    基于线阵CCD的二维码扫描识别实验

    李立富 仇渊亭 尹亚玲 李晓云 王春梅 郭超修

    华东师范大学

    摘    要：

    利用物理实验室常用线阵CCD实验系统对二维码进行逐行扫描, 得到二维码每一行的条纹图, 通过编程运用Python-pillow库解将采集到的条纹图还原为可识别的二维码。本实验可以作为大学物理实验的拓展内容。

    关键词：

    线阵CCD; 二维码; 识别;

    收稿日期：2018-11-7

    Experimental Research on Two-dimensional Code Scanning and Recognition Based on Linear Array CCD

    LI Lifu QIU Yuanting YIN Yaling LI Xiaoyun WANG Chunmei GUO Chaoxiu

    East China Normal University

    Abstract：

    Linear array CCD experiment systems are commonly used in college physics experiments.We use this instrument as an input device to scan the two-dimensional code line by line to obtain stripe figures.Then we programmatically use the Python-pillow library to restore the collected stripe figures to an identifiable two-dimensional code.This experiment is an extension of college physics experiments.

    Keyword：

    linear-array CCD; two-dimensional code; recognition;

    Received： 2018-11-7

    电荷耦合器 (Charge Couple Device, CCD) 是一种以电荷为信号载体的图像传感器, 它有光电转换、信息存储、延时和将电信号按顺序传送等功能, 且集成度高、功耗低, 因此是图像采集及数字化处理必不可少的关键器件。线阵CCD是一种特殊的矩阵式电荷耦合器件, 其传感器只有一行感光元素[1], 具有高扫描频率和高分辨率的特点[2], 广泛应用于工业生产和科研教学。至今, 基于线阵CCD开展的大学物理实验包括有利用线阵CCD进行条形码扫描与识别实验[3]、非接触测量物体尺寸实验[4,5]、非接触测量物体的振动实验等[6]。

    二维码 (Two-dimensional bar code) 是指在一维条码的基础上扩展出另一维具有可读性的条码, 可以认为是把一维条码的高度截短并按行堆积, 被设备扫描后可获取其中所包含的信息[7]。二维码与一维条形码有许多共同功能, 在从一维向二维发展的过程中, 也具备了二维码独特的功能[8]。二维码可以分为矩阵式二维码和行列式二维码, 其中较为常用的是矩阵式二维码。将黑、白像素块按照编码原理分布在矩形空间中, 其中黑色像素块表示二进制“1”, 白色像素快表示二进制“0”, 不同的排列组合确定了二维码所代表的意义。具有代表性的矩阵式二维码有:Code One、Maxi Code、QR Code、Data Matrix等。其中QR Code (Quick Response code, 即快速响应矩阵码) 具有超高速识读、全方位识读、纠错能力强、能有效表示汉字等特点, 在我国具有广泛的应用[9]。因此, 选用码制为QR Code的二维码来作为实验对象具有普适性。

    线阵CCD是大学物理实验中常用的测非接触式测量设备, 本文提出并完成一个实验拓展, 使用线阵CCD作为输入设备对二维码进行逐行扫描, 得到每一行的条纹图, 并用Python语言进行编程, 将采集得到的条纹图用Python-pillow库还原成可识别的二维码, 最后调用Python-Zxing库对二维码进行解析[10,11]。

    1 实验原理

    线阵CCD实验仪基本原理如图1所示。

    图1 实验光路原理图   下载原图

    当一束平行光入射到线阵CCD上时, 可以探测到的光强信号分布为一条直线。此时将二维码的某一行放置在光路中, 该行中的黑色像素块将平行光遮挡, 而白色像素块区域, 光可以直接透过, 透过白色像素块的光通过成像物镜成像在线阵CCD像敏面上, 这时线阵CCD探测到的光强信号分布如图1右侧所示的明暗分布, 明暗条码间的相对宽度与原图像的相对宽度一致。形成如图2 (a) 所示的一维时序信号, 经过实验仪软件二值化处理后得到如图2 (b) 所示的条形方波信号。

    图2 单行二维码扫描输出波形   下载原图

    图2单行二维码扫描输出波形, 其中 (a) 为一维时序信号, (b) 为条形方波信号。扫描二维码的实验装置如图3 (a) 所示, 图3 (b) 所示为可调节的狭缝和透光板。

    图3 实验装置 (a) 和可调节宽度狭缝 (b) 的示意图   下载原图

    线阵CCD的像元尺寸为11 μm×11 μm, 有效像元数为2 700。实验中要确保白色LED光源中心、矩形矩形狭缝和线阵CCD物镜中心应处于同一光轴。当白色LED平行光源透过矩形狭缝照射在二维码的某一行上, 透射的光被物镜和线阵CCD接收, 线阵CCD将信号输入到计算机中进行处理, 最后以光强分布曲线和条形码的形式呈现在屏幕上。依次移动二维码, 使得每一行依次经过矩形狭缝后被采集。因此可以通过这个方法依次进行二维码的采集, 然后通过编程将其还原和识别。

    2 实验内容

    结合实验器材及其可行性, 我们自制了一个码制为QR Code、尺寸大小为21×21 (Version 1) 和文本内容为“ECNU”的二维码, 如图4所示。仿照条形码的设计, 我们在二维码的左右两侧加上宽度均匀的辅助线, 以方便图像处理, 如图5所示。

    图4 内容为“ECNU”文本的QR Code   下载原图

    图5 添加辅助线的QR Code   下载原图

    为了简化实验操作, 我们制作了可调节矩形狭缝大小的支架和可移动的透光板 (见图3 (b) ) , 前为狭缝, 后为透光板。实验利用线阵CCD实验仪器, 将打印在常规A4纸上的二维码固定在支架透光板上。二维码的单位像元宽度即为单行二维码的宽度, 测量二维码正方形单位像元宽度为d1, 调节矩形狭缝宽度d2, 使得d1=d2。然后调节支架, 使矩形狭缝距试验台高度h1恰好与线阵CCD物镜最宽处距试验台高度h2相等。将二维码第一行对准矩形狭缝, 然后进行扫描并采集图像, 之后移动透光板, 使得二维码的每一行对准矩形狭缝, 重复上述步骤, 依次进行扫描。

    进行编程, 将二维码的每一行条纹图进行图像处理, 即裁剪图像多余部分和按照扫描顺序进行图片拼接, 最后解析还原得到的二维码, 读取相关内容。

    3 实验结果与分析

    3.1 逐行对二维码扫描得到条纹码

    依据实验内容逐行扫描图5所示的二维码, 得到21张原始图像, 由于图像众多, 我们仅展示部分图像。通过移动透光板, 使得每一行图像经过矩形狭缝后在线阵CCD试验仪软件上显示光学采集信号, 每一行对应不同的光学波形信号, 图像下方的条形码经过实验仪软件二值化处理后得到。图5中二维码的第1、8和21行, 分别对应二维码的定位图案区域和纠错区域, 它们的实验测量与比对结果如图6 (a) , (b) 和 (c) 所示。

    图6 二维码第1行 (a) , 第8行 (b) 和第21行 (c) 的实验测量结果   下载原图

    3.2 图像处理

    实验中共采集到21个条形码, 按照扫描顺序编号为1～21号, 这些条形码的尺寸都是有差异的, 所以我们使用程序批量化对条形码进行裁剪、缩放。裁取辅助线之内的图像, 然后从宽度中部开始向下裁取40像素, 之后统一缩放为尺寸为1 000×40像素的图片。同样, 我们以第1个条形码为例说明图像处理过程, 如示意图7所示。

    图7 图像处理示意图   下载原图

    3.3 图像还原及OR code解析

    编写程序将每个标准化的条形码按照扫描顺序拼接成可识别的二维码, 如图8所示。最后调用Python-Zxing库解析二维码。

    图8 图像还原所得二维码   下载原图

    由图9可以看出, 解析结果为{'ECNU', 'QR_CODE', 'TEXT', [ (197.5, 700.0) , (196.5, 140.0) , (805.0, 140.0) ]}。其中花括号中第一项为二维码的内容, 与实验初始设置的“ECNU”一致;第二项为其码制, 为QR码;第三项为二维码内容的类型, 说明实验所用二维码内容为文本类型;最后三项为二维码位置探测图形的坐标。

    图9 二维码解析结果   下载原图

    通过对图6的分析可知, 实验仪测得的条形码的相对位置、相对宽度与对应的二维码的黑白像元的位置、宽度是一致的。即使二维码的单位像元宽度很小, 但是我们可以通过改变矩形狭缝宽度或者选区适当的光强, 经过逐次测量二维码的每一行图像, 在经过图像处理, 就可以将其解析出来。

    从图像处理的最终结果 (图8) 来看, 还原所得的二维码仍有些部分是不能完全对齐的, 产生凹凸不平的现象, 其原因可能如下: (1) 由于光强不均匀, 导致二维码的黑白像元的交界处存在色差, 在图像裁剪过程中产生误差; (2) 矩形狭缝的边缘有一定的透光性; (3) 打印的二维码样品本身的对比度不够明显。但是只要所有行上下对应的像元的错位距离之和不超过一个单位像元的宽度, 该二维码仍能被正常解析。

    4 实验总结

    利用线阵CCD实验仪作为输入设备对二维码进行扫描, 并且通过简单的图像处理将条形码还原为二维码, 体现了二维码的设计哲学, 可以简单地将二维码理解为一维条码的高度截短并按行堆积。本实验设计的可变宽度矩形狭缝可适应各种单位像元大小的二维码, 制作的过程中要考虑器材是否能够契合何种实验条件, 锻炼了学生的设计能力和动手能力。图像处理只需简短的程序即可完成实验目的, 拓展了知识面并且让学生了解到实用程序批量化处理数据的高效率型, 同时也启发学生多思考、多拓展, 明白在原有的大学物理实验平台上也能够完成许多有趣的实验。
