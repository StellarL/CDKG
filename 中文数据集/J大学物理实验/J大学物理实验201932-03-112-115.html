    基于Python的牛顿环实验数据拟合及分析方法

    马天成 李全彬

    江苏师范大学物理与电子工程学院

    摘    要：

    牛顿环实验是经典的大学物理实验, 其实验数据量大, 手工计算非常容易出错。基于Python的第三方库对牛顿环实验数据进行拟合分析, 利用numpy进行数据的预处理和不确定度的计算, 利用scikit-learn建立线性回归模型拟合数据并求解, 最后利用matplotlib绘制图像实现数据的可视化。本方法能快速处理实验数据, 绘制拟合曲线并输出相对不确定度, 形象而又直观, 对牛顿环实验数据的分析和处理有很好的辅助作用。

    关键词：

    python; 牛顿环; 拟合; 一元线性回归; 不确定度;

    The Method of Data Fitting and Uncertainty Analysis in Newton Rings Experiment Baesed on Python

    MA Tiancheng LI Quanbin

    College of Physics and Electronics Engineering, Jiangsu normal university

    Abstract：

    Newton rings experiment is a classic university physics experiment.Its experimental data is large, manual calculation is often error prone.In order to contribute to simple data processing of Newton rings experiment and a visual representation of the result of that, there is a convenient method composed of fitting analysis on the experimental data based on the third-party function libraries of python, data preprocessing and calculation on the uncertainty by numpy, establishing a linear regression model with scikit-learn to fit the data as well as solve the equation and using matplotlib to achieve data visualization.

    Keyword：

    python; Newton's rings; fitting; unary linear regression; uncertainty;

    牛顿环是牛顿发现的一种薄膜干涉现象。当一个曲率很大的凸透镜的凸面紧贴平滑的玻璃时, 凸透镜和玻璃之间形成两边厚而中间薄的空气薄膜, 经单色光垂直入射凸透镜平表面, 便会产生明暗相间、间距随中心点的距离而逐渐变窄的同心牛顿环。作为经典的光学实验, 牛顿环实验被广泛收录于各大学物理实验, 但是其数据量大、数据计算复杂等问题也影响着此实验的数据处理效率。因此, 使用计算机处理实验数据进而达到化难为易、缩减时间的目的, 显得尤为重要。

    赵昆涛[1]利用Python的PyqQt模块设计了图形用户界面让实验数据录入更加直观, 但是在分析数据不确定度时, 仅考虑了A类不确定度而忽略了B类不确定度的影响;徐海英[2]使用Origin 7.0得到了相关性很强的拟合结果, 但是代码可读性不高, 不易让人准确理解。徐少刚[3]利用Matlab对牛顿环数据进行处理, 但Matlab在物理教育学生中并不十分普及, 软件门槛相对较高。郭晓春[4]使用Excel实现了对数据的线性拟合, 但还需手动计算不确定度, 没有明显减轻计算量。

    利用Python的第三方机器学习库scikit-learn来处理牛顿环实验的数据, 并通过matplotlib库实现数据及其分析结果的可视化。

    1 工具介绍

    Python[5]是由Guido van Rossum创造的面向对象的高级程序设计语言, 其设计风格优雅, 功能丰富, 使得初学者也很容易入门。此外, 该软件采用开源设计, 具有丰富的专用第三方库和众多开源科学计算软件包的接口, 已经成为应用于科学计算、数据库、网络工程、GUI设计等多领域的高级语言。

    采用Python的numpy, scikit-learn和matplotlib三个第三方库。numpy是一个开源的、高性能的数值计算库, 提供快速高效的数组矩阵运算。将牛顿环的实验数据全部通过该库转化为数组格式, 避免了循环语句的使用, 简化了处理的难度。scikit-learn是Python的机器学习库, 是用于数据挖掘、数据分析的工具, 通过数据训练, 可以轻松实现数据的拟合回归预测、聚类、模型选取等复杂算法。采用其中的一元线性回归模型来完成牛顿环实验数据的拟合。matplotlib是Python经典的二维画图库, 可以简单地画出折线图, 柱状图, 散点图等各类二维图像, 应用该库实现数据的可视化。

    2 实验原理及不确定度分析

    2.1 实验原理

    牛顿环是光等厚干涉的典型实验。当一束单色平行光垂直入射时, 形成明暗相间的干涉条纹。实验中是对暗纹进行测量, 根据波动光学的知识, 产生暗纹的条件和平凸透镜曲率半径的几何关系可知[6]:

    R=Dk24kλ          (1)

    即:

    Dk2=4λRk (2)

    一元线性回归模型y=a+bx, 设x=k, y=Dk2, 有b=4λR, 则:

    R=b4λ          (3)

    实验中采用钠光灯作为光源, 波长λ=589.3 nm。只要拟合出以环数k为x轴、以第k环直径的平方Dk2为y轴的数据的斜率b, 即可求出凸透镜的曲率半径R。

    2.2 不确定度分析

    牛顿环实验数据的不确定度包括A类不确定度和B类不确定度。

    (1) 不确定度的A类分量由实验标准差表征, 参考文献[7]给出了一元线性回归的方差估计公式:

    e2⌢=1n-2∑i=1n (yi-yi⌢) 2          (4) UA=e2⌢          (5)

    式子中n-2为自由度。

    (2) 不确定度的B类分量大体上认为由两个因素导致, 分别为单次测量而引入的仪器误差限Δ仪所导致的不确定度UB1和测量过程中条纹的定位精度所引入的不确定度UB2。由参考文献[8]可知:

    UB1=Δ仪3          (6) UB2=13e¯          (7)

    其中e¯=12 (Emax+Emin) , Emax、Emin 为测量范围中最大环与最小环的条纹宽度

    (3) 合成不确定度为Uy, 即:

    Uy=UB12+UB22          (8)

    (4) 设拟合出的斜率的不确定度为Ub:

    Ub=Uy (Xi-x¯) 2          (9)

    (5) 曲率半径的不确定度为UR:

    UR=Ub4λ          (10)

    最后结果为:R=R±UR, 相对不确定度为:

    ER=URR×100%          (11)

    3 实现过程

    3.1 软件环境配置

    本方法在Win10操作系统上实现, 基于Python3.6版本, 并利用Python自带的安装第三方库工具pip安装numpy, matplotlib和scikit-learn三个库。

    3.2 实现框图

    首先录入牛顿环实验数据, 再用numpy进行数据的预处理, 然后调用scikit-learn建立线性回归模型并求解, 最后用numpy分析不确定度并用matplotlib画出图像。实现框图如图1所示。

    3.3 实现代码

    (1) 导入第三方库:

    import numpy as np

    import matplotlib.pyplot as plt

    from sklearn import linear_model

    (2) 输入数据, 使用numpy的array () 函数把数据转化成行向量, 再用reshape () 函数把行向量转化为列向量, 从而满足sklearn中线性回归模型只接收列向量的要求。

    d_left=np.array ([])

    d_right=np.array ([]) #从第一环开始输入数据, 数据用逗号隔开

    y= (d_left-d_right) **2

    x=np.arange (3, 22) .reshape (-1, 1)

    y=np.array (y) .reshape (-1, 1)

    (3) 定义scikit-learn库的线性回归模型

    lrmodel=linear_model.LinearRegression ()

    (4) 利用线性回归模型的fit () 函数进行数据训练:

    lrmodel.fit (x, y)

    (5) 求解模型并用result、b接收拟合出来的结果:

    result=lrmodel.predict (x) #线性回归模型拟合出的数值

    b=lrmodel.coef_#斜率

    R=b/ (4*589.3e-6) #r为求得的曲率半径

    (6) 利用numpy的矩阵运算函数进行不确定度分析:

    n =y.size

    e_2=np.sum ( (y-result) **2) / (n-2) #方差

    u_B1=0.001/np.sqrt (3) #不确定度B1

    u_B2=0.04/6#不确定度B2

    u_y=np.sqrt (e_2+u_B1**2+u_B2**2) #合成不确定度Uy

    u_b=u_y/np.sqrt (np.sum ( (x-np.mean (x) ) **2) ) #斜率的不确定度Ub

    u_R=u_b/ (4*589.3e-6) #曲率半径的不确定度UR

    E_R=u_R/R#百分差

    print ("E_R=%f"%E_R)

    (7) 画图

    plt.figure ()

    plt.scatter (x, y, label=”true value”) #画出实验数据的散点图

    plt.plot (x, result, ”r”, label=”predict value”) #画出拟合直线

    plt.title (“b= %f R= %.2fmm”% (b, R) ) #在图上方标出数据

    plt.xlabel (“k”) #x轴名称

    plt.ylabel (“D^2”) #y轴名称

    plt.legend () #显示示例

    plt.show ()

    5 结 论

    利用Python软件, 借助一元线性回归模型对牛顿环实验数据进行拟合, 绘制拟合曲线并计算不确定度, 实现了牛顿环实验数据的可视化分析。消除了手动计算耗时较长, 易出错的弊端。将本代码移植到移动终端上运行, 为实验数据处理提供更灵活更便利的条件。
