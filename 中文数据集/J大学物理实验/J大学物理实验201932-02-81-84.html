    摘    要：

    针对温度传感器测量误差受温度影响比较大, 必须使用温度补偿的方法对测温数据值进行修正。本文介绍了一种温度补偿的方法:主要采用oringin软件利用最小二乘法拟合实验数据的方式, 进行系统误差分析处理后再对温度传感器进行温度精确标定, 可以有效消除传感器非线性误差。对AD590传感器温度精确标定后实测电压与温度线性关系良好, 证实此方法可以很好的修正温度对AD590传感器测量误差的影响, 系统测量精度可达±0.02℃。将此实验的温度标定后的AD590传感器直接应用到其它实验中加深学生对温度传感器具体应用的理解。

    关键词：

    AD590温度传感器; 温度补偿; 最小二乘法; 线性拟合; oringin软件;

    Least-Squares Methods in Linear Fitting of Temperature Sensor Measurement Data and Its Application

    SHANG Jimin WANG Haiyan JIANG Fengchun WEI Maocai CHEN Peng

    College of Physics and Electronic Engineering, Zhengzhou University of Light Industry

    Abstract：

    In view of the temperature sensor measurement is more affected by temperature, it must be used a method of temperature compensation to calibtrate the temperature.A temperature compensation method is introduced:mainly using the Origin software by adopting the least-squares linear fitting the experimental data, it is calibrated temperature accurately for temperature sensor after processing the error analysis and handling, so that the nonlinear error of temperature sensor is can be eliminated mostly.After the AD590 temperature sensor is accurately calibrated, the datas of measuing voltage and temperature indicate good linear relationship.The experiment proves that it can be good emperature compensation for AD590 temperature sensor measurement error, and the effect of system measurement precision can reach ±0.02 ℃.

    Keyword：

    AD590 temperature sensor; temperature compensation; least-squares methods; linear fitting; Origin software;

    Received： 2018-11-20

    传感器是温度测量仪表的核心部分, 即是指能感受温度并转换成可用输出信号的传感器, 按照传感器材料及电子元件特性分为热电阻和热电偶两类, 它们广泛应用于工业、农业、商业等部门。AD590温度传感器是美国ANALOG DEVICES公司开发的一种温度传感器, 它是利用PN结正向电流与温度的关系制成的电流输出型两端温度传感器, 即温度信号转换为电流信号输出。具有精度高、动态电阻大、响应速度快、线性好、使用方便等特点[1]。近年来诸多工科高校针对本科生开设温度传感器测试及半导体致冷控温实验等温度传感器使用系列的热学实验, 该类实验主要训练学生理解并掌握温度传感器的测温原理及准确标定传感器的温度电压关系。结合我校开设该类实验的实际教学情况, 对于学生如何理解并利用最小二乘法这一数据处理工具来进行温度传感器的温度精确的标定, 从而提高动手能力和分析解决问题能力显得尤为重要。

    按实验电路接线 (图1所示) , 将控温传感器Pt100和温度传感器AD590分别插入加热井中。

    首先在加热状态下进行P.I.D自整定。将仪器温度设置为比环境温度高至少20 ℃的温度进行P.I.D自整定, 并将自整定开关设置为开启状态。按下“加热”按钮, 开始P.I.D自整定。P.I.D自整定结束后, 将温度设置从环境温度起加热, 每隔10 ℃设置一次, 每次待温度稳定2 min后, 记录一次输出电压, 直到100 ℃为止。同时将输出电压从+5～+10 V 改变, 观察输出电压是否有变化。加热实验结束后, 先将仪器调节成致冷模式再进行致冷实验。在致冷状态下先进行P.I.D自整定。将仪器温度设置为比环境温度低至少15 ℃的温度进行P.I.D自整定, 并将自整定开关设置为开启状态。按下“致冷”按钮, 开始P.I.D自整定[1,2]。

    P.I.D自整定结束后, 将温度设置从环境温度起致冷, 每隔5 ℃设置一次, 每次待温度稳定2 min后, 记录一次输出电压, 一直到-15 ℃止。同时将输出电压从+5～+10 V 改变, 观察输出电压是否有变化。

    2 最小二乘法线性拟合原理

    高斯 (Gauss) 从解决一系列等精度测量最佳值问题中建立了最小二乘法原理, 即被测量真值的最佳值乃是能使各次测量偏差的平方和为最小的那个值。用数学表达式可写成

    ∑i=1k (xi-x最佳) 2=S (x最佳) =min

    应用最小二乘法则拟合出来的直线称为直线拟合。设y和x两个物理量有函数关系y=a+bx, a、b是两个待定参数, 现在由x和y的n组测量值{xi, yi} (i=1, 2, …, n) 对a、b参数进行估计。

    设x和y两个量中x的误差相对很小, 可忽略, 而y有测量误差, 且是等精度测量。按照最小二乘法原理, 参数a、b的估值a^, b^应使下式取最小值。

    S (a^a^, b^) =∑i=1n[yi-y (xi) ]2=∑i=1n[yi- (a^+b^xi) ]2

    为使S达到极小值, 就应该使S对a^、b^的一阶偏导数为零, 二阶偏导数大于零。实际上, 由于S总是大于零, 因此, 必然存在极小值。由一阶偏导数为零并经整理后得到:

    a^=y¯-b^x¯,  b^=x¯⋅y¯-xy¯ (x¯) 2-x2¯

    这样依据n组测量值{xi, yi} (i=1, 2, …, n) 确定了一个最佳的线性方程 (回归方程) y=a + bx, 方程中的两个待定参数a、b取其最佳估计值a^、b^。r表示两变量之间的函数关系与线性的符合程度, |r|的值总在0和1之间。|r|值越接近于1, 说明实验数据分布越能密集在所得直线的近旁, 越符合求得的直线, 即拟合直线有意义[1,3]。

    利用最小二乘法的公式可以进行直线拟合计算得出得出:B=1.02uA/K;A=272.8uA r=0.998

    其中A是灵敏度 (表示温度传感器的温度响应速度) 、B是拟合直线的斜率, r表示拟合直线的相关系数[4,5,6,7]。

    利用oringin软件处理数据得出结果, 并利用软件菜单命令“Analysis/Fit linear ”将曲线拟合成直线, 其斜率和相关系数在图中可以自动得出:B=1.017 63 uA/K, r=0.997 43, 如图2所示。

    在实际的教学过程中, 我们布置学生使用拟合公式计算、作图两种方法来比较处理实验数据结果的精度高低。明显地, 使用最小二乘法利用oringin进行线性拟合得到的温度传感器灵敏度、斜率及相关系数更加精确。将采用拟合后得到的数据进行温度传感器AD590温度的标定后再进行检验, 检测实验电路如图3所示。

    4 讨论及应用

    在实际的教学过程中, 我们发现:在做完温度传感器的整定, 并利用最小二乘法进行线性拟合后得到的数据处理结果, 无法直接应用, 即对所使用的仪器 (我校采用复旦天欣仪器公司FD-TM型温度传感器测试及半导体致冷控温实验仪) 进行具体的温度精确标定, 也即是无法把拟合数据精确反馈给仪器并进行设置。这样学生对做温度传感器AD590在实际环境温度测量下的应用特性实验—电压与温度对应关系的实验意义不明确。针对此问题:课题组结合我校开设的一个传统实验项目“惠斯通电桥测电阻的温度系数”, 我们直接将此实验温度整定后的AD590温度传感器直接应用到此实验项目中, 可将传感器的测温端直接插入需要升温的待测铜线圈所处的加热电热杯里的变压器油中, 可以非常精确的测量在快速升温过程中的油温, 代替原来测量温度滞后、精度低的电子温度计, 大大提高了测量的精度。也加深了学生对温度传感器具体应用的深入理解。

    5 结 论

    我校课题组在关于温度传感器实验教学中对原有教学内容进行较大的改革, 将最小二乘法这一数据处理工具融入温度传感器的温度精确标定这一传统的实验内容中, 对工科专业大学物理实验教学中温度量和电学量的对应关系, 对学生深入理解温度传感器AD590原理及使用方法, 对比分析物理实验数据处理工具的优劣和最小二乘法的理解掌握等多方面均有较好的效果, 实际教学效果反映良好。
