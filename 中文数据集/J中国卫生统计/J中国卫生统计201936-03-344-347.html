    基于平均影响值的SVM在遗传数据疾病分类和特征提取中的应用

    张阳阳 曹红艳 武淑琴

    山西医科大学卫生统计教研室

    摘    要：

    目的 探讨基于平均影响值 (MIV) 的支持向量机 (SVM) 在遗传数据疾病分类预测和变量筛选中的应用, 为遗传数据的疾病分类与特征提取方面提供方法学参考。方法 以GAW18 (genetic analysis workshop 18) 数据为例, 采用基于MIV的SVM建立预测模型, 并和logistic回归模型、SVM、多层感知机和决策树分类模型进行比较分析, 评价基于MIV的SVM预测分类和变量筛选效果。结果 经过平均影响值的支持向量机算法处理后, 六个SNPs位点 (13_28567172、3_127394820、1_1658093、9_123969834、1_174996637、17_17498492) 组合的变量子集, 获得78.125%的分类准确率, 明显优于其他分类模型。结论 基于MIV的SVM能比较有效的在实现遗传数据变量筛选的同时提高分类预测能力, 避免了变量间的交互作用, 为探索各种疾病发病机制和寻找易感SNPs位点提供线索, 具有一定的研究和应用价值。

    关键词：

    支持向量机; 平均影响值; 单核苷酸多态性;

    Application of SVM based on MIV in the Classification and Feature Extraction of Genetic Data

    Zhang Yangyang Cao Hongyan Wu Shuqin

    Department of Health Statistics, Shanxi Medical University

    Abstract：

    Objective The application of support vector machine (SVM) based on average impact value (MIV) in genetic data classification, prediction and variable selection is discussed to provide methodological reference for disease classification and feature extraction.Methods Taking GAW18 data as an example, a prediction model was built based on MIV SVM, and compared with the logistic regression model, the SVM, the MLP and the tree algorithms model, and MIV based SVM prediction classification and variable selection effect were evaluated.Results After the support vector machine algorithm with MIV is processed, the subset of six SNPs loci (13_28567172、3_127394820、1_1658093、9_123969834、1_174996637、17_17498492) is combined to get 78.125%, which is obviously better than that of other models.Conclusion The SVM based on MIV can be more effective in improving the classification prediction ability while implementing genetic data variables screening.It avoids the interaction between variables, and provides clues for exploring the pathogenesis of various diseases and finding vulnerable SNPs loci.It has research value and application value.

    Keyword：

    Support vector machine; Mean impact value; SNPs;

    随着DNA测序技术的发展和高通量测序的出现, 后续产生的大量遗传数据对统计分析方法提出新的要求。单核苷酸多态性 (single nucleotide polymorphisms, SNPs) [1]是基因组水平上单个核苷酸变异所引起的DNA序列多态性, 在人类基因组中广泛存在, 估计其总数可达到300万个甚至更多[2]。本文拟采用遗传分析工作组18 (genetic analysis workshop 18, GAW18) 中134个相互独立个体的全基因组测序数据, 每个样本拥有较多SNPs, 高维度和较小的样本量给疾病分类和特征提取带来了极大的挑战。而变量降维和筛选常见的方法有主成分分析、Wrapper法[3]、filter法[3]、逐步多元线性回归[4]和随机森林[5]等。本文采用Wrapper法的思想以分类的精度为指标, 在完成变量筛选的同时获得较高的分类准确率。由于该方法的特征寻找依赖于分类器的性能, 所以合适分类模型的选择尤为重要。

    针对分类模型的选择问题, 最初采用由Vapnik于1968年提出的支持向量机 (support vector machine, SVM) [6]。支持向量是一种机器学习算法, 广泛应用于数据挖掘、文本分类和图像识别等多个方面。该算法以统计学习理论为基础, 结构风险最小化为原则, 既考虑渐进性能的要求, 又在有限的信息条件下追求最优的结果, 是一种具有坚实理论基础的小样本学习方法[7]。该算法的判别函数最终由少数几个支持向量所决定, 在某种意义上避免了“维数灾难”。所以SVM在面对小样本和高维度数据时有较好的优越性。其理论基础和算法本质决定了该算法在生物遗传数据上良好的适应性。但由于SVM是一种将全部变量纳入模型进行训练和预测的机器学习方法, 无法对变量进行特征筛选。因此需要寻找合适的方法来解决该问题。平均影响值 (mean impact value, MIV) 被认为是神经网络中评价变量相关性最好的指标之一[8]。故考虑将SVM和MIV结合起来, 通过评价各个变量相对结局变量的影响程度, 找到对结果有较大影响的自变量, 继而实现使用SVM进行变量筛选的过程。因此, 本文提出一种基于平均影响值的支持向量机的方法, 能相对有效的解决生物遗传数据的特征提取问题。使用较少的变量构建更好的分类预测模型, 在提高分类预测模型准确率的同时实现了变量筛选。本文以GAW18数据为例, 采用基于平均影响值的支持向量机进行分析, 一方面为遗传数据的分析和处理提供方法学参考。另一方面, 对特征变量的提取有助于使人们的注意力集中到少数的变量上, 使得实验研究更具有针对性。

    原理与方法

    支持向量机最初是为了解决分类问题而提出的一种机器学习方法[6,9]。本研究将SNPs数据的每个样本与样本空间中的样本点一一对应。其基本原理是在样本空间中寻找一超平面, 使得高血压和非高血压样本点分别分布在超平面的两侧。如图1所示, 低维度的样本空间线性不可分, 通过非线性映射到高维空间后变得线性可分。在数据空间中, 设用于分离的超平面方程是ω·x+b=0 (ω是可调的权值向量, b是偏置) 。设最优的ω和b为ω0和b0, 则有:ω0x+b0=0。支持向量是最靠近决策面的样本点 (xi, yi) , 满足条件:ω·xi+b=yi, yi=±1。设 (x1, 1) 和 (x2, -1) 分别为最靠近决策超平面支持向量, 该两点满足ω·0x1+b=1和ω·x2+b=-1;则该两点的间隔为dis=ω∥ω∥⋅ (x1-x2) =2∥ω∥。对于任意的样本点 (xi, yi) 均满足yi· (ω·xi+b) ≥1。

    图1 支持向量机原理示意图   下载原图

    支持向量的间隔问题最终归结为一个求最小∥ω∥22二次规划问题。使用Lagrange乘子法可以将该问题转换为原问题的对偶问题, 此时可以求得原问题的最优解:w0=∑i=1Νa*iyixix;b0=1-w0x[s], y (s) =1;。至此, 原函数得到解决获得判别函数:sgn (∑i=1Νai*yixix+b0) , 其中sgn是数学中的符号函数, a*i表示最优的Lagrange乘子, x为测试集中的样本。而线性不可分的情况则是通过升维将线性不可分的数据通过一定的方法映射到高维空间中, 在高维空间中使用线性回归对空间中的数据点进行分类或者回归。其中需要注意的是, 高维空间中训练点的内积可以用原来空间中的函数 (核函数) 进行替代。支持向量机 (SVM) 就是该算法的典型代表[10,11]。核函数与支持向量机的性能好坏有着直接的关系。目前最常用的核函数有四类, 分别为:高斯径向基核、多项式核、sigmoid核、B-样条核[7]。

    平均影响值 (mean impact value, MIV) 是评价各变量对结局变量影响程度的重要指标, 其符号代表相关的方向, 绝对值大小代表影响的程度[12]。MIV的具体做法为:构建一个SVM模型, 再将SNPs的训练样本p的每个变量在原来的基础上分别加或减某个值构成两个新的训练样本P1、P2, 将P1、P2作为测试集带入训练好的模型进行预测, 得到两个结果A1、A2, 两者的差值就是该变量变动后对输出产生的影响变化值 (impact value, IV) 。再将IV按照例数进行平均得到该自变量SNP相对于因变量高血压的MIV。计算出每一个SNPs变量相对高血压的MIV, 按照绝对值降序排列, 然后依次去除对结局变量影响最小的若干个SNPs变量, 剩下的变量作为SNPs变量子集, 分析其对样本的分类能力, 从中寻找具有最小SNPs数和最高分类率的候选子集。

    实例分析

    本研究通过对GAW18数据进行分析和研究, 探寻高血压发生和SNPs位点间的关系, 将有助于利用遗传数据从SNPs的角度对高血压患者进行筛查和预测。并提供一些可能与高血压发生相关的候选SNPs位点。

    1.变量选择和样本确定

    本次研究采用的数据源于GAW18, 一个国际上公开的用于研究稀有变异关联方法的模拟数据平台, 研究者针对GAW的组织者所发布的遗传数据进行分析。该数据包括20个墨西哥裔美国家庭的1043个个体的全基因组测序数据 (whole genome sequencing data, WGS) 及其相对应的纵向血压表型数据。

    GAW18数据仅提供了奇数染色体遗传数据, 包括四种数据类型:基因型数据 (chrN-geno.csv.gz) , GWAS数据 (chrN-geno.csv.gz) , 原始测序数据 (chrN-seq.vcf.gz) 以及变异剂量数据 (chrN-dose.csv.gz) 。本文针对变异剂量数据 (chrN-dose.csv.gz) 进行分析。GAW18的表型数据包括真实测量数据和模拟数据。GAW18的“GAW18-TheAnswers”中提供了具体的模拟研究设置情况, 包括主要的效应基因和功能SNPs。本研究将信息完整的134个无亲缘关系、互不相关的个体作为研究对象。高血压的诊断标准为收缩压 (SBP) >140mmHg, 舒张压 (DBP) >90mmHg, 测量期间使用高血压药物也诊断为高血压。

    由于四次血压测量中第一次测量缺失最少, 因此选择第一次测量结果作为结局变量。自变量则选择影响SBP和DBP的SNPs位点。前十五个分别影响收缩压和舒张压的SNPs位点中共有7个SNPs的位点同时影响SBP和DBP。将其合并后总共23个SNPs位点作为自变量。利用MATLAB软件实现, 采用基于平均影响值的支持向量机对该高血压数据进行分析。 (表1)

    2.建立模型和变量筛选

    针对与高血压相关性较强的23个SNP位点进行进一步筛选以获得更好的自变量期望维数。采用MIV与SVM相结合的方法来进一步的筛选SNPs信息。

    从样本集中随机抽取70个高血压患者和30个非高血压患者作为训练集, 剩余样本作为测试集。训练集和测试集样本量的比例约为3∶1。使用训练集对模型进行训练和预测, 并记录测试集的分类准确率。把所有的SNPs纳入作为自变量, 高血压结局作为因变量进行模型训练, 使用10-折交叉验证进行模型寻优, 寻找到最佳参数, 使用训练好的模型对预测样本进行预测, 记录分类准确率。按照变量的重要程度进行排序, 得到结果见表2。

    讨 论

    基于MIV的SVM方法通过考虑不同变量的改变对结局变量的影响, 同时实现了对遗传数据的特征提取和模型构建。本文以高血压遗传数据的SNPs位点分析为例, 识别了与高血压的发生可能相关的SNPs位点, 并与其他方法进行了比较, 说明基于MIV的SVM在生物遗传数据的分析中有较好的变量筛选和预测能力。

    本文通过基于平均影响值的支持向量机的机器学习算法, 构建不同的SNPs集合, 避免了SNPs位点间的交互作用。该算法在构建了一个利用SNP信息数据对遗传数据进行预测分类模型的同时实现了变量的筛选, 即为疾病的发病机制提供线索, 又为疾病易感SNPs的研究提供了候选位点。此外, 通过相关生物医学方面的文献发现本文获得特征SNP突变位点与高血压存在一定联系:基因RAI1 (retinoic acid induced 1) 在基因的表达和转录过程中起着重要作用[13], 与体内所有的生物化学功能相关;基因TNN (tenascin N) 与细胞的生长、移行相关;基因GSN (gelsolin) 与肌动蛋白的相关活动有关[14], 通过切断和封盖调节动态肌动蛋白丝组织, 而肌动蛋白与血管收缩的功能密切相关;基因SLC35E2 (solute carrier family35, memberE2) 与离子的跨膜运输有关, 钠离子和钾离子对高血压的影响广为人知;基因FLT3 (fms-related tyrosine kinase3) 与细胞的增殖和凋亡有关。以上结果均显示筛选出的SNPs位点与高血压有一定的相关性, 有可能是高血压发生的中间环节。需要指出的是, 基于MIV的SVM筛选出的位点, 仍然需要进一步的生物学方面的实验验证。

    总之, 基于平均影响值的支持向量机不但考虑到生物遗传数据的分析难点, 而且对高维变量进行了降维, 在较低的维度内获得了分类准确率更高的分类预测模型, 是一种针对遗传数据比较有力的分析方法。一方面实现了疾病在SNPs层面的预测和分类模型的构建;另一方面, 将人们的注意力从大量的遗传变量转移到较少的变量上, 为科学研究提供方向。基于MIV的SVM为探索疾病发生时体内的生物学过程和病理机制提供遗传方面的线索, 将有可能在日益繁复的遗传数据分析中发挥重要作用。
