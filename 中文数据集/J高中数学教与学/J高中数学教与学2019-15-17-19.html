    复合函数零点繁 流程图法援手施

    李艳

    江苏省扬州大学数学科学学院

    求解复合函数零点问题不仅涉及到函数的各种性质, 蕴含着丰富的数学思想与方法, 同时还由于其具有“内”、“外”两层函数的特殊性, 增加了解复合函数的零点个数问题时所需的思维难度, 因此也更方便考察学生数学思维的灵活性与创造性.本文主要探究兼具分段函数与复合函数两重身份的函数零点个数问题.

    例1 已知函数

    f (x) ={ax+1, x≤0, log2x, x>0,

    求h (x) =f[f (x) ]+1的零点个数.

    分析 波利亚指出:“在求解问题的过程中, 我们经常可以利用一个简单的类比问题的解答, 我们可能利用它的方法或者可能利用它的结果.”h (x) 是在f (x) 的基础上复合成的, 我们可以依据f (x) 的特征类比出h (x) 的特征.这种不直接求解h (x) 的方法看似迂回曲折, 事实上在数学解题中颇具实效.

    图1   下载原图

    解 第一步:换元.令f (x) =t, 则h (x) 零点的个数等于满足

    {f (x) =t, f (t) =-1

    的x的个数.由内而外, 先根据y=f (t) 的大致图形, 讨论f (t) =-1根的情况.

    由图1可知, 当a≤0时, f (t) =-1有唯一解t=12;当a>0时, f (t) =-1有两解t1=-1a<0, t2=12.

    第二步:根据第一步中t的范围以及y=f (x) 的特点, 得出f (x) =t的根的个数.这里借助流程图 (图2) 分析更加方便直观.

    图2   下载原图

    第三步:综合一二步, 得出结论.

    综上, a≤0时, h (x) 零点个数为1; a>0时, h (x) 的零点个数为4.

    评注 复合函数零点个数问题的常见解题思路是换元、解套, 将复合函数零点问题转化为两个简方程解的个数问题, 然后由内而外, 数形结合求解.但由于复合函数的繁杂性, 解题时屡屡碰壁, 借助“流程图”来辅助解答, 思路清晰直观, 事半功倍.

    例2 已知函数

    f (x) =x3-3x2+1, g (x) ={ (x-12) 2+1, x>0, - (x+3) 2+1, x≤0,

    则g (f (x) ) -a=0 (a为正实数) 的实根最多为___个.

    解 第一步:换元.令f (x) =t, 由内而外, 先讨论g (t) =a根的情况.

    如图3, 根据y=g (t) 的大致图形, 可以得出:

    图3   下载原图

    图4   下载原图

    ① a≥1.25时, y=g (t) 与y=a只有一个公共点, 相应t>1;

    ② 1<a<1.25时, y=g (t) 与y=a有两个公共点, 相应交点横坐标0<t1<12, 12<t2<1;

    ③ a=1时, y=g (t) 与y=a有两个公共点, 相应t1=-3, t2=12;

    ④ 0<a<1时, y=g (t) 与y=a有两个公共点, 相应-4<t1<-3, -3<t2<-2.

    第二步:根据第一步中t的范围, 结合图4中y=f (x) 的图象, 得出f (x) =t的根的个数.这里借助流程图 (如图5) 更加方便直观.

    图5   下载原图

    第三步:综合一二步, 得出g (f (x) ) -a=0根的个数.

    综上, 当1<a<1.25时, g (f (x) ) -a=0 (a为正实数) 的实根最多为6个.

    评注 本题稍微复杂些, 涉及两个层次的讨论. 一是结合图形对 a 作4 种分类; 二是由于是分段函数, 涉及两个不同的函数嵌套, 要根据函数变量 x 的取值范围, 相应地对 t 进行讨论, 进而数形结合, 整体求解.

    事实上, 流程图法不仅在求复合函数零点个数问题中有着广泛应用, 还可以为已知复合函数零点个数求参数取值范围的问题提供很好的思路.

    例3 已知

    f (x) ={|lnx|+2, x>0, 3-x2, x≤0,

    若f (f (x) ) =a只有5个不同的实根, 求实数a的取值范围.

    图6   下载原图

    解 令f (x) =t, 本题转化为

    {f (x) =t, f (t) =a

    中x的解的个数为5.

    图7   下载原图

    结合图6中y=f (x) 的图象，画出流程图，如图7.

    根据流程图7，当2<t3<e时，满足f (f (x) ) =a只有5个不同的实根．又f (2) =2+ln 2f (e) =3，数形结合，得到a的取值范围为2+ln 2≤a<3.

    评注准确作图至关重要，作出内外层函数图象，并利用流程图结合图象合理分类，整合求解，直观便捷．

    “流程图”法的使用是为了帮助学生在数学解题中更好地理解解题思路，从而学会解题．因此，教师应优化问题设计，合理引导学生思考、反思，坚持下去，学生解题思维就会越来越活跃，创新意识和应用意识就能随之提高，从而实现有意义的学习．
