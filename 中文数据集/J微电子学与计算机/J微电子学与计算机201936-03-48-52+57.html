    摘    要：

    受不利因素影响, 传感器网络中的数据不可避免地存在缺失, 阻碍上层应用对数据的分析与处理.使用能够自主提取和学习有效特征结构的卷积神经网络对缺失数据问题进行研究, 提出一种基于卷积神经网络的缺失数据填充方法.首先, 分别针对时间序列数据的时间相关性和传感器节点间的空间相关性, 使用卷积神经网络填充模型对缺失数据进行单维度相关性的填充.然后, 根据时间维度和空间维度的填充结果, 考虑时空相关性对缺失数据进行填充.最后, 在真实数据集上进行实验验证, 实验结果表明考虑时空多维度相关性的填充结果优于只考虑单维度相关性的填充结果, 并与BP模型进行对比, 验证了卷积神经网络填充模型的有效性.

    关键词：

    缺失数据; 填充; 卷积神经网络; 时空相关性;

    Missing data imputation approach based on convolutional neural network

    ZHANG Wang-juan XU Guo-yan LI Min-jia ZHU Shuai

    College of Computer and Information, Hohai University

    Abstract：

    In the sensor network, the missing data is inevitable due to many of interference factors, which hinders data analysis and processing.In this research, by using the convolution neural network (CNN) which can extract and learn effective features data independently, a missing data imputation approach based on convolution neural network was proposed. Firstly, for the temporal correlation of time series and the spatial correlation between sensor nodes, the CNN imputation model was used to estimate the missing data with single dimensional correlation. Then, based on the imputation results of temporal dimension and spatial dimension, the missing data were imputed by considering the multidimensional correlation. Finally, experimental verification is performed on the real data set. Experiment results show that the imputation result considering the temporal and spatial multidimensional correlation is better than the imputation result considering the single dimensional correlation. And compared with the BP model, the validity of the convolution neural network imputation model is verified.

    Keyword：

    missing data; imputation; convolution neural network (CNN) ; temporal and spatial correlation;

    1 引言

    随着计算机和通信技术、传感器技术及嵌入式技术的迅速发展, 传感器设备被大量运用于军事、医疗、工业等多个领域.传感器节点采集的数据能帮助人们更好地了解监测区域, 然而由于设备软硬件故障、网络传输问题及工作环境恶劣等不可控因素, 传感器网络中的数据存在不同程度的缺失, 降低了数据的可用性[1].同时, 现有的数据分析工具大多是基于完整数据集提出的, 无法直接处理含有缺失数据的不完整数据集, 因此如何有效处理缺失数据成为当前需要解决的问题.

    直接丢弃缺失数据简单易行, 但当缺失数据比例较大时, 此方法会造成原始数据失真, 损害推理能力. 因此, 对缺失数据进行填充是一种更合理且非常必要的方法.对于缺失数据填充方法的研究, 前人已有大量的研究成果.均值填充是典型的基于统计的方法, 此方法可保持序列的均值不变, 但填充精度较低.K最近邻填充法通过K个最近邻居来估计缺失数据, 但K值的选择限制了该方法的实用性.针对传感器网络数据的特点, 文献[2]通过属性约简划分属性的重要性, 再根据属性的重要程度选择相应的填充算法.文献[3]考虑数据的时间相关性, 使用局部时间索引 ( (Local Time Index, LTI) 策略对时间序列中存在的缺失数据进行填充.文献[4]考虑数据时空相关性, 使用概率主元分析 (Probabilistic Principal Component Analysis, PPCA) 和混合概率主元分析 (Mixed Probabilistic Principal Component Analysis, MPPCA ) 论证了同时使用时间和空间信息可以提高填充的准确性.

    此外, 深度学习模型[5,6,7]通过其非线性和自学习的特点可对非线性函数进行逼近, 对缺失数据进行估计和填充, 如使用BP神经网络估计缺失值[6,7], 但考虑到参数学习效率, 实际应用中模型的构建一般不超过三层[8].卷积神经网络 (Convolutional Neural Network, CNN) 是一种常用的多层深度学习模型, 通过卷积和池化操作可以提取低层的局部特征, 然后在更高层综合局部特征得到全局特征.自2012年Krizhevsky等人使用扩展深度的CNN在ImageNet图像分类竞赛中取得最佳分类效果, 使得CNN越来越受到研究者的关注.当前, CNN在图像分类、人脸特征提取、图像语义分割等多个方面表现突出, 已有研究表明, CNN在数值回归问题上也有较好的效果[9].

    基于以上理论基础, 本文考虑将数据的时空相关性与深度学习模型相结合, 提出一种基于卷积神经网络的缺失数据填充方法.

    2 相关工作

    2.1 卷积神经网络

    卷积神经网络[8]的基本结构由输入层、卷积层 (Convolutional Layer) 、池化层 (Pooling Layer, 也称为下采样层) 、全连接层和输出层组成.在卷积神经网络中, 采用卷积层和池化层交替设置的方式进行模型的构造, 通过对多维矩阵进行逐层特征提取, 学习数据内部的信息.与BP神经网络相比, 虽然它们拥有相似的训练方式, 但全连接神经网络的深度较浅, 而CNN的局部连接、权值共享和池化操作, 可减少连接数目和训练参数, 增加神经网络的训练深度且易于模型的训练.

    卷积层包含多个特征面, 每个特征面中的神经元通过卷积核与上一层的特征面进行局部相连接, 使用非线性函数对局部加权和进行激活, 从而获得卷积层中每个神经元的输出值.卷积层可描述为:

    Hi+1=f (Ai⨂Wi+1+bi+1) (1)

    式中, Hi+1为卷积层的输出特征矩阵;Ai为原始输入矩阵或上一层池化层的输出特征矩阵;Wi+1为第i+1层卷积核权重向量;⨂表示进行卷积操作;bi+1为第i+1层的偏置;f (•) 为非线性激活函数.

    池化层同样包含多个特征面, 其每一个特征面与卷积层的一个特征面唯一对应, 因此池化操作不会改变特征面的个数.池化层是采用一定的下采样操作对特征图进行二次提取特征的过程, 常用的池化法有最大池化、均值池化和随机池化.池化层可描述为:

    Ai=fsub (Hi-1) (2)

    式中, Ai为池化层的输出特征矩阵;Hi-1为第i-1层卷积层的输出特征矩阵, 作为池化层的输入特征矩阵;fsub (•) 为下采样函数.

    CNN以最小化损失函数L (W, b) 为网络的训练目标, 本文选用均方误差 (Mean Squared Error, MSE) 作为损失函数:

    ΜSE (W, b) =1Ν∑i=1Ν (Y (i) -Y (i) ^) 2    (3)

    式中, Y (i) 表示期望值, Y (i) ^表示卷积神经网络的输出值.

    经多个卷积层和池化层的交替传递, 逐层提取输入矩阵的特征后, 通过全连接网络对提取的多个特征进行整合.

    2.2 二维信息矩阵

    卷积神经网络多用于图像分类领域, m×n的图像数据矩阵能够很好地满足卷积层和池化层的要求, 但本文的原始数据样本并不是以矩阵方式存储的, 因此为了满足CNN卷积层和池化层的操作要求, 将原始数据向量转换为二维信息矩阵[9].

    设x为原始数据向量, 且为列向量, 则二维信息矩阵为:

    X=xxΤ= (a12a1a2⋯a1ana2a1a22⋯a2an⋯⋯⋯⋯ana1ana2⋯an2)     (4)

    使用原始数据向量构建的二维信息矩阵, 不仅保持了原始数据向量的数据特征, 同时满足了CNN对输入数据的要求.

    3 基于卷积神经网络的缺失数据填充方法

    3.1 问题的形式化定义

    设传感器网络中有N个数据源S={s1, s2, …, sN}, 第n (n∈N) 个传感器节点的时间序列为sn={dn1, dn2, …, dnT} (共T个时间步) , 第t (t∈T) 个时刻N个传感器节点的数据序列为Dt={d1t, d2t, …, dNt}.本文研究的问题可定义为:在传感器网络中, 传感器节点n在t时刻数据发生缺失, 使用传感器节点n的时间序列数据和监测区域内其他传感器节点的数据对t时刻的缺失值进行估计和填充.

    3.2 缺失数据填充方法

    使用卷积神经网络对缺失数据进行填充有三种基本思路:时间相关性、空间相关性和时空相关性.

    3.2.1 时间相关性

    基于时间相关性的填充只利用缺失数据所在传感器节点的时间序列数据进行CNN回归, 填充缺失数据.本文采用文献[3]的局部时间索引 (LTI) 策略构建原始训练集和测试集, 具体方法如下:

    图1 含有缺失数据的时间序列   下载原图

    图1是一个含有缺失数据的时间序列, 其中dt, dt+1, dt+4为缺失数据, 记录上方的数值代表全局时间索引, 以窗口大小为2来建立基于时间相关性填充的原始训练集和测试集.非缺失数据被用于构建训练集, 缺失数据用于构建测试集.

    对于非缺失数据dt-1, 它的前两个非缺失数据为dt-3, dt-2, 对应的全局时间索引为t-3和t-2, 由于第一个非缺失数据下标为t-3, 因此局部时间索引表示为{0, 1, 2}, 因此dt-1所构建的原始训练向量为pt-1= ({dt-3, dt-2, 0, 1, 2}, {dt-1}) .以此类推, dt+2, dt+3构建的原始训练向量分别为pt+2= ({dt-2, dt-1, 0, 1, 4}, {dt+2}) 和pt+3= ({dt-1, dt+2, 0, 3, 4}, {dt+3}) .

    对于缺失数据dt, 它的前两个非缺失数据为dt-2, dt-1, 对应的全局时间索引为t-2和t-1, 由于第一个非缺失数据下标为t-2, 因此局部时间索引为{0, 1, 2}, 因此dt所构建的原始测试向量为qt= ({dt-2, dt-1, 0, 1, 2}) .dt+1, dt+4构建的原始测试向量分别为qt+1= ({dt-2, dt-1, 0, 1, 3}) , qt+4= ({dt+2, dt+3, 0, 1, 2}) .

    将原始训练向量和原始测试向量经过第2.2节的方法转变为二维信息矩阵, 构成基于时间相关性的训练集和测试集, 并使用CNN填充模型对缺失数据进行填充, 得到时间维度的填充结果.

    3.2.2 空间相关性

    部署在监测区域内的多个传感器节点所形成的传感器网络, 网络中节点与节点间的数据存在空间相关性, 因此可以利用与缺失数据在同一传感网络中的其他传感器节点的数据, 对缺失数据进行填充.

    以传感器节点n为例说明基于空间相关的原始训练向量和测试向量的构建, 设传感器节点在k时刻数据完好, 而在t时刻数据发生缺失.因此, 在k时刻, 以其他N-1个传感器的数据作为输入, 传感器节点n的数据作为输出构建原始训练向量, 即p= ({d1k, d2k, …, dn-1, k, dn+1, k, …dNk}, {dnk}) .同理, 在t时刻的原始测试向量可表示为q= ({d1t, d2t, …, dn-1, t, dn+1, t, …dNt}) .

    同样, 将原始训练向量和原始测试向量转变为二维信息矩阵, 构成基于空间相关性的训练集和测试集, 并使用CNN填充模型对缺失数据进行填充, 得到空间维度的填充结果.

    3.2.3 时空相关性

    基于时间相关性的填充是依据缺失数据所属时间序列之间的关系进行填充, 得到时间维度的缺失数据估计值;基于空间相关性的填充是利用传感器节点之间的相互关系进行填充, 得到空间维度的缺失数据估计值;因此, 基于时空相关性的填充方法就是同时利用时间和空间信息对缺失数据进行填充.本文构建一个3层全连接网络整合时间维度和空间维度的估计值, 获得时空相关性的填充结果.基于CNN的缺失数据填充方法的基本流程图如图2所示.

    4 实验与结果分析

    4.1 数据源与数据处理

    本文采用某大坝测站点的真实数据进行实验, 共有7个数据源, 每个数据源上记录了近27个月的温度数据, 其中前18个月的监测数据为每天更新一次, 后9个月的监测数据为每天更新3次, 将这7个数据源的时间序列数据进行时空整合共得到1 351个样本数据.由于所得到的温度数据含有少数缺失值, 因此在实验过程中将缺失值替换为邻近的前后数据的平均值, 形成一个完整的数据集.本文选择传感器节点n作为实验对象, 随机插入缺失数据, 使其拥有r%的缺失数据, 其中r∈{5, 10, 15, 20, 25, 30, 35, 40, 45, 50}.

    4.2 建立卷积神经网络填充模型

    本文在PyCharm编程环境下使用Python和TensorFlow机器学习框架建立神经网络模型, 以最佳回归效果为目标确立卷积神经网络填充模型结构, 模型结构如图3所示.

    图3 卷积神经网络填充模型结构   下载原图

    在传感器节点n上, 基于时间相关性进行填充时获得的原始训练集和测试集有5维特征, 基于空间相关性进行填充时获得的原始训练集和测试集有6维特征.经过二维信息矩阵变换后, 最终基于时间相关性的卷积神经网络填充模型的输入数据维度为None×5×5, 输出数据维度为None×1;基于空间相关性的卷积神经网络填充模型的输入数据维度为None×6×6, 输出数据维度为None×1, 其中None为模型输入样本个数.

    卷积神经网络填充模型, 选用大小为2×2, 步长为1的卷积核进行卷积操作, 选用大小为2×2, 步长为2的池化核进行池化操作, 卷积层和池化层的Padding都设置为SAME, 激活函数为线性整流单元 (Rectified Linear Unit, ReLU) .卷积神经网络填充模型以最小化损失函数均方误差 (MSE) 为目标, 具体来说就是为了减小填充值与期望的真实值之间的误差.在模型训练初期较大的学习率可加快模型收敛速度, 使模型快速靠近真实值;随着训练次数的增加, 较小的学习率可以使模型在真实值附近收敛, 鉴于以上考虑, 模型中的学习率为lr=0.001×0.95 epoch, 其中0.001为初始学习率, epoch为当前迭代次数, 模型训练的最大迭代次数epochmax=1 000.

    4.3 结果分析

    本文选用希尔不等系数 (Theil Inequality Coefficient, TIC) 度量填充结果的准确性.TIC的值介于0和1之间, 度量指标TIC的数值越小, 说明填充值与真实值的误差越小, 回归拟合度越好, 填充精度越高.TIC公式为:

    ΤΙC=1n∑i=1n (yi-yi∧) 21n∑i=1nyi∧2+1n∑i=1nyi2    (5)

    式中, 1n∑i=1n (yi-yi∧) 2表示均方根误差 (Root Mean Square Error, RMSE) ;yi表示真实值, yi∧表示填充值, n为缺失数据个数.

    本文采用三层BP神经网络作为基本的缺失数据填充模型, 与本文提出的基于卷积神经网络的缺失数据填充模型进行对比.依据填充时考虑到的时间与空间两方面的相关性, 分别利用MBP, TBP, SBP表示考虑时空多维度相关性、只考虑时间相关性和只考虑空间相关性的BP填充方法.同理, 利用MCNN, TCNN, SCNN表示考虑时空多维度、时间和空间相关性的CNN填充方法.本文选用十折交叉验证来确定模型的最优参数, 在实验过程中, 分别对每一个缺失率的实验重复20次, 取平均值作为最终的实验结果.

    5 结束语

    本文分别研究传感器网络的数据特点和卷积神经网络技术, 在此基础上提出一种基于卷积神经网络的缺失数据填充方法.首先基于时间相关性和空间相关性, 使用CNN填充模型分别对缺失数据数据进行单维度相关性的填充.然后考虑数据时空相关性, 得到基于时空相关性的填充方法MCNN.最后在真实数据集上进行实验验证, 实验结果表明基于时空多维度相关性的填充结果好于单维度相关性的填充结果, 具有较高的填充精度和回归拟合度.同时, 与BP神经网络相比, 本文所提方法的填充准确性更高.下一步, 将考虑时间、空间和属性三个维度的相关性, 使用神经网络模型对缺失数据进行填充.
