    大数据分析技术的无线通信网络安全风险预测

    庄海燕

    铁道警察学院

    摘    要：

    当前无线通信网络安全风险预测模型存在误差大, 预测耗时长等缺陷, 为了提高无线通信网络安全风险预测精度, 加快无线通信网络安全风险预测速度, 提出了基于大数据分析技术的无线通信网络安全风险预测模型.首先采集无线通信网络安全风险预测历史数据, 然后采用大数据分析技术建立无线通信网络安全风险预测模型, 最后进行时了无线通信网络安全风险预测仿真实验, 结果表明, 本文模型的无线通信网络安全风险预测精度高, 预测误差远远小于其它无线通信网络安全风险预测模型, 加快了无线通信网络安全风险预测速度.

    关键词：

    无线通信; 网络安全; 风险预测; 大数据分析;

    Network based on big data analysis technology

    ZHUANG Hai-yan

    Railway Police College

    Abstract：

    The current wireless communication network security risk prediction model has some shortcomings, such as large error and long prediction time. In order to improve the accuracy of wireless communication network security risk prediction and accelerate the speed of wireless communication network security risk prediction, a wireless communication network security risk prediction model based on large data analysis technology is proposed. Firstly, the historical data of wireless communication network security risk prediction are collected, and then the security risk prediction model of wireless communication network is established by using large data analysis technology. Finally, the simulation experiment of wireless communication network security risk prediction is carried out. The results show that the security risk prediction accuracy of wireless communication network based on this model is high and the prediction accuracy is high. The measurement error is far less than other wireless communication network security risk prediction models, which accelerates the speed of wireless communication network security risk prediction.

    Keyword：

    wireless communication; network security; risk prediction; large data analysis;

    1 引言

    无线网络系统给人们生活带来便利的同时, 一些非法分子提供了一些便利, 通过各种攻击手段, 对无线网络传输的敏感信息进行窃取, 每年攻击的频率, 窃取数据数量呈大幅增长趋势, 无线网络安全问题随之而来[1,2].风险预测是一种新的无线网络安全技术, 可以描述无线网络的整体安全状态, 成为当前无线网络安全研究的一个重要方向[3].

    无线通信网络安全风险预测模型可以分为两类:线性预测模型和非线性预测模型.线性预测模型目前已经处于淘汰状态, 没有什么实际应用价值[4].非线性预测模型主要包括:各种神经网络的无线通信网络安全风险预测模型以及支持向量机的无线通信网络安全风险预测模型[5], 它们从非线性角度对无线通信网络安全风险进行跟踪和建模.

    最小二乘支持向量机 (Least Squares Support Vector Machines, LSSVM) 是一种大数据分析技术, 将支持向量机的不等式约束改成等式约束, 提高了求解速度及收敛精度[6].为了解决无线通信网络安全风险预测过程存在的难题, 设计大数据分析技术的无线通信网络安全风险预测模型, 并与其它无线通信网络安全风险预测模型进行了具体的仿真测试, 验证了本文无线通信网络安全风险预测的可行性和优越性.

    2.1 大数据分析技术

    2.3 大数据分析技术的无线通信网络安全风险预测步骤

    (1) 对一个无线通信网络安全风险预测问题, 选择报警数量、被植入后门数量、攻击数量、新增安全漏洞数量作为无线通信网络安全风险指标, 收集它们一段时间的这些指标数据.

    (2) 将无线通信网络安全风险等级划分为5种, 它们具体分别为:优[0 1], 良 (1 2.5], 中 (2.5 6], 差 (6 8], 危 (8 10], 通过专家根据指标数据标记出相关的无线通信网络安全风险等级, 它们构成无线通信网络安全风险预测的样本集合.

    (3) 从无线通信网络安全风险样本集合按一定比例随机选择一部分作为训练样本集, 用于构建无线通信网络安全风险预测模型, 剩余下的样本作为测试样本, 以验证无线通信网络安全风险预测模型的有效性.

    (4) 为了提高最小二乘支持向量机的收敛效率, 加快最小二乘支持向量机学习速度, 对无线通信网络安全风险等级的值进行如下归一化处理.

    y'=y-yminymax-ymin          (9)

    式中, ymax和ymin分别表示无线通信网络安全风险等级的最大和最小值.

    (5) 采用粒子群优化算法对于核参数σ和正则化参数γ进行优化, 确定最小二乘支持向量机的参数, 采用最小二乘支持向量机对无线通信网络安全风险的训练样本集进行学习, 建立无线通信网络安全风险预测模型.

    (6) 采用验证本对无线通信网络安全风险预测模型的性能进行测试.

    3.2 结果与分析

    3.2.1 安全态势的预测精度对比

    采用RBF神经网络、支持向量机、最小二乘支持向量机分别对200个训练样本进行学习, 并对50个测试样本的值进行预测, 并统计每一次仿真实验的无线通信网络安全风险平均预测精度, 结果如图3所示, 对图3的无线通信网络安全风险平均预测精度进行对比和分析, 可以得到如下结论:

    (1) RBF神经网络的无线通信网络安全风险平均预测精度比较低, 而且稳定性比较差, 这是因为它们基于经验风险最小化原则进行建模, 建立的无线通信网络安全风险预测模型稳定不够, 无法获得理想的无线通信网络安全风险预测结果.

    (2) 支持向量机和最小二乘支持向量机的无线通信网络安全风险平均预测精度要高于RBF神经网络, 主要因为它们基于结构风险最小化原则进行建模, 可以更好对无线通信网络安全风险进行描述, 降低了无线通信网络安全风险预测误差, 同时最小二乘支持向量机的无线通信网络安全风险预测精度要高于支持向量机, 这是因为克服了支持向量机的缺陷, 建模效果更佳.

    统计各种模型的无线通信网络安全风险训练时间, 具体如图4所示.从图4可以发现, 无线通信网络安全风险建模的训练时间相差较大, 其中支持向量机的训练时间最长, 而最小二乘支持向量机的训练时间最短, 提升了无线通信网络安全风险预测效率.

    针对当前无线通信网络安全风险预测模型存在误差大, 预测耗时长等缺陷, 设计了基于大数据分析技术的无线通信网络安全风险预测模型, 测试结果表明, 本文模型的无线通信网络安全风险预测精度, 加快了无线通信网络安全风险预测速度, 具有十分广泛的应用前景.
