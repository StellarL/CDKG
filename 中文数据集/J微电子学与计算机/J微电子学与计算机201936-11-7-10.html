    一种Nios Ⅱ软核架构的高效GPS捕获方法

    田嘉 王伟 史平彦

    中国空间技术研究院西安分院

    摘    要：

    针对传统GPS接收机不能根据接收信号的特性自适应的调节频率搜索范围及积分时间的问题,本文提出了一种Nios Ⅱ软核架构的高效GPS捕获方法,其可以通过Nios Ⅱ软核自适应调节频率搜索范围、相干积分时间及非相干积分时间,流水线式的VHDL结构仅负责并行处理,实现了参数调节的灵活可控.通过采用仅有一片Stratix Ⅲ FPGA的基带开发板,实测结果表明,捕获功能性能正常,频率捕获范围可在100 Hz～80 kHz内灵活切换,积分时间可在1 ms～30 s内灵活切换.

    关键词：

    Nios Ⅱ; 信号捕获; Stratix Ⅲ FPGA; GPS;

    作者简介： 田嘉 男，（1987-），硕士，高级工程师．研究方向为空间通信与导航技术．E-mail:jia_epfl_tian@163.com．; 王伟 男，（1970-），硕士，研究员．研究方向为空间通信与导航技术．; 史平彦 男，（1964-），硕士，研究员．研究方向为导航与遥感技术．;

    收稿日期：2019-02-27

    The GPS acquisition method based on Nios Ⅱ soft core

    TIAN Jia WANG Wei SHI Ping-yan

    China Academy of Space Technology(Xi′an)

    Abstract：

    In order to solve the problem that the traditional GPS receiver cannot adjust the frequency search space and integration time adaptively according to the characteristics of the received signal, the GPS acquisition method based on Nios Ⅱ soft core is presented by this paper. The Nios Ⅱ soft core is responsible for accommodate the frequency search space, the coherent integration time and the non-integration time. The VHDL part of assembly line structure is responsible for the parallel data processing. Therefore, the parameters can be adjusted adaptively by the method. By using the conception product which embeds 1 Stratix Ⅲ chip, real test result shows that the acquisition works well, the frequency search space can be changed within 100 Hz-80 kHz and the integration time can be changed within 1 ms-30 s.

    Keyword：

    Nios Ⅱ; signal acquisition; Stratix Ⅲ FPGA; GPS;

    1 引言

    目前,由于灵活性及成本优势,基于GPS(Global Positioning System)星座的空间导航应用引起人们越来越多的关注.随着科学技术的发展,飞行器本身获得的辅助信息也越来越多,如轨道滤波器[1]、测控链路及通信链路都会为飞行器提供其大致位置和GPS星座的大致位置[2].基于这些先验信息,飞行器可以计算出接收信号功率及多普勒频偏.

    传统GPS接收机一般是基于VHDL(Very-High-Speed Integrated Circuit Hardware Description Language)结构设计,参数的设置是固定的,其会针对接收信号功率情况设置3种积分时间,以应对强信号、中等信号及弱信号的应用场景,同样,频率搜索范围也会包括全频带、1/2频带及1/4频带3种模式.但是,灵活性仍显不足,不能根据接收信号的情况进行最优的参数配置[3].这会带来捕获时间的大大增加,从而延长定位的时间[4].

    针对上述问题,本文提出了一种基于Nios II软核的高效捕获方法,VHDL结构仅用于并行计算,极大的增加了GPS接收机的灵活性,降低了捕获时间,提高了捕获算法的效率.

    2 系统设计

    图1为本算法的系统模型,其由下变频采样模块、串并转换模块、数据缓存管理单元、DDR2 SDRAM控制器、DDR2 SDRAM、基于流水线处理的捕获模块和Nios II软核组成.除外挂DDR2 SDRAM和Nios II软核,其余模块均由VHDL实现.

    图1 系统模型   下载原图

    变频采样模块实现了对输入信号变频至基带及将信号的采样率抽取至4.096Mbps的功能,串并转换模块将下采样后的数据转为并行32bit的数据流,数据缓存管理单元根据Nios II软核的指令,启动DDR2 SDRAM控制器,执行将指定长度数据写入DDR2 SDRAM或从其中读出来的操作,并将数据写入DDR2 SDRAM时所对应的1ms时标计数值返回至Nios II软核.基于流水线处理的捕获模块会根据Nios II软核指令给出的多普勒频偏、相干积分时间、非相干积分时间及PRN(Pseudo Random Noise)码,对接收信号进行快速的捕获,并将并行计算得出的峰值、峰值坐标、均值及标准差回传至Nios II软核,最终由Nios II软核判断是否捕获成功.

    2.2 数据缓存管理单元设计

    数据缓存管理单元同外部的接口如表 1所示,Nios II软核通过Avalon总线完成与本模块的交互,具体如下:

    (1)avsAddress=“0000000”时,总缓存数据长度通过avsWrite和avsWriteData写入数据缓存管理单元;

    (2)avsAddress=“0000001”时,通过avsWriteData来区分将当前数据写入DDR2 SDRAM的指令和从DDR2 SDRAM中读出缓存数据的指令;

    (3)avsAddress=“0000010”时,Nios II软核通过avsRead将数据写入DDR2 SDRAM时所对应的1ms时标计数值以avsReadData的形式返回至软核.

    同DDR2 SDRAM接口主要完成数据从外接缓存中读写的交互,同串并转换模块接口主要用于接收输入的基带数据及时标计数值.

    2.3 基于流水线处理的捕获模块

    基于流水线处理的捕获模块框架图如图 5所示,捕获算法采用的PCS(parallel code search)[7,8],FFT的长度为4096,积分分支的个数可根据需要调整.Nios II软核首先通过Avalon总线将频率搜索范围、相干时间、非相干时间和PRN序号发送至捕获管理单元,捕获管理单元接收到指令后,会根据频率搜索范围将搜索频率拆分为多个Doppler bin,并对每个Doppler bin所对应的频率转换至载波频率和码率配置给载波产生模块和伪码产生模块,将相干积分时间和非相干积分时间配置给相干积分模块和非相干积分模块,将PRN配置给伪码产生模块.

    判决模块会根据N条分支相关峰值的大小,选出最大的一条分支,并计算出当前分支的均值和标准差,最终将峰值、峰值坐标、均值及标准差回传至Nios II软核,由Nios II软核判断是否捕获成功.

    2.4 Nios II 软核

    Nios II软核采用C语言编写,其采用串行处理流程,处理步骤如下:

    (1)接收到工作指令后,从初始态进入业务态,并根据当前先验信息将总缓存数据长度配置给数据缓存管理单元,启动写数据指令,待写操作完毕后,启动读数据指令,将数据读取至基于流水线处理的捕获模块;

    (2)将PRN、频率搜索范围、相干积分时间和非相干积分时间配置至基于流水线处理的捕获模块,读取第一个Doppler bin所对应的峰值、峰值坐标、均值及标准差,并计算出SNR(signal noise ratio),将相关数据存储在软核内的存储区内;

    (3)读取第二个Doppler bin所对应的峰值、峰值坐标、均值及标准差,并计算出SNR,若此SNR大于第一个Doppler bin所对应的SNR,则将第二个Doppler bin所对应的参数覆盖至第一组参数之上,更新存储区,否则保持存储区不变;

    (4)以此类推,直至将所有Doppler bin处理完毕;

    (5)读取软核存储区内的SNR值,若其大于捕获门限,则捕获成功,若小于捕获门限,则对先验信息内的其他PRN进行捕获.

    3 实测结果

    本文采用Spirent GSS8000导航信号源来产生飞行器在不同轨道下可以接收到的GPS L1信号,并通过仅有一片Altera Stratix III FPGA的基带开发板进行接收,实测场景如图6所示.

    实测结果表明,捕获功能性能正常,频率捕获范围可在100 Hz～80 kHz内灵活切换,积分时间可在1 ms～30 s内灵活切换.并且可以适应不同飞行轨道下接收信号功率及频偏变化的需求.

    4 结束语

    本文针对传统GPS接收机不能根据接收信号的特性自适应的调节频率搜索范围及积分时间的问题,提出了一种Nios II软核架构的高效GPS捕获方法,其可以通过软核实现灵活的调度,流水线式的VHDL结构实现快速的处理.实测结果表明,本方法可以很好的适应不同应用场景中对快速捕获的需求.
