    摘    要：

    自王国俊教授在模糊推理中提出全蕴涵推理方法以来, 对于这个推理方法的推广和完善, 成为模糊推理研究的热点方向之一。特别地, 最近有文献提出了五I推理方法。由于这个新推理方法中的蕴涵全部为同一种蕴涵, 因此, 本文将对这种推理方法进行改进, 提出更一般的全蕴涵推理方法。同时分别对两个基本的推理模型FMP和FMT, 给出新算法的统一表达式, 并分别讨论改进方法在两个模型下算法的还原性和鲁棒性。

    模糊推理; 全蕴涵推理方法; 五蕴涵推理方法; 对称蕴涵方法; 一般全蕴涵推理方法;

    General Full Implication Method of Fuzzy Reasoning

    WANG Lan-ting PEI Dao-wu

    School of Sciences, Zhejiang Sci-Tech University

    Abstract：

    Since Professor Wang Guojun proposed the full implication inference method in fuzzy reasoning, the generalization and improvement of this method has become one of the hot research direction of fuzzy reasoning. In particular, the five implication inference method has been proposed in the recent literature. Since five implications in this new inference method are all the same, therefore, this paper improves the inference method, and proposes a more general full implication inference method by using two different implications. At the same time, unified expressions of the new algorithm are given for two basic inference models FMP and FMT, respectively. Moreover, the reductivity and robustness of the improved method under the two models are discussed separately.

    1 引言

    模糊推理在模糊系统控制、模糊专家系统、模糊模式识别、模糊决策分析、模糊数据挖掘和人工智能等领域中扮演着越来越重要的角色[1,2,3]。 众所周知, FMP和FMT是模糊推理中的两个最基本的模型, 简单地说, 就是给一个模糊规则“如果A那么B”, 且给出A*, 求出B*, 或者, “如果A则B”, 且给出B*, 求出A*. 对于这两个模型, 1973年, Zadeh提出了模糊推理的合成推理规则, 简称CRI方法。 王国俊教授指出, CRI方法采用了复合运算, 带有一定的随意性, 缺少严格的逻辑理论基础[4]。为了解决这一问题, 王国俊教授根据“规则和前提应该尽可能地支持结论”的思想, 提出了模糊推理的全蕴涵推理方法, 简称为三I算法[4]。这一推理方法将模糊逻辑与模糊推理的研究做了有机的结合, 在模糊推理中贯彻逻辑语义蕴涵的原则, 使得模糊推理回到语义蕴涵的正确轨道上来[5]。 本文第二作者在文献[6]中给出了全蕴涵推理算法关于FMP和FMT模型的统一解的形式。刘华文等在文献[15]中讨论了三I算法在一些蕴涵下的连续性问题。罗敏霞等在文献[13]中讨论了三I方法基于Schweizer-Sklar算子族的算法。 文献中对三I方法进行推广和完善, 得到了一些新的推理方法。 同时, 各种推理方法的性质也被广泛地研究, 例如文献[1]和文献[12]。关于模糊推理三I方法的主要研究工作, 请参阅综述文[16]。

    对于同一个模糊推理规则来说, 使用不同的蕴涵将会得到不同的结果。那么若一个模糊推理规则中含有几个蕴涵, 这几个蕴涵是否可以取不同的蕴涵?如果可以, 应如何去选择? 对于模糊推理来说, 蕴涵连接词是逻辑系统的基础。 在全蕴涵推理算法中, 我们可以把第一和第三个蕴涵, 看作是逻辑逻辑系统中的模糊连接词, 第二个蕴涵看作是“若…, 则…”推理模型中的关系[8]。鉴于这种思想, 经过深入的分析, 本文将提出新的推理规则, 对五I推理方法进行合理的推广, 同时也对这种新的推理方法进行研究, 给出这种新的推理方法关于FMP和FMT模型统一解的形式, 还讨论新推理方法的还原性和鲁棒性。

    3 一般全蕴涵推理方法

    对于FMP模型的全蕴涵推理算法, 文献[7]认为, 如果A和A*差异不大的话, 自然应该要求B 和B*的差异也不大。 作为三I方法的改进, 应该将已知条件A→B结合A*→A计算B*. 基于这个想法, 文[7]提出了“五I推理原则”, 即要求FMP模型的解B*应该是论域V上使得表达式

    (A (u) →B (v) ) → ( (A* (u) →A (u) ) → (A* (u) →B* (v) ) )

    恒取最大值的最小模糊集。

    类似地, 对于FMT模型, 可以将A与B, A*与B*的位置交换。 由此应该寻求论域U 上最小的模糊集A*, 使得表达式

    (A (u) →B (v) ) → ( (B (v) →B* (v) ) → (A (u) →A* (u) ) )

    恒取最大值。

    其实, 对于FMT 模型来说, 自然地, 我们也应该要求, 将两个已知条件A→B和B→B* 结合起来, 对结论给予尽可能大的支持。

    不难看出, 式 (1) 和 (2) 并不是完全对称的, 据我们分析, 其中的主要原因在于算法设计的可操作性。

    当然, 我们也可以尝试将公式修改为

    (A (u) →B (v) ) → ( (B (v) →B* (v) ) → (A* (u) →B* (u) ) )

    这样做似乎更合理, 也与式 (1) 完全对称。 但是, 如此一来, 使得结论的计算将变得比较困难。 我们将在以后的工作中详细讨论这个问题。

    关于五I原则的详细介绍, 请读者参阅文献[7]。

    值得注意的是, 在五I推理方法中, 五个蕴涵都取为同一个蕴涵算子[7]。

    我们知道, 对于模糊推理来说, 蕴涵连接词是推理方法的重要组成要素。

    在全蕴涵推理算法中, 我们可以把第一和第三个蕴涵看作模糊连接词, 第二个蕴涵看作是“若…, 则…”推理模型中的蕴涵关系, 这样修改后得到的推理方法称为对称蕴涵方法, 基于这个推理方法所建立的模糊控制系统可能具有比较好的响应能力[8]。

    基于这个思想, 本文对五I推理方法进行推广, 提出更一般的五I推理方法, 称之为一般全蕴涵推理方法。

    一般全蕴涵推理原则 (i) 对于FMP问题, 我们寻求论域V上的最小模糊集B*, 使得以下表达式恒取最大值:

    (A (u) →1B (v) ) →2 ( (A* (u) →1A (u) ) →2 (A* (u) →1B* (v) ) )

    (ii) 对于FMT 问题, 我们寻求论域U 上的最小模糊集A*, 使得以下表达式恒取最大值:

    (A (u) →1B (v) ) →2 ( (B (v) →1B* (v) ) →2 (A (u) →1A* (u) ) )

    不难看出, 当第一个蕴涵→1和第二个蕴涵→2取为同一个蕴涵时, 以上的一般全蕴涵原则即为文[7]的五I推理原则。因此, 由这里的全蕴涵推理原则得到的一般全蕴涵推理方法, 实际上是五I 推理方法的推广, 从而也是三I 推理方法的推广。

    对于任何u∈U, v∈V, 如果式 (3) 存在最大值, 则令M (u, v) 为其最大值; 如果式 (4) 存在最大值, 则令L (u, v) 为其最大值。

    作为一般全蕴涵推理原则, 命题3.3和命题3.4的一般情形, 现在我们分别给出α-一般全蕴涵推理原则, 以及关于FMP和FMT问题的解的统一形式。

    α-一般全蕴涵推理原则 设α∈E.

    (i) 对于FMP问题, 我们寻求论域V上的最小模糊集B*, 使得以下不等式成立:

    5 一般全蕴涵推理方法的还原性及鲁棒性

    命题5.1 如果∃u0∈U, 使得A (u0) =1, 那么由命题3.3给出的一般全蕴涵推理方法是还原的。

    证明 根据已知条件, 当A*=A 时, B是FMP模型的好集。 又, B*是FMP模型的最小好集。 于是, 我们有

    定义5.1 称FMT问题的某个推理方法是还原的, 如果根据这个推理方法, 由B*=B能够推出A*=A.

    命题5.2 如果∃v0∈V, 使得B (v0) =1, 那么由命题3.4给出的一般全蕴涵推理方法是还原的。

    6 结束语

    由于模糊推理方法已经被成功地应用于模糊系统控制等许多研究领域, 因此对于模糊推理方法的深入研究与改进就显得十分的有意义。本文的主要工作是结合对称蕴涵方法, 将五I推理方法做了有意义的推广, 提出了一般全蕴涵模糊推理方法, 并且分别给出了一般全蕴涵方法在FMP和FMT问题下的统一计算表达式, 并且研究了一般全蕴涵方法的还原性和鲁棒性。作为后续的工作, 我们还计划研究一般全蕴涵方法的其他性质及其应用。
