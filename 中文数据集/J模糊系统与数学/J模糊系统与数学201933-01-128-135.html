    摘    要：

    通过分析变精度概率粗集模型与概率S-粗集模型的不足, 给出了一般关系下的变精度S-粗集模型, 并将变精度与概率S-粗集相结合, 得到了变精度概率S-粗集模型。运用该模型处理相关动态问题会明显降低决策失误的风险。

    Variable Precision Dynamic S-rough Set Model and Its Applications

    QI Xiao-ding SHAO Ya-bin

    Key Lab of Intelligent Analysis and Decision on Complex Systems, Chongqing University of Posts and Telecommunications

    Abstract：

    In order to overcome the insufficient of variable precision probabilistic rough set model and probabilistic S-rough set model, we give the variable precision S-rough set model based on the general relations. In addition, we obtain the variable precision probabilistic S-rough set model by combining the variable precision with the probabilistic S-rough set. Using this model to deal with relevant dynamic problems will obviously reduce the risk of decision making errors.

    Received： 2018-06-20

    2002年, Kaiquan Shi对Z.Pawlak粗集[1]作出了改进, 给出了动态R-元素等价类的概念, 并提出了S-粗集[2]。S-粗集是以具有动态特性的R-元素等价类[x]定义的, 具有两种基本形式:单向S-粗集和双向S-粗集。在不完备信息系统中, 大多数集合都是动态变化的, 即存在元素的迁入和迁出, 故而在解决问题时考虑集合元素的动态性是很有必要的。S-粗集很好地解决了这一问题, 为动态数据挖掘及规律发现研究提供了理论支持。而后, 文献[3], [4], [5], [6]给出了S-粗集的结构、知识分离和分解还原定理, 对S-粗集的相关性质进行了研究。

    2008年, Tian Minge等[7]结合概率粗糙集[8,9]与S-粗集的思想, 给出了概率S-粗集模型, 该模型虽然考虑了元素双向迁移发生的概率及在描述[x]R下X发生的概率, 但由于模型中包含了两个参数α, β, 且α, β仅有相同的约束条件, 相互之间无严格约束, 故有可能导致对论域U中某些集合的刻画不精确, 从而使得决策失误。W.Ziarko最早给出了变精度粗糙集模型[10], 解决了Pawlak粗集局限性大、容错能力小的问题.然而, 上述模型依然要求绝对精确的包含关系, 2005年巩增泰等给出了结合一般关系下的变精度粗糙集模型[11], 克服了Pawlak粗集中由于要求绝对精确的包含关系致使有用信息失真的现象。文献[12], [13]通过限定参数的置信区间, 分别给出了变精度概率粗糙集模型和变精度粗糙集模型, 解决了由于有两个参数而导致的论域中某些集合刻画不精确的问题。但是上述两种模型均只能处理相关静态集合问题, 而不能解决相应的动态问题.随后, Cao Bingru等[14]和Zhao Huanhuan等[15]分别给出了基于集对势优势关系和基于集对顺势相似关系的变精度粗糙集模型。前者用集对势优势关系代替变精度粗糙集模型的等价关系, 后者则用集对顺势相似关系代替变精度粗糙集的不可区分关系, 使得对粗糙集的刻画更加精确合理。

    本文利用一般关系下的多数包含关系, 给出了一般关系下的动态S-粗集模型, 考虑了集合X的动态特性和知识库中的不完备信息。同时根据变精度思想, 通过限定参数α的置信区间, 将概率S-粗集模型中的两个参数变为一个参数得到一类变精度的概率动态S-粗集模型, 该模型解决了集合的动态变化即集合中元素的迁入与迁出问题, 也约束了参数α的置信区间 (12<α≤1) , 从而解决了论域U中某些动态集合刻画不精确的问题, 使原来的粗糙集 (即不可定义集) 成为可定义集或原来可定义的集合成为粗糙集的概率大大降低, 同时该模型充分结合了S-粗集中动态集合与变精度思想中参数设定的优点, 从而降低了决策失误的风险。

    概率S-粗集模型虽然考虑了论域U中集合X*的动态特性和近似空间中相关信息的不确定性, 但由于其参数间的约束不严格, 导致论域中的某些子集刻画错误.变精度概率S-粗集模型引入参数α, 并限定其置信区间为12<α≤1, 充分利用概率近似空间中的相关信息, 降低了决策失误的风险, 提高了近似决策的精确性。

    4 例子

    设U={x1, x2, x3, …, x20}, R是论域U上的等价关系, U中元素在R下的基本类为:D1={x1, x2, x3, x4, x5}, D2={x6, x7, x8}, D3={x9, x10, x11, x12, x20}, D4={x13, x15, x19}, D5={x14, x16, x17, x18}, X0={x3, x5, x8, x14, x16, x17, x19}, 则由条件概率计算得:

    Ρr (X0|D1) =25, Ρr (X0|D2) =13, Ρr (X0|D3) =0, Ρr (X0|D4) =13, Ρr (X0|D5) =34

    则对任何α (12<α≤1) , 都有恒等式成立。而由于0≤1-α<12, 有F*α (X0) 除包含D5外, 至少包含D1, D2, D3, D4中的某一个集合, 故对所有α (12<α≤1) 均成立。所以X0依参数α关于概率近似空间AP是粗糙的。

    (1) 当元素迁入X0时, 设存在迁移函数F使得元素x6, x9, x12迁入到X0中得到X1, 即

    X1={x3, x5, x8, x14, x16, x17, x19}∪{x6, x9, x12}={x3, x5, x6, x8, x9, x12, x14, x16, x17, x19}

    此时由条件概率计算得:

    Ρr (X1|D1) =25, Ρr (X1|D2) =23, Ρr (X1|D3) =25, Ρr (X1|D4) =13, Ρr (X1|D5) =34

    则对任何α (12<α≤1) , 都有恒等式成立。而由于0≤1-α<12, 有除包含D2, D5外, 至少包含D1, D3, D4中的某一个集合, 故对所有α (12<α≤1) 均成立。所以X1依参数α关于概率近似空间AP是粗糙的。

    (2) 当元素迁出X1时, 设存在迁移函数F¯使得元素x17, x19迁出X1外得到X2, 即

    X2=X1\{x17, x19}={x3, x5, x6, x8, x9, x12, x14, x16}

    此时由条件概率计算得:

    Ρr (X2|D1) =25, Ρr (X2|D2) =23, Ρr (X2|D3) =25, Ρr (X2|D4) =0, Ρr (X2|D5) =12

    则对任何α (12<α≤1) , 都有恒等式成立。而由于0≤1-α<12, 有除包含D2外, 至少包含D1, D3, D4, D5中的某一个集合, 故对所有α (12<α≤1) 均成立, 所以X2依参数α关于概率近似空间AP是粗糙的。

    可见, 变精度概率S-粗集模型不仅可以解决由于参数无严格约束而使得集合刻画错误的问题, 还可以解决具有动态特性的集合X*⊆U的问题, 明显减小了处理问题时决策失误的风险。
