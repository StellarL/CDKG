    包括解题反思在内的数学解题,是数学学习中不可或缺的核心内容.[1]本文从一道高考数列题出发,从解题反思的角度进行解题分析,并对新定义型数列题的解决提出几点建议.

    1 案例呈现

    例1 (2019·江苏卷)定义首项为1且公比为正数的等比数列为“M-数列”.

    (1)已知等比数列{an}(n∈N*)满足:a2a4=a5,a3-4a2+4a1=0,求证:数列{an}为M-数列.

    (2)已知数列{bn}(n∈N*)满足:b1=1,1Sn=2bn-2bn+1,其中Sn为数列{bn}的前n项和.

    ①求数列{bn}的通项公式;②设m为正整数,若存在M-数列{cn}(n∈N*),对任意正整数k,当k≤m时,都有ck≤bk≤ck+1成立,求m的最大值.

    解 (1)设等比数列{an}的公比为q,则a1≠0,q≠0.由

    {a2a4=a5,a3-4a2+4a1=0

    得

    {a1q⋅a1q3=a1q4,a1q2-4a1q+4a1=0,

    解得

    {a1=1,q=2.

    因此,数列{an}为M-数列.

    (2)①由1Sn=2bn-2bn+1,得Sn=12bn-2bn+1=bnbn+12(bn+1-bn), 且bn ≠ 0. 当n ≥ 2时, 由bn = Sn-Sn-1, 得bn=bnbn+12(bn+1-bn)-bn-1bn2(bn-bn-1). 因为 bn ≠ 0,所以1=bn+12(bn+1-bn)-bn-12(bn-bn-1)=bn+1(bn-bn-1)-bn-1(bn+1-bn)2(bn+1-bn)(bn-bn-1),即2(bn+1-bn)(bn-bn-1)=bn+1(bn-bn-1)-bn-1(bn+1-bn),即bn+1bn-2bn2+bnbn-1=0.因为bn≠0,所以bn+1+bn-1=2bn,即bn+1-bn=bn-bn-1(n≥2),所以{bn}是等差数列.

    又b1=1,b1=S1=2b1-2b2,于是b2=2,从而公差d=2-1=1,所以{bn}是首项和公差均为1的等差数列,其通项公式为bn=n(n∈N*).

    ②由①知bk=k(k∈N*).数列{cn}为M-数列,设公比为q,则c1=1,q >0.因为ck≤bk≤ck+1,所以qk-1≤k≤qk,其中k=1,2,3,…,m.当k=1时,有q≥1;当k=2,3,…,m时,有lnkk≤lnq≤lnkk-1.设f(x)=lnxx(x>1),则f′(x)=1-lnxx2.令f′(x)=0,得x=e.列表如下:

    x    		(1,e)e(e,+∞)

    f′(x)    		+0-

    f(x)    		↗极大值↘

    因为ln22=ln86<ln96=ln33,所以f(k)max=f(3)=ln33.取q=33,当k=1,2,3,4,5时,lnkk≤lnq,即k≤qk,经检验知qk-1≤k也成立.因此,所求m的最大值不小于5.

    若m≥6,分别取k=3,6,得3≤q3且q5≤6,从而q15≥243且q15≤216,所以q不存在.因此,所求m的最大值小于6.

    综上,所求m的最大值为5.

    2 解题过程的反思

    本题主要考查等差和等比数列的定义、通项公式、性质等基础知识,考查代数推理、转化与化归及综合运用数学知识探究与解决问题的能力.在问题的呈现方式上,既有许多让考生觉得比较熟悉的情境,也有少数相对新颖陌生的情境.本题属于新定义型数列题,既考查了学生的运算能力、逻辑推理能力,也考查了学生观察、分析、解决问题的综合能力以及独立获取数学知识的能力,符合“能力立意”的命题思想.

    第(1)问及第(2)问第①小问的情况较为简单,因此本文将分析重点放在第(2)问第②小问,主要从解题思路、思想策略两大方面对解题过程进行反思.

    2.1 解题思路的理解

    第(2)问第②小问属于难题.首先提炼其解题步骤,大致可以分为三大步,记为S1,S2,S3.

    S1:弄清题意.

    先将问题翻译成对应的数学语言.{cn}是M-数列,即c1=1,公比为q>0,可知cn=qn-1.ck≤bk≤ck+1成立,即qk-1≤k≤qk成立.故目标就是求满足下列条件的最大值m:存在q(q>0),使得当k≤m时,不等式qk-1≤k≤qk成立(k,m∈N*).

    S2:将指数不等式转换为对数不等式,构造f(x)=lnxx的新函数.

    S3:求出m的最大值.可分为三个小步:S31,S32,S33.

    S31:对函数求导,f′(x)=1-lnxx2.

    S32:判断函数的单调性,找出极大值f(k)max=f(3)=ln33.

    S33:对k的取值进行讨论.根据最初题干给出的不等式进行验证,发现k≤5时不等式恒成立,m≥6时,满足条件的q不存在,最终得出m的最大值为5.

    这一小问的逻辑比较复杂和抽象,一开始如何突破到第二大步呢?

    尝试将m取几个具体的值试试:m=1时,必然有k=1时成立;m=2时,必然有k=1,2时都要成立,即q≥1,且当k=2时,c2≤b2≤c3,即q≤2≤q2,得2≤q≤2;m=3时,必然有k=1,2,3时都要成立,即q≥1且2≤q≤2,当k=3时,c3≤b3≤c4,即q2≤3≤q3,得33≤q≤3.

    特殊化帮助我们理解:通过这里的特殊化,可以发现q必须满足(kk)max≤q≤(kk-1)min,其中k≤m,即(lnkk)max≤(lnkk-1)min成立的最大m即为所求.构造函数f(x)=lnxx(x≥3),g(x)=lnxx-1(x≥3),分别求解最大值和最小值,最后解不等式ln33≤lnmm-1即可.具体解法如下.

    当k=1时,有q≥1;当k=2时,c2≤b2≤c3,即q≤2≤q2,得2≤q≤2;当k≥3时,两边取对数,可得lnkk≤lnq≤lnkk-1(k≤m),即考察(lnkk)max≤(lnkk-1)min成立的最大的m.

    令f(x)=lnxx(x≥3), 则f′(x)=1-lnxx2.当 x≥ 3时,f′(x) < 0,此时f(x)递减,故当k ≥ 3时,(lnkk)max=ln33.令g(x)=lnxx-1(x≥3), 则g′(x)=1-1x-lnx(x-1)2.令h(x)=1-1x-lnx, 则h′(x)=1-xx2.当x ≥ 3时, h′(x) < 0,从而h(x) ≤ h(3) < 0, 即g′(x) < 0,故g(x) 在[3,+∞)上单调递减, 即k ≥ 3时,(lnkk-1)min=lnmm-1.下面考察ln33≤lnmm-1.化简得3ln m-(m-1)ln 3≥0.令u(m)=3lnm-(m-1)ln3,u′(m)=3m-ln3,当m≥3时,u′(m)<0,故u(m)在[3,+∞)上单调递减.又u(5)=3ln 5-4ln 3=ln 125-ln 81> 0,u(6) = 3ln 6 - 5ln 3 = ln 216 - ln 243 <0,所以存在m0∈(5,6),使得u(m0)=0.故当m≤5时,u(m)=3ln m-(m-1)ln 3>0,当m≥6时,u(m)=3ln m-(m-1)ln 3<0,故使ln33≤lnmm-1成立的最大的m是5,此时q∈[33,54].

    想法:如果继续取特殊值呢?

    当k=4时,c4≤b4≤c5,即q3≤4≤q4,得44≤q≤43;当k=5时,c5≤b5≤c6,即q4≤5≤q5,得55≤q≤54;于是q≥1,2≤q≤2,33≤q≤3,44≤q≤43,55≤q≤54同时成立,解得33≤q≤54,所以m≤5符合题意.

    因此,可以确定m不小于5.若m≥6,那么当k=1,2,3,4,5时得33≤q≤54;当k≥6时,ck≤bk≤ck+1,即qk-1≤k≤qk,得kk≤q≤kk-1.下证33≤q≤54与kk≤q≤4k-1(k≥6)不能同时成立.只要证kk-1<33(k≥6)(*).

    1° 当k=6时,因为(65)15=63=216,(33)15=35=243,所以(65)15<(33)15,从而65<33,(*)式成立.

    2° 假设当k=l(l≥6)时,(*)式成立,即ll-1<33,则当k=l+1时,令y=ll-1,两边取对数lny=1l-1lnl.利用对数法对其进行求导,y′y=-1(l-1)2lnl+1l(l-1),则y′=l(1-lnl)-1l(l-1)2⋅ll-1.由于l(1-lnl)-1<0,l(l-1)2>0,ll-1>0,则y′<0,因此y=ll-1在 [6,+∞)上单调递减,故l+1l<ll-1<33,即当k= l+1时,(*)式也成立.

    由1°,2°及数学归纳法,可知kk-1<33(k≥6)成立,即33≤q≤54与kk≤q≤kk-1(k≥6)不能同时成立,所以m≥6不合题意.

    综上,m≤5,即m的最大值为5.

    2.2 反思数学思想方法,实现陌生问题熟悉化

    本题考察思维的严谨性,需要扎实地掌握数列和导数知识,需要做题的韧性和耐心,同时还需要利用特殊化思想理解题意.“新定义”型数列问题考查学生的分析理解能力、抽象概括能力和推理论证能力,需要学生把握问题的本质,使问题比较自然地转化到熟悉的知识点与题型上来.

    数学能力    		逻辑推理能力、数学运算能力、探究问题的能力、分析解决问题的能力以及独立获取数学知识的能力

    如果考生能够在记忆中找到类似的“新定义”型数列问题,解答起来可能会容易许多.笔者发现,类似的解题策略在下面的例2中也有体现.

    例2 (2014·江苏卷)设数列{an}的前n项和为Sn,若对任意的正整数n,总存在正整数m,使得Sn=am,则称{an}为“H数列”.

    (1)若数列{an}的前n项和为Sn=2n(n∈N*),证明:{an}是H数列;

    (2)设{an}是等差数列,其首项a1=1,公差d<0,若{an}是H数列,求d的值;

    (3)证明:对任意的等差数列{an},总存在两个H数列{bn}和{cn},使得an=bn+cn(n∈N*)成立.

    解 (1)当n≥2时,an=Sn-Sn-1=2n-2n-1=2n-1,所以Sn=2n=an+1.又S1=a1,故{an}是H数列.

    (2)由已知,得S2=2a1+d=2+d.若{an}是H数列,则存在正整数m,使得S2=am,即2+d=1+(m-1)d,于是(m-2)d=1.因为d<0,所以m-2<0,故m=1,从而d=-1.

    当d=-1时,an=2-n,Sn=n(3-n)2是小于2的整数(n∈N*),于是对任意的正整数n,总存在正整数m=2-Sn=2-n(3-n)2,使得Sn=2-m=am,所以{an}是H数列.因此d的值为-1.

    (3)设等差数列{an}的公差为d,则an=a1+(n-1)d=na1+(n-1)(d-a1)(n∈N*).令bn=na1,cn=(n-1)(d-a1),则an=bn+cn(n∈N*).下证{bn}是H数列.

    设{bn}的前n项和为Tn,则Τn=n(n+1)2a1(n∈Ν*).于是对任意的正整数n,总存在正整数m=n(n+1)2,使得Tn=bm.所以{bn}是H数列.

    同理可证{cn}也是H数列.因此,对任意的等差数列{an},总存在两个H数列{bn}和{cn},使得an=bn+cn(n∈N*)成立.

    说明 该题中“对任意的正整数n总存在正整数m,使得Sn=am”说明{Sn}几乎是{an}的子列,命题者将此数列命名为“H数列”,取自“和谐”之意.

    这道题在数学思想和解法上与“M-数列”问题有着相通之处,也利用了转化与化归及构造的方法.第(3)问是寻找等差数列{an}的某种拆分,构造适合条件的数学对象,蕴含着创造,并且an的分解不唯一.基于问题,通过观察、类比、联想,大胆猜想,再进行验证.这一点与“M-数列”问题中利用构造新函数,将问题转化为求最值问题相似.两者都将转化思想蕴含其中,进行不断转化问题,将陌生、不规范、复杂的问题转化为熟悉、规范、简单的问题,使之逐步成为容易解决或已经解决过的问题模式.

    以下是an分解的一般性证明.

    先证:若{an}是公差为d的等差数列,且是“H数列”,则存在整数N≥-1,使a1=Nd.事实上,若{an}是等差数列且是H数列,则对任意的正整数n,总存在正整数m,使得Sn=na1+n(n-1)2d=a1+(m-1)d,整理得m-1=(n-1)(a1d+n2).特别地,当n=2时,有正整数m,使m-2=a1d.所以a1d为整数且a1d≥-1.记Ν=a1d,则a1=Nd.

    设{bn},{cn}为等差数列,且为“H数列”,使得an=bn+cn(n∈N*)成立,即使a1+(n-1)d=b1+(n-1)d1+c1+(n-1)d2(n∈N*)(*)成立,其中d1,d2分别是{bn},{cn}的公差.因为{bn},{cn}为等差数列且为H数列,由已证的结论得,存在整数N1≥-1, N2≥-1,使b1=N1d1,c1=N2d2.代入(*)式,得a1+(n-1)d=N1d1+(n-1)d1+N2d2+(n-1)d2=N1d1+N2d2+(n-1)(d1+d2).

    由此可知,要找到an的分解式,只要根据上式找到d1,d2并确定{bn},{cn}的首项b1和c1.为此,我们任意取定两个整数N1,N2满足N1≥-1,N2≥-1且N1≠N2,并令d=d1+d2,a1=N1d1+N2d2(** ).对于给定的a1,d,由方程组(** )总能解出d1,d2,此时得到H数列{bn}和{cn}.即对任意的等差数列{an},总存在两个H数列{bn}和{cn},使得an=bn+cn(n∈N*)成立.

    3 反思总结

    近年在高考或各地高考模拟试卷中出现的一些“新定义”型数列题,都是培养学生迁移和探究能力的良好素材.通过以上题目的解决,笔者根据自己的经验总结出以下几点建议:(1)深度挖掘题意,将其数学化;(2)掌握基本方法,尤其是基本方法的叠加;(3)学会反思、总结相似解题策略、思想方法,使陌生问题熟悉化;(4)大胆转化,积极创新.
