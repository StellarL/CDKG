提升学生数学思维素养深度和广度的教学实践

在高二文科零轮复习中,为了对恒成立与存在性问题的转化策略进行梳理总结与完善提升,笔者开设了一节“恒成立与存在性问题”复习课.以下是主要教学过程及反思,供读者参考.

1 投石问路,了解学生原有思维状况

课前给学生做了两个热身训练题:

(1)存在实数x∈[1, 2],使得不等式x2 - 2ax + 1≤0成立,则实数a的取值范围为______.

(2)已知不等式x ln x - a < 0对任意实数 x∈[1, e](e是自然对数的底数)恒成立,则实数a的取值范围为______.

生1:(1)a≥(x2+12x)min,x2+12x=12(x+1x)≥12×2x⋅1x=1,当且仅当x = 1时取得等号,所以a≥1.

生2:(2)a > (x ln x)max,令h(x) = x ln x(x ∈[1, e]),则h′(x) = ln x + 1 > 0,故h(x) = x ln x在区间[1, e]上单调递增.因此,h(x)max = h(e) = e,故a >e.

学生几乎千篇一律地运用了参变分离法.笔者询问是否有其他方法时,学生回答不出来.这说明学生思维还不够宽广.

2 从顺应到突破固有思维,提升学生思维的深广度

为了帮助学生突破思维定势,笔者设计了一组问题变式,让学生认识到解决恒成立与存在性问题的方法不仅仅有参变分离法,有时还需要有其他方法,以此进一步提升学生数学思维的深度和广度.

问题1 已知对任意实数x∈[1, 2],不等式x2-2ax+1>0恒成立,求实数a的取值范围.

解 (学生讲述,教师板书)参变分离,得a<x2+12x,又x2+12x=12(x+1x)≥12×2x⋅1x=1,当且仅当x = 1时取等号.对任意实数x ∈[1, 2],不等式x2-2ax+1>0恒成立⇔a<(x2+12x)min,所以a < 1.

求x2+12x=12(x+1x)的最小值时,有些学生会想利用“耐克”函数的单调性来求,而作为解答题就必须利用导数来确定.

总结参变分离法:利用不等式的等价变形,使不等式中的参数(或关于参数的代数式)与其他变量分别处于不等号的两侧,求出关于其他变量的函数的最值或值域,从而求出参数的范围.参变分离法处理恒成立与存在性问题的等价转化策略:

设计意图 热身训练1、问题1分别是存在性问题和恒成立问题,帮助学生理解恒成立与存在性问题的特征.解决恒成立与存在性问题优先考虑参变分离法,因为参变分离后新函数是确定的,可避开所含因参数的不确定性而需要的繁琐讨论;对此,学生已有的思维应该得到顺应和巩固.若用构造函数法,令f(x) = x2 - 2ax + 1,则要利用二次函数的图象性质分类讨论,学生基本不选择这种方法.

变式1 已知对任意实数x ∈ (1, 2),不等式x2-2ax+1≥0恒成立,求实数a的取值范围.

变式2 已知对任意实数x ∈ (1, 2),不等式x2-2ax+1>0恒成立,求实数a的取值范围.

变式3 已知对任意实数x ∈[-1, 2],不等式x2-2ax+1 > 0恒成立,求实数a的取值范围.

分析 自变量的取值范围发生了变化.变式1和变式2使得y=12(x+1x)的最小值取不到1,所以a≤1.对变式3,若仍然运用参变分离法,则需分-1≤x < 0, x = 0, 0 < x≤2三种情况讨论,较繁琐.若参变分离不便,或无法参变分离,或参变分离后所得新函数比较复杂、最值不好求,则可以直接构造原来含参数的对应函数来解决问题.构造函数f(x) = x2-2ax+1, x∈[-1, 2].对任意实数x∈[-1, 2],不等式x2-2ax+1 > 0恒成立 ⇔ f(x)min > 0,这里因为参数a的不确定性,需要分三种情况讨论来确定f(x)的最小值,如果参数a的范围能缩小,可能可以减少讨论的种类.对任意实数x∈[-1, 2],不等式x2-2ax+ 1>0恒成立,那么x取区间内的特殊值(例如区间端点处、中点处等)时,不等式也成立.

对任意实数x∈[-1, 2],不等式x2-2ax+1 > 0恒成立,令f(x) = x2-2ax+1,则必有

{f(1)>0,f(2)>0,

所以

{2+2a>0,5-4a>0,

所以-1<a<54.当-1<a<54时,函数f(x) = x2-2ax+1在区间 [-1, a]上单调递减,在区间[a, 2]上单调递增,f(x)min= f(a) = 1-a2.所以

{-1<a<54,1-a2>0,

所以-1 < a < 1.

设计意图 函数能否取得最值,这直接关系到所求参数a的取值.恒成立与存在性问题中,参数取值范围中等号能否取得,一直是学生的易错点;可以采用验证法,看取等号时是否符合题意,这是顺应学生已有思维,并使之更为严谨.变式3利用恒成立条件缩小参数的取值范围,进而减少分类讨论的情况,优化学生思维,使之更为灵活.

变式4 已知对任意实数x∈[1, 2],不等式x2-logax-3 < 0(0<a≠0)恒成立,则实数a的取值范围为______.

分析 无法参变分离时,可以直接构造原来含参数的对应函数f(x) = x2 - logax - 3.对此函数求最值需要借助导数,而分开的二次函数和对数函数都是我们熟悉的函数,由此考虑先变形得x2 - 3 = logax,变形后对应的两个函数y = x2 - 3和y = logax均可作图.

设计意图 利用数形结合思想转化为函数图象的位置关系,根据图象特征抓住符合条件的关键点列出不等式.突破学生的固有思维,使之更为宽广.

再问:若对任意实数x∈(1, 2)呢?(注意边界处等号是否能够取到.)

类比:让学生回顾热身训练2,如果对函数h(x) = x ln x不是很熟悉,能否变形转化成更熟悉的函数?

启发:x与ln x分开是大家熟悉的,那能否分开呢?由x ln x - a < 0,x ∈[1, e],得lnx<ax,是否也可以用数形结合法呢?帮助学生突破固有思维,抽象出解决同类问题的方法,使学生思维更为宽广深刻.

变式5 不等式x2 - 2ax + 1 > 0对任意实数a ∈[-1, 1]恒成立,求实数x的取值范围.

分析 学生习惯于看成关于x的二次不等式,但本题中a的范围已知、x的范围待求,所以可将a视作自变量、x视作参数,上述问题即可转化为在[-1, 1]内关于a的一次函数函数值大于0恒成立的问题,问题即可求解.令h(a) = -2xa + x2 + 1,则

{h(-1)>0,h(1)>0,

所以

{2x+x2+1>0,-2x+x2+1>0,

故x ≠ 1且x ≠ -1.

设计意图 不等式中出现了两个字母,解题的关键在于该把其中哪个字母看成变量、哪个作参数.总结处理恒成立与存在性问题的又一方法:参变换位法.引导学生从不同角度、不同方位去思考,使学生思维更为灵活开阔.总结处理恒成立与存在性问题的常用方法:参变分离法,构造函数法,数形结合法,参变换位法.

3 从单变量到双变量,培养学生思维的灵活创新

问题1及其变式是探究各种含单变量的恒成立与存在性问题的转化策略.笔者在此基础上进行了延伸拓展,设计了含双变量的恒成立与存在性问题变式系列,以增强学生的创新意识和应变能力,优化学生的数学思维品质,帮助学生构建多元化、变通性数学思维.

问题2 已知函数f(x) = x2 - 2ax + 1,且g(x) = xln x - 1.f(x)在区间(-∞, 2]上是减函数,对任意x1, x2 ∈[1, a + 1],总有| f(x1)-f(x2)|≤4,求实数a的取值范围.

解 令f(x) = x2-2ax + 1 = (x - a)2 + 1 - a2,因为f(x)在区间(-∞, 2]上是减函数,所以a≥2,故a+22≤a<a+1,所以f(x)max = f(1) = 2-2a,f(x)min = f(a) = 1 - a2.

对任意的x1, x2 ∈[1, a + 1],总有 |f(x1) - f(x2) |≤4 ⇔ | f(x1) - f(x2) |max≤4 ⇔ f(x)max - f(x)min≤4,所以(2 - 2a) - (1 - a2)≤4,故a2 - 2a - 3≤0,从而-1≤a≤3.又因为a≥2,所以2≤a≤3.

变式 存在x1, x2∈[1, a + 1],使得 |f(x1) - f(x2) | > 4.

设计意图 对于涉及一个函数的双变量恒成立与存在性问题,运用整体考虑法,变式也可以逻辑转化为原题,考虑命题的否定,即为原题,再取补集即可.

问题3 已知函数f(x) = x2 - 2ax + 1,且g(x) = xln x - 1.对任意x1∈[1, 2],x2 ∈[1, e](e是自然对数的底数),都有f(x1) < g(x2)恒成立,求实数a的取值范围.

分析 依次主元法f(x)max < g(x)min.

方法1 分两种情况讨论求f(x)max.

方法2 g(x)min = -1.优化,利用恒成立缩小参数范围,减少讨论的种类:

{f(1)<g(x)min,f(2)>g(x)min,

即

{2-2a<-1,5-4a<-1,

所以a>32.因为f(x) = (x - a)2 - a2 + 1,所以f(x)max = f(1).

方法3 利用依次主元法转化为例1中单变量问题:g(x)min = -1,问题转化为对任意x∈[1, 2],都有f(x) < -1恒成立,即对任意x∈[1, 2],x2 - 2ax + 1 < -1恒成立.

变式1 若对任意x1∈[1, 2],存在x2 ∈ [1, e],使f(x1) < g(x2)成立,求实数a的取值范围.

变式2 若存在x1∈[1, 2],x2 ∈[1, e],使f(x1) < g(x2)成立,求实数a的取值范围.

变式3 若对任意x1∈ [1, 2],存在x2 ∈[1, e],使f(x1) = g(x2)成立,求实数a的取值范围;

分析 变式1问题等价转化为f(x)max < g(x)max;变式2问题等价转化为f(x)min < g(x)max;变式3从不等关系变为了相等关系,问题等价转化为f(x)的值域 ⊆g(x)的值域.

设计意图 涉及两个函数的双变量恒成立与存在性问题,用依次主元法. 把两个变量进行分离,所求参数与其中一个变量放在一起,则不含参数的一侧可以求出最值,这样就减少了一个变量,从而双变量恒成立问题就转化为单变量的恒成立问题,体现了转化与化归思想,是解决问题的最基本的数学思想方法.题中含有量词两个“任意”与两个“存在”以及一个“任意”、一个“存在”,注意区分.

拓展 设函数h(x)=ax+x+b,对任意a∈[12,2],都有h(x)≤10在x∈[14,1]时恒成立,求实数b的取值范围.

分析 解决双变量问题一般是先解决一个变量,再处理另一个变量,即为依次主元法,通过求函数最值解决问题.

方法1 化归函数最值,h(x)≤10 ⇔ h(x)max≤10.

方法2 参变分离,b≤10-ax-x.变更主元,构造关于a的函数t(a)=-1xa-x+10.

方法3 不进行参变分离,直接变更主元,构造关于a的函数f(a)=1xa+x+b,a∈[12,2],f(a)≤10.

在解决函数存在性与恒成立问题时常常要构造适当的函数,通过函数性质(最值、值域)或图象进行求解.在含多个变量的存在性与恒成立问题中,需要分清变量和参数.

4 教学反思

(1)尊重学习主体,营造学生乐于接受的思维情境

课前的两个热身训练,笔者原本设想有一部分学生用参变分离法、一部分学生用构造函数法、一部分学生用数形结合法,但由于思维定势,学生几乎全都运用了参变分离法.若用一题多解法强行灌输构造函数法和数形结合法,有学生会觉得:“反正这题会做了,为什么还要用别的方法?况且别的方法可能还没有参变分离法便捷.”学生会不乐意听,这样上课费时费力效果还不好.为了帮助学生突破思维定势,所以设计了问题变式系列.

(2)用好问题变式,帮助学生构建思想方法体系

数学教育家乔治·波利亚在总结解题时说:“我们必须一再地变化它,重新叙述它、变换它,直到最后成功地找寻到某些有用的东西为止.”教学中,在学生思维水平的最近发展区里,对数学问题设置多角度、多方位的变式探究,有意识地引导学生观察、联想、类比、化归,从变的表象中发现不变的实质,让学生认识到不同知识方法之间的本质联系,对数学知识、解题方法、数学思想进行感悟、提炼和深化,建立起对一类问题的整体认知,提升学生思维深度和广度,提高复习的有效性.

(3)体验高考真题,提高学生解决问题的能力

布鲁纳说:“向学生提出挑战性的问题可引导学生智慧.”一个有效的学习方法,不但能使学生学习手头的材料,而且能使学生运用所学的知识去解决实际问题.后续教学中逐步让学生亲身体验有一定难度的恒成立与存在性问题的高考真题,让学生经历一个螺旋上升的学习过程,切实提高探究问题的能力.
