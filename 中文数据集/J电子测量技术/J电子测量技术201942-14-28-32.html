    摘    要：

    为了提高生产线作业品质保障中硬盘工件缺陷检测的准确率与稳定性, 以硬盘外壳表面为工件样本, 设计并开发了一种基于DOG特征与神经网络的硬盘表面缺陷检测系统。首先, 以工业相机、环形光、数字光源控制器、传送带和数据分析服务器为基础, 设计以图像识别为核心的机器视觉缺陷检测系统。然后, 利用高斯卷积表现尺度空间, 基于图像金字塔多分辨率, 进行差分处理, 得到DOG特征, 完成神经网络缺陷识别器的数据准备。随后, 结合多种深度神经网络的感知机与激活函数, 创建神经网络分类模型, 设置迭代终止准则, 训练带有标签的缺陷样本数据集, 完成分类预测, 达到准确检测缺陷的目的, 将带有缺陷的工件通过传送带引流到不良品聚集区, 保障生产线的产品品质输出。最后基于VS开发平台、OpenCV开源库与C++语言编程实现系统。设计对比实验, 与现有系统和人力对应流程进行实验对比, 经验证, 本系统具有更好缺陷检出能力。

    关键词：

    机器视觉; DOG特征; 深度学习; 缺陷检测; 激活函数;

    Workpiece surface defect detection algorithm based on DOG features and deep learning

    Chang Bo

    Shanxi Communications Vocational and Technical College

    Abstract：

    In order to improve the accuracy and stability of defect detection of hard disk workpiece in quality assurance of production line, this paper designs and develops a system based on DOG feature and neural network for defect detection of hard disk workpiece. Firstly, based on industrial camera, ring light, digital light source controller, conveyor belt and data analysis server, a machine vision defect detection system with image recognition as the core is designed. Then, using Gauss convolution to represent scale space, based on image pyramid multi-resolution, differential processing is carried out to obtain DOG features and complete the data preparation of neural network defect recognizer. Then, combining the perceptron and activation function of the neural network, the classification model of the neural network is created, the iteration termination criterion is set up, the defect sample data set with labels is trained, the classification prediction is completed, and the defect is detected accurately. The defective workpiece is drained to the defective product aggregation area through the conveyor belt to guarantee the students. Product quality output of production line. Finally, the system is developed based on Visual Studio, OpenCV open source library and C++ language programming. Designed comparative experiments, compared with the existing system and human corresponding process, the system has better defect detection ability.

    Keyword：

    machine vision; DOG feature; deep learning; defect detection; activation function;

    随着“中国智造2025”的感念不断深化, 我国从世界工厂转型为智造中心的趋势已经越来越明显, 市场倒逼企业转型, 从人力密集型向智能设备密集型逐渐过渡。制造业工厂内生产线作业流程与工序, 已经逐步被智能化MES系统、智能AGV、堆垛机器人、智能机械手、智能品质检测取代[1,2]。其中智能缺陷检测, 以计算机视觉技术取代人眼, 以每天24 h不间断工作模式, 客观化精准化地完成重复外观缺陷检测工作。这个智能化缺陷检测过程, 也不可避免的存在各种挑战, 比如:产品型号、产品外观缺陷种类会变化;现场环境光线改变, 影响成像质量;视觉硬件故障或者人员干预误操作等, 这些不确定因素, 都对我们的缺陷检测系统提出了更高的要求。

    目前已提出的缺陷检测图像算法中, 郭永平等[3]提出了基于机器视觉缺陷检测可重构系统, 采用阈值设置、感兴趣区域标识、算法选择、机器自动学习以及图像帧自动存储, 完成对产品表面缺陷的检测, 优点在于系统可重构, 可以对应多种型号产品展开缺陷识别, 但是缺点在于缺乏对某种特定型号的深耕式缺陷检测, 当产品外观缺陷种类发生较大变化时, 算法的检出能力往往大受影响。赵海峰等[4]提出基于机器视觉的小微零件缺陷检测算法, 耦合目标成像、图像滤波、图像增强, 利用图像差值算法得到标准图样对比, 推导出缺陷程度和位置, 但是该技术过度依赖标准化图样, 往往缺乏普适性, 当工件成像受环境影响因素较大或者缺陷种类变化较大时, 往往不能对应缺陷检测任务。

    为了提高硬盘表面缺陷的检出能力, 本文首先从打光实验开始, 最终确定视觉硬件, 组合工业相机、镜头、光源和计算机, 建立视觉缺陷检测的硬件平台。然后从底层自主开发视觉算法, 基于混合高斯模型和金字塔结构, 提取DOG特征;研究主流深度学习框架Tensorflow, 融合多种深度学习算法, 构成本文定制化的深度学习解决方案。最后以软件工程思维和标准, 完成从算法、模块到软件系统的开发, 达到缺陷检测算法在现场落地的目的。

    1 硬盘外观缺陷检测系统的架构

    2 缺陷检测算法

    为了准确检测出硬盘表面缺陷, 克服来自成像亮度和角度带来的干扰, 首先基于混合高斯[5,6]进行建模。

    G (x, y, σ) =12πσ2exp (-x2+y22σ2)           (1)

    式中:G代表二维高斯函数;x、y代表图像横、纵坐标;σ代表高斯核, 可以产生出多尺度空间的核[7]。

    根据式 (1) , 建立图像尺度空间。

    L (x, y, σ) =G (x, y, σ) ×I (x, y) (2)

    式中:L代表图像尺度空间函数。

    随后, 引入金字塔结构[8,9], 建立多个尺寸的子图像, 为后续高斯模型多尺度表达提供数据。金字塔是图像多尺度表达形式, 包括低通滤波器平滑图像, 以及降采样图像[9]。通过结合混合高斯尺度空间和金字塔结构, 建立本文的DOG特征, 即高斯金字塔特征。

    g=∂2G∂x2+∂2G∂y2          (3)

    式中:g代表DOG特征函数值。

    对于上述的图像尺度空间, 本文主要采用OpenCV这个开源视觉函数库实现, 而混合高斯模型则是基于函数GaussianBlur实现。高斯金字塔结构, 主要是利用基于buildGaussianPyramid函数实现。本文的DOG特征函数值, 基于buildDoGPyramid函数实现。

    通过式 (3) 可得到采集图像的DOG特征值, 构成特征训练集。首先, 借助工业相机来收集缺陷硬盘图像, 并标注特征图像帧, 作为后续深度学习的训练集, 在所提算法中, 总共收集14 000条特征集。再借助深度学习对其完成训练[10,11]。深度学习与传统的前馈多层网络的区别在于:深度学习网络具有更多的神经元, 而且具有更复杂的连接方式, 可以进行特征自动提取[11]。在所提算法中, 将训练集放入分割主流深度学习框架中:卷积神经网络、循环神经网络和递归神经网络, 以此来完成缺陷特征集训练。卷积神经网络基于共享权重在空间中进行扩展的标准神经网络, 可以通过内部卷积来识别图片, 内部卷积可以看作为待识别物体的边缘[12];循环神经网络是在时间上进行扩展的标准神经网络, 可以有效识别序列, 如语音信号或者文本。递归神经网络类似于分层网络, 其中输入序列没有真正时间面, 而输入序列必须以树状方式分层处理[13]。

    1) 将训练集数据输入卷积神经网络, 主要分为3步:卷积、最大池化和全连接神经网络计算[14,15]。其中, 卷积操作函数为:

    F (x, y) =f (x, y) ×K (4)

    Κ= (10-11)           (5)

    式中:f代表卷积前图像;F代表卷积后图像;K代表卷积核函数。

    2) 对卷积结果完成最大池化处理。对于图像中每个2×2个像素值的窗口, 选出4个像素值中最大的像素值作为输出矩阵的相应元素的值[15]:

    ga=ΜAX (cdef)           (6)

    式中:ga代表最大池化后的图像局部结果;c、d、e、f代表局部区域像素。

    3) 依据卷积神经网络训练与识别分类的过程, 完成最大池化后, 再对数据进行全连接。最后, 对神经元进行有权重连接, 通常全连接层在卷积神经网络尾部。在本文系统基于深度学习库tensorflow的卷积神经网络函数:卷积函数conv_op, 池化函数max_pool_2x2。本文系统是基于C++语言与TensorFlow函数来完成硬盘表面缺陷检测。

    本文的深度学习函数中权重初始化, 采取的是结合随机和经验并行的方式, 先考虑一个小批量数据, 初期在所需的特征激活的异常值进行标准化;初始层中的微小扰动都会导致后面层的非常大的变化;在反向传播过程中, 这些现象会导致梯度弥散;学习权重产生所需要的输出前, 必须对梯度的异常值进行补偿, 这将导致需要额外的时段来收敛, 由此完成本文的深度学习模型。基于以上检测方案, 对图3进行处理, 得到检测结果如图4所示, 可见缺陷轮廓检测精准, 达到了准确识别缺陷的工程目的。

    3 实验结果及分析

    本文开发的系统, 经过性能速度实验, 以满足现场单位工作时间要求的前提下, 通过测试验证, 得到一个性价比较高的计算机配置, 增强设备在市场上的竞争力。计算机配置选型为:Intel i5的双核CPU, 8 GB的内存, 操作系统为Windows 7的64 bit版本。系统基于微软Visual Studio2017平台, 结合OpenCV开源库视觉函数和TensorFlow深度学习框架, 对本文提出的基于DOG和深度学习的硬盘外观缺陷检测系统以及对照组的检测性能进行验证。

    为了评估系统的检测准确率, 本文设立了对照组, 对照组为文献[3]和文献[4], 分别记为A、B算法。如图5所示为本文技术系统界面, 系统功能有:图像读取、DOG特征提取、深度学习、缺陷检测。如图6所示为待检测的原图, 其中有一硬盘, 框线内有表面缺陷, 即待检测目标。分别用本文技术、A组技术、B组技术进行处理, 结果如图7～9所示。根据系统识别后自动标注的轮廓线发现, 本文技术自主设计打光方案, 完成视觉选型和图像采集, 基于混合高斯模型和金字塔结构, 完成DOG特征提取, 基于多种深度学习方法, 完成缺陷检测, 如图7所示, 本文技术精准定位缺陷轮廓, 完成缺陷检测。A组技术采用阈值设置、感兴趣区域标识、算法选择、机器自动学习以及图像帧自动存储, 完成对产品表面缺陷的检测, 优点在于系统可重构, 可以对应多种型号产品展开缺陷识别, 但是缺点在于缺乏对某种特定型号的深耕式缺陷检测, 当产品外观缺陷种类发生较大变化时, 算法的检出能力往往大受影响, 如图8所示, 未能准确检测出缺陷, 对非缺陷区域误识别。B组技术耦合目标成像、图像滤波、图像增强, 利用图像差值算法得到标准图样对比, 推导出缺陷程度和位置, 但是该技术过度依赖标准化图样, 往往缺乏普适性, 当工件成像受环境影响因素较大或者缺陷种类变化较大时, 往往不能对应缺陷检测任务, 如图9所示, 存在明显的漏检, 没有检测出典型缺陷。

    本文算法是根据图像目标特性, 设计视觉硬件系统, 有机融合工业相机、环形光、数字光源控制器、传送带和数据分析服务器, 建立视觉缺陷检测系统硬件平台, 采集凸显特征的图像。由于缺陷表面特征不明显, 本文算法利用了高斯卷积表现尺度空间, 基于图像金字塔多分辨率, 进行差分处理, 得到DOG特征。随后, 根据大批量数据图像样本, 启动深度学习训练, 结合多种深度神经网络的感知机与激活函数, 创建神经网络分类模型, 设置迭代终止准则, 完成分类预测。通过检测结果发现, 本文算法可以准确定位到缺陷位置, 而对照组算法存在漏检缺陷目标, 将良品位置判为缺陷, 这说明本文算法要优于对照组。

    为了客观反映出本文算法的稳定性, 以硬盘外壳为样本, 如图10所示, 通过对其外壳进行不同长度不同面积的划伤, 以及不同位置涂加污渍等方式, 采用工业相机对这些外壳存有缺陷的工件进行采集, 总共获取220幅图像。随后, 利用本文算法、文献[3]和文献[4]算法对220幅带有缺陷的工件图像完成检测, 统计每种算法对应的正确检测数量, 以获取正确率, 结果如表1所示。根据表1统计结果发现, 本文算法的正确检测数量最多, 约为208幅图像, 正确率约为94.54%, 而文献[3]和文献[4]技术的正确检测图像分别为194、185幅, 对应的正确率分别为88.18%、84.09%。这说明本文技术具有更好的稳定性与正确检测率。

    4 结 论

    为了提高硬盘表面缺陷的检出能力, 本文设计并开发了基于DOG特征与深度学习的硬盘表面缺陷检测算法。有机融合工业相机、光源、传送带、硬盘和计算机, 构建起一个基于视觉硬件的缺陷检测系统。利用传统图像算法, 完成特征提取, 利用主流深度学习方法, 完成基于数据驱动的缺陷检测。自主研发机器视觉缺陷检测算法, 完成对硬盘表面缺陷的准确检测, 达到了以机器视觉代替人眼人脑, 降低成本, 提高产品品质的目的。基于Visual Studio平台实现本文系统, 并进行了对比实验测试。结果表明, 本文系统在硬盘表面缺陷检测方面具有更高准确度。

    硬盘表面缺陷识别分类是本文目前研究成果上后续开展的内容, 如划痕、疵点和破损等, 在得到具体缺陷种类和程度后, 利于分类机械手做分流处理, 不仅进一步降低生产损耗, 同时提高工作效率。
