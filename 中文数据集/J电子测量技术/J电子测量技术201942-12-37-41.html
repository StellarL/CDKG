    基于MRAS的永磁同步电机无传感器滑模控制

    曹兆锦

    南京理工大学泰州科技学院

    摘    要：

    为了改善转速误差的估计信息及转子位置估计误差信息, 提高了误差的准确度, 在三相永磁同步电机的矢量控制算法中, 基于MRAS观测器采用滑模变结构速度控制器代替PI速度控制器, 将PI控制与模型参考自适应系统结合起来。并且针对滑模变结构的不连续性所引起的转速抖动, 利用正弦饱和函数的光滑特性对滑模速度控制器进行优化。首先在三相永磁同步电机的矢量控制系统的基础上, 根据算法公式建立了参考模型自适应观测器, 再设计滑模面建立控制模型, 最后通过MATLAB仿真软件进行实验。理论与仿真分析结果表明, 改进后的系统将PI控制器的超调量消除, 并且转速到达稳态的过程所花费的时间大大减小, 转速误差估计值与转子位置估计误差的精度更高, 系统整体鲁棒性得到加强, 检测精度增高。

    关键词：

    模型参考自适应; 滑模控制; 无传感器; PI控制; 永磁同步电机;

    Sensorless sliding mode control of permanent magnet synchronous motor based on MRAS

    Cao Zhaojin

    Institute of Intelligent Manufacturing, Taizhou Institute of Science and Technology, Nanjing University of Technology

    Abstract：

    In order to improve the estimation information of rotational speed error and rotor position error, and improve the accuracy of the error, in the vector control algorithm of three-phase permanent magnet synchronous motor, based on MRAS observer, sliding mode variable structure speed controller is used instead of PI speed controller, and PI control is combined with model reference adaptive system. In order to solve the speed jitter caused by the discontinuity of sliding mode variable structure, the sliding mode speed controller is optimized by using the smooth characteristic of sine saturation function. Firstly, based on the vector control system of the three-phase permanent magnet synchronous motor, the reference model adaptive observer is established according to the algorithm formula, then the sliding mode surface is designed to establish the control model, and finally the experiment is carried out by MATLAB simulation software. Theoretical and simulation results show that the improved system eliminates the overshoot of PI controller, and the time taken for the speed to reach steady state is greatly reduced. The accuracy of the speed error estimation and the rotor position estimation error is higher, the overall robustness of the system is enhanced, and the detection accuracy is increased.

    Keyword：

    model reference adaptation; sliding mode control; sensorless; PI control; permanent magnet synchronous motor;

    0 引 言

    随着时代的进步与科技的发展, 永磁同步电机 (permanent magnet synchronous motor, PMSM) 作为一种具有结构简单、电机损耗小、利用效率高等明显优点的优秀电机, 在矿业发展中得到广泛的推广与应用。PMSM是一种非线性的、耦合性较高的复杂系统, 在工业发展中, 需要发挥出电机的高性能, 所以高性能控制方式得到众多学者的研究, 如矢量控制、无传感器控制、直接转矩控制等, 以及衍生出的改进控制方式, 但都需要对转速的大小、转子的位置误差进行估计, 传统上一般是通过安装机械传感器来对转速、转子进行检测, 但是这种方式的可靠性较低, 且成本增加, 不适合现代电机控制, 于是低成本和可靠性高的无传感器控制方式得到推广。

    目前, 诸多学者已经对模型参考自适应 (model reference adaptive system, MRAS) 无传感器控制进行了较为广泛的研究。文献[1,2]建立了表贴式PMSM的数学模型, 设计了MRAS速度观测器, 实现了无速度传感器的控制。文献[3]在研究了MRAS无传感器控制的基础上, 采用一种基于自适应律的PI控制方式, 以期实现对转速与位置的精确估计。文献[4]为了提高控制系统的性能, 采用了基于RBF与MRAS混合控制方式, 利用自定义的误差信号经过RBF网络进行调节最终得到估计转速。文献[5]在PID的控制理论上结合anti-windup与模糊控制理论, 提出了一种变结构PID控制器, 通过MRAS观测到转速的超调量得到有效抑制。文献[6]在综合了滑模变结构控制与MRAS的优点上, 提出了一种变结构的模型自适应速度观测器, 最终实现对实际速度的有效跟踪。文献[7]为了达到使PMSM调速系统的高性能控制, 在MRAS无速度传感器控制的基础上设计出了一种模糊PI速度控制器, 与传统的PI控制器进行比较, 仿真验证了调速系统的优秀效果。

    上述研究效果各有千秋, 综合近几年来滑模控制的研究成果, 本文提出了一种基于MRAS的滑模速度控制系统, 传统的PI控制器响应速度慢, 超调量较高, 利用滑模速度控制器代替传统的PI速度控制器[8], 可以加快系统响应速度, 提高系统的稳定性。通过MATLAB平台进行仿真[9], 搭建系统模型, 最终结果证明, 在滑模速度控制下的电机可以准确估计转速与转子位置, 且转速效果明显优于传统PI控制。

    2 滑模速度控制器

    在多种三相PMSM无传感器矢量控制方式中, iq=0的控制策略对于表贴式PMSM的转子磁场定向控制有较好效果。现在较为普遍的控制系统中通常利用PI控制对iq进行控制, 滑模控制是变结构控制系统中许多策略之一, 其对多变量、非线性的高耦合的复杂系统具有良好的控制效果。

    电机运动方程为:

    Jdωdt=32Ρnψfiq-ΤL          (11)

    2.2 趋近律设计

    在滑模控制中, 需要选取一个合适的趋近律来保持滑模的动态特性, 一般有如下几种趋近律可供选择:1) 等速趋近律;2) 指数趋近律;3) 幂次趋近律, 还有许多改进型趋近律如双幂次趋近律、变阶次趋近律等。一个良好的趋近律会使滑模的正常运动阶段的品质得到提升, 所以本文选取指数趋近律[12]。

    一般采用理想滑动模态中的符号函数sign (s) 来保证滑模的优秀品质, 但是滑模固有抖震难以消除, 所以在趋近律中采用一种正弦饱和函数来消除滑模抖震[13], 提高滑模的运行品质。公式为:

    sat (s, σ) ={sign (s) , |s|>σsin (∂s) , |s|≤σ          (17)

    式中:∂=π2σ;σ为边界厚度。

    则趋近律为:

    s˙=-nsat (s) -cs          (18)

    其中:n>0, c>0.

    控制器的表达式为:

    u=1Μ[ax2+nsat (s) +cs]          (19)

    则q轴参考电流方程为:

    iq=1Μ∫0t[ax2+nsat (s) +cs]dτ          (20)

    2.3 李雅普诺夫稳定性证明

    滑模变结构运动控制系统由趋近运动和滑模点运动组成, 其中趋近运动需要满足滑模的存在性和可达性条件, 即ss˙<0, 滑模状态变量可在限定时间内达到滑模面s=0。

    选取李雅普诺夫函数:

    V=12s2          (21)

    据李雅普诺夫稳定性判据[14]可知, 满足条件:

    V˙=ss˙=s (-nsat (s) -cs) =-n*s*sat (s) -cs2 (22)

    由n>0, c>0可知:V˙<0证明成立, 滑模可以在限定时间内到达滑模面, 系统稳定。

    4 结 论

    在三相PMSM的无传感器控制系统中, 为了能够使电机高性能的运行, 在基于MRAS的控制方式下能够准确的估计出电机转速的大小误差与转子的位置误差, 在滑模控制与参考模型自适应结合的控制系统中, 滑模控制的抖振得到了有效抑制, 电机实际转速响应较PI控制快, 转子位置估计误差也相对更小, 从而可以证明滑模控制比PI控制的效果更好。本文建立的仿真模型在速度控制器方面已有一定的改进, 但是仿真不足以说明问题的准确性, 在后续的研究中, 还需要进行实验进一步验证仿真的效果。
