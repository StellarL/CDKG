    摘    要：

    顾及扩展贝叶斯分类算法(Extended Bayesian Classification Algorithms,EBCA)的计算机网络涉密信息安全应用系统,当前仅仅处于发展的初始阶段。多研究人员都无法描述相互之间的一些关系。本研究以计算机网络安全的涉密信息安全为核心,巧妙利用EBCA反向传播模型,综合使用非线性函数的功能提出了顾及EBCA的计算机网络涉密信息安全应用系统,有效解决了复杂的交联关系,同时通过仿真实验对所提出的模型进行了有关验证。顾及EBCA的计算机网络涉密信息安全应用系统的设计与实施提供理论支撑。

    关键词：

    扩展贝叶斯分类算法; 安全指标; 计算机网络; 安全涉密信息安全;

    Computer network secret-related information security application system considering extended bayesian classification algorithms

    Wang Bin

    Shangluo University

    Abstract：

    Considering the extended Bayesian classification algorithm, the computer network security application system is only in the initial stage of development. Many researchers are unable to describe some of the relationships between them. This research takes the security of computer network security as the core, skillfully utilizes the back-propagation model of extended Bayesian classification algorithm, and synthetically uses the function of non-linear function, proposes an application system of computer network security involving extended Bayesian classification algorithm, which effectively solves the complex cross-linking relationship. At the same time, the model is validated by simulation experiments. Considering the extended Bayesian classification algorithm, the design and implementation of computer network secret information security application system provides theoretical support.

    Extended Bayesian classification algorithm; Security index; Computer network; Security-related information security;

    计算机网络安全对航空交通安全及涉密信息安全至关重要[1]。在国家的产业中,民航产业是非常重要的一种信息化产业,它不但关系到国家的领空安全及涉密信息安全,而且关系到国家的经济安全及涉密信息安全,同时还关系到社会安全及涉密信息安全[2,3]。由此可见,加强国家网络安全及涉密信息安全防御战略是一项非常重要的工程[4]。采用网络安全及涉密信息安全,可以有效了解国家空间网络信息的整体概况,从而让很多安全事故防患未然[5]。在计算机网络涉密信息安全应用系统建设方面,使用最多的方法是贝叶斯分类算法,并且国内外很多学者都采用该方法获得了较大的研究突破[6]。从重要性角度分析,计算机网络涉密信息安全应用系统跟国家电力网络相比拟,EBCA的应用领域更广,并且也形成了许多较为成熟的技术成果[7,8]。但是也需注意,现阶段顾及EBCA的计算机网络涉密信息安全应用系统的研究还是处在初期阶段,特别是各个目标属性的关联性,表现为非线性关系的比例达到了至少一半以上,若是采用常规化的方法是很难来反映这种关系的;特别是针对于有的问题,其来源是不明确的,在分析的过程中常常会出现一些矛盾,甚至还会出现没有条理的现象[9,10]。

    顾及上文论述的情况,本研究以计算机网络安全的涉密信息安全为重点内容,合理选择EBCA反向传播模型,并根据该模型的具体特征,充分发挥非线性函数的特点,精确反映了计算机网络包含的各类复杂关系,从而有助于对网络和涉密信息安全都进行全面、准确的分析[11,12]。

    1 计算机网络涉密信息安全应用系统

    1.1 计算机网络信息安全系统介绍

    从组成结构上看,计算机网络信息安全系统包含了6个部分,分别为通信卫星系统、地面空中交管制(ATC)系统、空间导航系统、信息服务系统、空中机载电子系统、平面通信网络系统。[13,14,15]在本文的研究中,结合有关专业理论知识将计算机网络涉密信息安全类型分为三种层次,各层次结构关系见图1:

    1)最底层是物理层,也被称为新航行系统CNS层,在本层中,通常是由基础性设施构成,例如空间卫星系统、导航系统、地面通信系统等。

    2)中间层主要是由网络层构成,这一层通常也被称作数据传输层,该层基本是由传输性设施构成,包括地区空管骨干网络、空管局网络以及管理信息网络等。

    图1 计算机网络的层结构   下载原图

    3)最高层为应用层,也被称为空管业务层,在业务层中主要包括多种业务,比如说航行情报业务、通信导航业务、气象业务、信息网络业务等。

    设计计算机网络涉密信息安全应用系统时需以上述三层结构作为核心内容,并对各层实施保障性处理,例如可以根据不同的技术方法来完成入侵检测、数据加密以及设置有效的防火墙等。

    1.2 扩展贝叶斯分类算法

    EBCA是一种前馈逻辑定律,但它是一种多层结构的逻辑定律,在网络权值调节方面,通常选择后向传播的形式。使用扩展贝叶斯分类算法模型时,该模型功共包含三层结构,从第1层到第3层分别为输入层、隐含层与输出层,并且在隐含层中可能含有多层结构。该网络的具体模型结构可以参考图2。

    图2 扩展贝叶斯分类算法拓扑图   下载原图

    对图2模型结构进行分析可以发现,对于这种具有三层结构的前馈网络,将向量X=(x1,x2,…,xi,…,xn)T输入到网络中,经过隐含层处理后,再输出向量Y=(y1,y2,…,yj,…,ym)T,之后进入输出层完成后续处理,并输出向量O=(o1,o2,…,ok,…,ol)T,对向量进行处理后可以得到期望输出向量为d=(d1,d2,…,dk,…,dl)T。要注意输入层与隐含层之间的权值矩阵,我们可以将这个权值矩阵表示为V=(V1,V2,…,Vj,…,Vm)T,列向量Vj表示的是与第j个信息互相匹配的权向量,同时还要注意在隐含层与输出层间存在一个权值矩阵,通常可以将该权值矩阵表示成W=(W1,W2,…,Wk,…,Wl)T,其中,Wk代表列向量,对应第k个信息可以实现互相匹配性的权向量。

    1.3 计算机网络涉密信息安全

    本研究中,所设计的顾及扩展贝叶斯分类算法的计算机网络涉密信息安全应用系统,具体结构图可以参考图3。上文论述的三层计算机网络结构与扩展贝叶斯分类算法间形成了互相对应关系,如下所示:

    1) CNS层属于物理层,该层通常是由传感器设备组成,对应于扩展贝叶斯分类算法的为输入层,其中,输入向量是X=(x1,x2,…,xi,…,xn)T。xi代表传感器设备经过处理得到的输出信号。

    2) 网络层对应的是传输层,该层包含了具有复杂结构的网络系统,隐含层对应扩展贝叶斯分类算法,在这一层内的输出向量为Y=(y1,y2,…,yj,…,ym)T。Yj指的是所对应的计算机网络。

    图3 计算机网络涉密信息安全应用系统   下载原图

    3) 应用层对应输出层,该层包含了许多最终的业务数据,在扩展贝叶斯分类算法所对应的就是输出层,在该层中。输出向量为O=(o1,o2,…,ok,…,ol)T。ok指的是那些已经通过一定的方式处理过的计算机网络。

    EBCA安全体系是一个采用多输入单输出为特性的系统。在本研究中,采用扩展贝叶斯分类算法对空管信息安全及涉密信息安全进行全新的构建,以EBCA为基本核心,得出全新的安全涉密信息安全。

    若网络输入值和期望输出结果见存在差异时,会引起输出误差的问题,将其表示成E,可以根据如下公式得到:

    E=12(d-Ο)2=12∑k=1l(dk-ok)2         (1)

    将上述表达式扩展至隐含层之后,可以得到如下表达式:

    E=12∑k=1l[dk-f(netk)]2=12∑k=1l[dk-f(∑j=0mwjkyj)]2         (2)

    再将上述表达式扩展至输出层之后,可以得到如下表达式:

    E=12∑k=1l{dk-f[∑j=0mwjkyjf(netj)]}2=12∑k=1l{dk-f[∑j=0mwjkyjf(∑i=0nvijxi)]}2         (3)

    根据以上式子可以发现,网络输入误差受到不同层的权值wjk、vij影响,因此,为获得不同的误差E值就要求对权值进行合理调整。非常明显,我们调整权值的目的就是让误差不断减小。换句话说就是让误差梯度减小和权值调整量之间呈正相关的变化关系,具体而言,可通过以下表达式来得到:

    Δwjk=-h∂E∂wjk,Δvij=-h∂E∂vij,j=0,1,2,⋯,m;k=1,2,⋯,l;i=0,1,2,⋯,n;j=1,2,⋯,m         (4)

    在以上表达式中,负号代表梯度发生降低的过程,h为常数,对应一个特定的比例系数,介于0～1范围内,可以利用EBCA算法来计算误差梯度的变化。

    以下内容主要推导调整EBCA算法权值的计算式。首先分析输出层,存在如下关系:

    Δwjk=-h∂E∂netk∂netk∂wjk,j=0,1,2,⋯,m;k=1,2,…,l (5)

    再研究一下隐含层,在隐含层中有:

    Δvij=-h∂E∂netj∂netj∂vij,i=0,1,2,⋯,n;j=1,2,…,m (6)

    以上述两个表达式为基础,在输出层与隐含层中,分别定义误差信号令:dk0=-∂E∂netk和djy=-∂E∂netj,根据设定规则来处理第一个误差信号,同时把公式(5)转换为下述表达式:

    Δwjk=hd0kyj (7)

    运用相同的方法,对第1个误差信号也按照同样规则来处理,采用式(6)来完成权值调整的过程,也可将其改写成下述式子:

    Δvij=hdyjxi (8)

    综上所述,以公式(7)与公式(8)为基础,求解得到信号dk0与djy,此时可以推导得到权值调整表达式。在输出层内,展开dk0后将得到:

    dk0=∂E∂netk=∂E∂ok∂ok∂netk=∂E∂okf′(netk)         (9)

    隐含层中djy,将其展开之后可以得到:

    djy=∂E∂netj=∂E∂yj∂yj∂netk=yjf′(netj)         (10)

    之后再推导以上二个误差信号,在式(7)与(8)中重新代入式(9)和式(10),由此获得如下所示的EBCA 3层网络权值调整公式:

    Δwij=hd0kyj=h(dk-ok)ok-(1-ok)yj (11)

    Δvij=hdjyxi=h(∑k=1ldk0wjk)yj(1-yj)xi         (12)

    在EBCA算法中,关于扩展贝叶斯分类算法的所有信息,都要求激活函数满足特定条件,即函数要应具备可微的特性。通常应选择Sigmoid型对数,也可以选择线性函数或正切函数。从图4中可以看到正切函数和对数函数各自对应的曲线:

    图4 正切函数与对数函数的曲线分布图   下载原图

    对于Sigmoid型函数来说,其具备自身独有特性,该函数具有非线性放大系数的特性,一个输入信号,它可以由负无穷大被放大至正无穷大,并将输出结果控制在-1～1的范围内。实际放大系数通常取决于输入信号,当输入信号处于较大值的状态下,则会引起放大系数的减小,当输入信号较小时,则会得到较大的放大系数。因此巧妙使用Sigmoid激活函数,可以有效优化和逼近非线性的输入/输出关系。

    2 仿真测试

    本文主要采用计算机网络数据来开展仿真和测试过程。在选择这类运行数据时,其来源有着严格的要求,在本文中所采用的数据,取自于中国民航各地区空管单位的值班记录。这些运行记录数据是本文研究的样本数据,将这些数据用在EBCA安全涉密信息安全的研究之中。建设计算机网络涉密信息安全应用系统的最终目的是为了确保系统中的各个计算机网络设施能够实现正常运行。通常可以选择二个指标来衡量计算机网络设备的运行状态,分别为设备运行的正常率与完好率。以下内容将对上述二个指标的具体计算方法进行分析。

    1)计算设备的运行正常率时,可根据下述公式进行:

    设备运行正常率=计划运行时间(小时数)-不正常运行时间(小时数)计划运行时间(小时数)×100%         (13)

    其中,计划运行时间是指系统处于正常运行状态的总时间(本文是指包含的总小时数),可以把累计期限分成如下三类:月期限、季度期限与年度期限;不正常运行时间是指系统无法正常工作的时间(这里是指包含的总小时数)。

    其中,设备总台数是指处于设备保质期范围内的设备总台数,故障设备总台数是指处于设备保质期中发生故障问题的设备总台数。具体的计算过程阐述如下:

    (1)假设某个节点是攻击者的起源,起源可以为任意系统或设备,一般为外网攻击者。假定任意攻击者能获取起源设备的输入权限为 anonymous,也即其能查看到起源设备或系统的信息,但无法登录;

    (2)通过风险评估、安全检查等方式获得系统存在的漏洞信息,确定攻击者所在设备是否存在风险漏洞,如存在风险漏洞则检查漏洞利用后攻击者能获得的输出权限,如果输出权限为{user,root},则攻击行为可能继续,否则攻击过程结束。

    (3)攻击者通过所在节点的风险漏洞,使其权限得到提升,如果攻击者不能得到观察下一节点的权限(即user 或 root 权限),则攻击过程终止,攻击者所能影响的范围为当前已查看过的主机或设备;如攻击者利用漏洞能够获取高于 anonymous 的权限,则攻击者能其连接的其他节点,进而去尝试是否下一节点可达(取决于下一系统的输入权限);攻击者将不断地重复步骤(3),直到达到目标节点或由于耗费过高而放弃。

    (4)由于攻击行为往往会因遇到安全设备和安全策略的阻拦,而导致其成功实现其攻击目的的时间、精力甚至资金成本提高,攻击者为达到其攻击目标所付出的所有的行为成本即攻击代价,其可以接受的最大攻击代价为攻击代价阈值。假设网络安全策略对目标区域的防护策略使得攻击者使用任意攻击链路所产生的代价都要大于攻击代价阈值 t,则说明目标区域是安全的;否则说明存在一条代价可接受的攻击链路,使得攻击者得以进入目标区域。

    依据设备的分类,在本研究中所选取的对象都是一些非常常用的设备,同时在本文中还将这些保护对象进行了安全及涉密信息安全等级的划分。

    1)基础运行样本数据的选取

    ①无线通信设备(Wireless communication equipment): VHF(very hige frequency,甚高频)数据链系统、VSAT(very small aperture terminal,甚小口径终端)卫星系统。

    ②有线通信设备(Wird communication equipment):内话系统(Telephone system)。

    ③导航设备(Navigation equipment):DVOR(doppler vhf omni-direction range,多普勒全向信标)系统、ILS(instrument landing system,盲降)系统、DME(distance measurement equipment,测距机)系统、GPS(Global positioning system,全球定位系统)系统。

    ④监视设备(Monitoring equipment):SSR(secondary surveillance radar,二次雷达)系统与ADS(automatic dependent system,自动相关系统)。

    ⑤航管自动化设备(Air pipe automation equipment):飞行计划(Flight plan)、飞行情报(Flight information)和监控系统(Monitoring system)。

    上文所论述的设备,在计算机网络中都是典型代表,可以准确有效全面地反映安全及涉密信息安全状况。依据这些设备所提取的记录数据,可以全面准确有效地反映计算机网络的运行状况,依据这些数据,还可以有效地分析网络安全的可靠性和合理性及其完备性。

    3 结 论

    本文提出了一种顾及扩展贝叶斯分类算法的计算机网络涉密信息安全应用系统方法。通过仿真实验对所提出的方法进行了有关验证。仿真实验表明,提出的方法特别适合计算机网络安全的非线性特性,可以准确全面系统地反映网络安全及涉密信息安全运行状态。再通过仿真测试,测试发现本研究中的方法可以有效满足设备完好性与设备正常率的误差要求。
