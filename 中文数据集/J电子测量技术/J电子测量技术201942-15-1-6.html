    摘    要：

    针对以多颗GNSS卫星作为外辐射源的无源雷达目标跟踪问题, 提出一种基于机器学习的无源雷达目标定位跟踪算法。该算法通过线性回归和梯度下降算法训练得到时延和多普勒频偏的模型, 根据模型对低信噪比下互模糊函数估计出的多普勒频偏进行修正, 之后构建了基于4颗卫星的时延和修正后的多普勒频偏的定位算法模型实现对目标的定位跟踪。仿真结果表明, 该算法与直接利用互模糊函数估计的时延和多普勒定位跟踪目标的原始算法相比, 位置估计精度提高了11.1倍, 速度估计精度提高了3.6倍, 能够更加有效快速的定位跟踪空中目标, 同时, 与粒子滤波的算法相比, 该算法不需要长时间的累计, 可以在更短的时间内对目标进行定位跟踪, 效率更高。

    关键词：

    无源雷达; 梯度下降; 线性回归; 互模糊函数;

    Linear regression-gradient descent algorithm for passive radar target tracking

    Peng Zhangyou Xia Haiqin

    Key Laboratory of Specialty Fiber Optics and Optical Access Networks, Shanghai University

    Abstract：

    Aiming at the passive radar target tracking problem with multiple GNSS satellites as external radiation sources, a passive radar target tracking algorithm based on machine learning is proposed. The algorithm obtains the model of delay and Doppler by linear regression and gradient descent algorithm, and the value of Doppler shift which estimated by the cross-ambiguity function in the case of low SNR is corrected according to the trained model, and then a location algorithm model based on the value of four satellites' delay and Doppler is constructed to achieve the tracking of target. The simulation results show that the proposed algorithm can improve the position estimation accuracy by 11.1 times and the speed estimation accuracy by 3.6 times, which is more efficient and faster than the original algorithm that directly uses the value of delay and Doppler which estimated by cross-ambiguity function. At the same time, compared with the particle filter algorithm, this algorithm does not need long time accumulation, and can locate the target in a shorter time with higher efficiency.

    Keyword：

    passive radar; gradient descent; linear regression; cross-ambiguity function;

    基于全球导航卫星系统 (Global navigation satellite system, GNSS) 信号作为外辐射源对目标进行定位跟踪具有重大的意义[1,2,3]。现有的无源雷达目标定位跟踪算法主要有测向定位法、TDOA时差定位法、差分多普勒定位法和基于滤波的定位跟踪算法, 其中基于粒子滤波算法和各种变形的卡尔曼滤波技术已经应用广泛[4,5,6], 文献[7] 提出了一种改进的动态规划检测前跟踪算法, 通过指数平滑估计的测量和预测状态之间的角度偏差, 提出了一种新的状态转移概率模型, 提高微弱目标的检测和跟踪性能;文献[8]以单目标的递归检测前跟踪算法为基础, 提出了一种MM-MM-TBD算法, 有效的解决了在微弱目标的情况下时变多目标的跟踪和检测问题。文献[9]将动态规划与粒子滤波相结合, 解决复杂杂波环境下的微弱目标检测跟踪问题。此外, 利用多颗卫星的互模糊函数估计目标的时延差和多普勒频偏[10,11,12,13], 也可以实现对目标的定位和跟踪。

    然而, 由于卫星的辐射功率低, 采用互模糊函数估计目标的时延差和多普勒频偏时, 由于噪声的影响会导致虚假峰值点的存在, 从而增大了时延和多普勒的不准确性, 导致估计精度不高;另外, 基于粒子滤波的跟踪算法虽然估计精度较高, 但是该算法复杂度高, 且需要长时间的累积才能不断的修正误差提高精度。基于此, 本文提出了一种利用机器学习[14]方法对无源雷达系统中的微弱目标进行定位与跟踪处理算法, 建立时延和多普勒模型, 采用梯度下降算法对模型进行优化, 并通过仿真给出目标定位和跟踪的结果以及对应的位置和速度误差。

    2.1 线性回归

    本文利用线性回归算法学习时延和多普勒之间的关系, 算法的训练数据是由互模糊函数估计得到的时延、多普勒频偏值构成, 不同帧数的时延和多普勒频偏值经过预处理后作为该回归问题的训练集。预处理主要包括剔除离群点和数据标准化两大过程。假设需要剔除离群点的数据的集合为T, t (i) ∈T, 遍历T, 如果t (i) 满足|t (i) -μΤ|<σΤ, 那么t (i) 就是需要保留下来的数据, 将它归属到集合Tinline, 其中μT和σT分别是集合T中所有数据的平均值和标准差, 之后计算集合Tinline中所有数据的平均值和标准差, 记为μT_inline和σT_inline, 再次遍历T, 如果|t (i) -μΤ_inline|>σΤ_inline, 那么t (i) 就是需要剔除的离群点。数据标准化过程如式 (6) 所示:

    4 结 论

    本文研究了无源雷达系统中的目标定位算法, 该算法利用机器学习中的线性回归方法, 结合梯度下降算法可以实现对互模糊函数估计出的多普勒频偏进行修正, 修正之后的时延、多普勒数据可以用来实现对目标的定位跟踪, 与未修正的数据相比, 位置估计精度提高了11.1倍, 速度估计精度提高了3.6倍, 具有低复杂度、高效率和高准确度的特性。仿真结果显示, 本文所提出的算法能够定位跟踪4 km的目标, 实现被动雷达系统中的目标定位跟踪, 对定位跟踪空中目标有一定的意义, 同时, 采用线性回归得到的修正之后的时延和多普勒值, 可重新用于互模糊函数中提高峰值的大小, 进而提高检测率。
