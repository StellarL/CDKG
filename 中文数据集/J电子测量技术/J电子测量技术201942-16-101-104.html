    基于坐标函数弱约束的模糊度函数法

    赵芳 张骁 杜亚杰

    海军航空大学

    摘    要：

    为了提高GPS动态定位的精度和效率,在使用无人直升机装载GPS设备标校舰载三坐标雷达时,可事先依据雷达要标定的参数设计标校航迹。提出了一种基于坐标函数弱约束的模糊度函数法,根据已知的标校航迹,对模糊度函数法(AFM)解算整周模糊度时对坐标函数进行弱约束,完全消除了整周跳变的影响,从而提高了GPS的动态定位的效果。仿真结果表明,所提方法提高了模糊度函数法在动态定位中的精度,若轨道方程存有较大误差(通常dm级精度),采取该判别方法效果会更好。

    关键词：

    雷达标校; 航路设计; 坐标函数弱约束; 模糊度函数法;

    作者简介： 赵芳,博士,主要研究方向为武器系统运用与保障工程。; 张骁,助教,主要研究方向为武器系统运用与保障工程。; 杜亚杰,博士,主要研究方向为武器系统运用与保障工程。E-mail:850119362@qq.com;

    收稿日期：2019-03-07

    The AFM based on coordinate function weak constraint

    Zhao Fang Zhang Xiao Du Yajie

    Naval Aviation University

    Abstract：

    In order to enhance the precision and efficiency of GPS dynamic positioning, when we use the unmanned helicopter loading GPS devices to calibrate the implementation of shipborne 3 d radar, according to the radar calibration parameters, the school track can be designed in advance. The AFM based on coordinate function weak constraint is proposed, use the known school flight track and restrain the coordinate function when using AFM to calculate the whole week ambiguity, so the effect of the precision and efficiency of GPS dynamic positioning has been improved. The simulation experiments show that the precision of AFM in the dynamic positioning algorithm is improved. If the orbit equation has bigger error(the precision is usually dm level), the effect of this discriminant method will be better.

    Keyword：

    Calibrate; Air route design; Coordinate function weak constraint; AFM;

    舰载三坐标雷达能够提供目标距离、方位和仰角等参数,是对空预警探测装备之一,是一种最重要的舰载防空反导作战系统的信息源[1,2,3]。为维持三坐标雷达精确的工作状态,需要对雷达测量精度定期进行标定。文献[4,5]分析了传统标校方法的优缺点,提出了一种使用无人直升机平台加载GPS设备对雷达进行动态标校的方法,如何获取精准的目标GPS实时定位和规划无人直升机的标定航迹,是本文研究的重点。

    准确确定整周模糊度是高精度GPS动态定位的重点[6],模糊度函数法(AFM)由Couselman等[7]首先提出,是解算GPS整周模糊度的一种方法。之后,Remondi[8]在GPS伪动态定位和静态定位中使用了AFM。Mader[9,10]把AFM法用于快速动态和静态GPS定位。

    AFM将整周跳变的影响完全消除,只使用了载波相位观测值中的小数部分[11],该特性适用于雷达校飞(高动态性)的动态定位。但此方法的缺点在于搜索耗时太长,主要原因是大量的“正确解”存在候选网点中,通常需要分离正确的网格点,在时间上的累加进而造成模糊度函数法在动态定位中未得到广泛应用[12,13]。

    1 基本模型

    1.1 AFM

    AFM的定义为[14]:

    A(x,y,z)=∑i=1n∑j=1ni∑l=1nfcos[2π(∇Δφ0ijl-∇Δφcijl(x,y,z))]         (1)

    式中:∇Δφcijl(x,y,z)是依据移动站坐标(x,y,z)得出来的双差理论值;∇Δφ0ijl是移动站双差观测值;nf=1,2是载波频率L1于L2;n为历元数;ni是历元i上双差观测量个数。

    利用AFM对整周模糊度求解的基本步骤为:

    1)首先采用码伪距测量值求得待测点初始三维的坐标,以待测点初始坐标为中心,依据其精度来确定搜索的边长,构造出三维模糊度函数的搜索空间。

    2)搜索空间确定之后,针对每个网格点分别按照AFM公式计算其模糊度函数,判断此点是不是待测点的真实值坐标。

    经过搜索之后,如果只剩下满足条件的唯一网格点,那么该点就是待测点的真实坐标,依据待测点的位置可以反推出整周模糊度。若剩余很多个网格点,那么需要整合更多卫星或历元的数据进行搜索,直到剩余一个网格点停止。

    1.2 坐标函数约束

    坐标函数约束的一般表达式为[6]:

    fx(X)=fx(x,y,z)=0 (2)

    受无人直升机续航距离和雷达精度标定指标要求的限制,无人直升机的航路半径一般不会超过20 km,因此在进行坐标函数约束时一般采用站坐标。空间直角坐标与站坐标的转换公式为:

    [xpyphp]=Τ⋅[x-x0y-y0z-z0]=[-sinB0cosL0-sinB0sinL0cosB0-sinL0cosL00cosB0cosL0cosB0sinL0sinB0]⋅[x-x0y-y0z-z0]         (3)

    式中:(x0,y0,z0)是站坐标原点处的三维坐标;(L0,B0)是站坐标原点处的大地经纬度;(x,y,z)是测站的三维坐标;站坐标系(xp,yp,hp)将测站即无人直升机的平面高程和坐标分开。雷达标定中,为了使雷达获取数据与GPS动态相对定位数据的统一,在建立站坐标时,取舰载雷达天线的中心为原点。

    2 基于舰载雷达距离标定的约束函数

    标定雷达距离精度时,通常假设无人直升机沿着雷达上方位置hp高度的水平线由距离雷达D位置向雷达方向飞行,这种校飞航迹是标校中常见的一种航迹。hp和D取不同的值进行多次飞行,标定雷达的距离误差,还可以进一步分析雷达距离精度受目标俯仰角的影响规律。校飞航迹如图1所示。

    图1 标校距离精度的无人机航路   下载原图

    对一般空间的直线约束而言,需要提前获取两端点三维的坐标[6],用于建立包含两个线性条件的方程,方程的应用与表示比较复杂,在实际操作过程中必须提前定位,导致使用不便,在本文的实验环境里,由于校飞航迹比较特殊,考虑采用高程不变类型的空间直线约束,表示为:hp=c。

    若无人直升机可以限制在向雷达直飞时方位角不变,即航迹捷径最短,坐标函数约束表达式为:θ=c。

    3 弱约束下AFM判别算法

    利用坐标函数约束AFM整周模糊度解算,主要有采用两种思路:1)对航路与函数不进行强制符合,这时坐标函数只是用于加速模糊度搜索进程和验证搜索结果,称之为判别法,也可称为弱约束;2)作为已知量将约束方程代进模糊度函数中去,当遍历搜索空间坐标点的时候一起计算,称之为代入法,也可称为强约束。在雷达标校时对目标动态定位时,因为接收机本身和外界干扰等各种因素,坐标约束函数几乎都有误差,这样代入法会把误差加入到计算结果里,甚至会由于公式的复杂性导致耦合误差。而弱约束对约束方程线性没有太多要求,且通过约束条件可以减少搜索点,搜索时间得到大量节约,因此本文考虑采用弱约束。

    坐标函数下的AFM判别算法流程如图2所示:

    图2 坐标函数下的模糊度函数法判别算法流程   下载原图

    暂且不考虑坐标的函数约束,依据伪距差分方式或者其他方式来确定待测点的起始坐标,并把起始位置作为中心,搜索的边长依据其精度来确定,构造三维模糊度函数的搜索空间,如图3所示,之后对空间里各点计算模糊度函数法公式值来进行遍历。

    图3 搜索空间的网格部分   下载原图

    比如首先以载波波长的0.1倍为边长,即L=0.1λ,将搜索空间切分,变为一些粗网格,然后对每个粗网格点的模糊度函数值分别计算。通常采用标准化的模糊度函数以提高计算效率:

    NA(X,Y,Z)=A(X,Y,Z)/(K,J,L) (7)

    式中的值域是[0,1]。将满足NA(X,Y,Z)>T(T一般为0.85～0.95)的所有网格点搜索出来之后,检查坐标是否满足坐标函数的约束条件,此时由于是弱约束,是允许少量观测误差存在的,不要求坐标函数的强制符合,所以通常设误差限定值为γ,即:

    f(X,Y,Z)<γ (8)

    如果误差在这个限定范围内,那么认为这个坐标解可靠,通过这一步检测,网格点数大大减少,如果此时只剩唯一一个满足条件的网格点,那么认为此点为真实坐标,由此就可以反推整周模糊度。如果还剩下很多网格点,那么需要增加更多卫星或历元的数据来进行搜索,且需要进行坐标函数约束的检查,直到只剩唯一一个网格点为止。

    4 实验仿真分析

    实验采用的GPS接收机是Trimble R6型,时间为2018年8月12日下午,参考站置于海军航空大学的综合实验楼前广场上,流动站放在综合实验楼楼顶,它的表面粗糙,可以不考虑多径效应影响,实验楼周围视野开阔没有遮挡,卫星可见度较高。流动站在楼顶边缘的移动可近似为无人直升机相对于舰载雷达的移动。采样间隔都设为1 s,把基准站设为站坐标原点,使用WGS-84坐标系,流动站的运动范围为一条直线,在轨道上进行往复运动。

    在动态定位中,捕获的卫星数目如图4所示。从图4可以看出,大部分时间参考站和流动站观测到的卫星数都在7颗以上,其定位可靠性较高。

    图4 捕获卫星数量   下载原图

    由于模糊度函数法对周跳不敏感,所以初始化时刻可以不考虑周跳的影响。对于某历元,在开始动态定位之后,即使某颗卫星观测值出现了整周跳变,还可以通过剩余的其他卫星的观测值来解算基线向量的真值。在进行初始化时,发生周跳的单颗卫星的计算量通常很小,基本对实时的动态定位无影响。

    选取其中一段的直线轨道进行计来检测函数约束条件下动态定位解的性能,大概有185个历元。参考站和流动站之间的垂直距离约为45.07～45.15 m,轨道上不同测试点的高程hp近似一样,约束方程为高程不变:hp=45.110

    考虑楼体铺设和楼顶建筑等原因,将存在误差的轨道方程作为强约束代入模糊度函数法算法进行计算,其带来的误差累积效应是未知的。其次,本实验主要目的是验证弱约束模糊度函数法的效果,为了降低GPS信号的损失,在最大程度上降低由信号缺失造成的实验误差,采用沿楼体的边缘进行移动,未固定基线向量参数,因此需要依据坐标转换方程转换约束函数和线性化,而线性化也会带来误差,所以这里只考虑弱约束,不进行强制的航迹符合。

    将基准站作为坐标原点,在进行动态定位计算时只采用单频L1的数据,分别采用弱约束和无约束进行动态定位,高度已经进行了约束,因此,在绘图时只考虑平面的方向。动态定位之前,要先利用双频数据,得到较精确的轨道如图5(a)所示,动态定位结果如图5(b)～(d)所示。以上实验结果表明,在图5(b)中也可看出模糊度搜索在无约束情况下的错误率为15%。而在弱约束情形下,虽然轨迹平面是近似的,误差大概为4 cm,但是模糊度搜索在约束后正确率仍能达到95%,定位误差由误差数据可以看出基本在0.005 m的范围以内。使用了函数弱约束的模糊度函数法比无函数约束情形下的解算效率明显提高。当然,不同的约束条件和定位环境也会影响计算结果。如果使用双频定位,还会进一步提高定位精度和计算效率。实验计算结果表明,如果约束误差变大,模糊度解算的错误率会有所升高,但这只是在单频数据下的定位,当使用双频数据时,还能够适当地放宽约束方程的误差限。定位误差为:σx2+σy2+σz2,此处的定位误差包括了流动站和基准站之间的系统误差。

    5 结 论

    舰载三坐标雷达标校的重点是空中目标,即实时获取无人直升机的GPS动态位置,已有的方法忽略了动态定位算法和校飞航路之间的联系。而根据已知的校飞航迹对模糊度函数法算法进行坐标约束,可以大大提高GPS动态定位的精度和效率。进一步地,如果能精准控制无人直升机航迹,使得约束方程能够准确表示轨道方程(通常误差1 cm以内),那么可采用强约束法以获取更好的结果,还需要进一步具体讨论实现方法。若轨道方程存有较大误差(通常dm级精度),那么采取本文的判别方法效果会更好。如果可以提高获取坐标初值的精度,步那有可能提升模糊度解算的正确率,可在之后的研究中进一步讨论。
