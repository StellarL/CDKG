    摘    要：

    为了提高异构有向传感器网络异常流量检测能力,提出一种基于模糊聚类的异构有向传感器网络异常流量检测算法。对采集的异构有向传感器网络传输数据进行低维度的特征集构造,构建异构有向传感器网络异常流量分布的粗糙集模型,对网络的异常流量特征集采用自适应回归分析方法进行统计特征量提取,以少量的样本类别数据为测试集,采用模糊C均值聚类方法对异构有向传感器网络异常流量进行向量量化分析,对全部的异常样本进行抽样训练,提取异构网络异常流量的高阶谱特征量,将异构有向传感器网络异常流量的特征提取结果输入到BP神经网络分类器中进行数据分类,结合大数据融合聚类方法实现异构有向传感器网络异常流量检测。仿真结果表明,采用该方法进行异构有向传感器网络异常流量检测的准确性较高,抗异常流量干扰能力较好,提高了网络的流量安全监控能力。

    关键词：

    模糊聚类; 有向传感器网络; 异常流量; 检测算法;

    Research on Network Anomaly Detection Algorithm Based on Fuzzy Clustering

    Zhai Jianli Wang Yingli

    Huali College Guangdong University of Technology Guangzhou Huali Science and Technology Vocational College

    Abstract：

    In order to improve the detection ability of heterogeneous directed sensor networks, an anomaly detection algorithm based on fuzzy clustering for heterogeneous directed sensor networks is proposed. Based on the low dimensional feature set structure of the collected heterogeneous sensor network data, the rough set model of abnormal traffic distribution in heterogeneous directed sensor network is constructed. An adaptive regression analysis method is used to extract the statistical feature of the abnormal traffic feature set of the network, and a small amount of sample class data is used as the test set. The fuzzy C-means clustering method is used to analyze the abnormal traffic of heterogeneous directed sensor networks by vector quantization. All abnormal samples are sampled and trained to extract the higher-order spectral characteristic of abnormal traffic in heterogeneous networks. The feature extraction results of heterogeneous directed sensor networks are input into the BP neural network classifier for data classification, and big data fusion clustering method is used to detect the abnormal traffic in heterogeneous directed sensor networks. The simulation results show that the proposed method is accurate in detecting abnormal traffic in heterogeneous directed sensor networks and has a better ability to resist the interference of abnormal traffic, thus improving the traffic security monitoring ability of the network.

    Keyword：

    fuzzy clustering; directed sensor networks; abnormal traffic; detection algorithm;

    随着无线传感器网络通信技术的发展,采用异构有向传感器网络通信技术进行无线网络设计,提高数据的无线传输能力,在异构有向传感器网络进行数据传输过程中,对网络流量传输的稳定性提出了更高的要求[1]。采用异构有向传感器网络通信进行数据传输中受到传输信道的干扰以及码间特征扰动等因素的影响,导致网络输出出现异常,需要对网络的异常流量进行准确检测,提高网络安全传输能力[2]。研究网络异常流量检测算法对提高网络的安全性和稳定性方面具有重要意义。

    结合异构有向传感器网络异常流量的类别,采用模糊聚类方法进行流量异常检测,对异构有向传感器网络的异常流量检测是通过数据挖掘和特征提取实现,提取异构有向传感器网络异常流量的高阶谱特征量,采用抗干扰算法进行滤波检测[3],本文提出一种基于模糊聚类的异构有向传感器网络异常流量检测算法,采用模糊C均值聚类方法对异构有向传感器网络异常流量进行向量量化分析,将异构有向传感器网络异常流量的特征提取结果输入到BP(Back propagation)神经网络分类器中进行数据分类,结合大数据融合聚类方法实现异构有向传感器网络异常流量检测,最后进行仿真实验分析,展示了本文方法在提高网络异常流量检测能力方面的优越性能。

    1 异构有向传感器网络异常流量分布数据库及特征集构造

    1.1 异构有向传感器网络异常流量分布数据库构建

    为了实现异构有向传感器网络异常流量检测,利用模糊粗糙集聚类方法构建异构有向传感器网络分布大数据库模型,采用最近邻优先分布式信息挖掘方法进行异构有向传感器网络异常流量挖掘,采用自适应的关联规则调度方法进行异构有向传感器网络异常流量的特征检测和信息过滤,融合相关性检测方法构建传感器网络异常流量的分布式大数据库模型[4],将数据集进行向量化处理,在不确定数据频繁项集模式下计算异构有向传感器网络异常流量的频繁项集,采用期望频繁项(expected frequent term, EFI)与概率频繁项(probability frequent term, PFI)融合分析方法,得到异构有向传感器网络异常流量调度集函数为:

    Rdi(t+1)=min{Rs,max{0,Rdi(t)+β(nt-|Ni(t)|)}}(1)

    Ni(t)={j:‖xj(t)-xi(t)‖<Rdi;li(t)<lj(t)} (2)

    式中,xj(t)表示异常流量分布数据集D中的平均信息熵,描述了在第j个聚类中心的样本子集,lj(t)表示在异常流量检测过程中第t代学习的样本集,计算异构有向传感器网络异常流量在第j个聚类中心的输出标签属性。采用分裂信息检测方法分析异构有向传感器网络异常流量的统计特征量[5],初始化异构有向传感器网络异常流量检测数据分类的聚类中心F(xi,Aj(L)),i=1,2,…,m,j=1,2,…,k,提取异构有向传感器网络异常流量检测谱特征,得到异构有向传感器网络异常流量检测的统计特征量为:

    γi=1w∑l=0w-1[xi(k-l)-μi]3(1w∑l=0w-1[xi(k-l)-μi]2)3/2         (3)κi=1w∑l=0w-1[xi(k-l)-μi]4(1w∑l=0w-1[xi(k-l)-μi]2)2         (4)

    提取异构有向传感器网络异常流量高阶谱信息[δ1,δ2,…,δN],得到第k个异构有向传感器网络异常流量检测冲激响应函数δk变成了δik(t):

    δik(t)=G(V=k|Ui,Θ(t))         (5)

    对采集的原始异构有向传感器网络异常流量信息进行融合处理[6],进行波束集成处理,异常流量的传感信息融合模型为:

    xm(t)=∑i=1Ιsi(t)ejφmi+nm(t),-p+1≤m≤p         (6)

    对采集的异构有向传感器网络传输数据进行低维度的特征集构造,构建异构有向传感器网络异常流量分布的粗糙集模型,提高对流量的异常检测能力[7]。

    1.2 异常流量检测的传感信息融合

    对网络的异常流量特征集采用自适应回归分析方法进行统计特征量提取[8],异构有向传感器网络异常流量采集节点数目为N=n-(m-1)τ,计算异构有向传感器网络异常流量检测的三维图谱ri和功率谱密度ki,采用Langevin方程描述异构有向传感器网络异常流量检测模型为:

    dxdt=ax-bx2+s(t)+Γ(t)         (7)

    对流量序列进行离散化采样和传感信息融合跟踪识别,得到异常统计特征量模型为:

    AVGX=1m×n∑x=1n∑y=1m|GX(x,y)|         (9)

    式中,m,n分别是异构有向传感器网络异常流量的采样样本的类别数和采样节点,设D为不确定数据库,Ti为异构有向传感器网络异常流量的分类元素,采用频繁项集的挖掘方法得到异构有向传感器网络异常流量的频谱带宽为:

    其中,e^R2(k)表示异常流量检测的观测序列,zR2(k)为原始训练集中的信噪比,zΙ2(k)为模糊聚类的冲激响应函数,e^Ι2(k)为数据子集的输出误差。根据上述分析,采用关联规则挖掘方法,进行网络异常流量检测的传感信息融合处理[9,10,11]。

    2 网络异常流量检测算法优化

    2.1 异常流量特征提取

    在构建异构有向传感器网络异常流量分布的粗糙集模型的基础上,进行网络异常流量检测优化设计,本文提出一种基于模糊聚类的异构有向传感器网络异常流量检测算法。以少量的样本类别数据为测试集,采用模糊C均值聚类方法对异构有向传感器网络异常流量进行向量量化分析[12],在模糊C均值聚类中心中,异构有向传感器网络还书数据码元元素t的期望支持度esup(D)大于阈值θ,则称异构有向传感器网络异常流量检测的属性元素为一个频繁项,在满足约束条件下所有异常流量的分类属性元素满足:

    esupt(D)>θ (12)

    对异构有向传感器网络异常流量的聚类结果进行自适应调制[13],使用一个四元组结构来描述异构有向传感器网络异常流量的关联特征:FP(Xij,Pij,(supk1(D),…,supkf(D)),(Tk1,…Tkj)),其中,Xij为异构有向传感器网络异常流量在Tij时刻到达窗口的第j次出现的第i个数据元素,Pij为输出优化训练的最优概率,(supk1(D),…,supkf(D))为异常流量的低维度的特征集。采用相关性波束形成方法,进行网络异常流量检测,得到机器学习迭代式为:

    xΟi=xSi+Kdimax(xLi-xSi) (13)

    2.2 异构有向传感器网络异常流量检测

    采用灰阶量化特征分析方法,得到异构有向传感器网络异常流量检测的决策函数为:

    U(x)=-12ax2+14bx4         (17)

    上式中,系统参数取a=1,b=1,对采集的原始异构有向传感器网络异常入侵大数据信息进行融合处理,构建异构有向传感器网络异常流量检测的统计信息分析模型,结合信息融合方法进行异常流量检测的均衡控制[15],采自适应均衡方法,得到异构有向传感器网络异常流量检测的模糊聚类分析模型描述为:

    ωk=(vkek)=(xk-f(xk-1)yk-h(xk))         (18)

    采用自回归滑动平均模型(auto regressive and moving average model, ARMA)模型表示异构有向传感器网络异常流量的主成分特征量,描述为:

    xn=a0+∑i=1ΜARaixn-i+∑j=0ΜΜAbjηn-j         (19)

    提取异构网络异常流量的高阶谱特征量,将异构有向传感器网络异常流量的特征提取结果输入到BP神经网络分类器中进行数据分类,综上分析,实现对异常流量检测,得到检测实现步骤为:

    1.初始化异常流量检测的BP神经网络学习系数SWF=null,D=null,Pij=0,supki(ω)=0;

    2.对Xij,随机找到一个模糊聚类中心点,得到异常流量检测的所有聚簇的中心点

    3. 按照聚簇交叉计算异常网络流量检测的统计凭据概率Pij;

    4. 如果(当前窗口未满),采用模糊聚类方法进行异常特征重组

    5.更新当前窗口进行异构有向传感器网络异常流量的样本重组,计算异常类别的概率分布值supki(ω);

    6.计算异常流量的模糊样本集,结合累计概率分布方法得到统计特征量为Q=∑i=minsupnumt(D)supt(D);

    7. 如果 Q≥δ

    8. 将BP学习的异构有向传感器网络异常流量样本加入粗糙集集合D;

    9.进行统计特征量提取,将异常流量存入窗口集合SWF;

    10.返回异构有向传感器网络异常流量的频繁项集D

    根据上述步骤,实现了异构有向传感器网络异常流量检测算法的改进设计。

    4 结 论

    对网络的异常流量进行准确检测,提高网络安全传输能力,本文提出一种基于模糊聚类的异构有向传感器网络异常流量检测算法。融合相关性检测方法构建传感器网络异常流量的分布式大数据库模型,将数据集进行向量化处理,对流量序列进行离散化采样和传感信息融合跟踪识别。构建异构有向传感器网络异常流量检测的统计信息分析模型,结合信息融合方法进行异常流量检测的均衡控制,采自适应均衡控制和模糊聚类方法实现异常流量检测。研究得知,该方法能有效实现对异常流量检测,检测准确性较高,抗干扰性较好,未来将在检测的实时性等方面进行进一步的研究。
