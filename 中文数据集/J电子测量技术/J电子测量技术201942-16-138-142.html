    摘    要：

    为了对电网中的电力谐波进行检测,采用基于三相瞬时无功功率理论的ip-iq算法进行电力谐波的检测,并使用自适应滤波器作为算法中所需的低通滤波器来提取电流中的直流分量,同时将检测结果与使用巴特沃斯低通滤波器时的检测结果进行比较,最后根据检测原理建立仿真模型,进行仿真。仿真结果表明,与使用传统的巴特沃斯低通滤波器相比,检测的快速性得到了提高,同时通过比较基波电流THD指标可知检测的准确性也得到了提升。因此,所提方法能够成功检测出电力谐波,达到了预期目的。

    关键词：

    电力谐波; 检测; ip-iq算法; 自适应滤波器; 巴特沃斯滤波器;

    Power harmonic detection based on adaptive filter

    Cao Lei Zhang Liguang Jia Hongmin

    College of Electronic Information Engineering, Xi′an Technological University

    Abstract：

    In order to detect power harmonic in the power grid, the ip-iq algorithm based on three-phase instantaneous reactive power theory is used to detect power harmonics, and an adaptive filter is used as low-pass filter needed in the algorithm to extract the DC component in the current, the results obtained when using Butterworth low-pass filter are compared with the results obtained when using adaptive filter at the same time, finally, the simulation model is built according to the detection principle, and the simulation is performed. The results show that the rapidity of the detection is improved compared to using the traditional Butterworth low-pass filter and the accuracy of the detection is also improved by comparing fundamental current THD. So the method can detect power harmonic successfully and the intended purpose is achieved.

    Keyword：

    power harmonic; detection; ip-iq algorithm; adaptive filter; Butterworth filter;

    随着工业水平和生活水平的不断提高,越来越多的工业电气设备以及生活电器投入到使用中,电网提供的周期交流电源经过非线性设备就会产生非正弦电流,形成畸变电压,由此下去,电流也是非正弦波,这样就产生了与电网基波频率不同的电力谐波[1]。这些电力谐波影响着电网的电能质量,使得电能的使用效率变低。同时,谐波也可能扰乱通信设备的正常工作,使得通信变得不可靠。因此,对谐波的抑制具有现实的重要意义。所以,需要一种快速有效的电力谐波检测方法。

    当前,用来检测谐波的方法有很多,主要有模拟滤波器法、基于傅里叶变换法、基于瞬时无功功率法、基于小波变换法[2,3]以及基于神经网络检测法等[4]。模拟滤波器检测法在实时性上很好,但是检测精度及可靠性上均不高。基于傅里叶变换法与模拟滤波器一样,在检测精度和可靠性上不是很高。由于小波变换的理论和应用还在研究中,且小波变换在实际中实现复杂,运算速度慢,所以其目前还不适合应用到实际的谐波检测之中。基于神经网络检测法因为过多的依赖精度较高的训练样本,所以其在实际的工程中也不太容易实现。基于瞬时无功功率理论的检测方法相对于其它检测方法在实现程度上难度较小[5,6],同时在谐波检测的实时性以及准确性上也有较好的性能,因此本文将用此方法进行谐波的检测[7]。因为该理论需要低通滤波器来提取检测电流中的直流分量,而低通滤波器的阶数和截止频率对检测的结果存在影响[8],理论中经常使用巴特沃斯滤波器作为低通滤波器,但是其具有较长的延迟特性。所以,本文采用自适应滤波器取代巴特沃斯滤波器,并且通过MATLAB进行仿真,比较两种滤波器对检测结果的影响,以提高谐波的检测精度以及实时性。

    1 ip-iq的谐波检测法

    三相瞬时无功功率理论的提出为谐波的检测提供了一个新的思路,也为谐波的检测以及抑制提供了理论基础,后来又经过不断的发展逐渐成熟,提出了ip-iq算法,其与p-q算法相比能有效地避免电压畸变和三相不平衡带来的影响[9]。

    根据算法原理,首先需要进行三相到正交两相的坐标变换,设各相电压和电流的瞬时值分别为ea、eb、ec和ia、ib、ic,两相瞬时电压eα、eβ和两相瞬时电流iα、iβ,则有:

    [eαeβ]=C32[eaebec]         (1)[iαiβ]=C32[iaibic]         (2)

    其中,

    然后电压矢量eα、eβ和电流矢量iα、iβ分别合成电压矢量e和电流矢量i:

    e=eα+eβ=e∠φe (3)

    i=iα+iβ=i∠φi (4)

    合成电压矢量e和电流矢量i之后便可以求出瞬时有功电流和瞬时无功电流,即

    ip=icosφ,iq=isinφ (5)

    式中,φ=φe-φi。ip和iq图形如图1所示。

    图1 α-β坐标系中各矢量表示   下载原图

    同时,三相瞬时无功功率q和瞬时有功功率p也就可以求出,即

    p=eip,q=eiq (6)

    将φ=φe-φi代入式(5)展开得:

    ip=icosφecosφi+isinφesinφi (7)

    iq=isinφecosφi-icosφesinφi (8)

    即

    ip=iαcosφe+iβsinφe (9)

    iq=iαsinφe-iβcosφe (10)

    此外,又由于:

    eα=cosφeeα2+eβ2         (11)eβ=sinφeeα2+eβ2         (12)ea=usinωt         (13)eb=usin(ωt-120°)         (14)ec=usin(ωt+120°)         (15)

    将式(11)～(15)代入式(1)整理后可以得到:

    [cosφesinφe]=[sinωt-cosωt]         (16)

    式中:ω为三相系统中基波电压的角频率。

    将式(16)代入式(9)、(10)中得:

    ip=iαsinωt-iβcosωt (17)

    iq=-iαcosωt-iβsinωt (18)

    即:

    [ipiq]=C[iαiβ]         (19)

    其中:

    C=[sinωt-cosωt-cosωt-sinωt]         (20)

    根据式(1)、(2)和(20)得出,对于ip-iq法,可以先检测出各相电压及电流,将三相系统中的各相电流以及正余弦函数先进行坐标变换,然后就可以得到三相系统中对应的两相系统中的瞬时有功电流和瞬时无功电流,继而将它们经过低通滤波器进行滤波,这样就可以得到其中的直流分量,将得到的直流分量再进行与之前变换相反的反变换,就会得到滤波之后的直流分量在三相系统中的值,也就是基波电流iaf、ibf、icf,最终用最先检测出的三相系统中的各相电流减去得到的各相基波电流就可以检测出需要的谐波分量iah、ibh、ich[10]。整个算法清晰明了,其中不涉及到复杂的运算方式,只需要进行两个矩阵相乘以及它们的反变换,再而就是低通滤波器的实现,因此计算量不是很大,所以其在响应速度上相对其它检测方式较快,可以满足谐波检测的实时性要求。

    2 自适应滤波器原理

    由图2可知,谐波检测算法中需要使用低通滤波器,其作用是从有功电流中提取基波电流经过变换之后对应的直流分量,所以低通滤波器的性能对谐波检测的准确与否以及快速与否有着很大部分的影响。所以,有必要对所用的低通滤波器的特性进行分析,而算法中常使用巴特沃斯低通滤波器作为其滤波器的选择,但是巴特沃斯低通滤波器在检测的快速性与准确性之间存在矛盾,当增加滤波器阶数时,检测结果准确性得到提升,但是巴特沃斯低通滤波器会产生较大的延时,从而对谐波的检测有一个滞后。当减少滤波器阶数时,响应速度变快,但是准确性又会降低。而自适应滤波器可以准确的追踪被检测波形,可以根据所处的环境来调节权重系数,以达到滤波效果的最优化,所以,本文用自适应滤波器取代传统的巴特沃斯滤波器作为算法中的低通滤波器。

    图2 ip-iq法原理   下载原图

    自适应滤波器实际上就是调节某个系数来使得滤波器的输出最优地与期望进行匹配,其原理结构如图3所示。其中x(k)为滤波之前的信号,作为滤波器的信号输入端,d(k)为滤波器的期望信号,为滤波器的输出理想值,y(k)为滤波器的输出端,是信号经过滤波之后的输出,误差信号e(k)为参考输入与滤波器输出之间的差值,w(k)为滤波器的权系数,即所需调节的参数[11,12]。

    图3 自适应滤波器原理   下载原图

    由图3可知,

    y(k)=∑i=1Νx(k-i+1)ωi(k)         (21)e(k)=d(k)-y(k)         (22)

    其中N为横向滤波器的阶数。在谐波检测运用中,x(k)为图2中的ip序列,d(k)为滤波之后的期望信号序列,权重系数的初值为0。

    根据最小均方准则,滤波器的最佳权系数w*应该使滤波器的均方误差最小,即minE[e2(k)]。根据LMS递推算法可知:

    W(k+1)=W(k)-μ∇(k) (23)

    式中:W(k)=[ω1ω2⋯ωΝ]Τ,∇(k)为滤波器在k时刻的均方误差梯度;μ为步长,μ的大小影响滤波器的稳定性以及响应的速度,当μ的值变大时,滤波器响应变快但是可能会不收敛,当μ的值变小时,对滤波器的影响与前者相反。

    而∇(k)=-2e(k)X(k),其中:

    X(k)=[x(k)x(k-1)⋯x(k-i+1)]Τ         (24)

    从而可得权值迭代公式为:

    W(k+1)=W(k)+2μe(k)X(k) (25)

    对自适应滤波器的仿真,其中滤波器阶数为10,步长μ=0.01,仿真输入信号、干扰信号以及经过自适应滤波器滤除之后的信号如图4所示。

    由图4可以看出,自适应滤波器的输出信号中的幅值变化范围比输入信号小,趋近于标准信号,输入信号中的噪声信号明显被削弱,同时从自适应滤波器输出波形可以看出,滤波器的响应速度也比较快,所以自适应滤波器的滤波效果较好且响应的速度也满足要求,可以将其用在谐波检测算法中提高谐波的检测精度以及快速性。

    3 检测谐波电流的仿真模型

    在MATLAB仿真软件中的Simulink提供了方便的仿真环境,搭建相应的仿真模块[13],就可以得到谐波的检测模型,为了将整个模型变得清晰简洁,本文将检测算法封装成了一个子系统,这样有助于查看并且具有层次感,如图5所示。其中,交流电压源选择三相可编程电压源,峰值为380 V,频率为50 Hz。因为整流桥具有非线性特性,所以选择整流电路作为负载,以此来产生谐波,作为谐波源。其中直流侧的电阻为10 Ω,电感为10-3 H,交流侧的线路电感值为10-3 H。

    图5 谐波检测模型   下载原图

    由巴特沃斯滤波器特性可知,阶数和截止频率的选择影响滤波效果[14,15]。当阶数增加时,准确性会得到提高,但是计算量随之也会加大,致使滤波器的反应速度变慢,从而造成在检测初始阶段有较大误差。当阶数保持固定不变时,选择较高的截止频率,滤波器的响应速度变快,但是这与准确性相矛盾,除了基波对应的直流分量可以通过,谐波经过变换之后得到的分量也可以通过滤波器,这样就造成了检测结果的精度降低。通过不断地比较,发现当巴特沃斯低通滤波器的阶数为二阶,同时截止频率为30 Hz的时候,检测结果在实时性和准确性上都有较好的表现,在此状态下的谐波检测结果如图6所示。

    图6 N=2,fc=30 Hz Butterworth检测结果   下载原图

    由图6可以看出,基波电流在初始阶段有个较大的延迟,这样造成了谐波检测也有一个较大的延迟,影响了谐波检测的快速性以及准确性。

    基于最小均方算法的自适应滤波器计算量小,性能优越,而且较易实现,所以可以用它来替代传统的巴特沃斯低通滤波器作为ip-iq算法中的低通滤波器,以此来提高谐波检测的准确性[16]。和巴特沃斯滤波器一样,当自适应滤波器阶数增加时,其结构也会变得复杂,因而计算量也就会加大,从而响应速度也就会变慢。除此之外,步长μ的选取也会影响快速性,当步长较小时,算法需要迭代的次数就会增加,从而计算量增加,就会有相应的延迟,当步长较大时,算法迭代次数减少,但是精度下降。所以,要找合适的步长,经过比较检测结果,确定步长μ为0.007 6的时候比较合适。图7所示为自适应滤波器的步长为0.007 6,权重系数初值为0时的谐波检测结果。

    图7 自适应滤波器检测结果   下载原图

    由表1可知,用自适应滤波器进行基波电流的提取比使用巴特沃斯的效果要好,提取到的基波更加准确,从而使得谐波检测准确性得到提高,同时,自适应滤波器在响应速度上要比巴特沃斯快,因此,综合以上两种情况可以看出,自适应滤波器比巴特沃斯在谐波检测中具有更好的滤波效果和更好的适应能力,将其运用在谐波检测算法中可以获得更加准确的谐波值,提高电力谐波的检测性能。

    4 结 论

    本文在三相瞬时无功功率理论的基础上建立数学模型,考虑到低通滤波器对检测结果的影响,进而对自适应滤波器的原理进行了研究,并且通过仿真对其特性做进一步的分析,最后根据建立的数学模型建立谐波检测模型,并且对两种滤波器的检测结果进行了对比,通过比较两种滤波器检测结果中的基波电流响应速度以及基波电流THD,得出以下结论:

    1)自适应滤波器比巴特沃斯滤波器具有更快的响应速度,使用自适应滤波器可以提高谐波检测的快速性,从而增强系统的实时性。

    2)通过对基波电流的FFT分析可知,自适应滤波器比巴特沃斯滤波器在谐波检测中具有更好的检测精度,使用自适应滤波器可以提高谐波检测的准确性。

    综上所述,自适应滤波器比巴特沃斯低通滤波器更适合运用到ip-iq算法中,其能够快速准确地检测出谐波电流,从而为谐波的治理提供准确的谐波值。
