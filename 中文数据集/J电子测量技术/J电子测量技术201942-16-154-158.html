    基于深度学习的纠错编码方式识别

    刘洁 刘凯

    上海大学通信与信息工程学院 上海大学特种光纤与光接入网重点实验室

    摘    要：

    引入卷积神经网络(CNN)用于解决信道编码方式识别问题,提出了一种基于改进LeNet-5网络的编码方式自动识别方法,该方法将接收机端接收到的信号进行预处理,预处理之后的数据作为深度卷积神经网络(DCNN)的输入,根据训练好的模型对时域编码信号进行直接特征提取与识别分类。针对ldpc码、卷积码、汉明码三种典型的编码方式进行了仿真实验,信噪比大于8时,编码方式的识别准确率大于80%。不同码率的卷积码之间,信噪比大于8时,识别准确率可以达到95%以上。实验结果表明,基于DCNN的时域编码信号识别是可行的。

    关键词：

    信道编码识别; 卷积神经网络; 改进LeNet-5网络; 特征提取;

    Recognition of error correction coding based on deep learning

    Liu Jie Liu Kai

    School of Communication and Information Engineering, Shanghai University Key Laboratory of Special Fiber and Optical Access Network, Shanghai University

    Abstract：

    This paper introduces convolutional neural network(CNN) to solve the problem of channel coding identification. An automatic recognition method based on improved Lenet-5 network is proposed. This method preprocesses the signal received by the receiver and preprocesses it. The subsequent data is used as the input of the deep convolutional neural network(DCNN), and the direct feature extraction and recognition classification are performed on the time domain coded signal according to the trained model. The simulation experiments are carried out for three typical coding methods: ldpc code, convolutional code and Hamming code. When the signal-to-noise ratio is greater than 8, the recognition accuracy of the coding mode is greater than 80%. When the signal-to-noise ratio is greater than 8 between convolutional codes of different code rates, the recognition accuracy can reach more than 95%. Experimental results show that time-domain coded signal recognition based on DCNN is feasible.

    Keyword：

    channel coding identification; convolutional neural network; improved LeNet-5 network; feature extraction;

    无论是工业生产还是科研领域,人工智能的成果都已经初具规模,成功应用于图像处理[1,2,3]、语音识别[4,5]、视频处理[6]、生物医学、通信信号处理[7,8]等领域。深度学习对于信号的特征提取为我们丰富纠错编码识别领域提供了新的思路。在信道编码盲识别领域,对各种编码方式的研究均有涉及,大多集中在编码参数的盲识别上,对于编码方式的识别成果极少。文献[9]提出了一种基于搜索小重量向量的编码类型识别算法,但是其算法复杂度高且对噪声敏感。由于通信传输环境的多变和复杂性,传输到接收端的信号往往存在某种差错,因此在发射端对信号进行纠错编码是通信系统必不可少的步骤。本文研究了基于卷积神经网络(convolutional neural network,CNN)的纠错编码类型识别,该方法避免了特征提取的复杂性,而且可以适应更为复杂的通信传输环境。

    1 编码信号

    无论是在采用自适应调制编码技术(adaptive modulationand coding,AMC)的智能移动通信中[10]还是在非合作通信中的通信侦察和信息截获领域,都需要盲识别技术为后续的参数识别和译码工作提供支持[11],其中对信道编码的识别至关重要[12]。信道编码盲识别技术是指在通信系统中,接收端在不知道信道编码类型的情况下,识别出接收序列所使用的纠错编码方式和编码参数的技术。

    1.1 纠错编码

    传统纠错编码有汉明码、BCH码、RS码和卷积码[13],现阶段应用范围比较广的有Turbo码,以及5G即将使用的LDPC码和Polar码,其中LDPC成为数据信道编码,Polar码成为控制信道编码。不同的编码方法,有不同的检错或纠错能力。设编码序列中信息码元数量为k,总码元数量为n,则码率为k/n,监督码元数为(n-k),(n-k)/k称为冗余度。LDPC码是基于稀疏校验矩阵H构造的线性分组码,其性能最接近香农极限,通常用Tanner图表示Ldpc码的校验矩阵,变量节点Vn=(n1,n2,…,nn)对应Hm*n的列,校验节点Vm=(m1,m2,…,mm)对应Hm*n的行。在设计Ldpc码时,应避免Tanner图包含长度为4和6的短环。图1所示为一个非规则Ldpc码的校验矩阵及其Tanner图。卷积码是一种非分组码,卷积码的编码输出中,n bit码组中监督码元不仅和当前输入的k bit信息位有关系,还和前面m=N-1个信息段有关系。通常将N成为编码约束度,nN称为编码约束长度,将卷积码记作(n,k,N),码率则仍然定义为r=k/n,其编码原理如图2所示。汉明码(Hamming code)也是一种线性分组码,它能够纠正一位错码且编码效率较高。汉明码因为其简单的编码原理广泛用于内存和磁盘纠错编码。一般要求为:

    2r-1≥n或2r≥k+r+1(1)

    1.2 复杂的通信环境

    信号序列s(t)在传输过程中会经历信道衰落、噪声干扰等各种因素的影响,其传输模型可以近似为:

    r(t)=c×s(t)+n(t)(2)

    式中:s(t)经过调制编码等预处理的传输信号;c是信号损耗;n(t)是加性高斯白噪声。

    2 深度卷积神经网络(DCNN)

    2.1 构造CNN

    CNN属于深度学习方式的一种,它是一种深度前馈神经网络[14,15]。与传统的深度神经网络(deep neural network,DNN)相比,它具有局部连接,权值共享的优点,更符合生物神经元部分响应特性,大大简化了模型训练复杂度。CNN具有很强的数据特征提取能力,不需要数据进行复杂的预处理流程,非常适合于时域编码信号的识别。CNN一般由输入层、卷积层、池化层、全连接层、输出层等组成。

    卷积层包括过滤器个数、卷积核尺寸、步长等超参数。卷积核的尺寸决定了卷积层感受野的大小,每个特征图使用同一个卷积核,在数据输入后,卷积核以特定的步长在数据上进行遍历操作,即做卷积运算,一般设置滑动步长小于卷积核尺寸,将遍历的乘积与偏置参数相加,之后将和送入激活函数,最后输出该层的特征图作为下一层的输入。一个神经元的卷积运算如式(3)和图3所示。

    Z=f(∑Wj×Xj+b)(3)

    式中:X是输入数据,W表示权重,b是偏置参数,f为激活函数,Z表示下一层特征图中的某一值。

    图3 遍历卷积运算   下载原图

    池化层又称为下采样层,即对输入的特征图进行采样操作,进行信息过滤,从而有效防止过拟合现象。常用的池化方法有Lp池化、随机池化和谱池化等。本文采用最大池化算法,即以特定的步长提取对应区域内的最大值构成新的特征图。

    全连接层的每个神经元都与上一层的全部神经元进行连接,将前面层中学习到的特征进行线性组合,送入输出层进行分类识别。

    为了在系统中引入非线性特征,增强网络复杂能力,在卷积层使用ReLU激活函数,同时还可以防止梯度消失现象。在全连接层之后引入DropOut 层,随机隐藏网络中的部分神经节点使其不工作,可以有效防止过拟合现象的发生。输出层使用Softmax多分类器输出分类标签。在进行模型训练时,针对某一样本,通过Loss损失函数计算真实值与预测值之间的差距,然后通过一定的步长使用梯度下降法不断前馈更新权重和偏差,最后理想的结果是损失值无限逼近于0,这里的步长称之为学习率。常用的Loss函数包括mean_squared_error、mean_absolute_error和mean_absolute_error等。categorical_crossentropy loss(交叉熵损失函数)属于多分类的对数损失函数,它与Softmax多分类器相对应,交叉熵用来评估当前的训练分布于真实分布之间的差异。

    C=-1n∑x[ylna+(1-y)ln(1-a)] (4)

    式中:y为模型期望输出;a为神经元实际的输出(a=δ(z),z=∑Wj×Xj+b),通过计算损失函数的梯度来不断地更新参数达到理想模型的目的。常用的梯度下降算法有Adam、SGD、Momentum、Agagrad、Adadelta等,就收敛性等整体效果来说,Adam自适应学习率优化算法的效果是最好的,也是实际训练中最常用的算法。

    2.2 改进LeNet-5网络

    LeNet-5网络是第一个比较经典的CNN网络,它在卷积层后面加入池化层的思想至今还在使用。整个网络包括2个卷积层、2个池化层和3个全连接层。经过改进的Lenet-5网络如图4所示,第1层为卷积层,滤波器(filter)的个数为20,卷积核的大小为5,遍历步长为1。第2层为最大池化层,窗口大小为2,步长为2。第3层是卷积层,filter个数为40,卷积核大小为5,遍历步长为1。第4层是最大池化层,窗口大小为2,步长为2。第5、6层是全连接层,神经元个数为512,第6层之后的最后一个全连接层作为输出层,采用Softmax多分类器输出分类标签。与最初的LeNet-5原模型相比,现在的Lenet-5网络也有了许多的改进,比如非线性激活函数采用Relu函数,它比传统的Sigmoid和Tanh激活函数具有更快的收敛速度,且可以防止梯度消失现象。为了防止过拟合现象,在全连接层后加入Dropout层,以P=0.5的概率随机隐藏部分神经元,降低网络复杂度,防止卷积网络对于数据中特征的过度学习,导致训练出的模型泛化能力差,测试集的准确率低。

    3 模型训练与特征提取

    本文使用Keras深度学习框架,它是一种纯Python编写的高级的神经网络应用程序编程接口(application programming interface,API),支持CNN和循环神经网络(recurrent neural network,RNN)以及中央处理器(central processing unit,CPU)与图形处理器(graphics processing unit,GPU)的无缝切换,具有简单高效的优点。Keras框架提供完整的库函数用于模型构建、训练和测试,其中网络层、损失函数、梯度优化器、激活函数、正则化方法等都是独立的模块,可以使用它们很方便的来构建自己的网络模型,更适用于前沿的研究工作。Loss选用Bcategorical_crossentropy交叉熵损失函数,梯度下降选用Adam自适应学习率优化算法,其初始默认学习率Lr=0.001。

    作为人工智能(artificial intelligence,AI)的重要分支,深度学习具有自动提取特征的优点,避免了传统复杂的特征提取算法[16]。本文构造了一个复杂度较小的网络,通过两个卷积层,提取出数据集深层次特征,然后将所得到的特征集变换到一维空间,之后送入全连接网络进行特征的线性组合,最后通过逻辑回归的方法实现编码类型的识别分类。

    4 结 果

    4.1 数据集的准备

    为了使用CNN进行模型训练,需要产生大量的样本数据。本文分别利用MATLAB产生50 000帧长度为2 016的Ldpc码,码率为3/4,码率分别为1/2,1/3,2/3,长度为600的卷积码各20 000帧,30 000帧长度为560的汉明码,码率为4/7。然后将编码后的数据进行二进制相移键控(binary phase shift keying,BPSK)调制,之后送入加性高斯白噪声(additive white gaussian noise,AWGN)信道进行传输,信噪比取2～20 dB,对接收端的信号进行相应解调,解调之后的数据就构成了本文所用的数据集。本文所用样本数据集中,训练集和测试集的比例分布为8∶2。

    4.2 仿真分析

    本文采用的是使用CNN方法直接对时域编码信号进行处理,不需要对数据进行任何变换,避免了繁杂的预处理流程。使用准备好的数据集对改进LeNet-5 CNN进行训练。

    如图5所示,可以看到随着迭代次数的增多,训练损失迅速下降,准确率迅速上升,经过40次迭代之后就可以实现训练的收敛。

    5 结 论

    本文将DCNN应用于纠错编码类型识别问题,给出了改进的LeNet-5神经网络模型以及模型训练方法,实现了时域信道编码信号的直接特征提取与分类识别,丰富了信道编码盲识别的思路。仿真结果表明,改进的LeNet-5网络非常适用于编码类型的识别,两层的卷积层可以有效提取编码序列的时域特征,从而实现纠错编码方法的分类识别。基于深度学习的编码识别方法有效的避免了手动提取数据特征的复杂操作,省去了大量的人工成本,同时在信道环境更为复杂的情况下,该方法仍可以对编码方法进行有效识别,比传统方法更具有普遍适用性。
