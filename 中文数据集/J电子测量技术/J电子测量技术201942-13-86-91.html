    随着互联网飞速发展及智能终端的普及, 网购逐渐成为一种重要的购物方式。在这样的背景下, 细粒度情感分析成为了当前热门研究领域之一, 而评价对象抽取是细粒度情感分析的重要步骤。提出了一种基于深度学习的电商评论文本评价对象抽取模型, 模型通过双向LSTM获取词语上下文的整体信息特征, 且通过卷积神经网络获取词语周围n-gram的局部特征信息, 将融合两种特征信息融合后输入至CRF模块进行判别序列建模。实验结果表明该模型的抽取效果相比其他抽取模型有一定的提升。

    Research on opinion target extraction model of e-commerce review based on deep learning

    Tang Kang Wang Haitao Jiang Ying

    Yunnan Key Laboratory of Computer Technology Applications, Kunming University of Science and Technology

    Abstract：

    With the rapid development of the Internet and the popularity of smart terminals, online shopping has gradually become an important shopping way. Under this background, fine-grained sentiment analysis has become one of the current hot research fields. An opinion target extraction model based on deep learning is proposed. The model obtains the overall information feature of the word context through Bi-LSTM, and obtains the regional feature information of the n-gram around the word through the convolutional neural network. The model combines two kinds of feature information and input it to the CRF module for discriminant sequence modeling. The experimental results show that the extraction effect of the model is improved compared with other models.

    随着互联网飞速发展及智能终端的普及, 网购逐渐成为一种重要的购物方式。因此, 以淘宝、京东商城、苏宁易购为代表的电商平台发展迅速, 平台上的用户评论性文本数量也呈指数级增长。消费者作为商品服务对象, 其对商品的评价对于商家改进商品质量具有非常大的指导意义。在这样的背景下, 细粒度情感分析成为了当前热门研究领域之一, 而评价对象抽取是细粒度情感分析的重要步骤, 因此, 评价对象方法研究有着重要意义, 只有准确提取评价对象后才能够进行针对商品评论有意义的情感分析。评价对象是指观点词所描述的实体或者实体属性[1], 如酒店评论文本中的“房间”、“床”、“卫生间”等, 评价对象抽取即用机器学习方法或者自然语言处理相关技术手段将评论文本中所涉及的评价对象识别的过程。评价对象可以由传统的机器学习方法抽取, 也可以由深度方法抽取。

    评价对象抽取通常被视为序列标注任务[2]。序列标注任务是指给每一个待标注序列元素, 标注一个集合中标记的过程。现有的评价对象抽取模型大多都是基于传统的机器学习方法和基于句法、规则的方法, 不仅工作量大, 而且适应性差。深度学习近年来广泛应用于自然语言处理相关任务。它是一种表示学习方法, 通过多层神经网络学习浅层网络无法获得的特征, 同时显著地提高了模型泛化能力[3]。Zhuang 等利用依存句法分析, 抽取出频率比较高的依存关系模板, 结合预定义的评价对象和评价此列表与依存关系模板对评论文本进行“特征-观点”对抽取。Poria等使用卷积神经网络 (convolutional neural network, CNN) 抽取评价对象。Huang等使用BiLSTM-CRF组合模型抽取评价对象。

    本文提出了一种双通道CNN-BiLSTM-CRF的评价对象抽取模型。实验结果表明, 该模型取得了不错的效果。

    1 相关工作

    1.1 文本表示

    文本是多个字符串构成, 而字符串是多个词语的有序集合, 人可以直接理解, 但是计算机不能直接理解, 所以在使用计算机对文本进行文本分类、情感分析等自然语言处理任务之前, 务必将文本表示为计算机能够“理解”的形式。

    1) one-hot 文本表示方法

    one-hot 文本表示是布尔模型, 也称独热模型[4]。One-hot文本表示方法是最早用来进行文本表示的方法, 也是最简单的文本表示方法, 在使用之前需先建立语料词表库。One-hot 文本表示的基本思想是将词表库中的每一个词语用相同维度的稀疏向量来表示, 维度大小等于词表库词语的总数, 该词语位置对应的向量元素为1, 其余皆为0。例如“门口”、“窗户”、“床垫”, “枕头”分别表示为:门口=[1, 0, 0, 0, 0, 0, 0, 0, …]窗户=[0, 1, 0, 0, 0, 0, 0, 0, …]床垫=[0, 0, 1, 0, 0, 0, 0, 0, …]枕头=[0, 0, 0, 1, 0, 0, 0, 0, …], 虽然这种文本表示方法通俗易懂, 实现简单, 但也存在着2个明显的缺陷[5]:

    (1) 词汇鸿沟:在one-hot编码中, 词语是孤立的, 词与词之间都是正交的, 无法体现出语义层面上的联系, 因此在文本分类、情感分析等自然语言处理任务相关模型无法捕捉到两个词联系的特征。

    (2) 维数灾难:在语料库较大的情况下, 通常词汇总数会过万, 使用one-hot方法进行文本表示将会导致词向量维度极高, 而且向量的维度会随着词库表的增大而增大, 这对于后续相关任务的训练是个巨大的挑战。

    2) 分布式表示

    为了克服独热模型的缺陷, Hinto等学者于1986年提出了分布式文本表示方法[6]。分布式表示的基本思想是通过对语言模型的学习, 将词汇映射成为相对低维度的实数向量, 使得词表中所有词语对应的词向量构成一个词向量空间, 其中每个词语则是该向量空间的一个点。如上一小节的几个词语“门口”、“窗户”、“床垫”, “枕头”则有可能表示为:门口=[0.151 25, 0.512 45, 0.522 11, 0.723 49, 0.823 12, …]窗户=[0.297 12, 0.359 12, 0.345 12, 0.211 24, 0.114 53, …]床垫=[0.641 24, 0.821 41, 0.172 41, 0.124 51, 0.125 12, …]枕头=[0.163 21, 0.641 24, 0.158 54, 0.126 03, 0.857 32, …]。

    如此一来, 就可以在词向量空间中的两点之间引入“距离”的概念, 并以此“距离”来衡量它们之间语义上的相似性。

    Word2vec是一组用于训练生成词向量的相关模型[7]。Word2vec以大量文本语料作为输入, 产生低维度向量空间。训练语料中每一个单词都对应着一个向量, 并且语料中有着共同上下文的单词在该向量空间中的距离相对近。

    Word2vec有两种训练词向量的结构, 它们分别是连续词袋 (CBOW) 或连续skip-gram。CBOW的核心思想是根据上下文来预测中心词。Skip-gram模型的核心思想与CBOW模型相反, 是以中心词来预测周围词[8]。

    1.2 循环神经网络

    循环神经网络 (recurrent neural network, RNN) 是一种常用于语音识别和自然语言处理 (NLP) 的人工神经网络, 最早于上世纪八十年代被提出[9]。RNN旨在识别数据的顺序特征, 并使用模式来预测下一个可能的情况。与前馈神经网络一样, RNN可以处理从初始输入到最终输出的数据。与前馈神经网络不同, RNN在整个计算过程中使用反馈循环, 例如反向传播时间或BPTT, 以将信息循环回网络。这将输入连接在一起, 使RNN能够处理顺序和时间数据。RNN的基本神经单元如图1所示:

    图1 循环神经网络   下载原图

    图1显示了循环神经网络的多个神经单元, 每个单元之间是有连接的, 具体表现在隐藏层的输入包括了输入层的输入以及上一时刻的隐藏层的输出。设输入为长度为L的文本序列, Xt表示t时刻隐藏层节点的输入。则模型在t时刻的涉及到的计算公式如下所示:

    at+b+Wht-1+Uxt (1)

    ht=tanh (at) (2)

    ot=c+Vht (3)

    yt=soft max (ot) (4)

    式中:at表示t时刻隐藏层的值;ht表示隐藏层到下一时刻隐藏层的输出;ot表示t时刻隐藏层到输出层的输入;yt表示t时刻输出层的输出;U为输入层到隐藏层的权重;W为自身递归的权重;b为偏置;tanh为t时刻到下一时刻隐藏层的激活函数;soft max为输出层的激活函数。

    对RNN的训练的目的就是要得到U、V和W的最优值[10,11]。循环神经网络的参数训练是基于时间的反向传播算法[12,13] (backpropagation through time, BPTT) 进行, 其基本原理与反向传播算法一样, 该算法包括3个步骤:首先是前向计算每个神经元的输出值;然后通过反向计算每个神经元的误差值, 它是误差函数E对神经元j的加权输入的偏导数;最后计算每个权重的梯度, 并使用随机梯度下降算法 (stochastic gradient descent, SGD) 来更新权重。但是在计算过程中, 会出现2种情况, 一是梯度趋近于无穷, 容易导致梯度爆炸问题。另一种情况是梯度趋近于零, 容易导致梯度消失的问题。

    1.3 长短期记忆网络

    理论上, RNN能够在序列的整个历史中对连续信息建模, 但是在实践中, “消失梯度问题”限制了RNN的能力。为了克服RNN的不足, 有学者于上世纪九十年代提出了长短期记忆网络 (long-short term memory network, LSTM) , 经后人改进后真正地被广泛使用, 它是目前使用最多的RNN变体之一[14,15]。LSTM的神经元通常有由存储器单元, 输入门, 输出门和忘记门组成。其结构如图2所示:

    图2 长短期记忆网络   下载原图

    设在t时刻, 词向量输入为Xt, 单元状态为ct, 隐藏层输出为ht, 在上一个时刻t-1, 单元状态为ct-1, 隐藏层输出为h。

    “遗忘门”控制到该时刻中单元状态应该保留多少上一时刻的单元状态, 其公式如下:

    ft=σ (Wf·[ht-1, xt]+bf]) (5)

    式中:Wf为遗忘门的权重矩阵;bf为遗忘门的偏置;[ht-1, xt]表示两个向量的拼接;σ为激活函数。

    “输入门”控制该时刻的输入对该时刻单元状态的影响, 其公式如下:

    it=σ (Wi·[ht-1, xt]+bi) (6)

    式中:Wi为输入门的权重矩阵;bi为输入门的偏置。当前输入的单元状态是根据上一时刻的输出和t时刻的输入来计算, 其公式如下:

    c′t=tanh (Wc[ht-1, xt]+bc) (7)

    式中:Wc为该过程的权重矩阵;bc为该过程的偏置。

    t时刻的单元状态由t-1时刻的单元状态与上述3个门的共同决定, 其公式如下:

    ct=ft·ct-1+it·c′t (8)

    式中:⊙表示按元素乘操作。

    “输出门”决定长期记忆对当前输出的影响, 其公式如下:

    ot=σ (Wo·[ht-1, xt]+bo) (9)

    LSTM神经单元的输出由输出门和单元状态共同确定:

    ht=ot·tanh (ct) (10)

    2 本文提出的评价对象抽取模型

    本文提出了一种基于双通道CNN-BiLSTM-CRF评价对象抽取模型, 其框架图如图3所示:

    图3 模型整体框架   下载原图

    2.1 模型的结构

    模型主要由5个模块组成, 分别是双通道嵌入模块、双向长短期记忆网络 (Bi-LSTM) 模块、卷积神经网络 (CNN) 模块、条件随机场 (CRF) 模块和BIO模块。

    2.2 双通道嵌入模块

    双通道嵌入模块主要作用是将单词序列转化成后续模块可接受的输入形式, 即词向量表示, 并从上下2个通道分别输入到Bi-LSTM模块和CNN模块。

    图4 双通道模块   下载原图

    如图4, 评论文本分词后形长度为l的单词序列, 将其输入至双通道嵌入模块后得到两个大小为l×n的词向量矩阵, 其中n为word2vec训练出的词向量维度。

    2.3 Bi-LSTM模块

    Bi-LSTM模块旨在获取文本单词序列的全局特征信息。

    图5 双向LSTM   下载原图

    如图, 双向长短期记忆网络是长短期记忆网络的重要变体之一, 由前向LSTM与后向LSTM组成。前向LSTM捕获词语上文的特征信息, 后向LSTM捕获词语下文的特征信息。Bi-LSTM的隐藏层最终保存有2个值, 包括前向输出向量h⇀和后向输出向量h↼, 计算方式与LSTM单向模型类似, 具体计算过程为式 (5) ～ (10) , 2个方向的隐藏层向量为:

    h→t=ot⋅tanh (ct) (11)

    h←t=ot⋅tanh (ct) (12)

    Bi-LSTM模块t时刻的输出为2个方向隐藏层输出的融合向量, 融合方式有3种, 分别是向量相加、向量平均、向量相乘, 该模块取向量相加方式, 即

    ht=h→t+h←t (13)

    最后该模块的输出为:

    Obi-lstm= (h1, h2, h3, …, hl) (14)

    式中:l为嵌入层输入文本长度。

    2.4 CNN 模块

    CNN模块旨在获取文本单词序列的局部特征信息。该模块用卷积方式获取以每个词语为中心的n-gram特征作为其上下文信息。

    图6 CNN模块   下载原图

    如图6, 设输入为长度为l的词语序列Sentence= (X1, X2, X3, …, Xl) , Xi为第i个词语n维的分布式向量表示, h为卷积窗口高度, w为卷积窗口宽度, k为卷积核数量, s为卷积步长, p为填充数量, 因为每个词向量具有完整性, 所以使卷积窗口宽度与词向量维度一致, 即w=n, 则每个卷积层卷积后是一个b×1的特征图, 高度b为:

    b= (l-h+2p) s+1 (15)

    为了使得能够与Bi-LSTM模块的输出融合, 该模块输出特征图需具有与原始输入矩阵相同的宽度, 即输出是大小为l×k的特征图, 也就是要使每个卷积核卷积结果是l×1的矩阵。在获取3-gram特征信息情况下, 获取的上下文宽度为2, 即获取每个词前后序列各2个词的信息, 取s=1, p=2, h=5由式 (15) 得b=l。最后得到该模块的输出:

    Οcnn= (h→1;h→2;h→3;…;h→k) (16)

    hi为第i个卷积核卷积后l×1的特征图。

    2.5 融合特征模块

    融合特征模块的作用是将Bi-LSTM模块的输出特征矩阵和CNN模块的输出特征矩阵融合。融合过程如图7:

    如图7, Bi-LSTM模块的输出是一个大小为l×n的特征矩阵, CNN模块中每个卷积核的输出是一个l×1的特征图, 将k个卷积核的输出组合成一个大小为l×k的特征矩阵。两个特征矩阵拼接成l× (n+k) 的特征矩阵, 这样就可以把两个模块获取到的特征信息融合在一起。

    2.6 CRF模块

    条件随机场 (CRF) 是一个判别序列模型, 作用是考虑序列标签之间的关系。上一模块的输出向量序列, 即给定观察序列X={x1, x2, x3, … xl}, 正确预测的标签序列y={y1, y2, y3, …, yl}, Y (X) 表示输入序列所有可能形成的序列。设每个标注序列的分数score (X, y) , 其计算公式如下:

    score (X, y) =∑i=0nAyi, yi+1+∑i=1npi, yi (17)

    通过指数函数与归一化操作将分数转化为概率分布:

    p (y|X) =escore (X, y) ∑y′∈Y (X) escore (X, y′) (18)

    使用动态规划算法计算出概率最高的标签序列:

    ymax=argmax{p (y|X|, y∈Y (X) } (19)

    2.7 BIO模块

    评价对象抽取是序列标注问题, 该模型序列标注使用的是BIO方法。本文的BIO方法是使用3个标签来标注序列, 分别是:“B”、“I”和“O”。“B”表示评价对象的开始, “I”表示评价对象的中间或末尾, “O”表示不是评价对象。评价对象只包含1个单独的词语时, 用“I”标注。例如评论文本分词序列“送货-速度-快-服务-也-好”的序列标注为“B-I-O-I-O-O”。

    3 实验与分析

    3.1 实验环境与实验数据

    实验机器选用MacBook Pro (13-inch, 2017, Two Thunderbolt 3 ports) , 相关配置:系统macOS High Sierra, 版本10.13.3, 处理器2.3 GHz, Intel Core i5, 内存8 GB, 2 133 MHz, LPDDR3。

    本文使用的实验数据是通过爬取京东、淘宝等商品评论文本, 经过清洗、过滤和人工标注的10 000条数据。其中6 000条作为训练数据, 2 000条作为验证数据, 2 000条作为测试数据。

    3.2 实验设计与评价标准

    为了验证本文提出抽取模型的效果, 实验通过使用LSTM-CRF模型和Bi-LSTM-CRF模型进行对照实验。

    F1 指标是一种衡量分类总体效果的常用评估方法, 其公式为:

    F1= (2*p*r) (p+r) (20)

    式中:p和r为正确率和召回率。

    共进行了3组实验, 分别为LSTM-CRF模型, Bi-LSTM-CRF模型和本文提出的双通道CNN-Bi-LSTM-CRF模型, 每组分别在词向量维度n设置100, 200, 300做对比实验。实验多组结果如下:

    4 结 论

    针对现有的评价对象抽取Bi-LSTM-CRF模型, 本文提出了一种双通道CNN-Bi-LSTM-CRF的电商评论评价对象抽取模型。通过双通道词嵌入双向长短期记忆网络和卷积神经网络两种网络提取文本特征信息并融合, 充分利用了Bi-LSTM与CNN两种网络获取特征信息的特点, 强化了模型对词语周围n-gram局部特征信息的提取, 最后利用CRF判别序列建模。实验结果表明, 强化模型对词语周围n-gram局部信息的提取, 可提高评价对象抽取的效果。
