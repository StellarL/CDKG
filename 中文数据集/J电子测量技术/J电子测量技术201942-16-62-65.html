    基于BP神经网络的配网系统光伏输出功率平滑控制分析

    郝宁 王新娜 宁博扬 樊浩 林琳 夏瑞

    摘    要：

    为充分降低配网系统光伏输出功率神经网络输入变量,基于BP神经网络的配网系统光伏输出功率平滑控制。选择BP神经网络来训练数据,建立基于光伏预测模型及储能配合的光伏功率平滑控制。以MATLAB构建了神经网络预测模型实例验证得到:在12:00～14:00时段内光伏输出功率明显偏大,引起配网节点电压越限的结果,以沃兹低通滤波器对光伏输出功率进行了平滑优化处理后光伏实际输出功率变化得更加平缓,在9:00～15:00时段内光伏输出功率明显偏大,最大时段有明显的增加。可以利用构建的模型完成光伏输出功率的精确预测。

    关键词：

    配网系统; 光伏输出功率; 平滑控制; BP神经网络;

    Analysis of pv output power smoothing control in distribution system based on BP neural network

    Hao Ning Wang Xinna Ning Boyang Fan Hao Lin Lin Xia Rui

    State Grid Jibei Electric Power Company Limited Skills Training Center Metering Center of State Grid Hunan Electric Power Company

    Abstract：

    In order to fully reduce the input variables of the photovoltaic output power neural network of the distribution network system, the photovoltaic output power smoothing control of the distribution network system based on BP neural network is proposed.BP neural network is selected to train the data, and a photovoltaic power smoothing control based on photovoltaic prediction model and energy storage coordination is established. With MATLAB neural network prediction model is constructed to verify and the results get: photovoltaic(pv) power output during 12:00～14:00 obviously slants big, which cause the limit of the distribution network node voltage as a result, in watts low-pass filter on the photovoltaic power output smooth optimization photovoltaic(pv) of the actual output power changes more gently, during 9:00～15:00 photovoltaic(pv) power output is significantly larger, there has been a marked increase maximum period of time.The constructed model can be used to accurately predict the photovoltaic output power.

    Keyword：

    distribution network system; photovoltaic output power; smooth control; BP neural network;

    0 引 言

    随着全球经济快速发展,对能源的需求量不断提高,同时传统化石能源储量持续降低,促进了全球能源减排过程的明显加快,这也因此促进了分布式能源的快速推广应用[1,2,3,4,5]。由于光伏发电具备可持续性、环保、建设周期短等优势,已经成为一种重要的新能源并获得了人们的深入研究。考虑到光伏发电过程受到气象与环境因素的共同影响,其运行过程具有明显的波动性与随机性,同时其自身结构也会对输出功率造成显著影响,无法在配电网内实现高效利用。当光伏发电功率过高或过低时将影响电网的正常运行,不利于实现大规模光伏系统并网运行的过程,由此可见,必须对光伏发电预测模型进行深入分析,实现光伏功率的精确、快速预测,同时选择合适的配网功率调控措施可以使电网运行达到更安全的状态并有利于灵活调度。其中,光伏功率预测时间尺度包括超短期、短期以及中长期共3类[6,7,8,9]。同时根据不同的建模方法也可以将其分成物理分析方法与数据统计方法共两类,其中物理分析方法是根据光伏电站运行参数以及实际地理信息得到的结果,数据统计方法是对历史数据进行统计与分析的处理方法。结合各类不同的光伏发电预测方法,可以将其分为利用预测太阳辐射强度实现间接预测与利用光伏功率数据来直接预测共两种情况[10]。对于光伏功率的预测,现阶段国内外大部分学者为光伏发电输出功率构建不同的预测模型并采用多种理论方法[11,12,13,14,15]。为充分降低神经网络输入变量,本文针对光伏功率的各项影响因素进行了深入探讨,考虑到将光伏发电系统接入配网后将会引起交流母线PCC点出现电压波动的情况,本文构建了一种BP神经网络预测模型,可以利用该模型完成光伏输出功率的精确预测。

    1 光伏发电直接并入配网

    先研究在配网中接入光伏发电系统后交流母线PCC点的改变情况。通过分析配网线路、变压器以及光伏电池各个部分的阻抗,可以得到图1的配网模型。

    图1 光伏接入配网等效模型   下载原图

    图1中,Xg代表滤波电抗,Qg是无功功率,Vs是线路末端电压,Pg代表注入光伏系统的有功功率,Vg是PCC点电压,ig是PCC点电流,此处不研究接入负载的情况。

    结合图1结果,并利用线路电压公式可获得固定配网线路阻抗的PCC点电压,

    Vg=Vs-[PgR1-QgX1]Vg (1)

    根据该式可以推导得到

    Vg=Vs/(1-PgR1-QgX1) (2)

    可以看出PCC点电压的变化与配网系统光伏输出功率有关。

    图2给出了光伏发电直接并入配网拓扑结构流程图,可以看出该拓扑结构由光伏电池单元组成的光伏发电系统、蓄电池与电导储能组成的混合储能系统组成。

    图2 光伏发电直接并入配网拓扑结构   下载原图

    在恒定的线路阻抗取值下,PCC的电压变化主要由注入该节点的无功功率决定,光伏输出负载投切都会影响并网PCC点电压波动。

    2 光伏功率平滑控制设计

    2.1 BP神经网络模型的设计

    本文设计了一种光伏预测配网优化方法。先对收集到的光伏发电系统历史发电功率以及气象统计数据实施信息融合,再利用处理得到的数据训练本文的网络模型,接着采用训练得到的网络模型对光伏功率进行预测。本文将预测间隔设定为每小时一次,再根据预测功率进行配网优化。

    考虑到BP神经网络存在明显的非线性映射、自适应特点与优异的学习能力,所以,选择BP神经网络来训练数据。此建模方法不同于之前根据天气情况进行划分的模型,本文不对阴天、下雨或晴天分类,该模型的具体结构原理见图2。根据图2可知,当输入节点数等于13时,BP神经网络预测日前1 d与预测日平均影响因素为λ(t)与λ(t+1),其中, 在预测日前1 d中以G(t)代表7:00～17:00各小时内光伏输出功率。输出节点数共11个,同时以G(t+1)来表示预测日内7:00～17:00各小时段中的光伏输出功率。形成较多数量隐含层时一方面会明显提高预测模型的复杂性,另一方面还会导致过度拟合,同时明显影响预测结果的现象,在实际应用过程中应隐含层数量设定在1～2层。本文对1层隐含层进行分析。从图2中可以看到,wij是实现隐含层和输入层相互连接的权值;wik代表输出层和隐含层的连接权值。当整个模型构建结束后,为防止网络参数发生局部小的情况,可以进一步提高训练的质量和训练速度,并进一步对比分析经过优化的输入层和隐含层权值wij及输出层权值wjk的结果。

    2.2 基于储能装置配合的光伏输出功率平滑控制

    根据滤波原理为光伏发电输出功率的平滑控制选择合适的控制方法,将该方法与分时电价结合后,再根据实际所需的负载条件,通过预测得到电价与功率都较高的时间段后开始对电网进行电能出售,对于预测出功率高与电网电价低的时段则将电能存储到蓄电池中,之后在高电价时段把储能向电网出售,到达输出功率以及电价都低的时候从电网进行大量购电,并通过储能装置完成部分能源的储存过程,到达高电价时将储能出售给电网。达到配网高效经济运行的状态,保证PCC点的电压稳定,有效防止节点电压越限情况。

    从图3中可以看到,通过一阶巴特沃兹低通滤波器来完成间歇式电源功率平滑控制的结构框图,采用此方法时需将滤波器设定在一个固定的时间常数,或者根据储能装置状态完成实时调整的过程。利用低通滤波器处理具有波动性的光伏输出功率PDG后可以生成功率参考值PDG0,在储能装置功率为Pbat0=PDG0-PDG时,联络线功率增大至期望值PDG0,当Pbat0>0时,电池处于放电状态,Pbat0<0代表电池充电。

    3 实例验证

    根据以上理论分析结果,为了对策略有效性进行验证,以MATLAB构建了神经网络预测模型。同时和外部公司合作,从屋顶光伏发电系统采集功率数据,从当地气象局获得气象数据。

    当光伏发电系统的电池板处于固定角度状态下时,可以将单位面积输出功率Pg表示成如下式子:

    Pg=ηSI[1-0.005(T0+25)]

    式中:η代表光伏电池的光电转换效率;I是光照强度;S代表光伏电池面积;T0表示环境温度。对于固定的光伏结构,光电转换效率以及面积都等于一个常数,此时光伏输出功率取决于光照强度和温度条件。

    本研究直接采用文献[16]给出的光伏功率和各项气象参数,并从中选择跟光伏发电量相关的气象因素,结果见表1。

    表1 光伏功率与气象因素的相关性系数 导出到EXCEL


    参数    		TgpaHws

    值    		0.516-0.362-0.5480.295

    参数    		CtClIrG

    值    		-0.308-0.2120.9641

    按照上述方法,确定模型输入,分别为预测日前1 d综合影响因子与发电功率以及预测日综合影响因子,其中,模型输出对应的是预测日中的预测光伏输出功率。比较了各类典型天气条件下的预测功率与实际功率数据,包括晴天、多云、雨天,结果见图4。可以发现,图4(a)是对晴天预测的结果,对图4(a)进行分析可以发现晴天条件下该模型达到了很高的预测精度。图4(b)显示了雨天下的预测结果,此时预测功率与实际功率误差较大,为减小两者间的误差可利用更精确气象数据来实现。

    图4 预测功率与实际功率对比曲线   下载原图

    在配网内应用该预测功率并结合储能技术,所选择的储能装置容量为30 kW。根据图4晴天预测功率可以发现,在12:00～14:00时段内光伏输出功率明显偏大,引起配网节点电压越限的结果,以沃兹低通滤波器对光伏输出功率进行了平滑处理,得到图5所示的优化结果。可见经过优化处理后光伏实际输出功率变化的更加平缓,在9:00～15:00时段内光伏输出功率明显偏大,最大时段有明显的增加。

    图5 光伏优化实际输出功率   下载原图

    4 结 论

    1)选择BP神经网络来训练数据,建立基于光伏预测模型及储能配合的光伏功率平滑控制。为防止网络参数发生局部小的情况,可以进一步提高训练的质量和训练速度,并进一步对比分析经过优化的输入层和隐含层权值wij及输出层权值wjk的结果。

    2)以MATLAB构建了神经网络预测模型实例验证得到:在12:00～14:00时段内光伏输出功率明显偏大,引起配网节点电压越限的结果,以沃兹低通滤波器对光伏输出功率进行了平滑优化处理后光伏实际输出功率变化的更加平缓,在9:00～15:00时段内光伏输出功率明显偏大,最大时段有明显的增加。本文构建的模型可以利用该模型完成光伏输出功率的精确预测。
