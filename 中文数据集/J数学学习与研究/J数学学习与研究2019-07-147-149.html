    浅谈小学、中学、大学教材中最大公因数的两种定义对研究的影响

    吴林隽

    广东省梅州市梅县区高级中学

    摘    要：

    很多高中生在读小学时都学过公因数、最大公因数 (或称公约数、最大公约数) 的定义, 学生一直到高中毕业时, 头脑中对最大公因数的概念还停留在“公因数中最大的一个”这个根深蒂固的印象上.这对高中生毕业后进入大学再次学习最大公因数的定义是不利的, 为此, 笔者举出一些现行的小学、中学、大学本科教材中对最大公因数的定义, 并尝试从不同的定义对一些比较重要的最大公因数的性质进行研究.

    关键词：

    公因数; 最大公因数; 整除; 定义; 性质;

    很多高中生在读小学时都学过公因数、最大公因数 (或称公约数、最大公约数) 的定义, 现行的北师大版小学五年级上册数学教材[1]第77页是这么定义公因数、最大公因数的:“12和18相同的因数是它们的公因数, 其中最大的一个是它们的最大公因数.”第31页还规定了小学阶段的研究范围:“我们只在自然数 (零除外) 范围内研究倍数和因数.”学生一直到高中毕业时, 头脑中对最大公因数的概念还停留在“公因数中最大的一个”这个根深蒂固的印象上.这对高中生毕业后进入大学再次学习最大公因数的定义是不利的, 为此, 笔者举出一些现行的大学本科教材中对最大公因数的定义.

    张禾瑞、郝鈵新主编的《高等代数 (第五版) 》[2]第20页对最大公因数的定义是:“设a, b是两个整数.满足下列条件的整数d叫作a与b的一个最大公因数: (ⅰ) d|a且d|b; (ⅱ) 如果c∈Z, 且c|a, c|b, 那么c|d.”“一般地, 设a1, a2, …, an是n个整数.满足下列条件的整数d叫作a1, a2, …, an的一个最大公因数: (ⅰ) d|ai, i=1, 2, …, n; (ⅱ) 如果c∈Z, 且c|ai, i=1, 2, …, n, 那么c|d.”第21页:“我们把n个整数a1, a2, …, an的那个非负的最大公因数记作 (a1, a2, …, an) .”

    闵嗣鹤、严士健主编的《初等数论 (第三版) 》[3]第4页对最大公因数的定义是:“整数a1, a2, …, an的公因数中最大的一个叫作最大公因数, 记作 (a1, a2, …, an) .”

    高中的一些数学竞赛教材, 比如, 李迅、陈德燕主编的《高中奥数专题讲座》[4]第117页对最大公因数的定义是:“若a, b不全为0, 则a, b的公约数中有一个最大数, 称为a, b的最大公约数, 记为 (a, b) , 类似地, 可以定义n个整数a1, a2, …, an的最大公约数, 记为 (a1, a2, …, an) .”

    由此可见, 不管是小学、中学还是大学的教材, 对公因数的定义基本是一致的, 对最大公因数的记法 (a1, a2, …, an) 也基本上是一致的.但是对最大公因数的定义, 可以分为两种类型的定义:1.公因数中最大的一个称为最大公因数[1][3][4];2.能够被所有公因数整除的公因数称为最大公因数[2].

    笔者整合了各种教材中常见的与最大公因数有关的一些性质, 结合上面两种类型的定义, 浅谈两种类型的定义对最大公因数的一些性质在研究方向上的异同.

    各种教材常见的与最大公因数有关的性质 (文字叙述不一定和教材上的原始文字相同) :

    性质1 设 (a1, a2, …, an) 是整数a1, a2, …, an的一个最大公因数.那么存在整数t1, t2, …, tn, 使得t1a1+t2a2+…+tnan= (a1, a2, …, an) .[2][3][4]

    性质2 若m是整数a1, a2, …, an的一个公因数, 则m| (a1, a2, …, an) .[3][4]

    性质3 若 (a, c) =1, c|ab, 则c|b.[3][4]

    注:性质3在证明算术基本定理的过程中有着非常重要的作用.而证明了算术基本定理后, 最大公因数的其他很多性质以及最小公倍数的诸多性质的证明都能够迎刃而解.

    一、以第1种类型的定义推导

    定义1 不全为0的n个整数a1, a2, …, an的公因数中最大的一个叫作最大公因数, 记作 (a1, a2, …, an) .[3]并且规定n个0的最大公因数是 ( 0, 0, ⋯, 0 n个0) =0.

    注:在定义1下, 最大公因数一定存在, 因为起码1是a1, a2, …, an的一个公因数.

    性质1 设 (a1, a2, …, an) 是整数a1, a2, …, an的一个最大公因数.那么存在整数t1, t2, …, tn, 使得t1a1+t2a2+…tnan= (a1, a2, …, an) .[2][3][4]

    证明 若a1, a2, …, an全为0, 那么 (a1, a2, …, an) =0, 此时取t1=t2=…=tn=0即可.

    若a1, a2, …, an不全为0, 构造集合I={t1a1+t2a2+…+tnan|t1, t2, …, tn∈Z}, 显然±a1, ±a2, …, ±an∈I.由于a1, a2, …, an不全为0, 所以I中含有正整数.记I+={s|s∈I, s>0}, 则I+是N+的一个非空子集, 由最小数原理可知, I+中含有最小的正整数, 记为d, 那么d具有形式:d=t1a1+t2a2+…+tnan, t1, t2, …, tn∈Z.

    下面证明d= (a1, a2, …, an) 即可.

    首先, 证明d是a1, a2, …, an的一个公因数.倘若不然, a1, a2, …, an中存在一个不能被d整除的数, 不妨设d|a1, 根据带余除法, 存在整数q, r, 使得a1=dq+r, 0<r<d.那么r=a1-dq=a1- (t1a1+t2a2+…+tnan) q= (1-t1q) a1-t2qa2-…-tnqan∈I+, 这与d是I+中最小数相矛盾.

    其次, 证明d是a1, a2, …, an的所有公因数中最大的一个.设c是a1, a2, …, an的任一个公因数, 则c|ai, i=1, 2, …, n, 那么c|t1a1+t2a2+…+tnan, 即c|d.所以c≤|c|≤|d|=d.

    证毕.

    性质2 若m是整数a1, a2, …, an的一个公因数, 则m| (a1, a2, …, an) .[3][4]

    证明 由性质1的证明过程可知此性质成立.证毕.

    性质1可以用来证明最大公因数的很多性质, 可是如果我们没有熟悉辗转相除法[3], 我们恐怕不容易想到构造一个集合然后用最小数原理的证明方法.我们的高中教材 (除了奥赛教材) 使得高中生对最大公因数的理解还停留在“公因数中最大的一个”这个根深蒂固的印象上, 把性质2 (所有公因数整除最大公因数) 当作是一种很自然的结论, 并没有严格进行证明.所以, 笔者在此跳过性质1, 给出一个对性质2, 在n=2的情况下的不太成熟的证明, 主要采取的方法是高中生学过的数学归纳法.

    当n=2时, 先改写性质2:若m是整数a, b的一个公因数, 则m| (a, b) .

    证明 如果a, b中含有0, 结论显然成立.如果a, b中含有负数, 当然可以转成正数来研究.下面假设a, b均大于0, 并用数学归纳法证明.

    当max (a, b) =1时, 经验证, 结论成立.

    假设当max (a, b) ≤k (k≥1, k∈Z) 时结论都成立.那么当max (a, b) =k+1时, 分4类讨论:

    Case1.若a≤k, b≤k, 由归纳假设, 结论成立.

    Case2.若a=b=k+1, 结论显然成立.

    Case3.若a=k+1, b≤k, 易证 (a, b) = (a-b, b) (由于 (a, b) |a且 (a, b) |b, 所以 (a, b) |a-b且 (a, b) |b, 那么 (a, b) ≤ (a-b, b) .由于 (a-b, b) |a-b且 (a-b, b) |b, 所以 (a-b, b) | (a-b) +b=a且 (a-b, b) |b, 那么 (a-b, b) ≤ (a, b) .) .又注意到a, b的公因数和a-b, b的公因数完全相同 (同上易证) 且1≤a-b≤k, 1≤b≤k, 由归纳假设可知, m| (a-b, b) = (a, b) , 结论成立.

    Case4.若a≤k, b=k+1, 同Case3的证明.

    那么当max (a, b) =k+1时结论也成立.

    综上所述, 性质2在n=2的情况下的结论成立, 证毕.

    后续:对n≥3时的证明, 我们可以使用 (a1, a2, a3, …, an) = (a1-a2, a2, a3, …, an) 以及a1, a2, a3, …, an和a1-a2, a2, a3, …, an的公因数完全相同 (这两个结论均易证, 且无需用性质1来证明) , 仿照n=2的证明过程来证明性质2.

    性质3 若 (a, c) =1, c|ab, 则c|b.[3][4]

    此结论也是高中生觉得非常自然 (虽然没有性质2那么自然) 的一个结论, 如果采用性质1, 它的证明过程将变得非常简单.

    证明 由性质1, 存在整数s, t使得sa+ct=1, 则sab+ctb=b.因为c|ab, 所以c|sab, 而c|ctb是显然的, 所以c|sab+ctb=b.证毕.

    下面同性质2的研究一样, 我们还是不用性质1得到的结论, 看看我们能否证明性质3, 这里需要用一下最小数原理.

    证明 若a, c中含有0, 由于 (a, c) =1, 根据定义1, 可以得到a=0, c=±1或a=±1, c=0, 经检验, 结论都成立.

    若a, c均不为0, 不妨设a, c>0, 不影响证明.下面固定a, c.

    构造集合B={b|c|ab, b∈Z}, 显然B≠∅且B中含有正整数 (至少正整数c∈B) , 设B+={b|b>0, b∈B}, 根据最小数原理, B+中含有最小数, 设其为b0.

    下证b0|b.

    假设b0|b, 根据带余除法, 存在整数q1, b1, 使得b=q1b0+b1 (0<b1<b0) .则c|ab=a (q1b0+b1) =aq1b0+ab1, 由于上面已经证明了c|b0, 所以c|ab1, 那么b1∈B+.这与b0是B+中最小数相矛盾.

    然后我们证明b0=c即可.

    倘若b0≠c, 注意到c∈B+且b0是B+中的最小数, 那么0<b0<c.由上面的证明可知, b0|c, 不妨设c=tb0 (t∈Z) , 则t>1.注意到c|ab0, 所以tb0|ab0, 那么t|a.又注意到t|c, 那么a, c存在一个大于1的公因数t, 这与 (a, c) =1相矛盾.

    证毕.

    二、以第2种类型的定义推导.

    定义2 一般地, 设a1, a2, …, an是n个整数.满足下列条件的整数d叫作a1, a2, …, an的一个最大公因数: (ⅰ) d|ai, i=1, 2, …, n; (ⅱ) 如果c∈Z, 且c|ai, i=1, 2, …, n, 那么c|d.

    并且我们把n个整数a1, a2, …, an的那个非负的最大公因数记作 (a1, a2, …, an) .[2]

    注:在定义2下, 不必特意去规定n个0的最大公因数为 ( 0, 0, ⋯, 0 n个0) =0了.

    在定义2下, 最大公因数的存在性就不那么“明显”了.

    性质1 设 (a1, a2, …, an) 是整数a1, a2, …, an的一个最大公因数.那么存在整数t1, t2, …, tn, 使得t1a1+t2a2+…tnan= (a1, a2, …, an) .[2][3][4]

    证明 若a1, a2, …, an全为0, 那么 (a1, a2, …, an) =0, 此时取t1=t2=…=tn=0即可.

    若a1, a2, …, an不全为0, 构造集合I={t1a1+t2a2+…+tnan|t1, t2, …, tn∈Z}, 显然±a1, ±a2, …, ±an∈I.由于a1, a2, …, an不全为0, 所以I中含有正整数.记I+={s|s∈I, s>0}, 则I+是N+的一个非空子集, 由最小数原理可知, I+中含有最小的正整数, 记为d, 那么d具有形式:d=t1a1+t2a2+…+tnan, t1, t2, …, tn∈Z.

    下面证明d= (a1, a2, …, an) 即可.

    首先, 证明d是a1, a2, …, an的一个公因数.倘若不然, a1, a2, …, an中存在一个不能被d整除的数, 不妨设d|a1, 根据带余除法, 存在整数q, r, 使得a1=dq+r, 0<r<d.那么r=a1-dq=a1- (t1a1+t2a2+…+tnan) q= (1-t1q) a1-t2qa2-…-tnqan∈I+, 这与d是I+中最小数相矛盾.

    其次, 证明a1, a2, …, an的所有公因数都能够整除d.设c是a1, a2, …, an的任一个公因数, 则c|ai, i=1, 2, …, n, 那么c|t1a1+t2a2+…+tnan, 即c|d.

    证毕.

    注:上面性质1的证明过程还说明了在定义2下最大公因数的存在性.

    性质2 若m是整数a1, a2, …, an的一个公因数, 则m| (a1, a2, …, an) .[3][4]

    证明 不用性质1, 直接由定义2可得.证毕.

    性质3 若 (a, c) =1, c|ab, 则c|b.[3][4]

    类似于 (一) 定义1的研究过程, 证明如下.

    证法1 由性质1, 存在整数s, t使得sa+ct=1, 则sab+ctb=b.因为c|ab, 所以c|sab, 而c|ctb是显然的, 所以c|sab+ctb=b.证毕.

    证法2 若a, c中含有0, 由于 (a, c) =1, 根据定义2, 可以得到a=0, c=±1或a=±1, c=0, 经检验, 结论都成立.

    若a, c均不为0, 不妨设a, c>0, 不影响证明.下面固定a, c.

    构造集合B={b|c|ab, b∈Z}, 显然B≠∅且B中含有正整数 (至少正整数c∈B) , 设B+={b|b>0, b∈B}, 根据最小数原理, B+中含有最小数, 设其为b0.

    下证b0|b.

    假设b0|b, 根据带余除法, 存在整数q1, b1, 使得b=q1b0+b1 (0<b1<b0) .则c|ab=a (q1b0+b1) =aq1b0+ab1, 由于上面已经证明了c|b0, 所以c|ab1, 那么b1∈B+.这与b0是B+中最小数相矛盾.

    然后我们证明b0=c即可.

    倘若b0≠c, 注意到c∈B+且b0是B+中的最小数, 那么0<b0<c.由上面的证明可知, b0|c, 不妨设c=tb0 (t∈Z) , 则t>1.注意到c|ab0, 所以tb0|ab0, 那么t|a.又注意到t|c, 那么a, c存在一个大于1的公因数t, 又 (a, c) =1, 由定义2得t|1, 这是不可能的.

    证毕.

    三、对定义1和定义2的研究过程小结

    我们由上面两种定义的研究过程可以看到:

    1.不管是哪种定义, 对性质2和性质3这两个“很自然的结论”的证明, 都可以跳过形如t1a1+t2a2+…tnan的构造方法, 但是形如t1a1+t2a2+…tnan的构造方法, 确实让我们的研究前进了一大步, 是一个很妙的方法和结论.

    2.定义1和定义2看上去都是很自然的定义, 但是定义1给我们的感觉 (特别是中小学生) 更像是一个定义, 而定义2给我们的感觉 (特别是中小学生) 更像是最大公因数的一个结论 (虽然它看上去很自然、很“显然”) .我们可以证明最大公因数的这两种定义是相互等价的, 大学教材如果在编写时能够说明这一点就更好了.

    3.最小公倍数的定义, 我们可以类似地在小学、中学、大学教材中找到两种不同的定义方法, 同样可以根据两种不同的定义方法, 进行不同的研究 (与此文中最大公因数的研究方法类似, 但是最小公倍数的不少性质可能用算数基本定理会更加容易证明和理解) .
