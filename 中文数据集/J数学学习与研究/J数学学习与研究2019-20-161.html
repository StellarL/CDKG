    以问题为导向,促进小学生深度学习的实现

    摘    要：

    深度学习是指在教师的组织和引导下,让学生经历从理念到实践的一整套思维方式和行为模式的转化.本文简要阐述如何实现深度学习,让学生主动参与到学习过程中来.

    关键词：

    问题; 导向; 深度学习;

    小学数学课堂,知识相对比较简单,受知识量的局限,师生教学程序相对比较单一,对教学内容的学习更多注重对知识本身的掌握和对解题能力的训练,往往对知识背后所隐藏的规律和本质、思想和方法的探讨相对较少.

    要真正实现深度学习,让学生主动参与到学习过程中来,以问题导向为主题和任务的教学设计是把学生引入深度学习的一种有效的教学手段.

    “三角形的面积”是小学数学图形与几何知识板块中的内容,是概念性知识和程序性知识相结合的知识类型,其主要的学习方法是在理解与探究、操作与体验的过程中完成对知识的学习.教师在教学设计中以问题为导向,通过四个问题把学生一步一步推向学习的主体,帮助学生深度参与、深度理解学习的全过程.

    问题设计一:如何求出这面流动红旗的面积?请选择合适的方法,尝试证明你的想法.

    学生已有求平行四边形面积的基础,为自主探究奠定了良好的基础,大部分学生能独立找到以下三种三角形面积的求法.

    方法一

    2个完全一样的三角形平行四边形

    底 = 底

    高 = 高

    2个三角形的面积=平行四边形的面积=底×高

    1个三角形的面积=底×高÷2

    方法二

    2个完全一样的三角形长方形

    底 = 长

    高 = 宽

    2个三角形的面积=长方形的面积=长×宽

    1个三角形的面积=底×高÷2

    方法三

    1个三角形长方形

    底÷2 = 长

    高 = 宽

    1个三角形的面积=长方形的面积=长×宽

    1个三角形的面积=(底÷2×高)=底×高÷2

    这些方法的发现都极具思考价值,因为是自己独立探究的结果,学生也因此获得一种极大的心理满足.表现为课堂气氛热烈、交流发言积极主动.学生们的数学表达在相互的启发中不断地完善.

    问题设计二:这三种方法有什么相同和不同?

    这个问题的抛出使得学生的思考向更深的数学思考挺进.在充分的交流、启发下,学生们发现:三个数量中面积可以变、底也可以变,那高也可以发生变化.在学生兴奋的目光中,方法四的得出就水到渠成了.

    方法四

    1个三角形平行四边形

    底 = 底

    高÷2 = 高

    1个三角形的面积=平行四边形的面积=底×高

    1个三角形的面积=底×高÷2

    由于关注的变量不同,路径就不同.
