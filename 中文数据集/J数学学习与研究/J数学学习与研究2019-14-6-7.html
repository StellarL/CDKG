    Filter在优化问题最优解中的应用

    郑雪莲

    摘    要：

    优化问题的求解通常需要满足求目标函数的极小值, 同时使迭代点列收敛到原问题的可行域内.大多数的线性搜索均采用精确罚函数方法, 这种方法的优点在于具有全局收敛性, 在最优点附近有较快的收敛速度.但是在应用罚函数时, 却有许多的困难.本文结合Filter的概念, 用Filter方法代替线性搜索, 决定步长, 使迭代点列收敛到优化问题的可行域内.

    关键词：

    Filter; 线性无关;

    本文考虑如 (P) 形式的约束优化问题:

    (Ρ) {minx∈Rn, f (x) , s.t.ci (x) ≤0, i∈Ι={1, 2, ⋯, m},

    其中f (x) 和ci (x) , i∈I都是二阶连续可微的实值函数.对任意的x∈Rn, 定义:I (x) ={i∈I|ci (x) =Ψ (x) }, 其中Ψ (x) =max{ci (x) , i∈I;0}.记c (x) = (c (x) , …, cm (x) ) T, 结合Filter的概念将问题 (P) 转化为如下形式:这里, h (c (x) ) =‖c+ (x) ‖1, 其中c+ (x) = (c+1 (x) , …, c+m (x) ) T, c+i (x) =max (0, ci (x) ) , i∈I.

    记F (k) ={j|j≤k, (h (c (xj) ) , f (xj) ) 为当前Filter中的元素}.若迭代点x被Filter接受, 则对所有的j∈F (k) , 满足

    h (c (x) ) ≤ (1-αη) h (c (xj) )

    或f (x) ≤f (xj) -γh (c (x) ) , (1)

    其中η∈ (0, 1) , γ∈ (0, 1) 为趋于0的参数.

    我们假设以下条件成立:对任意的x∈Rn, 向量组{∇ci (x) , i∈I (x) }是线性无关的, 产生的迭代序列{xk}包含在一个紧致凸集S中.

    引理1 在假设条件下, 则对任意的迭代指标k, 有h (c (xk+αdk) ) ≤ (1-α) h (c (xk) ) +α2mM‖dk‖2∞且|Δf-Δl|≤α2M‖dk‖2∞.

    引理2 在假设条件下, 对任意的迭代指标k, 有|h (c (xk+αdk) ) -h (c (xk) ) |≤αδ1‖dk‖∞且|f (xk+αdk) -f (xk) |≤αδ2‖dk‖∞, 其中δ1, δ2>0.

    证明 由h (c (x) ) 的定义, 有

    |h (c (xk+αdk) ) -h (c (xk) ) |

    ≤‖c (xk+αdk) -c (xk) ‖1

    =‖c (xk) +α∫01∇c (xk+tαdk) Tdkdt-c (xk) ‖1

    =‖α∫01∇c (xk+tαdk) Tdkdt‖1

    ≤∫01α‖∇c (xk+tαdk) Tdk‖1dt

    ≤∫01α‖∇c (xk+tαdk) T‖1‖dk‖1dt.

    由假设条件, 不妨设‖∇c (xk+tαdk) Tdk‖1≤M1, M1>0, 则|h (c (xk+αdk) ) -h (c (xk) ) |≤αM1‖dk‖1≤αδ1‖dk‖∞, 其中, δ1=nM1.由泰勒展开式,

    |f (xk+αdk) -f (xk) |

    =|f (xk) +α∇f (y) Tdk-f (xk) |=α|∇f (y) Tdk|

    ≤α‖∇f (y) ‖2‖dk‖2.

    y是介于xk与xk+αdk之间的一个点, 再由假设条件, 可设‖∇f (y) ‖2≤M2, M2>0, 则|f (xk+αdk) -f (xk) |≤αδ2‖dk‖∞, 其中δ2=nΜ2.

    定理1 若假设条件成立, 新的迭代点可以被F (k) 中的所有元素接受.

    证明 我们通过以下两种情况进行讨论:

    (1) h (c (xk) ) >0.

    由引理1, 存在α>0, 使得h (c (xk+αdk) ) ≤ (1-α) h (c (xk) ) +α2mM‖dk‖2∞成立.特别地, 当α≤ (1-η) h (c (xk) ) mΜ∥dk∥∞2时, 有h (c (xk+αdk) ) ≤ (1-αη) h (c (xk) ) 成立.

    令τk (+) =minj<kh (c (xj) ) , 则满足h (c (xk) ) <τk (+) , 因此, 有h (c (xk+αdk) ) < (1-αη) h (c (xk) ) ≤ (1-αη) τk (+) 成立.

    若令f (n) =minj<kf (xj) , 因为 (h (c (xk) ) , f (xk) ) 能被Filter接受, 有f (xk) ≤f (n) -γh (c (xk) ) 成立, 又由引理1和引理2, 有

    f (xk+αdk) -f (n) +γh (c (xk+αdk) )

    =f (xk+αdk) -f (xk) +f (xk) -f (n) +γh (c (xk+αdk) )

    ≤αδ2‖dk‖∞+f (xk) -f (n) +γ[ (1-α) h (c (xk) ) +α2mM‖dk‖]2∞

    ≤f (xk) -f (n) +γh (c (xk) ) +α2γmM‖dk‖2∞+αδ2‖dk‖∞-αγh (c (xk) )

    ≤α[αγmM‖dk‖2∞- (γh (c (xk) ) -δ2‖dk‖∞) ].

    当α≤γh (c (xk) ) -δ2∥dk∥∞γmΜ∥dk∥∞2时, 有f (xk+αdk) ≤f (n) -γh (c (xk+αdk) ) 成立.所以 (h (c (xk+αdk) ) , f (xk+αdk) ) 满足条件 (1) , 因而, 能被F (k) 中的所有元素接受.

    (2) h (c (xk) ) =0, 由引理1,

    h (c (xk+αdk) ) ≤α2mM‖dk‖2∞≤αmM‖dk‖2∞.

    令τk=min{h (c (xj) ) , j∈F (k) },

    若α≤τkητk+mΜ∥dk∥∞2, 则h (c (xk+αdk) ) ≤ (1-αη) τk.那么 (h (c (xk+αdk) ) , f (xk+αdk) ) 满足条件 (1) , 因而, 能被Filter接受.
