    小学数学翻转课堂研究回顾与展望

    王广木

    福建省厦门市集美区乐海小学

    摘    要：

    随着信息技术在教学中的应用, 小学数学翻转课堂也应运而生, 丰富了以班级授课制为主导的教学方式.本文通过对已有文献进行分析, 梳理了翻转课堂的发展概况, 分析了小学数学翻转课堂研究现状, 并且指出当前小学数学翻转课堂研究面临的问题, 旨在为深入开展小学数学翻转课堂研究提供思路.

    关键词：

    小学数学; 翻转课堂; 研究;

    基金： 福建省厦门市集美区教育科研“十三·五”规划2016年度课题《翻转课堂在小学数学教学中的实践探究》阶段性成果, 课题编号16061;

    一、翻转课堂发展概况

    翻转课堂于美国兴起, 随后传播到全世界.加拿大的《环球邮报》在2011年给予翻转课堂“影响课堂教学的重大技术变革”的美称.美国阿拉帕霍高中的卡尔·费舍和林地公园高中的伯格曼和亚伦, 最早开始在基础教育领域中研究翻转课堂教学.2011—2012学年, 加拿大奥卡纳中学格拉哈姆·约翰逊和卡罗琳·多莉开展的翻转课堂教学效果显著, 受到众多学生和家长的好评.新加坡裕廊景中学的李嘉昌对翻转课堂教学模式的研究, 在2013年获得了“李光耀奖”.澳大利亚的昆士兰大学和昆士兰州立高中的研究认为:“翻转课堂能够让学生主动学习, 给学生足够的学习时间, 满足学生的学习需要.”中国重庆市聚奎中学于2011年总结了“课前四步”“课中五环”的翻转课堂教学模式.深圳南山实验教育集团于2012年提炼了“三步五环节”的翻转课堂教学模式.2015年5月, 清华大学宣布将利用翻转课堂教学方式, 培养国内首个数据科学与工程专业硕士学名学生.2015年11月香港乐善堂于近卿中学采用Schlllogy网上学习平台, 开展翻转课堂教学.2016年5月, 青岛日报发表了《山东省青岛市将在全市推广翻转课堂教学》为题的文章, 表示该市将加强教育统筹管理, 实现优质教育资源的共享, 并计划三年内完成所有学科微视频制作, 以便大范围开展翻转课堂教学.

    二、小学数学翻转课堂国内研究现状分析

    笔者以“翻转课堂”为关键词, 利用中国知网数据库进行文献跨库检索.翻转课堂于2011年开始在美国兴起, 在2012年以前, 中国知网内没有关于翻转课堂教学研究的文献, 因此, 对文献检索日期设定为2012年1月1日到2018年1月1日.利用知网库进行篇名模糊检索, 得到如下结果:文章题目含关键词“翻转课堂”的共15 031篇, 含关键词“小学数学”和“翻转课堂”的共68篇.缩小检索范围, 利用中国知网数据库, 通过篇名进行精确检索, 尽数据库的可能进行检索, 得到如下结果:文章题目含关键词“翻转课堂”的共11 295篇, 含关键词“小学数学”和“翻转课堂”的共54篇.笔者以通过文章题目含关键词“小学数学”和“翻转课堂”进行精确搜索所搜集到的54篇文献为最终研究样本.文献从发表时间来看, 数量随着时间的推进而呈上升趋势, 相关成果不断得到丰富, 仅2016年一年的数量就占了总数的48.15%, 小学数学翻转课堂受到越来越多学者的关注.

    笔者通过对这54篇文献样本进行分析和整理, 学者们对“小学数学翻转课堂教学”的研究主要围绕以下问题进行探讨.

    (一) 关于小学数学翻转课堂模式的研讨

    翻转课堂作为一种蕴含着先进教学理念的教学模式, 如何在小数学学科教学中实施和操作, 值得当前学者广泛关注.关于小学数学翻转课堂教学模式研究, 主要包括:小学数学翻转课堂教学步骤探究、结合课例介绍小学数学翻转课堂的教学模式、小学数学翻转课堂模式下的教学策略研究、小学数学翻转课堂模式下的教学方式研讨以及如何构建个性化小学数学翻转课堂教学模式等.如, 江苏省泰州市高港实验小学的凡晓雨在《融合“翻转课堂”, 优化小学数学教学》 (《学子》 (理论版) , 2015 (22) ) 一文中介绍了小学数学翻转课堂三个步骤:学生课前观看视频自学;课上师生互动讨论, 解决疑难;课后教师总结经验, 优化教学.广东省佛山市顺德区伦教培教小学何雪文在《“人人通”云环境下小学数学翻转课堂教学实践》 (《教育信息技术》, 2016 (3) ) 一文中介绍了基于“人人通”云环境下小学数学翻转课堂的教学模式, 并指出该模式可以提升课堂教学效率, 促进学学生学习水平发展.广州市海珠区海联路小学黎启韶在《小学数学“翻转课堂”的实践与思考——以“画角”一课为例》 (《教育信息技术》, 2015 (03) ) 一文通过具体实例, 介绍了小学数学翻转课堂教学模式, 并提出了关于翻转课堂的一些思考.

    (二) 关于小学数学翻转课堂价值的探讨

    实施翻转课堂教学, 可以提升课堂教学实效, 满足学生学习的个性化需求, 促进教师的教和学生的学.有关在小学数学领域开展翻转课堂教学价值的探讨主要包括小学数学翻转课堂对学生的价值、对教师的价值、对学习的价值和对教学效果的价值等.海南师范大学翟树红和海口寰岛实验小学于跃在《“翻转课堂”在小学数学教学中的应用价值研究》 (《亚太教育》, 2016 (4) ) 一文中探讨了小学数学翻转课堂的应用价值.青岛香港路小学的郑静在《翻转课堂在小学数学教学中的应用》 (《教师博览 (科研版) 》, 2015 (06) ) 一文中指出小学数学翻转课堂关注学生自主构建、注重实践生成、兼顾个体差异, 让学生成为课堂学习的主人.即墨市第三实验小学孙进平在《翻转课堂在小学数学教学中的使用》 (《吉林教育》, 2015 (01) ) 中指出, 小学数学翻转课堂能够培养学生的创新思维和品质, 提升创新力.江苏省南通师范学校第二附属小学曹彬在《关于运用微视频“翻转课堂”对小学数学教学的启示》 (《新课程导学》, 2015 (04) ) 一文指出, 小学数学翻转课堂教学使学生在学习中的主人翁地位得到体现, 增加了师与生、生与生的交流, 提高了学生的学习效率.江苏省昆山市培本小学张水平在《微课程“翻转课堂”对小学数学教学的启示》 (《数学学习与研究》, 2014 (08) ) 一文中指出, 小学数学翻转课堂重新分配教学时间, 重构学习流程, 变革了数学课堂, 挑战教师的教学方式, 让学习成为学生的自主需要, 让学生成为数学学习的主人.

    (三) 关于小学数学翻转课堂教学实践的思考

    随着小学数学翻转课堂被人们所熟知, 越来越多学者参与到翻转课堂实践中, 并表达了自己的思考.关于翻转课堂的思考主要涉及的问题是关于实践中翻转成败的思考、关于实践中教学方式的探讨、关于实践注意事项及建议、关于通过实验数据分析得出结论的实践探讨等.如江苏省溧阳市实验小学薛强在《玩转“翻转课堂”彰显数学本质——小学数学“数字与信息”“翻转课堂”教学实践与反思》 (《文理导航》, 2013 (08) ) 一文中进行了两次翻转实践, 指出微视频的制作是翻转课堂成功的关键, 学生的自学还需要家长监督, 教师和学生的信息技术能力需要加强.江苏省高邮市城北实验小学张芹在《“翻转课堂”模式在小学数学教学中的实践》 (《内蒙古教育》, 2015 (21) ) 一文中指出, 小学数学翻转课堂教学要善于利用视频吸引学生注意, 组织学生在课前自学, 上课时要运用小组合作学习促进学生主动发展.江苏省苏州工业园区胜浦实验小学的周丽在《小学数学翻转课堂的实践与思考》 (《中国信息技术教育》, 2014 (17) ) 一文中指出, 教师要选择合适的内容进行翻转, 要制作有效的微视频, 并对微视频制作、课堂组织、教师语言、教师指导、课堂总结提出建议.

    (四) 关于小学数学翻转课堂局限的讨论

    小学数学翻转课堂具有其优势, 但也具有一定的局限, 不少学者对其局限也进行了一定的探讨.有关小学数学翻转课堂的局限主要在硬件条件限制、教师综合水平、学生自我管理能力、学生自学能力的以及家庭环境的局限等.广东省梅州市梅县区丽群小学林群燕在《“翻转课堂”模式在山区小学数学教学中的实践》 (《教育信息技术》2015 (03) ) 一文中指出, 实施翻转课堂教学, 面临着学生没电脑无法上网、家长不支持上网、制作视频太耗时间、学生无法坚持自学微视频、学习质量不高等问题.浙江省湖州市湖师附小教育集团范新林、朱国平在《小学数学“翻转课堂”的冷思考》 (《小学数学教师》, 2015 (Z1) ) 一文中指出, 由于受地区经济、教育资源、家庭信息化条件、学校信息化环境、学生的信息化素养、师资力量、学生自律能力等影响, 还无法大面积推广翻转课堂, 建议大家对翻转课堂不要排斥, 也不要过于热衷.

    三、国外小学数学翻转课堂的研究现状分析

    国外小学数学翻转课堂的研究文献多是介绍翻转实施翻转课堂的经验.如, 黄发国、张福涛在《翻转课堂导学案编写指导与案例分析》一书中提及, 萨尔曼·可汗利用雅虎涂鸦给表妹娜迪亚辅导数学作业, 辅导结束后让表妹完成配套作业, 以检验她的学习效果, 在可汗的辅导下, 表妹的数学突飞猛进.2011年, 美国的明妮苏达州石桥小学开始在小学数学教学中尝试翻转课堂教学, 他们以五六年级学生为对象, 利用Moodle平台监控学生自学的情况, 及时地对学生的学习做出评价和反馈, 在上课时帮助学习有困难的学生.国外的小学数学翻转课堂主要体现人性化学习理论, 让学生按照自己的学习方式进行自学, 教师在课堂上给学习有困难的学生更多帮助, 让学生成为课堂学习的主人.翻转课堂教学的实践是多元化的, 不同国度、不同地区、不同学校的教育工作者实践翻转课堂, 都应该从实际出发, 按照系统最优化原则选择技术方案.

    四、小学数学翻转课堂研究展望

    学者对小学数学翻转课堂模式进行了研讨, 并且结合教学课例对如何在小学数学课堂教学中实施翻转课堂教学进行了研究, 肯定了小学数学翻转课堂教学的价值, 针对在小学数学翻转课堂教学过程中遇到的问题提出思考, 并且指出了小学数学翻转课堂的局限, 这对深入研究小学数学翻转课堂打下良好的基础.但是, 当前小学数学翻转课堂研究的文献资源还不够丰富, 还存在着一定的不足.首先, 从研究内容上看, 相关研究大多偏向于介绍个人经验或体会, 讨论小学数学翻转课堂价值的文章居多, 文献创新点不够突出, 对一些值得借鉴的独特做法还缺少研究.其次, 实践与理论结合不够理想.从研究者的身份上看, 大多为一线小学数学教师, 理论性较薄弱, 对小学数学翻转课堂的相关理论, 如掌握学习理论、最邻近发展区等鲜有提及, 对翻转课堂的理论研读不够透彻, 在实践过程中与理论结合不够紧密.最后, 研究方法较为单一.从研究运用的方法上看, 文献大多采用了个案分析法、经验总结法为主, 文献多是具体的小学数学翻转课堂案例介绍, 大多是个案案例, 对如何设计翻转课堂教学缺乏系统性阐述.同时, 翻转课堂是否具有优越性, 对实施翻转教学前后学生的成绩、学生的学习态度、教师的教学态度等都缺乏数据说明, 不够让人信服.
