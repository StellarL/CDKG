    摘    要：

    针对传统的离群点检测算法在机场大数据集上存在着检测精度较低、检测运行时间运行较高等问题,提出基于嵌套循环的机场大数据集离群点检测算法。通过对机场大数据集进行分析,采用划分方法对机场数据集进行初步处理,将较大规模的机场数据集划分成小规模的数据子集。利用嵌套循环算法对机场大数据集离群点进行检测,引入两个剪枝规则对其数据集离群点进行高效剪枝,减少了离群点检测时机场数据点之间距离的计算次数,此完成检测。实验结果表明,所提算法具有较高的检测精度并且检测运行时间运行较低。

    关键词：

    机场大数据集; 离群点; 检测; 嵌套循环;

    Simulation Research on Outlier Detection Algorithm of Airport Big Data Set

    FAN Jiang Guo Yong

    Institue of Geographic Spatial Information, Information Engineering University

    Abstract：

    Traditional outlier detection algorithms have low detection precision and high detection running time for the airport big data set. Therefore, this article proposed an outlier detection algorithm for airport big data set based on nested loop. After analyzing the airport big data set, we used the partitioning method to preliminary process the airport big data set. And then, we divided large-scale airport data sets into small-scale data subsets. Moreover, we used the nested-loop algorithm to detect the outliers in airport big data set. According to the two pruning rules, we pruned the outliers in data set efficiently, reducing the computing times of the distance between the airport data points during the detection of outliers. Thus, we completed the detection. Simulation results show that the proposed algorithm has higher detection accuracy and lower detection running time.

    Keyword：

    Airport big data set; Outliers; Detection; nested loop;

    1 引言

    机场大数据集离群点是指明显偏离其它对象的数据点,也可以将它称作为孤立点和异常点[1]。对机场大数据集目标点检测是机场数据集挖掘领域中一项重要的挖掘技术[2,3]。对离群点检测能够发现机场大数据集中隐藏的有价值知识。但现阶段机场大数据集目标点检测中普遍存在着,检测精度较低、检测运行时间运行较高等问题。针对这一问题,提出一种高效的目标点检测算法,成为当今社会亟待解决的问题[4,5]。

    目前,对机场大数据集目标点检测已经得到了很多专家学者们的高度关注,并且取得了一定的成果,其它检测方法有很多种。例如,基于分化距离的数据离群点检测算法。该算法主要根据K邻居之间的距离来确定机场数据是否离群。基于K-means聚类的数据离群点检测算法。该算法将大数据集划分为多个子集,不属于任何子集的数据点就是离群点。上述两种方法都能对机场大数据集离群点进行检测,但是局限于全局离群点检测,对局部离群点检测不出来,并且还存在检测精度较低等问题。

    针对上述问题,提出基于嵌套循环的机场大数据集离群点检测算法。实验结果表明,所提算法具有较高的检测精度并且检测运行时间运行较低。

    2 方法

    2.1 机场大数据集预处理

    通过对机场大数据集进行分析,采用划分方法对机场大数据集进行初步处理,将较大规模的机场数据集分割成小规模的数据子集。具体过程如下。

    机场大数据集目标点检测是数据挖掘中的主要部分,最开始是在统计学中出现的[6],后来被主要运用到对数据挖掘的领域中,关于机场大数据的离群点,也是一个观察点,它偏离其它机场大数据集中其它的观察点十分的明显。机场大数据集D中的随意一个元素x,当它和机场大数据集中最多pE个对象之间的距离小于pD时,这时就是说明x是关于参数pE和pD的离群点。当x在pD范围内小于pE个邻居时,则x是离群点,并且其中参数pD用来确定邻域,pE则用来判断x是否是离群点。

    为了能够更加清晰的表达出机场大数据集目标点的离群程度,专家学者们提出了离群度的概念[8],它是指在机场大数据集中,目标点与它邻域的偏离情况,离群度是通过机场数据对象到K个最近邻的距离之和来得到的,与它距离越大,就可以看成是离群度越高。

    对机场大数据集处理的整个过程是将海量的机场大数据集划分成多个子集,通过分析可知,机场数据点的近邻只能和它挨着的数据点之间存在联系,和它较远的数据之间没有任何关联,所以要对机场大数据集进行分割[9]。通过与机场大数据集中计算近邻进行对比,在机场大数据子集中计算数据点的近邻计算量将会大幅度下降。并且在分割边缘的数据点将会出现丢失的现象,机场大数据集划分之前和划分之后的边缘数据点的近邻也会存在一定的变化,将它强行进行划分,这个部分所丢失的信息将会直接影响机场大数据集目标点检测的精度,为此采用特定的数据集划分方法对机场大数据集进行划分。

    文中引用软硬结合的划分策略来对机场大数据集进行划分。首先对机场大数据集进行硬划分,利用K-Means聚类算法[10]对机场大数据集进行初步处理,以此来得到n个类簇,ci(i=1,2,…,n)代表聚类中心,并且将每个簇都可以看成机场大数据集划分后的核心集,利用Di(i=1,2,…n)来代表,又因为在机场大数据集预处理的过程中只需要获得机场大数据集的粗糙分割,所以对聚类的精度并不算太高。

    软分割主要是构建机场大数据集核心集的扩展集[11],该集是在机场大数据集核心集外所引入的一部分多余的数据点,这些数据点可以有效的保障机场大数据核心集中数据点的K近邻不出现丢失情况。

    第一步要先得到每个簇的簇半径,在机场大数据每个核心集中求出距离ci最远的簇内数据点,则这个距离称作为簇半径ri,利用下式给出其表达式

    ri=max{d(x1,ci),d(x2,ci),…,d(xm,ci)} (1)

    式中,x代表Di中的数据点,则该点总共有m个。对机场大数据核心集进行扩展,利用下式给出得到的扩展集D′i

    式中,x′代表机场全体数据集中的所有数据点;rErtend代表扩展系数。以式(2)为基础,可以得到机场大数据集中的数据集预处理表达式D*i

    D*i=D′i-Di (3)

    通过上述可知,机场大数据核心集的硬分割,即D=D1∪D2∪…∪Dn,且Di∩Dj=∅(i≠j,i,j=1,2,…,n),则扩展集是机场大数据集的软分割,并且扩展集中还含有核心集,则Di⊂D′i,且D=D1′∪D2′∪…∪Dn′。这样,机场大数据集会被分割成Nc个子集,当得到合适的扩展系数时,则机场大数据核心集近K邻信息基本保存完好,并采用启发式规则将聚类方法对机场大数据集划分Nc子集,根据ODC值,按照从大到小进行排序,其中ODC值代表衡量机场数据集类簇内含有离群点的可能性大小的一个指标。

    2.2 基于嵌套循环的机场大数据集离群点检测算法

    在2.1节机场大数据集初步处理的基础上,采用嵌套循环算法对机场大数据集目标点进行检测,并根据两个剪枝规则对其数据集离群点进行高效剪枝,减少了离群点检测时机场数据点之间距离的计算次数,以此完成检测。具体过程如下。

    文中改进了传统的嵌套循环算法[12],并在其中引入两个剪枝规则。

    第一条规则,当Da(q)<Omin时,Omin代表剪枝阈值,则机场数据点p被修剪并且不用进行k近邻搜索。

    第二条规则,对机场数据点p进行k近邻搜索。假设,机场大数据集中数据点q和数据点p之间的计算距离主要以检查是否为近邻,又假设,机场数据点q已经计算过Da(q)值,这时就可以利用Da(q)和三角不等式来求出Da(p)的上界值,当该值小于Omin时,则机场数据点p被修剪并且不用进行k近邻搜索。

    对第二条规则进行继续推导,假设,已经求出的机场数据点q的离群因子Da(q)。并且对p进行k近邻搜索时,取数据点q和p计算距离来检测是否为近邻,当点p、q和q个k个最近邻之间构成k个三角形,这时采用三角不等式可得到

    k×dist(p,q)+∑i=1kzi∈kΝΝ(q)dist(q,zi)≥∑i=1kzi∈kΝΝ(q)dist(p,zi)         (4)

    在式(4)两端同时除k,则

    dist(p,q)+Da(q)>1k∑i=1kzi∈kΝΝ(q)dist(p,zi) (5)

    q的k个最近邻也不一定就非得是p的k个最近邻,所以有

    1k∑i=1kzi∈kΝΝ(q)dist(p,zi)>Da(p) (6)

    通过式(5)和式(6)可以得到

    dist(p,q)+Da(q)>Da(p) (7)

    根据上可以得到Da(p)的上界值,当该值小于剪枝阈值Omin时,则式(8)成立,这时说明机场数据点p不可能是离群点,不用在进行k近邻搜索。飞机大数据集离群点检测表达式为

    dist(p,q)+Da(q)<Omin (8)

    按照上述对机场大数据集进行划分,并利用启发式规则对其进行排序作为基础,利用嵌套循环算法依次从C1到CNc来对目标点进行检测。当检测出机场数据点的ODC值较大时,则导致剪枝阈值Omin会快速增加,所以占机场大数据集中的正常数据,就会根据剪枝第一条规则和第二条规则的作用,不用精确得到数据点的k个最近邻,就能确定它为非离群点,而终止k近邻搜索,来减少机场数据点间距离计算的次数,当Omin增加的越快,内循环k近邻搜索是的剪枝效率就越高,对机场大数据离群点检测精度就越高。
