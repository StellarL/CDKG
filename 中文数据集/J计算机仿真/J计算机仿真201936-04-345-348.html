    摘    要：

    对分布式网络资源用户的信息进行快速获取, 能够有效整合网络用户, 提高网络运行效率。对资源用户信息的获取, 需要对BP神经网络误差函数权值进行更新调整, 对数据结构神经元之间连接权值进行修正。传统方法根据本征模函数上的数据结构包络, 采用3α原则与计算获取阈值的对比, 但忽略了对数据结构权值的修正, 导致获取精度偏低。提出一种基于BP神经网络的资源用户信息快速获取方法, 对数据结构原信号的尺度缺失进行补充, 采用BP神经网络计算其误差函数, 同时利用梯度下降法对BP神经网络误差函数权值进行更新调整;引入动量因子α, 将BP神经网络学习模式不断输入到输入层, 对数据结构神经元之间的连接权值进行修正, 实现对分布式网络资源用户信息的快速获取。实验结果显示, 所提方法具有高快速获取率、低误报率的资源用户信息快速获取性能, 且大大提高了收敛速度。

    关键词：

    分布式网络; 网络资源; 用户信息; 快速获取;

    作者简介： 暴占彪 (1976-) , 男 (汉族) , 河南滑县人, 硕士, 副教授, 研究方向:网络系统与信息计算。;

    Simulation of Rapid Acquisition of User Information in Distributed Network Resources

    BAO Zhan-biao

    Cyberspace Administration Center, Henan University of Economics and Law

    Abstract：

    The fast acquisition of user information in distributed network resources can effectively integrate network users and improve running efficiency. The traditional method compares the 3α principle with the threshold value, but ignores the correction of weight value of data structure, resulting in the low acquisition precision. Therefore, this article focuses on a method to fast acquire user information based on BP neural network. This research supplied the scale deficiency of the original signal of data structure and used BP neural network to calculate its error function. Then, our research used the gradient descent method to update and adjust the weight value of error function of BP neural network. Moreover, the research introduced the momentum factor αto constantly input the learning mode of BP neural network into the input layer. Finally, we modified the connection weight value between data structure neurons to achieve the rapid acquisition of user information in distributed network resource. Simulation results show that the proposed method has fast acquisition performance of resource user information with fast acquisition rate and low false alarm rate, which greatly improves the convergence rate.

    Keyword：

    Distributed network; Network resource; User information; Fast acquisition;

    1 引言

    近年来, 随着网络信息技术水平的不断提升和计算机应用技术的不断更新发展, 分布式网络数据共享环境发生了改变, 用户可以在分布式网络数据共享环境下, 通过网络平台发布信息和获取信息, 这给人们的生活带来了极大的便利的同时, 由于资源用户信息的产生, 严重影响了数据集成的质量[1]。因此, 研究出一种有效的资源用户信息快速获取方法对保证分布式网络高质量的数据集成具有重要现实意义[2]。

    文献[3]提出一种基于3α原则与快速获取阈值相结合的资源用户信息快速获取方法, 该方法通过对分布式网络训练数据结构信号的经验模态分解得到本征模函数上的数据结构包络, 并对该数据结构保留进行高斯建模和自适应学习, 计算快速获取阈值, 通过采用3α原则与计算得到的快速获取阈值的对比实现资源用户信息快速获取。结果表明当分布式网络规模较大时, 该方法的收敛速度较慢。文献[4]提出一种基于小波模极大值模糊熵的资源用户信息快速获取方法, 对采集获得的分布式网络的数据结构信号进行零漂移修正和趋势项消除等预处理;对预处理后的数据结构信号进行一维连续小波变换获得分布式网络所有尺度空间中数据结构信号的小波模极大值序列;将原始结构数据信号与处理后的信号序列的模糊熵构成特征向量输入到支持向量机分类器中, 根据数据结构信号模糊熵的变化情况实现冲突快速获取。结果表明, 该方法的误报率较高。文献[5]提出一种基于小波模极大值多重分析的资源用户信息快速获取方法, 在对采集获得的分布式网络数据结构信号预处理的基础上, 利用小波模极大值多重分形的方法对预处理后的数据结构信号进行分析获得相应分形特征, 将数据结构分形特征输入到分类器中, 实现冲突快速获取。该方法存在响应时间较长, 平均快速获取率较低的缺点。

    研究提出一种基于BP神经网络的资源用户信息快速获取方法, 试图解决上述快速获取方法存在的缺点与不足, 仿真结果证明了所提方法的优良性能。

    2 分布式网络资源用户信息快速获取原理

    对分布式网络一数据结构时间序列进行相空间重构获得重构矩阵, 将数据结构时间序列中的重构分量按照从低到高的顺序重新排列得到不同符号序列的排列熵[6];采用排列熵最大值对不同符号序列的排列熵进行归一化处理, 根据数据结构时间序列的排列熵变化情况实现分布式网络资源用户信息快速获取[7]。

    假设m和τ分别表示分布式网络数据结构的嵌入维数和延迟时间, 对分布式网络一数据结构时间序列{X (i) , i=1, 2, 3, ⋯, n}进行相空间重构, 可以得到以下矩阵

    [x (1) x (1+τ) ⋯x (1+ (m-1) τx (2) x (2+τ) ⋯x (2+ (m-1) τx (j) x (j+τ) ⋯x (j+ (m-1) τ⋯⋯⋯⋯x (Κ) x (Κ+τ) ⋯x (Κ+ (m-1) τ]    (1)

    其中, j=1, 2, 3, …, K, K=n- (m-1) τ。上式 (1) 中的每一行表示分布式网络数据结构时间序列的一个重构分量, 整个时间序列包含K个重构分量。将数据结构时间序列X (i) 中的第j个重构分量按照从低到高的顺序重新排列, 表达式如下

    x (i+ (j1-1) τ) ≤x (i+ (j2-1) τ) ≤⋯≤x (i+ (jm-1) τ)     (2)

    其中, j1, j2, …, jm代表数据结构时间序列重构矩阵分量中各个元素所在列的索引。

    如果数据结构时间序列重构矩阵分量中存在相等的值, 即

    x (i- (j1-1) τ) ≤x (i- (j2-1) τ)     (3)

    根据上式计算, 对于分布式网络数据分享下, 任意一个数据结构时间序列X (i) 都可以得到一组符号序列, 表达式为

    S (l) = (j1, j2, ⋯, jm) , l=1, 2, ⋯, k    (4)

    其中, k≤m!, m!表示数据结构时间序列m维相空间映射不同的符号序列数量, S (l) 只是其中一种排列。假设P1, P2, …, Pk表示数据结构时间序列相空间映射中每一种符号序列出现的频率, 则X (i) 的k种不同符号序列的排列熵计算公式如下

    Ηp (m) =-∑j=1kΡilnΡj    (5)

    当Pj=1/m!时, 排列熵Hp (m) 的值可以达到最大ln (m!) , 采用排列熵最大值ln (m!) 对上式 (5) 进行归一化处理, 实现分布式网络资源用户信息快速获取, 表达式如下

    0≤Ηp=Ηp/ln (m!) ≤1    (6)

    式中, Hp代表数据结构时间序列重构矩阵X (i) 的随机程度;Hp的值越大, 则说明分布式网络数据结构时间序列越规则, 相反, 如果Hp的值越小, 则说明分布式网络数据结构时间序列越随机, 根据Hp值可以提取并放大数据结构时间序列中的微小变化, 实现资源用户信息快速获取。

    综上所述, 为分布式网络资源用户信息快速获取原理, 根据该原理实现了分布式网络资源用户信息快速获取。

    3 分布式网络资源用户信息快速获取方法

    在分布式网络数据结构原信号中加入白噪声, 对数据结构原信号的尺度缺失进行补充, 使其在不同尺度上具有连续性, 能够减小数据结构信号模态混叠程度, 更加真实地反映原始数据结构信号[8];在此基础上, 采用BP神经网络计算分布式网络数据结构误差函数, 同时利用梯度下降法对BP神经网络误差函数权值进行更新调整;将BP神经网络学习模式不断输入到输入层, 再根据输入层产生的误差对数据结构神经元之间的连接权值和数据结构神经元不断进行修正, 直到学习模式全部输入到BP神经网络中, 或者误差值达到预设值[9];为了解决计算量过大影响收敛速度的问题, 引入动量因子α, 最终实现对分布式网络资源用户信息的快速获取[10]。

    3.1 数据结构模态混叠消除

    EMD是根据分布式网络数据结构信号的局部极值特征将数据结构信号分解为有限个模态函数IMF和一个余项之和。其中, 每个IMF均满足两个条件:一是分布式网络整个数据结构信号范围内过零点数目与极值点数目相差小于等于1;二是数据结构信号上任意一点, 根据其局部极大值和局部极小值分别确定的数据结构信号包络线均值为0。假设分布式网络任意数据结构实信号表示为s (t) , 其EMD计算公式如下

    s (t) =∑i=1nci+rn    (7)

    式中, ci (i=1, 2, …, n) 表示数据结构信号的模态函数分量;rn表示数据结构信号分解的余项。

    EEMD方法在分布式网络数据结构原信号中加入白噪声, 对数据结构原信号的尺度缺失进行补充, 使其在不同尺度上具有连续性, 能够减小数据结构信号模态混叠程度。在进行多次EMD之后, 由于在原信号中添加的白噪声之间互不相关, 对所有数据结构信号的EMD结果相应的模态函数分量进行求整平均, 可以将人为添加的噪声抵消。具体步骤描述如下:

    假设N表示执行EMD的次数;δ表示数据结构信号中加入的白噪声n (t) 幅值系数;Nc表示数据结构信号总模态函数数目, 向s (t) 中加入幅值系数为δ的白噪声得到sω (t) , 即:

    sω (t) =s (t) +k⋅n (t)     (8)

    对上式 (8) 进行EMD可以得到Nc个模态函数分量, 记作c1, i′ (i′=1, 2, …, Nc) , 假设cj′, i′ (j′=2, 3, …, N, i′=1, 2, …, Nc) 表示重复执行EMDN-1次得到的结果, 对其求平均值, 表达式如下:

    ci¯=∑i=1ncj′, i′/Ν    (9)

    从上述EEMD过程中可以看出, 得到的数据结构信号IMF分量并不一定是真实的IMF分量, 需要对上述结果进行进一步处理, 假设aci′表示分布式网络数据结构信号进行EEMD后各个IMF分量;ac1表示对数据结构信号进行EEMD的结果进行EMD得到结果的第1个IMF分量, 将该结果作为最终EEMD的第1个IMF分量, 同时将数据结构信号EEMD处理剩余部分的resi′加到acNc上, 并对其进行EMD, 可以推导出以下关系式

    {ac1EΜD→ec1+res1ac1+res1EΜD→ec2+res2⋮acΝc+resΝc-1EΜD→ecΝc+resΝc    (10)

    根据上述计算, 可以消除分布式网络数据结构信号的模态混叠, 表达式为

    s (t) =∑i′=1Νcaci′+resΝc    (11)

    3.2 基于小波变换的资源用户信息快速获取

    假设n′、r和m′分别表示BP神经网络输入层节点数目、隐含层节点数目和输出层节点数目;W= (wl) 和V= (wκ) 分别表示BP神经网络隐含层与输入层、隐含层与输出层之间的权值矩阵;θl和θ′l分别表示BP神经网络隐含层节点阀值和输出层节点阀值;γ表示分布式网络数据结构误差函数, 它是数据结构期望输出值与实际输出值之间差的平方和, 计算公式如下

    γ=12∑l (Vlm-yl) 2⋅s (t)     (12)

    式中, Vlm和yl分别表示分布式网络BP神经网络输出单元的数据结构期望输出值和实际输出值。

    采用梯度下降法对BP神经网络权值进行更新调整, 表达式为

    ΔWll′=-ε⋅dlκ⋅Vlκ-1    (13)

    式中, ε表示BP神经网络学习步长参数;Vjκ-1表示BP神经网络上一层神经元的数据结构输出;djκ有以下两种情况:

    ①如果l是BP神经网络输出层 (第m′层) 的神经元, 则:

    dlκ=Vlm (1-Vlm) (Vlm-yj)     (14)

    ②如果l不在BP神经网络输出层, 而处于隐含层的神经元, 则

    dlκ=Vlκ (1-Vlκ) ∑jWldlκ+1    (15)

    其中, Vlκ表示在BP神经网络中该神经元的数据结构输出, 将BP神经网络学习模式不断输入到输入层, 再根据输入层产生的误差对数据结构神经元之间的连接权值和数据结构神经元不断进行修正, 直到学习模式全部输入到BP神经网络中, 或者误差值达到预设值。

    上述方法在分布式网络数据共享规模较大时, 面临计算量过大的问题, 收敛速度受到影响, 容易陷入局部最优解, 为了解决这一问题, 引入动量因子α, 同时考虑权值更新, 对分布式网络资源用户信息快速获取, 表达式如下

    ΔWll′ (t+1) =-ε⋅dlκ⋅Vlκ-1+αΔWll′ (t)     (16)

    根据以上步骤完成了分布式网络资源用户信息的快速获取。
