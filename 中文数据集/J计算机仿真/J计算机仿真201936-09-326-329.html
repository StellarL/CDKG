    摘    要：

    研究一种有效的脚本病毒信息主动预警方法,能够降低病毒信息预警的漏报率,提高预警的准确率,增强超密集网络安全性。为了防止脚本病毒信息入侵超密集网络,提出了基于自适应阈值的脚本病毒主动预警方法,通过构建脚本病毒信息入侵信号模型,利用小波变换方法对脚本病毒信号进行线性变换,得到去噪后的脚本病毒信号。利用传统的残差比异常检测方法对超密集网络信息数据的观测值进行检测,获取决策函数,利用更新机制叠加前一时段的信息,计算标准差设置置信度区间,利用决策函数对脚本病毒信息进行预警,实现了超密集网络脚本病毒信息主动预警。仿真结果表明,所提方法能够在保证预警检测率和准确率的基础上降低误警率,提高了对网络脚本病毒的准确预警能力。

    关键词：

    脚本病毒信息; 主动预警; 检测率; 误报率; 漏报率;

    Research on Active Early Warning Simulation of Ultra-Dense Network Script Virus Information

    Dong Wei

    Xinzhou Teachers University Department of Computer Science

    Abstract：

    An effective active early warning method for script virus information can reduce the false negative rate of virus information warning, improve the accuracy of early warning and enhance the security of super-dense network. In order to prevent the script virus information, this paper proposed an active warning method for script virus based on adaptive threshold. By constructing the model of intrusion signal in script virus information, we used the wavelet transform method to linearly transform the script virus signal, so as to obtain the script virus signal after the noise reduction. Then, we used traditional residual ratio detection method to test the observation value of ultra-dense network information data, and thus to obtain the decision function. Moreover, we used the update mechanism to superimpose the information in previous period. After that, we calculated the standard deviation and set the confidence interval. Finally, we used the decision function to forewarn script virus information. Thus, we achieved the active warning for the script virus information in super-dense network. Simulation results show that the proposed method can reduce the false alarm rate and improve the accurate early warning ability for network script virus on the basis of ensuring the early warning detection rate and accuracy rate.

    1 引言

    现今社会,网络技术的不断发展和计算机的广泛应用给人们生活增添了新的转变,网络信息传播速度不断加快,具有较强的共享性,网络信息的安全性逐渐得到人们的重视。小到企业重要信息大到国家机密在网络传送和通信中,被各种不同目的的黑客进行网络入侵[1]。在超密集网络中,很容易受脚本病毒信息的入侵,并通过拒绝服务的方式将计算机中的重要信息盗取和利用,导致计算机不能正常使用以至于停止响应[2]。脚本病毒是计算机网络一种新式病毒,主要运用脚本语言编写,利用脚本对电脑系统进行传播和破坏。脚本病毒是互联网病毒中特别流行的网络病毒信息之一,为了防止脚本病毒信息入侵超密集网络,需要研究一种有效的脚本病毒信息主动预警方法刻不容缓[3,4]。

    文献[5]提出一种基于Apriori算法的超密集网络脚本病毒信息主动预警方法,通过分析超密集网络中的脚本信息,利用Apriori算法挖掘出脚本程序用于编辑信息,搜索出隐藏在编辑信息中的脚本病毒信息,实现超密集网络脚本病毒信息主动预警。该方法存在预警漏报率较高的缺点。文献[6]提出一种基于关联分析技术的密集网络脚本病毒信息主动预警方法,该方法对脚本程序编辑信息进行预警信息挖掘,获取预警信息规则集,并以该规则集对超密集网络传输脚本病毒信息进行预测,获取脚本病毒信息预测值,实现了脚本病毒信息主动预警。该方法存在预警误报率较高的缺点。文献[7]提出一种基于免疫和代码重定位的超密集网络脚本病毒信息主动预警方法。结合生物免疫系统机理,计算电脑系统中的自体、非自体、抗体和病毒检测器的免疫值,运用感染型病毒独特的代码定位特点提取出脚本病毒特征,并构建脚本病毒特征集,构建自体和非自体、脚本病毒特征和脚本病毒自主预警器动态演化模型,实现超密集网络脚本病毒信息主动预警。该方法存在预警准确率较低的问题。

    综上所述,提出了基于自适应阈值的脚本病毒主动预警方法,通过实验结果验证,该方法可以解决上述三种预警方法存在预警漏报率较高、误报率较高和准确率较低等问题。

    2 超密集网络脚本病毒信息主动预警方法

    2.1 脚本病毒信息入侵信号模型构建与去噪

    脚本病毒信息入侵信号模型构建与去噪过程是通过构建脚本病毒传播信息流,得到脚本病毒信息入侵信号模型,再结合小波变换方法对脚本病毒信号进行线性变换,利用硬阈值或软阈值进行阈值处理,再运用新的小波系数重构小波系数得到去噪后的脚本病毒信号[8]。

    1)构建脚本病毒信息入侵信号模型

    在超密集网络中,假设Υ0表示超密集网络脚本病毒入侵的方位聚类系数,α表示脚本病毒转移中心权向量,l0表示具有干扰性的杂波信号,脚本病毒传播信息流的表达式如下:

    f(g)=cos[2πl0g+παf2+Υ0] (1)

    依据式(1),构建脚本病毒信息入侵信号模型,表达式如下

    t(ε,g)=∑mbm(g)e-ι2mltεm(g)η(g-εm(g)) (2)

    式中,bm(g)表示脚本病毒信息入侵超密集网络的第m条路径上接收信号的衰减因子,εm(g)表示第m条路径的滞后时间;lt表示瞬时频率,ι表示超密集网络中链接的病毒空集群体。

    2)脚本病毒信息入侵信号去噪

    假设a(t)表示原始病毒信息入侵信号,r(t)表示方差为δ2的高斯白噪音,且服从M(0,δ2),一维脚本病毒信号的表达式如下

    u(t)=a(t)+r(t) (3)

    将u(t)进行离散采样,取得M点的离散病毒信号是u(m),m=0,1,2,…,M-1,则小波变换的计算公式如下

    Ρu(i,j)=2-i/2∑m=0Μ-1u(m)Φ(2-i-j),(i,j)∈Ζ (4)

    在实际使用中,由于式(4)比较复杂,且小波函数Φ(t)没有明显表达式,则以小波变换的递归实现方式可表示为

    Hu(i+1,j)=Hu(i,j)*k(i,j),

    Pu(i+1,j)=Fu(i,j)*f(i,j), (5)

    式中,k表示尺度函数σ(t)相应的高低通滤波器,f表示小波函数Φ(t)相应的高低通滤波器,Hu(0,j)表示原始病毒信息入侵信号;Hu(i,j)表示尺度系数。Pu(i,j)表示小波系数,与式(5)对应的重构表达式如下

    Ηu(i-1,j)=Ηu(i,j)*k∼(i,j)+       Ρu(i,j)*f∼(i,j)         (6)

    式中,k∼表示尺度函数σ(t)相应的重构高低通滤波器;f∼表示小波函数Φ(t)相应的重构高低通滤波器。

    由于小波变换属于线性变换,将Hu(i,j)用hi,j来表示,则u(j)=a(j)+r(j)经过离散小波变换以后获取小波系数为hi,j,依旧是两个部分组成,一部分为病毒信号a(j)相应的小波系数Hu(i,j),标记成oi,j,还有一部分是噪音r(j)对应的小波系数Hu(i,j),标记成wi,j。

    若hi,j小于假设临界值,hi,j是因噪音造成的,可舍弃;若hi,j大于假设临界值,hi,j因脚本病毒信号造成,则运用硬阈值或软阈值进行阈值处理,再运用新的小波系数重构小波系数得到去噪后的脚本病毒信号,其具体步骤如下:

    1)运用含有噪音的脚本病毒信号u(j)进行小波变换,获取一组小波系数hi,j。

    2)经过hi,j运用硬阈值或软阈值函数进行阈值处理,取得估计小波系数h⌢i,j,将∥h⌢i,j-oi,j∥保持为最小。

    3)将h⌢i,j经过小波重构后,估计病毒信号u¯(j),为去噪后的脚本病毒信息入侵信号。

    2.2 超密集网络脚本病毒信息主动预警方法

    基于上述脚本病毒信息入侵信号模型构建与去噪结果,提出了基于自适应阈值的脚本病毒主动预警方法,通过利用传统的残差比异常检测方法对密集网络信息数据的观测值进行检测获取决策函数,再通过更新机制叠加前一时段的信息,获取自适应阈值,确认目前时段检测点是否含有脚本病毒数据,通过计算标准差,设置置信度区间,利用决策函数对脚本病毒信息进行预警,实现了超密集网络脚本病毒信息主动预警。

    1)传统的残差比异常检测方法

    ①假设在时间点…,xv-1,xv,xv+1,…上的超密集网络信息数据的观测值序列是…,Z(xv-1),Z(xv),Z(xv+1),…,当A(xv)=sx时,代表x=v时的数据观测值,设sx+1,sx+2,sx+V+1,…表示部分平稳数据,对其进行均值处理,即

    s¯=1V∑τ=1V+1sx+1 (7)

    zx+τ=sx+1-s¯,τ=1,2,⋯,V+1 (8)

    ②对零均值化后的序列运用AR(2)模型拟合取得残差,即:

    …,px+1,px+2,…,px+V+1,…:

    px+1=zx+τ-μ1zx+τ-1-μ2zx+τ-2,τ=1,2,…,V+1 (9)

    式中,μ1和μ2表示AR(2)模型系数。

    ③确认决策函数

    Uτ(V+1)=px+V+1φ^ (10)

    式中,φ^2=(px+1+px+2+⋯+px+V+1)/(V+1)。

    ④利用决策函数判定sx+V+1中是否含有脚本病毒信息,当Uτ(V+1)>K或者Uτ(V+1)<-J时,检测出sx+V+1含有脚本病毒信息,超密集网络主动预警。当Uτ(V+1)>K时,sx+V+1脚本病毒信息数量较大,当Uτ(V+1)<-J时,sx+V+1脚本病毒信息数量较少,相反,sx+V+1中不含有脚本病毒,K和J属于常数[9]。

    2)超密集网络脚本病毒信息主动预警步骤如下:

    ①在超密度网络中,从网络管理协议信息库中定时读取端口组变量,假设I表示设备接收端口;Y表示超密集网络信息传送字节数,T表示路由器每秒收到的超密集网络信息,Q表示超密集网络转发数据包数。

    ②运用滑动窗口模型最新超密度网络管理信息库中的n个数据,超密集网络数据越大,统计信息会一直更新,窗口数据一直平移。

    ③对滑动窗口中多组变量进行定时采样,依照时间顺序排列出一个时间序列,将该时间序列当作病毒信息观测值序列,采样的时间间隔可用毫秒、秒、分、小时等。

    ④确认自适应阈值

    假设x+1表示时段正常模型;d(x+1)表示x+1时段的阈值预测值,sx+V表示x时段的观测值;sx+V+1表示x+1时段的观测值;β表示加权常数,利用更新机制将前一天的行为叠加,表达式如下:

    d(x+1)=(β[sx+V+1-sx+V]+sx+V)u¯(j) (11)

    式中,利用加权常数β限制最新的信息数据在模型中占据比例,并适应部分行为的速度。

    构建正常超密度网络信息模型,其中含有模型更新[10],若目前观测值全部符合正常超密度网络信息模型,则观测值是属于正常的,这种判断比较严格,在实际应用中符合度会出现偏差,需设置置信区间,即:

    式中,υ2表示标准差。

    置信区间的范围是需通过建立正常超密度网络信息模型来获取的,不同地方是正常超密度网络信息数据来源于观测值结果,置信度区间范围的信息数据是通过计算观测值的标准差来获取,是指在每天的相同时间段,会产生一个观测值。在前一周的每个时间段会有七个观测值,利用该观测值可计算出标准差υ2,将得到的标准差加入正常超密度网络信息模型中,获取上边界K(x+1),用K(x+1)减去标准差υ2,得到下边界J(x+1)。依据加入标准差个数的不同,能取得级别不等的容差区域,通常情况下是标准差的3倍,当-J(x+1)≤Uτ(V+1)≤K(x+1)时,是超密度网络中的正常信息,当Uτ(V+1)>K(x+1)或Uτ(V+1)<-J(x+1)时,则为超密度网络脚本病毒信息,超密度网络发起预警。

    3 仿真及分析

    为了验证基于自适应阈值的脚本病毒主动预警方法的有效性,主要选取预警漏报率、准确率、误报率以及预警检测效率作为评价指标进行仿真测试。

    仿真在CPU为Intel Core i7-8565U,主频为2.9GHz,内存为8GB,操作系统是Windows10,运用Python语言编写的环境下构建测试平台。

    4 结束语

    随着互联网技术的不断发展,超密集网络易被脚本病毒信息的入侵,导致计算机不能正常使用以至于停止响应。为了防止脚本病毒信息入侵超密集网络,提出了基于自适应阈值的脚本病毒主动预警方法,该方法通过设置自适应阈值判断超密集网络是否含有脚本病毒,利用更新机制叠加前一时段的信息,设置置信度区间,对超密集网络脚本病毒信息进行预警,实现了超密集网络脚本病毒信息主动预警。仿真测试结果证明,所提方法能够克服了超密度网络脚本病毒信息的入侵,提高病毒信息入侵预警效率,具有一定的应用价值。
