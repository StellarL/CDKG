    摘    要：

    针对当前方法检测档案信息系统关键信息干扰特征时由于没有提取关键信息干扰特征, 导致干扰检测率不高、通信误码率较高的问题, 提出基于连续均值消除算法的档案信息系统中关键信息抗干扰模型。模型在构建档案信息系统中关键信息传输信道模型基础上, 采用概率统计法将档案信息系统关键信息通信随机过程中的各随机变量簇相同频率对应频谱幅值组成一个新的序列, 构建系统关键信息干扰频谱矩阵, 得到系统全时通信时关键信息干扰频谱统计特征, 完成干扰特征提取。在此基础上, 将系统接收关键信息干扰特征做快速傅里叶变换, 并将变换后的关键信息干扰特征功率谱值与预先设定的干扰检测门限值做比对, 如果变换后的关键信息功率谱值高于预先设定的干扰检测门限值则认为系统关键信息受到了干扰并将其滤除;相反则认为系统关键信息没有受到干扰。仿真测试结果表明, 所提模型提高了档案信息系统中关键信息干扰检测率的同时, 降低了通信误码率, 具有较好的抗干扰性能。

    Simulation of Key Information Anti-interference Model in Archives Information System

    JIANG Xiao-bo

    City College, Jilin Jianzhu University

    Abstract：

    A key information anti-interference model in the file information system based on continuous mean elimination algorithm is proposed. The interference model is based on the key information transmission channel model in the archive information system. The probability and statistics method is used to form a new sequence of the corresponding frequency spectrum amplitudes of each random variable cluster in the random information process of the archive information system. The key information of the system is constructed to interfere with the spectrum matrix, the statistical characteristics of the key information interference spectrum of the system during full-time communication are obtained, and the interference feature extraction is completed. On this basis, the system receives the key information interference features and performs fast Fourier transform, and the transformed key information interference characteristic power spectrum value is compared with a preset interference detection threshold value. If the converted key information power spectrum value is higher than a preset interference detection threshold value, the system key information is considered to be interfered, and filtered out; on the contrary, the system key information is not interfered. Simulation results show that the proposed model improves the interference detection rate of key information in the file information system, reduces the communication error rate and has better anti-interference performance.

    Keyword：

    Archive information system; Key Information; Anti-interference; Model;

    1 引言

    档案信息系统是办公自动化和电子政务发展的重要产物, 系统中包含了各个企事业机关单位产生的海量电子文件和档案信息, 档案信息系统的出现大规模取代了传统纸质载体, 成为国家企事业机关单位保存重要文件和信息的主要形式[1,2]。档案信息系统中包含的电子文件和档案信息绝大多数都与个人隐私信息相关, 这些关键信息在通信传输过程中通常是转化为数字信号的形式发送, 一旦被居心不良者中途拦截、复制、篡改、删除会给企事业机关单位和个人带来无法弥补的损失, 而且档案信息数字信号传输过程中即使没有遭遇恶意攻击, 但如果存在其它信号干扰, 也会造成关键信息传输失败或传输的现象, 同样会给单位和个人带来一定的损失[3]。针对这一问题, 需要研究一种有效的方法应对档案信息系统中关键信息干扰现象, 建立一个行之有效的关键信息抗干扰模型。

    文献[4]提出并建立了基于循环平稳差异的档案信息系统关键信息抗干扰模型, 该模型将档案系统关键信息转换为数字信号形式, 建立数字信号盲源分离抗干扰模型, 计算档案系统源信号的循环平稳差异性自相关函数, 并根据计算结果获得一个典型极值点用于对比有用信号与干扰信号差异度特征参数的大小, 根据对比结果将系统中关键信息干扰成分滤除掉, 即实现了抗干扰。结果表明, 该模型干扰检测率不高。文献[5]提出并建立了基于多PRI采样的档案信息系统关键信息抗干扰模型, 该模型通过计算档案信息系统关键信息距离密集假目标干扰构建基于了多PRI采样的抗干扰数学模型, 运用独立分量分析法提取系统关键信息回波信号, 同时采用匹配滤料器检测提取获得系统关键信息回波信号峰值, 将高峰值回波信号对应的关键信息滤除, 即可达到抗干扰效果。结果显示, 由于该模型没有较好地实现系统关键信息抗干扰检测, 导致通信误码率急剧增加。

    为了解决上述模型存在的问题, 提出了基于连续均值消除算法的档案信息系统中关键信息抗干扰模型。

    2.3 档案信息系统中关键信息干扰检测

    在实现了档案信息系统关键信息传输信道模型建立和干扰特征提取基础上, 提出并建立了基于连续均值消除算法的档案信息系统中关键信息抗干扰模型, 该模型将系统接收关键信息干扰特征做快速傅里叶变换, 并将变换后的关键信息干扰特征功率谱值与预先设定的干扰检测门限值做比对, 如果变换后的关键信息功率谱值高于预先设定的干扰检测门限值则认为系统关键信息受到了干扰将其滤除;相反则认为系统关键信息没有受到干扰[9,10]。

    通常情况下档案信息系统接收到的关键信息r (t) 由提取获得的有用信息s (t) 、干扰信息i (t) 和高斯白噪声n (t) 三部分构成, 具体计算公式如下

    r (t) =s (t) +i (t) +n (t)     (11)

    将系统接收到的关键信息做快速傅里叶变换可得

    R (n) =∑j′=0Ν-1r (t) ej2πniΝ    (12)

    其中, N表示采样点数;2πn/N表示系统关键信息归一化中心频率;n=0, 1, …, N-1。

    假设系统关键信息干扰频率与频率响应为sin (Nx) /sin (x) 的窄带BP滤波器组中心频率相同, R (n) 即为t=N时刻该滤波器组中第φ个BP滤波器的输出值, 则有

    Νφ (t) =aφ (t) cos[2πfot+φ (t) ]    (14)

    3 仿真测试及结果分析

    为了检验研究提出并建立的基于连续均值消除算法的档案信息系统中关键信息抗干扰模型的抗干扰效果, 设置档案信息系统关键信息载波频率为70MHz, 关键信息采样频率为3600MHz, 基于上述构建的档案信息系统关键信息传输信息模型, 在MATLAB2012a软件环境下搭建仿真平台, 仿真中选取脉冲干扰、扫频干扰、调制干扰和梳状谱干扰四种不同干扰类型 (在表1中分别用①、②、③、④代表) , 分别采用文献[4]模型、文献[5]模型和所提模型在四种不同干扰类型下的检测率和误码率, 结果如图所示。

    从图1～图3的实验结果中可以看出, 文献[4]模型在检测档案信息系统四种不同类型干扰时, 检测率始终较低, 最高没有超过40%;文献[5]模型在检测档案信息系统四种不同类型干扰时, 检测率波动较大, 这可能与四种不同类型干扰强度相关, 在同等迭代次数下, 所提模型的干扰检测率平均在80%左右, 最低没有低于40%, 最高可以达到99%, 充分证明了所提模型只需要较少的迭代次数就能达到较高的检测率。

    4 结束语

    研究针对当前方法检测档案信息系统关键信息干扰特征时存在检测率不高、通信误码率较高的问题, 提出并建立了基于连续均值消除算法的档案信息系统中关键信息抗干扰模型, 通过搭建仿真平台对所提模型进行了性能测试, 结果表明, 由于所提模型在提取了关键信息干扰特征基础上通过设定干扰检测门限值滤除了系统中高于该门限值的关键信息包络, 提高了干扰检测率的同时, 降低了通信误码率, 具有较好的抗干扰性能。
