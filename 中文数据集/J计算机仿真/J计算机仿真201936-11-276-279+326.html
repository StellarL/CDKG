    摘    要：

    针对当前通信网络对等数据节点选取方法中存在的选取的节点数据检索时延较长,以及选取的节点易出现过负载状况等问题,提出多线程电子通信网络对等数据节点选取方法。对多线程电子通信网络及其节点布局进行分析,给出网络建模假设,结合网络内对等数据节点的应用带宽累积分布函数,分析对等数据节点数据接收速率的均匀分布状态,综合节点效益与开销,构建对等数据节点的效用函数;根据网络内各对节点的直接信任值与推荐信任值,进行综合信任评价计算与节点信誉值计算,结合网络内各节点的效用值,在候选节点中选取对等数据节点。实验结果表明,与当前节点选取方法相比,所提方法选取节点的数据检索时延明显缩短,节点负载能够比较稳定的处于均衡状态。

    关键词：

    通信网络; 对等节点; 效用函数; 信誉值; 节点选取;

    Multi-Threaded Electronic Communication Network Peer-to-Peer Data Node Selection Simulation

    ZHANG Zhuo

    Collegeof Humanities & Information Changchun University of Technology

    Abstract：

    In the current communication network peer data node selection methods, the retrieval delay of node data is too long and the selected nodes are prone to the overload. Therefore, this article focuses on a method to select peer-to-peer data node in multi-thread electronic communication network selection. The multi-thread electronic communication network and its node layout were analyzed. Then the network modeling hypothesis was given. Combined with the application bandwidth cumulative distribution function of peer-to-peer data nodes in network, the uniform distribution state of the data receiving rate of peer-to-peer node was analyzed. On the basis of integrating the benefits and costs of nodes, the utility functions of peer-to-peer data node were constructed. According to the direct trust value and the recommended trust value of each pair of nodes in network, the comprehensive trust evaluation and the node reputation value were calculated. After combining the utility values of all the nodes in network, the peer-to-peer data nodes were selected from the candidate nodes. Following conclusion can be drawn from experimental results. Compared with the current node selection method, the data retrieval delay of the proposed method is shortened greatly. Meanwhile, the node load can be relatively stable in equilibrium.

    Keyword：

    Communication network; Peer-to-peer node; Utility function; Reputation value; Node selection;

    1 引言

    对等网络是用于对等者之间任务与负载分配的分布式计算机网络应用架构,是对等计算模型在应用层形成的一种组网或网络形式。多线程电子通信网络能够在同一时段内,通过不同线程实现通信数据的实时传输,该网络技术的应用提高了电子通信网络的运行效率。在多线程电子通信网络运行过程中,受到自身或外界因素影响,会存在部分对等节点离开网络,在此情况下,需要及时选取对等数据节点,及时恢复多线程电子通信网络的正常运行[1]。

    有学者提出了基于代理信任的通信网络对等数据节点选取方法,此方法降低了网络节点的通信开销,但节点选取决策需要以大量的网络节点信任先验知识为基础[2]。还有学者提出了基于自信息理论的对等数据节点选取方法,对于各网络节点对应的维护信息向量,采用加权平均算法进行计算,根据计算结果选取出节点能力与在线时长最优的网络节点作为对等数据节点[3]。针对通信网络节点选取的研究较多,当前节点选取方法的研究中,关注的重点大多在于节点的能力,如存储空间、运行带宽等,忽略了通信网络节点的其它属性。

    针对当前方法存在的不足之处,通过相关研究成果的探究与学习,提出了多线程电子通信网络对等数据节点选取方法:

    1)引入效用函数,分析多线程电子通信网络节点的数据传输效用;

    2)计算各节点的综合信任值与信誉值,结合节点效用,从多线程电子通信网络候选节点中选取合适的节点作为对等数据节点;

    3)通过仿真,检验了该方法的综合有效性。

    2 多线程电子通信网络对等数据节点选取方法

    2.1 多线程电子通信网络对等节点效用函数

    通过对多线程电子通信网络及其节点布局分析,得到对等数据节点的图示如图1所示,对等数据节点的组内数据共享方式,能够减少网络流量消耗,降低通信过程中各节点的数据传输能耗[4]。

    图1 对等数据节点图示   下载原图

    针对多线程电子通信网络的对等数据节点选取问题,为便于构建通信网络系统的数学模型,给出假设内容如下:

    1)多线程电子通信网络的各数据传输节点主要分布在X-Y平面内,每一个数据节点可利用GPS或其它方式获取对应的位置信息,并处于静止状态。

    2)多线程电子通信网络内,各节点的通信半径相同,传输一定量的数据所消耗的能量相同,每一个节点的通信半径内至少存在一个邻节点。

    3)在一定的节点检索时间内,检索结果可通过非层次化多线程电子通信网络内数据处理方式得到,通信数据源节点位置与数据发送速率可利用Directed Diffusion方式获得。

    4)多线程电子通信网络内的各节点具备一定的计算能力,必要时可对通信数据进行处理。

    根据上述假设,多线程电子通信网络可描述为无向图形式,用V=(e,E)表示,通信网络中各节点v∈V,通信网络中两个节点之间的数据传输链路e∈E。假设多线程电子通信网络中节点数量为N,记作v1,v2,…,vN,全部网络节点中,有s个节点可作为对等数据节点,对等数据节点与组外节点间的数据接收速率即为二者之间的通信数据传输速率,设各对等数据节点的通信数据接收速率分别为r1,r2,…,rs,则对应组内对等数据节点的总数据接收速率ds可利用下式计算

    ds=∑i=1sri (1)

    多线程电子通信网络内的s个对等数据节点通过随机、独立选取组外数据通信节点来接收通信数据,通信网络内各对等数据节点的数据接收速率ri为随机变量。根据多线程电子通信网络对等数据节点的应用带宽累积分布函数,构建对应的线性函数,可得到对等数据节点接收通信数据的下载带宽概率密度函数呈现均匀分布[5],综合上述分析可知,对等数据节点的数据接收速率ri为均匀分布。

    多线程电子通信网络内对等数据节点无缓冲情况下,若对等数据节点的综合通信数据接收速率大于等于通信数据源的数据发送速率r,则组内对等数据节点可获取实时通信数据,此时各对等数据节点可取得的收益为G。反之,会出现一定的延迟,多线程电子通信网络对等数据节点的收益为0。

    在通信数据接收速率ds与通信数据源节点数据发送速率r已知的情况下,若多线程电子通信网络节点vi为对等数据节点,则其效用函数表达式如下

    Up,i(s)=Pr(ds≥r)G-Ci (2)

    其中,节点编号s的数值范围为1≤s≤N,Pr(ds≥r)代表对等数据节点获取实时通信数据的概率,Ci代表节点vi为对等数据节点时的开销。Pr(ds≥r)可根据相关定理计算。

    若r1,r2,…,rs在[rL,rU]区间内呈现独立均匀分布时,Pr(ds≥r)的计算公式如下

     Ρr(ds≥r)=12s!∑m=0s(-1)m(sm)×   [(s-m)2-sgn(r^-m)(r^-m)s]         (3)

    m表示[L,U]区间内的对等数据节点数量与节点数量的比,r^表示对等数据节点发送速率平均值,s足够大的情况下,Pr(ds≥r)可简化为以下形式

    Pr(ds≥r)≈Q(r^-s2s12) (4)

    r^=r-srLrU-rL (5)

    Q(x)=∫x∞12πexp-x22dx (6)

    式中,Q(x)为高斯尾函数,此函数可以表示为多线程电子通信网络对等节点效用函数。

    2.2 多线程电子通信网络对等数据节点选取

    分析获得多线程电子通信网络对等数据节点的效用函数后,对通信网络中的各对等数据节点进行综合信任计算与信誉评价,在此基础上,从各候选对等数据节点中选取合适的对等数据节点。

    多线程电子通信网络对等数据节点的信任值计算,包含节点的直接信任值与推荐信任值两部分内容。

    节点的直接信任值可根据评估节点SE及通信数据中继节点SP的过往数据交互情况来计算,通过实例分析,得到节点直接信任值DTSΡSE的计算公式如下

    DΤSΡSE=tSΡSE+1tSΡSE+fSΡSE+2 (7)

    其中,tSΡSE表示通信网络节点SE与SP之间信息成功传递的次数,fSΡSE代表通信网络节点SE与SP之间信息传递失败的次数。当通信网络节点SE与SP之间无信息传递时,tSΡSE和fSΡSE的值均为0,结合实际应用情况,通信网络节点SE对SP的信任与不信任概率均等,此时,节点直接信任值DΤSΡSE=12。

    节点信任值具备动态变化性与时衰特性,在多线程电子通信网络中,两个节点之间长期无信息传递的情况下,通信节点之间的信任度会逐渐降低[6]。通过对多线程电子通信网络的分析,根据信息传递与时间的关系来度量节点之间的信任衰减特性。

    假设t时间内,通信数据成功传输次数为N(t),时间-通信次数影响函数为f(N(t)),其中,N(t)满足条件为:

    1)N(t)的值为不小于0的整数;

    2)当时间s′<t时,有N(s′)≤N(t),N(t)-N(s′)的值等于[s′<t]时段内通信数据成功传输的次数。

    在任一时间t内,多线程电子通信网络中通信节点SE与SP成功进行通信数据传输的次数N(t)服从泊松分布,参数为λt,综合分析可得函数f(N(t))表达式

    f(Ν(t))=Ρ{Ν(t+s´)-Ν(s´)}   =e-λt(λt)kk!         (8)

    λ=E(Ν(t))t (9)

    其中,k代表通信数据传输次数,λ代表单位时间内通信数据传输的平均次数。

    4 结束语

    针对多线程电子通信网络对等数据节点选取问题,在当前节点选取方法的研究基础之上,进行多线程电子通信网络分析,引入节点效用函数,结合节点直接信任度、推荐信任度与信誉值计算,提出了多线程电子通信网络对等数据节点选取方法。通过仿真,与当前节点选取方法进行对比,验证了多线程电子通信网络对等数据节点选取方法的可行性。下一研究阶段,将对该方法的约束性及对等组内的节点异构情况进行深度研究,使其更接近实际应用情况,针对性的进行完善与优化。
