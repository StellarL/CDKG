    摘    要：

    由于每个光纤光栅所反射出波长光存在差异,传统测温系统不能很好的探测得出波长变化情况,导致测温系统存在准确性不高,稳定性不佳等问题,为此,提出并设计了电气设备光纤光栅自动测温系统。分别描述了光纤光栅的结构以及特性、光纤光栅元件自动测温原理。根据上述原理,设计得出测温系统架构,包括数据采集层、数据监测层和远程监控层。分别从硬件设计和软件设计两方面完成系统设计,对于硬件部分,分析了温度传感器模块、传输系统、信号解调系统、PLC以及上位机的功能,采用FTM3501光纤温度变换器,实现电器设备光纤光栅自动测温的报警和显示分析等;对于软件部分,系统利用模块化的设计方式,使每个功能模块间相互联系并且相互独立,共同实现软件功能模块,其中软件的功能模块包括异常报警、系统设置、数据查询、温度显示、数据采集和数据处理等,由此完成系统设计。实验结果表明,所设计系统在自动测温时,温度误差较小,波长变化数据与实际数值拟合情况较好,验证了上述系统的准确性和稳定性,且故障确定耗时短。

    Simulation of Fiber Bragg Grating Automatic Temperature Measurement System for Electrical Equipment

    ZHANG Zi-hao AN Ying

    China Three Gorges University North China University of Science and Technology

    Abstract：

    Because of the difference of wavelength light reflected by each fiber grating, the traditional temperature measurement system cannot detect the wavelength change well, which leads to the problems of low accuracy and poor stability of the temperature measurement system. Therefore, an automatic temperature measurement system for electrical equipment fiber grating is proposed and designed. The structure and characteristics of fiber grating and the principle of automatic temperature measurement of fiber grating element are described respectively. On this basis, the architecture of temperature measurement system is designed, including data acquisition layer, data monitoring layer and remote monitoring layer. The system design is completed from two aspects of hardware design and software design respectively. For the hardware part, the temperature sensor module and transmission system are analyzed. The functions of system, signal demodulation system, PLC and upper computer are adopted to realize the alarm and display analysis of fiber grating automatic temperature measurement in electrical equipment by using FTM3501 fiber temperature converter. For the software part, the system makes use of modularization design method, which makes each functional module contact and independent with each other, and realizes the software function module together, in which the software function module includes abnormal alarm, system setting, data query, temperature display, data acquisition and data processing, etc., thus completing the system design. The simulation results show that the temperature error of the designed system is small when the temperature is measured automatically, and the data of the wavelength change is fit well with the actual value.

    Keyword：

    Electrical equipment; Fiber grating; Automatic temperature measurement; Wavelength;

    1 引言

    电气设备运行时的温度对高压或者超高压配电系统的安全起到制约作用,电气设备系统温度提升[1,2],有两方面原因,一是来源于电阻,二是自然气候对电气设备安全运行的影响,为了使电气设备可以安全运行,需要构建电气设备光纤光栅自动测温系统。在国家不断对电力行业的投入中,电气设备光纤光栅测温的研究受到了广泛关注,通常情况下,电气设备装在封闭开关柜的里面,受空气中氧化的作用以及潮湿环境的影响,还有在检修中或者施工中存在的缺陷,使电气设备接头的电阻变大,出现发热现象,电气设备的接头温度升高,并且不断的恶性循环,还有一部分电气设备因为绝缘缺陷导致发热,不能够对温度进行监测,最终引气电气设备出现接地短路事故,甚至起火[3,4]。电气设备过热时出现的事故,会带来较严重的经济损失,电气设备光纤光栅自动测温的系统设计可以保证电网的安全运行。

    王伟等人提出并设计了基于FPGA的电气设备光纤光栅自动测温系统,根据放大自发辐射光源得到的红外带宽光辐射,对光纤光栅传感器进行照射,利用高斯拟合寻峰方法对反射光的波长进行确定,通过波长的偏移量来判断电气设备的温度,并对其展开了理论分析和验证,实验结果表明,该方法所设计的系统可以高速的调节光纤光栅传感器,但温度误差较大,不能够准确的完成对电气设备光纤光栅的自动测温[5]。卢建中等人提出并设计了基于折反射星敏感器的电气设备光纤光栅自动测温系统,排除应变给传感器带来的影响,设计适用于光纤光栅的封装结构,并测试了光纤光栅温度传感器的振动情况、温度重复性和温度标定,该方法所设计的系统简单快捷,但波长变化数据的拟合情况较差,系统的稳定性差[6]。于丽霞和秦丽提出并设计了改进型光纤布拉格的电气设备光纤光栅自动测温系统,该系统利用差分处理法对光纤采集温度,可以消除掉测温时的随机误差,提高光纤光栅测温的稳定性,并计算了光纤光栅波长偏移量以及与温度变化之间的函数关系,设计新式光纤光栅结构,完成其自动测温系统的设计,该方法所设计的系统能够提高测温的灵敏度,但系统的稳定性较差[7]。

    针对上述研究结果中存在的缺陷,提出并设计了电气设备光纤光栅自动测温系统。

    2 光纤光栅自动测温原理

    光纤被当作传感器的历史悠久,在光通信的快速发展下,光纤受到德国以及日本等国家的广泛应用。光栅处于光纤的内部,并且为波长编码型,与普通的光纤传感不同,有可靠、温度测量稳定的特点,适用在恶劣环境中[8]。

    1)光纤光栅的结构以及特性

    Bragg光纤光栅属于主要器件,是掺杂敏感材料的一种光纤,根据光敏光纤的特殊效应,利用紫外激光从侧面进入,可以增加光纤芯的折射率,构成具有折射率周期变化的结构,如图1所示。

    图1 光纤光栅结构   下载原图

    根据透射光、入射和反射光和与其有关联的能量等条件,可以得到其波长的变化情况。结合光纤的轴向应变与光纤光栅的波长变化两者之间的关系,能够构成物理量之间(例如电压、电流、速度、位移等)的函数关系[9]。

    光纤光栅的性能十分优良,并且具有较高的反射率,带宽窄可以小于0.2nm,当其在符合相位匹配条件的情况下,入射光受到反射,其对应峰值的反射波长表达式如下

    λB=2neffΛ (1)

    其中,Λ表示光栅周期,neff表示有效折射率,假设轴向应变为ε,温度为T,则其波长的相对变化量为

    ΔλB/λB=(α+ξ)T+(1-Pe)ε (2)

    式中,α、ξ分别代表光纤光栅的热膨胀系数以及热光系数,通常情况下,α=0.55×10-6,ξ=8.3×10-6,其光纤光栅温度的灵敏度约为0.0136nm/℃,Pe代表有效光弹系数,根据上式对波长的变化量进行测量,可测出电气设备光纤光栅的温度和应变量。

    2)光纤光栅元件自动测温原理

    为了提高其测温的灵敏度,将光纤封装在增敏材料的基座中[10,11],温度的灵敏度可以表示为

    MT=((α+ξ)+(1-Pe)(αj-α))λB (3)

    其中,αj表示基座的热膨胀系数。

    3 自动测温系统设计

    3.1 自动测温系统架构

    光纤光栅自动测温系统是在一根光纤中将不同的光栅常数串接在一起,宽带光源发出的宽带将经过Y型分路器,并且通过全部的光纤光栅,每个光纤光栅所反射出波长的光存在差异,通过Y型分路器中的另外一个端口耦合调仪。当电气设备的温度发生变化时,经过解调仪可以探测出波长的变化情况,能够得知被测电气设备光纤光栅的情况[12]。

    与电气设备温度监测的要求相结合,电气设备光纤光栅自动测温系统的构架如图2所示。

    图2 自动测温系统的构架   下载原图

    可以将该系统的构架分为三层,分别为:

    1)数据采集层

    是指分布在电气设备的接头表面或者电缆桥架中的传感器对其表面温度的采集;

    2)数据监测层

    是指经过传输光缆将所采集到的数据传输到信号处理器上,可以实现电气设备光纤光栅温度的在线监测;

    3)远程监控层

    对监测得到的温度数据进行打包,再经过网络发送给计算机监控中心,可以实现对电气设备光纤光栅温度的远程监控。

    3.2 系统的硬件设计

    电气设备光纤光栅自动测温系统的硬件结构如图3所示。

    图3 自动测温系统的硬件结构   下载原图

    该系统硬件采用FTM3501光纤温度变换器,该变换器属于光纤温度解调模块,并且是单通道的,可以输出4～20mA的工业标准电流,也可以提供给计算机或者数据采集器处理,实现光纤光栅自动测温低费用的功能。FTM3501中有ST多模光纤接口,能够支持OPTIC3000以及OPTIC3001温度传感器,两者的区别是有不同的温度量程,利用多模光纤来连接变换器和电气设备的温度传感器,距离大约为50m。电气设备光纤光栅自动测温系统利用S7-300系列CPU312,该系列属于小型PLC,可以应用在快速处理过程中或者小型的自动化控制系统。S7-300系列PCL都是利用模块化结构,模块和机架共同组成S7-300系列PCL,电气设备光纤光栅自动测温系统按照实际需要测量8通道的3个模拟量模块,16通道的1个继电器输出模块以及以太网的通讯模块CP343-1。FTM3501将经过转换之后的电流信号发送给PLC,利用PLC处理数据,可以实现报警功能。同时CP343-1将数据传输到PC机中,实现电器设备光纤光栅自动测温的报警和显示分析等。

    经过上述分析,实现了自动测温系统硬件部分的设计。

    3.3 系统的软件设计

    该系统利用模块化的设计方式,每个功能模块间相互联系并且相互独立,可以共同实现软件的功能。该设计方式有以下几个优点:系统在发生故障时,开发者能够按照故障模块对其位置进行确定,大大节省了处理时间,并且不会给系统的其它功能带来不良影响。

    通过对系统软件的需求建模和分析,并与用户建议和实际情况相结合,详细的设计和划分了自动测温系统的软件功能模块如图4所示。

    图4 系统的软件功能模块   下载原图

    该电气设备光纤光栅自动测温系统的软件功能模块分别有异常报警、系统设置、数据查询、温度显示、数据采集和数据处理等。每个模块中的子模块不同,子模块的功能为该系统软件所实现的功能,其中,系统软件功能模块的配置只对管理员开放,其它模块能够对普通用户和管理员同时开放权限,最终实现了系统软件部分的设计。

    图6 温度误差对比结果   下载原图

    本文提出并设计了电气设备光纤光栅自动测温系统,主要描述了自动测温系统的工作原理,并相应的设计了光纤光栅自动测温系统的整体架构、硬件和软件部分。分别测试了该自动测温系统所测得的温度误差以及波长变化数据的拟合情况,实验结果验证了系统的准确性和稳定性,提出方法所设计的系统适用于对电气设备容易出现故障处进行温度监测,该系统也可以广泛应用在电力领域的其它研究中。
