    摘    要：

    通过对智能家居系统进行近距离无线自动控制, 可以提高用户的居住舒适度和满意度。进行家庭近距离无线自动控制时, 需要获取系统控制量的输出误差, 并对其进行实时调整完成无线自动控制, 而传统的控制方法多是采用模糊PID控制方法对系统控制量进行在线控制, 忽略了系统控制量的输出误差, 降低了控制精度, 数据包接收成功率也较低。针对上述问题, 提出一种基于免疫优化的智能家居系统家庭近距离无线自动控制方法。建立智能家居系统模型, 以模型为基础各个执行器节点由相应的控制器控制, 控制器根据智能家居系统误差的变化实时调整参数, 得到系统控制量。将智能家居系统控制量作为操作变量, 利用PID神经网络算法对系统控制量进行无线自动控制。针对PID神经网络连接权值易陷入局部最优的问题, 采用免疫算法优化连接权值, 实现系统家庭近距离无线自动控制。实验结果表明, 所提方法数据包接收成功率可达到100%, 可实现近距离无线自动控制, 能够为用户提供舒适、方便的智能家居环境。

    Smart home system home proximity wireless automatic control simulation

    GUI Jing-yi

    School of Electrical & Electronic Information Engineering Hubei Insitute of Technology

    Abstract：

    By performing close-range wireless automatic control of the smart home system, the user's living comfort and satisfaction can be improved. When performing home short-range wireless automatic control, it is necessary to obtain the output error of the system control quantity, and adjust it in real time to complete the wireless automatic control. The traditional control method mostly uses the fuzzy PID control method to control the system control quantity online, ignoring the output error of the system control quantity reduces the control precision and the packet reception success rate is also low. Aiming at this problem, this paper proposes a home proximity wireless automatic control method based on immune optimization for smart home system. A smart home system model is established. Based on the model, each actuator node is controlled by a corresponding controller. The controller adjusts the parameters in real time according to the change of the error of the smart home system, and obtains the system control quantity. The control quantity of the smart home system is used as an operation variable. The PID neural network algorithm is used to wirelessly control the system control quantity. For the problem that the PID neural network connection weight is easy to fall into the local optimum, the immune algorithm is used to optimize the connection weight to realize the system home short-range wireless automatic control. The simulation results show that the proposed method can achieve 100% success rate of data packet reception, and can realize close-range wireless automatic control, which can provide users with comfortable and convenient smart home environment.

    Smart home system; Family close range; Wireless automatic control; Immune algorithm;

    1 引言

    计算机网络通讯技术的飞速发展, 社会生活越来越智能[1]。在此基础上, 无线网络逐渐兴起, 并应用在智能家居系统中, 将与家庭生活相关的各个子系统与网络综合智能管理平台有效联系起来[2], 其家庭近距离无线自动控制的实现有助于提升智能家居系统的性能, 打造更舒适的家居环境[3]。当前阶段实现智能家居控制是建立在多传感器基础上, 在应用平台上进行家庭近距离无线自动控制, 但普遍存在控制精度不高的问题。针对上述问题, 研究智能家居系统家庭近距离无线自动控制具有重要意义。

    国内学者对智能家居系统控制方案进行了大量研究, 提出了很多好的方法。例如史斌斌等人[4]提出一种基于真假值的家居系统控制方法。构建针对家居电器的描述模型, 在此基础上采用TF算法对描述模型进行自动控制。该方法的缺陷是当室内环境发生改变时, 易造成目标点位置也发生改变, 此时需要重新调整控制其参数才能达到较好的控制效果。刘桂涛等人[5]提出基于灰色理论的家居系统在线控制方法。构建智能家居系统实时控制模型, 采用模糊PID控制方法对系统控制量进行在线控制。该方法控制响应性较强, 但忽略了系统控制量的输出误差, 造成控制精度有所降低。

    针对上述方法存在的不足, 将神经网络算法和免疫优化算法相融合, 应用于系统的不同执行器节点的控制器, 构成了一种基于免疫优化的智能家居系统近距离无线自动控制方法。实验结果表明, 所提方法控制下的智能家居系统具有较好的适应性, 控制精度较高。

    2 智能家居系统家庭近距离无线自动控制原理

    在进行智能家居系统家庭近距离无线自动控制过程中, 建立智能家居系统模糊控制器阶段, 引入加权因子, 将系统控制量偏差以及控制响应误差绝对值积分的和视为目标函数, 通过粒子群算法通过种群迭代搜索获取最优加权因子, 调整模糊控制规则实现家庭近距离无线自动控制, 具体过程如下所述:

    其中, X和Y分别表示系统状态变量、输出变量, u表示智能家居系统输入控制变量, E表示系统能耗, w0表示系统扰动信号, A表示传感器数据。

    假设, 种群中某个粒子在n维空间中搜索, xi= (xi1, …, xin) 表示粒子在空间中的位置, 描述问题的一个解, vi= (vi1, …, vin) 表示种群中粒子i当前搜索速度, 以下给出种群中个体迭代过程中的速度与位置更新式

    其中, xid (k) 表示种群中某个粒子i在第k次迭代过程中的位置, vid (k) 表示种群中某个个体i在第k次迭代中的搜索速度。Pi (k) 表示种群中个体i当前最优位置, Pg (k) 表示种群当前搜索阶段全局最优位置, c1和c2分别表示学习因子, w、k表示种群惯性因子和迭代次数, Nr1、Nr2表示区间[0, 1]内的随机数。

    利用系统控制量偏差和偏差时间误差绝对值积分之和作为粒子群算法[6]优化系统模糊控制器的性能指标

    Sw (u) =∑j=1kW (xi) /F    (6)

    其中, W (xi) 表示能够改变系统输出偏差的权重。

    根据式 (6) 求解出的最优加权因子调整模糊控制规则实现智能家居系统家庭近距离无线自动控制

    C˜=∑j=1kpiΦ (xi)     (7)

    其中, Φ (xi) 表示智能家居系统模糊控制器的输出上限值, pi表示传感器节点所测参量。

    3 基于免疫优化的系统无线自动控制方法

    3.1 获取系统控制量

    智能家居系统控制器的设计以及家庭近距离无线自动控制均需要以被控系统模型为依据, 对仿真平台进行建模。系统中每个执行器节点均由相应的控制器控制, 控制器根据智能家居系统误差的变化实时调整参数, 得到系统控制量, 具体过程如下所述:

    设定M′、N′分别表示智能家居系统中执行器、传感器节点的数量, aι和sη分别表示执行器节点和传感器节点。U=[u′1, …, u′M′]T表示智能家居系统控制器输出, Y=[y′1, …, y′N′]T表示目标传感器节点所得测量值, 由该值可推导出第η个传感器节点处的测量值, 即

    y′η (k) =∑ι=1Μ′φηι (u′ι (k) ) +Y⋅dη (k)     (8)

    其中, φηι (u′ι) 表示正常家居家居环境下, 系统控制器输出为u′ι时, 第η个传感器节点的测量值, dη (k) 表示传感器节点η测量的光照强度。考虑到家居系统模型的结构, 各个传感器节点的测量值受到执行器节点的影响, 在仿真阶段仅需考虑距离较近的执行器对该节点测量值的贡献。将传感器节点sη所在区域的执行器节点的集合描述为Azη, 在仿真阶段将系统模型式 (8) 转换为

    y′η (k) =∑aι∈Azηφηι (u′ι (k) ) +dη (k)     (9)

    智能家居系统传感器节点处于中心位置, 执行器节点到任意一个传感器节点之间的距离都是相同的, 即φηι (u′ι) 相同, 将系统传感器节点sη放置在区域中心, 在室内环境不发生改变的情况下, 标记系统传感器节点的测量值, 将测量结果通过线性拟合[8], 智能家居系统家庭近距离自动控制目标是将被控设备的测量值达到设定值R=[r1, …, rN′]T, 即

    minJ (k) =∑η=1Ν′ (rη-yη (k) ) 2    (10)

    智能家居系统传感器节点sη的测量值yη与设定值rη进行对比可获得误差eη, 将该结果传送给执行器节点, 根据执行器节点与传感器节点之间的相对位置关系, 将系统控制任务一一分配给各个执行器节点, 区域zeη中的执行器节点aι将系统控制任务划分至区域η的误差为

    eηι=αηιeη    (11) 0≤αηι≤1, ∑aι∈zeηαηι=1    (12)

    其中, αηι表示控制任务分配函数。

    从系统执行器角度分析, 执行器节点不仅仅属于同一个区域, 设定执行器节点aι所处区域数量为Atι, 相应的会有Atι个区域误差, 设定智能家居系统执行器节点控制器的输入为

    eι=∑η=14m′eηι    (13) m′={1, aι2属于区域zeη0, aι不属于区域zeη    (14)

    智能家居系统控制器输出结果uι是依据执行器节点所在的全部区域的误差计算获得, 为了均衡不同区域间存在的矛盾, 利用折中思想[9], 系统控制量为全部控制分量的均值, 即

    u′ι=uι/Atι    (15)

    获取系统控制量的输出误差, 并对其进行实时调整, 在一定程度上可有效提高控制精度。

    5 结论

    针对当前智能家居系统控制方法存在的控制精度不高的问题, 提出一种基于免疫优化的智能家居系统家庭近距离无线自动控制方法。将PID控制算法与神经网络算法相融合对智能家居系统进行近距离无线自动控制, 在实际应用过程中发现神经网络连接权值易陷入局部最优, 针对此问题, 采用免疫优化算法对连接权值进行优化。实验结果表明, 所提方法系统控制成功率可达到100%, 且能够满足家庭近距离无线自动控制。
