    摘    要：

    对水印图像进行抗攻击隐形优化处理, 能够有效保护数字图像版权保护, 对水印图像进行隐形优化处理, 需要对高光谱水印图像中的第一谱段进行预测, 并对像素预测参考值进行修正。传统方法建立高光谱水印图像编码块的边信息, 并对边信息选择最优的预测阶数, 但忽略了修正像素预测参考值, 导致处理过程耗时长。提出基于多波段预测的水印图像抗攻击隐形优化处理方法, 采用中值滤波器对高光谱水印图像中的第一谱段进行预测, 消除谱间冗余;采用线性预测根据第一谱段内像素对当前谱段像素进行预测, 获得当前像素预测参考值, 建立上下文模型对像素预测参考值进行修正。采用8级查表搜索预测算法对水印图像各谱段像素进行预测, 获得最终图像像素预测值, 利用熵编码算法对该预测值嵌入编码获取水印图像抗攻击隐形优化处理结果。实验结果表明, 所提方法处理过程运算时间短, 且处理后的图像质量较优。

    关键词：

    多层嵌入; 水印图像; 隐形; 线性预测;

    Simulation of Stealth Optimization of Watermark Image Against Attack

    XIN Chen CUI Bing-de WANG Yi TIAN Zhi-min

    Computer Department, Hebei University of Water Resources and Electric Engineering

    Abstract：

    Traditional method establishes the edge information of coding block in hyper-spectral watermark image and selects the optimal prediction order of side information, but ignores the corrected pixel prediction reference value. Therefore, a method of invisible optimization for attack-resistant of watermark image based on multiband prediction was presented. First of all, the median filter was used to predict the first spectral coverage in hyper-spectral watermark image and eliminate the inter-spectral redundancy. Then, linear prediction was used to predict the current spectral pixel according to the pixel in the first spectral coverage, and the reference value of current pixel prediction was obtained. In addition, the context model was built to correct the reference value of pixel prediction. Meanwhile, level-8 search table prediction algorithm was used to predict the pixel of each spectral coverage in the watermark image, so as to obtain the final prediction value of image pixel. Finally, the entropy coding algorithm was used to embed the coding into predictive value. Thus, we could obtain the result of invisible optimization of attack-resistant of watermark image. Simulation results show that the operation time of proposed method is short. Meanwhile, the image quality after processing is better.

    Keyword：

    Multi-layer embedded; Watermark image; Invisible; Linear prediction;

    1 引言

    高光谱图像因其在地物分类以及识别上具有的特殊能力, 在农业、医疗以及军事等方面有着广泛的应用[1,2]。在图像传输过程中, 需要对图像进行水印处理, 以提高图像的抗攻击能力。图像质量的不断提高, 相应的数据量也快速增长, 这对图像存储和传输无疑是一个巨大的挑战[3]。高光谱图像在上述应用领域中是具有重要价值的信息, 对其进行水印, 可能引起的图像数据失真, 对后续图像识别与处理产生不利影响。在这种情况下, 研究高效的水印图像抗攻击隐形优化处理方法势在必行[4,5]。

    现阶段, 相关专家学者在水印图像抗攻击隐形优化处理方面已经取得了很多好的研究成果, 例如文献[6]提出一种基于分布式信源编码的水印图像抗攻击隐形优化处理方法。采用多元线性回归模型建立水印图像编码块的边信息, 对边信息选择最优的预测阶数, 依据水印图像预测阶数实现抗攻击隐形处理, 这种方法处理耗时较长。文献[7]提出一种基于串匹配的水印图像抗攻击隐形优化处理方法。对水印图像, 以像素为搜索单位, 对图像像素未匹配串长度以及匹配后偏移程度等参数进行编码, 完成水印图像抗攻击隐形优化处理, 但影响图像的传输质量。

    针对上述问题, 在串匹配方法的基础之上, 提出一种基于多层嵌入的水印图像抗攻击隐形优化处理方法。

    2 水印图像抗攻击隐形优化处理原理

    在水印图像抗攻击隐形优化处理原理中, 运用K均值聚类方法对水印图像矢量进行划分, 将相同类别矢量划分至相同聚类群中, 对各类光谱矢量聚类群依据最小二乘法进行预测, 去除图像光谱矢量空间冗余。对预测误差图像进行转换, 采用算术编码器对转换后的图像序列以及转换相应关系进行水印, 完成水印图像抗攻击隐形优化处理结果。具体步骤如下:

    1) 确定高光谱图像光谱矢量, 聚类群的数目, 结合先验可知, 当确定16个光谱矢量聚类群时, 可以获得最佳水印效果。

    2) 对光谱矢量群聚类中心中的元素进行线性修正。

    3) 将光谱矢量修正后的图像划分至其近邻的矢量聚类群中, 利用式 (1) 描述高光谱图像矢量之间的距离

    d (x, y) =∑i=1l (xi-yi) 2    (1)

    其中, xi和yi分别表示图像中某条光谱矢量, l用于描述图像光谱波段数。

    4) 对待水印的高光谱图像中多段光谱矢量完成一次聚类后, 将重新生成新的矢量聚类群中心点。

    5) 重复运行步骤3) 和4) 。

    3 水印图像抗攻击隐形优化处理方法

    3.1 谱内预测及修正

    采用中值滤波器模型对水印图像第一谱段进行预测, 消除该谱段空间相关性及谱间相关性。采用线性预测算法根据第一谱段预测结果对当前谱段像素进行预测, 对预测残差值建立上下文预测模型进行预测修正。具体过程如下所述:

    假设, yn、yw、ynw表示水印图像当前像素y相应的三个近邻的像素, 利用下式给出y的预测值

    y={min (yn, yw) , ΝW≥max (yn, yw) max (yn, yw) , ΝW≤max (yn, yw) yn+yw-ynw, 其他    (12)

    为使式 (12) 获得的预测残差值为最小, 利用二阶线预测器[9]对其进行预测

    x^n, i, j=an-1⋅xn-1, i, j+bn-1    (13)

    其中, xn-1, i, j用于描述水印图像中第n-1波段第i行第j列像素的灰度值, n, i, j用于描述由第一谱段像素xn-1, i, j预测得到的与第n个波段相同位置上像素对应的灰度值, an-1和bn-1分别用于描述光谱图像像素值的预测系数。

    假设, M×N用于描述水印图像大小, 利用下式计算水印图像预测误差的平方值表示为

    εn2=1ΜΝ∑i=0Μ-1∑j=0Ν-1 (xn, i, j-an-1xn-1, i, j-bn-1) 2    (14)

    在满足水印图像预测均方根误差为最小的约束条件下, 最后解出an-1和bn-1的值

    {an-1=r (n, n-1) -u (n-1) u (n) r (n-1, n-1) -u2 (n-1) bn-1=u (n) -r (n, n-1) -u (n-1) u (n) r (n-1, n-1) -u2 (n-1) u (n-1)     (15)

    采用线性预测, 依据上一谱段预测结果, 可获得当前谱段像素的参考预测值, 将该值输入至构建的上下文模型中, 对其进行重新排列编码, 可获得当前谱段像素的预测值。

    3.2 基于多层嵌入的水印图像抗攻击隐形处理

    在多层嵌入的水印图像进行抗攻击隐形处理过程中, 预测器使用3.1节获得的预测值, 利用LUT搜索预测方法获得8个LUT预测值, 将上一节获得的参考预测值与其进行对比, 获得最终的高光谱图像预测值, 对预测差值进行熵编码, 完成水印图像抗攻击隐形优化处理。具体过程如下所述:

    为了提高水印图像像素搜索速度, 采用LUT预测搜索算法[10]对其进行搜索, 采用8个LUT预测器对图像像素进行预测更新, 具体过程描述如下。

    假设, fmax表示LUT长度, 由3.1节求解获得的像素f (x′, y′, z′) 相应的参考预测值为P′ref, 利用前一谱段的像素预测值f (x′, y′, z′) 可预测当前谱段的像素预测值f′ (x′, y′, z′) , 即:

    f′ (x, y, z) =f′min{ε1, ε2, ε3, ε4, ε5ε6} (x, y, z)     (16)

    式中

    {ε1=agrmin{|Τ1 (z) [f (x, y, z-2) ]-Ρ′ref|}⋮ε6=agrmin{|Τ6 (z) [f (x, y, z+1) ]-Ρ′ref|}    (17)

    {Τ2 (z) [f (x, y, z-2) ]=Τ1 (z) [f (x, y, z-2) ]⋮Τ6 (z) [f (x, y, z+1) ]=Τ5 (z) [f (x, y, z+1) ]    (18) {Τ1 (z) [f (x, y, z-2) ]=f (x, y, z) ⋮Τ5 (z) [f (x, y, z+1) ]=f (x, y, z)     (19)

    最后将P′ref、Τ1 (z) [f (x, y, z-2) ]、Τ2 (z) [f (x, y, z-2) ]、Τ3 (z) [f (x, y, z-1) ]、Τ4 (z) [f (x, y, z-1) ]、Τ5 (z) [f (x, y, z+1) ]进行对比, 获得最终的高光谱图像当前谱段像素预测值。综上所述, 完成水印图像抗攻击隐形优化处理。

    分析表1和表2结果可知, 基于串匹配的水印图像抗攻击隐形优化处理方法取得了对比方法中最差的图像水印比, 并不适用于高光谱图像。基于分布式信源编码的水印图像抗攻击隐形优化处理方法依据图像预测阶数实现优化处理, 相比基于串匹配的水印图像抗攻击隐形优化处理的计算时间较短, 但水印比结果仍然不尽人意。而所提基于多层嵌入的水印图像抗攻击隐形优化处理方法, 第一阶段采用线性预测根据第一谱段内像素对当前谱段像素进行预测, 获得当前像素预测参考值;建立上下文模型对像素预测参考值进行修正。第二阶段采用LUT搜索预测方法获得8个LUT预测值, 并将其结果与参考预测值进行对比, 将最终预测值进行熵编码, 完成水印图像抗攻击隐形优化处理。充分利用了水印图像多层谱间相关性, 使得所提方法在水印计算时间上优于其它两种图像处理方法, 验证了所提方法在获取图像高水印性能的同时保持了较低的处理计算时间。

    5 结论

    提出一种基于多层嵌入的水印图像抗攻击隐形优化处理方法, 第一阶段, 采用中值滤波器对高光谱图像中的第一谱段进行预测, 消除谱间冗余, 减少了图像水印数据量, 提高了水印性能。第二阶段, 依据第一谱段内像素预测结果对当前谱段进行线性预测, 获得当前像素预测参考值, 针对预测参考值的误差问题构建上下文模型对其进行修正, 提高了平均水印比。第三阶段, 采用LUT搜索预测方法对第二阶段修正后的预测参考值进行对比, 获得最终的图像像素参考值, 对该参考值进行熵编码, 完成水印图像抗攻击隐形优化处理, 进一步提高了图像的抗攻击能力。
