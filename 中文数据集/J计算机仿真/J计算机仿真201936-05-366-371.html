    侯增选 张微 黄磊 史有志

    大连理工大学机械工程学院

    摘    要：

    为了增强人机交互的实时书法体验, 获得理想的行书风格, 引入力反馈技术, 提出基于力反馈技术的行书笔划判别方法, 从而实现行书书法仿真。对大量行书特征进行了分析, 构建行书笔划连接件及笔段模型的特征库, 建立了一套行书笔划判别规则对特征库进行调用。最后介绍了虚拟绘制系统, 将Phantom Desktop力反馈设备与所搭建的软件系统结合, 实现了在虚拟系统中的行书风格化绘制。仿真结果表明, 该方法能够获得较好的行书书法虚拟绘制效果。

    关键词：

    行书特征库; 笔划判别规则; 虚拟绘制系统;

    Running Script Calligraphy Simulation Based on Force Feedback Technology

    HOU Zeng-xuan ZHANG Wei HUANG Lei SHI You-zhi

    College of Mechanical Engineering, Dalian University of Technology

    Abstract：

    In order to enhance the real-time calligraphy experience of human-computer interaction and get the ideal style of writing script, this paper introduced the force feedback technique and put forward a method of determining the stroke line of the script based on the force feedback technique to achieve running script calligraphy simulation. In this paper, a large number of characteristics of line writing books were analyzed to construct the feature library of line drawing stroke connection and pen segment model, and a set of line-of-sight stroke identification rules was established to call feature library. At last, the virtual rendering system was introduced. The Phantom Desktop force feedback device was combined with the software system to realize the stylized drawing of the running script in the virtual system. The simulation results show that this method can get a good virtual drawing effect of calligraphy.

    Keyword：

    Semi-cursive script feature library; Discriminant rules; Virtual rendering system;

    1 引言

    中国书法历史悠久, 源远流长, 从甲骨文、钟鼎文衍变成秦篆、隶书再到当今的行书、草体、正楷等, 散发着独有的艺术魅力[1]。伴随虚拟绘制技术逐渐的发展, 中国书法的计算机仿真随之也成为了研究热点。

    Ip和Wong[2,3]运用参数化方法合成书法作品, 而后改良了原有的毛笔模型, 仿真出各种书法风格;朱墨子等[4]以统计分析法为理论基础, 构建虚拟毛笔模型, 可仿真出纸面与毛笔的交互区, 书写效果良好, 但不能实现笔头部分实时的形变;郭丽等[5]结合楷书字体的形状以及在运笔中毛笔笔头形变, 采用贝塞尔曲线建立笔划模型, 并对笔划的边缘进行了过渡处理。但书法与书画不同, 讲求笔路与笔法, 得以形成迥异的书法风格。为实现人机实时交互书法体验及优质的书法风格化效果, 本文选取中国书法中极具代表性的行书, 提出基于力反馈技术的行书笔划特征实时判别方法。文中通过对大量的汉字笔划特征分析及总结规律, 建立一套行书笔划判别规则对特征库进行调用, 利用硬件软件开发平台, 并验证其系统功能。

    2 特征库的构建

    通过观察大量汉字, 根据转折点有无的情况将笔划分成基本笔划与复合笔划。没有拐点的笔划称作基本笔划, 其中有点、提、撇、捺、横、竖;复合笔划为含拐点的笔划, 可视为基本笔划与转折处连接的组合, 如竖提、横折钩、竖弯钩等[6]。

    将笔划打断为连接件和笔段, 连接件为笔划的笔头、笔尾及转折处, 笔段为连接件之间的笔划部分, 如图1所示。

    图1 连接件与笔段   下载原图

    笔划 = 起笔连接件+ k* (笔段+连接件) +笔段+ n*收笔连接件[7], 其中:k≠1;n=0或1。

    在实现虚拟书法实时行书化的过程中, 分别需要构建行书笔划连接件、笔段特征库, 笔划判别及行书风格化重构。

    在行书特征库的建立过程中, 采用贝塞尔曲线来绘制出行书的各个连接件、笔段和连笔的曲线轮廓, 以便于在虚拟行书绘制中调用, 其所具有的参数为默认方向、端口宽度等。同时, 可存储轮廓曲线的控制点。

    3 笔划判别及行书风格化

    3.1 基本笔划判别及行书化

    使用者运用力反馈设备来控制虚拟毛笔进行书写, 在毛笔与平面分离时, 该笔划记为笔1, 继续运笔进行书写, 抬笔, 再次与平面分离, 该笔划记为笔2, 以此类推。此时由搭建的虚拟绘制系统结合两笔情况按照相应规则对笔1或笔2的笔划进行判别并进行行书化风格重构。具体判别规则如下:

    1) 首先由笔划的折弯度判别笔1是否含拐点, 判别其为基本笔划或复合笔划;

    2) 对于基本笔划, 通过轨迹点数判别。若采样点的数目大于点、提的给定值, 则非点提笔划, 若小于或等于设定值, 认定为点、提, 再根据始末向量判别点还是提, 矢量方向在0°-90°时为提, 180°-360°时为点, 再根据笔2的相对位置矢量判定为点a还是点b, 若2.3秒内无下一笔, 则为点c。

    3) 对于非点提的基本笔划, 根据始末向量规则判别其矢量在哪个角度范围, 即对应相应的基本笔划, 基本笔划调用规则如图4。

    4) 每一笔结束后, 2.3秒 (经验值) 内若有下一笔, 则若下一笔的相对位置矢量在0°-180°, 记下一笔起始点位置为上方, 180°-360°, 记下一笔起始点位置为下方, 左右方的范围则为-90°-90°及90°-270°, 由于左右方判定与上下方判定是针对不同笔划, 则判断区间不会干涉;若2.3秒内无下一笔, 则记为无下一笔。

    5) 对笔1和笔2的矢量进行判断, 若两者基本平行 (即两笔划夹角在设定值内) , 则认定笔1和笔2为平行笔划;当书写完笔3时, 对笔2和笔3两笔划夹角进行判断, 若在设定值内, 则判定笔2与笔3亦为平行笔划, 以此类推。反之, 为非平行笔划。平行笔划的调用规则见图5。

    6) 最后, 根据相应的调用规则从行书风格特征库调用相应的连接件和笔段, 进而实现笔划的行书风格处理。

    3.2 复合笔划判别及行书化

    当某笔划被判定为复合笔划时, 记录拐点数量。复合笔划以拐点分界, 由若干连接件和笔段构成。

    当某一复合笔划拐点数为1时, 可将之拆分为两个基本笔划, 对所拆分的两部分采用上文基本笔划 (非平行) 判别规则。同时记录第二段基本笔划采样点个数, 若小于给定值, 则第二段基本笔划为钩, 再由前一部分基本笔划判定结果判定横钩、斜钩或竖钩。随后在已构建的行书特征库中按照图6中复合笔划调用规则选取相应连接件、笔段等进行行书化处理;相反, 若个数大于给定值, 则此处为折, 再根据第一笔和第二笔笔划的笔划夹角、矢量方向及采样点数比例判定属于三种折中的哪一种, 继而按照图6中调用规则对该复合笔划进行行书化处理。并定义采样点比例为拐点前笔划采样点数与拐点后点数的比值。

    当复合笔划含拐点多于1个时, 可认为拐点将之划分为多个基本笔划, 按上文判定基本笔划及最后一部分基本笔划采样点个数, 最后一部分是否为钩。针对竖弯钩有特殊情况需要说明, 若笔划矢量为先↓再→后↑, 并判别出最后一笔为钩, 则该笔划为竖弯钩, 另要根据其第一个拐点处的采样点比例, 若值在0.2-0.5之间, 则调用竖头a+竖弯钩笔段a+钩a, 若值在0.5-0.8之间, 则调用规则为竖头a+竖弯钩笔段b+钩a, 如图7。

    4.2 仿真结果

    使用本文中的系统进行行书化绘制的具体过程如下:

    首先, 进行系统初始化, 使用者通过操控力反馈设备控制虚拟毛笔与二维平面接触, 使毛笔发生弹性变形。

    然后, 使用者通过力反馈设备控制虚拟毛笔进行书写, 通过相应的笔道书写算法获取完整笔道, 系统通过对采样点信息、毛笔变形大小、压力及摩擦力信息的收集得出书写位移、反馈力, 并将反馈力反馈给使用者, 再控制笔道宽度。笔道生成后, 系统通过行书笔划判别规则对笔划类型进行判别, 调用行书特征库的连接件及笔段, 进行行书风格化处理。

    最后, 如果获得了使用者满意的效果, 可将绘制结果保存, 结束绘制。
