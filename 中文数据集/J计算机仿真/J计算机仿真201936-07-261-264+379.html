    摘    要：

    运用传统方法对相依网络安全态势进行评估的过程中存在评估结果不准确, 效率低等问题, 为此提出一种基于攻击模式识别的相依网络安全态势评估方法。结合网络安全态势攻击信息中的态势值关系得到BP神经网络相应节点, 从态势信息中的关系经过多次迭代推算得到态势值, 根据态势值结合布谷鸟搜索算法获取BP网络最佳权值和阈值, 将结果赋予目标网络态势评估模型中, 得到最优网络态势评估模型。基于一种攻击识别的安全评估方法, 对网络信息进行融合识别, 根据识别出的信息进行网络安全评估量化, 结合网络状态转移图, 实现网络安全的预测, 达到评估效果。通过实验结果验证, 所提评估方法与传统评估方法相比, 评估结果更准确、效率更高, 上述方法的实际应用性较高。

    关键词：

    灰色信息攻击; 网络安全; 安全态势评估; 攻击模式识别算法;

    Simulation of Interdependent Network Security Assessment Based on Gray Information Detection

    Zhang Yu

    College of Physics & Electronic Information, Yan'an University

    Abstract：

    In this article, a method to evaluate the security situation of interdependent network based on attack pattern recognition was proposed. Based on the relation of situation value in network security situation attack information, the corresponding nodes of BP neural network were obtained. Through multiple iterations, the situation value was calculated from the relationship in the situation information. According to the situation value and cuckoo search algorithm, the best weight and threshold of BP network were obtained. Then, the result was given to the situation evaluation model of target network, so that optimal network situation assessment model was obtained. Based on the method of security evaluation for attack identification, network information was fused and identified. According to the identified information, network security evaluation was quantified. Combined with network state transition diagram, network security prediction was achieved. Thus, the evaluation effect was realized. Simulation results show that, compared with the traditional evaluation method, the proposed evaluation method has more accurate assessment result and higher efficiency. Meanwhile, the practical application of this method is higher.

    Keyword：

    Gray information attack; Network security; Security situation assessment; Attack pattern recognition algorithm;

    1 引言

    随着互联网技术的发展, 一些网络安全漏洞开始出现, 各式各样的网络安全事故频繁发生对个人和国家都造成了极大的损失, 在这种情况下对网络安全态势进行评估[1]显得尤为重要, 然而传统的评估方法存在评估结果不准、CPU占用过高的问题, 针对上述问题, 如何有效、精准的对灰色信息攻击下网络安全态势[2]进行评估是十分重要的研究课题。

    目前对于灰色信息攻击下相依网络安全态势评估方法的相关研究有很多, 例如文志成等人[3]提出了基于聚类分析的评估方法, 建立主机上多个二级正方体, 通过对网络主机异构信息源进行高效聚类, 得到该主机的安全态势, 结合Dirichlet对分布平滑和安全概率进行验证, 优化聚类结果;建立网络上的立方体, 对受攻击的各主机安全态势进行量化评估。虽然运用该方法可以对网络的安全态势进行评估, 但是评估结果不够准确。王豪璞等人[4]提出一种针对多步攻击的网络安全态势评估方法。识别网络中的攻击信息, 对相应的攻击场景进行关联, 找到对应的攻击路径与攻击阶段, 构建安全态势评估标准, 根据攻击阶段的危害指数, 实现网络安全态势的评估。该方法能够对受攻击网络的安全态势进行较为准确的评估, 但是评估效率过低。王法玉等人[5]提出一种多源事件相融合的网络安全态势评估方法。将多源数据变换为统一形式, 对于网络攻击中的大量误报情况, 利用属性相似度算法结灰色信息攻击下的网络进行关联、融合;采用模糊综合评价法的同时结合AHP确定攻击信息、漏洞信息得到安全态势评估结果, 该方法存在的弊端是评估结果不够准确。

    根据上述方法中存在的问题, 提出一种新的网络安全评估方法——基于攻击模式识别的相依网络安全态势评估方法, 根据布谷鸟搜索算法获得最佳网络态势评估模型。通过仿真验证了所提方法在评估准确性方面的优势。

    2 安全态势评估模型构建

    针对相依网络安全态势输入指标值与态势值关系不明确的问题, 根据相依网络输入指标数量和网络输出态势值确定神经网络的输入输出节点[6]数量。针对随机选择的神经网络权值以及阈值易导致目标网络收敛速度慢、CPU值过高等问题, 利用布谷鸟搜索算法通过多次迭代获取神经网络最优权值和阈值使最终的评估结果更为精确。

    通常情况下, 随机选取的神经网络权值及阈值可能不是最优的, 采用非最佳的权值[7]和阈值评估网络安全态势存在评估时间较长的问题, 因此结合相依网络特点, 设计了一种新的态势评估模型。该模型主要由态势信息输入、态势信息映射以及态势信息输出部分构成, 以上三部分的主要功能如下:

    1) 目标网络态势数据输入。间隔采集相依网络中节点的态势信息, 将这些信息视为神经网络的输入部分。

    2) 目标网络态势数据映射。态势数据映射过程主要分为三层, 分别是输入层、隐含层以及输出层。输入层主要负责从输入部分获取态势信息, 这些信息经过隐含层的处理传递给隐含层的输出端口, 通过输出层的处理, 传递给输出层的输出端口上, 最终完成目标网络态势信息数据的映射过程。

    神经网络中输入层的节点数量与输入层输入的态势数据相同;输出层的节点数量与实际评估出的态势结果数量相同;隐含层可依据实际需要进行设置, 但设定的隐含层数量有限, 不可过大, 过大将导致神经网络训练时间过长, 根据此特性, 利用隐含层叠加隐含层节点数量的形式即可满足实际评估需求, 同时还能够提高目标网络的准确性。利用试凑法设定神经网络中隐含层节点的数量, 并运用经验公式设定适量的隐含层节点数量, 每次增加相同数量的节点数量, 在采用相同评估样本的前提下, 选择神经网络训练误差最小时相应的隐含层节点数。初始阶段, 隐含层节点数量为

    m=n+l+δ (1)

    其中, n表示BP网络中输入层所含有的节点数量, l表示输出层所含有的节点数量, δ表示区间[0, 10]内的常数, 当n+l的开方是一个小数时, 在满足评估精度的要求下, 应当选择较少的隐含层节点数量m。

    选定BP网络输入层的节点数量为i, 隐含层设定的节点数量为j, 输出层的节点数量为k, wij表示神经网络中输入到隐含层间的衔接权值, θj表示设定的隐含层阈值, 隐含层与输出层之间的连接权值可表示为vjk, rk表示输出层阈值, yj表示BP中隐含层节点j的输出值, yk表示BP中对应输出层节点k的输出值, 则有

    yi=f (∑i=1nwij×xi-θj) (2)

    yk=f (∑i=1nwjk×yi+rk) (3)

    式中, f (x) 表示sigmoid函数, 该函数用于描述BP传输函数, xi表示中BP输入层的对应系数。

    3) 态势信息输出。当接收到映射部分传输的态势信息, 该值即为采集时段的目标网络态势值。

    相依网络安全态势评估是为了实现隐含层态势信息到输出层态势数据集的映射, 采用布谷鸟搜索算法优化目标网络态势评估模型。设定, M表示优化的目标网络输入层节点数量, I表示优化网络的隐含层节点数量, J表示优化的网络输出层节点数量, W表示优化的网络输入到隐含层的对应矩阵, V表示优化的网络隐含层到输出层之间的权值矩阵, θ表示优化的网络隐含层阈值矩阵, r表示目标网络中输出层阈值矩阵。优化目标网络中全部权值, 阈值联合编码形式为布谷鸟, 以下给出布谷鸟的编码形式

    F=W11W21…WM1V11V21…VIJθIr1…rJ (4)

    结合神经网络训练误差越小越好的特性选取布谷鸟搜索算法的适应度函数, 假设, 神经网络总误差E小于阈值ε, 当第k个神经网络输出节点的期盼值为dk, 真实输出值为yk, 则有:

    E=12∑i=1Ι∑m=1Μ[dk-yk]2≤ε (5)

    给出布谷鸟搜索算法适应度函数设计策略

    F (w, v, θ, r) =10-3/[∑i=1Ι∑m=1Μ (dk-yk) 2] (6)

    根据种群适应度判断当前个体是否满足迭代终止条件, 将满足要求的个体进行解码, 获取神经网络最优权值、阈值, 将结果赋予目标网络态势评估模型中, 获得最佳网络态势评估模型

    Q=∑m=1Μ[dk-yk]2F (w, v, θ, r) yk (7)

    综上, 完成对网络安全态势评估模型的构建, 为相依网络安全态势评估提供基础。

    3 基于攻击识别法的相依网络安全态势评估

    根据2.1节中得到的最佳网络态势评估模型, 对相依网络中信息进行融合, 通过融合识别出灰色攻击信息, 依据灰色信息进行网络安全评估量化, 根据网络状态转移图, 得到网络安全预测[8]结果。

    3.1 信息融合与识别

    相依网络中的攻击信息无法直接进行识别, 不同防御模式造成攻击处理的混乱, 防御系统的缺陷导致反馈信息中含有大量的错误信息, 针对这种问题, 需要对攻击信息进行信息融合, 以便获得更为精确的反馈信息。

    攻击信息关联度为攻击信息间的关联情况, 两个攻击在同一攻击中的有效性, 记为cor (a, b) ;其中, a、b为正常信息, 其函数表达式为

    cor (a, b) =[∑k=1nakFeaturek (a, b) ]/[∑k=1nak] (8)

    式中Featurek (a, b) 和ak为第k个特征信息特征属性间的关联度和权重信息, 在攻击阶段识别中, 先将信息融合得到的正常信息与攻击模式下的信息库模板进行匹配, 引用关联性计算受灰色信息攻击的具体阶段

    D=cor (a, b) *Ea/b (9)

    式中, 通过对每个反馈信息与攻击信息库进行对照, 根据阈值的安全事件计算灰色信息攻击阶段支持概率

    s (h) =∏i=1nm (Τi) ⋅∏i=1ncor (Τi, Τi+1) (10)

    式中Ti代表i个攻击阶段的灰色攻击信息, n为当前灰色信息攻击所处的攻击阶段。

    3.2 态势评估量化

    利用由上而下, 先部分后全部的策略, 对网络节点及对应的权重进行结合, 根据结合结果对网络节点态势进行评估:

    假设, s (h) 为相依网络中灰色信息攻击阶段的支持率, t (h) 为攻击信息的威胁程度, 可获得灰色信息攻击对网络节点的影响

    e=s (h) •t (h) (11)

    在同一测试区域测试节点时受到多方攻击, 在这种情况下, 网络节点安全态势[9]SA可表示为多个灰色信息攻击对网络态势的影响

    SA=∑i=1nei (12)

    式中i为测试到的攻击数量。

    ΝSA=∑i=1nSAi⋅wi (13)

    结合网络中节点的重要度i, 基于状态转移图对网络安全态势进行预测, 状态转移图的具体公式如下

    STG= (S0, Ai, E) (14)

    式中S0为未攻击下的状态, Ai为灰色信息攻击下的状态, S0与Ai的攻击部分组成的攻击点的集合用E表示, 对网络中攻击信息的漏洞情况进行测试并计算其转移概率, 利用STG得到对应攻击阶段的转移条件, 通过该条件与系统漏洞及配置数据一一对应, 如符合该条件, 则有ns (h) =1;如不符合, 则有ns (h) =0, 根据发生概率[10,11]与攻击威胁以及网络节点权重取得灰色信息攻击下的安全态势预测值

    SA=∑i=1ns (h) ⋅ns (h) ⋅t (post (h) ) (15)

    根据预测值得到未来阶段网络安全态势。

    受到灰色信息攻击的节点数量越多则网络威胁越为严重, 同时网络安全态势更为严峻。将文献[4]方法、所提方法的安全态势预测值与实际安全态势情况进行对比, 文献[4]方法的预测值要高于实际安全态势值, 并且其变化轨迹与实际轨迹大不相同, 而所提方法的评估每一时刻的安全态势值变化轨迹与实际轨迹基本一致, 由此可以看出, 所提方法在提高评估准确性方面具有显著优势。

    分析图4可知, 运用所提方法对对称相依网络安全态势进行评估时, 评估结果的精度较高, 而对非对称相依网络安全态势进行评估时, 结果精度较低, 且起伏较大, 说明所提方法适用于一般相依网络, 但是对于非对称性网络来说并不适用。因此, 将来会以非对称网络作为研究对象进行相关研究。

    5 结论

    针对传统评估方法存在的一系列问题, 提出一种基于攻击模式识别的相依网络安全态势评估方法, 根据网络态势值建立安全态势最优模型, 利用攻击模式识别的安全评估方法进行态势预测, 得到对应的评估结果。仿真结果表明, 所提评估方法与传统评估方法相比, 评估效率更高、结果更加准确。在今后的研究工作中将对非对称相依网络进行研究, 争取扩大所提方法的适用范围, 对该评估模型进行进一步的改善, 争取能够面向不同研究对象得到更为精确的评估结果。
