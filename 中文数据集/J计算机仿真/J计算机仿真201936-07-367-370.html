    摘    要：

    对大数据分析下的不完备数据进行填补, 能够有效提高数据的利用率。对不完备数据进行多重准确填补, 需要计算所有数据的向量属性均值与标准差, 并将不完备数据填补模拟中重复应用。传统方法对数据填补变量间关系予以考虑, 根据与待填补数据之间的相关性完成缺失填补, 但忽略了计算所有数据的标准差, 导致填补效率低。提出基于logistic的大数据分析下不完备数据多重准确填补方法。对所有数据向量属性值均值与标准差进行计算, 采用估计的形式得到数据平均向量与协方差函数, 并对各观察对象缺失值进行独立模拟填补, 通过logistic回归模型选择存在缺失值的变量所需填补值, 得到完备数据。重新估计数据平均向量与协方差函数, 并将其在不完备数据填补模拟中重复应用。对上述过程进行迭代, 直到达到迭代条件, 将不完备数据多重填补结果输出。实验表明, 上述方法填补效率较高, 可为该领域研究发展奠定基础。

    关键词：

    大数据分析; 不完备数据; 多重填补;

    Incomplete Data Multiple Precision Filling Simulation under Big Data Analysis

    WANG Li-wen HUANG Xu

    Xi'an University of Science and Technology

    Abstract：

    Traditional method ignores to calculate all the standard deviation of data, resulting in the low efficiency. This paper focuses on a multiple accuracy imputation method for incomplete data based on Logistic in big data analysis. Firstly, mean and standard deviation of all data vector attribute values were calculated, and then the form of estimation was used to obtain average vector and covariance function of data. Meanwhile, the missing value of each observed object was simulated and filled independently. Moreover, the logistic regression model was used to choose the imputed data needed by variable with the missing value, and then get the complete data. In addition, average vector and covariance function of data were estimated again, which were applied to simulation of incomplete data imputation repeatedly. Finally, the above process was iterated until reaching the iteration condition. Thus, the result of multiple imputations of incomplete data was output. Simulations prove that the proposed method has high-efficient data imputation, which can lay the foundation for the research and development in this field.

    Keyword：

    Big data analysis; Incomplete data; Multiple imputations;

    1 引言

    大数据分析下完备数据作为数据质量的关键维度之一, 对于数据的利用和划分有着重要的作用[1]。但在实际应用过程中, 很多数据处于缺失待填补的状态。数据不完备现象十分普遍, 有些情况下这些数据会成为不可或缺的事件解决影响因子, 为此对海量数据下不完备数据进行填补成为了亟待解决的问题[2]。当前基于动态概率的数据填补法, 将RFID数据由数据层转换至逻辑层当作数据处理的粒度, 根据挖掘已知区域事件顺序相关程度, 实现后续发生事件的判断以及填补。该法简约性能较强, 但不具备高准确性填补性能。由于不完备数据的快速、准确填补现实意义较大, 在社会各界中占据着至关重要的地位, 因此引起了多方学者的高度重视, 并纷纷对其进行了深度研究。

    郑奇斌[3]等人提出基于局部敏感哈希的数据填补法。利用与之相对的局部敏感哈希法对各待填补数据实行局部敏感哈希操作, 依据所获哈希值寻找和其具有相似性的候选数据。在候选数据中逐个计算数据间的相似程度, 以此找到相似度较高的若干个数据, 利用kNN算法完成不完整数据的填补。实验表明, 该法具有一定的填补准确性, 但经过填补过程中, 未对数据进行归一化, 导致其填补效率较低。韩飞[4]等人提出基于双聚类的数据缺失填补法。通过双聚类下数据簇的均方残基为0和数据簇中属性值的波动具有一致性等特性, 实现不完备数据填补。将查找具有缺失的最大数据簇问题转换成缺失对象和其他对象间最大相似属性的问题, 于最大相似属性集合中, 将缺失值众数当作填补值。该法运行能耗较低, 但无法得到精确数据填补结果。杨杰[5]等人提出基于k-means聚类的数据缺失填补法。首先对数据填补变量间关系予以考虑, 根据和待填补数据之间的相关性完成缺失填补, 以降低弱相关或者不相关的数据对不完备数据填补造成的影响, 提升数据填补精确性。为实现对对未知的不完备数据分块, 提出基于k-means的数据分块法。实验结果表明, 该法可有效提升缺失数据填补准确性, 但过程相对繁琐, 填补效率偏低。

    当前不完备数据填补方法或算法中, 存在着填补效率和填补准确性低的问题, 提出基于logistic的大数据分析下不完备数据多重准确填补方法。

    2 大数据分析下不完备数据填补原理

    在大数据分析下不完备数据填补原理中, 根据属性重要度实现数据补齐。过程中, 利用同类数据分布具有的内聚性、异类数据具有的耦合性对数据属性重要程度进行定义, 对重要属性进行优先填补。利用加权欧式距离以及余弦相似从距离方面对数据样本之间的相似程度进行度量, 根据数据间的相似度度量结果, 完成不完备数据填补。详细过程如下所示。

    假设, 将输入定义为不完备决策体系IDS, 将输出定义为完备决策体系CDS。

    假设c代表决策体系样本任意条件属性, 于IDS上对数据属性重要程度进行计算

    Sig (c) =1/∑i=1mσci (∑|i∩j||i∪j|) (1)

    其中, m代表决策类数量, σci代表条件属性c中第i种数据类的标准差。如果c是离散属性, 那么|i∩j|代表条件属性c中第j种数据类的属性值与第i种数据类的属性值具有相等样本数量;如果c是连续属性, 那么|i∩j|代表属性c中第j种数据类的属性值于第i种数据类的属性值具有不等的样本数量。|i∪j|代表属性c中第j种数据类的属性值与第i种数据类样本总和。

    将IDS划分成完备子集CS与不完备子集IS。根据不完备数据值隶属条件属性重要度由大至小对不完备子集IS重新排列。

    针对不完备数据的填补, 将IS中各样本定义为sam, 利用sam的存在值对权值wi进行设置。在此, 将数据属性重要度当作权值, 通过加权欧式距离对数据样本之间的相似程度进行计算。其中, 权值wi计算式可表示为

    wi=Sig (c) ∑i=1mSig (c) ⋅sam (2)

    式中, Sig (ci) 代表待填补数据样本的第i个属性重要程度。

    根据式 (2) 的计算对欧式距离d (x, y) 进行计算

    d (x, y) =∑i=1mwi|xi-yi|2 (3)

    式中, xi和yi分别代表第i个属性的位置横纵坐标。欧氏距离度量作为空间各点绝对距离, 与各点具体位置有直接关系, 可以体现出个体数据特征绝对差异, 不过并不能在方向上对数据差异进行区分[6,7]。

    为解决上述问题, 利用余弦相似于角度上度量两个数据样本之间的相似程度, 得到与sam最为相似的数据样本sam′。综上, 则有

    cos (α) =∑i=1md (x, y) ∑i=1mxi∑i=1myi (4)

    其中, α代表两个数据样本之间在空间上所成角度。余弦值取值在[-1, 1]中, 其取值越接近1, 则表示两个数据样本越相似。

    利用式 (4) 对sam′进行计算

    sam´=d (x, y) ⋅cos (α) ∑i=1mSig (c) ⋅sam (5)

    利用sam′属性值代替sam属性值, 实现数据填补

    CS=Sig (c) ·sam′⨂sam (6)

    其中, ⨂代表替换标识。对上述过程进行迭代, 直到IS为空。将集合CS输出, 这时的输出即为填补之后的CDS。

    根据上述步骤, 完成了大数据分析下不完备数据填补原理。依据该原理, 实现了大数据分析下不完备数据的准确填补。

    3 大数据分析下不完备数据多重准确填补方法

    3.1 数据归一化

    为将不完备数据填补过程中产生的不平衡消除, 并提高数据填补准确性和效率, 要对数据进行归一化[8]。过程中, 对所有数据向量属性值均值与标准差进行计算, 将计算结果代入数据记录格式化中, 实现原始数据记录的格式化。通过z-scores法将所有数据压缩至设定范围内, 得到数据属性值归一化结果。详细结果如下:

    为解决当前不完备数据填补方法中存在的问题, 需要将数据属性值进行归一化, 为确保归一化效果, 将所有属性值归一化至已经确定的数值范围, 尽可能地将拥有不同属性的数据统一权重值。在此, 通过标准偏差实现属性集中连续特征的归一化。

    假设, C代表完整数据集, 其可表示为C={c1, c2, …, ct}, 式中的各数据ci根据n个属性{a1, a2, …, an}构成的n维空间中特征向量{ai1, ai2, …, ain}表征, aij代表属性aj在第j个数据上的取值。那么数据集的所有向量属性值均值avg (ci (aj) ) 以及标准差可表示为

    avg (ci (aj) ) =1n∑i=1n (aj) (7)

    std (ci (aj) ) =1n-1∑i=1n (aj) -avg (ci (aj) ) (8)

    根据式 (7) 和式 (8) 的计算, 将原始数据集合中记录实行格式化操作

    ci (aj) =ci (aj) -avg (ci (aj) ) std (ci (aj) ) (9)

    根据z-scores法将全部数据压缩于[0, 1]范围。

    ci (aj) =f (ci (aj) ) ·Δci (aj) +min (ci (aj) ′) (10)

    式中

    f (ci (aj) ) =ci (aj) -min (ci (aj) ) max (ci (aj) ) -min (ci (aj) ) (11)

    Δci (aj) =max (ci (aj) ′) -min (ci (aj) ′) (12)

    其中, f (ci (aj) ) 代表信息函数, Δci (aj) 代表格式化后的记录。max (ci (aj) ) 、min (ci (aj) ) 分别代表z-scores法规范化之间数据最大值与最小值, max (ci (aj) ′) 、min (ci (aj) ′) 分别代表代表z-scores法规范化之后数据最大值与最小值, 在此分别将其设置为0和1。根据上述计算, 可将数据属性值归一化至[0, 1]中, 得到最终的数据归一化结果

    Ζ (ci (aj) ) =f (ci (aj) ) ⋅Δci (aj) ∑10C⋅λ (13)

    其中, Z (ci (aj) ) 代表归一化数据属性值, λ代表归一化系数。式 (13) 计算结果即为数据属性值归一化结果。

    3.2 基于logistic的不完备数据多重填补方法

    在基于logistic的不完备数据多重填补方法中, 将3.1节得到数据属性值归一化结果引入数据填补中。过程中, 采用估计的形式得到数据平均向量与协方差函数, 并对各观察对象缺失值进行独立模拟填补, 通过logistic回归模型在不存在缺失值的变量后验概率分布结果中选择存在缺失值的变量所需填补值, 得到完备数据。重新估计数据平均向量与协方差函数, 并将其重复在不完备数据填补模拟中应用。对该过程进行迭代, 直到达到迭代条件, 将不完备数据多重填补结果输出。详细过程为:

    在不完备数据多重填补中, 利用填补值表示缺失值中一个随机数据样本, 将其当作统计工具, 可以将观测到的数据信息利用地较为充分[9,10]。下面通过两个步骤的迭代完成不完备数据多重准确填补:

    第一步:利用估计得到的数据平均向量与协方差函数, 对各观察对象缺失值进行独立模拟填补。

    假设Q为估计的参数, 其中Q能够通过一个点的估计值Q′及一个标准误表征, 由此对于M个完备数据集, 即有M个Q′与标准误。依据上述内容, Q最终估计值平均向量Q¯可表示为

    Q¯=1Μ∑i=1ΜQ´ (14)

    据式 (14) 的计算, 可得到协方差函数V (Q¯)

    V (Q¯) =1Μ∑i=1Μ1Μ-1∑i=1Μ (Q´-Q¯) 2 (15)

    通过式 (14) 和式 (15) , 实现观察对象缺失值的独立模拟填补。即Yi (mis) 为第i个观察对象存在缺失值的变量, Yi (obs) 为第i个观察对象不存在缺失值的变量。通过logistic回归模型指定Yi (obs) 的后验概率分布结果中选择Yi (mis) 填补值, 利用Yi (obs) 替代Yi (mis) , 可得到完备数据。

    利用上述内容, 可进行第二步:根据第一步得到的完备数据, 重新对数据平均向量与协方差函数进行估计, 将新估计值在第一步中重新利用, 实现不完备数据多重准确填补。

    反复实行上述两个步骤, 直到达到迭代条件, 一个与缺失值近似的填补值被选择存储。为合适地反映缺失值具有的不确定性质, 填补值在条件分布中反复选取M次。

    依据上述的迭代过程, 可得到基于logistic的不完备数据多重准确填补结果

    CDS (Μ) =∑i=1ΜQ¯⋅V (Q¯) ∑i=1ΜYi (obs) ⊗Yi (mis) (16)

    其中, CDS (M) 代表基于logistic的不完备数据填补结果。综上所述, 完成了大数据分析下不完备数据的多重准确填补。

    4 实验结果与分析

    为验证基于logistic的不完备数据多重准确填补方法的整体有效性, 需要进行一次仿真实验。以UCI与HIGGS不同规模数据集作为实验数据来源, 实验环境为:Inter Xeon E5 2609 V2处理设备, 其主频为2.5GHz, 内存为128GB, 操作体系为Centos6.4。实验指标为:不完备数据填补准确性和填补效率。实验结果如图1。

    图1 不同数据填补法准确性对比   下载原图

    分析图1可知, 基于双聚类的数据缺失填补法填补准确性系数最高为0.5。基于logistic的不完备数据多重准确填补方法填补准确性系数最高为0.99。根据数据对比可知所提方法可实践性较强。

    图2 不同数据填补法效率对比   下载原图

    由图2可知, 基于局部敏感哈希的数据填补法在应用过程中, 随着待填补数据量的不断增长, 其填补效率越来越低, 所耗时间越来越长, 耗时曲线持续上升。基于k-means聚类的数据缺失填补法随着待填补数据量的增加, 呈现出了不稳定的一面, 耗时曲线波动较大, 总体来看呈上升趋势。基于logistic的不完备数据多重准确填补方法虽然也随着数据量的增长, 在填补时间上有所波动, 但数据填补整体耗时较少, 且当待填补数据量为6000万时, 填补耗时呈下降趋势。经过上述对比可见, 基于logistic的不完备数据多重准确填补方法与当前数据填补法相比效率更高。

    5 结束语

    不完备数据填补关系到数据利用价值的体现, 针对当前不完备数据填补算法和方法中存在的填补效率和填补准确性低的问题, 提出基于logistic的不完备数据多重准确填补方法。利用在数据填补之前对所有数据进行归一化提升数据填补效率和准确性。经实验证明, 该方法具有较强的鲁棒性和可行性。针对未来的发展方向做出如下规划:在数据量持续增长的过程中, 单个计算节点可能会存在计算瓶颈, 这就需要并行化的方式提升方法整体可延续性;数据填补时, 周边干扰因子应列入考虑范围, 以进一步提升数据填补精度和填补效率。
