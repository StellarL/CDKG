    摘    要：

    针对现有移动网络混合安全漏洞检测方法存在的混合安全漏洞检测准确率低、漏检率高、检测速度较慢等问题, 提出了一种基于静态防御的移动网络混合安全漏洞检测方法, 利用信息特征项的权重值、用户需求与信息间的关系、信息语义与用户需求语义间的相似度以及用户感兴趣信息与信息特征项相似度等信息, 完成对移动网络信息的过滤。将过滤后的移动网络信息作为检测移动网络混合安全漏洞的样本, 创建一个自回归模型, 对模型中的数据做零均值化处理, 选择一个恰当的自回归模型阶数拟合时间序列, 利用得到的时间序列拟合二阶自回归模型, 根据二阶自回归模型检测移动网络信息样本, 利用异常信息判定条件对检测的移动网络信息样本进行判定, 实现对移动网络混合安全漏洞的检测。实验结果表明, 所提移动网络混合安全漏洞检测方法可以快速、准确检测移动网络中的混合安全漏洞。

    Mobile Network Hybrid Security Vulnerability Detection Simulation under Static Defense

    QI Lu-jie

    China University of Political science and Law, College of Criminal Justice

    Abstract：

    Due to low detection accuracy and high missed detection rate in current mixed security hole detection methods, this paper puts forward a method to detect mixed security hole in mobile network based on static defense. Firstly, the weight value of information feature item, the relationship between user requirement and information, the similarity degree between information semantics and user requirement semantics, and the similarity between user interest information and information feature item were used to filter mobile network information. Then, the filtered mobile network information was used as the sample to detect the mixed security hole in mobile network, so as to build an autoregressive model. The data in model were zero-centered, and an appropriate autoregressive model order was selected to fit time series. After that, the time series was used to fit the second-order autoregressive model. Based on the second-order autoregressive model, the mobile network information sample was detected. Finally, the abnormal information decision condition was used to judge the mobile network information sample. Thus, the detection for the mixed security hole in mobile network was achieved. Simulation results show that the proposed method can quickly and accurately detect mixed security hole in mobile network.

    Keyword：

    Static defense; Mobile network; Mixed security hole detection;

    1 引言

    网络技术与信息技术的快速发展, 使人们正在经历一个前所未有的移动网络时代。在移动网络时代中, 一些曾经无法想象的事情步入了人们的生活, 社会对移动网络的依赖性愈来愈强, 甚至已经达到不可缺少的程度, 一旦没有了移动网络, 很多高精度工作都将无法正常进行, 很多需要网络的服务也将瘫痪。移动网络新技术的开发在提升了人们日常工作效率的同时, 也带来了一些不可忽视的安全威胁问题, 种类繁多的网络病毒、层出不穷的黑客技术使各种网络入侵事件屡见不鲜, 各国政府逐渐意识到移动网络安全的重要性, 对网络安全漏洞的侦查越来越受到重视。越早的侦查出软件安全漏洞, 所引发的损失就越小[1], 研究出一种能够在静态防御状态下有效检测移动网络混合安全漏洞的方法是非常重要的, 专家们也对此展开了多项研究[2], 研究成果如下。

    文献[3]提出PHP程序混合型安全漏洞检测方法, 该方法通过PHP内置函数对PHP程序进行深入剖析, 跟据剖析结果形成了一个抽象剖析树, 利用剖析树生成控制流图, 创建内置特征模型与敏感特征模型, 准确分析数据流;提高分析的准确性, 并测试两个常用的PHP应用程序, 完成对移动网络混合安全漏洞的检测。文献[4]提出基于用户行为模拟的混合型安全漏洞检测方法, 根据研究网页结构的方式, 找出非格式化注入点, 通过字符长度、种类等信息优化攻击向量, 以绕开服务器过滤函数, 减少漏洞检测所需要的时间。文献[5]提出静态分析结合动态符号执行的混合型安全漏洞侦查方法, 该方法利用静态分析的方式找出目标程序中调用危险函数的指令地址, 在动态符号执行过程中碰到危险函数指令地址时, 通过求解约束危险路径的条件, 完成对移动网络混合安全漏洞的检测。

    针对上述三种方法均存在的安全漏洞检测准确率低、漏检率高、自适应性差的问题, 提出了基于静态防御的移动网络混合安全漏洞检测方法。

    2 静态防御下移动网络混合安全漏洞检测

    2.1 移动网络信息过滤

    移动网络中的信息领域本体IO是4元组, IO= (C, SR, IR, P) , 其中C是概念名字、SR和IR分别是概念间的结构关系和非结构关系、P是概念属性。如果移动网络信息概念的重要性高于给定阈值, 称这个概念是特征项[6], 记为CT。CT如果存在性质描述集合CS, CS的单个元素csk⊆CS是CT的解释, cs的值被记做cs_val。

    信息I的完整语义表示形式为

    Ι={ (ct1, (cs11|cs11_val, cs12|cs12_val, ⋯) , right1) , ⋯, (ctn, (csn1|csn1_val, csn2|csn2_val, ⋯) , rightn) }    (1)

    其中, 特征项cti的权重是right, cti⊆CT, 序对csik|csik_val是cti的第k个解释值。

    IO给定的特征项和IO解释集合如果有赋值映射N, 则有Nct→IOc, 存在解释csk=IOcpj, 则称该信息特征项可被IO理解[7], 如果全部的解释满足csk=IOcpj, 则该信息特征项能够被IO理解。如果IO能够理解信息I, 则全部的信息特征项cti⊆CT可被IO理解。

    2.2 移动网络混合安全漏洞检测

    将上述处理过滤后的移动网络信息作为侦查移动网络混合安全漏洞的样本, 创建一个自回归模型。对模型中的数据做零均值化处理, 选择一个恰当的自回归模型阶数拟合时间序列, 根据获得的时间序列拟合二阶自回归模型, 将其用于检测移动网络信息, 结合异常信息判定条件对待检测的移动网络信息样本进行判定, 实现移动网络混合安全漏洞检测。

    对滑动窗零均值化, 尽管从总体上说, 过滤后的移动网络信息观测值序列是不平稳的, 但是可将其局部看作统计上近似平稳, 将这个局部作为一个滑动时间窗, 窗的大小设为N′+1, 每次取出N′+1个数, 滑动窗的N′+1个数表示为y1, y2, …yN+1, 用前N′个数创建自回归模型AR, 判断第N′+1个数是否出现异常。创建自回归模型前需要对前N′个数据做零均值处理[9]。设Y¯是y1, y2, …yN+1的平均值, Y¯的计算公式为

    Y¯= (1/Ν′) ∑t=1Ν′yt, yt∈Ι, t=1, 2, ⋯, Ν′x′t=yt-Y¯, t=1, 2, ⋯, Ν′, Ν′+1.    (10)

    由式 (10) 可知, x′1, x′2, …, x′N′+1为做零均值处理后的时间序列。

    选择一个恰当的AR阶数q′, 设滑动时间窗近似平稳, 窗口大小N′应该相对小一些, AR (q′) 拟合时间序列时, 准确使用FPE衡量, 对应最小FPE的AR阶数q′即为最佳模型阶数。N′和q′的约束条件为

    0≤q′≤0.1Ν′    (11)

    综上所述, 所提基于静态防御的移动网络混合安全漏洞检测方法能够在高准确率、低漏检率的情况下, 快速的实现对移动网络混合安全漏洞的检测。

    4 结论

    计算机新技术的发展, 在提升人们工作效率的同时也给计算机带来了安全威胁。针对这个威胁提出了基于静态防御的移动网络混合安全漏洞检测方法。首先对移动网络信息进行了过滤处理, 然后创建一个自回归模型, 对模型中的数据做零均值化处理, 利用异常信息判定条件对过滤后的移动网络信息进行判定, 最终实现对移动网络混合安全漏洞的检测。

    为了能够实时、准确的检测出移动网络中的混合安全漏洞, 接下来的工作会针对移动网络不同攻击类型进行更深一步的防御研究。
