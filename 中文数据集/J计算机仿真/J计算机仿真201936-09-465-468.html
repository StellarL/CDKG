    摘    要：

    针对传统的多障碍场景下不规则建筑图像布局标定中,普遍存在着标定准确率较低、运行时间过长、成本耗费过大等问题。提出一种基于内核投影的建筑图像布局标定方法。通过对多障碍场景下不规则建筑图像进行分析,采用超像素技术方法对给出的多障碍场景下不规则建筑图像进行大致分割,构造不规则建筑图像特征描述算子,并采用内核投影技术对不规则图像特征描述算子进行改进,利用改进特征描述算子降低图像特征描述子的维度,并以欧式距离对图像特征点进行匹配,引入随机抽样一致性算法得到特征点匹配后的不规则建筑图像变换模型,利用该模型来对其进行标定。实验结果表明,所提方法标定准确率较高、成本耗费较低、运行时间较短,具有一定的应用价值。

    关键词：

    多障碍场景下; 不规则建筑; 图像布局; 标定;

    Irregular Building Image Layout Calibration Simulation in Multi-Obstacle Scene

    ZHANG Bo

    School of Life Sciences, Changchun Normal University

    Abstract：

    This article presents a calibration method of architecture image layout based on kernel projection. Firstly, irregular architecture image under multi-obstacles scene was analyzed and the provided irregular architecture image was segmented roughly via super-pixel technique. On that basis, feature description operator of the irregular architecture image was constructed and the operator was modified via the kernel projection. Then, the modified operator was used to reduce dimension of the feature description operator and image feature point was matched via the Euclidean distance. Consistency algorithm of random selection was introduced to obtain transformation model of the irregular architecture image after feature mapping. Finally, the model was used to calibrate the image. Simulation results show that the method has high calibration precision and low cost. The runtime is shorter. It has certain application value.

    Keyword：

    Multi-obstacles scene; Irregular architecture; Image layout; Calibration;

    1 引言

    我国计算机视觉技术快速发展,人们开始对户外建筑进一步了解,其复杂的应用前景已经成为专家学者们研究热点[1]。多障碍场景下不规则建筑图像布局标定技术是当前户外场景的重要研究方向[2,3]。在现阶段多障碍场景下不规则建筑图像布局标定中,很少对不规则建筑图像目标区域进行分割,普遍存在着标定准确率较低的问题。在这种情况下,如何有效地提出一种具有较高精度的标定方法成为当今社会亟待解决的问题[4,5]。

    文献[6]提出一种基于PCA的相机响应函数模型标定方法。该方法采用主成分分析法对相机采集的图像数据进行分析,并结合相机响应函数的约束条件,构建相机采集图像响应函数较低参数的经验模型,在模型中输入图像合适的参数,根据不同相机图像曝光量的输入图像,采用最小二乘法求解出模型的系数,利用该模型系数实现对相机响应函数的标定。该方法具有较高的标定精度,但是在标定的过程中运行时间较长。文献[7]提出一种基于固定视点的视觉标定方法。该方法利用机器人的尾端夹起一个棋盘进行一系列运动,并且每次运动结束时摄像机都会采集棋盘格图像,并对机器人尾端的位姿进行记录,待所有运动结束后,利用最小二乘方法对摄像机和基座之间的关系进行视觉标定。该方法标定的过程较为简单,并且标定的速度较快,但是标定准确率较低。文献[8]提出一种自动、快速的Kinect标定方法。该方法提取彩色图像角点并构成角点集合,在对点云图像进行分割,提取出图像棋盘格点云,利用三维哈夫变换检测方法将提取出的棋盘格点云映射到深度图像的平面上,在深度图像中获取图像中的角点,并将深度图像角点映射到棋盘格点云中,构成角点点云,将角点点云与彩色图像角点集合进行匹配,得到图像的空间坐标,完成标定。该方法标定速度快,但是标定所需成本耗费较大。

    针对上述问题,提出基于内核投影的建筑图像布局标定方法。实验结果表明,所提方法标定准确率较高、成本耗费较低、运行时间较短,具有一定的应用价值。

    2 多障碍场景下不规则建筑图像布局标定

    2.1 不规则建筑图像分割

    通过对多障碍场景下不规则建筑图像进行分析,采用超像素技术方法对给出的多障碍场景下不规则建筑图像进行大致分割。具体过程如下:

    将多障碍场景下不规则建筑图像看成一张无向图G(V,E),不规则建筑图像上各个像素都可以看成是无向图上的一个顶点v,由于不规则建筑图像是用一个二维矩阵进行储存的,围绕在多障碍场景下不规则建筑图像一个像素周边含有8个邻近的像素[9],用E来表示不规则建筑图像像素与周边8个邻近的像素之间边e的集合,对不规则建筑图像像素间每条边e设定一个权值,该值用w(e)来表示;它主要描述e所连接的无向图中两个顶点vi,vj所表示不规则建筑图像像素的亮度差异;V代表无向图中所有顶点的集合,一个分割S就可以将集合V分成不同部分的一种划分,利用这个划分将S分割成多个C,则每个C均对应G(V,E′)的一个子图,用E′来表示E的非空子集,将各个子集之间的内部差异定义为不规则建筑图像局部区域最小生成树上的最大权值MST(C,E),利用下式给出表达式

    Ιnt(C)=maxe∈ΜSΤ(C,E)w(e) (1)

    同属多障碍场景下不规则建筑图像局部区域V的两部分子集C1和C2,两个子集之间的差异是连接这两部分之间的最小权边

    Dif(C1,C2)=minvi∈C1,vj∈C2,(vi,vj)∈Ew(e) (2)

    假设,多障碍场景下不规则建筑图像局部部分C1和C2两者之间的亮度差异Dif(C1,C2)大于C1和C2中至少一个部分的内部差异Int(C1)和Int(C2),则说明不能将这两部分互相融合[10],反之,就可以合并,利用下式给出其表达式

    D(C1,C2)=Dif(C1,C2)>MInt(C1,C2) (3)

    将不规则建筑图像中最小内部差异定义为

    MInt(C1,C2)=min(Int(C1)+D(C1,C2),Int(C2)) (4)

    式中,π(C)代表一个界限函数,主要用它来控制不规则建筑图像块间差异Dif(C1,C2)和块内差异之间的度Int(|C|),不规则建筑图像分割结果表达式为

    P(C)=k/|MInt(C1,C2)| (5)

    式中:k代表图像分割常数。

    2.2 基于内核投影的建筑图像布局标定方法

    以2.1节对多障碍场景下不规则建筑图像进行大致分割为依据,构造不规则建筑图像特征描述算子,并采用内核投影技术对不规则图像特征描述算子进行改进,降低图像特征描述子的维度,以欧式距离对图像特征点进行匹配,引入随机抽样一致性算法得到不规则建筑图像变换模型,利用该模型来对其进行标定。具体过程如下:

    构造多障碍场景下不规则建筑图像特征描述算子是为了确定不规则建筑图像特征点的主方向[11,12],在进行匹配时就可以把不规则建筑图像旋转到主方向,从而保证了建筑图像的旋转不变性,利用下式给出,将建筑图像旋转到(x,y)处时的梯度值和方向分别为

    θ(x,y)=tan-1(L(x,y+1)-L(x,y-1))/

    (L(x+1,y)-L(x-1,y)) (7)

    以不规则建筑图像特征点为中心的邻域窗口内取样,并运用梯度直方图来统计不规则建筑图像邻域像素的梯度方向,直方图的最高点所对应的方向则为主方向。

    内核函数只有+1和-1两种元素,利用它大大的提高了运算速度,这一点对不规则建筑图像处理来说是最为重要的。

    根据不规则建筑图像特征点的位置和建筑图像特征点存在的尺度将以建筑图像特征点为中心的一个区域看成建筑图像子块,并将多障碍场景下不规则建筑图像子块中所有像素点的梯度方向都调整至主方向,获取规范化的建筑图像子块,根据以往经验,取以不规则建筑图像特征点为中心的32×32大小的区域效果最好。

    在得到规范化的建筑图像子块之后,求出该子块的不同方向梯度矩阵Gθ(x,y),即

    Gθ(x,y)=cosθGO(x,y)+sinmGπ/2(x,y) (9)

    式中,GO(x,y)和Gπ/2(x,y)分别代表不规则建筑图像水平方向和竖直方向梯度。为了避免因光照变化和视点变化造成的不利影响,需要式(9)中的不同方向梯度矩阵进行高斯滤波处理

    Gθσ(x,y)=G(x,y,σ)*Gθ(x,y)Ηn+1 (10)

    式中,σ代表不规则建筑图像子块宽度的1/4。利用高斯滤波处理后,远离子块中心点的梯度值对最终布局结果的不利影响也被消除。

    在生成新的不规则建筑图像特征描述算子之后,以欧式距离作为相似度准则对2个不规则建筑图像特征点进行匹配。假设,不规则建筑图像特征点对p和q的特征描述算子分别为Desp和Desq,利用下式对欧式距离进行定义

    d=∑i=0,1,⋯,35(Desp(i)-Desq(i))2/|Gθσ(x,y)| (11)

    最后采用随机抽样一致性算法不规则建筑图像变换模型,利用该模型来对其进行标定。

    4 结束语

    针对传统的多障碍场景下不规则建筑图像布局标定方法中,存在着各种不足,针对所存在的问题,提出基于内核投影的建筑图像布局标定方法。该方法引入超像素技术对不规则建筑图像进行分割,大大增加了建筑图像结构的粒度,在此基础上,利用内核投影技术对不规则建筑图像特征描述子进行改进,以达到降低图像特征描述子维度的目的,以此构建不规则建筑图像变换模型,实现多障碍场景下不规则建筑图像布局标定。实验结果表明,所提方法标定准确率较高、成本耗费较低、运行时间较短,具有一定的应用价值。
