    摘    要：

    传统的多相水平集图像分割算法普遍存在图像分割运行时间较长、成本消耗较大、错误率较高等问题。为了解决上述问题, 提出一种基于树形结构的多相水平集图像分割算法。通过改变图像背景, 完成多相水平集的重新收敛;根据图像亮度对图像背景进行填充, 引入加性算子分裂格式对水平集图像给出统一的数值。以二相水平集分割方法为基础, 利用水平函数进行多相水平图像分割。实验结果表明, 所提算法图像分割运行时间较短、成本消耗较小、错误率较低。

    关键词：

    树形结构; 多相水平集; 图像分割; 水平函数;

    作者简介： 陈娥祥 (1978-) , 女 (汉族) , 福建龙岩人, 硕士研究生, 讲师, 研究方向:电子商务, 软件工程, 计算机科学与技术。;

    收稿日期：2018-12-10

    基金： 电子商务专业核心课程工学结合改革探索;项目编号:DFJG2018002;

    Simulation of Multi-phase Level Set Image Segmentation Algorithm Based on Tree Structure

    CHEN E-xiang

    Dong Fang college of the Fujian Agriculture And Foresty University

    Abstract：

    Traditionally, the multi-phase level set image segmentation algorithm generally has the problems about long running time, high cost and high error rate. Therefore, an algorithm to segment multi-phase level set image based on tree structure was presented. By changing the background of image, the multi-phase level set was converged again. According to the brightness of image, the background of image was filled. Moreover, the additive operator splitting scheme was used to give a uniform value to the level set image. Based on the two-phase level set segmentation method, the level function was used to segment the multi-phase level image. Simulation results show that the proposed algorithm has shorter running time of image segmentation, lower cost and lower error rate.

    Keyword：

    Tree structure; Multi-phase level set; Image segmentation; Level function;

    1 引言

    图像分割是对图像进行处理和分析的基础, 很多专家学者们提出了大量基于不同理论的图像分割算法, 其中, 水平集图像分割算法成为研究人员广泛关注的重点[1,2,3]。但现阶段在多相水平集图像分割过程中, 普遍存在着图像分割运行时间较长、成本消耗较大, 错误率较高等问题, 在这种情况下, 如何有效提出一种具有较好分割效果的多相水平集图像分割算法成为当今社会亟待解决的问题[4,5]。

    文献[6]提出一种基于改进近邻传播分类的图像分割算法。该算法先衡量图像内部中数据点之间的相似度, 充分考虑到图像色彩密度差异对数据点造成的影响;采用密度分类思想设定偏向参数, 引入图像内部数据点邻近位置信息构建图像相似度矩阵, 增强分类内聚性, 完成图像分割。该算法具有较好的图像分割效果, 但是图像分割运行时间较长。文献[7]提出一种基于超像素和模糊聚类的图像分割算法。该算法采用简单的线性迭代分类方法对图像进行多个超像素区域划分, 比较各个超像素区域间特征的相似性, 利用模糊分类算法对这些分割过的区域进行合并, 实现图像有效分割。该算法在一定程度上提高了图像分割精度和分割效率, 但是图像分割成本消耗较大。文献[8]提出一种基于二维直方图双斜率划分的图像分割算法。该算法将二维直方图上的图像像素点之间的关系变换成斜率值, 利用图像斜率值与频率相结合建立一维直方图, 根据双阈值方法对一维直方图取双阈值, 利用取得的双阈值对图像二维直方图进行重新分割, 获取最终图像分割结果。该算法具有较高的图像分割速率, 但是图像分割过程中错误率较高。

    针对上述问题, 提出一种基于树形结构的多相水平集图像分割算法。具体过程如下:

    ①通过改变图像背景, 促使多相水平集在新的图像上可以重新进行收敛, 根据图像亮度的不同, 对图像背景进行填充。

    ②引入加性算子分裂格式对水平集图像给出统一的数值, 以二相水平集分割方法为基础, 利用水平函数进行多相水平图像分割。

    ③仿真实验对比不同方法的图像分割运行时间、成本消耗、分割错误率, 实验结果表明, 所提算法具有一定的应用价值。

    2 多相水平集图像分割算法

    2.1 图像背景填充

    通过改变图像背景, 促使多相水平集在新的图像上可以重新进行收敛, 根据图像亮度的不同, 对图像背景进行填充。具体过程如下:

    在进行目标图像背景填充之前, 需要通过Chan-Vese模型将目标图像背景与前景区域进行分离。利用背景填充方法, 将目标图像平均灰度代替图像背景中每个像素的灰度值, 利用下式给出原图像与填充后的背景组成的新图像:

    u′=u⋅Η (ϕ) +c0 (ϕ) ⋅ (1-Η (ϕ) )     (1)

    背景填充方法能够使目标图像与新的背景拥有一样的平均灰度值, 有效减小了目标图像与背景之间的对比度, 但是这种方法会产生一个特殊的情况, 假设, 当μ=0时, 得到EulerLagrange方程为:

    ϕt=1δ (ϕ) (u-cb-u+c0) (u-c0+u+cb)     (2)

    在此情况下设原图像u不均, 这时目标图像内部就不存在子目标, 则对新图像进行第二次划分, 在新的多相水平集函数相对稳定时, 利用下式可以得到一个新的目标图像和新背景:

    {u′0=u′Η (ϕ′) u′b=u′ (1-Η (ϕ′) )     (3)

    当利用Chan-Vese模型收敛后的目标图像和背景的均值不一样时, 对新图像u′进行第二次划分, 将这次划分看成是一个独立的划分过程, 当目标图像轮廓曲线收敛后得到的新目标的平均灰度与新背景的平均灰度不一致, 即当u′=u·H (ϕ) +c0· (1-H (ϕ) ) 时, c′0≠c′b, 新图像的平均灰度c′0和第一次分割后的目标图像平均灰度c0之间的关系为c′0≠c0。原因在于原目标图像填充后的背景与原目标图像的平均灰度都等于c0, 所以目标图像的平均灰度也为c0, 又因为新图像与新背景的平均灰度满足c′0≠c′b, 所以新图像平均灰度不等于原目标图像的平均灰度, 即c′0≠c0。

    对原目标图像进行第二次划分够得到的新图像和第一次划分后得到的原图像不同, 且新图像位于原图像内部, 因为在新图像的背景填充区域没有生成新的边界, 所以新图像轮廓曲线在曲率项作用下不会在背景填充区域停止, 又因为c′0≠c0, 所以新图像与原目标图像不同, 新图像只能位于原图像内部, 完成图像背景的填充, 使得多相水平集重新收敛。

    针对传统的多相水平集分割方法存在各种问题。提出一种基于树形结构的多相水平集图像分割算法。该算法自身具有易扩展和较稳定的特点, 拥有较快的图像分割运行速度, 水平集图像分割算法与传统的图像分割算法的区别在于该算法只采用一个水平集曲线, 通过二相分割实现对多区域的有效分割, 可有效降低图像分割错误率。实验结果表明, 所提算法图像分割运行时间较短、成本消耗较小、错误率较低, 能够实现对图像多个区域进行准确分割。
