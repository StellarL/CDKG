    摘    要：

    为了提高移动物联网在运行过程中的安全性和准确性, 需要对移动物联网入侵检测模型进行研究。采用当前入侵检测模型对移动物联网进行入侵检测时, 存在检测结果准确率低和节点背离率高的问题。提出一种移动物联网中重复博弈论入侵检测模型, 通过四元组构建重复博弈模型, 在重复博弈模型的基础上得到入侵检测系统和攻击者的收益函数, 采用期望效用函数计算重复博弈模型的收益矩阵, 根据计算结果得到模型的纳什均衡。采用均衡算法将贴现因子引入效益函数中对重复博弈论入侵检测模型进行优化, 并在重复博弈论入侵检测模型中引入惩戒机制, 完成移动物联网的入侵检测。仿真结果表明, 所提模型的检测结果准确率高、节点背离率低。

    关键词：

    移动物联网; 重复博弈论; 入侵检测;

    Simulation of Repeated Game Theory Intrusion Detection Model in Mobile Internet Of Things

    ZHANG Lin-jiang LIU Zhi-long ZHOU Hong-lei LIU Jian-peng

    Henan University of Animal Husbandry and Economy, Network Management Center Network Management Center

    Abstract：

    Research on intrusion detection model can improve accuracy and security of Internet of Things during operational process. Accuracy of detection result of current detection model is low. Meanwhile, deviation rate of node is high. Therefore, a new intrusion detection model based on repeated game theory was proposed in this article. Firstly, repeated game model was built via tetrad. Based on the model, revenue function of intrusion detection system and intruder was obtained. Payoff matrix of the model was calculated via expected utility function and s, Nash equilibrium of the model was obtained according to the calculation result. Then, discount factor was introduced into benefit function to optimize the intrusion detection model using load-balancing algorithm and punishment mechanism was introduced in the intrusion detection model. Thus, the intrusion detection of mobile Internet of Things was completed. Simulation results show that accuracy of detection result of the method is higher and deviation rate of node is lower.

    Keyword：

    Mobile Internet of Things; Repeated game theory; Intrusion detection;

    1 引言

    移动物联网是一种泛在网络, 移动物联网将物联网和无线网络与有线融合, 通过全球定位系统、处理终端和传感器实现人和物、物和物以及网络和物品的连接, 方便用户的控制、管理和识别[1,2]。移动物联网成为世界各国的关注焦点, 研究移动互联网有关的技术具有重要意义。移动物联网采用无线通信方式通过海量微型传感器节点构成自组织网络系统, 感知、收集并处理移动物联网中存在的感知对象数据, 并将数据传输给用户[3]。入侵检测模型在移动物联网的安全运行中起重要作用, 目前大部分入侵检测模型的防御都是被动的, 不能积极的对入侵事件进行判断和分析, 无线信道和节点的开放性、移动性降低了移动物联网的安全性, 不能遏制网络用户的自私行为, 且通常情况下节点之间互相关联、互联影响, 不能集中进行管理[4]。当前移动物联网入侵检测模型存在检测结果准确率低和节点背离率高的问题, 需要对移动物联网中入侵检测模型进行研究[5]。

    石云、陈钟、孙兵提出了一种基于均值聚类分析的网络入侵检测模型, 通过K-means算法得到核心集, 利用核心集替代顶层核心集对移动物联网中存在的异常数据进行定位, 采用KM.MulCA算法构建移动物联网入侵检测模型, 该方法得到的检测结果准确率低[6]。韩红光、周改云提出了一种基于Makov链的网络入侵检测模型, 利用马尔科夫链对移动物联网中存在的事件进行概率建模, 根据隐马尔科夫模型对移动物联网中的入侵事件进行预测和检测, 该方法的节点背离率较高[7]。许勐璠、李兴华、刘海等人提出了一种基于信息增益率和半监督学习的网络入侵检测模型, 根据网络数据受到攻击时的特点, 利用半监督学习算法在模型训练阶段得到训练数据集, 在信息增益率定量的基础上在模型检测阶段分析模型检测性能受不同特征的影响, 完成移动物联网的入侵检测, 该方法得到的检测结果与实际结果不符, 存在检测结果准确率低的问题[8]。

    综上所述, 提出一种移动物联网中重复博弈论入侵检测模型。

    2 移动物联网入侵检测原理

    当前移动物联网入侵检测方法结合主成分分析算法和模糊C均值聚类算法对移动物联网进行入侵检测。采用模糊C均值聚类算法对移动物联网中的样本数据进行聚类, 通过主成分分析算法对聚类后的样本数据做降维处理[9]。将降维后样本主成分的聚类特征集与对应的维度数据进行对比, 设置阈值, 当阈值小于等于比较结果时报警, 完成移动物联网的入侵检测。

    采用主成分分析方法通过下式对样本矩阵X进行线性变化, 将其映射到低维空间向量Y中, 表达式如下

    Y=WTX (1)

    式中, X代表的是移动物联网中的样本矩阵, 通过M个待观测数据变量和N个列向量数据构成, M维中存在的坐标都代表一个数据;W代表的是通过样本协方差矩阵组成的正交投影矩阵, 通常情况下由样本矩阵计算得到;T代表的是转置矩阵。设C代表的是样本的协方差矩阵, 其表达式为

    C=Ν-1∑i=1Ν (X-ui) 2 (2)

    式中, ui为平均值, 存在下式

    CWi=λiWi (3)

    式中, i=1, 2, …, N, Wi代表的是正交矩阵W中存在的第i列样本协方差矩阵;λi代表的是矩阵C中存在的特征值, Wi代表的是特征值对应的特征向量。按照从小到大的顺序对特征值λi进行排列, 组合前L个特征值相应的特征向量, 获得降维处理后的L维数据。

    设U代表的是初始化的随机隶属矩阵, 在区间[0, 1]中取值, 满足下式的约束条件

    设vi代表的是c个聚类中心, i=1, 2, …, c, 聚类中心vi的计算公式为

    vi= (∑j=1nuij⋅xi) /∑j=1nuij, ∀i (5)

    根据式 (5) 计算聚类中心和第i个样本之间的欧几里得距离dij

    dij=‖vi-xj‖ (6)

    通过目标函数J得到模糊最优解

    J=∑i=1C∑j=1nuijmdij2 (7)

    式中, m代表的是模糊加权因子, 目标函数J代表的是各个聚类中心与各类样本之间的距离平方和。设置阈值ζ, 当目标函数值J大于阈值ζ时, 证明样本数据集为入侵数据, 进行报警, 完成移动物联网的入侵检测。

    3 重复博弈论入侵检测模型

    3.1 重复博弈模型

    通过四元组构建重复博弈模型, 根据模型得到攻击者和入侵检测系统的收益函数, 在收益函数的基础上建立攻击策略集合和防御策略集合, 采用期望效用函数得到博弈模型的收益矩阵, 根据收益矩阵算出博弈模型的纳什均衡。

    设RRDM代表的是重复博弈模型, 通过四元组Attender、Action、Profits、Times对模型RRDM进行表示

    RRDM= (Attender, Action, Profits, Times) (8)

    其中, Attender中存在入侵检测系统和攻击者, 将攻击者记为a, 入侵检测系统记作b, 入侵检测系统和攻击者采用的行动空间不一样, Aa= (N, A, M, P) 代表的是攻击者a的行动空间, 其中包含正常、进攻、不正常、预进攻, Ad= (C, R, W, D) 代表的是入侵检测系统的行动空间, 其中包含持续执行、推荐执行、预警和防护。

    设Ua (Aa) 和Ud (Ad) 分别代表的是攻击者和入侵检测系统的收益函数, T代表的是博弈总数, 将式 (8) 转换为下式

    RRDM= (a, d;Aa, Ad;Ua (Aa) , Ud (Ad) ;T) (9)

    通过式 (9) 建立攻击策略集合Sa和防御策略集合Sd, 其表达式分别如下:

    Sa= (SN, SM, SP, SA) (10)

    Sd= (SC, SR, SW, SD) (11)

    式中, SN、SM、SP、SA分别代表的是正常、进攻、不正常、预进攻行动策略;SC、SR、SW、SD分别代表的是持续执行、推荐执行、预警和防护行动策略。博弈双方的行动策略用Sad= (sa, sd|sa∈Sa, sd∈Sd) 表示。

    当行动策略Sad= (SA, SC) 时, 攻击者在移动物联网中获得最高利益;当行动策略Sad= (SP, SD) 时, 入侵检测系统获得最高效益, 即攻击者攻击移动物联网时, 入侵检测系统使用放行策略对攻击者有利, 当攻击者准备攻击移动物联网时, 入侵检测系统采取防范策略[10]。通过以上分析将入侵检测系统和攻击者的行动策略转化为对应的偏好集, 采用期望效用函数将偏好集规范在[0, 1]区间之内, 设U (X) 代表的是博弈双方的效益函数, U (X) 的表达式为:

    U (X) =∑SadΡ1u (x1) +⋯+Ρku (xk) (12)

    式中, Pi代表的是入侵检测系统或攻击者在移动物联网中每次采取不同策略的概率;u (xi) 代表的是不同策略的收益。在理想条件下, 攻击者和入侵检测系统采取不同行动处理的概率均为0.25时, 能够算出博弈模型的收益矩阵, 得到博弈模型的纳什均衡。

    3.2 重复博弈论入侵检测模型优化方法

    采用随机最优反映均衡算法对概率Pi进行调整, 使调整后的概率Pi在移动物联网中的适用性更强, 并将收益因子引入效益函数中, 对重复博弈论入侵检测模型进行优化。

    设P*= (P*1, P*2, …, P*i) 代表的是重复博弈模型中存在的随机最优反映均衡解。其中P*i代表的是在某次博弈中入侵检测系统或攻击者采用某种策略的概率。设P*i= (P*1, P*2, …, P*k) 代表的是入侵检测系统或攻击者的混合策略概率;Δi代表的是混合策略概率P*i的集合, 其表达式为

    Δi={P*i= (P*1, P*2, …, P*k) } (13)

    通过式 (13) 得到入侵检测系统和攻击者的混合策略空间Δ=∏Δi。通过策略空间Δ得到入侵检测系统和攻击者选择行动策略的概率函数π*i

    πi*=λu (xi) /∑k=1iλu (xi) (14)

    式中, λ代表的是加权因子, 当加权因子λ趋近于无穷大时, 随机最优反映均衡接近纳什均衡, 式 (13) 可以得到最终的结果。

    在正常移动物联网中理想概率出现的几率几乎为零, 攻击者会通过不同的攻击方式和策略对移动物联网进行攻击。对式 (12) 和式 (13) 进行优化, 提高重复博弈论入侵检测模型的检测准确率。设δ∈[0, 1]代表的是收益因子, 双方博弈过程中的收益可以通过收益因子进行度量, 收益因子δ的值越大, 代表入侵检测系统或攻击者对整体收益的重视度越高, 收益因子δ的值越小, 代表入侵检测系统或攻击者对整体收益的重视度越低。将收益因子δ引入效益函数中, 得到下式

    U (X) =∑k=1∞ (1-πi*) k-1πi*δk-1 (15)

    式 (15) 为收益函数, 在不同的博弈中博弈双方会主动对行动策略概率进行调整, 使收益值最高, 根据收益函数检测移动物联网中存在攻击行为的恶意节点, 并将恶意节点中存在的数据包删除, 提高移动物联网的安全性。

    3.3 惩戒机制

    通过惩戒机制威慑移动物联网中存在的恶意节点。惩戒机制的主要思想是:有的节点表现出攻击行为, 从下一时隙起, 其它节点不转发以恶意节点为目标的节点。惩戒机制主要分为以下几步:

    1) 如果没有在前一个移动物联网博弈中检测出恶意节点, 所有节点在网络中保持现有状态, 如果在移动物联网中检测到恶意节点, 转到下一步。

    2) 通过惩戒机制对恶意节点ϑ进行惩戒, 在惩戒过程中移动物联网中的其它节点保持原有状态。

    3) 恶意节点执行策略后在移动物联网中的效用值为 (U1, …, UN) 。如果在第三步中存在恶意行为, 返回第二步对恶意节点进行惩戒。

    如果恶意节点ϑ变为正常状态, 那么节点ϑ在背离时隙的最大收益为v¯k, 节点ϑ在惩戒时间Tk中的收益为v˙k, 节点ϑ重新回到正常状态的收益为v′k。可以得到节点ϑ在移动物联网中的平均折现效用值U^k

    U^k= (1-δ) v¯k+δ (1-δΤk) v˙k+δΤk+1vk´ (16)

    如果在博弈过程中节点ϑ一直为正常状态, 收益为vk, 此时节点ϑ的平均折现效用值为Uk

    Uk= (1-δ) ∑t=0∞δvk=vk (17)

    通过式 (16) 和式 (17) 可以得到节点ϑ在移动物联网中的背离收益ΔUk, ΔUk的计算公式为

    ΔUk=U^k-Uk (18)

    式 (18) 中的背离收益一定小于零, 代表节点ϑ的背离收益小于协作收益, 此时任何理性节点都不会在移动物联网中从正常状态中背离。

    4 实验结果与分析

    5 结束语

    移动物联网被广泛的应用在生活服务、军事、医疗和商业等领域中, 一旦在移动物联网中出现了病毒入侵、病毒破坏以及受到攻击时, 会丢失信息资料, 造成巨大损失。当前移动物联网入侵检测模型存在检测结果准确率低和节点背离率高的问题, 提出一种移动物联网中重复博弈论入侵检测模型, 解决了当前方法中存在的问题, 提高了移动物联网的安全性和稳定性。
