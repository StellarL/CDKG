    摘    要：

    对大数据分析下攻击性数据进行超前安全预警能够识别攻击性数据的攻击手段, 对保证网络安全具有重要意义。针对当前大数据分析下攻击性数据超前安全预警方法存在的预警及时性较差, 且无法准确对攻击性数据进行识别的问题, 提出一种基于RBF神经网络的大数据分析下攻击性数据超前安全预警方法, 通过计算攻击性数据样本的聚类中心, 对大数据分析下攻击性数据样本进行自适应聚类, 并通过对径向基函数的宽度进行扩展, 实现大数据分析下攻击性数据样本自适应分类, 根据分类结果, 构建RBF神经网络, 确定攻击性数据超前安全预警的目标函数, 并对RBF神经网络参数进行调整, 建立大数据分析下攻击性数据超前安全预警模型, 实现攻击性数据超前安全预警。实验结果表明, 所提方法对攻击性数据识别速度较快, 保证了预警及时性, 且对攻击性数据的识别准确度较高。

    大数据分析; 攻击性数据; 超前; 安全预警;

    Big Data Analysis Under Attack Data Advance Security Early Warning Simulation

    College of Mobile Telecommunications, Chongqing University of Posts and Telecommunications

    Abstract：

    This article puts forward an advancing security precaution method of aggressive data under big data analysis based on RBF neural network. Firstly, our research carried out adaptive clustering for sample of the aggressive data via calculating center of clustering of the sample and achieved adaptive classification of the sample via extending width radial basis function, then built the RBF neural network according to classification results. The research confirmed objective function of the advancing security precaution and adjusted parameter of the RBF neural network to build model of the advancing safety precaution. Thus, we achieved the advancing security precaution. Simulation results show that the method has fast recognition speed for the aggressive data. It ensures timeliness of the precaution and has higher recognition accuracy.

    Big data analysis; Aggressive data; Advancing; Security precaution;

    1 引言

    计算机互联网技术的发展与普及使得网络中的数据量不断增加, 为保证数据的利用效率, 需要对网络中的大数据进行分析[1]。但在分析过程中, 一些数据中夹带病毒等攻击手段, 影响网络的安全[2]。为保证网络安全, 数据的安全得到了人们的高度重视。网络数据中的病毒具有较强的模糊性和随机性, 导致当前预警方法难以准确对攻击性数据进行识别[3]。攻击性数据超前安全预警作为保证数据安全的新手段, 同时也受到了相关研究者们的重视, 许多学者在不同角度对其进行不断深入研究, 已取得了一些应用价值较高的研究成果[4]。

    文献[5]主要采用攻击模式识别对攻击性数据进行预警, 该方法通过对攻击性数据进行因果分析, 确定攻击性数据的攻击意图以及当前的攻击阶段, 并根据攻击阶段进行数据安全态势评估, 建立攻击阶段状态转移图, 进而实现攻击性数据的安全预警。但该方法存在计算过程复杂、预警精确度较低的问题。文献[6]方法通过对大数据分析下安全数据进行场景聚类, 确定攻击性数据的攻击目标, 并对攻击性数据的每一攻击场景进行因果关联, 确定攻击性数据的攻击轨迹和阶段, 建立安全态势量化标准, 并结合攻击性数据的威胁指数, 实现攻击性数据安全预警。但该方法缺少对攻击性数据属性的深入分析, 进而导致攻击性数据识别精度低、数据安全预警效果不佳的问题。文献[7]提出一种基于多源事件融合的攻击性数据安全预警方法, 通过对网络大数据进行预处理, 统一数据格式, 并通过数据属性相似度计算的方法, 以及DS证据理论, 对攻击性数据的因果关系进行关联融合。通过模糊综合评价和层次分析相结合的方法确定攻击性数据的威胁权值系数, 并通过融合攻击性数据的各因素态势, 构建攻击性数据的安全态势图, 实现攻击性数据安全预警。但这种方法应用过程中, 数据的处理效果较差, 影响安全预警的准确度。

    5 结束语

    网络的发展推动大数据时代的到来, 对数据的处理利用逐渐增多。但数据中的病毒等攻击手段对网络安全产生不利影响。对大数据分析下攻击性数据进行超前安全预警作为保证网络安全的重要方法, 受到了国内外许多专家学者的关注。针对当前数据分析下攻击性数据超前安全预警方法存在的预警及时性较差, 且无法准确对攻击性数据进行识别的问题, 提出一种新的攻击性数据超前安全预警方法, 实验结果表明, 与当前方法相比, 采用所提方法能够有效解决当前方法存在的问题, 具有较高的应用价值。
