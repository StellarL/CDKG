    存在参考站位置误差的三星时差定位算法

    任凯强 孙正波

    摘    要：

    在三星时差定位系统中,通常需引入位置已知的参考站来校正星历误差和时间同步误差对定位精度的影响,然而当参考站位置存在误差时,便无法实现高精度校正。为解决上述问题,首先推导了存在参考站位置误差、星历误差和时间同步误差时三星时差定位系统的定位精度CRLB。然后利用线性均方最小均方误差(Linear Minimum Mean Square Error,LMMSE)估计准则对参考站位置误差进行估计,并对参考站位置进行更新;最后将更新后的参考站位置带入定位方程,减少参考站位置误差对定位精度的影响。仿真结果表明,上述算法可以有效降低参考站位置误差,且对目标辐射源定位精度可达克拉美-罗界(Cramer-Rao Lower Bound, CRLB)。

    关键词：

    三星时差定位系统; 参考站位置误差; 星历误差; 时间同步误差; 克拉美-罗界;

    A Tri-satellite TDOA Localization Algorithm Using Reference Stations at Inaccurate Locations

    REN Kai-qiang SUN Zheng-bo

    National Key Laboratory of Science and Technology on Blind Signal Processing

    Abstract：

    Reference stations, whose locations are known, are always used to calibrate location error caused by ephemeris error and synchronization error in tri-satellite TDOA localization system. When the exact locations of reference stations are not available, it is unable to the calibrate location error of the target source. Firstly, the paper derived the CLRB of tri-satellite TDOA localization system with reference station location error ephemeris error and synchronization error, then, proposed an algorithm based on LMMSE to estimate the location error of reference stations. Finally, the refined locations of reference stations were used in the localization equations of target source to reduce the location errors caused by location error of reference stations. Simulation results demonstrate that location error of the proposed algorithm can reach CRLB.

    Keyword：

    Tri-satellite TDOA localization system; Location error of reference stations; Ephemeris error; Synchronization error; CRLB;

    1 引言

    三星时差定位系统是星载无源定位的重要手段之一,其原理是利用辐射源信号达到三颗卫星的到达时间差实现对辐射源的定位,具有定位精度高、作用距离远、隐蔽性好等优点,广泛应用于移动通信和卫星导航等领域[1,2,3]。在实际应用中,由于星历误差和时间同步误差的存在,严重影响了三星时差定位系统的定位精度[4,5,6,7]。

    有源校正是定位系统中常用的误差校正技术,其原理是通过引入位置已知的参考站增加对定位系统的观测,来消除星历误差和时间同步误差对定位精度的影响。很多行之有效的有源校正算法相继被提出,能够较好地解决星历误差和时间同步误差导致的三星时差定位系统定位精度恶化的问题[8,9,10,11,12,13,14],文献[14]提出一种基于正交投影的两步迭代有源校正算法,该算法可在三参考站时基本消除星历误差和时间同步误差对定位精度的影响,且定位精度可达CRLB。

    在实际应用中,当参考站GPS受到干扰或参考站为非合作参考站时,参考站的准确位置难以获取。绝大部分现有的有源校正算法都假定参考站位置是精确已知的,未考虑参考站位置误差对定位精度的影响。

    本文在文献[14]算法的基础上,进一步考虑存在参考站位置误差时,三星时差定位系统如何准确定位的问题。首先推导了存在星历误差、时间同步误差和参考站位置误差时,三星时差定位系统定位精度的CRLB;然后利用线性均方最小均方误差估计准则对参考站位置误差进行估计,并对参考站位置进行更新;再将更新后的参考站位置带入定位方程,利用文献[14]所提的定位算法求解辐射源位置;最后采用计算机仿真验证了本文算法的有效性。

    2 问题建模

    考虑地球表面静止辐射源,其所在位置的经度和纬度分别为θ和φ,记a=[θ,φ]T。假定地球模型为球模型,则辐射源在地心坐标系下的位置矢量u=[x,y,z]T可表示为

    三颗卫星星历的观测值为

    si=sio+δi (2)

    式中:sio为第i颗卫星的真实位置,δi为对应的星历误差。令s=[s1Τ,s2Τ,s3Τ]T,so=[s1oΤ,s2oΤ,s3oΤ]T,δ=[δ1Τ,δ2Τ,δ3Τ]T,并假定δ为零均值、协方差矩阵为Qs的高斯白噪声。

    记c为电磁波传播速度,ti1为第i颗卫星相对于第1颗卫星接收到辐射源信号的TDOA观测值,则对应的到达距离差(Range difference of arrival, RDOA)为ri1=c·ti1。在时差定位中TDOA与RDOA是等效的,为便于表示,下文中均采用RDOA形式。辐射源信号的RDOA观测值为

    ri1=ri1o+τi1+ni1 (3)

    式中:ri1o=‖u-sio‖-‖u-s1o‖为对应的真实RDOA,τi1为第1颗卫星与第1颗卫星之间时间不同步引起的RDOA偏差,ni1为RDOA观测噪声。

    为了校正星历误差和时间同步误差,引入M个位置精确已知的自主参考站和N个位置已知但存在位置误差的非合作参考站,记自主参考站真实位置为cjo,j=1,2,…M,已知的非合作参考站的位置为

    cj=cjo+ej,j=M+1,…,M+N (4)

    式中,cjo为非合作参考站的真实位置,ej为非合作参考站的位置误差,可假定其为零均值、协方差矩阵为Qc的高斯白噪声。若三颗卫星同时或在较短时间间隔内接收辐射源和参考源信号,则可认为辐射源信号和参考源信号对应的星历误差和时间同步误差是相同的,因此参考源信号的RDOA观测值为

    r∼i1j=r∼i1jo+τi1+n∼i1j (5)

    式中:r∼i1jo=∥cj-sio∥-∥cj-s1o∥为参考源信号RDOA真实值,ni1j为RDOA观测噪声。

    将式(3)和式(5)改写成矩阵形式:

    m=mo+Hτ+n (6)

    式中

    一般假定n为零均值协方差矩阵为Qr的高斯白噪声,⨂表示Kronecker积,1(M+1)×1是元素为1的列矢量,I2是2维单位矩阵。

    6 结论

    本文针对三星时差定位系统中,参考站位置误差使得定位系统无法有效校正星历误差和时间同步误差的问题,提出一种基于LMMSE的参考站位置估计算法。计算机仿真结果表明,本文算法可有效降低参考站位置误差,从而提高对目标辐射源的定位精度,且定位精度可达CRLB。本文算法适用于无法获得精确的参考站位置情况下三星时差定位系统的有源校正,具有较高的普适性和应用价值。
