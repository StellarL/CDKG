    摘    要：

    警示性图像颜色纹理特征选取的有效性直接影响着多个领域的发展。针对当前方法存在的警示性图像颜色纹理特征选取准确率低的问题, 提出了一种基于维纳滤波和增强矩阵的警示性图像颜色纹理特征选取方法, 采用维纳滤波对警示性图像中的噪声进行消除, 噪声消除后利用横向增量与纵向增量计算警示性图像中像素点的灰度值, 求解每个像素点灰度值所对应的横纵坐标轴的二阶偏导数之和, 根据这个偏导数推导出警示性图像的颜色纹理特征矩阵, 利用该矩阵得到了一个警示性图像的颜色纹理特征模型, 通过这个模型实现了对大数据处理下警示性图像颜色纹理特征的选取。仿真结果表明, 所提方法对警示性图像的噪声具有很好的抑制效果, 可以较好的保留警示性图像中的信息, 并且所提方法能够准确的在大数据处理环境下对警示性图像颜色纹理特征的选取。

    关键词：

    大数据处理; 警示性图像; 颜色纹理; 特征选取;

    Big Data Processing Warning Image Color Texture Feature Selection Simulation

    ZHANG Zi-dong ZHANG Jie-min MAO Jian

    Computer Engineering College, Jimei University

    Abstract：

    Current feature selection method of color texture of warning image has many disadvantages, such as poor selection accuracy. In order to overcome the disadvantages, this research proposed a new feature selection method based on Wiener filtering and enhanced matrix. Noise of the warning image was eliminated via the Wiener filtering. After that, gray value of pixel in the warning image was calculated via horizontal increment and vertical increment. Sum of second partial derivative of abscissa axis and ordinate axis corresponding to the gray value of each pixel was solved. According to the partial derivative, Feature matrix of the color texture was deduced. A model of feature of the color texture was obtained via the matrix. The feature selection was achieved via the model. Simulation results show that the method has excellent inhibitory effect for noise of the warning image. It can reserve information of the image preferably and select the feature of color texture under big data processing accurately.

    Keyword：

    Big data processing; Warning image; Color texture; Feature selection;

    1 引言

    20世纪90年代时, 人们通过以图找图的方式实现对警示性图像的检索。随着互联网技术的快速发展, 互联网中交互的数据急剧增长, 人们步入了大数据时代[1]。21世纪时, 采用以图找图的警示性图像检索方式已经不能满足用户对警示性图像检索的需求。研究出一种能够准确及时的处理警示性图像颜色纹理特征选取方法是非常重要的。为了满足大数据时代下用户对图像处理的需求, 提出了以下几种图像特征选取方法[2]。

    文献[3]提出了一种基于检测算子的图像特征选取方法, 该方法利用检测算子获取图像特征点构造的最小生成树, 定义特征点选取准则, 利用删除最小生成树的手段选取符合准则的一组特征点, 实现对图像特征的选取。文献[4]提出了一种基于图像深度层次特征提取的图像特征选取方法, 该方法选取卷积输出层的特征图构建图像阶层结构, 采用信息熵描述低层级特征图, 采用区域平均的方法描述高层级特征图, 构建具有较强表达能力的特征, 实现对图像的特征的选取。文献[5]提出了一种基于二进制十字对角纹理矩阵的图像特征提取方法, 该方法首先提取煤岩图像的二进制十字对角纹理矩阵, 然后利用二进制十字对角纹理矩阵提取图像特征向量, 结合稀疏表示实现对图像特征的选取。

    上述方法均存在着警示性图像颜色纹理特征选取准确率低的问题, 为了解决这个问题提出一种基于维纳滤波和增强矩阵的警示性图像颜色纹理特征选取方法。

    2 图像颜色纹理特征选取的原理

    计算警示性图像的灰度共生矩阵, 得到计算结果后利用定义的熵、逆差距、角二阶距、对比度和相关系数等特征量对图像的颜色纹理进行分析, 从而实现对图像颜色纹理特征的选取。

    假设有一幅灰度量化的警示性图像, 图像的量化级别为L, 图像中两个像素点之间的距离是l, pl, θ (i, j) 是顺着θ方向在距离为d= (Δx, Δy) 的两个像素点的出现次数。它们的像素值分别用i和j表示。θ的取值通常情况下为如下0°、45°、90°和135°几个方向, 像素点的d和θ能够改变l得到不同的组合方式, θ是通过计算d的得到的。利用这种方式能够提取到不同的警示性图像的颜色纹理特征。

    1) 熵

    3 大数据处理下警示性图像颜色纹理特征选取

    采用维纳滤波对警示性图像进行去噪操作, 噪声去除后利用横、纵向两个增量计算警示性图像中像素点的灰度值, 计算警示性图像中每个像素点灰度值所对应的横纵坐标轴二阶偏导数之和, 根据这个偏导数求解警示性图像的颜色纹理特征矩阵, 利用警示性图像颜色纹理特征矩阵得到了警示性图像的颜色纹理特征模型, 通过这个模型实现了对大数据处理下警示性图像颜色纹理特征选取。

    本次实验平台是由7台操作系统均为Windows10、十核、内存是1TB的计算机所组成。为了更加准确地评价所提方法的有效性和可行性, 选取了一组警示性图像作为实验样本, 从不同角度衡量警示性图像的质量, 分别采用文献[3]方法、文献[4]方法、文献[5]方法和所提方法对颜色纹理特征提取效果进行对比。

    总体来说, 通过实验验证, 所提的基于维纳滤波和增强矩阵的警示性图像颜色纹理特征选取方法实现了大数据处理警示性图像颜色纹理特征的选取, 并且采用所提方法处理图像时, 图像的还原程度与特征点的提取准确率均是几种方法中最高的。

    5 结论

    在大数据处理环境下提取警示性图像颜色纹理特征时, 首先需要对警示性图像进行去噪处理, 将那些无用的图像信息去除, 利用横向增量与纵向增量计算警示性图像中像素点的灰度值, 每个像素点的灰度对图像数字化显示形成了一个警示性图像的灰度矩阵, 计算警示性图像中每个像素点灰度值所对应的横纵坐标轴二阶偏导数之和, 根据这个偏导数求解警示性图像的颜色纹理特征矩阵, 利用警示性图像颜色纹理特征矩阵得到了警示性图像的颜色纹理特征模型, 通过这个模型可以达到对大数据处理下警示性图像颜色纹理特征选取的目的, 具有高图像还原度、高图像特征点提取准确率的特点。

    接下来的工作方向是在所提方法的基础对图像的特征点提取准确率进行进一步的研究, 使警示性图像颜色纹理特征的提取准确率更高。
