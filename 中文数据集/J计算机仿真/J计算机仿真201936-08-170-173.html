    摘    要：

    在多媒体网络中, 图像目标的动态特征与噪声干扰比较明显, 加之传统图像检测算法对于图像旋转的敏感, 导致其图像目标检测性能欠佳。为了提高图像检测的准确性与稳健性, 提出了改进局部二值二次图像多特征检测算法, 同时对图像的纹理特征、颜色特征、边界特征做提取, 从图像多种特征进行检测。算法利用原始图像分割得到二次图像, 根据二次图像灰度设计了改进局部二值特征提取, 得到图像局部具体的差异度, 并对各个差异度做加权调整, 再结合感知压缩降低计算维数, 根据差值平均进行特征矢量编码量化, 采用汉明距离计算并判断图像的相似性。通过CBIR图像库结合Matlab工具搭建仿真平台, 验证了在多媒体网络图像目标检测中, 改进局部二值二次图像多特征检测算法能够更好的处理图像动态特性, 具有更好的准确性。

    关键词：

    图像检测; 局部二值; 差异度; 特征提取; 编码量化;

    Simulation Research of Multimedia Network Image Target Detection Algorithm

    HUANG Pin

    Guangdong Ocean University

    Abstract：

    In multimedia networks, the dynamic characteristics of image targets and noise interference are more obvious. In addition, the traditional image detection algorithm is sensitive to image rotation, resulting in poor image target detection performance. In order to improve the accuracy and robustness of image detection, an improved local binary image multi-feature detection algorithm was proposed. At the same time, the texture features, color features and boundary features of the image were extracted, and various features of the image were detected. The algorithm used the original image segmentation to obtain the secondary image. According to the secondary image grayscale, the improved local binary feature extraction was designed to obtain the local specific degree of difference, and the weighted adjustment was made for each difference degree. Then combined with the perceptual compression, the computational dimension was reduced. The feature vector coding was quantized according to the difference average, and the Hamming distance was used to calculate and judge the similarity of the image. The simulation experiment platform was built with CBIR image library combined with Matlab tool. It is verified that the improved local binary image multi-feature detection algorithm can better process image dynamic characteristics and achieve better accuracy in multimedia network image target detection.

    Keyword：

    Image detection; Local two value; Difference degree; Feature extraction; Coding quantization;

    1 引言

    多媒体网络是在多媒体和现代网络技术基础之上发展而来的, 其网络中有大量的图像需要传输与存储, 如何对这些大量图像进行管理和检索, 就成了亟待解决的难题。网络中复杂的噪声与安全攻击, 以及网络图像自身的动态特性, 将严重影响图像的检测识别, 因此, 关于多媒体网络图像目标的检测成为研究热点。

    现有的图像检测算法中, 图像哈希处理最为引人注目[1,2,3,4], 其稳健性较现有的其它方法都要明显, 且它的改进优化空间更大, 与其它算法结合容易, 因此, 在哈希图像基础上提出了一些改进优化算法。文献[5]基于数据依赖关系提出了多索引哈希, 该方法通过调整投影方式, 将特征向量哈希表对应元素下标近似平均分布, 有利于改善检索速度, 但是该方法没有考虑到投影数据可能导致原始数据失真的情况;文献[6]设计了相关性函数, 并根据该函数估算出数据和编码关系, 从而得到线性哈希编码, 该方法由于编码更加合理, 对图像描述的失真程度得到降低, 因此取得了一定的检测精度, 但是该算法忽略了局部信息, 且抗噪能力不佳;文献[7]通过引入局部信息, 设计出基于加权局部信息哈希图像检索, 该方法通过图像原始信息的线性处理, 构建出对应的局部信息, 然后对其进行投影量化等处理, 最终完成图像检测, 该方法提高了检测准确率, 但是计算复杂度过高, 图像检测的实时性不好。

    为了更好的实现多媒体网络图像目标检测的性能, 本文基于哈希图像设计了改进局部二值二次图像多特征检测算法, 为了提取到准确合理的特征向量, 该算法利用图像像素距离得到二次图像, 并利用灰度进行局部二值特征的提取, 从而得到具体的差异度值。为了体现图像的纹理特征, 提高检测的准确性, 调整差异度权值, 再采用感知压缩降维处理, 降低计算复杂度。

    2 图像目标检测原理

    多媒体网络图像目标检测通常包含两个任务, 其一是线下学习, 一般完成图像特征的编码与压缩, 并更新训练数据集;其二是在线查询, 主要是提取出图像包含的特征, 并利用距离等方式进行图片检索。由于哈希在处理图像的过程中, 所需的存储与能耗少, 且检测快, 其改进优化的空间也很大, 所以在图像目标检测领域备受关注。

    在进行图像特征编码压缩的时候, 对图像目标采用哈希编码, 其实质是把高维数据转换至低维空间内进行处理, 对于任一向量X=[x1, x2, …, xn]⊂Rd, 若要将其转换至b维的二值空间内, 则对应的哈希函数集合为{h1, h2, …, hb}, 集合中的每个元素均为二值哈希函数, 向量X向函数集的映射关系可以表示成哈希码Hi∈{0, 1}b, 哈希函数描述为

    h (x) =sgn (ωΤx+t)     (1)

    这里, x⊂Rd表示原始数据对应的特征向量, ω⊂Rd表示映射向量, t表示偏移量。根据式 (1) , 可以把原始数据特征向量转换至哈希码。

    在进行图像检测的检索阶段, 计算图像之间相似度的时候, 大多采用匹配距离的方法, 一般都是对哈希码计算距离, 因此, 采用哈希技术进行多媒体网络图像目标检测的过程可以描述为:

    1) 对图像目标进行分割, 得到目标图像的候选区;

    2) 对分割区的图像特征进行提取;

    3) 对图像的特征向量利用哈希技术做量化与编码处理;

    4) 利用距离匹配算法得到图像间的相似性。

    3 改进局部二值二次图像特征检测算法

    3.1 改进局部二值二次图像多特征提取

    将多媒体网络图像做环形分割, 利用图像中各个像素到图像中心距离, 得到关于像素距离的集合, 根据距离的排序和插值计算, 得到一个新图像, 这里将其叫做二次图像, 为了有效处理图像旋转问题, 本文采取二次图像分析。

    局部二值算子[8]是将图像以某矩形作为参考, 矩形中的像素采取二值化计算, 比较基准采用该矩形中心对应灰度值, 通过各个像素的加权和求出局部二值, 求解过程如下:

    QLBΡΡ, R=∑i=1ΡS (gi-gc) 2i-1, i=1, 2, ⋯, Ρ    (2)

    其中, gc是矩形中心对应的像素, gi是以R为半径的圆环上像素i, 2i-1为像素加权。

    利用局部二值可以很好的描述图像纹理特征, 然而, 该方法易受噪声影响, 且不能得到局部的具体差异值, 导致其稳健性不好, 因此, 本文通过二次图像特征提取, 对其进行改进, 提高其准确性。根据式 (2) , 对二次图像灰度做如下处理, 得到像素记录

    Τ=∑i=1ΡS (fi-fc) 2i-1, i=1, 2, ⋯, Ρ    (3)

    其中, fc表示邻域像素, fi表示当前像素, 此时的比较函数描述为

    S (x) ={1, x≥00, x<0    (4)

    根据阈值比较, 可以发现图像中像素灰度的细微差别, 从而可以识别出其中的噪声像素。另外为了得出邻近像素差异的不同程度, 这里采用如下函数计算差异度

    (fi-fc) ={0, fi-fc<Τ12[1+2πarctan (-fi-fcd) ], otherwise    (5)

    这里d为相似度距离, 其计算表达式为

    d= (fi-fc) 2fi+fc    (6)

    根据式 (6) 可以看出, 相似度距离越大, 表明像素间越相似, 与式 (5) 一起完成了对像素差异的有效描述。

    采用递增方式进行差异度排序, 从而对差异度的加权进行重新调整, 经过排序后得到的局部二值如下

    QLBΡΡ, R={0, Τ=0∑i=1ΡΗ (fi-fc) 2i-1, R≤Τ≤Ρ-10, Τ=Ρ    (7)

    由此可知, 如果T=0, 则说明像素记录数量为0, 此像素是孤立状态;如果T=P, 则说明此像素可能是孤立的, 也可能是处于平滑区。将改进局部二值做归一化得到如下公式

    Η (b) =1Μ×Ν∑i=1Μ∑j=1Νf[Q (i, j) , b], b∈[0, S]    (8)

    把经过归一化后的H (b) 作为特征矢量, 则有F=[v1, v2, …, vN]。由于改进后的局部二值法是采用超过中心像素灰度所对应的像素数作为比较基准的, 因此提高了抗干扰能力, 同时引入相似度距离计算, 用以代表像素差异, 表述图像像素在空间中存在的联系, 增加差异度大的像素加权, 有利于体现图像的纹理特征, 经过比较舍弃掉不符合条件的像素, 增加了检测的准确性。

    为了进一步提高多媒体网络图像目标检测性能, 除了采用二次图像处理, 本文在提取图像特征的过程中, 还对图像颜色和图像形状进行特征提取, 这样可以提高对简单图像目标的检测速度, 以及对复杂图像目标的检测精度。

    对于任何图像, 颜色是最为明显的特征, 这里采用HIS模型进行颜色特征分析, 假定某分割区中包含n个像素, 则图像的一阶色矩可以表示为

    Μ=1n∑i=1nΗ (pi)     (9)

    式中, H (pi) 是分割区p中像素i对应的色调分量。

    图像边界能够较好的对形状进行描述, 因此在提取形状特征时, 转换为对边界特征的提取。假定构成图像边界的离散点表示为 (x (n) , y (n) ) (n=0, 1, 2, …, N-1) , 将N作为边界循环周期, 可以得到如下复变函数

    u (n) =x (n) +jy (n)     (10)

    于是, 其对应的傅里叶变换表达式为

    f (k) =∑n=0Ν-1u (n) exp (-j2πknΝ) , 0≤k≤Ν-1    (11)

    由复数f (k) 构成的序列{f (k) }即为边界特征。

    4 仿真分析

    4.1 仿真设置

    为了检验本文方法在多媒体网络图像目标检测方面的性能, 本文采用CBIR图像库获取目标图像, 该图像库具有多达十种以上的自然图像, 同时, 通过Matlab从图像库中选取仿真所需图像目标。仿真过程中, 分割环形数量n=6, 半径R=1, P=8。采用文献[7]提出的局部信息加权哈希算法作为图像检测仿真对比。

    5 结束语

    多媒体网络中的图像检测容易遭受噪声干扰, 现有的图像目标检测算法无法较好地处理复杂噪声, 以及图像的几何变换, 为此本文提出了改进局部二值二次图像多特征检测算法。为了提高图像特征提取精度, 首先采用环形分割得到二次图像, 并根据二次图像灰度设计了改进局部二值方法, 从而得出图像的具体差异度, 提高了算法的抗噪能力。然后对差异度进行感知压缩, 将其差值平均采取特征矢量编码, 这种优化后的哈希编码对于图像特征的描述更准确。最后采用汉明距离计算并判断图像的相似性。本文在特征提取阶段, 分别对图像的纹理、颜色、边界进行特征识别, 在实际检测过程中, 特征简单的图像不需要完成所有特征对比即可得出检测结果, 特征复杂的图像则通过多种特征对比提高检测精度。通过实验结果, 证明了改进局部二值二次图像多特征检测算法具有更好的抗干扰能力和更高的检测准确性, 能够有效检测发生几何改变的图像。
