    基于组合核函数SVM的中小流域流量预测研究

    刘昕玥 姚建国 万定生

    河海大学计算机与信息学院 淮河水利委员会水文局

    摘    要：

    传统支持向量机在水文预测中大多采用单一核函数,忽略了核函数的选择对模型预测结果的影响。研究了线性核函数、多项式核函数以及径向基核函数对中小流域流量预测的影响,建立基于组合核函数的流量预测支持向量机模型,将线性核函数与多项式核函数线性组合,运用改进的自适应遗传算法对组合系数进行寻优。通过对屯溪流域汛期小时流量进行预测研究,表明在最优参数下组合核函数的支持向量机预测效果优于单一核函数,从而证明上述模型具有可行性,并在预测精度方面有所提高。

    关键词：

    支持向量机; 组合核函数; 自适应遗传算法; 流量预测;

    Research on Middle and Small Basins Prediction Model Based on SVM with Mixed Kernel Function

    LIU Xin-yue YAO Jian-guo WAN Ding-sheng

    College of Computer and Information, Hohai University Bureau of Hydrology,Huaihe River Commission

    Abstract：

    Traditional support vector machines mostly use single kernel functions in hydrological forecasting, ignoring the influence of the choice of kernel functions on the model prediction results. This paper studied the influence of linear kernel function, polynomial kernel function and radial basis kernel function on small and middle basins prediction, and a flow prediction support vector machine model based on mixed kernel function was established. Linear kernel and polynomial kernel function were linearly combined, and an improved adaptive genetic algorithm was used to optimize the combination coefficients. Through forecasting the flood hourly flow in Tunxi valley, it is shown that the SVM with combined kernel function is superior to single kernel function in the optimal parameters, which proves that the model is feasible and has improved prediction accuracy.

    Keyword：

    Support vector machine(SVM); Mixed kernel function; Adaptive genetic algorithm; Flow prediction;

    1 引言

    以统计学习理论作为基础的支持向量机[1](Support Vector Machine,SVM)在处理小样本数据上具有较为突出的优势,且结构风险最小化理论保证了其具备良好的推广能力[2]。近年来,对传统支持向量机模型进行改进或是与其它方法相结合逐渐成为研究热点[3]。为了确保预测结果更为准确,核函数的选择及参数优化成为支持向量机预测模型的关键。

    然而,在国内外很多运用SVM进行水文预报的研究中,大部分研究直接采用单一的径向基核函数[4,5,6,7],缺少对核函数选择的研究。因此,本文提出基于多项式核函数与线性核函数的组合核函数的中小流域流量预测SVM模型,并采用改进的自适应遗传算法对组合核函数系数进行求解,以屯溪流域汛期小时流量数据进行实验分析。

    2 基于支持向量机的中小流域流量预测模型

    2.1 支持向量机原理

    支持向量机是一种以结构风险最小化原理为依据的统计学习机器,其目标在于构造最优超平面。SVM的基本思想是通过一个非线性映射Φ:x∈Rn→H将样本映射到高维特征空间H。假设训练样本集为{(x1,y1),(x2,y2),…,(xn,yn)},其中xi∈Rn表示输入向量,yi∈Rn为目标值,n为样本数。则目标函数为

    min[12∥ω∥2+C∑i=1n(ξi+ξi*)] (1)

    其中,C为惩罚因子,ξi*、ξi为引入的非负松弛变量,ε为不敏感损失函数。最终得到函数表达式为

    f(x)=∑i=1n(αi-αi*)Κ(xi,x)+b (3)

    αi和αi*为引入的拉格朗日乘子,K(xi,x)称为核函数(kernel function),b为偏置量。

    2.2 SVM核函数

    核函数将原输入空间中两个简单函数的求值作为高维特征空间中两点的内积运算结果。通常使用的核函数有线性核、多项式核、高斯径向基核以及sigmoid核等。其中,线性核函数的形式为K(u,v)=u*v,多项式核函数的形式为K(u,v)=(g*(u,v)+c)d,c≥0,d=1,2…,高斯径向基核函数的形式为Κ(u,v)=exp(-∥u-v∥22σ2),sigmoid核函数的形式为K(u,v)=tanh(g*u*v+c),其中g为常数。

    2.3 基于SVM的流量预测模型

    2.3.1 数据选择及处理

    3 基于组合核函数的SVM模型应用研究

    3.1 核函数的组合方法

    组合核函数是指将两种或两种以上单一核函数进行组合后得到的新的核函数。多项式核函数是一种典型的全局性核函数,线性核函数是一种局部性核函数,考虑到这两种核函数在应用中的不足及优势,提出将这两种单一核函数线性组合作为新的核函数。构造组合核函数如下

    K(u,v)=ρ(u*v)+(1-ρ)(g*(u,v)+c)d (7)

    其中,0<ρ<1。

    在上述函数中,单一核函数在组合中所占的比重由组合系数ρ决定,该系数值通过改进的自适应遗传算法确定。

    3.2 改进的自适应遗传算法

    3.2.1 自适应遗传算法

    遗传算法[8](Genetic Algorithm,GA)中交叉概率Pc和变异概率Pm的选取在很大程度上会影响遗传算法的性能,需要进行多次实验才能确定。针对这种缺陷,M.Srinivas等人[9]提出自适应遗传算法(Adaptive Genetic Algorithm,AGA),使Pc和Pm根据适应度值自动调整,即当种群中的个体适应度收敛到局部最优时,增加Pc和Pm;相反,当适应度较发散时,减少Pc和Pm。改进后Pc和Pm的计算公式分别为

    3.2.2 改进的自适应遗传算法

    自适应遗传算法仍然会出现早熟现象,针对这种缺陷,引入种群最优个体ρbest和最小适应度fmin,并在三个基本操作中增加插入操作。具体步骤可总结如下:

    1)初始化种群X,令fmax=max[f(1)],其中f(1)为初始种群适应度值的集合,即f(1)=[f11,f12,…,f1n],n为种群个数,ρbest为对应最大适应度值的染色体编码,适应度函数采用均方根误差RMSE的倒数;

    2)对第i-1代种群进行选择、交叉、变异、插入操作后得到第i代种群及其适应度值,即ρ(i)和f(i)。如果第i代种群中的最大适应度值max[f(i)]大于前i-1代的最大适应度值fmax,则令fmax=max[f(i)],ρbest更新为相应最大适应度值的染色体编码;

    3)如果fminfmax>a,0<a<1,则认为种群可能陷入局部最优,为跳出局部最优解,对种群进行初始化;

    4)如果达到迭代次数则停止循环,此时fmax为最佳适应度值,对应的ρbest即为所求的最优解。否则重复步骤2)-3)。

    3.3 实验分析

    实验采用本文2.3.1节中的数据。设初始种群个数为20,种群迭代次数为100,惩罚因子C的取值范围为[0.1,10],核函数参数g的取值范围为[0.1,10]。通过改进的自适应遗传算法不断进行寻优,得出最优的组合系数ρ为0.3850,即组合核函数为K(u,v)=0.3850(u*v)+0.615(g*(u,v)+c)d,此时RMSE为9.3012。实验结果对比情况如图5。

    从图5可以看出,将线性核函数与多项式核函数线性组合后的SVM预测模型误差比分别采用两种单一核函数时的误差小,即采用基于组合核函数并运用改进的自适应遗传算法进行参数寻优的SVM预测模型的拟合效果相比单一核函数时更好。由此可以证明,基于组合核函数的流量预测模型在预测方面具有显著优势。

    4 结语

    本文建立基于组合核函数的中小流域流量预测SVM模型,对屯溪流域汛期小时流量数据进行应用分析。实验结果说明,与单一的多项式核函数或线性核函数相比,预测模型采用多项式核函数与线性核函数的组合核函数时得出的实验结果更加准确。
