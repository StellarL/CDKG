    摘    要：

    针对当前网络拓扑控制算法中存在的节点能耗较高、组网时间较长、节点存活周期较短等问题,提出基于节点负载均衡模型的低损耗无线传感器网络拓扑控制算法。利用适应度函数与通信效率函数调整节点自身的发射功率,增强节点的网络拓扑适应性,调控邻接节点数量,结合通信范围内邻节点分布情况,构建网络节点负载均衡模型,通过模型调节实现节点负载均衡。采用博弈理论收益函数及自有空间模型,构造邻节点选择策略集,根据较优反应策略选择下跳拓扑节点,控制节点损耗。实验结果表明,所提算法相比当前网络拓扑控制算法,网络拓扑控制过程耗时更短,网络节点的存活时间更长,且网络节点的能量损耗更低。

    低损耗; 网络拓扑; 发射功率; 负载均衡;

    Simulation of Topology Control Algorithm for Low Loss Wireless Sensor Networks

    SI Yong-jie ZHANG Jian

    Qiannan Normal College for Nationalities

    Abstract：

    Due to high energy consumption, long networking time and short survival time of nodes of current network topology control algorithms, this article presented an algorithm of network topology control for low-loss wireless sensor based on node load balancing model. The fitness function and the communication efficiency function were used to adjust the transmitting power of node itself and enhance the network topology adaptability of node. The number of adjacent nodes was adjusted. Based on the distribution of adjacent nodes in the communication range, the network node load balancing model was constructed, so that the node load balancing was realized through model adjustment. Moreover, the game theory revenue function and its own space model were used to construct the adjacent node selection strategy set. According to the optimal response strategy, the next-hop topology node was selected to control the node loss. Simulation results show that, compared with the current algorithm, the proposed algorithm is less time consuming. Meanwhile, the network node has longer survival time and lower energy loss.

    1 引言

    无线传感器网络综合了传感器、网络、通信及信息处理等多项技术,是当前互联网的重要组成之一,应用范围越来越广泛[1]。实际使用过程中,传感器网络节点的能量难以及时补增,减少能量损耗是无线传感器技术研究领域的重点问题。无线传感器网络部署与优化控制,受到了研究界及应用领域的高度重视,现已出现一些较好的网络拓扑控制算法[2,3]。

    文献[4]提出了基于Voronoi覆盖机制的网络拓扑控制算法。根据Voronoi无线网络覆盖机制,调控节点的运行状态,以达成目标区域内的网络节点全覆盖,结合点集三角剖分技术与节点功率控制策略,构建有效的网络拓扑结构。该算法拓扑得到的传感网络为刚性网络结构,且拓扑组网效率较高,但网络传输过程中节点能量损耗较高。文献[5]提出基于节点度估计的网络拓扑控制算法。构造传感器节点度估计模型,利用该模型计算得到的节点度因数判断网络拓扑结构的性能,在此基础上,进行网络节点布局调控与节点发射功率优化,创建完整的网络拓扑结构。该算法的网络拓扑结构各节点的能量损耗较低,但拓扑组网时间较长。文献[6]提出基于簇头选举更新的网路拓扑控制算法。结合网格划分策略,将传感器网络节点划分成若干个簇,根据节点性能与剩余能量选取簇头节点,在各簇簇头间构建动态链接,形成具有小世界属性的网络拓扑结构,通过减小全网节点与汇集节点的路径长度均值,以及周期性簇头更新,缩短网络通信延时,均衡各节点的能量损耗。该算法的网络拓扑结构能量损耗较低,但节点生命周期较短。

    针对上述方法中存在的不足之处,提出基于节点负载均衡模型的低损耗无线传感器网络拓扑控制算法,结合当前拓扑控制算法进行仿真,对比该算法形成的拓扑网络的节点存活时间、能量损耗均值及组网时间,验证了所提算法的鲁棒性。

    2 低损耗传感网络拓扑控制算法

    2.1 构建网络节点负载均衡模型

    无线传感器网络节点负载均衡模型含有传感器节点适应度函数以及通信效率函数;适应度函数根据其邻域节点能量调节自身发射功率,以提高自身适应度;通信效率函数主要通过节点自身发射功率的变化控制其邻域节点的数量,结合节点通信范围内邻居节点分布情况,提高节点通信效率,实现节点负载均衡[7]。

    当目标节点的邻域节点所含能量不多时,会增加该节点的负载,负载过大会加快节点死亡,降低了传感器节点适应度;若目标节点的邻域节点剩余能量较多,可缓解目标节点的负载及能耗,进而增强目标节点适应性。目标节点的适应性可通过其邻域节点的剩余能量情况来描述,其邻域节点剩余能量消耗越快,目标节点所承受的负载越大。

    传感器网络节点随机分布时,任一传感器节点处于另一个节点通信区域内的概率是相对独立的,遵从二项分布,则某一传感器节点t的节点度为kt时的概率可利用下式计算

    Ρ(εt=kt)=(AtA)kt(1-AtA)Ν-1-kt   (1)

    其中,εt表示一常数,N代表传感网络内的节点总数,At代表某一节点t可通信区域,A代表节点t分布范围所占面积。假设At≪A,则上式近似遵从普阿松分布[8],则处在节点分布区域内有kt个节点的概率可利用下式计算

    Ρ(Νt=kt)=λtktexp(-λt)kt   (2)

    其中,Nt表示随机变量,λt表示随机变量的平均值。

    上式是以无线传感器网络内全部节点信息为基础建立的,采用这些信息建立网络拓扑结构时,信息之间的转换次数较多,加大了计算量。针对此问题,采用局部节点信息建立网络拓扑结构时,传感器节点i在其可通信区域Ai内节点数量为ki的概率可利用下式表示

    Ρ(i,ki)=λtkiexp(-λt)ki   (3)

    传感网络节点在网络空间内随机分布,且其总数满足N>>1,当节点密度近似时,有ρ=N/A。则在任意节点i的通信范围Ai内,节点适应度为ki=ρAi。当任意节点i的发射半径为di时,其可通信的面积为Ai=πdi2。结合先验知识可知,传感器节点的发射功率pi与对应通信半径di之间为正比关系,则有pi=adi2,a为常数。通过上述分析过程可得

    ki=ρAi=ρ(πdi2)=ρ(πpia)=ΝAπpia   (4)

    设定b=ΝAπa,则b为常数且满足ki=bpi,可将式(3)转变成

    Ρ(i,ki)=(λt)bpiexp(-λt)bpi   (5)

    在实际拓扑结构中,节点适应度为整数,且当大部分传感器节点的适应度值为1时,存在节点失效情况,这种情况会导致网络通信异常,无法完成数据传输。针对此情况,设定的传感器节点适应度应不小于1,即bpi-1≥0。

    在无线传感器网络通信中,通信链路质量直接影响数据传输的完整性及效率。选取高质量通信链路进行网络数据传输,通信信道干扰较少,数据传输速率较快,重传的概率较低;相反,数据传输速率较慢,重传的概率较高,丢包率也较高。但是当传感器节点与地面接收基站通信时,传输路径之间通常表现出不对称性,致使网络数据传输出现错误,节点无法准确传输待传输信息,可通过双向接收的信号强度值描述无线传感器网络通信链路质量。

    对无线传感器网络中的随机一条通信链路(i,j),RSSIj(i)、RSSIi(j)用于描述传感器节点i与j接收信号的强度值,则节点j与i所在的双向通信链路质量可设定为

    bid(i,j)=RSSΙj(i)RSSΙi(j)   (6)

    通信链路双向的信号接收强度值RSSIj(i)、RSSIi(j)相等时,说明此链路能够有效完成双向、完整数据传输过程。

    传感器节点i可通信区域中邻居节点数量为ki时,各邻居节点与其通信的机会均等。由式(6)可计算节点i与任意一个邻域节点j进行通信的概率

    ΡRj=bid(i,j)∑k∈Aibid(i,k)   (7)

    其中,k和j均为i的邻域节点。

    无线传感器网络运行过程中,邻域节点耗能越少,说明节点自身承担的负载越少,负载变化越平稳。传感器节点适应度函数可根据邻居节点的剩余能量,描述目标节点i所负担的压力。由式(5)、(7)可设定节点i的适应度函数

    Sti=Ρ(i,ki)⋅∑j(ΡRj⋅EjE0j)=(λt)bpiexp(-λt)(bpi)⋅∑j(bid(i,j)bid(i,k))   (8)

    其中,Ej、E0j代表网络结构中任意一节点j的剩余能量和初始能量。∑j(ΡRj⋅EjE0j)代表邻域节点的剩余能量总和。Sti表示关于节点i发射功率适应度ki和其邻居节点剩余能量的函数。通过调整自身发射功率获取节点的最佳适应度函数值;当节点发射功率为恒定时,若其邻域节点剩余能量均较多,则该节点的适应度函数也越大,节点能够承担的负载更多。

    在无线传感器网络信息传输过程中,要实现数据传输,传感器节点需与多个节点通信,且节点与节点之间的距离会影响目标数据初始发送节点与最终接收节点之间的通信路径长度。假设待传输的目标数据有限,节点i将数据传输至其邻域节点的距离越短,越能够降低节点i本身的负载;相反,节点i则需增加本身功率,导致能量损耗。根据传感器节点的负载特性构造相应的效率函数,即

    Ιt=1Ν-1∑t1d(t,i)   (9)

    It为传感器节点效率函数,是指节点t与其通信的节点之间距离倒数之和的均值,此值越高说明该节点越容易传输数据至其它节点,耗能越少。

    传感器节点适应度函数与效率函数均可衡量节点所承担的负载。以这两项为依据,分析这两者之间的关系,并据此构建目标网络节点负载均衡评测模型。

    优化两项函数Sti与It属于多目标优化问题,求解此问题需要结合归一化思想,构造综合这两项优化目标的负载均衡评测函数,以下给出网络负载均衡评测函数

    w(i)=αStmax-StiStmax+βΙmax-ΙiΙmax   (10)

    其中,α、β分别代表权重调节系数。通过对网络负载均衡评测函数的拓展,可将上述问题转变成单目标优化问题,获得一个非线性约束优化的网络节点负载均衡评测模型

    minw(i)=α(1-1Stmax⋅(λt)bpiexp(-λt)bpi)+β(Ιmax-ΙiΙmax⋅(∑j(bid(i,j)bid(i,k)))   (11)

    当满足α=1,β=0时,上式给出的模型即为节点适应度函数;当满足α=0,β=1时,上式给出的模型即为节点效率函数。当α、β、pi已知时,w(i)与其通信范围内邻域节点的能量比总和为负相关;若已知邻节点的能量比总和,调节α、β的值使w(i)最小,最大化网络中节点适应度及效率,实现传感器节点负载均衡,延长无线传感器网络生命周期。

    2.2 网络拓扑控制算法

    利用上一节构建的无线传感器网络负载均衡模型实现网络节点的负载均衡,减少节点能量损耗,在此基础上,结合博弈模型的收益函数[9]与较优反应策略,在邻居节点中选择下一跳节点,逐渐形成完整的网络链路。具体过程如下:

    网络拓扑控制所形成的网络结构,可看作一个无向图,记作H,描述式如下

    Η=(Ν′,E,Ω)   (12)

    式中,N′为点集,代表所有网络节点的集合,N′={1,2,…,N};E为边集,代表网络节点中两个节点间通信链路的集合,Ω为网络链路功率矩阵,Ω=[Wij],其中,Wij为维持i-j段网络通信链路的最小功率。

    设所有网络节点在目标区域内随机分布,各节点的最大发射功率为pimax,定义发射功率向量p=(p1,p2,…,pn):pi∈[0,pimax],无向图H的子图G(p)代表网络节点的全部双向链路集合图,当全部节点通信功率均为最大发射功率时,构成连通网络拓扑图Gmax。

    无线传感器网络拓扑控制过程中,构建网络节点收益函数,利用该函数描述各节点在网络连接时,得到的收益与网络连接代价的权衡,给出各网络节点的收益函数

    ui(pi,p-i)=-apimaxE0iEi+fi(pi,p-i)(apimaxE0iEi+bE¯i(pi))   (13)

    式中,a、b为权重系数,且a>0,b>0;p-i代表除节点i外的其余节点的发射功率;fi(pi,p-i)代表网络各节点的连通性,网络连通时取值为1,网络不连通时取值为0;E0i和Ei分别代表节点i的初始能量和剩余能量;i(pi)代表邻居节点选择函数。i(pi)对应的公式为

    E¯i(pi)=1k′∑j=1k′EjE0j   (14)

    其中,k′为节点i的邻居节点数量,j代表发射功率为pi时,节点i的一跳邻居节点,E0j和Ej分别代表该节点的初始能量和剩余能量。

    节点i将本身发射功率初始化为pimax,以广播形式传送“邻节点寻求信息”,该信息包括:节点ID、发射功率pimax及剩余能量Ei;同时,采集邻节点反馈的确认字符信息。邻节点j接收到节点i的“邻节点搜索信息”后,利用自有空间模型[10]计算两节点间的最小联结功率pij,并反馈对应的确认信息,包括:节点ID、剩余能量Ej及联结功率pij。节点i接收到该确认信息后,将该邻节点的各项信息添加到自身的邻居表内,通过与各节点的信息交换,生成该节点的邻居节点选择策略集Pi

    Ρi={pimax=pi1,⋯,pij,⋯,pik′=pimin}   (15)

    其中,pij≥pij+1,且j∈[1,k′]。根据所得邻居节点选择策略集,结合较优反应策略,调整节点功率值,并从邻居节点中选择下一跳节点,循环上述过程,进行无线传感器网络拓扑控制,直至形成满足需求的无线传感器网络。

    3 实验结果与分析

    4 结论

    通过相关算法的学习研究,提出基于节点负载均衡模型的低损耗无线传感器网络拓扑控制算法。通过节点能量及发射功率与节点负载间的关联分析结果,构建网络节点负载均衡模型,以此模型为基础,结合博弈理论的收益函数与自有空间模型,构造邻居节点选择策略集,综合较优反应策略,完成下跳节点选择,最终得到网络拓扑结构。通过仿真,检验了所提方法的拓扑控制性能。
