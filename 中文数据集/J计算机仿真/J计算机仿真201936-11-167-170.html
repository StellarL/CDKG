    摘    要：

    针对传统大规模MIMO部署过多天线导致硬件成本和空间资源的浪费,选择基于虚拟信道的大规模MIMO系统,能够复用天线资源,在一根天线上传输多路数据流。针对传统的大规模MIMO信号检测方法,无法直接应用在虚拟大规模MIMO,提出了联合MMSE-ML检测算法,先检测出每根天线叠加的混合数据,再区分出单路数据,能够在保证良好检测性能下,降低检测算法的复杂度。仿真结果和复杂度分析表明,该方法在大规模MIMO中,相比传统大规模MIMO能节约天线数量保证检测性能,复杂度因天线数减少而降低。

    关键词：

    大规模多输入多输出; 虚拟信道; 复杂度;

    An Improved Signal Detection Algorithm Based on Virtual Channel Massive MIMO System

    MIAO Rong-zhen ZHOU Yuan-ping MA Yao-ting CHEN Min'e

    College ofElectronic Information, Sichuan University

    Abstract：

    The traditional massive Multiple Input Multiple Output(MIMO) deployment of too many antennas leads to the cost of hardware and the waste of space resources. A large-scale MIMO system based on virtual channels can be selected to reuse the antenna resources and transmit multiple data streams on one antenna. In view of the traditional large scale MIMO signal detection method, it can not be applied directly to the virtual large-scale MIMO. A joint Minimum Mean Square Error-Maximum Likelihood(MMSE-ML) detection algorithm is proposed, which first detects the mixed data of each antenna, and then divides the single path data. It can reduce the complexity of the detection algorithm under the good detection performance. The simulation results and complexity analysis show that the method can save the number of antenna detection performance compared with the traditional large scale MIMO in large scale MIMO, and the complexity is reduced because of the decrease of the number of antennas.

    Keyword：

    Large scale MIMO; Virtual channel; Complexity;

    1 引言

    人工智能,无人驾驶,虚拟现实等众多新技术的出现和5G时代的到来,对小区拥有更快数据传输速率和更高系统吞吐量的需求越来越大。在第四代移动通信中,多输入多输出(Multiple Input Multiple Output, MIMO)技术的应用大大改善了系统频谱效率和传输速率,但天线数上限设置为8根。随着未来移动数据流量的爆炸式增长,将大规模MIMO技术引入第五代移动通信技术,能够调用更多的发射天线和接收天线实现较高的频谱效率和能源效率,大规模MIMO技术被称为是5G的一项关键技术[1]。大规模MIMO与传统MIMO技术相比,它的收发天线少至数十根,多至成百上千根,虽然能充分挖掘空间自由度,但同时随着天线数的增加,信号检测的复杂度也同时增加,因此传统MIMO系统的信号检测算法(ML等)在大规模MIMO系统存在复杂度高,性能不理想等问题[2]。

    在大规模MIMO系统中,通过增加更多天线的数目实现更高数据传输速率,但也同时增加了系统的复杂度和硬件成本,而结合大规模MIMO复用天线资源的基于虚拟信道的大规模MIMO系统[3][4],能达到获得同样数据传输速率下,节省一定的天线数目,降低硬件成本,改善部署大量天线带来的空间资源的浪费。但采用基于虚拟信道的大规模MIMO系统不同于传统的大规模MIMO系统,每根发射天线将复用多路数据,并且发端的天线可多至几十上百根,所以一些传统的大规模MIMO线性检测方法MMSE,ZF等,仅能恢复出单路混合信号,而非线性理论上性能最优的ML检测算法,其复杂度很难达到系统的要求,而其它近ML的检测算法,如SD算法同样难以单独采用[5]。本文研究基于虚拟信道的大规模MIMO系统的检测方法,提出了一种结合系统模型的联合MMSE-ML检测算法,能在保证性能的同时,很好地平衡复杂度对系统带来的不利影响。

    2 系统模型

    考虑基于虚拟信道的大规模MIMO系统模型,发射端部署M根天线,接收端部署L根天线。本文的系统模型如图1所示。

    图1 虚拟信道大规模MIMO系统模型   下载原图

    假设接收端的天线数目少于发射端天线数目。发射端的M根天线,每根将混合传输Q路数据信号,每路数据用smq表示,m为发射天线标记,q为发射路序号标记。每单根的数据信号smq在虚拟信道因子wmq的作用下,将发送信号smq加权混合发送出去。虚拟信道因子wmq=Amqejθmq ,Amq表示信道因子的幅度值,θmq表示信道因子的相角。则第m根发射天线的输入数据可表示为:sm=[sm1,sm2,...,smQ]T,其对应的虚拟因子ωm=[ωm1,ωm2,...,ωmQ]T,经过虚拟因子作用后的第m根天线的发射数据为xm=ωmΗsm=∑q=1Qωmqsmq。

    M根发射天线的信息经过实际物理信道传输,在接收端被L根接收天线所接收。接收信号用y表示,y=[y1,y2,...,yL]T∈CL×1 ,第l根天线接收发射端的所有M根天线的数据信息,l=1,2,...,L。则接收信号可以表示为

    y=Ηx+n=∑m=1ΜhmwmΗsm+n (1)

    其中,H表示L×M空间物理信道矩阵, H=[h1,h2,...,hM],hm=[h1m,h2m,..,hLm]T表示对应于第m根发射天线的物理信道向量,n=[n1,n2,...,nL]T 是具有统计独立分量的加性高斯白噪声向量,且每个元素服从CN(0,σ2)分布。x=[x1,x2,...,xM]T是系统的发射向量。

    3 传统检测算法

    3.1 最小均方误差(MMSE)检测算法

    最小均方误差误差检测是一种线性检测误差的算法,线性检测算法通过接收信号矢量通过变换矩阵的作用,恢复出原始的发送信号。而最小均方检测算法作为一种简单的检测方式,其检测准则在于使得信号矢量与理想发射信号之间的均方误差在统计意义上达到最小[6]。其在MIMO无线通信系统中的操作方法是,将接收到的信号向量通过加权矩阵操作,恢复出原始信号。

    具体的操作步骤为:

    计算加权矩阵

    WMMSE=(HHH+σ2I)-1HH (2)

    加权矩阵与接收信号相乘得检测结果

    xMMSE=WMMSEy=x+(HHH+σ2I)-1HHz (3)

    3.2 最大似然(ML)检测算法

    最大似然检测算法是一种非线性的检测算法,通过计算接收信号与所有可能的发射符号进行距离求解,选择距离最小的发射符号作为检测结果[7][8]。最大似然检测算法的检测结果由下式获得

    x∼ΜL=argminx∈CΝtx∥y-Ηx∥2 (4)

    这里C表征包含调制信号的所有可能的星座集,Ntx表示发射天线数量,CNtx表示包含了发射信号所有可能组合的向量空间。‖y-Hx‖2是实际接收信号与潜在的星座点之间的欧式距离,即最大似然度量。虽然最大似然检测方法能获得最有的误码率性能,但是由于需要穷举式搜索,其复杂度随着发射天线的数量和调制阶数的增加呈指数倍增长。

    在最小误差概率的意义之下,最大似然检测算法是理论上最优的检测方法。这种方法以实际接收信号与理想接收信号的距离作为比较的依据,选择实现距离最小的符号作为检测的发射信号。这种算法的复杂度随着发射天线数与调制阶数的增加指数型增加,在高阶通信系统中复杂度极大。

    4 基于虚拟信道处理检测算法

    传统的大规模MIMO系统,每根发射天线仅发送单路数据,在接收端通过线性检测算法(如MMSE,ZF等)或非线性检测算法(ML)将发送信号检测出来。总体上讲,发射和接收天线部署的数量越多,就会有更多可能的信号路径,传输得到的数据速率会更高,可靠性会更好,但复杂度会显著增加同时能耗也会上升[9]。对于采用虚拟信道的大规模MIMO系统,能够减少一定数量的天线,复用天线资源,但传统的大规模MIMO系统的检测方法应用在虚拟信道大规模MIMO系统上有很大弊端。用MMSE或ZF算法,算法检测虽然操作简便,但检测的是单根天线上多路数据的叠加,无法分离出具体的单路信号;对于ML算法检测,虽然能够实现单路数据的检测,但算法复杂度过大,带来检测过程消耗大量的时间成本,均没有达到一个理想的检测效果。

    鉴于此,提出一种MMSE与ML结合的基于虚拟信道的大规模MIMO系统检测算法,将虚拟信道大规模MIMO利用MMSE检测出的混合信号再逐个经过ML检测,实现单路信号的区分。

    4.1 联合MMSE-ML检测算法

    检测算法具体步骤:

    1)接收端的信号为y=[y1,y2,...,yL]T,假设在检测之前已经完全知晓信道状态信息(Channel State Information, CSI),无线信道矩阵记为H ,噪声功率记为:σ2。首先通过MMSE检测算法检测出每根发射天线所发射的混合信号,MMSE的算法处理矩阵为:GMMSE=(HHH+σ2IM)-1HH 那么,经过MMSE处理矩阵的接收信号为:x^=GΜΜSEy

    2)经MMSE算法获得的每根天线的多路混合信号为x^=[x^1,x^2,...,x^Μ]Τ,假设进行下一次检测之前,每路信号的虚拟因子ωmq准确已知,将得到的信号x^经过ML算法检测,计算第m个输入信号向量sm的估计值如下

    s^m=argminsm∈JQ∥x^m-wmΗsm∥2

    其中,s^m=[s^m1,s^m2,...,s^mQ]Τ,J是QAM信号的星座集,JQ 为所有发射信号的解空间。

    结合MMSE检测和混合信号的ML检测,能够有效地区分出每一路信号,有效地克服了单独使用MMSE检测无法直接检测出单路信号和单独使用ML检测算法复杂度太高的弊端,实现了大规模MIMO系统下天线资源的复用,并能够保证系统性能优良。

    4.2 性能分析

    基于虚拟信号的大规模MIMO系统检测算法结合了MMSE和ML两种检测方法,有效地解决了单根天线混合多路数据下区分单路信号的问题,首先将单根天线的混合信号看做合成信号,合成信号发送到恢复用MMSE算法实现,而之后的合成信号区分出单路信号通过ML算法实现,这一方面发挥了MMSE算法在大规模MIMO系统中检测复杂度低,操作便捷的优势,另一方面采用ML算法区分单路信号保证了检测的高精度。

    5 算法仿真与分析

    5.1 性能准确度分析

    为了验证基于虚拟信道的大规模MIMO检测算法的有效性,在保证发射相同路数据的前提下,与传统的大规模MIMO线性检测算法MMSE算法进行了对比。

    通过仿真对发送数据总路数相同条件下的传统大规模MIMO检测算法MMSE算法,与本文提出的基于虚拟信道的大规模MIMO检测算法在发射总路数为20路和30路的情况下进行了对比,图2为发射总路数为20路的传统检测算法MMSE与基于虚拟信道联合MMSE-ML检测的仿真结果。系统采用的调制方式是QAM调制,信道为静态信道,噪声为独立同分布加性高斯白噪声。在基于虚拟信道的大规模MIMO系统,当发射20路数据时,假定每根天线发2路数据,则发射端只需要10根发射天线。在检测之前,虚拟信道的虚拟因子与实际信道的CSI假设已经准确知道。仿真结果描绘了信噪比在-20dB～10dB之间的误码率,随着信噪比的增大,联合MMSE-ML方法的性能优越性体现地越来越明显,在信噪比达到7dB时,联合MMSE-ML能够达到10-4以下的误码率,而传统的大规模线性检测算法误码率随着信噪比的增加而缓慢降低,在高信噪比时没有达到联合MMSE-ML所具有的低误码率性能。

    5.2 复杂度分析

    联合MMSE-ML的基于虚拟信道的大规模MIMO检测算法,复杂度由两部分组成,由MMSE检测带来的复杂度和由ML检测带来的复杂度。其中由MMSE算法的复杂度为O(LM2+M3) ,其中复杂度O(LM2)来自矩阵的乘积,而复杂度O(M3)是由矩阵的求逆计算所产生的[11]。由结合算法的第二个步骤,ML算法的复杂度为O(MJQ)。因此结合两种算法的虚拟信道大规模MIMO检测算法,总的计算复杂度为O(LM2+M3+MJQ),能够看出联合MMSE-ML检测方法的复杂度很大程度上取决于信号路数。相比传统大规模MIMO采用线性检测算法如MMSE,在发送同样多总路数的数据并采用低阶调制的前提下,基于虚拟信道的大规模MIMO检测算法将采用更少的发射天线数量,使得采用联合MMSE-ML的算法复杂度会低于传统大规模MIMO采用线性检测算法复杂度。这种检测算法将能很好地应用在工程实际中,带来高效的实际利用价值。

    6 结论

    本文主要研究了传统大规模MIMO和基于虚拟信道的大规模MIMO的信号检测,鉴于传统大规模MIMO部署过多天线导致硬件成本增加,浪费空间资源,引入基于虚拟信道的大规模MIMO系统。但传统的线性检测方法只能检测出单路混合信号,非线性检测方法复杂度太高,引入联合MMSE-ML算法,先用线性算法检测出混合信号,再用非线性算法检测出单路信号。在传输同样多数量的数据时,既保证了检测性能的优越性,同时相比传统大规模MIMO大幅减少了发射天线的数目,降低了算法复杂度,节约了硬件成本,节约了空间资源,能满足未来通信系统的需要。
