    多社交网络权威用户身份发现算法仿真

    庄海燕

    摘    要：

    网络权威用户在产品销售、舆情议论和信息传播等方面有着重要影响,检测社交网络权威用户意义重大。当前算法存在查准率和查全率低的问题,提出基于影响力的多社交网络权威用户身份发现算法。分析社交网络演化特征,并以网络用户连通性变化和社团结构化为主进行了详细探讨。利用TFIDF可以衡量文章中词对文档重要程度特性,衡量一个用户对于其他用户的重要程度,实现候选权威用户筛选。根据社交网络特性,分别构建用户话题影响力模型和用户行为影响力模型,并结合候选权威用户筛选结果,构建多社交网络用户影响力模型。通过用户影响力模型得到多社交网络所有用户中权威性强的用户,即影响力大的用户权威性强。仿真结果表明,所提算法查准性能和查全性能均优于当前算法,可很好地应用在权威用户识别中。

    关键词：

    多社交网络; 权威用户; 身份发现;

    作者简介： 庄海燕(1976-),女(汉族),河南南阳人,硕士,副教授,研究方向:信息安全,电子数据取证与分析。;

    收稿日期：2018-12-26

    Multi-Social Network Authoritative User Identity Discovery Algorithm Simulation

    ZHUANG Hai-yan

    Image and Network Investigation Department, Railway Police College

    Abstract：

    This article focuses on an algorithm of authoritative user identity discovery based on influence. Firstly, the evolution characteristics of social network were analyzed. The change of network user connectivity and the structuralization of community unity were discussed in detail. TFIDF was used to measure the importance of words to the document and the importance of a user to other users, so as to achieve screening of candidate authoritative users. According to the characteristic of social network, user topic influence model and user behavior influence model were constructed respectively. Meanwhile, the candidate authoritative user screening results were combined to construct the influence model of multi-social network user. Based on the user influence model, the authoritative user in all the users of multi-social network was obtained. That is to say, the user with great influence has strong authority. Simulation results show that the proposed algorithm has better precision rate and recall rate than that of current algorithm, which can be applied to the authoritative user identification.

    Keyword：

    Multiple social network; Authoritative user; Identity discovery;

    1 引言

    当前社交网络中对于用户身份影响力的定义主要以用户粉丝数量多少为依据,粉丝数量越多的用户知名度也会越高[1,2]。要将源自大众媒体的信息传播给更多用户,一般会经由权威用户的中转[3]。事实上,粉丝数量只是衡量社交网络具有影响力的其中一个指标,要高精度识别网络权威用户还需要结合不同方面。而要树立网络的正面形象和提升信息传播效率与范围,就必须对社交网络权威用户进行识别。

    2 相关研究简析

    文献[4]中陈毅恒等人认为社交网络中用户影响力分析是当前非常重要的研究课题。文章中将研究重点放在了利用网络结构对用户影响力进行分析方面,其中包含最大度算法和距离中心点算法等,并在性能上对这些算法进行对比,选出最适合挑选出社交网络中权威用户,即具有影响力的用户的算法。文献[5]中周飞等人认为权威用户是社交网络中正面信息的传播者与信息发展方向的引导者,对此类用户的影响力和权威性进行识别是一项非常重要的内容。过程中,将用户权威性与PageRank意见领袖识别算法相结合,并综合考虑用户本身权威性和用户动态行为的影响力给社交网络中的动态信息带来的实际影响,实现社交网络权威用户身份识别。利用在知乎社交网络采集的海量数据当作实验数据对所提算法进行了验证。文献[6]中郑志蕴等人对于社交网络中心性节点,即影响力较大节点的发现,提出将中心启发式理念应用在社交网络影响力较大用户发现中。过程中,根据线性阈值模型得到网络节点潜在的影响力,采用贪心算法筛选出影响力最大的扩展节点。

    社交网络权威用户发现和检测是引领网络走向积极正面的必要环节,对网络健康发展意义重大。上述文献所提算法在查准率和查全率方面均有待提升,急需提出一种可靠算法。为此,本文提出基于影响力的多社交网络权威用户身份发现算法。

    3 多社交网络权威用户身份发现

    3.1 社交网络演化分析

    选择某社交网络,并采集了该网络2017年3月至2017年9月真实数据,其中包含用户创立时间和发表的信息、评论等数据。将用户账号创立时间定义为网络节点创立时间。将节点创立时间当作网络演化时间的参考项,以此考察该社交网络用户网络演化情况。将用户账号创立时间按照每7天当作一个时间步,从而获取网络演化特征,以此分析社交网络发展特性,为发现权威用户提供辅助。

    1)网络用户连通性变化:

    社交网络演化的初级阶段,用户连接模式与一般的网络模型连通并不相同,而是存在一些用户是孤立的情况,也会出现少数用户聚集而导致整个社交网络由多个相互不连通的子图构成[7,8]。上述现象主要原因为该社交网络用户在创立账号的初期,只有比较少的用户,可能不会关注任何人,也可能仅和现实生活中认识的人组成小团体。还存在用户可能会优先关注自己感兴趣的用户,以此加入某社团,进而使整个社交网络变为了连通网络。

    图1的横坐标为时间,纵坐标为每个时间步连通节点占据的百分比,以此构成了连通节点在社交网络演化中占据的比例情况示意图。

    图1 连通节点在社交网络演化中占据的比例   下载原图

    分析图1可知,随着时间不断推移,用户逐渐加入,社交网络的规模也在日益扩大。连通节点占据的比例也越来越大,所谓的孤立节点也会加入至整个网络中,此种情况下,当网络发展到一定规模之后,该社交网络会成为一个连通网络。

    2)社团结构:

    实际社交网络中一般是通过兴趣或现实朋友圈构成社团结构的,根据当前社团划分方式能够得到每个时间步社交网络中的社团数量,以此得到的网络社团数量变化情况如图2所示。

    图2中,在时间不断推移下,该社交网络社团数量呈上升趋势,仅在比较小的范围存在波动情况。该状况出现的主要原因为网络中社团之间新连接的关系会对社团的个数产生比较小的影响。

    3.2 候选权威用户身份筛选

    在上述简介与分析中,了解了社交网络演化发展呈现出的特征和主要影响因素。接下来利用TFIDF筛选出候选权威用户,以此提升权威用户身份发现查准率。TFIDF主要用在权重计算方面,衡量一个词针对整篇文章的重要程度,一般用在词袋模型中[9,10]。TFIDF权重包含两部分:词频TF,也就是一个词在文章中出现频率,出现频次越多则表示该词越重要;词逆文档频率IDF,一般认为一个词在文章中出现频率越少,则该词具有的区分性能就越强。

    TFIDF主要是用来对文章中词对文档重要性进行衡量,在此,利用其来衡量一个用户对于其他用户的重要程度。

    综上,一个用户对于其他用户的重要程度表达式为

    tagtfidfp,t=tagtfp,t·tagidft (1)

    式(1)中,tagtfidfp,t代表词频率,即一个用户对于其他用户的重要程度,tagtfp,t代表用户在社交网络所有用户中出现的次数,tagidft代表区分于其他用户能力。

    3.3 基于影响力的权威用户身份识别

    基于上述社交网络演化和候选权威用户身份筛选,对用户影响力建模,利用话题影响力模型、行为影响力模型等实现权威用户身份发现,其中话题和行为间存在的关系主要利用显式变量用户维系。

    1)用户话题影响力模型构建

    针对社交网络中的用户来说,所有话题均来自于消息。假设利用#(hahstag)标签文字、纯文字消息表征短文字话题。信息的处理流程为:解析每个用户发布出来的推文,检测并提取出相应的内容特征,构成将用户当作单位的两个特征集合:#(hahstag)标签文字、纯文字消息集合。上述中的两个集合均利用词向量予以表征,在表征之前利用word2vec模型训练全部用户的#(hahstag)标签文字、纯文字消息,且向量维度定义为50维。综合上述内容,对各个推文词向量实行加权操作,构成各推文向量,并以此为依据,计算话题影响力。

    上述表征的话题模型实际为隐式的话题模型,文章研究的目标是获取用户影响力树。其中,话题影响子树与行为影响子树能够构成用户影响力树。

    话题影响子树是用户在某个话题下对社交网络中所有用户产生影响的树。假设P(Tc|u)代表用户u话题影响树,Tc代表话题影响树。P(Tc|u)表达式为

    P(Tc|u)=P(Tc|z)·P(z|u) (2)

    其中,z代表一个话题,P(z|u)代表用户u对于z感兴趣的几率,P(Tc|z)代表对z感兴趣的用户占据全部用户的比例。

    话题影响树构建时,设定话题间是一种互为独立的关系。假设用户u在#(hahstag)标签构成的集合概率为P(i|uh),用户u在纯文字集合中概率为P(j|ut),P(Thc|i)和P(Ttc|j)代表概率子树。根据P(i|uh)、P(j|ut)、P(Thc|i)和P(Ttc|j)能够确定概率P(Ta|u)与概率子树P(Tc|u)。其中,P(T|u)概率描述的是用户u会以多大概率构成影响力树P(T|u),以此对生成的影响力树P(T|u)中全部用户产生一定程度上的影响。上述中的i代表#(hahstag)标签下话题,j代表纯文字信息下话题。

    以上述计算与分析为基础,用户u话题影响力树被约简化成式(3)

    P(Tu|u)=P(Ta|u)·P(Tc|u) (3)

    2)用户行为影响力模型构建

    目前,针对多社交网络中用户行为的分析主要侧重点为用户之间转发和被关注等行为,特别是转发行为被网络中其他用户认可是表征用户之间实际影响的重要指标。以此,从用户推文和引发行为间相互关系角度出发,结合用户行为影响力树模型,为网络权威用户发现提供支撑。用户在社交网络影响力树模型构建中,需要考虑以下三种行为:转发、涉及与回复。综合上述内容,可将行为影响力树模型Au定义为

    P(Au|u)=∑P(Au|z)P(z|u) (4)

    其中,P(Au|u)代表用户发表的全部推文导致的其他用户行为,其中包含转发、涉及与回复构成的影响力树Au概率。

    3)多社交网络用户影响力模型构建

    结合上述用户话题影响力模型和用户行为影响力模型及候选权威用户身份筛选结果,则户影响力模型可定义为

    Ρ⌢u=Ρ(Τ|u) (5)

    P(T|u)=P(Au|u)·P(Tu|u)·tagtfidfp,t (6)

    通过用户影响力模型构建可得到多社交网络所有用户中权威性强的用户,以此实现权威用户身份发现。依据用户话题影响力模型和用户行为影响力模型两方面发现网络权威用户,可有效增强算法查全性能。

    4 实验结果与分析

    该节中,在规模比较大用户名为真实的数据集上开展权威用户身份发现实验。实验环境如下:5台服务器构建的hadoop分布式运行环境。每台服务器配置:3.10 GHz Intel Xeon E3-1220 CPU,32GB的内存,操作系统为CentOS 6.0。

    4.1 实验数据来源

    在多社交网络中收集用户名称过程中,以省时省力省成本为出发点,利用得到用户在社交网络中用户名称及Email实现数据集合构建。数据来源于某社交网络2018年12月用户身份信息数据库。与文献中方法对比验证所提算法,利用采集到的数据构建3个测试数据集合,各数据集合包含2000万用户名。

    4.2 权威用户发现查准性验证

    在上述实验环境下,以查准率为实验指标验证基于影响力的多社交网络权威用户身份发现算法与文献算法性能优良。

    查准率是验证算法发现社交网络权威用户精准性的有效指标。从图3中可以看出,在不同实验次数和用户数量下,文献几种算法查准性能均不是很理想。其中文献[5]算法在运行初期具有较高的查准率,但持续性不是很好,最终查准率曲线呈下降趋势。与当前算法相比,基于影响力的多社交网络权威用户身份发现算法查准率较高,最高约为97%。该算法利用TFIDF对候选权威用户进行初步筛选,TFIDF中的词逆文档频率IDF,具有区分性能,以此可为提升权威用户身份发现查准率奠定有效基础。

    4.3 权威用户发现查全性验证

    不同算法的查全性实验结果如图4所示。

    查全率是检验算法可靠性的指标,该值越高表示算法鲁棒性越强。图4中文献算法查全率曲线波动较大,虽然存在较高情况,但整体实用性较差。文章提出的权威用户身份发现算法查全率高,最高约为99%。算法运行过程中,为了提高查全性能,以用户话题影响力模型和用户行为影响力模型两方面作为构建用户影响力模型的依据,以此发现网络权威用户。在整体可观性方面而言,本文算法更具可行性。

    5 结束语

    鉴于权威用户在网络中的实质性作用及当前权威用户检测算法性能待改善等一系列问题,提出基于影响力的多社交网络权威用户身份发现算法。通过简单的社交网络演化分析为后续构建用户影响力模型奠定基础,采用TFIDF筛选出候选权威用户,结合话题影响力和行为影响力得到最终的用户影响力模型,完成权威用户身份识别。经实验可知,文中算法相较当前算法在性能上更为完善。在现实社交网络中,相似用户名称数量巨大,如果不将此类用户名称清理掉,会严重影响权威用户身份发现效率。针对该问题,下一步应将重点放在清洗网络用户名方面,更好地拓展算法性能。
