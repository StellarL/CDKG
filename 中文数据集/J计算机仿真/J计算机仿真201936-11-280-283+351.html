    摘    要：

    传统网络恶意行为特征提取方法,无法对关联性弱的行为进行高精度检测,存在误报率高的问题,无法有效感知网络威胁。提出一种基于关联规则的网络态势感知系统中恶意行为特征关联分析方法。利用粒子群算法优化D-S证据理论,对不同信度流量数据授予不同权值,完成网络安全数据多源融合,实现网络安全态势要素提取;对提取出的网络安全态势要素进行态势理解,将不同来源的网络警报数据去伪存真,提取网络恶意攻击行为特征,采用FP-Stream算法对恶意行为特征关联规则进行挖掘,完成网络态势感知系统中恶意行为的特征关联。实验结果表明,所提方法能够有效提高特征关联检测精度、降低误报率,上述方法对网络安全态势的感知效果好,具有应用前景。

    关键词：

    网络态势感知; 恶意行为; 行为特征; 特征关联分析; 关联规则;

    Correlation Simulation of Malicious Behavior Characteristics in Network Situational Awareness System

    ZHANG Ju-guang

    Information Center, China Association for Science and Technology(CAST)

    Abstract：

    For network malicious behavior feature extraction, traditional methods can not precisely detect behavior with the weak relevance, resulting in high false alarm rate. Therefore, this paper presented a method to analyze feature association of malicious behavior in network situation awareness system based on association rule. At first, we used particle swarm optimization to optimize D-S evidence theory. Secondly, different weights were given to flow data with different reliability, so as to complete multi-source fusion of network security data and realize the extraction of network security situation element. Moreover, we conducted the situation comprehension on extracted network security situation elements. For the false alarm data from different sources, we discarded the false and retained the true and then extracted feature of malicious network attack was extracted. Finally, we used FP-Stream algorithm to mine the association rule of malicious behavior characteristic, and thus to complete the association of malicious behavior characteristic in network situation awareness system. Experimental results show that the proposed method can effectively improve the accuracy of feature association detection and reduce the false positive rate. Meanwhile, the method has good effect on network security situation awareness which is in prospect of applying.

    Keyword：

    Network situation awareness; Malicious behavior; Behavior characteristic; Feature association analysis; Association rule;

    1 引言

    计算机网络技术的飞速发展促使网络中的数据量也飞速增长,人类社会已进入网络信息化时代[1]。网络大数据给人们日常生活和工作带来极大便利的同时,也给网络安全带来了挑战[2]。网络数据来源广泛,具有多样性且价值密度低等特点。很多违法者根据这些数据作为网络攻击的敲门砖发起恶意攻击,通过这种方式可快速控制大量的对象且攻击持续时间长、具有隐蔽性[3]。传统的网络攻击检测技术面对复杂的网络攻击形式已经变得束手无策,目前已成为网络安全管理领域中不可忽视的问题,引起了相关专家学者们的高度关注。

    网络入侵检测技术是目前进行网络恶意行为防御的有效手段之一[4]。网络入侵检测技术是通过对网络数据流进行特征提取,对提取的特征进行分类,来识别出恶意攻击行为。国内学者针对网络入侵检测方面进行了大量研究,文献[5]提出一种基于SMOTE和GBDT的网络入侵检测方法。对网络数据进行预处理过程中,采用SMOTE技术对少数类别的网络数据进行属性约简,在约简后的数据集中训练GBDT分类器,根据分类结果完成网络入侵检测。该方法约简后的网络恶意行为特征数据集之间关联性较弱,存在检测命中率低、难以有效感知网络安全威胁的问题。文献[6]提出一种基于隐马尔可夫模型的入侵检测方法。采用K均值聚类算法划分目标网络状态,根据设定的目标网络状态构建隐马尔可夫模型,通过该模型对网络数据流进行入侵检测。该模型感知范围有限,导致实际网络安全态势感知结果与真实结果存在较大的偏差。

    将关联规则应用到网络态势感知系统中,对恶意行为进行分析,挖掘恶意攻击行为特征之间的关联,快速识别出恶意攻击发生的时间、地点以及强度等多种信息,能够有效提高网络入侵检测精度和效率,同时能够帮助网络安全管理人员快速掌握当前网络整体安全态势情况,并及时给出相关防御措施。近年来国内学者针对网络态势感知方面也进行了大量研究,文献[8]根据布谷鸟算法特性,对神经网络模型进行优化,将优化后的模型用于网络安全态势评估过程中。该方法证实了优化后的神经网络模型层次感知的可行性,但无法有效感知网络安全威胁。文献[9]通过对网络中的恶意数据流进行攻击类型聚类以识别出攻击者;对恶意攻击类型因果关联,识别出相应的恶意攻击轨迹以及攻击阶段,构建网络安全态势量化评估标准,计算各个攻击阶段的威胁指数实现网络安全态势评估。该方法计算出的威胁指数受到主观依赖影响较大,难以有效感知网络安全威胁。

    针对当前网络安全检测方法存在的不足,为了有效保障网络信息安全,提高网络防御性能,提出将关联分析技术和网络安全态势感知应用到网络恶意攻击行为分析中。所提方法能够有效提高检测命中率、降低误报率,有效感知网络安全态势威胁情况,达到了有效感知网络安全态势的目的。

    2 网络安全态势感知原理

    通过粗集理论从网络恶意攻击样本数据集中构造恶意攻击态势决策表,计算恶意攻击态势决策表中属性之间的依赖度,提取出恶意攻击规则。将提取的规则根据相关转换机制转换成机器可识别的语言,在线对网络数据流进行恶意行为关联分析,将分析结果提供给实时态势分析,根据恶意攻击感知的结果对整体网络安全安全态势进行分析。具体过程如下所述。

    粗集理论将网络世界抽象为一个信息结构。所描述的信息结构用S表示,S=〈U,A,V,f〉,U表示论域,V表示信息系统中属性值的值域,A表示属性值集合,f表示信息函数。论域U实际上是简单攻击彼此融合后新生成的复杂攻击形式,将每个简单攻击视为条件属性值,将新的攻击形式对网络安全造成的威胁程度视为决策属性集D。构造如表1所示的恶意攻击态势决策表。表1中,C表示条件属性。

    表1 恶意攻击态势决策表 导出到EXCEL


    U
    C    		D

    C1    		…Cn

    1    		C1,1-Cn,1D1

    2    		C1,2-Cn,2D2

    …    		…-……

    n    		C1,n-Cn,nDn

    对表1中的条件属性集进行属性约简,提出对决策结果产生不利影响的条件属性集。从约简后的恶意攻击决策表中获得各个属性之间的依赖关系,提取出恶意攻击规则C→D,此规则的可靠程度cf可利用下式计算

    cf=|U/eq(C)⊂U/eq(D)||U/eq(C)| (1)

    恶意攻击决策条件属性C可根据cf的可信度来确定D,攻击规则的可信度cf越高,说明C对D的影响越大,eq(C)表示C的无效攻击因子,eq(D)表示D的无效攻击因子。

    将提取的规则转换成事件流处理引擎可以识别的语言[10],实现对通过该引擎的网络数据流进行恶意行为关联分析

    Η(z)=cf*Ν(z)eq(D) (2)

    其中,N(z)表示当前攻击威胁值。

    将分析结果提供给实时态势分析,从网络服务器、主机以及系统三个维度分析网络安全态势。网络恶意攻击对服务器的威胁程度可利用下式表示为

    TS=∑num×att (3)

    其中,num表示发动攻击的次数,att表示网络恶意攻击值。网络恶意攻击对主机的威胁程度利用下式表示为

    TH=∑SP×TS (4)

    其中,SP表示网络攻击等级。网络恶意攻击对系统的威胁程度利用下式表示为

    TN=∑HP×TH (5)

    其中,HP表示主机所占重要度的权重。

    以上为网络态势感知系统中恶意行为特征关联原理,根据该原理提出一种基于关联规则的网络态势感知系统中恶意行为特征关联分析方法。

    3 网络态势感知系统中恶意行为特征关联分析

    3.1 网络安全态势要素提取

    利用粒子群算法优化D-S证据理论对系统中不同信度流量数据授予不同的权值,实现网络安全数据多源融合。在融合的基础上,对网络威胁因子进行监控,实现网络安全态势要素提取,具体过程如下所述:

    将粒子群算法与D-S理论相融合处理网络安全数据融合问题,根据种群中个体之间的竞争与合作关系搜索出不同数据源的优化指数权值。在种群迭代搜索过程中,对含有t个个体的种群,其中任意一个个体i根据下式更新其搜索速度

    vi=c1r1(pi-xi)+c2r2(pg-xi) (6)

    其中,c1和c2分别表示常数,r1、r2表示区间[0,1]内的随机数,pi表示种群中个体i的局部速度更新值,pg表示种群全局速度更新值,xi表示种群中个体i当前所处位置。

    为了能够快速获取全局最优值,融合于D-S证据理论,应当满足下式所示的适应度函数

    Fi=max{m(Ai)-max{m(Aj)}} (7)

    其中,m表示D-S证据理论中的命题数量,决策目标Ai相应的基本概率分配为m(Ai),决策目标Aj相应的基本概率分配为m(Aj),Fi表示Ai相应的适应度函数。

    在式(7)给出的适应度函数约束下,对系统中不同信度的数据设定不同的融合权值w1,…,wn,以下给出D-S融合规则,即

    m(A)=∑nm1(A1)w1,⋯,mn(An)wn1-∑nm1(A1)w1,⋯,mn(An)wn (8)

    在融合的基础上,将正态分布进行离散化处理,设定共有n类不同的网络安全事件,并将划分为m个等级的威胁因子。设定Xn=∑i=1nxi,Y=Xn/n,对正态分布形式进行描述

    f(x)=1σ2πe (9)

    其中,σ表示随机变量的均值,e表示随机变量的标准差,x表示满足均值为0的正态分布。

    在时间窗口T内,网络系统NS中共有v台主机,λHl表示主机Hl的权重,对网络威胁因子进行监控,可实现网络安全态势要素提取

    VΝS=∑i=1v(λΗl⋅VΗl) (10)

    其中,网络服务器重要性依据服务器上运行服务数量、资产值和是否存在价值数据VHl等决定。

    3.2 基于态势要素理解的恶意行为特征关联规则挖掘

    对提取出的网络安全态势要素进行态势理解,将不同来源的网络安全态势感知警报数据去伪存真,挖掘出真实的网络恶意信息。将不同来源的警报数据去伪存真,挖掘出真实的网络恶意信息。通过真实的网络恶意信息进行预处理,从处理好的网络恶意信息中提取出网络恶意攻击行为特征,采用FP-Stream算法实现恶意行为关联规则挖掘。具体过程如下所述:

    对提取出的网络安全态势要素进行态势理解,设t时刻网络安全态势要素的总数Qt为

    Qt=∑n=1kΟn (11)

    其中,k表示t时刻网络中恶意攻击类型,On表示某个安全态势要素的权限特征。

    网络安全态势警报数据主要来源于入侵检测系统提供的数据以及采集的日志报警信息,这些警报数据中含有很多冗余、无效信息,针对此问题,采用入侵检测方法对这些不同来源的警报数据去伪存真,挖掘出真实的网络恶意信息

    Ρi=∑i=1Ν(Dj/Qt) (12)

    其中,Dj表示入侵检测系统提供的恶意攻击数据包,N表示Dj中攻击类型字段。

    根据实际网络需要,将网络恶意信息进行预处理,包括数据净化和冗余警报去除处理

    xi´=xi-xminxmax-xmin (13)

    其中,xi表示网络恶意信息数据值,xmax和xmin分别表示数值上下限。从处理好的网络恶意信息中提取出网络恶意攻击行为特征

    Xi=Uk∏i=1nΡ(X1=t1|X2=t2)∏i=1nΡ(C=cj)∏i=1nΡ(X1=t1|X2=t2) (14)

    其中,P(X1)和P(X2)分别表示网络恶意攻击行为特征X1、X2的先验概率,t1和t2分别表示网络恶意攻击行为特征向量空间,P(C=cj)表示特征的后验概率。利用关联规则的置信度Uk作为网络恶意攻击行为特征间的条件概率,采用FP-Stream算法实现恶意行为关联规则挖掘

    σ=1n-1∑i=1n(Uk⋅Xi) (15)

    5 结论

    针对当前网络安全检测方法存在的多方面不足,提出基于关联分析的网络安全态势感知系统中恶意行为特征关联分析方法。从网络安全态势要素提取、网络恶意攻击行为特征关联分析、安全态势感知等方面实现对网络恶意攻击行为的分析与检测。实验结果表明,所提方法能够有效感知网络恶意攻击行为,为制定网络安全策略以及防御措施等提供了有效手段。
