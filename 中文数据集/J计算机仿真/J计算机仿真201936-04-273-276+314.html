    摘    要：

    对多层差异网络分布式数据进行实时加密, 可以有效保证数据的完整性。传统方法存在数据传输稳定性差、加密耗时长的问题, 为此提出一种基于椭圆曲线加密算法的多层差异网络分布式数据实时加密方法。分析基于椭圆曲线加密算法的多层差异网络分布式数据实时加密原理, 采用波动函数对数据传输稳定性进行控制, 依据横向波动因子和纵向波动因子计算出传输控制误差, 根据误差值, 利用反演波动函数非线性跟踪控制方法设计传输控制模型。引入小扰动原理分层设置密钥, 将不同的密钥与模型中不同的数据明文进行匹配, 实现多层差异网络分布式数据的实时加密。实验结果可得, 基于椭圆曲线加密方法分布式数据传输稳定性较强, 对分布式数据进行加密耗时较短。

    关键词：

    多层差异网络; 分布式数据; 反演波动函数; 加密;

    Abstract：

    The real-time encryption of distributed data in multi-layer differential network can effectively ensure the data integrity. Traditionally, the data transmission stability is poor and encryption time is long. Therefore, a method of real-time encryption for distributed data in multi-layer differential network based on elliptic curve cryptography algorithm was proposed. Firstly, the principle of real-time encryption of distributed data in multi-layer differential network based on elliptic curve cryptography algorithm was analyzed. Secondly, the wave function was used to control the stability of data transmission. According to the transverse fluctuation factor and the vertical fluctuation factor, the transmission control error was calculated. Based on the error value, the nonlinear tracking control method for inversion wave function was used to design the transmission control model. In addition, the small disturbance principle was introduced to hierarchically set the key. Finally, different keys were matched with different data plaintexts in the model. Thus, the real-time encryption of distributed data in multi-layer differential network was achieved. Simulation results show that the distributed data transmission based on elliptic curve cryptography algorithm is more stable. Meanwhile, the encryption of distributed data takes less time.

    Keyword：

    Multi-layer difference network; Distributed data; Inversion wave function; Encryption;

    1 引言

    网络技术的飞速发展, 各种网络形式蜂拥而至, 其中, 多层差异网络具有不同层次、不同深度网络空间的特点, 可以将海量网络资源, 按层次差异的特点分散存储到不同网络数据节点中, 大大的提高了网络的运行效率, 因此, 越来越受到人们的关注。随着网络资源数量的不断增长, 对多层差异网络要求也逐渐增高, 分布式数据库在多层差异网络的基础上, 可以将原来集中式数据库数据分散存储到多个通过网络连接的数据存储节点上, 以获取更大的存储容量和更高的访问量。多层差异网络因其具有不同层次结构的特点, 能够弥补其它网络形式不能对海量资源分层次的存储、传输的缺点, 因此, 被广泛应用于商业、军事、企业等诸多领域中。随着网络技术水平的不断提高, 对分布式数据库关注度越来越高, 分布式数据库改善了传统集中式数据库冗余数据造成存储空间的浪费, 提高了响应时间, 降低了通信成本[1]。但是由于多层差异网络结构的复杂性, 也存在其弊端。多层差异网络分布式数据传输过程也较为复杂, 导致存在安全隐患, 例如:分布式数据传输不稳定性的特性, 就会造成网络运行的不稳定性, 甚至造成网络的瘫痪[2]。因此, 对多层差异网络分布式数据稳定性的控制一直是网络科技研究的难点问题, 最有效的方式就是对分布式数据进行实时加密, 以此控制分布式数据的稳定性。由于多层差异网络的多层次的特点, 在多层差异网络运行和传输阶段, 需要对分布式数据进行实时加密。

    文献[3]提出一种无线传感器网络轻量级数据加密机制设计, 为提高密文的安全性与数据加密强度, 使用双密钥对明文进行两级加密, 并引入了随加密钥管理机制, 使网络节点每次加密时都能使用不同的密钥, 提高了密钥的安全性, 分布式数据加密机制还使用了节点ID认证、带有身份标识的密钥池认证等多种安全认证机制来阻止非法节点的接入。但该方法加密过程繁琐, 加密耗时时间长。文献[4]引入Storm分布式实时计算平台进行WAMS数据的实时加密处理。设计了基于云平台的WAMS数据存储系统, 并在此系统之上设计了基于Storm框架的高级加密标准 (AES) 算法。该算法将加密过程分为数据接入、快速并行数据加密及加密结果云存储等几个过程, 并在Storm预定义的编程组件中进行了编程实现。但该方法实现的复杂度高, 可操作性差。文献[5]提出一种基于模糊映射的计算机网络安全加密方法.通过模糊计算理论将计算机网络节点中的分布式数据排列成伪随机序列, 在此基础上导入模糊约束的分布式数据属性, 从而精确地完成对数据参数不定条件下的优化加密.但该方法实现过程复杂, 加密设置的密钥长度过长, 数据传输过程数据稳定性较差。

    针对上述问题, 本文提出基于反演波动函数的多层差异网络分布式数据实时加密方法, 实验结果表明, 基于椭圆曲线加密的方法可以有效控制数据传输的稳定性, 提高了网络数据的运行效率, 控制了加密成本。

    2 多层差异网络分布式数据实时加密系统原理

    提出的基于椭圆曲线加密算法 (ECC) 的多层差异网络分布式数据实时加密方法, 椭圆曲线加密算法具有高运算效率、高安全度、可运算的优点。分布式数据传输过程会出现数据不稳定的状态, 首先需要对数据稳定性控制, 并为传输数据设置密钥。然后对分布式数据进行分层次设置密钥, 最后, 在保证分布式数据传输稳定性的基础上, 对多层差异网络分布式数据进行实时加密, 以保证数据传输过程中的完整性。

    由于多层差异网络分布式数据传输过程会出现自由离散状态, 造成传输数据以游离态进行传输, 此时会产生多层差异网络分布式数据传输不稳定的状态。分析多层差异网络数据节点运动控制数学模型, 得到分布式数据节点波动过程中的纵向波动函数

    其中:节点波动矢量;ωx、ωy—多层差异网络分别横向波动因子和纵向波动因子;Jy—纵向波动所耗能量算子;Jx为横向波动所耗能量算子;e1—数据控制误差。

    为实现多层差异网络分布式数据实时加密算法, 建立一个精确的数学模型进行参数整定描述, 根据分布式数据传输状态空间模型, 得到KdV控制方程为

    通过控制方程, 对分布式数据传输状态时, 进行有效控制, 在游离状态的数据流得到有效控制的基础上, 对分布式数据进行私钥加密, 基于ECC的加密改进算法。如果使用相同私钥k加密明文, 一个私钥被破解, 在攻击者手中所有的密文将全部被破解, 后果不堪设想。为此, 为了加强整体分布式数据的安全性, 提出了使用不同的私钥k对明文进行加密的新思想, 进一步加强基于ECC的加密算法的安全性。

    3 多层差异网络分布式数据实时加密方法

    5 结论

    本文提出了基于反演波动函数的多层差异网络分布式数据实时加密方法, 通过对分布式数据加密的密钥长度对比, 在同安全等级条件下, 基于反演波动函数的加密方法长度最短, 说明该方法加密步骤简洁易操作, 可是操作性强。通过对比不同方法加密后数据传输的稳定性的对比, 基于椭圆曲线加密的方法对数据加密后, 数据传输稳定性要高与传统方法, 提高了运行效率
