    摘    要：

    无线传感器网络移动节点进行路径规划, 对最优路径有效的选择, 可提升无线传感器的性能。针对当前无线传感器网络移动节点路径规划中, 存在着路径规划所需完成时间过长, 能源消耗过大等问题, 提出基于粒子群优化的移动节点路径规划方法。在无线传感器网络移动节点路径规则中, 根据无线传感器网络移动节点的驻足位置, 利用数学公式计算出各个网格的节点能源消耗, 建立移动节点路径规划模型, 利用粒子之间互相协作得到各移动节点的位置和速度, 引用网络移动节点路径障碍物斥力场求出适应度函数, 根据适应度值求解出移动节点全局最优路径。实验结果表明。利用所提方法对无线传感器网络移动节点的路径进行规划, 所需完成时间较短, 能源消耗较少。

    关键词：

    无线传感器; 网络移动节点; 路径规划;

    Wireless Sensors Mobile Node Path Planning Simulation

    CHEN Xiao-ru

    South China Institute of Software Engineering, Guangzhou University

    Abstract：

    The path planning for mobile nodes in wireless sensor networks and the effective selection of optimal path can improve the performance of wireless sensors. In this paper, we focus on a method to plan the path of mobile node based on particle swarm optimization. In the path rule of mobile node in wireless sensor network, we used the mathematical formula to calculate the node energy consumption of each grid based on the stop position of mobile node in wireless sensor network, and then built the path planning model of mobile node. Moreover, we used the interaction between particles to get the position and speed of each mobile node. Finally, we used the repulsive field of obstruction in network mobile node path to determine the fitness function. Thus, we found the global optimal path of mobile node based on fitness value. Simulation results show that the proposed method needs shorter completion time and less energy consumption in planning the path of mobile node in wireless sensor network.

    Keyword：

    Wireless sensor; Mobile node; Path planning;

    1 引言

    无线传感器是由多个节点组成, 是根据通信方式形成的无线网络系统, 主要是对网络数据采集和处理网络监测区域中对象信息等问题[1]。无线传感器网络节点能量是有限的一般情况下不能变换, 大量的降低能耗成为无线传感器的主要问题, 为此需要找出无线网络传感器节点数据传输之间的最优路径[2,3]。但现阶段在无线传感器网络移动节点路径规划中, 普遍存在着路径规划所需完成时间过长, 能源消耗过大等问题。在这种情况下, 如何有效的设置高效、节能的无线传感器网络移动节点路径规划方法成为当今社会亟待解决的问题。

    目前, 俸皓、罗蕾、王勇[6]提出一种基于萤火虫算法的无线传感器网络移动sink节点路径规划方法。该方法根据无线传感器特性对空间进行了压缩, 为了提高空间的搜索效率, 利用离散萤火虫粒子算法对其进行全局最优路径搜索。该方法提高了收敛速度, 但是在路径规划过程中能源消耗较大。侯贵升、吴晓蓓、黄成[7]提出一种基于传感器网络中移动数据收集器的路径规划方法。该方法对传感器节点边缘度不整齐的进行适量节点去除, 利用节点筛选出优先度, 获取能效较低的节点, 以此完成路径规划。该方法能够有效的减小了网络中的负载现象, 但是路径规划所用的时间较长。屈应照、胡晓辉、宗永胜等人[8]提出一种基于数据融合的MobileAgent路径规划方法。该方法采用K-means聚类算法对网络数据进行分类, 利用Sink节点根据分类后的数据派发多个MA, 对MA进行节点访问时考虑到节点采集数据的规模和可用的空间, 根据每个MA的路径访问进行规划。该方法在进行路径规划过程中所用的能源消耗较低, 但是最优路径规划的成功率较低。

    针对上述问题, 提出基于粒子群优化的网络移动节点路径规划方法。具体内容如下:

    1) 根据无线传感器网络移动节点的驻足位置, 利用数学公式计算出各个网格的节点能源消耗, 建立网络移动节点路径规划模型。

    2) 利用粒子之间互相协作得到各移动节点的位置和速度, 引用网络移动节点路径障碍物斥力场求出适应度函数, 根据适应度值求解出移动节点全局最优路径。

    2 无线传感器网络移动节点路径规划原理

    在无线传感器网络移动节点路径规划原理中, 首先, 利用路径评价函数进行无线传感器最佳网络移动节点路径的搜索;其次, 通过蚂蚁搜索信息素来对传感器网络负载进行平衡;最后, 结合量子行为粒子群优化算法进行全局网络移动节点最佳路径求解, 完成网络移动节点路径的规划, 其实验该过程的具体步骤如下:

    在无线传感器网络中, 每一个移动节点经蚂蚁走过的路径上会含有信息素, 通过路径中的信息素和启发因子, 计算移动节点i向j节点的转移概率为

    Ρi, j (t) = (τi, j (t) ) α (ηi, j (t) β) ∑d∈tabuQ ( (τi, j (t) ) α) (ηi, j (t) β)     (1)

    式中, d为无线传感器任意网络节点, tabuQ为禁忌表, τi, j (t) 为t时刻在无线传感器链路上信息素, ηi, j (t) 为无线传感器网络节点启发因子, α, β为传感器信息素和启发因子的影响因子。

    以式 (1) 为基础, 利用下式给出无线传感器信息素规则为

    τi, j (t+Δt) =ρτi, j (t) +Δτi, j (t, t+Δt)     (2)

    式中, ρ为无线传感器信息素的挥发因子, Δτi, j (t, t+Δt) 为无线传感器信息素增量, 利用下式给出信息素释放能量为

    ∑k∈ΝΔτi, jk (t, t+Δt) =di, j+τi, j (t+Δt)     (3)

    式中, N为在无线传感器链路上蚂蚁数量集合, k为N当中的一只蚂蚁, di, j为传感器移动节点i到节点j的欧式距离。

    依据蚂蚁对网络移动节点最短路径的搜索规则, 利用下式给出无线传感器链路对蚂蚁的启发因子为

    ηi, j (t) β=CE (i, j)     (4)

    假设无线传感器中移动节点k当前位置的适应值为

    Gk (t) =φ (t) ⋅Rek (t) +γ (t) (τk (t) )     (5)

    式中, φ (t) 、γ (t) 为无线传感器适应值和信息素值的权重因子, τk (t) 为网络移动节点在相邻链路上的信息均值, Re k (t) 为移动节点k的剩余能量。

    以式 (5) 为基础, 利用下式给出蚂蚁搜索出的最优路由路径为

    xbest>xa, (xbest-xa) 2+ (ybest-ya) 2≤r    (6)

    上述过程即是无线传感器网络移动节点路径规划的原理。该原理把无线传感器延迟、带宽、丢包率作为评价指标, 获取移动节点有效路径评价函数, 将蚁群搜索算法和粒子群适应度相结合, 具有较好的网络移动节点路径规划效果。

    3 无线传感器网络移动节点路径规划

    3.1 无线传感器网络移动节点路径规划模型构建

    构建无线传感器网络移动节点路径规划模型, 是实现网络移动节点路径规划的前提和关键。根据网络移动节点的驻足位置, 利用数学公式计算出各个网格的节点能源消耗, 完成无线传感网络移动节点路径规划模型的构建, 其构建该模型的具体过程为

    假设, 当前的网络节点当前停留位置坐标为网格的中心i的位置坐标为 (xs, ys) (xi, yi) , 利用下式给出无线传感器网络移动节点与网格中心的直线方程为

    y=ys-yixs-xi (x-xi) +yi (xc, yc)     (7)

    式中, (xc, yc) 为网络移动节点监测区域边界交点。

    以式 (7) 为基础, 计算出边界交点 (xc, yc) 经过网格中心到达移动节点的最大跳数为

    Νi, max= (xs-xc) 2+ (ys-yc) 2r    (8)

    式中, r为无线传感器网络移动节点的密度。

    假设, 当无线传感器移动节点沿着路径Ρ=[p1, p2, ⋯, pm]移动到m个驻足位置得到无线传感器网络节点数据时, 该路径到网格中心的总节点能源消耗为

    Ei=∑p∈Ρε    (9)

    式中, ε为无线传感器发送数据能耗系数。

    以式 (9) 为基础, 利用下式给出无线传感器网络移动节点网格生存时间为

    Τ=min (Einital/Ei) V    (10)

    式中, V为所有网格中心集合, Einital无线传感器网络移动节点初始能源消耗。

    通过上述分析, 利用无线传感器网络节点移动距离和驻足个数, 建立网络移动节点路径规划模型

    max (Τ) =num (Ρ) +lenth (Ρ)     (11)

    式中, num (P) 为任一网络移动节点路径P的中心位置, lenth (P) 为任一网络节点移动路径P的全部路程。

    3.2 基于粒子群优化的网络移动节点路径规划

    以3.1节获取的网络移动节点路径规划模型为依据, 利用粒子之间互相协作得到各网络移动节点的位置和速度, 引用网络移动节点路径障碍物斥力场求出适应度函数, 根据适应度值求解出网络移动节点最佳规划路径, 其具体步骤为:

    若想获取网络节点的全局最优路径, 需要计算在搜索过程粒子的速度和位置, 假设, 粒子i的向量为Xi= (xi1, …, xim) , 粒子的搜索速度为Vi= (vi1, …, vim) , 利用下式给出粒子速度和位置为

    {Vi, j (t+1) =ωVi, j (t) + (pi, j (t) -Xi, j (t) ) Xi, j (t+1) =Xi, j (t) +Vi, j (t+1)     (12)

    式中, Vi, j (t+1) 和Xi, j (t) 分别为蚂蚁移动节点的速度和位置, ω为惯性权重因子, 权重因子越大, 全局最优路径搜索能力就越强。

    以式 (12) 为基础, 利用下式给出网络移动节点路径中障碍物附近斥力场函数为

    Urep={12η (1ρ (p) -1ρ0)     (13)

    式中, η为无线传感器网络移动节点路径障碍物的斥力系数, p为传感器网络节点在移动时的任意位置, ρ (p) 为无线传感器移动节点路径障碍物的斥力场作用距离。

    在式 (13) 的基础上, 引入传感器网络移动节点与目标点的相对位置, 获取网络移动节点在空间的任意位置的安全度为

    safei, j=∑m=-ss∑n=-sssign (m, n)     (14)

    式中, s为模板窗口尺寸因子, sign (m, n) 为移动节点路径障碍物标志。

    以式 (14) 为基础, 得到网络移动节点路径的安全度为

    Μp=d0+∑j=1mdpj+dend=∑j=0m+1dpj    (15)

    式中, d0为网络移动节点起始点的安全度, ∑j=1mdpj为节点p1到节点pm的安全度总和, dend为节点目标位置的安全度, Mp越小表示路径的安全度就越高。

    以式 (15) 为基础, 利用下式给出传感器网络移动节点第i个粒子的适应度函数为

    fp=ω1∑j=0n+ω2∑j=1mdpj    (16)

    利用适应度函数对无线传感器网络移动节点路径求解出全局最优路径, 适应度函数值越小, 路径规划最优。

    5 结论

    针对无线传感器网络移动节点路径规划中, 提出一种新的基于粒子群优化算法的无线传感网络移动节点路径规划方法。根据传感器网络移动节点的驻足位置, 利用数学公式计算出各个网格的节点能源消耗, 建立网络移动节点路径规划模型, 利用粒子之间互相协作得到各网络移动节点的位置和速度, 引用移动网络节点路径障碍物斥力场求出适应度函数, 根据适应度值求解出网络移动节点全局最优路径。仿真结果表明, 所提出方法能够获取路径规划的最优解。
