    摘    要：

    为了确保集成化网络存储的信息安全,需要实时进行节点安全脆弱性评估,检测安全脆弱性区域,以便及时修复。针对当前网络存储安全脆弱性区域检测方法存在的检测结果反馈延时较长,误检率较高,查全率较低等问题,提出基于神经网络的集成化网络存储安全脆弱性区域检测方法。根据贝叶斯攻击图分析集成化网络存储节点间的可达关系,结合证据理论与后验概率理论,综合计算原子攻击可达概率,据此评估对应区域的安全脆弱性;利用神经网络模型预估安全脆弱性区域检测器运行状态,并计算节点状态回执信息估计值,通过该估计值与安全脆弱性评估结果的比较,判定节点是否为安全脆弱性节点,检测出网络存储安全脆弱性区域。实验结果表明,所提方法的检测结果反馈延时短于实验对比方法,误检率低于实验对比方法,且查全率更高,具备较好的可操作性与扩充能力。

    Detection and Simulation of Integrated Network Storage Security Vulnerability Area

    BAO Hai-yan

    School of Information Technology and Engineering, Jinzhong University

    Abstract：

    Real-time evaluation for node safety vulnerability can ensure information safety of integration network storage. Safety vulnerability area is detected so that it can be repaired timely. Current detection method has many disadvantages, such as long feedback delay of detection result, high false detection rate, and low recall ratio. Therefore, based on neural network, a new detection method for safety vulnerability area is proposed in this research. According to the Bayes attack graph, accessibility relation between storage nodes of the integration network was analyzed. Integrated with evidence theory and posterior probability theory, accessibility probability of atomic attack was calculated comprehensively. Hereby, safety vulnerability of the corresponding area was evaluated. The neural network model was used to estimate operation state of detector of the safety vulnerability area and estimation value of receipt information of node state was calculated. The estimation value was compared with the estimation result of the safety vulnerability to judge whether the node was the safety vulnerability node. Thus, the safety vulnerability area of network storage was detected. Simulation results show that the feedback delay of detection result of the method is shorter. The false detection rate is low and the recall ratio is high. The method has better operability and expansion capacity.

    1 引言

    网络存储是基于数据存储的一种特殊的专用数据存储服务器,集成化网络存储是将各类特殊的专用数据存储服务器集合在一起,互相关联,形成一个性能更优的网络存储有机整体。集成化网络存储能够根据实际应用需求进行集成组合,采用集成化方式控制应用成本,提高集成整体的网络存储安全性与适用性[1,2]。集成化网络存储安全脆弱性区域检测,是网络信息安全领域的重点研究内容,通过检测及时发现脆弱性区域,便于快速进行漏洞修复,预防、抵御攻击[3]。相关检测方法的研究备受关注,现已出现一些较好的网络存储安全脆弱性区域检测方法[4]。

    文献[5]提出了基于漏洞扫描技术的网络存储安全脆弱性区域检测方法。利用漏洞扫描与挖掘技术,结合静态评估方法,评估集成化网络存储各节点的安全脆弱性,根据评估结果,完成网络存储安全脆弱性区域检测。该方法的误检率较小,但查全率较低,且检测结果反馈延时较长。文献[6]提出了基于资源图的网络存储安全脆弱性区域检测方法。结合层次分析法,构建网络存储节点脆弱性变更及安全状态关系网络资源图,结合变点检测方法实现集成化网络存储安全脆弱性区域检测。该方法的查全率较高,且检测结果反馈较快,但误检率较高。

    针对上述方法中存在的不足之处,提出基于神经网络的集成化网络存储安全脆弱性区域检测方法。结合贝叶斯攻击图、证据理论与及验概率理论,提高网络存储节点安全脆弱性评估结果的准确性;运用神经网络模型对脆弱性区域检测结果进行自适应学习与训练,减少计算量,提高检测效率。通过仿真,与当前方法进行比较,验证了所提方法的可行性。

    2 集成化网络存储安全脆弱性区域检测方法

    2.1 集成化网络存储安全脆弱性评估

    根据贝叶斯攻击图[7],简化网络存储服务器节点攻击状态的分析过程,结合攻击证据与评分计算,进行集成化网络存储节点安全脆弱性评估。具体过程如下:

    为评估集成化网络存储安全脆弱性,首先计算各网络存储节点的可达概率,记作P(Ti),∀Ti∈T0,其中To表示攻击初始状态,通常P(To)=1。

    若∀Ti∈Tin∪T1,根据Thomas Bayes理论,可得

    P(Ti)=P(Ti|Pa(Ti),Pa(Ti)→Ti) (1)

    其中,Pa(Ti)代表对应节点间的路径连通概率。根据优化的贝叶斯攻击图,网络存储节点的攻击可达概率,与对应的父节点及父节点到达该节点的原子攻击相关。

    集成化网络存储的父节点与子节点间存在的关系[8]如下

    1)AND关系,对应的公式为

    Ρ(Τi)=∏Τj∈Ρa(Τi)Ρ(Τj)×Ρ(Ρa(Τi)→Τi) (2)

    2)OR关系,对应的公式为

    Ρ(Τi)=1-∏Τj∈Ρa(Τi)(1-Ρ(Τj)×Ρ(Ρa(Τj)→Τi)) (3)

    3)MIX关系,对应的公式为

    Ρ(Τi)=1-∏k´=1Κ´(1-∏Τj∈Ρa(Τi)k´Ρ(Τj)×Ρ(Ρa(Τj)→Τi)) (4)

    举例说明:若网络存储节点T1、T2对应的可达概率为P(T1)=0.6、P(T2)=0.7,原子攻击路径T1→T3的可达概率为0.6,路径T2→T3的可达概率为0.5,通过计算可得网络存储节点T3的攻击可达概率为0.584。

    集成化网络存储节点规模较大,且所处网络环境相对复杂,进行网络存储节点安全脆弱性评估过程中,需要考虑相关的动态因素影响。随着集成化网络存储运行时间的变化,系统能够采集到部分或全部攻击行为,在此前提下,引入攻击证据E,提高网络存储节点攻击可达概率计算准确性。设攻击证据E={T′1,T′2,…,T′n},用于描述系统观察到的原子攻击到达状态,Ti'=1,Ti∈T-E表示攻击可达概率待确定的网络存储节点,根据后验概率理论[9],Ti的攻击可达概率利用下式计算

    P(Ti|E)=P(E|Ti)×P(Ti)/P(Ei) (5)

    根据评分基本度量组指标量值表,选择攻击途径AV、身份验证AU及入侵复杂程度AC作为指标,计算原子攻击自身成功概率P(a)

    P(a)=F(AV)×F(AU)×F(AC) (6)

    其中,F(AV)、F(AU)和F(AC)分别为各项指标对应的等级值。

    攻击能否成功的另一决定性因素是节点的可达概率,结合所有节点攻击可达概率,及各父节点与对应子节点的关系,综合计算原子攻击可达概率P′(a)

    根据计算得到的原子攻击可达概率,评估集成化网络存储对应节点或对应区域的安全脆弱性,为安全脆弱性区域检测提供判定依据。

    2.2 集成化网络存储安全脆弱性区域检测

    集成化网络存储的节点数量较多,神经网络模型的容错能力及自适应能力较强,是一种比较理想的脆弱性区域检测方法,能够较好描述被检测节点的状态与回执信息之间的关联[10]。以检测器得到的各节点状态信息及回执信息为训练样本,通过自适应学习构建网络模型,利用该模型计算节点状态回执信息估计值,将该估计值与脆弱性评估结果进行比较,根据二者之间的差值检测节点是否为安全脆弱性节点,实现集成化网络存储安全脆弱性区域检测。

    式中,Ky代表神经网络的占比系数;xSi和yS分别代表神经网络的输入值与输出值;θy代表脆弱性检测阈值,根据上一节脆弱性评估值确定;uq代表检测器输出;ωi、aij分别代表与神经网络输入、输出相对应的权重;biq代表对应的检测器输出的权重。

    在检测器输出误差的基础上,进行神经网络的样本训练与自适应学习[12],检测器输出误差J的描述式如下

    J=‖Yd(k+1)-Y(k+1)‖2

    =‖E(k+1)‖2 (15)

    式中,Yd(k)、Y(k)分别代表神经网络的期望输出值与实际输出值;E(k)代表Yd(k)、Y(k)之间的偏差值。

    运用神经网络算法,在三层神经网络结构的前两层完成自适应学习,在神经网络结构的最后一层,设定如下学习规则:

    ωijL-1(k+1)=JωijL-1(k)+θyσiL(k)yijL-1(k) (16)

    σiL(k)=2biq(k)[1-uj]Yd(k+1)+Q(k)xSi (17)

    各节点脆弱性检测器都对应一个神经网络,通过正常运行数据进行自适应学习训练,获取对应的神经网络输出值,利用神经网络预测下一次输出,并与实际输出值进行对比,若二者偏差大于阈值,说明对应的节点为脆弱性节点。

    综上所述,检出网络存储的所有安全脆弱性节点,最终确定集成化网络存储安全脆弱性区域。

    在网络技术飞速发展的今天,网络攻击与破坏数量也不断增长,脆弱性检测是集成化网络存储安全风险评估的重点。通过相关资料研究,针对当前方法中存在的问题,提出基于神经网络的集成化网络存储安全脆弱性区域检测方法。在集成化网络存储节点安全脆弱性评估的基础上,利用检测器进行脆弱性区域检测;采用神经网络算法对检测过程进行优化,提高检测效率,提高检测结果的准确性;通过仿真,检验了所提方法的性能。
