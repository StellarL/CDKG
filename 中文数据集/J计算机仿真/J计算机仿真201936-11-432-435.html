    摘    要：

    针对当前攻击路径长度预测方法存在网络攻击成功概率低,路径长度预测准确性差的问题,提出基于吸收Markov链的网络攻击路径长度预测方法。采用转移概率度量法对网络中节点访问的期望值展开分析,利用比例系数计算网络攻击图状态转移边的概率比,以网络过渡状态下期望转移概率、非零矩阵等为基础分析网络攻击者从开始到达结束的概率,并计算网络受到攻击后,初始节点的转移概率,得到网络攻击的数学期望矩阵,利用上述矩阵计算网络节点访问次数的期望值总和,利用节点的转移次数计算网络攻击路径的期望长度,构建网络过渡状态下期望转移概率矩阵,通过对矩阵的求解,最终实现多数据源层次化网络攻击路径长度的预测。实验结果表明,提出方法在对多数据源层次化网络攻击路径长度预测时,网络目标节点的置信度较低,并且网络攻击成功概率较高,不仅验证了提出方法良好的性能,也提高了网络攻击路径长度预测的准确性。

    关键词：

    多数据源; 层次化; 网络攻击; 路径长度预测;

    Multi-Data Source Hierarchical Network Attack Path Length Prediction Simulation

    ZHANG Ke

    School of Data Science and Information Engineering, Guizhou Minzu University

    Abstract：

    Current method to predict the attack path length has low successful probability and low prediction accuracy of path length. Therefore, a method to predict the network attack path length based on absorbing Markov chain was proposed. The measure method of transition probability was used to analyze the expected value of node access in network and the proportionality factors were used to calculate the probability ratio of state transition edge of network attack graph. Based on the expected transition probability and non-zero matrix in the network transition state, the probability of network attacker from start to end was analyzed. And then, the transition probability of initial node after the network attack was calculated to obtain the mathematical expectation matrix of network attack. In addition, this matrix was used to calculate the sum of the expected values of network node access times. Furthermore, the expected length of network attack path was computed by the number of node transition. The expected transition probability matrix under the network transition state was constructed. By solving the matrix, the prediction for the multi-data source hierarchical network attack path length was achieved. Simulation results verify that the proposed method has lower degree of confidence of network target node in predicting the length of multi-data source hierarchical network attack path and the successful probability of network attack is high. Meanwhile, the good performance of proposed method is proved and the prediction accuracy of network attack path length is improved.

    Keyword：

    Multiple data sources; Hierarchical; Network attack; Path length prediction;

    1 引言

    在网络攻击技术的快速发展下,导致多数据源层次化网络攻击具有多步性,网络攻击行为多步性是指一些攻击者根据网络中存在的漏洞蓄意进行多步骤攻击,严重的可以对个人、企业、政府的网络安全造成严重威胁,例如僵尸网络利用窃取用户私人信息、传播病毒、主机感染等方式给网民带来巨大的经济损失[1]。近年来,APT攻击也成为威胁较大的攻击行为。一些研究学者纷纷开始关注多数据源层次化网络攻击问题,网络攻击的防护也逐渐被重视起来,网络攻击路径长度的预测是指根据攻击行为的产生机制,建立攻击场景,预测出网络攻击路径的长度,可以保护多数据源层次化网络的安全,也是应对网络攻击的主要技术。

    文献[2]提出基于大数据的多数据源层次化网络攻击路径长度预测方法,通过对网络报警数据进行预处理以提高效率,经过大数据分析后,将隐马尔科夫模型用在攻击路径长度的预测中,根据获得的报警信息挖掘网络中存在的攻击行为,根据关联规则法和网络攻击行为概率分布情况来获得观察矩阵,引进粒子群算法来优化参数,最终完成了对网络攻击路径长度的预测,该方法的收敛性较好,但网络目标节点的置信度较高,攻击路径中存在的冗余较多。文献[3]提出基于树型结构的多数据源层次化网络攻击路径长度预测方法,引入杀伤链建模原理,分析网络攻击过程的阶段性特征,构建网络攻击模型,对日志记录进行关联分析,引入可信度对网络攻击事件进行确定,并计算攻击路径,实现了网络攻击路径长度的预测,该方法有较好的实用性,但同样存在目标节点置信度高、路径冗余多的问题。文献[4]提出基于可能图的多数据源层次化网络攻击路径长度预测方法,将该方法与不确定图概念相结合来扩展攻击图,得到可能攻击图,构建网络攻击图,利用最大攻击子图和最大可达概率求解网络攻击路径,该方法可以有效推测网络的攻击意图,但网络攻击成功的概率较低,不能够对网络攻击路径的长度进行准确预测。

    针对上述三种研究方法中存在的缺陷,提出基于吸收Markov链的多数据源层次化网络攻击路径长度预测方法。

    2 多数据源层次化网络攻击图

    多数据源层次化网络攻击图是指占有网络资源节点的步骤,其中包括攻击行为节点、资源节点与两个节点间的有向边[5],对多数据源层次化网络攻击图分析:

    攻击图NAG属于九元组,NAG=(⨂,R,A,E,S,Rv,F,D,∏)表示多个由OR或者AND连接关系节点组成的有向五环图,其中,⨂代表网络节点的状态集合,有

    ⨂={TRUE,FALSE} (1)

    假设⨂(ri)、⨂(ai)分别为节点ri、ai的状态,R为网络资源节点集合,则有R={ri|i=1,2,...,n},ri表示资源节点,当⨂(ri)=TRUE时,表示资源节点ri被占有,当⨂(ri)=FALSE时,表示资源节点ri未被占有,A为网络攻击行为的节点集合[6,7],有A={ai|i=1,2,...,n},ai表示网络攻击行为节点,当⨂(ai)分别为TRUE和FALSE时,表示网络攻击行为ai已经发生和未发生。E表示各类节点关联的有向边集合,可以表示为

    E={e|∀ai,aj∈A,ri∈R,ri>∪<ri,aj>} (2)

    式(2)代表e在<ai,ri>和<ri,aj>中的一个节点。D=(di∪d′i),其中,di代表网络攻击节点ai发生的概率,d′i代表ai使目标⨂(ri)=TRUE的概率,有下列表达式

    di=P(⨂(ai)=TRUE|Per(ai)=TRUE),d′i=P(ai|⨂(ai)=TRUE) (3)

    S={si|∀ai∈A,ri∈R,i=1,2,...,n}表示网络攻击的属性集合,其中si表示边<ai,ri>的威胁属性。Rv{v(ri)|ri,i=1,2,...,n}表示网络中资源节点的脆弱属性集合,其中,v(ri)为ri的威胁属性,F={TRUE,FALSE}表示网络攻击威胁的状态集合,当e∈E,并且e属于<ai,ri>时,F(e)=FALSE说明ai没有威胁性。其中,∏为多数据源层次化网络攻击图中被攻击节点出现概率,∏∈[0,1],∏(ri)、∏(ai)表示在网络攻击图中的资源占有情况和发生攻击行为的概率分布。

    多数据源层次化网络攻击图中,一个网络节点的边有OR或者AND连接关系,AND连接关系代表网络的父节点满足指定条件即可到达子节点,OR连接关系代表网络中的父节点满足一个条件即可到达子节点[8],经过安全检测系统以及专家知识的观测值能够获得di和d′i的概率。多数据源层次化网络攻击图的表现形式如图1所示。

    3 基于吸收Markov链的网络攻击路径长度预测

    根据CVSS的转移概率度量方法对网络节点访问次数的期望值进行分析,并对网络攻击路径长度的期望值展开计算。

    3.1 转移概率度量

    采用网络攻击难度对状态转移概率进行度量,假设原子攻击α依赖漏洞可利用性得分为score(α),得分越高,网络攻击难度越小,漏洞越容易被利用,状态转移成功的概率越大,在网络节点转移失败的情况下[9],可利用得分被规定为10。假设网络攻击图中score(α)和状态转移边e′的概率为正比,则有如下表达式

    Δe′=f×score(α) (6)

    式中,f表示比例系数。

    N=(I-Q)-1=S1·1/2S2+S3S4 (13)

    3.3 网络攻击路径长度预测

    在对节点访问的期望值预测的基础上,对网络攻击路径长度进行预测,网络攻击者从Si′开始,Ti′为吸收之前网络攻击路径的长度,假设T、C分别表示大小为t×1的矩阵和单位矩阵,Ti′赋值给T位置上(i′,1)的元素,则T为网络攻击期望路径的长度矩阵,表达式如下

    T=N·C (14)

    对于网络节点Si′来说,在吸收之前转到S1,S2...,St的输出次数Ni,1,Ni,2,...,Ni,t,从Si′开始的网络攻击路径长度为

    Ti′=Ni,1+Ni,2+...+Ni,t (15)

    则有

    T=(Ti′)=(Ni,1+Ni,2+...+Ni,t)=N·C (16)

    多数据源层次化网络攻击路径长度预测步骤为:

    输入网络中漏洞可利用性得分score(α)和网络攻击图;输出网络状态节点Si′为网络的初始节点,网络攻击路径的期望长度为Ti′;构建t×t的网络过渡状态下期望转移概率矩阵Q;计算t×t的期望矩阵N=(I-Q)-1;令C=(1,1,...,1)T,对t×1网络攻击路径长度期望值展开计算,Si′开始的网络攻击路径长度期望值EARL(Si′)=Ti′,完成对攻击路径长度的预测。

    计算复杂度的分析:该方法中包括矩阵相乘、相加和求逆,当有两个n′×n′矩阵相乘时,此时网络攻击路径长度预测的时间复杂度阶数为O(n'3);维护t×1矩阵C和T、t×t矩阵、t×t矩阵Q以及n′×n′矩阵P′,此时网络攻击路径长度预测的存储复杂度为O(n'2)。经过上述分析与计算,完成了对网络攻击路径长度的预测,表达式如下:

    W=(T+Ti′)·N(Si′+Sj′) (17)

    分析图2可知,对网络攻击路径长度预测时,提出方法的网络目标节点置信度在0.5以下,文献[2]方法的网络目标节点置信度在0.6～0.8之间变化,文献[3]方法的网络目标节点置信度在0.7～0.9之间变化,经过对比可以得到,文献[2]和文献[3]方法没有考虑到网络攻击路径中存在冗余的问题,选择网络中全部的路径进行攻击,没有筛选攻击路径,同时也忽略了攻击成本的问题,则得到的网络目标节点有比较高的置信度,提出方法的网络目标节点置信度较低,攻击者选择淘汰无益或者选择有益的多数据源层次化网络攻击路径,通过减少网络攻击路径来降低网络目标节点的置信度,验证了提出方法良好的性能。

    在对网络目标节点置信度变化测试的基础上,利用网络攻击成功的概率进一步测试多数据层次化网络攻击路径长度预测的准确性,攻击概率越高,暴露出的攻击行为则越多,攻击足迹越完整,因此就能够得到准确的攻击路径长度预测结果,测试结果如图3所示。

    分析图3可知,通过对网络节点访问期望值的计算,提出方法在6次实验中,网络攻击成功的概率均在90%左右,从图中可以看出,网络攻击成功概率的最高点接近100%,文献[4]方法的6次实验中,网络攻击成功的概率均在60%以下,而该方法的网络攻击成功概率才接近60%,对比可知,提出方法在对多数据源层次化网络攻击路径长度预测时,网络攻击成功的概率较高,说明提出方法具有较好的准确性,可以有效的对路径发生的概率进行判断。

    5 结论

    多数据源层次化网络攻防具有复杂性和动态性,为了研究网络的攻击路径,提出基于吸收Markov链的网络攻击路径长度预测方法,分别测试了网络目标节点置信度的变化情况和网络攻击成功的概率,结果表明,提出方法在对网络攻击路径长度预测时,网络中目标节点的置信度较低,并且网络攻击成功的概率较高,不仅验证了提出方法良好的性能,还通过网络攻击成功概率的结果证明了提出方法的准确性,为网络实时防御以及管理工作提供重要的参考意见,还为网络攻击路径长度预测的进一步研究奠定基础。
