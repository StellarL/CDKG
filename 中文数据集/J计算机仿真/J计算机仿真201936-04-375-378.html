    摘    要：

    针对传统稳定性能测试方法存在测试成本较高、测试时间过长以及查准率较低等问题, 提出了一种新的稳定性能测试方法——基于混沌特征分析的监控系统稳定性能精准测试方法。根据选择性集成策略选取系统稳定性的部分特征进行集成, 利用改进后的搜索方法搜取最优特征子集。利用最优特征子集构建无线医疗监护系统数据库模型, 根据网络数据库模型, 建立稳定性系数偏差函数, 对不同数据库的稳定性数据进行偏差统计, 根据偏差函数的统计结果进行最小值求解, 通过求解描述最小化的阈值请求, 完成无线医疗监护系统运行稳定性能精准测试。实验结果表明, 所提方法有效提高了查准率, 降低了测试成本。在保证工作效率的前提下, 有效减少了测试时间。

    无线医疗; 监护系统; 运行稳定性能; 精准测试;

    Wireless Medical Monitoring System Running Stability Can Accurately Test Simulation

    LIU Zi-jian LIU Jia-lei

    Department of Design and Art Shaanxi University of Science & Technology

    Abstract：

    In this article, a new method to accurately test running stability of monitoring system based on chaotic feature analysis was proposed. According to the selective integration strategy, some features of system stability were selected for integration, and then the improved search method was used to find the optimal feature subsets. After that, the optimal feature subsets were used to construct the database model of wireless medical monitoring system. Based on the network database model, the stability coefficient deviation function was established. The deviation statistics was applied to the stability data in different databases. Then, the statistical result of deviation function was used to solve the minimum value. Finally, the minimum threshold request was described based on the solution. Thus, we completed the accurate test of wireless medical monitoring system stability. Simulation results verify that the proposed method effectively improves the precision ration and reduces the test cost. On the premise of ensuring work efficiency, the test time is effectively reduced.

    1 引言

    物联网技术的飞速发展, 给医疗行业带来了深远的影响[1,2]。无线医疗监护系统的开发彻底改变了传统的监护方式, 可以有效预防突发情况, 实时进行提醒[3,4]。新的监护方式相比传统方法具有一定的优越性, 但是这种新型监护系统的使用需要进行验证[5]。目前, 如何能够保证监护系统的稳定运行, 成为了当前研究的重点内容。

    目前, 已经研究出了许多新的稳定性能测试方法。例如文献[6]提出基于小波动稳定理论的系统运行稳定性测试方法。该方法根据对系统运行时的效用时间常数、输出值变化及系统控制方程, 结合小波动稳定理论构建系统运行稳定性分析模型, 利用模型完成系统稳定性测定。该方法所需的测定时间较短, 但系统运行稳定性测试的结果准确度较低。文献[7]提出一种基于STM32嵌入式系统Ni-MH电池性能测试方法。该方法利用平均最大信号方法建立大信号模型, 根据模型获取变换器控制方法, 利用非线性切换函数实现变换器控制平滑度, 以此完成测试。该方法在进行测试过程中, 所需测试时间过长。

    针对传统方法存在的问题, 提出了一种新的稳定性能测试方法——基于混沌特征分析的监护系统稳定能精准测试方法。

    2 无线医疗监护系统运行稳定性能测试原理

    对于无线医疗监护系统输出的各种稳定性指标, 采用聚类算法对各指标值进行聚类计算, 获得各类指标的聚类标准和各类指标间的关系, 并据此绘制聚类后的稳定性指标谱系图, 根据各类指标的差别度判定系统运行的稳定性。具体过程如下:

    利用聚类算法对无线医疗监护系统运行稳定性的指标相似性进行度量, 以“距离”表征各类指标间的相似度。设每个系统xi的运行稳定性指标为p个, 各测试指标观测值为xi= (x1i, x2i, …, xpi) T, i=1, 2, …, n, 将其映射到多维空间中, 以各点之间的距离来度量各系统指标之间的相似程度。

    假设d (xi1, xi2) 为指标xi1与xi2间的距离, 需要满足以下条件:

    1) d (xi1, xi2) ≥0, 且当且仅当xi1=xi2时, d (xi1, xi2) =0;

    2) d (xi1, xi2) =d (xi2, xi1) 且任意三项指标对应空间点之间的距离值间的关系为d (xi1, xi2) ≤d (xi1, xi3) +d (xi3, xi2) 。

    根据无线医疗监护系统的监护环境、监护需求与空间范围等条件, 选择几个输入值与表征系统运行稳定性的评测指标。各输入值对应的输出值Xi为一类, 输出值中的各指标为x′p′i, 得到输出值的表征系统稳定性的观测值x′i= (x′1i, x′2i, …, x′p′i) T。p′的值根据系统输出值所包含的指标数量来确定。对所测试的输出值进行聚类划分, 为简化公式, 用i′、j′分别表示输出值Xi和Xj, 则输出值i′、j′间的距离d (Xi, Xj) 可记作di′j′;以Gp、Gq代表两个类别, 类别内包括的输出值数量分别记作np和nq, 类间距离记作D (Gp, Gq) , 其计算公式为

    D (Gp, Gq) ≙min{di′j′|i′∈Gp, j′∈Gq}    (1)

    即以类别Gp、Gq中最接近的两个输出值的距离作为类间距离。类间最短距离的递推过程为:假设Gp、Gq的组合结果为Gr, 则Gr与其它类别Gk (k≠p, q) 间的最短距离计算公式如下:

    {D (Gp, Gk) =min{di′j′|i′∈Gp, j′∈Gk}D (Gq, Gk) =min{di′j′|i′∈Gq, j′∈Gk}    (2) D (Gr, Gk) =min{di′j′|i′∈Gr, j′∈Gk}  =min{D (Gp, Gk) , D (Gq, Gk) }    (3)

    按照计算结果, 对各组输出值进行聚类, 步骤为:

    1) 将n组输出值作为n类, 计算类间距离, 得到对称矩阵D= (di′j′) n×n, 矩阵对角线元素皆等于零, 则有D (Gp, Gq) =dpq。

    2) 选择初始矩阵D0中下三角区域的最小元素, 记作D (Gp, Gq) , 使Gp、Gq合并, 得到Gr={Gp, Gq}, 划除Gp、Gq所对的矩阵行列, 加入Gr, 构成新的矩阵D1。

    3) 循环上述操作, 直到所有输出值聚集成一个大类。在操作过程中, 根据各类合并时的类间距离, 绘制类别间的谱系图。

    4) 根据各组输出值间的差值度, 评测无线医疗监护系统运行的稳定性, 差值度小, 表明相似性高, 判定系统运行性能稳定;差值度大, 表明相似性低, 判定系统运行不稳定。

    综上所述, 即为无线医疗监护系统运行稳定性能测试原理。

    3 无线医疗监护系统运行稳定性能精准测试方法

    3.1 系统稳定性最优特征子集的获取

    根据选择性集成策略选取系统稳定性的部分特征进行集成, 利用改进的搜索算法搜寻二次搜索最优特征子集。具体过程如下:

    假设在m个特征上属性上的期望输[8]出为

    D=[d1, d2, ⋯, dm]    (4)

    其中, dj代表第j个属性的期望输出值, dj∈ (-1, 1) , (j=1, 2, …, m) 。如果fi代表第i个特征选择器的实际输出, 则

    fij∈{-1, 1} (i=1, 2, ⋯, Ν, j=1, 2, ⋯, m)     (5)

    利用下式给出不同特征的泛化误差[9]为

    Ei=1m∑j=1mError (fijdj)     (6)

    则Error (x) 可以设定为

    Error (x) ={1,  if x=-10.5,  if x=00,  if x=1    (7)

    式中, Sumj代表所有个体特征选择器在第j个属性上的输出为

    Sumj=∑i=1Νfij    (8)

    利用下式给出第j个属性上的输出为

    fj=sgn (Sumj)     (9) sgn (x) ={1,  if x>00,  if x=0-1,  if x<0    (10)

    则集成的泛化误差为

    E˜=1m∑j=1mError (fjdj)     (11)

    如果集成中剔除第k个特征选择器, 则新集成在j个示例上的输出为

    f′j=sgn (Sumj-fkj)     (12)

    利用下式给出新集成的泛化误差为

    E′=1m∑j=1m∑j=1mError (fj′dj′)     (13)

    利用准确率评估准则对特征提取进行排序

    g″=1m∑j=1mError (fjdj) ⊕kj    (14)

    如果设定N为流量样本数量[10], m代表应用类型种类, 利用下式给出综合评价的表达式为

    ΟA=∑i=1mΤΡi∑i=1m (ΤΡi+FΝi)     (15)

    式中, TPi代表尺度空间, FNi代表信息熵。

    S={S1, S2, ⋯, Sn}    (16)

    C (fi) =Ffi-FminFmax-Fmin    (17)

    3.2 基于混沌特征分析的监护系统稳定性测试方法

    利用特征构建无线医疗监护系统数据库模型, 根据网络数据库模型, 建立稳定性系数偏差函数, 对不同数据库的稳定性数据进行偏差统计, 根据偏差函数的统计结果进行最小值求解, 通过求解描述最小化的阈值请求, 完成无线医疗监护系统运行稳定性能精准测试。具体过程如下:

    利用下式给出Duffing混沌振子无线医疗监护系统数据库模型表达式为

    y=ω (k+ax+βx2+γcos (ωt) )     (19)

    式中, k代表无线医疗监护系统运行的数据规模, ax+βx2代表无线医疗监护系统数据库的非线性恢复力。

    假设, 无线医疗监护系统时间序列X中的分段数量为B, HB为分段数量集合, 利用下式给出, 整个无线医疗监护系统时间序列的偏差集合为

    Ex (ΗB) =∑i=1BF (bi)     (20)

    式中, F (bi) 代表时间序列均方差。

    假设, 无线医疗监护系统时间序列中第i个分段用bi表示, 当bi个系统时间序列的分段中带有偏差数量为t, 利用下式给出, 无线医疗监护系统稳定性系数偏差函数为

    F=a (j1, ii) Τ+E (i, j)     (21)

    式中, a (j1, ii) Τ代表无线医疗监护系统运行稳定性均值, E (i, j) 代表无线医疗监护系统运行稳定性总偏差。

    在进行无线医疗监护系统运行稳定性测试的过程, 就是对系统时间序列偏差函数进行最小值求解的过程。

    利用式 (21) 给出, 根据偏差函数统计结果进行最小值求解

    E (i, j) =F* (l+1⋯i, k-m)     (22)

    式中, F* (a…b, c) 代表无线医疗监护系统时间序列的分段中偏差数量为c的均方差最小值。

    无线医疗监护系统运行稳定性分段数量为j的时间序列是x1…xi, 稳定性偏差数量为k, 对偏差函数最小值求解结果为E (i, j) , 通过上述可知, 当e≥b-a+1时, F* (a…b, c) 为0时, 利用下式给出, 通过求解描述最小化的阈值为

    S*[a, b, c]=∑a≤i≤bxi    (23)

    当b为异常数据点时, 在无线医疗监护系统稳定性时间序列中需要对它进行去除, 并对F* (a…b, c) 进行重新更新为

    {S*[a, b, c]=S*[a, b-1, c-1]F*[a⋯b, c]=F*[a⋯b-1, c-1]    (24)

    当b为正常数据点, 对F* (a…b, c) 进行重新更新为

    S*[a, b, c]=S*[a, b-1, c-1]+xb    (25)

    通过式 (23) 和式 (24) 进行对比, 获取最小的F* (a…b, c) 所对应的S*值。

    由上述方法可以对无线医疗监护系统运行稳定性进行确定, 以完成对稳定性能的精准测试。
