    摘    要：

    针对传统的移动学习资源自动整合方法含有过多无效数据,存在资源整合所需时间较长、整合后的无效资源过多等问题,提出一种基于片段式移动学习资源自动整合方法。采用不完全资源管理器中的片段式资源特征进行相似度度量,运用近邻传播算法对不完全资源进行聚类,根据聚类结果将同一类资源对象划分到相同簇中,利用同一类对象的属性值对缺失值进行填充。将获取的完整资源集中的各资源进行小波分解,获取小波系数,通过该系数对各个资源的小波方差进行计算,得到小波熵,运用小波域移动学习资源整合算法以小波熵作为资源融合权值,对得到的系数进行整合,经过小波逆转换得到移动学习资源自动整合结果。实验结果证明,所提方法能够缩短整合所需时间,减少整合过程中出现的无效资源,从而提升整合的效果。

    关键词：

    相似度度量; 近邻传播算法; 小波系数; 小波熵;

    Simulation of Automatic Integration Method for Segmented Mobile Learning Resources

    LI Jing-li WANG Huai-yu

    Information Engineering Institute,Baoding University

    Abstract：

    Aiming at the problems of excessive invalid data in traditional automatic integration method of mobile learning resources, such as long time for resource integration and excessive invalid resources after integration, this paper proposes an automatic integration method of mobile learning resources based on fragments. The fragmented resource features in incomplete resource managers are used to measure similarity, and the nearest neighbor propagation algorithm is used to cluster incomplete resources. According to the clustering results, the same class of resource objects are divided into the same cluster, and the missing values are filled by the attribute values of the same class of objects. The wavelet coefficients are obtained by decomposing the resources in the complete resource set and calculating the wavelet variance of each resource through the coefficients. The wavelet entropy is obtained. The coefficients are integrated by using the wavelet entropy as the weights of resource fusion in the mobile learning resource integration algorithm in the wavelet domain. The coefficients are reversed by the wavelet transform. In exchange for the result of automatic integration of mobile learning resources. The experimental results show that the proposed method can shorten the integration time, reduce the invalid resources in the integration process, and improve the integration effect.

    Keyword：

    Similarity measure; Nearest neighbor propagation algorithm; Wavelet coefficient; Wavelet entropy;

    1 引言

    移动学习属于一种非正式的学习方式[1],其是在移动的环境中进行的,可以使学习者在非预先设定和非固定的环境下进行学习,具有及时性、移动性和虚拟性,能够使学习者摆脱空间和时间的限制[2]。由于移动学习的移动性使学习者无法进行系统且连贯的学习,因此需要对零碎的、非系统的移动学习资源进行整合[3,4]。如何应用移动技术的研究成果,整合片段式移动学习资源的应用潜力,构建适合学习者学习的移动学习资源,值得深入研究[5]。

    文献[6]提出了一种基于异常数据驱动的簇内移动学习资源整合方法。在移动学习资源采集过程中,当异常资源发生时将其传送给簇首,减少监测网络的移动学习资源传输量。在簇首异常移动学习数据融合过程中,构建资源采集设备间的相互支持度矩阵,支持度值较低的资源将被剔除,对支持度值较高的资源进行最优加权融合。该方法资源整合效率较低。文献[7]提出一种基于深度学习模型的移动学习资源整合方法,在移动学习资源汇聚节点上创建资源特征提取模型进行训练,不同终端节点通过特征提取模型获取原始移动学习资源特征,最终向汇聚节点传送整合后的移动学习资源。该方法资源整合所需时间较长。文献[8]提出一种面向多目标追踪的PHD滤波移动学习资源整合方法。构建高斯混合滤波的多传感器移动学习资源整合系统框架,通过混合滤波算法对移动学习资源进行局部状态估计,对其估计结果进行关联度计算,根据计算结果构建自适应混合参数,引入协方差交叉算法对关联状态进行融合。该方法在资源整合过程中,出现较多的无效资源。

    针对上述问题,提出一种片段式移动学习资源自动整合方法。实验结果表明,所提方法能够缩短资源整合所需时间,减少整合过程中的出现的无效资源。

    2 片段式移动学习资源自动整合方法

    2.1 对不完整的移动学习资源填充

    对不完整移动学习资源集中存在的部分资源对象的离散属性进行归一化处理,获取片段式连续资源特征值,通过区间相似度度量方式将连续资源特征进行相似度度量,获取相似度度量系数,通过近邻传播对其系数进行聚类[9],将聚类结果中的相同资源对象划分到对应的簇,计算各个簇中心的相似度,运用相同一类资源对象的属性值对缺失值进行填充。

    对不完整移动学习资源进行聚类,假设整个资源对象集合为O,且O={o1,o2,…,on},集合O中各个对象有m个属性{a1,a2,…,am},即A={a1,a2,…,am}为属性集合,首先将全部资源集分成C和I两个集合,即O=C∪I。其中,C∩I=φ且t+p=n;C为完整资源集,即C中全部资源对象的属性值都是完整的,I代表不完整移动学习资源集,即I中资源对象都含有缺失属性值[10],在对资源进行划分后,运用近邻传播算法对资源集C进行聚类处理,这样能够有效降低数据聚类过程中受到的干扰程度,同时,采用相似度度量方法将不完整资源集I中的资源对象划分到完整资源集中相应簇中。

    资源对象属性集合具有离散属性,采用相似度距离可以将离散属性变成资源属性,这样可以使离散属性参与到计算中。假设n维资源能够代表具有n个离散属性,而这些离散属性具有不同的取值范围,则通过转换可以保证资源不被流失。

    由于离散属性范围不同会导致不平衡状况的产生,因此为了降低不平衡性,需要对所有的属性值进行处理,即将它们划分到统一的取值范围内,并且使每一个学习资源都具有其自身的权重值,再通过标准偏差实现属性集合中特征值的归一化处理。

    针对连续移动学习资源的属性,采用区间划分的相似度量方式,假设完整资源集C聚类后得到k个簇,C={C1,C2,…,Ck},这些簇满足以下条件:Ci≠φ,i=1,2,…k;Ci∩Cj=φ,i,j=1,2,…k。

    假设,第i个簇Ci中资源对象的第j个属性aj为连续性资源属性,且在其所属簇中的取值范围为[x,y]。用aji代表第i个簇的资源中心ci在第j个属性上的取值,同时使aji∈[x,y]。假设两个相似度度量系数α与β,其中:

    对于不完整资源集I中的任意资源对象b,使b在第j个属性aj上的取值为ajb,则b与ci在属性aj下取相同值的概率Paj(b,ci)计算如下:

    1)当|ajb-aji|≤α时,Paj(b,ci)=1;

    2)当α<|ajb-aji|≤β时,Paj(b,ci)=1/|V|;

    3)当|ajb-aji|>β时,Paj(b,ci)=0;

    假设第i个簇Ci中移动学习资源对象的第k个属性ak为离散型属性,用aki代表第i个簇的资源中心在第k个属性上的取值,假设b在第k个属性ak上的取值为akb。资源对象b与ci在属性aj下取相同值的概率Paj(b,ci)为:

    1)当akb=aki时,Paj(b,ci)=1;

    2)当akb≠aki时,Paj(b,ci)=0;

    为了避免传统相似度度量产生大量0值的不足,根据上述设定对不完整移动学习资源集I中的资源对象b与完整移动学习资源集簇中心ci的任意相似度进行计算,具体过程如下式

    Similarity(b,ci)=∑j=1qΡaj(b,ci) (2)

    式中:q表示资源属性的个数,该资源属于连续型资源;k代表离散属性个数。

    根据区间划分结果得出不完整资源集,根据该资源集的相似度进行资源聚类,具体过程如下:

    1)将移动学习资源集O划分为完整资源集C与不完整资源集I,即O=C∪I。

    2)完整型资源集C中的资源具有离散属性,对该集合进行转换,将其转变成具体的数值,并对资源集C中所有资源对象的属性值进行归一化,得出归一化处理结果。

    3)对移动学习资源集C中资源对象的相似度进行计算,得出相似度矩阵S。

    4)对归属度矩阵A和吸引度矩阵R进行初始化处理,并根据上述公式(4)和公式(5)对归属度矩阵A和吸引度矩阵R进行更新处理。

    5)假如聚类中心完成迭代次数,或者聚类中心没有产生变化,需要停止计算;如果聚类中心产生变化,则需要返回步骤4)。

    6)对角线值a(k,k)+r(k,k)>0的资源点k为自动发现的聚类中心,对于资源点i,使满足a(i,k)+r(i,k)的最大候选类中心作为最终的归属聚类中心。

    7)针对移动学习资源集C中的连续性资源属性,计算其在各个簇中的相似度度量系数α与β。

    8)对不完整移动学习资源集进行划分,将资源集I中的资源对象划分到相应的簇中,再通过公式(1)对资源集I中各簇中心的相似度进行计算,根据计算结果选取和相似度最贴近的簇当做聚类中心。

    对不完整移动学习资源进行聚类后,利用缺失属性对象在相同簇中资源属性的加权值作为其属性的预测值,通过资源对象间相似度提供的信息来判定加权系数。

    对于不完整移动学习资源集中的资源属性对象o,假设与其处在相同簇中的完整资源集中的资源对象分别为o1,o2,…om,对于不完整移动学习资源集中的资源对象进行填充,主要步骤如下:

    1)计算o与o1,o2,…om的相似度,记为s1,s2,…sm。

    2)将s1,s2,…sm进行单位化,具体过程如下式

    pi=si/∑i=1msi,i=1,2,⋯,m (3)

    3)计算每个资源对象的权重,具体过程如下式

    wi=(1-hi)/(m-∑i=1mhi),i=1,2,⋯,m (4)

    4)根据下式计算资源对象o不完整的移动学习资源集中的资源属性值f

    f=∑i=1mwi×xi,i=1,2,⋯,m (5)

    式中:xi为oi资源对象相应的属性值。

    2.2 片段式移动学习资源自动整合

    根据2.1节对不完整移动学习资源的聚类填充,得到一个完整的移动学习资源集。通过小波域多尺度数据融合算法,将资源集中所有资源特征进行重构,通过小波基函数对重构后的资源集进行分解,获取资源小波系数;根据其小波系数来计算资源管理器中的资源小波方差,得到小波熵,将小波熵作为权值对小波系数进行融合,根据小波逆转换,得到整合结果。

    假设,q个资源管理器同时对多个资源X进行管理,各资源管理器的资源储备值可表示为:

    Zi=X+Vi , i=1,2,…,m (6)

    上式中:Vi 代表资源管理器中的资源参数。

    假设各资源管理器中权值与资源参数的方差分别为δi与σi2,则整合值为Ζ=∑i=1qδiΖi,且有∑i=1qδi=1,0≤δi≤1,总均方误差为:

    σ2=E[(X-Ζ)2]=∑i=1qδi2σi2 (7)

    根据多元函数求极值理论,可求出上式的最小值

    1/∑i=1q1σi2 (8)

    此时对应加权因子为

    δi=1/σi2∑i=1q1/σi2 (9)

    以式(9)获得的加权因子重构随机资源特征,具体公式为

    ΖJ=∑khn(J,k)φJ,k+∑J∑kgn(J,k)ψJ,k (10)

    式中:ZJ代表重构后的资源;φJ,k与hn(J,k)分别代表第J层尺度函数与第k个逼近系数,ψJ,k与gn(J,k)分别代表J层的小波资源函数与k个细节资源函数。

    假设,Wh,i与Wg,i分别代表资源逼近参数与资源细节参数的加权因子,则:

    σ2=σh2+σg2=E[ΖV2-∑i=1qWh,i212ηiφJ,k] (11)

    由上式可知,小波域资源整合方案的加权结果均方差不大于时域上最优加权的平均方差,通过移动学习资源整合后,资源的均方差将随尺度的分解而减小,其结论为片段式移动学习资源整合提供了理论依据。

    假设X代表管理的资源参数序列,通过帕斯瓦尔方程可知,正交小波基下的小波转换具有能量守恒性质,基于时间序列的能量可以在尺度域进行划分,通过小波基函数对资源管理器上的资源能量可分解为

    ∥X∥2=∑i=1J∥WJ∥2+∥VJ∥2 (12)

    则实测资源的方差为

    σX2=1Ν∥∥∑Ν=1Ν(X-X¯)2=(∥Wj,n∥2+∥Vj,n∥2) (13)

    由于Vj为X¯的逼近,由上式设定尺度j上的平均小波能量或小波方差进行归一化处理

    pj(E)=Ej/E=(1Ν∥Wj∥2)/E (14)

    式中:资源总能量为E=∑j=1JEj,并有∑j=1qpj(E)=1。归一化后的资源能量序列{pj(E)}称为资源能量序列的经验分布,为各尺度的小波资源能量与资源总能量的比例。结合信息熵的设定,运用小波各尺度资源能量序列的分布P=[p1(E),p2(E),…,pm(E)],取代资源参数的概率分布,这种基于资源能量分布获取的熵成为小波熵,具体过程为

    Hwe=H(P)=H(p1(E),p2(E),…,pJ(E)) (15)

    小波域移动学习资源整合算法以小波熵作为资源融合权值,对得到的系数进行整合,经过小波逆转换得到移动学习资源自动整合结果。

    3 仿真证明

    为了验证所提片段式移动学习资源自动整合方法的综合有效性,需要进行一次仿真,实验采用Win7x64操作系统,8GRAM,1T硬盘,搭载Intel Core-i5处理器,选取经典的MushRoom数据集作为数据来源。运用所提方法与文献[7]方法进行片段式移动学习资源整合,在样本数量相同情况下,对二者的资源整合时间(s)进行对比,对比结果如图1所示。

    由图1可知,两种资源整合方法所需时间随着样本数量不增加而发生改变,样本数量在550个以下时,所提方法资源整合所需时间随着样本数量的不断增加,呈现出缓慢增长趋势;样本数量在550个以上,资源整合所需时间曲线在不断的下降,总体上看,所提方法的整合所需最长时间与最短时间相差8s,平均都在15s以下,具有较高的稳定性;而文献[7]方法在样本数量为350个至600个这个区间内时,资源整合时间较短,当样本数量为600个以上时,整合时间曲线迅速的上升,最终达到了30s;对比上述数据可知,文献[7]方法在资源数量较少时,可有效整合移动学习资源,但随着数据量的不断增加,实用价值逐渐下降。

    4 结束语

    针对传统的移动学习资源自动整合方法含有过多无效数据,存在资源整合所需时间较长、整合后的无效资源过多等问题,提出了一种新的片段式移动学习资源自动整合方法。实验结果表明,与传统方法相比,所提方法能够缩短整合所需时间,减少整合过程中的出现的无效资源,从而增加整合的效果。
