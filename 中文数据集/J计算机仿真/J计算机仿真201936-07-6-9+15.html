    摘    要：

    针对当前方法存在伪码跟踪结果不准确的问题, 提出基于改进扩展卡尔曼算法的伪码跟踪优化方法。利用振幅、采样时间和码相位等条件计算输入信号, 获取到伪码信号的初始位置, 建立伪码跟踪环路模型。通过代价函数的计算来分析伪码的跟踪性能, 获取状态的预测值和预测方差, 并计算代价函数的极小值, 利用高精度的迭代方法对方差和协方差展开计算, 得到分解因式, 计算伪码的容积点, 完成对伪码的跟踪, 并求得伪码环路模型的解, 实现伪码的跟踪优化。实验结果表明, 所提方法在对伪码跟踪优化时, 具有较好的跟踪性能, 并且伪码的相位误差较小, 能够准确的完成对伪码的跟踪。

    改进扩展; 卡尔曼算法; 伪码; 跟踪优化;

    An Improved Extended Kalman Algorithm for Pseudo Code Tracking Optimization Simulation

    LU Lai DENG Wen

    Experimental teaching center, Guangdong Ocean University Cunjin College

    Abstract：

    Tracking result of current pseudo code tracking method is inaccuracy. Therefore, this research proposes an optimization tracking method for pseudo code based on the modified extended-Kalman algorithm. Input signal was calculated via amplitude, sampling time, and code phase, and initial position of pseudo code signal was acquired. Tracking loop model of the pseudo code was built. Tracking performance was analyzed via calculating cost function. The predicted value and variance of status was acquired and minimum of the cost function was calculated. The variance and covariance were calculated via iterative method with high precision to obtain factorization. Volume point of the pseudo code was calculated. Thus, the tracking was completed and solution of the tracking loop model was solved. The optimization of tracking was achieved. Simulation results show that the method has better tracking performance and lower phase error of the pseudo code. It can track the pseudo code accurately.

    Keyword：

    Modified extension; Kalman algorithm; Pseudo code; Tracking optimization;

    1 引言

    伪码跟踪是数据处理的主要部分, 在新概念雷达和新型雷达的不断发展下, 信号处理能力走向新的阶段, 数据处理的信息量变大, 对伪码的跟踪提出了较高的要求, 通常情况下, 伪码动态参数与观测数据之间的关系属于非线性关系, 针对伪码跟踪问题[1,2], 国内外研究学者对其展开大量研究。伪码跟踪属于比较典型的估计问题, 常用的算法有扩展卡尔曼算法, 粒子算法和不敏卡尔曼算法。其中扩展卡尔曼算法的滤波速度最快, 并且计算量小、统计效果好, 可以应用在雷达对多普勒的测量中, 为了进一步研究伪码的跟踪优化, 利用一种改进扩展卡尔曼算法对其展开研究。

    朱照阳[3]和高勇提出基于正交特性的伪码跟踪优化方法, 在已知伪码周期和码片速率的基础上, 将信号划分为两倍码元宽度的长度, 存在50%的重叠数据, 并对伪码的最大和次大特征向量做估计。最大和次大特征两相为相互正交, 计算伪码序列的信噪比, 并利用该特性对扩频序列和伪码序列做估计, 得到估计结果, 完成对伪码的跟踪优化, 但该方法伪码跟踪优化过程中的噪声较多, 跟踪性能较差。王雪君[4]、孙进平和张旭旺提出基于压缩感知的伪码跟踪优化方法, 建立稀疏测量模型, 利用压缩采样法获取伪码的多普勒量测值, 并利用SEKF法对其更新, 改善伪码跟踪的性能。在跟踪过程中, 引入伪量测来减小伪码状态与多普勒量测之间的误差, 完成伪码跟踪优化。该方法跟踪误差较大, 存在伪码跟踪性能差的问题。贺成艳[5]和卢晓春提出基于ASPeCT的伪码跟踪优化方法, 分析伪码跟踪环路的性能和数据, 利用接收机接收伪码的信号, 得到伪码的偏置载频调制信号, 可以解决伪码跟踪点模糊的问题, 实现伪码的跟踪优化, 但该方法的伪码相位误差较大, 不能够准确的完成对伪码的跟踪优化。

    针对上述三种方法存在的问题, 也为得到更理想的伪码跟踪效果, 提出基于改进扩展卡尔曼算法的伪码跟踪优化方法。

    2 多路径信号对伪码跟踪的影响

    一般情况下, 多路径信号按照产生原理可分成两种:第一种是镜面反射, 能够模拟一个或者几个伪码所制作的延迟信号;第二种是散射, 是接收以及延时信号之和, 属于一个附加的噪声通道, 不会对伪码产生较大的影响[6,7]。镜面反射多径信号对伪码跟踪影响的分析如下。

    M路镜面反射时, 对应的接收信号为

    r (t) ∑i=0Μaip (t-τi) cos (ω0t+φi) +n (t) (1)

    式中, ai、τi、φi分别表示第i路信号中的信号幅度、信号延时和信号的载波信息, i=0表示镜面反射多径的直达信号, p (t) 表示伪码, ω0表示镜面反射多径的载波角频率, n (t) 表示噪声。通常情况, 多路径信号的特性为:

    1) 传播路径长, 到达接收机的时间晚于直达信号的到达时间[8]。

    2) 出现能量衰减现象, 除了直达信号被遮蔽时, 其它情况下功率小于直达信号的功率。

    假设i≠0时, 有ai<a0、τi<τ0。伪码跟踪可以在延时锁定环路中实现。通过载波解调的接收信号有如下表达式

    s (t) =∑i=0Μaip (t-τi) cos (φi-φc-φe) +n´ (t) (2)

    式 (2) 中, n′ (t) 代表窄带噪声, 此时跟踪的是直达与多径信号合成得到的信号, φc、φe分别表示信号的相位和估计误差, 不受噪声的影响, 假设信号的环境为静态环境, 则φe=0, s (t) 与早码sE (t) =p (t-τ^0-d/2) 和晚码sL (t) =p (t-τ^0+d/2) 相关, 表达式如下

    RL (ε) =∑i=0ΜaiR (ε-Δτi-d/2) cos (φi-φc) (3)

    RE (ε) =∑i=0ΜaiR (ε-Δτi+d/2) cos (φi-φc) (4)

    其中, τ^0表示时延估计, ε=τ^0-τ0, Δτi=τi-τ0, d表示早码与晚码之间的时间间隔, 式 (3) 和式 (4) 相减能够得到鉴相函数

    S (ε) =RL (ε) -RE (ε) (5)

    当多径与合成信号为反相或者同相时, 伪码跟踪误差最大, 综上, 完成多路径信号对伪码跟踪的影响分析。

    3 伪码跟踪优化研究

    3.1 伪码跟踪环路模型

    利用振幅、采样时间和码相位等条件计算输入信号, 可以得到伪码信号的初始位置, 采用采样器的抽头系数和采样长度计算得到输入信号和参考信号对应的压缩采样值, 通过对伪码参考信号的分析可以得到积分清零结果, 并利用三角函数计算积分清零结果, 获取伪码的总噪声项, 最终实现伪码跟踪环路模型的建立。

    r´[k]=s´[k]+n˝[k]=2AD[kt0-η]Ρ[kt0-η]   ⋅cos[2π (f0+fd) kt0+θ0]+n˝[k]          (6)

    其中, A表示振幅, D、P分别表示数据和扩频波形, t0表示次采样时间, η表示码相位, f0、fd分别表示中频和多普勒频率, θ0表示初始位置, n″表示高斯白噪声, k表示信号, s′表示伪码, 假设压缩采样器的抽头系数为Pm[·], 并且 (Pm[·]) 2=1, 输入信号和参考信号对应的压缩采样值分别为

    rRD´[m]=∑k=0L-1 (ΡmL+k) (L+k) (7)

    URDΙΡ[m]=∑k=0L-1 (ΡmL+k) UΙΡ (L+k) (8)

    利用乘法器将其输出

    rRD´[m]URDΙΡ[m]=∑k=0L-1s´ (ΡmL+k) UΙΡ (L+k) +     n˝ (ΡmL+k) UΙΡ (L+k) +ncip˝[m]          (9)

    其中

    ncip˝[m]=∑k=0L-1 (ΡmL+k) (10)

    其中, L代表采样器的长度, 同时代表压缩比, 式 (8) 中的UIP代表参考信号, 式 (9) 中包括三项:一是累加处理高速率的采样值, 二是将输入噪声转换, 三是引入交叉噪声项。积分时间选取伪码的周期为TPN=NPNt0, NPN表示伪码一个周期中采样的点数, 积分清零器在压缩采样工作后工作, 实际的积累点数为MPN=NPN/L, 可以得到I路的积分清零结果

    ΙΡ=∑ΝΡΝΝΡΝ-1UΙΡ+∑ΝΡΝΝΡΝ-1ncip˝[m] (11)

    设数据位与伪码的周期相参, 并且伪码周期和积分清零周期处于同步状态, 积分清零周期中的数据位不变化, 当伪码相位与载波相互独立, 可以利用三角函数计算得到

    ΙΡ=2AD[Δη]ΝΡΝ⋅sinΝΡΝcosθ0+nΙΡ˝ (12)

    其中, n″IP表示总噪声项, 将伪码的总噪声项与上述计算结果相结合, 能够完成对伪码跟踪环路模型的建立, 表达式为

    W={n″IP+IP·r′RD[m]URDΙΡ[m]}r′[k] (13)

    3.2 基于改进扩展卡尔曼算法的伪码跟踪优化

    在对伪码跟踪环路模型建立的基础上, 通过对非线性动态系统的状态方程和量测方程进行计算, 得到对应的过程噪声和量测噪声。为提高伪码的跟踪性能, 计算代价函数, 得到对应状态的预测值和预测方差, 并计算该函数的极小值, 计算伪码的容积点, 对伪码进行跟踪, 并对伪码环路模型求解, 实现伪码的跟踪优化。

    设非线性动态系统的状态和量测方程分别为

    xk′=f′ (xk′-1) +wk′-1 (14)

    zk′=h (xk′) +vk′ (15)

    式中, xk′、zk′分别表示k′时刻系统的状态和测量值, wk′-1、vk′分别表示过程噪声和量测噪声, 伪码的权值和加权高斯积分的基本容积的计算公式如下

    γj=1m´ (17)

    其中, m′代表容积点总数, 为了提高对伪码的跟踪性能, 计算代价函数

    J (xk´) = (xk´-x¯k´) ΤΡ¯k´´ (xk´-x¯k´) (18)

    式 (18) 中, x¯k´表示状态的预测值, Ρ¯k´´表示预测方差, 式 (18) 的极小值与状态的最大似然值等价, 利用Gauss-Newton非线性迭代法将式 (15) 线性化, 求得式 (18) 的极小值, 表达式为

    x^k´i´+1= (x¯k´+Ρ¯k´´) Τ⋅ (zk´-xk´) (19)

    量测方程存在的传递误差较大, 降低了伪码的跟踪精度, 需要利用高精度的迭代方法对方差和协方差展开计算, 计算公式如下

    Zj, k´ (i´) =h (Xj, k´ (i´) ) , j=1, 2, ..., m′ (20)

    z^k´ (i´) =∑j=1m´γjΖj, k´ (i´) (21)

    Ρ´zz, k´ (i´) =∑j=1m´γjΖj, k´ (i´) (Ζj, k´ (i´) ) Τ-z^k´ (i´) (z^k´ (i´) ) Τ (22)

    Ρ´xz, k´ (i´) =∑j=1m´γjXj, k´ (i´) (Ζj, k´ (i´) ) Τ- (z^k´ (i´) ) Τ (23)

    其中, Xj, k´ (i´) 代表i次迭代的容积点, Zj, k´ (i´) 代表i次迭代的传播容积点, z^k´ (i´) 代表量测的预测值, 将x¯k´和Ρ¯k´´当作初始值的过程和测量更新过程。假设第i′次迭代对应的方差为P′k´ (i´) , 估计值为x^k´ (i´) , 得到的分解因式为

    S^k´ (i´) =chol (P′k´ (i´) ) (24)

    计算伪码的容积点

    Xj, k´ (i´) =S^k´ (i´) ζ+x^k´ (i´) (25)

    增益的计算公式为

    Kk´ (i´) =P′xz, k´ (i´) (P′ (i′) zz, k′) -1 (26)

    迭代的终止条件为

    ‖xk´ (i´+1) -xk′‖≤ε (27)

    其中, ε表示误差的极小值, 通过对容积点的分析与计算, 完成对伪码的跟踪, 可以求得伪码跟踪环路模型的解, 最终实现伪码的跟踪优化, 表达式为

    ϑ=W⋅{Κk´ (i´) +S^k´ (i´) +Xj, k´ (i´) }/z^k´ (i´) ⋅γj (28)

    式 (28) 中, ϑ代表伪码跟踪环路模型的解, 通过对模型的求解来完成伪码跟踪优化。

    4 仿真设计与结果分析

    实验平台为MATLAB6.5, 其操作系统为WindowsXP, CPU的主频为2.5GHz。利用基于改进扩展卡尔曼算法的伪码跟踪优化方法、基于正交特性的伪码跟踪优化方法和基于压缩感知的伪码跟踪优化方法测试伪码的跟踪性能, 伪码跟踪过程中会存在很多噪声, 噪声越多并且分布越不均匀时, 伪码的跟踪性能越差, 测试结果如图2所示。

    5 结论

    为了研究伪码的跟踪优化, 提出基于改进扩展卡尔曼算法的伪码跟踪优化方法, 分别利用跟踪噪声和伪码相位误差测试伪码的跟踪性能和伪码跟踪优化结果的准确性, 实验结果证明了提出方法在对伪码跟踪优化时, 具有较好的跟踪性能, 并且跟踪优化结果的准确性较好, 能够准确的完成对伪码的跟踪优化, 说明提出方法具有较好的跟踪效果, 提出方法也为伪码跟踪优化的进一步研究奠定基础, 并且具有一定的参考价值。
