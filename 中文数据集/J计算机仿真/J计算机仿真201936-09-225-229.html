    摘    要：

    针对微小型物体的三维重建,采用线结构光点云数据三维重建的方法,首先采用线结构光三维扫描技术来获取物体的原始三维点云数据,并使用八叉树来存储物体的原始点云数据,然后通过对原始点云数据进行配准、去噪以及简化等处理得到高质量的物体三维点云数据,最后在高质量三维点云数据的基础之上采用三角剖分对物体进行网格的生成与网格优化,最终得到高精度的物体三维几何模型。实验结果表明,线结构光点云数据三维重建的方法与其它方法相比能够快速准确的重建出硬盘外壳的三维几何模型,并且误差小于其它重建方法。

    Three-Dimensional Reconstruction of Micro-Small Objects Based on Line Structure Point Cloud Data

    TANG Rui-yin QIAO Shi-guang HE Hong-kun

    College of Electrical Engineering, North China University of Science and Technology

    Abstract：

    For the three-dimensional reconstruction of microscopic objects, a three-dimensional reconstruction method based on line structure point cloud data was adopted in the paper. First, the three-dimensional line light structure was used to acquire the original three-dimensional point cloud data of an object, and an octree was used to store the cloud data the original point of object. Then, high-quality object 3D point cloud data were obtained by registering, denoising, and simplifying the original point cloud data. Finally, triangulation was used to mesh the object on the basis of high-quality 3D point cloud data. The grid generation and grid optimization finally result in a high-precision 3D geometric model of the object. The experimental results show that the three-dimensional reconstruction method based on line structure point cloud data can reconstruct the three-dimensional geometric model of the hard disk enclosure quickly and accurately compared with other methods, and the error is less than 0.05 mm.

    1 引言

    近年来,随着新的加工制造方法和工艺的出现,制造和加工业在原有大型尺寸和常规尺寸工业产品的基础之上,涌现出了大量以生产和加工大批次的微小型产品为主的高新技术企业[1]。随着三维点云技术快速的发展,在这些企业中存在着大量对微小型物体三维表面重建的需求。

    Kazhdan[2]使用快速傅里叶变换高效地重建了有向点云,由于需要对点云的周围进行均匀细分,因此较高的分辨率会导致内存要求过高。Baron D[3]等人通过空间Poisson公式重建出了有向点云,虽然它补偿了因细节产生的过高内存需求,但是重建的效率却大大地降低了。钱归平[4]等人使用Splatting曲面来筛选采样选择样本逼近物体点云,采用非Delaunay三角剖分的方法实现点云重建。这些算法基于对象特征点的实际三维坐标,或者使用投影来处理特征点,需要匹配大量特征点,计算操作等复杂并且整体精度也大大降低。

    本文采用线结构光点云数据三维重建的方法,具有量程大、速度快、非接触、效率高、操作简单以及精度更高等优点,与文中其它方法相比,避免了大量特征点的匹配和大量的复杂计算等问题,无论是在运算速度上还是测量的精度上都有大幅度的提高,也简化了实际的操作。针对微小型物体的三维重建,将三角激光测量技术与ICP算法Delaunay三角剖分技术相结合最后得到高精度的物体三维几何模型。在有关微小型物体的测量与重建领域中有着广阔的应用前景。

    2 三维重建原理

    算法流程主要分为三步,分别是点云获取、点云处理和网格优化。首先建立三角激光测量模型,利用线结构光三维扫描技术来获得物体的三维点云数据;随后对物体的三维点云数据进行后续的处理,包括点云配准、点云去噪、点云简化以及点云封装。点云的配准采用ICP(iterative closest point)算法,点云去噪采用高斯滤波的方法,点云简化采用统一采样的方法,点云封装则采用贪婪式Delaunay三角化方法。网格优化采用减少相邻三角形方法来完成。算法流程如图1所示。

    2.1 点云数据的获取

    物体的三维点云数据是通过线结构光三维扫描技术获取的。线结构光三维扫描技术的原理是三角激光测量原理[5]。在三角激光的测量中,激光器发出的激光投射在被测物体的表面上,由相机获取打在物体表面的光条图像,当被测物体在光平面下移动时,物体上的光条会随着物体的移动而发生相应的形变,在相机上所成的像也会发生相应的变化。通过解调光条的形变,提取光条中心并计算相应的解析几何关系,最后获取被测物体的三维点云数据。

    三角激光是一种非接触式视觉检测方法,具有精度高、测量简单等优点,通过对物体上的光条进行测量来替代传统的直接测量。运用此方法需要对系统做精确标定和精确的光条中心提取,系统标定包括摄像机位姿的标定,光平面位姿的确定和移动位姿的标定;光条中心提取采用传统的灰度重心法,提取的精度能达到亚像素级别。这主要是为了确保被测物体的实际坐标对应于计算机的像素坐标,随后对图像进行信息的采集,把提取的光条中心坐标当做是物体实际坐标,最后获得被测物体的三维点云数据。八叉树用于存储数据结构以进行邻居点的快速查找。

    2.2 点云数据处理

    2.2.1 点云配准

    ICP算法为最常用的精确点云配准方法,过程分为两个步骤,第一步是确立对应点集,第二步根据对应点集确立坐标变换矩阵。然后重复以上两步骤,直到误差函数满足要求的精度为止。

    首先确立对应点集P与Q要遵循一定的准则,对应点对的个数记为n,最后用最小二乘法迭代计算最优的旋转矩阵R和平移矢量t来使以下目标方程的值为最小[6]:

    E(R,t)=1n||Qk-(RΡk+t)||2 (1)

    由于物体点云密集度较高,肉眼会很难观测到每个点的位置是否准确,因此在物体原始三维点云阶段是难以观察点云是否配准,但是在封装的时候是会出现明显的区别。没有经过配准的点云封装后模型会出现一些孔洞,更严重甚至会导致物体信息的缺失,点云配准之后的模型会更加的平滑,孔洞也相对较少,并会较好的还原物体的信息。

    2.2.2 点云去噪

    在对物体进行三角激光测量获取点云时,由于物体表面粗糙程度和材质的差异,采集数据环境中的照明条件以及测量设备的差异,由于测量人员操作水平的不同,不可避免的在整个测量过程中会产生不同程度的误差,出现一些偏离的点,即噪声点。噪声点的存在会影响后期封装的精确度,使得重建后的物体模型与真实物体相差较大,因此对原始数据进行去噪处理是很有必要的。为了更真实地还原物体的三维信息,需要对物体的三维点云数据进行去噪处理,在这里采用高斯滤波对数据进行去噪处理。

    高斯滤波属于线性平滑去噪滤波,具有简单而快速的优点。由于绝大多数的噪声是可以看似高斯分布的,因此使用低通高斯滤波去除噪声就成为既高效又实用的方法。高斯滤波去除噪声首先要选择窗口的固定大小,然后确定窗口内任意像素点与中心像素点的间距,权重的分配最终是通过高斯函数来完成的,高斯滤波能够写成如下形式[7]

    Ιgf(x,y)=∑(i,j)∈Wx,yωd(i,j)Ι(i,j)∑(i,j)∈Wx,yωd(i,j) (2)

    其中,Wx,y表示中心像素(x,y)的M×M(M奇数)大小的邻域;ωd为空间距离相似度权重因子。

    2.2.3 点云简化

    原始点云数据包含大量的点云,虽然可以直接进行三角化,但是直接处理时会占用大量的计算机资源,会大大的降低运算速度。因此需要对点云进行简化处理,这样做的主要优点是在进行三角化的时候运算速度更快,可视化的程度更好。处理时会得到更平滑的点云,进而会得到更加平滑的三角形网格,会得到更好的点云封装模型效果。由于统一采样的简化方法能够使平坦曲面上点数目的减少量一致,是以密度规定来减少曲面上的点数目,所以采用统一采样的方法来对点云数据实现点云简化。

    2.2.4 点云封装

    由于三角网格能够表示任意形状和拓扑结构的几何体,并且有利于计算机存储、计算和分析等,三维点云数据需转换为网格模型方可应用。采用Delaunay三角剖分方法,对点云数据进行三角连接,该方法常用于逆向工程中。三角剖分是三维重建中极其关键的部分,后续物体重建的精度直接取决于生成网格质量的好坏。

    假设在平面上的散乱点集为P={P1,P2,……Pn}三角剖分就是以P为顶点在平面上的最大划分,也就是说,用不同长度直线段将平面上散乱点连接起来,在该平面上形成紧邻三角形集,这些三角形集既不重叠又无间隙[8]。三角剖分简单来说就是把散落的点集连成由三角形组成的物体表面。

    2.3 网格优化

    直接将点云网格化后存在着网格锯齿、孔洞缺失等问题,封装后会出现一些沉余和一些不规则锯齿等错误网格,所以需要对网格进行优化。

    三角剖分的优化准则都是基于一个出发点,为了避免过尖三角网格的出现。根据逼近理论分析,网格的逼近误差是由三角网格最小内角决定的,避免出现过尖三角网格就可提高逼近精度[9]。具有代表性的优化准则有圆准则、Thiessen准则、PLC准则和ABN准则。本文选用的是圆准则算法,圆标准是指严格凸四边形的三个顶点构成一个圆,假如第4个顶点在圆的内部,那么就把这个点连接到其相对的顶点;否则,将另两个顶点相连。该准则简单明了,有利于用程序实现减少相邻三角网格的数量来达到对网格的优化。
