    摘    要：

    针对当前方法进行智能楼宇网络中冗余信息并行采集时, 普遍存在着采集时间代价较大、错误率较高、成本耗费较大等问题。提出基于云计算的网络中冗余信息并行采集方法。为获取智能楼宇网络中的冗余信息, 对整个楼宇网络冗余信息集合进行划分, 获得若干个相同几何大小的子空间格;利用K近邻算法对各子空间格内的楼宇网络冗余信息记录进行搜索, 得到智能楼宇网络冗余信息的部分拓扑数据, 完成冗余信息的特征提取, 构建网络冗余信息处理模型, 获取样本信息特征矢量, 将冗余信息特征和正常信息特征进行比较, 引入云计算方法对网络中冗余信息进行采集。实验结果表明, 所提出方法在智能楼宇网络中冗余信息并行采集时, 能够有效提高采集过程中时间代价、降低错误率, 减少一定的成本耗费。

    Simulation of Redundant Information Parallel Acquisition Method in Intelligent Building Network

    ZHENG Heng-he

    Jining University

    Abstract：

    When the current method was used to collect redundant information in intelligent building network, the collection time cost and error rate were high. Therefore, a method to parallel collect redundant information based on cloud computing was proposed. In order to obtain redundant information in intelligent building network, the set of building network redundancy information was divided to get several Subspace lattices with the same geometrical size. K-nearest neighbor algorithm was used to search record the redundant information record of building network in each subspace lattice, so as to obtain partial topology data of intelligent building network redundancy information. Thus, the feature extraction of redundancy information was completed. Moreover, network redundant information processing model was constructed to obtain sample information feature vectors. Finally, redundant information features were compared with normal information features, and the cloud computing method was introduced to collect the redundant information in network. Simulation results show that the proposed method can effectively improve the time cost, reduce the error rate and the cost during parallel collection of redundant information in intelligent building network.

    1 引言

    计算机技术和经济飞速发展的同时, 智能楼宇网络中数据传输速度也随之加快, 当前网络内容采集需求正在增大[1]。在智能楼宇网络中大量相同信息被用户反复请求, 造成网络中大量冗余信息出现, 因此需要对冗余信息进行采集, 减少不必要的网络信息传输[2,3]。但现阶段智能楼宇网络冗余信息并行采集时, 存在着采集时间代价较大、错误率较高、成本耗费较大等问题, 在这种情况下, 如何提高采集准确率成为当今社会亟待解决的问题[4,5]。

    目前, 杨浩、王喜伟[6]提出一种基于压缩感知的无线网络数据采集方法。该方法利用拓扑模型将网络数据随机划分多个区域, 在各个区域中选择出一个节点来接受其它区域传递出的节点采样数据, 将节点采样数据传递到Sink节点上进行数据重构, 根据重构数据测量值完成数据采样。该方法的数据采集性能优于各种混合采集方法, 但是在采集过程付出时间代价较大。蔡文郁、张美燕[7]提出一种基于稀疏水下传感器网络中AUV数据采集方法。该方法将AUV数据覆盖区域内的节点作为临时Sink节点, 利用最小生成树方法将网络数据传输到临时Sink节点上, 在通过这些节点将数据信息传递给AUV, 根据AUV的移动轨迹利用水下传感器将数据进行采集。该方法有效的保证了数据采集效率, 但是在采集过程中成本消耗过大。陈静[8]提出一种基于联合智能优化分类CS的稀疏数据采集方法。该方法构建稀疏数据网络通信模型, 根据网络数据总跳数, 对网络稀疏数据进行分类, 以分类结果完成数据采集[9]。该方法有效的提高了网络数据采集时间代价, 但是在采集过程中容易出现偏差, 导致采集错误率较高。

    针对上述问题, 提出一种基于云计算的网络中冗余信息采集方法。仿真证明, 所提出方法能够有效提高采集过程中时间代价、降低错误率, 减少一定的成本耗费。

    3 智能楼宇网络中冗余信息并行采集方法

    3.1 冗余信息特征提取

    为获取智能楼宇网络中的冗余信息, 首先对整个楼宇网络冗余信息集合进行划分, 获得若干同样几何大小的子空间格;利用K近邻算法对各子空间格内的楼宇网络冗余信息记录进行搜索, 得到智能楼宇网络冗余信息的部分拓扑数据;通过获得的拓扑数据, 进一步完成冗余信息的特征提取。具体过程如下。

    利用协方差分析法对智能楼宇网络冗余信息点pi的K个近邻点集合P进行分析, 获得的近邻点协方差矩阵Zi为

    Ζi=[pi1-p¯i ⋯piΚ-p¯i]Τ[pi1-p¯i ⋯piΚ-p¯i]    (13)

    式中, 该矩阵为三阶对称矩阵;i代表冗余信息点编号, i代表智能楼宇网络冗余信息集合的中心, 计算式为

    p¯i=1Κ∑j=1Κpij    (14)

    式中, j代表冗余信息点的近邻冗余信息点编号, pij代表第i个冗余信息点的第j个近邻点网络冗余信息。

    网络冗余信息点的邻近区域弯曲度可以近似描述该点的特征凸显程度, 且计算相对简单。冗余信息点附近区域的弯曲度大, 说明该区域信息的特征明显, 反之, 则说明该区域信息的特征不明显。假设某一信息点pi的特征值所对应的单位特征向量为e1, 则各冗余信息点的邻近区域弯曲度k (i) 的计算式如下

    k (p¯i) =1Κ∑j=1Κfabs (〈 (pij-pi) , e1〉) |pij-pi|    (15)

    式中, 〈 , 〉代表向量间的点积符号, ‖‖表示向量模长计算符, fabs代表绝对值计算函数。根据上式的计算结果, 选择弯曲度值较大的信息点作为特征点, 并依据对应的特征, 完成智能楼宇网络信息特征提取, 具体步骤为:

    1) 结合K近邻搜索, 计算各点的局部区域弯曲度值k (pi) , 根据计算结果, 将各信息点分别划分为四个冗余信息点集合, 分别记作S1、S2、S3和S4。

    2) 将智能楼宇网络信息点的局域弯曲度值大于设置参照阈值γ的点标记为“留存”, 剩余的标记为“待定”。阈值γ的计算式为:

    γ=σΝ∑i=1Νk (pi)     (16)

    式中, N表示网络信息点的数量, σ为调节算子。

    遍历所有网络信息点集合并进行精简, 对精简后的信息点, 仅筛选出标记为“留存”状态的信息点, 得到冗余信息集合S1、S2、S3和S4的特征信息点, 根据各信息点间的拓扑结构, 实现冗余信息特征提取。

    5 结论

    为获取智能楼宇网络中的冗余信息, 楼宇网络冗余信息集合进行合理划分, 利用K近邻算法对各子空间格内的楼宇网络冗余信息记录进行搜索, 得到智能楼宇网络冗余信息的部分拓扑数据, 对冗余信息的特征进行提取, 构建网络冗余信息处理模型, 获取样本信息特征矢量, 将冗余信息特征和正常信息特征进行比较, 采用ADASYN算法对冗余信息进行删除, 引入云计算方法对网络中冗余信息进行采集。实验结果表明, 所提出方法在智能楼宇网络中冗余信息并行采集时, 能够有效提高采集过程中时间代价、降低错误率, 降低成本耗费。
