    摘    要：

    当前网络恶意入侵数据识别方法误检率高、识别效率低,提出基于蚁群的恶意入侵数据识别方法。选取数据集一级分类属性集合,得到第一次数据分类依据值,给出数据分类依据集合。将网络总数据集合划分成若干个不重叠的子数据集合,将第一次数据分类集合中的部分数据当作操作数据进行二级分类。随机从总数据集合划分结果中选取一个子集合,对子集合中数据相似度进行计算,设置一个固定大小的窗口,对升序排列后的子数据集在窗口中实施分割截取操作。对比截取结果中某两段相邻数据之间的相似度差值,将差值与定义阈值进行比较,实现网络冗余去除和异常数据初步识别。根据去冗和异常初步识别,计算蚂蚁状态转移概率。利用转移概率计算结果获取蚁群适应度函数,并更新蚁群寻优路径上的信息浓度。判断是否找到信息浓度最高的位置或是否达到最大迭代次数,将网络恶意入侵数据多状态识别结果输出。实验结果表明,上述方法误检率低、识别效率高,具有科学性。

    关键词：

    交互式网络; 恶意入侵; 数据; 识别;

    Multi-State Recognition Simulation of Interactive Network Malicious Intrusion Data

    ZHANG Yong

    Technical College for the Deaf, Tianjin University of Technology

    Abstract：

    This paper proposes a method to recognize malicious intrusion data based on ant colony. Firstly, we chose the first-level classification attribute set of data set to get the value of first data classification basis and then gave the set of data classification basis. Moreover, we divide the network total data set into several nonoverlapping sub-data sets. Then we used some data in the first data classification set as operation data for secondary classification. Randomly, we selected sub-set from the total data set partitioning result and calculated the data similarity in subset. In addition, we set a fixed window to perform segmentation and interception on the sub data set after the ascending sort in window. Compared with the similarity difference between two adjacent data in the interception result, the difference was compared with the defined threshold to achieve network redundancy removal and primary recognition of abnormal data. According to redundancy removal and abnormal initial identification, we calculated the ant state transition probability. After that, we used the calculation result of transition probability to obtain ant colony fitness function. Meanwhile, the information concentration on the ant colony optimization path was updated. Finally, we judged whether the position with maximum information concentration was found or whether the maximum number of iterations was reached, and then we output the multi-state recognition result of network malicious intrusion data. Simulation results show that the method has low false detection rate and high recognition efficiency.

    Keyword：

    Interactive network; Malicious invasion; Data; Recognition;

    1 引言

    交互式网络的不断发展,网络安全问题逐渐出现在大众眼前。网络恶意入侵数据多状态识别检测依据主动防御方案,对网络交互中存在恶意入侵与否进行检测,其为网络安全研究中的关键技术手段之一[1,2]。但交互式网络中的数据不断增加和网络速度的逐渐提升,使得恶意入侵数据的识别检测耗时和检测率要求也随之提升。当前网络入侵数据识别检测方法中基于支持向量机的交互式网络恶意入侵数据多状态识别方法,采集高维数据样本,并将支持向量机在该样本中推广,以此构建RBF核函数,同时根据核函数实现有监督C-SVM与无监督的One-Class SVM的建立对网络恶意入侵数据进行识别。该方法检测效率和检测率均较低。网络恶意入侵数据识别检测研究的不断深入,引起了广大学者的关注和钻研[3]。

    张福勇[4]等人提出基于属性相似度的网络恶意代码检测方法。过程中,将网络数据样本转化为十六进制模式,将样本中的全部n-gram提取出来,并对各n-gram信息增益进行计算,选取具有最大增益的若干个n-gram当作特征属性,对其恶意代码与正常文件每维属性平均值进行计算。利用对待检测样本属性和恶意代码与正常数据类别属性的均值进行比较,判断待检测数据样本的类别。实验结果表明,该方法检测准确率较高,但耗时较长。韩红光[5]等人提出基于马尔科夫链的网络入侵检测方法。通过马尔可夫链实现网络事件概率建模,并将建模结果应用至网络入侵检测。利用聚类法对网络状态进行定义,依据状态定义结果,对概率状态转移与初始概率分布的HMM进行构建,采用模型对网络异常数据进行检测。实验结果表明,该方法漏检率较低,但存在检测效率低的问题。杨忠明[6]等人提出基于目标预判的网络恶意入侵数据检测算法,对采集到的网络数据进行分析检测,并将潜在恶意数据视为高危数据,剩余数据视为低危数据,高频检测高危数据,低频检测低危数据,以此提升异常数据检测效率。实验结果表明,该算法具有一定的检测效率,但存在误检率较高。

    网络安全问题不容小觑,鉴于当前相关研究方法和算法中存在的问题,提出基于蚁群的交互式网络恶意入侵数据多状态识别方法。

    2 交互式网络恶意入侵数据多状态识别原理

    在交互式网络恶意入侵数据多状态识别原理中,利用互信息可信度对网络入侵进行检测。过程中,对处理后的样本进行训练,计算各信息样本属性聚类时的相对可信程度,并对其进行排序,将排列在靠前的特征属性构建成新集合。利用统计方式对某类别于样本集合中出现概率进行计算,结合新集合中特征属性值的条件概率和互信息下相对可信度,对网络恶意入侵数据多状态进行识别。详细过程如下:

    假设现已将网络样本信息数据进行了离散化处理,对样本进行训练。通过式(1)对各信息样本属性Ai聚类相对可信程度Ri进行计算:

    R=ΜΙ1-ΜΙ2ΜΙ2 (1)

    其中

    ΜΙ=ΜΙ(A,cj)=∑i=1p(ai,cj)logp(ai,cj)p(ai)p(cj) (2)

    式(1)和式(2)中,MI1代表样本属性A和各数据类别之间互信息最大值,MI2代表样本属性A和各数据类别之间互信息次大值。MI1-MI2值越大,则表示样本属性A对交互式网络恶意入侵数据多状态识别作出的贡献就越大。(A,cj)表示以样本属性A为特征向量的类别cj,p(ai,cj)代表数据样本归属于一个类别的概率,p(ai)代表数据样本子元素为入侵数据识别作出的贡献概率,p(cj)代表某数据类别为入侵数据识别作出的贡献概率。

    综合式(1)和式(2)的计算,对Ri按照从大至小的顺序进行排列,并将前K个属性特征当作最佳属性,以此获取特征选取之后的新属性集合B,其表达式为

    B=(B1,B2,…,BK) (3)

    根据式(3)的计算,对样本于样本集合出现概率进行计算

    p(ci)=BiB (4)

    其中,p(ci)代表类别是ci的样本于样本集合出现概率,Bi代表样本中类别为ci的数量。

    利用式(4)计算结果,对式(3)属性集合中BK条件概率进行计算

    p(BΚ)=∏i=1Κp(xj|ci) (5)

    式(5)中,p(BK)代表属性集合中BK条件概率值,p(xj|ci)代表类别是ci训练样本集合中属性BK与xj相等的概率。

    将互信息下相对可信度R当作权值,与式(5)计算结果相融合,对交互式网络数据进行分类,完成网络恶意入侵数据多状态识别。结果可表示为式(6)

    p(Ci|X)=argmax∏i=1ΚRp(xj|ci) (6)

    式(6)中,p(Ci|X)代表交互式网络恶意入侵数据集合X在总样本集合Ci中出现的概率。

    在基于互信息可信度的交互式网络恶意入侵数据多状态识别方法中,比较少地考虑样本中冗余成分,导致该方法运行时的效率较低。为此,提出基于蚁群的交互式网络恶意入侵数据多状态识别方法。

    3 交互式网络恶意入侵数据多状态识别方法

    3.1 交互式网络数据去重和初步异常识别

    为保障交互式网络恶意数据检测率,提高异常数据识别效率,利用数据属性权重对交互式网络中的冗余数据进行去除,并实现初次异常识别[7,8]。定义交互式网络总数据集合,选取数据集一级分类属性集合,并对其相应权值进行计算,得到第一次数据分类依据值,以此给出数据分类依据集合。将交互式网络总数据集合划分成若干个不重叠的子数据集合,同时将第一次数据分类集合中的部分数据当作操作数据,进行二级分类。随机从总数据集合划分结果中选取一个子集合,对子集合中数据相似度进行计算,设置一个固定大小的窗口,对升序排列后的子数据集在窗口中实施分割截取操作。对比截取结果中某两段相邻数据之间的相似度差值,将差值与定义阈值进行比较,实现网络冗余去除和异常数据初步识别。

    依据交互式网络数据集属性实现一级分类:选择数据集一级分类属性集合C′={C′1,C′2,…,C′d},通过该集合获取的数据对应权值可表示为

    ∂={∂1,∂2,…,∂d} (7)

    根据式(7)能够获取第一次数据分类依据值

    X′=C′1∂1+C′2∂2+…+C′d∂d (8)

    通过式(8)构建与数据集相对应的数据分类依据集合

    X′={X′1,X′2,…,X′n} (9)

    将交互式网络总数据集合L划分为不重叠的m个子数据集L1,L2,…,Lm。

    将一级分类对象中部分数据当作操作数据,以上一级分类提到的属性权重计算方式,对相应的属性权值大小进行判断。以L1为例,进行二级分类,确定L1中数据集相似程度M′,其中,相似度计算属性可表示为d={d1,d2,…,dn},属性对应权值可表示为β={β1,β2,…,βn},那么M′可表示为

    M′=d1β1+d2β2+…+dnβn (10)

    基于相似度值M′,通过快速排序法对子数据集合L1实施升序排列操作,由此使相似度高的数据尽量排列在一起。

    假设有一个固定大小窗口,该窗口的宽度为w′,利用该窗口实现已经排好序的L1集合截取,对截取结果中某两段相邻数据之间的相似度差值进行计算,将计算结果定义为z′。设置一个阈值Z′,假设z′<Z′,则将比较中的前者数据删除,实现冗余去除。在两重分类的过程中,实现了交互式网络恶意入侵数据多状态初步识别。

    3.2 基于蚁群的网络恶意入侵数据多状态识别

    依据3.1节去冗和初步异常识别,利用蚁群算法进一步识别交互式网络中恶意入侵数据。过程中,将网络去冗和初步异常识别代入恶意入侵数据多状态识别中,计算蚂蚁状态转移概率。利用转移概率计算结果获取蚁群适应度函数,并更新蚁群寻优路径上的信息浓度。判断是否找到信息浓度最高的位置或是否达到最大迭代次数,将网络恶意入侵数据多状态识别结果输出。详细过程如下:

    将3.1节去冗和初步识别结果定义为G(x)。交互式网络节点是所有蚂蚁均需经过的节点,每一次循环,各蚂蚁将遍历所有节点数据特征。各节点数据特征自身有一个选择概率,特征被选择的概率越大,则表示其为恶意入侵数据的概率也就越大[9,10]。蚂蚁由特征i′转移至j′的概率pi´j´k´(t′)可表示为

    式(11)中,γi′j′代表启发因子,其可有效提升交互式网络恶意入侵数据识别率,γi′j′值越大,表示蚂蚁转移至特征j′概率也就越大。εi′j′(t′)代表t′时刻由特征i′转移至j′的信息素,tabuk′代表蚂蚁k′的禁忌表。

    在式(11)中,ξ代表信息素权重,ϑ代表启发因子权重。ξ是一个常数,ϑ利用式(12)计算

    ϑ=ϑ0(1-Ν´Νmax´) (12)

    式(12)中,N′代表迭代数量,N′max代表最大迭代数量,ϑ0代表启发因子初始权值。

    根据上述计算和分析,对蚁群适应度函数Fopt进行计算

    Fopt=ξ⋅νr(D)+ϑL-|r|L⋅pi´j´k´(t´) (13)

    式(13)中,νr(D)代表相对D维的数据条件属性集合r聚类质量,|r|代表选择特征子集长度。

    蚁群算法主要利用信息实现反馈机制,可将信息更新机制分为两种形式:局部信息与整体信息。其中,整体信息能够加速蚂蚁搜索速度,得到全局最佳特征越大,为此,每实现一轮搜索,每条路径的信息浓度均要进行更新。根据式(13)的计算,更新表达式可表示为

    εi′j′(N′+1)=Fopt[ρ·εi′j′(N′+1)+γi′j′(t′)] (14)

    式(14)中,ρ代表信息素的残留因子。

    迭代上述过程,判断是否找到信息浓度最高的位置或是否达到最大迭代次数,如果为是,则将信息浓度最高的位置输出,即将交互式网络恶意入侵数据多状态识别结果输出。式(15)为网络恶意入侵数据多状态识别结果输出表达式

    L(X)=Fopt·Hεi′j′·pi´j´k´(t′) (15)
