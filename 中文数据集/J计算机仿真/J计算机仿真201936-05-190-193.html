    摘    要：

    针对传统倾斜车牌图像字符识别率较低、识别精度不高的问题, 提出一种基于小波分析的倾斜车牌图像字符识别方法。首先对原始倾斜车牌图像转换成灰度图像预处理, 获得图像像素点, 利用小波矩对预处理后的车牌图像像素点提取字符特征, 运用直线拟合方程矫正字符倾斜角度目标图像, 然后通过主分量分析对提取的字符特征进行选择, 并降低字符特征维数, 获得特征向量, 并输入到BP神经网络, 通过对该网络权值初始化, 分析输入层与输出层之间的线性映射, 并通过线性映射关系完成字符识别。实验结果表明, 所提方法能有效提高倾斜车牌图像字符识别率。

    关键词：

    字符识别; 小波分析; 倾斜车牌; 神经网络;

    Character Recognition Simulation of Tilting License Plate Image Based on Wavelet Analysis

    HE Yu-fei

    School of mathematics and statistics, Yulin University

    Abstract：

    A method to recognize characters in slanted license plate image based on wavelet analysis was proposed. First of all, this method converted the original slanted license plate image into gray image for pretreatment, and then the pixel points of image were obtained. After that, our method used wavelet moment to extract the character features from the pixels in license plate image after the pretreatment. Moreover, this method used straight-line fitting equation to correct the target image of inclination angle of character. Meanwhile, we selected the extracted character features by principal component analysis and reduced the character feature dimension to get feature vector which was inputted to BP neural network. By initializing this network weight, we analyzed the linear mapping between the input layer and the output layer. Thus, we completed the character recognition through the linear mapping relationship. Simulation results show that the proposed method can effectively improve the recognition rate for characters in slanted license plate image.

    Keyword：

    Character recognition; Wavelet analysis; Slanted license plate; Neural network;

    1 引言

    倾斜车牌识别技术是现代智能交通系统中的识别技术类的重要核心, 它的处理过程包括倾斜车牌图像采集、图像预处理、倾斜校正、字符提取和字符识别。有时会受到采集设备与车牌之间角度的影响, 导致采集的车牌图像总会存在倾斜问题, 车牌图像倾斜状况大体分为水平倾斜、垂直倾斜以及水平垂直倾斜。不同倾斜的车牌字符识别方法也不同。传统意义上对倾斜车牌图像字符识别主要是通过计算模板和未检测图像的相关内容, 来对其进行识别, 然而, 实际应用中大多数图像字符识别时提取的图像字符特征向量过多, 这种情况的存在极大的降低了图像字符识别的精确性。

    当前阶段, 如何提高倾斜车牌图像字符识别率已经成为首要解决问题。因此倾斜车牌图像字符识别研究成为学者们关注的焦点课题, 受到了广泛的关注。文献[1]提出一种基于卷积神经网络的车牌字符识别方法。通过对图像进行预先处理, 去除背景干扰, 获得准确的字符结构;采用卷积神经网络对预处理后的车牌图像字符数据集进行训练、识别。该方法较为简单, 但是存在神经网络的输入节点数量较多, 对倾斜车牌图像字符识别率较低的问题。文献[2]提出一种基于栈式降噪自编码神经网络的倾斜车牌图像字符识别方法。利用降噪自编码模型提取倾斜车牌字符图像相关特征, 利用贪婪逐层预训练和有监督微调的方式训练神经网络算法, 提高字符识别精度。该方法字符识别精度较高, 但存在相似字符特征相近, 识别用时较长, 识别效率较低的问题。文献[3]提出一种基于提升小波的矩不变量倾斜车辆图像字符识别方法。利用小波变换方式准确提取车辆图像字符的结构特征, 采用不变矩描述车辆图像字符, 获得可以描述车牌图像字符结构和统计特性的联合特征向量, 车牌图像字符分解利用第二代提升小波算法, 最终完成倾斜车牌图像字符识别, 但该方法识别准确率较低。

    针对上述问题, 提出一种基于小波分析的倾斜车牌图像字符识别方法。实验仿真证明, 所提方法有效提高了倾斜车牌图像字符识别率。

    2 基于小波分析的倾斜车牌图像字符识别方法

    2.1 倾斜车牌图像预处理

    由于识别倾斜车牌图像字符会受到一定外在因素的干扰, 无法直接对其进行使用, 需要先对采集的倾斜车牌字符图像进行预处理, 具体过程如下:

    由于彩色图像受到分辨率的影响, 在对其进行识别的过程中, 会受到外部因素的影响, 识别误差较高, 因此需要对原始图像进行预处理, 经过预处理后的原始图像, 其各方面的性能指标数值均有明显的提高, 在对其字符进行提取的过程中, 能够有效减少计算量提高识别效率。

    为了达到倾斜车牌图像字符清晰度增强的效果, 需要通过非线性指数进行变换, 可以大幅度提高图像的清晰度, 在此基础上, 对处理过的图像进行非线性灰度变换[3], 具体过程如下所示:

    在研究倾斜车牌字符图像滤波的过程中采用平滑的处理办法时, 该过程用到了低通滤波器, 可以消除图像噪音的干扰。避免了噪音的干扰。在时域中的图像中计算像素颜色的平均值, 并且平均值被认为是图像该点的像素值的平均表达式, 即:

    g (x, y) =∑ (x, y) n∈sf (x, y) (2)

    x表示图像像素色彩因子, y表示图像像素灰度因子n表示时域内无限颜色像素值, g (x, y) 表示x和y图像颜色像素值的平均值, f (x, y) 表示二维连续函数。其中像素点数目的图像字符反相操作。研究人员最为感兴趣的是研究目标字符区域的像素问题[5]。为此, 通过反转操作, 突出显示图像目标区域, 并且在对图像目标区域进行图像字符识别后, 进一步完成对识别图像字符的特征提取。通过反转操作确定最终坐标灰度值, 依据灰度值设置图像像素的最优值及像素最大取值, 并用像素最大值中减去坐标灰度值, 以上就是反向操作过程。

    2.2 BP神经网络字符识别

    通过对倾斜车牌图像字符图像的特征提取, 并对特征提取结果进行筛选处理, 以此实现图像传输特征的有效识别。人工神经网络方法和模板匹配是当前较为流行的字符识别方法[8]。其中, 人工神经网络方法的起源相对较早, 它最初是针对于生物学的研究。人工神经网络方法通过模拟神经系统, 以此获取信号在传输过程中表现出的主要特征, 最后通过提取传输原理进行开发。人工神经网络的最大可取之处是将大规模的处理单元整理为一个网状单元, 并将此完整的输入到系统中, 除此之外, 通过学习生物神经网络的许多优点, 快速的将海量知识信息存储到自身系统, 这些优点是的人工神经网络方法得到了国内外各个领域的广泛关注。

    神经网络有很多种类, BP神经网络就是其中的一种, 它与许多其他的神经网络差别不大, BP神经网络也是多层次结构, 是不可逆的单方向形式, 即由输入到输出。BP神经网络的拓扑结构主要由输入层, 隐式层和输出层组成[9]。尽管BP神经网络为单向传输方式, 但它具有双向学习的功能。通过多次调整和优化后, 最终输出结果与实际值相同。

    激励功能对于整个网络至关重要。S型激励函数已被广泛使用[10]。S型函数的表达式为:

    f (Netkj) =1/1+e-Netej (17)

    设uj值是θj, 则对于S型激发函数:

    δkj/okj= (tkj-okj) (1-okj) (20)

    实验仿真证明, 从结果可以得到由于本文采用了小波分析的字符图像能量作为特征量, 降低了神经网络的输入节点数量, 网络收敛速度较快, 识别的准确率高达99.89%, 提高了其图像字符的识别率, 文献[1]中使用的方法准确率为52.21%, 随着检测倾斜车牌字符图像数量的增加准确率有明显的降低, 该方法主要是由于神经网络的输入节点数量较多导致的, 节点数量多就会导致处理时间延长。文献[2]的方法, 在文献[1]的基础上进行了优化, 最高识别率为64.97%, 利用降噪自编码模型提取倾斜车牌字符图像相关特征, 提高字符识别精度。但是该方法随着检测倾斜车牌字符图像数量的增加识别准确率也会降低, 主要是由于相似字符特征相近, 出现了识别误差率。

    针对传统倾斜车牌图像字符识别的准确率不高的问题, 提出一种基于小波分析的倾斜车牌图像字符识别的方法。实验仿真证明, 所提方法提高了对倾斜车牌图像字符的识别率。能够更好的满足车牌字符识别应用的需求。但仍存在对倾斜车牌图像字符识别耗过多的问题。因此进一步研究降低对倾斜车牌图像字符识别的耗时问题成为下一阶段的首要课题。
