    摘    要：

    针对当前网络数据交换方法存在实时性和安全性差的问题, 提出基于DDET的复合可重构无线网络信息交换方法。采用树突状细胞算法检测网络中异常数据, 利用选取细胞单元时影响参数构建弱分类器, 采用弱分类器对阈值集合进行训练, 以阈值集合能够适应不断变化的抗原刺激为依据, 通过抗原加权之后的成熟度集合指标来评估目前数据状态异常与否。基于网络信息安全检测, 引入能量有效的信息交换协议DDET, 采集复合可重构无线网络中节点信息数据, 排列所有节点本地信息数据请求率。在每一轮信息交换过程中, 选取请求率最高的节点加入数据交换树中, 实现信息交换。实验结果表明, 所提方法信息交换效率高, 且具有较强地安全性, 与当前研究成果相比更具可行性。该方法性能优越, 可为该领域发展提供可靠支撑。

    关键词：

    复合; 可重构; 无线网络; 信息交换;

    Simulation of Composite Reconfigurable Wireless Network Information Exchange Method

    WU Yan FAN Hai-hong

    College of Electronic and Information Engineering, Guangdong Ocean University

    Abstract：

    Due to low real-time performance and poor security of current network data exchange methods, this paper presented a method to exchange the information in composite reconfigurable wireless network based on DDET. The dendritic cell algorithm was used to detect the abnormal data in network. The influencing parameters in selecting the cell unit were used to construct the weak classifier. The threshold set was trained by the weak classifier. Based on the fact that threshold set was able to adapt to the changing antigenic stimulation, the indexes of maturity set weighted by antigens were used to evaluate whether the current data was abnormal. Based on the network information security detection, the energy-efficient information exchange protocol DDET was introduced to collect the node information data in composite reconfigurable wireless network. After that, the local information data request rate of all nodes was arranged. In each round of information exchange, the node with the highest request rate was selected to add to the data exchange tree. Finally, the information exchange was achieved. Simulation results prove that the proposed method has high information exchange efficiency and strong security. Meanwhile, this method is more feasible than the current research result. This method has superior performance, which can provide reliable support for the development of this field.

    Keyword：

    Composite; Reconfigurable; Wireless network; Information exchange;

    1 引言

    复合可重构无线网络主要利用具备感知性能、计算性能和通信性能的传感器节点基于自组织方式构建的, 其中综合了传感器和嵌入式计算等多种技术[1,2]。此种类型网络是一种无处不在的运行模式, 应用领域非常广泛, 其中包含环境监测领域和定位跟踪领域等。数据交换作为无线网络中最为基础和重要的操作之一, 主要理念为网络节点收到本地数据之后, 在数据中继节点辅助下, 利用多跳路由将收集的数据传输目的地节点[3]。尽管目前已经出现了很多相关研究成果, 但其中仍存在很多值得探究的内容。

    吴亦奇[4]等人对于设计和制造条件下协同构建的CAD模型安全交换共享, 提出基于服务的数据交换方法。依据模型变形的方式改变数据特征模型参数, 完成模型敏感信息加密;利用特征数据交换方式完成异构CAD体系之间参数化交换;通过面向服务的框架, 将数据特征CAD模型数据交换和数据安全保护方式, 利用云端服务的形式给相关人员, 同时融合访问与授权控制理念, 进而保障服务合法性。通过实验证明所提方法有效性, 结果表明, 该方法安全性较强, 但存在数据交换耗时长的问题。曹健[5]等人提出基于数据流转发的数据交换方法。依据队列缓冲方法, 完成消息性数据异步传输和批量确认;构建流式数据转发策略, 进而消除数据在各个传输节点上缓存时延以及缓存代价问题, 并提升数据传输效率和并发性;引入分布式和数据路由理念, 完成第三方透明节点组网以及数据路由。利用模拟实验证明所提方法, 实验结果表明, 所提方法并行性较强, 但存在安全性较差的问题。杨小东[6]等人提出基于代理重签名的数据交换方法。通过离散对数构建双陷门的哈希函数, 同时证明所建函数可以满足陷门碰撞和抗碰撞以及抗密钥泄露等一系列特性。融合陷门哈希函数以及代理重签名策略, 将数据安全交换归纳到代理重签名方法及陷门哈希函数自身安全性能方面。实验结果表明, 该方法数据交换后占据的存储空间小, 但存在实时性差的问题。

    数据交换是数据利用的一种方式, 以上关于数据交换的方法中存在耗时比较长和安全性能不理想的问题, 提出基于DDET的复合可重构无线网络信息交换方法。

    2 复合可重构无线网络信息交换

    2.1 网络信息安全检测

    为了提升信息数据交换时的安全性, 需要利用一定的技术手段检测出网络中的异常数据。

    在网络信息数据安全检测中, 将树突状细胞算法DCA引入其中。DCA能够提供一组表征抗原异常度的数据信息成熟环境抗原值MCAV, MCAV为完全成熟抗原数目M和提呈抗原总数目Ag间比值, 其可表示为:

    ΜCAV=ΜAg    (1)

    DCA算法中包含一定数目的DC。其中, 在采集抗原信息数据时, 抗原上下文不同, 则抗原上下文输入数据信号组合不同, 抗原数据在正常情况下收集抗原上下文代表DC正处在半成熟时期, 存在潜在异常的上下文代表DC为成熟时期。那么式 (1) 可转换为:

    ΜCAVave=∑Μi (∑Ag1i+∑Ag2i)     (2)

    式 (2) 中, MCAVave代表数据序列抗原值。根据式 (2) 可实现初步异常检测。

    随机选取DCA细胞单元时, 会存在很多可变阈值, 导致算法性能受到不同因素影响, 针对这些参数值设置, 构建了弱算法分类器, 利用弱算法对阈值集合进行训练, 并迭代逼近配准度最高的阈值参数。

    假设Ad代表异常阈值, 其中:

    Adi (t) =diAi, i=1, ⋯, n    (3)

    式 (3) 中, di代表复合可重构无线网络中危险信号集合, Ai代表此类抗原数量, n代表训练样本数量。

    根据上述计算分析, 以此类推, As代表安全阈值, 假设:

    Asi (t) =SiAi, i=1, ⋯, n    (4)

    式 (4) 中, Si代表安全信号集合。

    通过弱算法分组训练Ad (t) ={Adi (t) |i=1, ⋯, n}和式 (4) , 挖掘出对应抗原权重集合。

    设定Μ (t) ={Μi (t) |i=1, ⋯, n}为整个抗原样本集合分组集合预测项j中权重值集合, 是抗原加权成熟度构建的集合。

    以阈值集合能够适应不断变化的抗原刺激为目的, 按照抗原加权之后的成熟度集合指标来评估目前数据状态异常与否。

    综上, 异常度量值表达式为:

    Dε (t) =∑i, h (t) (xi) ≠yiAdt (i) Μ (i)     (5) Sε (t) =∑i, h (t) (xi) ≠yiAgt (i) Μ (i)     (6)

    式 (6) 中, h (t) 代表弱分类器。通过加权之后选择的训练数据替代随机选择的训练样本, 由此将训练焦点集中划分较难的数据样本上[7]。

    依据式 (6) 计算, 对比Sε (t) 和Dε (t) 的值, 假设Dε (t) >Sε (t) , 则此类抗原是异常数据, 将其剔除;反之为正常数据, 保留用作交换。

    2.2 基于DDET的网络信息交换

    针对2.1节安全检测之后数据信息交换生成树T, 假设T中某个节点有两个或者两个以上的子节点, 则该节点全部子孙节点中应该至少还包含两个叶子节点, 其中, T的叶子节点一定是传感节点, 因此该节点全部子孙节点中应该至少存在两个传感节点。因接下来的操作为信息交换, 提到的两个传感节点需完成数据交换操作, 由数据交换生成树拓扑构造得知, 两个节点间实行信息交换最佳路径一定是经过距离自身最近的共同祖先节点。

    传感节点s′要加入交换树T过程中, 希望最小化节点s′自身携带的能耗增量值。节点s′选择树T中节点t′当作接入节点之后, s′的实际输出率S′s′为max (S′t′, Rt′) 。假设Rt′小于设定阈值R (s′, ε) , 那么表示本地请求率比较高的节点s′会加入交换树, 其将使节点t′实际数据请求率Rt′产生变化, Rt′与R (s′, ε) 相等。一旦出现这种情况, 就需要对树T上部分节点实际输出率进行重新调节, 另外, 优先选择本地请求率比较低的节点组建的生成树, 高请求率节点申请加入之后会降低生成树性能。如果Rt′比R (s′, ε) 小, 则s′的加入针对目前信息交换树中节点实际请求率与实际发送率不存在相互影响。综上, 保障比较大的本地请求率节点能够优先加入信息交换生成树中, 对树的组建十分重要[8,9]。

    假设传感节点s′利用节点t′加入到T中, 新增加的能耗包含两部分, 一种是由s′至t′路段消耗的能量, 另一种是在原树T链路中传输新增加的传感节点s′中数据消耗的能量。对于任意非根节点∀p∈NT, NT代表子孙节点集合, 假设R (s′, ε) <Rp, 则s′由任意节点加入至树T对于原来就存在的链路来说消耗的能耗相同, 在此种状况下, 只需要关注新增加的能耗, 也就是由s′至t′路段消耗的能量, 将其记作Enew。尽管s′可能不能直接参与到与t′的数据传输交换中, 但可利用式 (7) 评估Enew:

    Enew (t′, s′) =l⋅d2 (t′, s′) ⋅[max (S′t′, Rt′) +|Νs′now|⋅R (s′, ε) ]    (7)

    由式 (7) 能够看出, 新增能耗比较依赖新加入的节点s′与接入节点之间存在的距离d (t′, s′) , Ns′now代表当前加入到树中的节点集合。在信息交换树构建时, 假设选择了适当的接入点, 则能够尽可能地节省能耗。

    依据上述数据交换代价分析, 详细描述DDET数据交换算法流程。

    数据交换流程在刚开始运行时, 有一个信息数据采集操作, 对全部节点进行编号, 以提升信息交换时的实时性。编号之后, 位置信息与本地数据信息请求率传输至与区域中心最接近的节点M′。这里使用的寻路机制为贪心寻路法:复合可重构无线网络节点将由邻居节点中筛选出满足式 (8) 条件的节点nb当作路由中继, 一直到目的地节点, 以此实现链路构建。

    Ο=min (d2 (sensor, nb) +d2 (nb, Μ′) )     (8)

    节点M′在受到全部节点信息数据之后, 将各节点本地数据请求率当作权值, 并实行降序排序操作。将排好序列的信息数据打包好之后放至引导消息中, 同时传输至本地数据信息请求率最大节点中, 收到引导消息节点就变成激活节点ac, 并将本节点信息由引导信息中清理掉。

    激活节点由引导消息节点序列中筛选出待加入节点, 将待加入节点定义为next, 同时为该节点找到最优接入点。选取出本地信息数据请求率最高节点成为待加入的节点, 假设有很多个本地信息数据请求率最高节点, 那么激活节点即选择距离本身最为接近的节点[10]。为待加入节点筛选最佳的接入点, 假设AcN代表激活节点在目前交换树上邻居节点集合, 如果目前激活点满足Enew (ac, next) ≤mint′∈AcN (Enew (t′, next) ) , 则目前激活节点就是最优的接入点, 反之满足mint′∈AcN (Enew (t′, next) ) 的节点t′会成为新激活点, 然后在t′中实行找寻机制, 一直到寻到最优的接入点。在这之后, 目前激活节点通过贪心寻路法实现路由节点组建, 以此使该节点加入到树。路由构建之后, 则待加入节点不仅会接收引导消息, 还会变为激活节点, 且该节点由引导消息将本地节点信息清理完毕之后, 继续筛选新的一轮待加入节点。一旦引导消息中全部节点均加入该树, 则数据交换截止。

    最后的激活节点利用虚拟减枝法找到树根节点。在每一轮的数据交换过程中, 全部叶子节点会临时被分离出交换树, 此时, 各个节点实际信息数据请求率能够利用式 (9) 得到。

    Rt′=max (Rcj, ε)     (9)

    式 (9) 中, cj代表子孙节点集合中的元素。

    减枝算法能够保障获取一颗深度比较小的生成树, 进而使最终获取的信息交换树时延达到最佳。减枝时, 以同步全部叶子节点减枝流程为目的, 方法实现主要依赖节点间时钟同步理念。当树根节点选定之后, 树上各个节点关系会变得明确, 这时, 各个节点可利用式 (10) 获取实际信息数据输出率:

    St′=maxcj (R (cj, ε) )     (10)

    其中, St′代表节点t′位置实际信息数据输出率。

    4 结束语

    复合可重构无线网络是当前新兴的领域, 面对数据分发与数据采集问题, 信息交换应运而生。为此, 提出基于DDET的复合可重构无线网络信息交换方法。在信息交换之前, 采用树突状细胞算法检测网络异常, 以此确定信息交换安全性。在信息安全检测基础上, 利用DDET构建信息数据交换树, 实现信息交换, 依据实验证明了所提方法具有可靠性和实用性。对于下一步研究提出建议为:实际信息交换中, 经常存在移动节点, 由此适应移动节点信息交换也可作为未来研究方向, 从而进一步提升网络数据交换性能。
