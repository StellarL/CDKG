    摘    要：

    动态用户网络连接干扰和安全认证问题始终是影响无线网络成功通信的重要因素, 对于当前方法存在通信信号传输防干扰性能较差, 以及无法实现动态用户网络连接安全认证的缺点, 提出了基于分布式干扰对齐与TNC相结合的动态用户网络连接防干扰与安全认证方法, 在建立网络连接通信信号传输模型基础上, 采用分布式干扰对齐的方式确保动态用户在网络连接后能够实现无干扰通信, 同时在网络接收节点通过简单的迫零接收消除来自网络内部的干扰, 实现动态用户网络连接防干扰;在此基础上, 设计基于可信网络连接规范 (TNC) 的动态用户网络连接安全认证协议, 给出了网络连接与安全认证协议执行具体流程, 按照该流程, 同时采用对称密钥算法安全认证动态用户网络连接身份。仿真测试结果表明, 所提方法能够实现动态用户网络连接防干扰和通信双方身份安全认证, 且具有执行效率较高、机密性能较好的优点。

    关键词：

    动态用户; 网络连接; 防干扰; 安全认证;

    Simulation of Dynamic User Network Connection Anti-interference and Security Authentication Method

    GAO Jin

    Business College of Shanxi University

    Abstract：

    Interference and security certification of network connection of dynamic user are always key factors influencing successful communication of wireless network. Anti-interference property of communication signal transmission of current method is poor. Meanwhile, the method cannot achieve the security certification. Therefore, integrated with distributed interference alignment and TNC, a method of anti-interference and security certification of network connection of dynamic user is proposed in this article. Based on transmission model of communication signal of network connection, the distributed interference alignment was used to ensure that dynamic user could achieve non-interference communication after network connection. Meanwhile, interference from network was eliminated via simple zero-forcing receiver in network receiving node and the anti-interference of network connection was achieved. On that basis, based on the Trusted Network Connect Specification (TNC) , the security certification was designed and specific flow of implementation of protocol of network connection and security certification was provided. According to the flow, symmetric key algorithm was used to certificate network connection identity of the dynamic user. Simulation results show that the method can achieve the anti-interference and security certification. It has higher execution efficiency and better secret property.

    Keyword：

    Dynamic user; Network connection; Anti-interference; Security certification;

    1 引言

    发展到现阶段的无线网络连接已经不仅仅是一种简单的网络接入方式, 企业的许多诸如语言服务、视频服务以及定位服务等重要应用都逐渐部署到无线网络上。用户也更加注重网络连接后的浏览流畅性, 但随着网络多媒体应用市场的快速发展, 各种网络应用数目急剧增加, 动态用户在网络连接过程中遭受到的无线干扰问题随之增加, 导致网络服务质量越来越差, 很难达到用户预期体验[1]。给网络通信安全带来了极大威胁, 为了保证网络安全稳定运行的同时给用户带来更加良好的网络体验, 需要研究一种动态用户网络连接防干扰与安全认证方法[2,3]。

    文献[4]提出了基于斜投影滤波器的动态用户网络连接防干扰与安全认证方法, 该方法利用斜投影滤波器滤除动态用户网络连接过程中的干扰信号;在此基础上, 采用RFID安全认证协议作为动态用户网络连接过程中的安全认证协议, 实现了动态用户网络连接防干扰与安全认证。实验结果表明, 该方法的防干扰性能较差。文献[5]提出了基于自适应MVDR的动态用户网络连接防干扰与安全认证方法, 通过采用自适应MVDR方法有效提高了动态用户网络连接过程中的防干扰效果, 并基于此利用EAP-AKA协议实现了动态用户网络连接安全认证。实验结果证明, 该方法无法较好地实现动态用户网络连接安全认证。

    研究针对上述方法存在的缺点与不足提出了基于分布式干扰对齐与TNC相结合的动态用户网络连接防干扰与安全认证方法。

    2.2 动态用户网络连接安全认证方法

    在上述实现了动态用户网络连接防干扰基础上, 设计基于可信网络连接规范 (TNC) 的动态用户网络连接安全认证协议, 设计原理如图1所示。

    从图1中可以看出, 动态用户都工作在网络特定平台上, 网络平台可信是动态用户网络连接可信的重要前提。为了使该安全认证协议具有较好的通用性能, 同时为了简化设计假设动态用户网络接入终端均包含TPM, 则基于TNC动态用户网络连接安全认证协议的第一步为通过采用TPM提供的网络平台可信服务 (PTS) 和完整性收集代理采集的网络终端数据认证和衡量动态用户网络连接的安全性和完整性。

    如果动态用户网络连接平台认证顺利通过, 下一步即认证网络连接动态用户身份。在动态用户网络连接完整性检验中添加对称加解密步骤, 即平台认证顺利通过后, 应用公钥加密传输的方式为动态用户之间的通信传输协商一个对称密钥, 动态用户之间可以通过共享该对称密钥实现通信传输, 同时不受网络连接过程中其他用户的干扰, 能够有效避免第三方猜测密码实行恶意攻击等。最重要的是, 通过在动态用户网络连接关键点布置防火墙等和流量控制器等网络安全防御装置将网络连接状况实时反馈到元数据存储点, 能够使网络后台管理者 (即如图1所示的策略决策者) 及时断开存在安全隐患的动态用户网络连接, 从而实现动态用户连接过程中的实时动态控制[9]。

    观察表1和表2的实验结果可以发现, 相比于其他两种对比方法而言, 所提方法的安全认证协议执行效率最高, 大大缩短了动态用户网络连接安全认证时间, 有利于减小通信延时, 且安全认证协议执行机密性更好保障。

    4 结束语

    鉴于动态用户网络连接过程中遇到的来自网络内部的各种干扰信号影响以及双方通信过程中的身份认证问题, 提出了基于分布式干扰对齐与TNC相结合的动态用户网络连接防干扰与安全认证方法, 并在仿真平台TinyOS上做了一系列性能测试, 证明了相比于其他方法, 所提方法能够实现动态用户网络连接防干扰和安全认证, 且具有执行效率高、安全性能好的优点。
