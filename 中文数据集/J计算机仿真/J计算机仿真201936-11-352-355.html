    摘    要：

    针对传统用户需求网络信息的优先协同过滤推荐方法存在推荐准确度较低、推荐范围小等问题,提出基于相似性度量的用户需求网络信息优先协同过滤推荐方法。根据目标网络用户对网络信息需求的感兴趣程度对用户进行分组,引用贝叶斯算法计算用户具有不同特征时对网络信息需求的喜好程度。在项目近似度公式中引入用户对项目的喜好程度概率值计算项目间的相似度,得到最高相似度邻居项目的评分。再利用云模型计算用户的相似度得到邻近用户,得到最终的预测评分,将预测评分值较高的前几个项目优先推荐给用户,完成推荐。仿真结果表明,相比传统的协同过滤推荐方法,所提方法可在一定程度上提高推荐准确度和推荐覆盖率,能够更准确的为用户提供所需要的网络信息。

    关键词：

    用户需求; 网络信息; 优先协同过滤推荐;

    User-Required Network Information Prioritized Collaborative Filtering Recommendation Simulation

    GENG Xiao-li DENG Tian-wen

    South China Institute of Software Engineering, Guangzhou University

    Abstract：

    Traditional collaborative filtering recommendation methods have some problems such as low recommendation accuracy and small recommendation range. Therefore, a preferential collaborative filtering recommendation method for user demand network information based on similarity measure was proposed. According to the interest degree of target network user to the network information demand, user was put into groups. Then, Bayesian algorithm was used to calculate the preference for the network information demand when user had different features. In the item approximation formula, the probability value of preference for item was introduced to calculate the similarity between items, and then score of neighbor item with highest similarity was obtained. Moreover, the cloud model was used to calculate the similarity of user and thus to obtain the neighboring user. After that, the final predicted score was obtained. Finally, the first few items with higher predicted values were preferentially recommended to users. Thus, the recommendation was completed. Simulation results show that, compared with the traditional collaborative filtering recommendation method, the proposed method can improve the recommendation accuracy and recommended coverage rate. Meanwhile, this method can provide users with the required network information more accurately.

    User demand; Network information; Preferential collaborative filtering recommendation;

    1 引言

    互联网和网络技术的不断发展,导致信息负载和个性化需求之间的矛盾越来越大[1,2],推荐系统作为对信息负载现象进行信息筛选,得到了广泛的应用[3]。协同过滤方法是当今社会最为成功的推荐方法之一,但现阶段用户需求网络信息的优先协同过滤推荐方法中[4,5],存在准确率和覆盖率较低等问题,在保护用户隐私前提下,如何实现准确协调过滤的方法成为当今社会亟待解决的问题

    文献[6]通过构建贝叶斯网络模型进一步实现了协同过滤推荐方法。该方法采用贝叶斯网络模型计算出认知反馈的分布概率,以此对评分数据进行降维处理,构建商品反馈模型,定义出用户认知属性,给出评价信息。最后利用相似度计算方法给出推荐。该方法推荐覆盖范围较小,无法为用户提供更多的信息。文献[7]提出一种融合网络信息的协同过滤推荐方法。该方法对物品相似度进行计算,得出物品网络信息的评分预测模型,利用社交网络对模型中的缺失值进行信息填充,最后依据最大化平均矩阵进行协同过滤推荐。该方法在评分误差上优于其它任何方法,但是该方法为用户推荐的信息范围有限。文献[8]提出一种基于信任的协同过滤方法。该方法构建用户信任模型,求出对用户信任感兴趣的最多邻居用户,引用时间遗忘函数来显示用户兴趣变化,根据用户兴趣变化相似度来进行协同过滤推荐。该方法具有一定的有效性,但是该方法为用户推荐信息的准确度偏低

    针对上述问题存在的缺陷,提出了一种基于相似性度量的用户需求网络信息优先协同过滤推荐方法。

    2 用户需求网络信息的优先协同过滤推荐原理

    对于用户需求网络信息的优先协同过滤推荐,首先对用户需求信息中的非结构化文本内容进行处理,通过处理得到文本内容关键词,结合关键词与用户需求构建用户需求信息关键词模型;利用马尔可夫链模拟用户需求信息的网络更新过程,获取用户需求信息关键词矩阵,矩阵关键词对应的文本内容即为协同过滤推荐结果。具体过程如下:

    用户需求网络信息的核心是获取对应关键词,采用中文词法分析对用户i所需求的网络信息内容ci进行分词处理,得到的关键词集合为ci={k1,k2,…,kl},其中kj表示关键词,l为关键词数量,1⪯j⪯l;得到用户需求信息关键词矩阵KST={ki,j},KST矩阵为n行m列矩阵,n为用户数量,m为用户需求信息的关键词数量,ki,j为用户i搜索关键词j的次数,矩阵中的各行Ki即为对应用户i需求信息的关键词集合。采用离散化方法度量用户需求信息关键词的重要程度,根据度量结果剔除无用关键词,用数字1～5代表关键词对于用户需求信,息的重要程度,1表示一般重要,5表示极其重要;选择对数函数对关键词按重要程度进行分类,过程为

    fi表示log2(min(ki,j)),log2(max(ki,j))的五分位点。通过上式得到用户需求信息关键词描述模型KS=u×K',其中u表示用户需求信息关键词的重要度系数,K'={K'1,K'2,…,K'n},K'i为用户i的需求信息关键词模型。

    依据用户需求信息的个性化偏好关键词k'i,j以及数据库文本关键词k^i,j的匹配度对用户需求信息协同过滤推荐模型进行规范化处理,得到模型如下

    Ρ=argmax{∑jf(ki,j´,k^i,j)|ResSi} (2)

    其中,ResSi∈ResS表示被推荐给用户i的关键词集,ResSi=<KS,R,U,K>;U表示用户集,R表示用户关联矩阵,R⊂U×U。

    用户需求信息的更新近似于马尔科夫性质,是一种自组织过程,对于用户偏好信息在t时刻的更新过程,利用马尔可夫链模型进行描述:

    P(ksi,jt|ksk,jt-1,R)=(ksk,jt-1×τi,j)/max(KSt) (3)

    其中,ksi,jt、ksk,jt-1、τi,j分别表示用户需求信息关键词矩阵KS及用户关联矩阵R中的元素,当用户需求信息关键词的重要度越大,说明用户对该关键词的偏好度越大,通过叠加法计算整体网络信息更新值

    Ρ(ksi,jt|ΚSt-1,R)=∑kΡ(ksi,jt|ksk,jt-1,R)(ksk,jt-1×τi,j)  =∑k(ksk,jt-1×τi,j)/max(ΚSt)         (4)

    上式体现了信息随机游走的属性,刻画了信息更新的过程,结合系统动力学与信息更新理论得到信息更新的稳定状态,记作KS″,该状态融合了KS和R,且刻画了用户信息需求的偏好,KS″的计算式为

    KS″=RT×KS (5)

    式中,RT=R×R×…×R。

    依据用户个性化需求偏好,得到用户需求信息关键词矩阵,并对协同过滤模型进行规范化处理;结合马尔可夫随机游走及用户需求信息关键词矩阵和用户关联矩阵进行网络信息更新;对于更新后的网络信息,利用规范化处理后的协同过滤推荐模型完成用户需求信息推荐。

    针对传统方法存在准确率较低、推荐覆盖率家偏低等问题,提出基于相似性度量的用户需求网络信息优先协同过滤推荐方法。仿真证明,所提方法有效提高了推荐准确度,并且推荐覆盖率也大幅度提高,能够为用户推荐更好的信息。
