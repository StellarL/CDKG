    基于相似度的多重信息协同过滤算法优化仿真

    摘    要：

    针对当前协同过滤算法准确度低、耗时长的问题,提出一种新的多重信息协同过滤优化算法。构建用户项目的评分矩阵,通过用户评分的先后情况对用户间的时序行为影响力进行计算;将用户的评分差异作为条件,对其加权信息熵的相似度展开计算;利用用户和项目的总数量构建用户项目种类的兴趣矩阵,计算兴趣相似度,得到全部项目种类和调节参数等条件;利用调节参数对相似度的计算精度进行调节;构建项目种类评分矩阵,分析获得用户和项目种类之间的关系,将项目种类兴趣矩阵和项评分矩阵相融合,获得最终的相似度,实现多重信息的协同过滤。实验结果表明,所提算法中得到的平均绝对误差较小,并且算法耗时短、效率高,能够有效、准确的完成对多重信息的协同过滤,具有较高的应用优势。

    关键词：

    相似度; 多重信息; 协同过滤; 算法优化;

    Optimization Simulation of Multiple Information Collaborative Filtering Algorithm Based on Similarity

    XUE Ya-fei

    Nanjing Normal University Zhongbei College

    Abstract：

    At present, the collaborative filtering algorithm is time-consuming. Therefore, a new algorithm to optimize multi-information collaborative filtering was proposed. Firstly, the scoring matrix of user project was constructed, and the influence of the time series behavior between users was calculated by the user scoring order. Then the difference in score was used as the condition and the similarity degree of the weighted information entropy was calculated. The total number of users and projects was used to construct interest matrix of user project types and calculate interest similarity, so that all project types and adjustment parameters were obtained. Moreover, adjustment parameters were used to adjust accuracy of similarity degree. In addition, the scoring matrix of project category was constructed to analyze the relation between user and project category. Finally, the interest matrix and the scoring matrix were combined to obtain the final similarity degree, and thus to achieve the collaborative filtering of multiple information. Simulation results show that the average absolute error obtained by the proposed algorithm is smaller. Meanwhile, the algorithm needs shorter time and has higher efficiency. Thus, this algorithm has high application advantages, which effectively and accurately complete the collaborative filtering of multiple information.

    Keyword：

    Similarity degree; Multiple information; Collaborative filtering; Algorithm optimization;

    1 引言

    在电子商务快速发展的背景下,为了减少客户对无关信息的浏览所占用的时间和精力,需要对信息的过载问题进行高效的处理,因此产生了有关于无关信息智能过滤、推荐的方法[1,2]。在电子商务信息推荐系统中,多重信息协同过滤是一种应用十分广泛的推荐算法。利用邻居集和目标用户兴趣类别得到最终的推荐信息是该算法的核心。多重信息协同过滤算法能够对非结构化的数据进行处理,还能够发掘用户的兴趣[3],并且在用户评分的不断增加下,使推荐质量得到升高。目前,已有一部分研究学者在该领域进行了深入研究,并提出了一些信息协同过滤的方法。

    张景龙等人[4]提出基于标签优化的多重信息协同过滤算法优化,首先确定用户评分矩阵的邻近,对各个目标项进行用户邻居的计算,当项目评分没有邻近数量或者数量过少时,需要加入标签信息扩展的近邻,再将近邻当作目标项进行预测和评分,最终实现了多重信息协同过滤。该算法缓解了数据的稀疏问题,但平均绝对误差较大,准确度较低,应用性能不强。孟桓羽等人[5]提出一种基于图和改进K近邻的多重信息协同过滤算法的设计方法,构建数据评分模型,将改进相似度模型和图消息相结合对用户做筛选处理并对其进行相似度的计算和处理,将用户的节点结构当作基础,利用图最短路径方法定位待评分项,根据图框架优化算法,实现了多重信息协同过滤。该算法的可扩展性较好,但存在平均误差大,准确度低和耗时长的问题。林皓等人[6]提出一种基于用户情境的多重信息协同过滤算法,将用户情境加入到算法中,对用户所在的情境处进行分析,并对预测评分做加权处理,用来确保推荐结果与用户的偏好相符,在生成推荐时,增加权重排对推荐候选项,大大提高了其时序的多样性,完成了多重信息协同过滤。该算法可以保多重信息协同过滤的准确性,但耗时较长、效率低较,应用性能不强。

    针对上述传统过滤算法中存在的过滤准确度低、耗时长的问题,在相似度研究的基础上,提出一种多重信息协同过滤算法,通过计算和调节用户相似度以及兴趣矩阵和项评分矩阵的融合提高过滤准确度,实现快速过滤。

    2 相关工作

    2.1 时序行为影响力

    经过研究得知,网络中与节点相对应的对象间存在多样性的关系,由于这些多样性的关系,与节点相对应的对象间也存在互相影响,因此可将将用户关系当作粒子。通常用户在购买一些商品时会被朋友的意愿、倾向影响到,调查结果表明,受在好友消费商品后将该商品推荐给目标用户,比相似度推荐商品更加可靠。

    用户购买商品的时间信息中包含部分规律,根据这些规律能够将识别系统中的特征识别出来,或是对特定事件的发生情况进行预测。假设根据用户消费商品的时间,对用户在该段时间中的消费行为构成一个序列,该序列能够揭示出隐藏的规律[7]。用户的消费行为很容易被亲近的人影响到,存在的规律为:假设用户i受用户j的影响较大,则j为i亲近的人,用户i与用户j的选择结果将更加相似,时序行为影响力的表达式如下

    effectij=ωijunionij (1)

    式中,对于用户i和j一起购买的商品来说,ωij代表i比j先购买商品的数量,unionij代表i和j购买商品的并集,effectij代表i对j的有向影响力。

    2.2 信息熵相似度

    信息论之父克劳德·香农提出了信息熵的概念,能够将信息熵当作衡量信息分散和分布程度的概念。信息熵的大小随着信息的分散程度变化,分散程度越小,信息熵也越小。经过相关研究得知,加入相似度量在多重信息协同过滤算法优化中,能够对用户间评分项差异的离散程度进行衡量[8],其差异熵越小,说明用户之间存在的差异程度越小,相似性越高。加权信息熵相似度的表达式为

    Η(i,j)=-1n∑k=1np(dk)log2p(dk)×|dk| (2)

    式(2)中,k表示邻居用户的个数,p(dk)为用户评分差dk在差异向量D中的概率,dk越大差异性越高,加入权重|dk|;n为用户同时评分项目的数量,n越大相似性越好,对应的熵值则越小。引入用户同时评分项目数量的权重1/n。

    此时,用户i和j对H(i,j)不能够参与到下一步的计算中。用户i与全部用户的加权信息熵向量H(i,:)中的元素均取值[0,∞],需要对其进行归一化处理:

    entropy(i,j)=Μax(Η(i,:))-Η(i,j)Μax(Η(i,:))-Μin(Η(i,:)) (3)

    其中,Max(H(i,:))、Min(H(i,:))分别代表H(i,:)中元素的最大和最小值,entropy(i,j)代表经过归一化处理之后用户的相似度情况,通常情况下,取值为[0,1],该值越大,用户间的相似性越好。

    3 多重信息协同过滤算法优化

    为了考虑数据的稀疏问题,优化对相似度的计算,将计算过程分成两部分,即在项目种类评分矩阵以及项目种类兴趣矩阵中对其相似度进行计算,并加入调节项来修正相似度。

    3.1 用户项目种类兴趣矩阵

    算法的主要思想是对用户的相似度进行计算,兴趣相似度可以用用户间的关注点以及兴趣进行描述。利用用户对项目种类中全部项目评价数量之和来表示该指标,该指标的值越高,说明用户对该类的兴趣度越高。且该值为有限整数。

    用户的兴趣所描述的是对于某项目种类的感兴趣情况,利用降低维度的方法难以有效的对数据稀疏问题做处理[9],需要与另外一种方法进行结合,可以提高精度上的要求,构建兴趣矩阵Tx×y

    该矩阵可以表示出用户对每个种类的感兴趣情况。式中,x表示用户的总数量,y表示项目种类的总数量,txy表示x个用户给y个项目种类中含有的全部项目评价次数的总和。为了可以有效提高兴趣相似度的准确性,考虑了用户年龄带来了影响,根据日常经验来看,年龄相近的用户兴趣相同的概率较大,而年龄相差较大用户间的关注点以及兴趣差异也较大,例如音乐,年轻人喜欢流行或者摇滚歌曲,中年人则对抒情歌曲比较感兴趣,老年人对年代性强的歌曲感兴趣[10]。除了对用户历史评分总数的考虑外,引入年龄调节项,可以使相似精度得到提高,用户u和v间项目种类兴趣相似度的表达式为

    sim(u,v)=[∑c∈Cuv(tu,c-t¯u)(tv,c-t¯v)]=∑c∈Cuv(tu,c-t¯u)2∑c∈Cuv(tv,c-t¯v)2⋅ww+|uage-vage|         (5)

    式中,Cuv代表集合,用户u和v评分之后的全部项目种类共同组成该集合,tu,c表示u对项目种类c中含有的累计评价,t¯u代表用户u对全部项目种类评价次数的均值,uage代表u的年龄,w代表可以调节参数,可以对相似度的计算精度进行调节。

    3.2 用户项目种类评分矩阵

    评分相似度可以通过对用户评分的偏好和倾向计算,确定用户的评分相似度,用户之间对项目种类评分的接近程度直接影响了其相似度,评分越接近,相似度则越高。分析用户打分范围和打分习惯的差异,利用评分的均值表示项目种类的评分,取值和评分的范围相同。通常情况下,评分的均值在[0,5]之间[11]。

    由于每个商品的自身属性存在不同,因此可按照其属性将商品归类到不同的种类中。同一个种类中也会包括一些不同的商品。项目与项目种类属于多对多的关系,项目种类为一个集合,其中包括多个属性相同的项目,项目种类的概念是指利用聚类的方法来降低评分矩阵维度,缓解了数据的稀疏问题,项目种类评分矩阵可以分析出用户和项目种类之间的关系,矩阵表达式如下

    式(6)中,pu×v表示u个用户给v个项目种类中含有项目评分的均值,利用该矩阵能够表示出用户对项目种类中全部项目的满意情况。

    若想要提高项目评分相似度的准确性,需要对用户评价项目是否一致进行分析。对一个极端的情况进行分析,假设两个用户同时对一个项目种类评分,并且评分较为接近,但所评价的项目之间无关联,则不能说明两个用户之间的相似性较高[12]。除了对项目种类历史评分进行分析之外,加入调节项σu,vc代表评分项目的一致性,从而相似度的精度将得到提高。

    采用改进的皮尔森相似度对其展开计算,详细的计算公式为:

    simR(u,v)=∑c∈Cuv(ru,c-r¯u)(rv,c-r¯v)⋅σu,vc∑c∈Cuv(ru,c-r¯u)2∑c∈Cuv(rv,c-r¯v)2         (7)

    其中,ru,c为用户u对项目种类c中全部项目的平均评分,r¯u为用户给分均值,调节项σu,vc为在c中,u和v一起评价项目数与两用户所评价项目的总数量之比。

    3.3 多重信息协同过滤

    经过3.1节和3.2节的介绍可知,simR(u,v)主要为评分相似性,仅对用户评分相似性进行了分析,可能会存在评分相似兴趣差异大的现象;sim(u,v)主要为用户之间关注点以及兴趣的相似性,不存在与评分相关的信息,该指标对多重信息协同过滤相似度的计算来说十分重要,进行多方面的考虑可以提高其相似度的精度,利用权重参数α将以上两种相似度做融合处理,可以获得最终的相似度,完成多重信息协同过滤优化,表达式如下

    sim(u,v)=αsim(u,v)+(1-α)simR(u,v) (8)

    利用式(8)最终实现了相似度的多重信息协同过滤算法优化。

    分析图2可知,在所提算法的12次实验中,算法的耗时均小于0.6μs,在文献[5]算法和文献[6]方法的12次实验中,算法整体耗时的变化范围分别为0～1.0μs和0.8～1.1μs。文献[5]算法的耗时情况变化波动较大,算法的最短和最长耗时相差较多。经过对比可知,所提算法的所用耗时较短,能够证明所提算法的多重信息协同过滤算法优化效率更高,推荐质量较好。

    5 结论

    为了对多重信息协同过滤算法进一步优化,提出基于相似度的信息协同过滤优化算法:

    1)通过用户项目的评分矩阵对用户间的时序行为影响力进行计算,在此基础上对其加权信息熵的相似度展开计算;计算并调节兴趣相似度,构建项目种类评分矩阵,将项目种类兴趣矩阵和项评分矩阵相融合,获得最终的相似度,实现多重信息的协同过滤。

    2)在实验阶段,分别利用平均绝对误差和耗时情况测试了算法的准确度和效率。实验结果表明所提算法的平均绝对误差相对较低,过滤耗时均小于0.6μs,比现有方法减少了一半以上,并且所提算法还能够为用户提供出更理想的推荐,应用价值更高。
