    摘    要：

    目前网络用户登录特征数据分类挖掘方法分类准确性和安全性差, 提出基于Leader的社交网络用户登录特征数据分类挖掘方法。根据用户登陆经常一起使用的命令之间存在紧密性, 不常组合使用的命令之间关系疏远的理念, 设定社交网络用户登陆活动与合法用户不同, 其组合命令方式不同这一条件。计算社交网络用户登陆两个命令间紧密度, 将所得值视为判断异常用户阈值, 如果紧密度高, 则表示该用户为正常用户, 反之为异常用户, 将其剔除。根据社交网络异常用户剔除结果设定分类挖掘输入集合, 并将输出集合定义为空。针对输入子集, 寻找输出子集, 并给出输入子集和输出子集之间的质心距离最短值, 将最短值与距离阈值进行比较, 将比较结果分别添至输出子集和输出总集中。给出质心计算公式, 对新用户登陆特征数据分类的属性集合进行计算, 并将计算结果分别添至输出子集和输出总集中, 完成用户登陆特征数据分类挖掘。实验结果表明, 上述方法安全系数平均为0.92, 分类准确率高。

    关键词：

    社交网络; 用户登录; 特征; 分类挖掘;

    Social Network User Login Feature Data Classification Mining Simulation

    ZHANG Shu-yue

    Guangzhou College of Technology and Business, Electronic Information Engineering Department

    Abstract：

    A method to classify and mine user logon feature data in social network based on Leader is proposed. According to the tightness between commands used by the user login frequently and the alienation of relation between commands that were not often used together, we determined the condition that login activity of social network users was different from legitimate users and the combination command mode was different. Then, we calculated the tightness between the two commands of social network user login and regarded the value as the abnormal user threshold. If the tightness was high, the user was a normal user. Conversely, it was an abnormal user. We needed to remove it. According to the elimination result of abnormal user in social network, we set classification mining input set and the defined the output set as empty. For the input subset, we found the output subset and gave the shortest centroid distance between the input subset and the output subset. Compared the shortest value with the distance threshold, we respectively added the comparison results to the output subset and the output overall set. After the centroid calculation formula was given, we calculated the attribute set of new user login feature data classification. Meanwhile, we added the calculation result to the output subset and the output total set respectively. Thus, we completed the classification mining of user login feature data. Simulation results show that the average safety factor of proposed method is 0.92 and the classification accuracy is high.

    Keyword：

    Social network; User login; Feature; Classification mining;

    1 引言

    互联网络的发展, 使得网民规模逐渐扩大, 导致社交网络用户登录特征数据也发生了很大的变化[1]。社交网络用户登录行为是一种虚拟行为, 是为达到某种目的而进行的一种活动。然而, 在用户实施此项行为时, 会有很多隐含的特征信息, 需要利用一定手段进行挖掘之后才可以发现[2,3]。综上可知, 社交网络用户登录特征数据分类挖掘的研究十分重要, 已成为亟待解决的问题。当前基于Scrapy的社交网络用户登录特征数据分类挖掘方法, 通过Scrapy网络爬虫对社交网络用户登录数据进行获取, 从收集到的数据中对特征值进行提取。针对训练样本, 要于人工标记之后, 再应用至机器学习中, 数据样本的训练分类使用SVM法实现。该方法安全性和分类准确性无法满足该领域发展需求。关于网络用户特征数据分类识别的相关研究有很多, 以下成果在当前应用较为广泛。

    郑晓雪[4]等人提出基于用户身份特征的网络用户多标签数据分类法。过程中, 引入启发式规则, 并结合Adaboost.MH算法, 将数据集分片概念摒弃, 算法运行的最后将单条数据当作分析焦点, 实现用户间的关系分析, 完成网络用户多标签数据分类。实验表明, 该法进行数据分类过程中的网络能耗较低, 但分类准确性较差。袁红[5]等人提出基于决策树分类的用户特征数据挖掘法。过程中, 将微信用户当作研究对象进行问卷调查, 根据决策树分类法实现微信用户特征数据挖掘, 完成隐私悖论行为生成规则以及存在性分析。实验结果表明, 该法分类效率高, 但分类准确性不是十分理想。周锐[6]等人提出基于IP过滤和特征组合的网络用户数据分类法。过程中, 通过Wireshark对网络中不同游戏类型产生的数据流进行抓取, 其中包含大型的多人角色扮演类别、以第一人称进行射击类别、以及卡牌类游戏, 对用户扮演的数据流进行识别。通过协议过滤与IP过滤法实现数据流的预处理, 分析下行及上行数据的统计特征, 依据分析结果, 将下上行包的大小信息熵和下上行包数量比及PPS等特征作为网络用户数据分类依据, 完成数据挖掘。实验结果表明, 该法分类准确率较高, 但网络安全性较差。

    社交网络用户登录特征数据十分复杂多变, 针对当前相关研究成果中存在的问题, 提出基于Leader的社交网络用户登录特征数据分类挖掘方法。

    2 社交网络用户登录特征数据分类挖掘原理

    在社交网络用户登录特征数据分类挖掘原理中, 利用拉普拉斯实现网络用户登录特征映射分类。过程中, 根据拉普拉斯算法目标, 定义社交网络用户登录数据分类挖掘目标函数。通过构建加权邻接函数, 得到实际社交网络用户登录特征数据的表示形式。将社交网络用户登录数据若干个最小特征值相应特征值映射至低维空间, 结合网络用户登录特征数据的表示形式, 得到网络用户数据映射分类结果。

    在拉普拉斯算法中, 其目标为寻求平均意义上社交网络用户登录数据点的局部特征映射, 进而完成网络数据分类挖掘。在此, 将目标函数定义为

    L (f) =12∑i, j (fi-fj) 2Wij (1)

    式 (1) 中, L代表拉普拉斯中的贝尔拉特米算子, 也表示图的拉普拉斯矩阵, f代表社交网络中潜在的数据特征维, fi代表用户i在社交网络登录中的特征维, fj代表用户j在社交网络登录中的特征维, Wij代表用户i和用户j在社交网络登录中的特征加权邻接函数。

    3 社交网络用户登录特征数据分类挖掘方法

    3.1 社交网络异常用户剔除

    为保障社交网络用户登陆特征数据分类挖掘过程的安全性, 要将其中的异常用户剔除[8]。根据用户登陆经常一起使用的命令之间存在紧密性, 不常组合使用的命令之间关系疏远的理念, 设定社交网络用户登陆活动与合法用户不同, 其组合命令方式不同这一条件。计算社交网络用户登陆两个命令间紧密度, 将所得值视为判断异常用户阈值, 如果紧密度高, 则表示该用户为正常用户, 反之为异常用户, 将其剔除。

    在社交网络用户登陆中两个命令共现的程度叫作紧密度, 在此, 利用滑动窗口对用户登陆命令间紧密度进行度量。社交网络用户历史操作记录为一串shell命令构建的序列。通过滑动窗口对命令序列进行采集, 经常一起组合使用的命令会在一个窗口出现。假设对滑动窗口进行编号, 同时对各命令出现在的窗口进行统计, 则各命令会各自对应一个特征向量, 并表示已经出现的窗口, 那么向量间相似程度即为社交网络用户登陆命令间紧密度。

    设置与滑动窗口有关的两个参数:滑动窗口大小s, 其为窗口可以容纳命令的数量。窗口变化步长l′, 其为窗口每次滑动命令数量。在窗口进行滑动时, 统计窗口各条命令出现次数。设定训练数据长度为L′, 训练数据中全部命令集合为Σ, 不同命令数量为m=|Σ|, 窗口数量为n′= (L′-s) /l′+1。社交网络用户登陆特征数据能够表示为m×n′的矩阵V′, 其表达式为

    V´=[u11u12⋯u1mu21u22⋯u2m⋯⋯⋯⋯un´1un´2⋯un´m] =[u1, u2, ⋯, um]          (5)

    其中, V′中各列向量值表示与其相应的命令于不同窗口出现频次, V′中各行向量值表示其相应窗口不同命令出现频次。

    设定与V′中第i′列相应的命令为si′, 其中, i′=1, 2, …, m, 以度量随机两个命令si′和sj′间紧密度为目的, 在矩阵V′生成两个矩阵, 分别如式 (6) 和式 (7) 所示

    B=[bi′j′] (6)

    B′=[b′i′j′] (7)

    式 (6) 和式 (7) 中, B′和B为二值矩阵, bi′j′表示命令sj′于窗口wi′出现与否。同一命令于窗口出现次数比较多, 可以看作该命令和其孪生命令交替出现产生的结果, b′i′j′代表命令sj′孪生命令s′j′于窗口wi′出现与否。

    窗口中平均命令紧密度可叫作窗口聚合度, 假设窗口中命令为合法用户经常使用的, 则窗口聚合度越高;否则, 窗口聚合度越低[9,10]。

    根据式 (8) 的社交网络用户登陆两个命令间紧密度Clo (si′, sj′) 计算公式, 可有效将聚合度低的社交网络用户登陆命令检测出来, 并将此类用户当作伪装用户剔除, 以此保障网络用户登陆特征数据分类挖掘的安全性。

    3.2 基于Leader的用户登陆特征数据分类挖掘

    基于3.1节社交网络异常用户剔除分析, 利用Leader算法对网络用户登陆特征数据进行分类挖掘。过程中, 根据社交网络异常用户剔除结果设定分类挖掘输入集合, 并将输出集合定义为空。针对输入子集, 寻找输出子集, 并给出输入子集和输出子集之间的质心距离最短值, 将最短值与距离阈值进行比较, 并将比较结果分别添至输出子集和输出总集中。给出质心计算公式, 对新用户登陆特征数据分类的属性集合进行计算, 并将计算结果分别添至输出子集合和输出总集中, 完成用户登陆特征数据分类挖掘。详细过程如下:

    社交网络用户登陆特征数据分类为利用数据子集的相似程度实现数据分类挖掘的一个过程。

    将输入定义为已将异常用户剔除后的集合Clo (X) , 将输出定义为社交网络用户登陆特征数据分类挖掘结果L (X) ={X1, X2, …}, Xi″∈Clo (X) 。

    根据上述定义, 假设L (X) 初值为空, 针对Clo (X) 中各登陆特征Clo (x) 做以下操作:

    寻找类别L (x) , 并使Clo (x) 与L (x) 质心距离d′ (Clo (x) , L (x) ) 最短, 将其记作d′min。

    假设d′min比距离阈值Dis小, 那么将Clo (x) 添加至L (x) , 反之将{Clo (x) }添加至L (X) 。

    根据以上分析, 能够通过平均值对质心进行计算, 假设有类别L (x) ={Clo (x) 1, Clo (x) 2, …, Clo (x) k}, 那么L (x) 质心g属性集合可表示为

    F (g) =F (Clo (x) 1) +F (Clo (x) 2) +⋯+F (Clo (x) k) k (9)

    式 (9) 中, F (Clo (x) i″) 代表社交网络用户登陆特征数据属性集。

    利用式 (9) 计算, 设当前有一个新用户登陆特征数据p加入至社交网络用户登陆特征数据分类中, 那么分类的特征属性集合能够通过式 (10) 计算得到

    F (p) =F (g) -1k+1 (k⋅F (g) ) (10)

    其中, F (p) 代表质心p属性集合。将式 (10) 所得结果按照上述过程分别添至L (x) 和L (X) , 迭代此过程, 最终输出的L (X) ={X1, X2, …}, Xi″∈Clo (X) 即为基于Leader的用户登陆特征数据分类挖掘输出结果。

    基于Leader的用户登陆特征数据分类挖掘方法与当前相关方法相比, 面对用户的加入, 动态性地实现用户登陆特征数据分类挖掘, 大大提高了运算效率和分类准确性。

    社交网络用户登陆存在一定隐含信息, 对该网络用户登陆特征数据进行分类挖掘十分必要。当前相关研究方法和算法存在一定问题, 提出基于Leader的社交网络用户登录特征数据分类挖掘方法。利用异常用户剔除为网络数据安全分类奠定基础, 通过Leader分层算法实现数据分类挖掘。通过实验验证了所提方法的整体性能, 可为该领域发展提供支撑。
