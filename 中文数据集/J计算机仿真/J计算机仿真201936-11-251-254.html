    摘    要：

    深网作为网络数据的载体,其中包含了大量的网络数据,可为科学研究及挖掘应用提供优质的分析数据集。但非合作结构化深网数据亦是异构的,使得传统方法挖掘结果普遍存在着挖掘精度较低、挖掘时间较长、内存占用率较高等问题。提出基于分层抽样的深网重叠数据特征自动挖掘方法。通过对非合作结构化深网重叠数据进行分析,采用贝叶斯网络方法确定相应的标签,建立动态全局模式,在此模式下,利用元组分层抽样的方法对深网重叠数据特征进行分层抽样,实现对深网重叠数据特征自动挖掘。实验结果表明,所提方法挖掘精度较高、挖掘时间较短、内存占用率较低。

    关键词：

    非合作结构化; 深网重叠数据; 挖掘; 全局模式;

    Automatic Mining Simulation of Overlapping Data Features in Non-Cooperative Structured Deep Networks

    YANG LU-han

    School of Management, Xi′an University of Architecture and Technology

    Abstract：

    As the carrier of network data, the deep web contains massive network data, which can provide high-quality analytical data sets for scientific research and mining application. The non-cooperative structured deep web data is heterogeneous. The traditional mining methods generally have low mining precision, long mining time and high memory occupancy. Therefore, an automatic mining method for deep web overlapping data feature based on stratified sampling was presented. By analyzing the overlapping data in non-cooperative structured deep web, we used Bayesian network method to determine the corresponding labels and build a dynamic global model. On this basis, we used the tuple stratified sampling to sample the characteristics of deep web overlapping data. Finally, the automatic mining of deep web overlapping data features was achieved. Simulation results prove that the proposed method has higher mining precision, shorter mining time and lower memory utilization.

    Keyword：

    Non-cooperative structure; Deep web overlapping data; Mining; Global model;

    1 引言

    深网是指一些保存在网络数据库中,不能利用链接来进行访问,只能通过网页技术进行访问的资源集合[1]。深网数据根据保存的方式不同,可以将它分为文本型和非合作结构化两种类型,对文本型深网数据的研究已经取得了一定的成果,但是对非合作结构化深网数据的研究正处于起步阶段[2,3]。当前研究的重点主要偏向于查询和数据之间的相关性问题,往往忽略了数据之间的重叠关系,导致不同的深网数据源上一样的数据被重复的进行查询,增加了一定的工作负载。在这种情况下,如何有效提出一种具有较高挖掘效果的深网重叠数据特征挖掘方法成为当今社会亟待解决的问题[4,5]。

    为了能够避免深网重叠数据特征带来的困扰,很多专家学者们提出一些关于深网重叠数据特征挖掘的方法。例如,基于动态调度的深网重叠数据挖掘方法。该方法在考虑运行环境和负载情况等多方面因素的基础上,以规模最小的深网重叠数据集作为挖掘单元,并将每个挖掘单元之间利用全局通信方式进行连接,降低了深网重叠数据挖掘成本。基于改进FSVM的深网重叠数据挖掘方法。该方法要先选择合适的支持向量,减少深网重叠数据训练样本数量,还要定义一个新的隶属度函数,来加强支持向量构建模糊支持向量机,最后根据邻居深网数据样本密度来完成分类。该方法具有较高的分类正确率。基于关系矩阵的深网重叠数据挖掘方法。该方法对深网重叠数据进行关系矩阵分析,并根据它们之间的关联性来挖掘深网重叠数据集。该方法具有较好的深网重叠数据挖掘精度。基于环路紧密度的深网重叠数据挖掘方法。该方法根据环路间密度值来对深网重叠数据进行分类,并将最紧密的核最为归属,实现对深网重叠数据的挖掘。该方法具有较高的分类精度。

    上述几种方法都具有一定的重叠数据挖掘效果,但是挖掘过程中还存在着挖掘时间较长的问题。

    因为在深网上能够查询到的数据一般情况下都是动态的,不能直接就被利用,所以必须要将深网数据抽取到本地。针对这种情况,文中提出基于分层抽样的深网重叠数据特征自动挖掘方法。采用贝叶斯网络方法确定相应的标签,建立一个全局模式,根据所建立的全局模式,有效的抽取出深网数据,采用分层抽样法对深网重叠数据特征进行挖掘。

    2 深网重叠数据特征自动挖掘方法

    2.1 建立全局模式

    通过对非合作结构化深网重叠数据进行分析,采用贝叶斯网络方法确定相应的标签,建立动态全局模式,可以大幅度提升数据挖掘的准确度,具体实现过程如下:

    全局模式建立的好坏会直接影响到最后的深网重叠数据特征挖掘的效果,其中深网数据特征属性的数量也非常的重要。当数据属性较少时,很大程度上会造成数据的不完整,导致有用的深网数据丢失;相反,当数据属性较多时,又会出现大量的无用深网数据,造成数据重叠或者冗余。针对这种情况,采用贝叶斯网络,对它进行改进,然后在网页中选取出重要的深网重叠数据的属性,来确保做到全局模式。

    利用公式(1)给出贝叶斯公式

    Ρ(Y|X)=Ρ(Y|X)Ρ(Y)Ρ(X) (1)

    式中,X、Y代表一对随机变量。

    贝叶斯网络是一个联合概率分布,它可以提供出深网数据因果关系的结构,并在此基础上进行学习。当这个网络结构中所有的数据都是给定,就可以直接进行计算,但是如果深网数据是隐藏的,只是存在这样的关系时,就需要对其进行概率估算。

    在一般情况下,在网页上会有多个深网数据属性,多个深网数据属性之间存在一定的关联,并且有的深网数据属性出现的频率较多,利用贝叶斯网络求出深网数据属性出现的概率,(C1,C2,…,Cd)代表深网数据所有变量,CG(j)代表G中第j个区域内最高的变量;π(CG(j))={CG(1),…,CG(j-1)}代表在CG(j)之前的所有深网数据变量集合,从π(CG(j))中删除对Cj不会存在任何影响的深网数据变量,得到第j个深网数据变量的概率P(Cj),并将它作为深网数据全局模式的变量。以此为基础,利用公式(1)求出深网重叠数据最小支持概率Pmin

    Ρmin=1-Ρ(C1|C3)Ρ(C1)×Ρ(C2|C3)Ρ(C2)×⋯×  Ρ(Cd-1|Cd)Ρ(Cd-1)×Ρ(Cd|C1)Ρ(Cd)         (2)

    当深网重叠数据所有变量相互之间没有任何的关联时,这时Pmin=0。

    当深网重叠数据所有变量之间都保持着紧密的联系时,即概率大于Pmin的深网重叠数据属性变量,这时G={(C1,C2,…,Ci…,Cn)},从中选择出深网重叠数据的先验概率的变量。

    通过上述方法对深网重叠数据进行预处理之后,还需要对每个深网重叠数据特征属性设定权限,可以随时产生全局模式,其过程如下:

    初始化4个深网重叠数据集合,分别为L、I、P、M;使它们都为空,然后建立一个深网重叠数据查询集合,把输入的深网重叠数据特征属性放在集合L中,在把深网重叠数据查询集合中的所有元素输入到深网重叠数据特征属性中,将深网数据实体放入集合I中,并对其进行记录,反复重复上述步骤,直到得到一个全局模式。

    2.2 全局模式下深网重叠数据特征挖掘

    以2.1节建立的全局模式为依据,利用元组分层抽样的方法对深网重叠数据特征进行分层抽样,具体过程如下:

    1)确定深网重叠数据抽样层次

    选取深网重叠数据特征属性分层集合,可以根据信息增益的方法对深网重叠数据特征属性进行排序,对于深网叠数据特征属性分层集合利用数据特征属值域发现方法,得到所有的深网重叠数据抽样层次{x0,…,x|d(Φ)|-1}。由于各深网重叠数据集合数量不同,所以对不一样的深网重叠数据的抽取数量是不一样的,利用误差约束的方法来控制深网重叠数据抽样的数量,保证每一层中抽取的深网重叠数据数量都能够保证在抽取结果误差在所给范围内。

    2)确定每层深网重叠数据样本量

    给出x∈{x0,…,x|d(Φ)|-1},d(x),查询Q,置信度为0<δ<1,创建指示函数,随意选取深网重叠数据集合ti∈d(x)。

    如果ti∈Q(d(x)),Yi=1;否则,yi=1,则Pr[Yi=1]=p=|Q(d(x))||d(x)|,Ρr[Yi=0]=1-|Q(d(x))||d(x)|,则Y1,…,YN为独立的泊松观测。

    当Y¯=∑i=1ΝxYi,μ=E[Y¯]=Νxp,对于随机给定的相对误差ε>0,则

     Ρr[Y¯-E[Y¯]]≥Νxpε≤2exp(-Νxpε2/(2+ε))=1-δ         (3)

    为此,层次x确定抽取的深网重叠数据的样本数量为

    总的深网重叠数据样本数量为N=∑x∈[x0…xd(Φ)-1]Nx,根据公式(4)可知,Nx和|d(x)|成正比,与Q|d(x)|成反比。当Q|d(x)|=d(x)时,能够得到最少数量的深网重叠数据样本,对于层内集合数量大于ΚS(ε)=[(2+ε)ε2⋅Ιn21-δ]的层次来说,所有深网重叠数据抽样数量都是KS(ε),对于层内集合数量小于KS(ε)的层次来说,深网重叠数据抽样数量为|d(x)|。

    3)每层随机抽样选取深网重叠数据样本。

    由于只能通过查询接口来得到深网重叠数据样本,在此引入“下钻”查询方法来得到深网重叠数据样本,并对每个x,从d(x)中随机抽样,构成深网重叠数据样本Sx(d),则每个深网重叠数据集合生成步骤如下:

    (a)给出x,得到查询Q:Φ=x在d上的基数|Q(d(x))|,当|Q(d(x))|>topK时,topK代表网页中最大的输出深网重叠数据集合数量,将其转向(b);否则,转向(c)。

    (b)|Q(d(x))|>topK时,利用“下钻”查询方法产生新的查询Q′:Φ=xAi=vi,其中Ai代表随机选取的深网重叠数据特征属性,转向(a)。

    (c)当|Q(d(x))|≤topK时,从查询结果中任意选取一个集合,如果这个集合已经被选取,就回到(b);否则,将该集合加入样本集合中。

    反复重复上述步骤,直到获得Nx个不同的深网重叠数据集合为止。

    3 实验与仿真证明

    为了验证所提基于分层抽样的深网重叠数据特征自动挖掘方法的综合有效性,需要进行一次仿真实验,实验环境为:IntelCore3-530 24G内存,操作系统为Windows10。在实验中随机准备500个真实深网重叠数据,将所提方法与基于关联规则的深网重叠数据挖掘方法和基于决策树的深网重叠数据挖掘方法进行深网重叠数据挖掘性能对比实验,实验结果如下所示。

    4 结束语

    针对传统的重叠数据特征挖掘方法中,普遍存在着各种各样的问题。针对所存在的问题,提出基于分层抽样的深网重叠数据特征自动挖掘方法。该方法利用贝叶斯网络来构建一个全局模式,在全局模式下,利用分层抽样方法实现对深网重叠数据特征的有效挖掘。实验结果表明,所提方法相对于其它两种方法,可以快速、有效地对深网重叠数据特征进行挖掘。
