    摘    要：

    针对永磁同步电机控制系统中的传统滑模速度控制器输出存在系统抖振, 从而导致转速、转矩超调, 动态响应慢等问题, 设计出一种改进型的滑模变结构速度控制器。首先, 利用模型参考自适应的辨识思想, 以及合适的自适应律实现对电机转速参数辨识, 并作为速度控制器的输入。再通过预设负载转矩上、下界的滑模结构将速度误差与系统状态量的变化相关联, 以实时改进滑模控制率。最后, 变指数趋近律引入, 让系统以变速和指数两种速率实时趋近滑模面。由于滑模变结构的加入, 使速度控制器的鲁棒性大大加强。通过预设负载转矩上下界的滑模结构, 极大的削弱了系统抖振, 提高了电流、转矩波形的平滑性。而变指数趋近律的引入, 缩减了转速, 转矩的超调时间, 且使系统动态响应能力也大大加强。通过在Matlab中搭建模型, 仿真结果表明所设计控制方案在满足控制精度的同时, 减弱了系统抖振, 抑制了状态变量的超调, 提高了系统的鲁棒性。

    关键词：

    永磁同步电机; 滑模变结构; 变指数趋近律; 模型参考自适应; 速度控制器;

    Research on Robust Control of Speed Characteristics of Permanent Magnet Synchronous Motor

    LIU Chang ZHAO Kai-xuan NIU Qing

    Xinyang Vocational and Technical College Institute of Automation and Electrical Engineering, Lanzhou Jiaotong University

    Abstract：

    Aiming at the problems of the traditional sliding mode speed controller output in permanent magnet synchronous motor control system, such as system chattering which results in the overshoot of speed and torque, and slow dynamic response, the paper presented a modified and variable sliding model controller. Firstly, Model Reference Adaptive system and adaptive law were used to identify motor parameters for increasing estimated accuracy of the rotor speed, which acted as the input of the speed controller. Then the speed error was associated with the change in system state quantity via pre-designed up-down boundary of load torques so that the sliding mode control rate with real time was improved. Finally, the variable exponential approaching law was added to cause the system to approach to sliding model surface in two kinds of speed rates. Due to the sliding mode variable structure, the robustness of speed controller was greatly enhanced. By pre-designed the sliding mode structure of up-down load torques, the system chattering was greatly weakened, and the smoothness of the current and torque waveforms was improved. The introduction of variable exponential approaching law, reduced the speed and torque overshoot time, and the system dynamic response capability was also greatly enhanced. By building the model based on Matlab, the simulation results show that the designed control scheme can reduce the chattering, suppress the overshoot of the state variables and improve the robustness of the system while satisfying the control precision.

    Keyword：

    Permanent magnet synchronous motor; Variable sliding model structure; Variable exponential approaching law; Model Reference Adaptive system; Speed controller;

    1 引言

    永磁同步电机 (Permanent magnet synchronous motor, PMSM) 因为其具备结构简单、功率因数高、损耗小等优点在工业生产中中得到广泛的应用[1,2]。但是由于PMSM是典型的多变量非线性系统, 在实际控制中存在很多外界干扰, 如转矩波动, 负载扰动, 温度升高而导致的参数温漂等问题。而滑模控制由于自身对扰动与参数不敏感、响应速度快等优点在鲁棒控制设计中广泛采用[3], 但也正是滑模控制结构特点在带来强鲁棒性的同时, 也造成了系统抖振, 从而阻碍了其在实际工程中的应用。现阶段对滑模结构的改进, 大多利用与各种算法相结合方法, 如积分滑模, 模糊滑模, 神经滑模等等, 而这些方法并未从而导致抖振现象根源上削弱, 从而又带来新的问题。文献[4]为了使滑模控制系统的抖振削弱, 采用神经网络在线预估滑模增益的方法, 该设计虽然能减弱系统抖振, 然而太多参数设计, 使结构变得复杂。文献[5]利用改进型趋近律来降低滑模控制导致的系统抖振, 虽然在一定情况下达到了目的, 但是控制偏差变化为零时, 滑模增益也为零, 对控制系统不利。文献[6]将变结构控制思想和直接转矩控制理论相结合, 使得系统动态性响应更快, 鲁棒性大大加强, 但是这导致变结构切换增益变大, 系统抖振问题依然存在。文献[7,8]在直接转矩控制中引入滑模变结构与指数趋近律, 有效的改善了系统的稳定性, 缺点在于指数趋近律的滑动模态使系统在靠近原点时, 是趋于原点周围的一个振动, 而不是趋向于原点, 增加了系统的负担。文献[9]在SVM调制技术的中, 采用ILC算法作用于新的控制器, 大大增强了永磁同步电机转速和转距的稳定性, 但是由于在SVM中, 磁链的空间角度的变化是连续的, 但是电压矢量的变换却是步进式的, 这样就会导致电机转矩的脉动。文献[10]为解决永磁同步电机中速度控制问题, 利用多滑模面切换处理的思想, 但过多的滑模平面切换实现存在困难, 同时也会导致系统鲁棒性减弱。文献[11,12,13]采用模糊智能控制策率, 不足之处在于在线模糊推理的计算量大, 难以实现。文献[14,15]为实现无速度传感器控制, 且使系统具备优越的动态响应能力, 将滑模结构与自适应观测器相结合, 同时也提高了系统对外界干扰的鲁棒性。

    本文采用预设负载转矩边界的滑模结构与改进型趋近律相结合的速度控制器来取代原有的滑模结构指数趋近律的速度控制器。该速度控制器从产生抖振的滑模控制器函数根源出发, 将负载转矩上下界通过数学公式推导巧妙引入, 使速度误差与系统状态量变化相关联, 以实时改进控制器函数。首先, 建立PMSM的数学模型, 分析优化控制的难点问题, 然后针对难点问题, 提出了改进型滑模速度控制器, 并对该控制器中算法难点技术过程做出详细的说明。最后通过对仿真结果对比分析, 验证了提出算法的可行性。

    5 结论

    本文拟选用基于负载转矩上下界的滑模函数并与改良后的指数趋近律相结合, 同时引入模型参考自适应的控制思想。通过预先设计负载转矩边界值实时改进滑模控制器函数, 实现对参考转速与误差速度的跟踪控制, 利用改进型指数趋近律来实时改变趋向滑模面速度, 并结合模型参考自适应控制策略来实现对电机转速无传感器精准估算, 以改善进入滑模面初期阶段系统振动。整个系统动态响应快, 极大的缩减了原系统变量的超调时间, 大幅度削弱了系统振动, 对外部扰动也有很强的适应性, 曾强了鲁棒性。
