    摘    要：

    针对传统方法数据单向隔离过程中存在隔离率较低、误隔离率较高、数据可用性较差的问题, 提出了基于事务关系的信息广域网不可信数据单向隔离可视化方法, 该方法依据数据影响关系确定受恶意事务影响的不可信数据集合;当网络子节点收到主节点发来的访问申请时, 根据节点级隔离检测协议判断该访问申请数据对象是否已被隔离, 并将隔离检测结果以消息的形式发送给网络主节点;如果申请通过则依据可疑失误隔离协议限制数据库可疑事务访问行为, 对受恶意事务影响的不可信数据集合执行单向隔离。仿真测试结果表明, 提出方法与其他隔离方法相比, 具有较高的单向隔离效率, 同时在降低信息广域网不可信数据误隔离率的同时, 保证了数据的高可用性。

    关键词：

    信息广域网; 不可信数据; 单向隔离; 可视化;

    Information wide area network untrusted data one-way isolation visual simulation

    WU Meng-xuan JIANG Tian-yu TONG Sheng-wei ZHANG Li-lai

    China Agricultural University LEETC College of Electrical and computer Science, Jilin Architecture University

    Abstract：

    Aiming at the problems of low isolation rate, high error isolation rate and poor data availability in the traditional data isolation method, a new visualization method based on transaction relation is proposed for untrusted data isolation in WAN. The method determines the set of untrusted data affected by malicious transactions according to the data influence relation. When the network sub-node receives the access request from the master node, it determines whether the access request data object has been isolated according to the node level isolation detection protocol, and sends the isolation detection result to the network master node in the form of message. If the application is passed, restrict database suspicious transaction access based on the suspicious fault isolation protocol access behavior to perform one-way isolation on an untrusted set of data affected by a malicious transaction. The simulation results show that the proposed method has higher unidirectional isolation efficiency than other isolation methods, and at the same time reduces the false isolation rate of untrusted data in the wide area network (WAN) and ensures the high availability of the data.

    Keyword：

    Information wide area network; Untrusted data; One-way isolation; Visualization;

    1 引言

    计算机技术的飞跃发展使人类迈入了信息时代, 互联网应用的普及更是大大推动了人类信息化进程。在网络高度互联的当今社会, 数据资源由于具有可处理、可应用、可再生、可共享、可传播等多种特性, 已经逐渐成为促进国民经济快速发展的一种特殊武器[1]。但高度互联的网络给数据资源传输、共享带来便捷的同时也给信息系统带来了许多安全隐患, 给攻击者以可乘之机, 通过拦截、复制、篡改、伪造等恶意手段造成信息系统数据资源的丢失、泄密, 乃至各类病毒的入侵、传播等等, 信息系统安全面临安全挑战。特别是对于一些重要行业, 例如银行、电力系统、军事防御等信息系统而言, 即使短暂停止服务都会造成难以估量的经济损失, 数据库作为信息广域网信息系统的重要组成部分, 担负着集中存储和处理海量数据资源的重任, 其安全性能至关重要, 如何保障数据库安全, 及时隔离信息广域网数据库中的不可信数据已然成为各个领域信息系统亟待解决的关键问题[2,3]。

    文献[4]提出了基于应变式传感器的信息广域网不可信数据单向隔离方法, 该方法通过采用较高频率的周期性方波信号作为信息广域网数据载波信号以及解调信号, 运用调制解调技术对应变式传感器输出的模拟数字信号做单向隔离;同时采用最小二乘回归模型重构应变式传感器单向隔离信号。文献[5]提出了基于蚁群算法的信息广域网不可信数据单向隔离方法, 该方法根据蚁群算法的正反馈特性分析信息广域网不可信数据的属性, 将种群最终收敛到的路径作为不可信数据传输异常路径, 计算异常传输路径上各个节点的O-measure值, 对于大于设置阈值的不可信数据做单向隔离。文献[6]提出了基于邻域散度的信息广域网不可信数据单向隔离方法, 该方法将信息广域网数据点所在邻域的离散度作为不可信测度度量标准, 对于信息广域网中数据点邻域离散度大于设定阈值的不可信数据予以隔离。

    针对上述单向隔离方法存在隔离率较低、误隔离率较高、数据可用性较差的问题, 提出了基于事务关系的信息广域网不可信数据单向隔离可视化方法。

    2 信息广域网不可信数据单向隔离可视化研究

    2.1 信息广域网不可信数据的确定方法

    信息广域网恶意事务的非法修改会造成网络数据库中原本可信数据受到攻击, 变得不再可信, 如果用户访问了不可信数据将会导致损坏大规模传播, 导致更大规模的数据损坏, 由此说明, 实现信息广域网不可信数据单向隔离的首要前提是确定信息广域网数据库恶意事务造成的数据损坏情况, 判断数据可信性[7]。

    DB={x1, x2, ⋯, xn}    (1)

    Η=∪i=1nΤi    (5) ∪i=1n∝i⊆∝Η    (6)

    ①信息广域网数据库事务依赖关系是指对于数据库不同事务Ti, Tj∈H, 如果事务Tj的发生依赖于Ti, 即Depend (Ti, Tj) , 则满足以下条件:

    其中, Ti.Write_Set和Tj.Read_Set分别代表数据库事务T的写集和读集。

    ②信息广域网数据库事务影响关系是指对于数据库不同事务Ti, Tj∈H, 如果事务Ti影响Tj, 即AffectT (Ti, Tj) , 满足以下任意条件:

    1) 对于数据库中的合法事务T, 如果事务T提交读或者写数据d的操作申请时, 满足以下规则R1, 则发送允许申请消息到信息广域网主节点send (true) ;如果不满足以下规则R1, 则事务T提交的读或者写数据d的操作申请失败, 同时发送请求失败的消息给网络主节点send (false) :

    R1:ΙF∃dΝ (dΝ∈DBΝ) ∧⇁∃dΙ (dΙ∈DBΙ) ΤΗEΝsend (true)     (17)

    上式中, dN和dI表示数据库中每个数据对象存在的两种形态, 前者为常态数据, 后者是隔离态数值;DBI和DBN表示不同形式数据对象构成的集合。

    2) 对于数据库中的可疑事务S, 如果其提交读或者写数据d的操作申请满足以下规则R2、R3之一, 则发送允许申请消息到信息广域网主节点;相反则可疑事务S提交的操作申请失败, 同时发送请求失败的消息给网络主节点:

    根据图1～图6的实验结果可以清楚地看出, 当信息广域网数据库事务执行历史集合H中事务数量n不变, 集合H中恶意事务数量m增大时, 文献[5]方法和文献[6]方法的不可信数据单向隔离率逐渐下降、误隔离率直线攀升、数据可用性基本呈线性下降趋势, 提出方法的不可信数据单向隔离率、误隔离率以及数据可用性并没有发生大幅度变化, 单向隔离率为100%, 误隔离率为0%, 数据可用性平均值也在95%以上;相反, 当信息广域网数据库事务执行历史集合H中恶意事务数量m不变, 集合H中事务数量n不同时, 文献[5]方法和文献[6]方法的不可信数据单向隔离率、误隔离率以及数据可用性变化趋势与n不变, m增大时基本一致, 同样, 提出方法的三项度量指标均表现出了明显优越性, 获得了较好的隔离效果。这是由于提出方法在对信息广域网不可信数据执行单向隔离前, 分析了信息广域网数据库事务依赖关系、事务影响关系以及数据影响关系, 确定了不可信数据度量方法, 获得较低误隔离率的同时, 保障了更高的数据可用度。

    4 结束语

    数据库作为信息广域网信息系统的重要组成部分, 担负着集中存储和处理海量数据资源的重任, 其安全性能至关重要。据相关统计资料显示, 近几年数据库入侵比例占所有入侵比例的四分之三左右, 由于数据库直接存储了大量有价值的信息资源, 成为攻击者的主要攻击目标, 为了保证数据库的可用性、完整性以及用户访问服务的连续性, 需要及时对信息广域网不可信数据进行准确隔离, 提出了基于事务关系的信息广域网不可信数据单向隔离可视化方法, 通过仿真对比测试, 取得了比较理想的隔离效果, 为及时阻断可疑访问, 保证信息系统安全奠定了良好基础。
