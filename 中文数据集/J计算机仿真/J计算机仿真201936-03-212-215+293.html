    摘    要：

    针对当前方法无法兼顾多媒体交互网络用户私密信息隐藏有效性以及在提供用户私密信息隐藏和查询服务质量方面的高效性的问题, 提出一种基于PR-四叉树的用户私密信息智能隐藏方法, 利用信息熵、条件熵和互信息测量多媒体交互网络用户隐私偏好, 得到用户私密信息访问者属性-群组之间的相关性。在了解用户隐私偏好基础上, 采用PR-四叉树时空隐匿算法, 设置一个用户私密信息队列和一个双向映射表, 网络匿名服务器需要抽取出访问者的坐标位置, 并判断其是否在用户私密信息敏感区域, 如果是, 则认为访问者所在区域是否大于最小隐藏区域, 如果访问区域小于用户敏感区域, 服务器将采用敏感区域代替设置的最小隐藏区域, 并进行适当扩展, 即可得到用户私密信息智能隐藏约束区域, 实现所有用户私密信息的智能隐藏。仿真测试结果显示, 所提方法不仅在用户私密信息智能隐藏方面具有较好的性能, 而且在提供用户私密信息隐藏和查询服务质量方面具有高效性。

    User Privacy Information Hiding Method Simulation of Multimedia Interactive Network User

    CHEN Zhi-bo WANG Li

    School of Computer Engineering, Shangqiu University

    Abstract：

    This article puts forward a method for intelligently hiding user private information based on PR-quadtree. At first, information entropy, conditional entropy and mutual information were used to measure user privacy preferences in multimedia interactive network and thus to obtain the correlation between visitor attribute of privacy information and group. Based on user privacy preferences, PR-quadtree space-time privacy algorithm was adopted and a user private information queue and a bidirectional mapping table were set. The network anonymous server extracted the coordinate position of visitor and determined whether it was in the sensitive area of user privacy information. If yes, the server judged whether the visitor area was greater than the minimum hidden area. If the access area was smaller than the user sensitive area, the server used the sensitive area to replace the set minimum hidden area, and then expanded it. Finally, the intelligent hidden constrained area of private information for users could be obtained. Thus, all the intelligent hiding of user private information was achieved. Simulation results show that the proposed method not only has better performance in intelligently hiding user privacy information, but also has higher efficiency in providing user privacy information hiding and querying service quality.

    Keyword：

    Multimedia; Interactive network; User privacy information; Intelligent hiding;

    1 引言

    由于计算机技术与互联网技术的大跨步发展, 各种各样, 功能丰富的诸如电脑、手机、手表等移动智能终端设备的使用变得越来越普及。天涯社区、新浪微博、微信等多媒体交互网络的应用得到了空前发展[1]。就国内现状而言, 新浪微博的成功足以彰显多媒体交互网络服务所蕴藏的巨大市场价值。但是多媒体交互网络的应用和发展在为社会创造巨大经济价值的同时, 也面临许多危机和挑战。用户在登录多媒体交互网络时, 必然会输入个人私密信息, 如果这些用户私密信息被不法分子以非法手段窃取, 甚至是篡改以做它用, 必然会给用户带来不可预估的经济和名誉损失[2]。如何实现多媒体交互网络用户私密信息智能隐藏, 在避免用户私密信息泄露与多媒体交互网络服务质量之间取得较好平衡已经成为一个重点研究课题[3]。

    文献[4]提出一种基于BitTorrent协议的用户私密信息智能隐藏方法, 在多媒体交互网络用户私密信息隐藏前设置容量分析单元模块, 判断用户私密信息隐藏量是否超出隐藏容量上限值;在私密信息隐藏时采用改进的奇偶映射信息编码方式, 将用户私密信息隐藏到非私密信息序列中, 通过利用循环冗余校验的方法提取隐藏信息, 并验证用户私密信息传输过程中是否发生错误。文献[5]提出一种基于R-LWE的用户私密信息智能隐藏方法, 采用R-LWE算法对多媒体交互网络用户私密信息载体明文进行加密处理;通过对用户私密信息单位比特明文在密文空间映射区域的权重以及对应的信息密文进行再编码, 并根据加密过程中用户信息的分布特征进行编码以嵌入私密信息, 实现了多媒体交互网络用户私密信息智能隐藏。文献[6]提出一种基于改进LZW的用户私密信息智能隐藏方法, 该方法通过对LZW算法进行改进, 在多媒体交互网络用户私密信息压缩编码数据空间的细分, 实现了用户私密信息在不同子空间中隐藏不同数量的用户私密信息。

    上述方法无法兼顾多媒体交互网络用户私密信息隐藏有效性与服务质量的高效性, 提出一种基于PR-四叉树的用户私密信息智能隐藏方法。

    3 多媒体交互网络用户私密信息智能隐藏方法

    采用信息熵、条件熵和互信息测量多媒体交互网络用户隐私偏好, 得到多媒体交互网络用户私密信息访问者属性-群组之间的相关性[9];在了解多媒体交互网络用户隐私偏好基础上, 采用PR-四叉树时空隐匿算法, 设置一个用户私密信息队列和一个双向映射表, 多媒体交互网络服匿名服务器需要抽取出访问者的坐标位置, 并判断其是否在用户私密信息敏感区域, 如果是, 则认为访问者所在区域是否大于最小隐藏区域, 如果访问区域小于用户私密信息敏感区域, 为了保证用户私密信息安全性, 系统服务器将用用户私密信息敏感区域代替设置的最小隐藏区域, 并进行适当扩展, 即可得到用户私密信息智能隐藏约束区域, 实现所有用户私密信息的智能隐藏[10]。

    5 结论

    针对多媒体交互网络中用户私密信息保护问题, 提出一种基于PR-四叉树的用户私密信息智能隐藏方法, 理论分析方面阐述了方法的具体执行过程, 并通过仿真测试证明了所提方法在用户私密信息智能隐藏方面的有效性以及在提供用户私密信息隐藏和查询服务质量方面的高效性。

    但是这种方法在应用过程还面临网络攻击、木马病毒等, 下一步的研究工作将针对所提方法的抗攻击性能开展。
