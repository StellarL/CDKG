    摘    要：

    CT图像分割对医学图像辅助临床应用具有重要的临床价值, 是应用最广泛的医学影像技术之一。很多常用的图象处理技术, 并不能自然地适应CT图像的分割。研究基于组件树的CT图像分割算法。通过快速建立CT图像组件树, 利用树结点中携带的属性, 实现树的剪枝、结点合并和树的归约, 完成CT图象目标的分割。实验结果表明, 提出的分割算法实现了分割目标更准确, 运行速度更快的目的。

    A Medical Image Segmentation Algorithm Based on Component Tree

    SUN Xue-guang QIU Wei-gen

    School of computer Science and Technology, Guangdong University of Technology

    Abstract：

    CT image segmentation has important clinical value for clinical application of medical image, which is one of the most widely used technologies. Due to its complexity, many commonly image processing technology can not be adapted naturally to CT image segmentation. This paper studied the CT image segmentation algorithm based on component tree. Through the rapid establishment of CT image component tree, and by using the attributes carried on the node data structure, tree pruning, reduction and node merging were realized efficiently. The experimental results show that this segmentation algorithm realizes the segmentation target more accurately and shorter running time.

    Images; Component tree; Image segmentation;

    1 概述

    图像分割是把图象目标子区域从复杂背景中分离出来, 是更高层次图像处理和分析的基础。近年来, 学者们提出了大量优秀的分割算法[1]。在临床应用中, 医学图象分割技术对计算机辅助诊断、影像引导外科手术、放射治疗等具有重大的意义[2,3]。Frank等人[4]通过阈值优化, 使用最优阈值成功分割出T1加权的大脑MR图像中的灰质、白质和脑脊液流体。文献[5]中, Wang H等将基于区域的分割方法, 成功地应用于海马分割。与普通图像分割相比, 医学图像分割有其应用领域的特殊性, 同时, 在分割精度和速度等方面有更高的要求, 许多在其它领域表现良好的分割方法不能满足医学领域的特殊要求。例如, 基于体素的方法对最优阈值的选择具有一定的主观性, 基于区域的方法不适合于医学图象复杂结构的分割, Snake方法并不适用于肿瘤之类的无规则形状的目标分割。

    近二十年来, 组件树方法已经广泛地应用在医学图象分割算法的研究, 特别是在共聚焦显微镜[6]、皮肤科数据[7]、脑磁共振[8]、CT/MR血管造影[9]等领域。本文研究基于组件树的CT图像分割方法。本文首先使用高斯拉普拉斯 (LoG) 变换对图象进行噪声抑制和细节增强;然后使用浸没算法和并查集算法[10]对图像快速建立组件树;再利用相邻组件属性相似性判定准则对组件进行合并, 达到组件树剪枝和归约的目的;最后输出CT图象分割结果。实验结果表明, 本文分割方法, 时间复杂度低, 算法效率高, 不仅可以有效地去除图象中的噪声, 也抑制了过分割现象, 达到了预期的分割效果。

    2 图象与组件树

    组件树[11,12]利用一簇连续阈值算子获取灰度图象的二值截集, 从而描述灰度图象的特征。设Ω是非空有限集, V是非空有限偏序集, 并假设其有最小值⊥。一个图象是函数f:Ω→V, 集合Ω和V分别称为图象f的支持集和值空间。对任意的x∈Ω, f (x) ∈V称为图象f在点x上的像素值。不失一般性, 假设f-1 ({⊥}) ≠Ø。如果 (V, ≤) 是全序 (或者偏序) 集, 则称f是灰度图象 (或多值图象) 。不妨设对任意x∈Ω, f (x) ≤nmax, nmax∈V。如果∀x∈Ω, f (x) ∈{0, 1}, 则f退化为二值图象。对任意x∈Ω, 记NΩ (x) 为像素点x的邻域。对任意x1, xn∈Ω, 从点x1到xn的路径P (x1, xn) 定义为点的序列 (x1, x2, …, xn) , 其中xi+1∈MΩ (xi) , 任意1≤i<n。子集C⊆Ω称为连通分支, 如果对任意xa, xb∈C, 都存在路径P (xa, xb) , 其上全部点都属于C;并且对任意C′, C′⊇C, 这个性质都不成立。扁平域C是图象f的连通组件, 并且对任意xa, xb∈C, 都有f (xa) =f (xb) 成立[13]。

    下面是部分关于组件树相关概念的定义[14]。

    定义1:设X⊆Ω和v∈V, 阈值函数λv:VΩ→2Ω, 对任意图象f:Ω→V, λv (f) ={x∈Ω|f (x) ≥v}。对任意v∈V, 阈值集λv (f) 上连通分支的全体记为C[λv (f) ]。

    定义2:设X⊆Ω和v∈V, 桶函数C (X, v) :Ω→V, 对任意x∈X, C (X, v) (x) =v;否则C (X, v) (x) =⊥。

    利用阈值函数λv可以将f分解成桶函数, 相反地, 图象f也可以由这些桶函数实现重构, f=∨v∈V∨X∈C[λv (f) ]C (X, v) , 其中∨表示点集函数的逐点上确界函数。由于连通分支X可能属于多个C[λv (f) ], 记m (X) =max{v∈V|X∈C[λv (f) ]}。一般地有m (X) =minx∈X{f (x) }。

    记Ψ={ (X, m (X) ) |X∈∪v∈VC[λv (f) ]}, 在Ψ上定义偏序关系≤: (X1, m (X1) ) ≤ (X2, m (X2) ) ⇔X1⊆X2。由于f是灰度图象, V是全序, 因此必有X1⊆X2⇒m (X1) >m (X2) 成立。由偏序集 (Ψ, ≤) 确定的哈斯图 (Ψ, L) 是一个树结构, 其根结点是上确界R=sup (Ψ, ≤) =Ω, 根树 (Ψ, L, R) 称为f的组件树, Ψ, L, R分别是树的结点集合、根和边的集合, 结点元素 (X, m (X) ) 称为图象f的组件。对任意结点 (X, m (X) ) ∈Ψ, 如果不存在结点 (Y, m (Y) ) ∈Ψ, 使得 (Y, m (Y) ) ≤ (X, m (X) ) , 则称结点 (X, m (X) ) 为组件树 (Ψ, L, R) 的叶结点。对任意结点 (X1, m (X1) ) , (X2, m (X2) ) ∈Ψ, 如果 (X1, m (X1) ) ≤ (X2, m (X2) ) , 则称结点 (X1, m (X1) ) 是结点 (X2, m (X2) ) 的儿子, 或者 (X2, m (X2) ) 是 (X1, m (X1) ) 的父亲。如图1, 图象f组件树构造示意图。

    3 基于组件树的图象分割方法

    组件树最初是在统计领域提出, 后来被纳入到数学形态学的理论框架中, 变成了一种形态学算子。L.Najman和M.courprie, Bertrend等人提出用组件树结构来表示图像, 为组件树的建立奠定了深厚的理论基础[15]。一般地, 在构造组件树的过程中, 它的每个结点上都带有一些定性或定量的例如形状、面积等称为属性的统计信息。这些信息为进一步的图象操作提供基础, 例如树的剪枝, 即所谓的图象过滤。设定剪枝函数FP, 其基于某个属性集上的谓词测试P上, 能够移除所有不满足测试P的结点, 其象素归约到邻近的其它结点上, 从而f的组件树 (Ψ, L, R) 演化成 (Ψ′, L′, R) , 记FP (Ψ, L, R) = (Ψ′, L′, R) 。这个过程称为组件树的归约。由于组件树算法的高效性以及复杂知识处理的应用, 组件树已经被用于设计各种图象的处理算法, 包括图像滤波到图像检索等几乎全流程的图象处理过程中, 并在机器视觉中动作抽取, 特征提取, 分割, 3D视觉等方面得到了很好的应用[15]。

    本文算法包括:图象预处理, 图像组件树构造, 组件规约, 边界判定四个过程, 如图2所示。首先对图像进行预处理, 对图象进行噪声抑制和细节增强。然后使用浸没算法快速构造图像组件树, 将图象处理转化成树结构的处理。再利用树结点携带的相关属性, 对树结点进行剪枝、合并、融合, 达到组件树归约的目的。最后利用边界判定阈值条件确定边界, 输出CT图象分割结果。

    3.1 图象预处理

    本文采用的图像预处理方式为高斯-拉普拉斯 (LoG) 变换, 实验结果如图4所示, 经过LoG变换后的图像噪声得到有效抑制, 图像细节得到显著增强。高斯-拉普拉斯算子为

    LoG=x2+y2-2σ2σ4e- (x2+y2) /2σ2    (1)

    σ越小, 增强效果越好, 反之, 平滑效果越好。在增强过程中应该选择合适的σ值, 图4所示的实验中σ取值为0.8。

    3.2 组件树的构造

    组件树的构造算法主要分为三种:浸没算法、流型算法、基于合并的算法, 本文采用浸没算法快速构造组件树。在构造组件树过程中, 本文使用并查集算法[10]实现路径压缩 (图3 (d) ) 。并查集算法由于本身的时间复杂度低, 算法效率高的特点广泛应用于连接操作。如图3 (a上) 所示, 简单图f为四连接, 图像f的所有像素点是R上的全序关系, 并按灰度值降序排列。灰度值相同的像素点按照视频扫描顺序排列。R表示用字典编码对每一个像素点进行标记, 如图3 (a下) 。图像f不同灰度下的组件区域可以用组件树来表示。为了避免数据冗余, 本文在为图像建立组件树的过程中, 树的每个节点只保存Lk (x) -Lk+1 (x) 的像素点。如果组件树的一个节点存储了多个像素点, 选择最后一个像素点代表该节点。

    图3 组件树构造过程   下载原图

    并查集是一种树形结构, 又叫“不相交集合”, 保持了一组不相交的动态集合, 每个集合通过一个代表来识别, 代表即集合中的某个成员, 通常选择根做这个代表。并查集算法主要分为以下三个步骤:

    MakeSet (x) :建立一个新的集合, 其唯一成员就是x, 因此这个集合的代表也是x, 并查集要求各集合是不相交的, 因此要求x没有在其它集合中出现过。

    Find (x) :返回能代表x所在集合的节点, 通常返回x所在集合的根节点。

    Link (x) :将包含x, y的动态集合合并为一个新的集合。合并两个集合的关键是找到两个集合的根节点, 如果两个根节点相同则不用合并;如果不同, 则需要合并。

    从图3 (b上) 可以看到, 图像f在水平截面V=4中有二个组件, 分别为 (A, C) , B;由于A, C同属于一个节点, 按照视频扫描顺序选择C作为结点 (A, C) 的代表元素。图3 (b下) 在截面V=3中有两个组件, 分别为 (E, D) , F, 选择E为结点 (E, D) 的代表元素;V=3与V=4形成包含关系, 为了避免数据冗余, 所以在水平截面V=3中只包含了灰度值为3的组件区域。图4 (c上) 表示, 水平截面L2中有两个组件H, G。L1中有一个组件 (J, I) , J为结点 (J, I) 的代表元素。由于层与层之间存在的包含关系, 将每一层结点以树结构形式连接起来构成组件树 (图3 (d右) ) 。

    3.3 组件归约及边界判定

    起初构造的组件树包含了大量的相似组件。这些相似组件大都属于同一个目标区域。为了更准确的分割出图像中的目标, 需要进行进一步的组件归约。组件归约应该基于两个基本原则:①两个组件之间的灰度差;②组件内部像素间的差异。在这两个原则以及组件树结构的指导下, 本文提出了基于组件灰度相似性和面积近似性组件归约准则。

    图像组件树中结点的面积和灰度值需要被统计。根据相邻结点之间的灰度差异和结点的面积进行组件规约。T表示构造的图像组件树, C表示组件树的结点集合, K表示图像灰度值集合。

    1) 结点面积计算:

    Compute_Area (T, C, k)

    1 area=0;

    2 for each p∈Cikarea=area+1;

    3 return area;

    2) 查找结点所处的灰度值:

    Find_Level (T, C, k)

    1 ∀p∈Cik, parent (p) ;

    2 return L (parent (p) ) ;

    3) 组件树归约:

    Merge_node (c1, c2)

    1 parent (c1) ←parent (c2) ;

    2 area (c1) =area (c1) +area (c2) ;

    3 return c1;

    定义两个组件C1, C2之间面积变化率, 灰度变化率分别为:

    DArea=|Area (C1) -Area (C2) |max (Area (C1) , Area (C2) )     (2) DLevel=|L (C1) -L (C2) |max (L (C1) , L (C2) )     (3)

    DArea取值越接近1, 表示两个组件的面积差异越大, 同理, DLevel取值越接近于1表示两个组件的灰度差异越大。综合考虑面积差和灰度差可以避免了将噪声作为独立组件而被分割出来, 从而实现图像准确分割。

    diff (C1, C2) ={1ifDLevel≥ΤLandDArea<ΤA0elses    (4)

    其中, “1”表示存在边界, “0”表示不存在边界, TL和TA分别表示灰度变化率和面积变化率阈值。在试验中, 分割的图像不同, 二者的取值也发生变化。

    4 本文算法描述

    下面给出图像组件树建立之后, 基于灰度和面积二重属性的组件合并的算法描述。

    Input:组件树T, 阈值TA, TL

    Output:合并后的组件树T′, 分割完成的图像f ′

    1) 根据输入的图像组件树T, 计算所有灰度级中每个组件的面积。

    for k←10 to 250

    for each cki∈Ckdo Compute_Area (T, C, k)

    2) 依据合并准则, 按照灰度级递减的顺序进行组件合并。

    for each cki∈Ck

    do Find_Level (T, parent (cki) , k)

    if DLevel (cki, parent (cki) ) ≥TLand DArea (cki, parent (cki) ) <TAcontinue;

    else

    merge_node (cki, parent (cki) ) ;

    3) 根据新的组件树T′, 输出分割后的图像f ′。

    5 实验结果分析

    本文设计两组实验, 第一组实验选用比较典型的大脑CT图像进行分割, 第二组选用某患者的胸部CT进行分割。实验中选取的TA的取值范围为 (0.10～0.20) , TL取值范围为 (0.50～0.60) 。模糊聚类算法在医学图像分割中是一种出色的算法, 广泛应用于CT图像分割[16]。Canny算子是最经典的基于边缘的分割算法[17]。所以本文的分割方法与模糊聚类算法和Canny算子进行对比。

    6 结语

    本文研究基于组件树方法的CT图像分割算法。考虑到图像本身灰度模糊性的特点, 在预处理阶段首先采用LOG增强, 使图像增强, 同时噪声降到最低。然后利用浸没算法和并查集算法快速构造图像的组件树, 并在树结点数据结构中预先安排了所需要的属性数据, 再属性相似性判定准则实现对组件合并、树剪枝和结点归约, 最后实现图象分割。实验结果表明, 本文分割方法, 时间复杂度低, 算法效率高, 不仅可以有效地去除图象中的噪声, 也抑制了过分割现象, 达到了预期的分割效果。本文定义了比较复杂的结点数据结构, 目的是为了以后能扩展到非灰度图象的分割领域, 例如彩色图象的分割。
