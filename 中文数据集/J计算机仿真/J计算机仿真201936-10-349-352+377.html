    摘    要：

    针对当前方法在进行动态分块网页主题信息自动提取是存在提取准确率较低、错误率较高、耗时较长的缺点,采用混合加权方法对动态分块网页主题信息进行自动提取。在对动态分块网页主题信息进行预处理的基础上,构建预处理后动态分块网页主题信息的分层树模型,确定网页主题信息的内在联系,采用二元集合序列描述目标提取的动态分块网页主题信息,计算不同的网页主题信息文本对全网页主题信息的贡献程度;采用空间向量模型描述动态分块网页主题信息特征,并利用混合加强的方法从空间向量模型中提取动态分块网页主题信息。仿真结果证明,采用的方法耗时可控制在0.1s内,对样本数据提取的准确率较高。说明采用的方法能够实现动态分块网页主题信息的准确、高效提取。

    关键词：

    动态分块网页; 主题信息; 自动提取;

    CUI Yan-qing

    Institute of Computer Information, Inner Mongolia Medical University

    Abstract：

    Currently, the method has low accuracy, high error rate and long time consumption in automatically extracting topic information from dynamic partitioned web page. In this paper, the mixed weight method was used to automatically extract topic information from dynamic partitioned page. On the basis of preprocessing the topic information of dynamic partitioned web page, the hierarchical tree model of topic information of dynamic partitioned web page after the pretreatment was constructed, and then the internal relation of topic information of web page was determined. Moreover, a binary set sequence was used to describe the topic information of dynamic partitioned web page of object extraction and calculate the contribution degree of different page topic information texts to the whole web page topic information. Finally, the spatial vector model was used to describe the feature of topic information of dynamic partitioned web page. Meanwhile, the method of mixed enhancement was used to extract the topic information of dynamic partitioned web page from the spatial vector model. Simulation results prove that the time consumption of proposed method is controlled within 0.1 s. Meanwhile, the accuracy of sample data extraction is high. Therefore, the proposed method can achieve accurate and efficient extraction of topic information from dynamical partitioned web page.

    Keyword：

    Dynamic partitioned web page; Topic information; Automatic extraction;

    1 引言

    根据最新统计数据显示,截止到2017年底,全球大约有将近十亿个有效的在线网站,且目前全球最大的搜索引擎“谷歌”已经索引了接近500亿左右的网页,这些数据说明互联网正在高速发展,成为越来越庞大的资源信息库[1]。另外,中国网民的数量也与日俱增,互联网应用的普及范围也越来越广,人们更多的时候是通过互联网网页获取需要的信息资源。但随着互联网动态分块网页数量的爆炸式增长,如何从如此庞大的动态分块网页资源信息库中及时、准确找到人们所需要的信息是当前面临的重点和难点。动态分块网页主题信息提取方法的优劣直接影响到互联网信息资源检索工作的顺利进行[2]。

    针对这一现状,文献[3]提出一种基于融合机制的动态分块网页主题信息自动提取方法,该方法通过“动态分块网页模板库匹配-基于网页模板库匹配的信息筛选-动态分块网页分类-网页主题信息自动提取”四个步骤实现对动态分块网页主题信息无关模板与有关模板的有机融合,从融合后的新的模板库中自动提取需要的动态分块网页主题信息。该方法的提取准确率不高。文献[4]提出一种基于LDA模型的动态分块网页主题信息自动提取方法,该方法通过利用LDA模型对互联网中各动态分块网页主题信息进行混合建模,分别计算动态分块网页主题信息与各个文档段落之间的相似度,根据相似度计算结果,选取相似度较高的前若干个结果作为网页主题信息进行自动提取。该方法的提取错误率较高。文献[5]提出一种基于Kmeans聚类算法的动态分块网页主题信息自动提取方法,该方法利用动态分块网页的主要内容和关键特征词构建Labeled-LDA模型,形成文档-主题特征向量,并利用Kmeans聚类算法对模型中的各个文档进行分类,提取每类文档集中的网页主题信息。该方法存在提取耗时较长的问题。研究提出一种基于混合加权的动态分块网页主题信息自动提取方法,在对动态分块网页主题信息进行预处理以保证提取精度的基础上,采用一个二元集合序列来描述要提取的动态分块网页主题信息,以降低准确提取动态分块网页主题信息的时间消耗。

    2 动态分块网页主题信息自动提取原理

    计算动态分块网页主题信息与STU-DOM树结点的局部相关度和上下文相关度,分别设置动态分块网页主题信息与STU-DOM树结点的局部相关度阈值和上下文相关度阈值[6],利用剪枝器判断态分块网页主题信息与STU-DOM树结点之间的关联度,根据关联度计算结果与相关度阈值的大小进行动态分块网页主题信息提取[7],完成动态分块网页主题信息自动提取原理。

    假设contentlength和linkcount分别代表STU-DOM中对应子树中动态分块网页的总字符数和总链接数;STUcij代表STUi树的第j个子树;linkcount(STUi)代表STUi树的总链接数;contentlength(STUi)代表STUi树的总字符数,STU-DOM树结点的主题信息相关度是指该STU-DOM树结点与动态分块网页主题信息之间的关联程度,主要包括两部分,分别为动态分块网页主题信息与该结点的局部相关度和动态分块网页主题信息与该结点的上下文相关度,其中,动态分块网页主题信息与该结点的局部相关度的计算公式如下

    LocalCorrelativity(SΤUi)=linkcountSΤUicontentlength(SΤUi)   (1)linkcountSΤUi=∑j=1Νlinkcount(SΤUcij)   (2)contentlength(SΤUi)=∑j=1Νcontentlength(SΤUcij)   (3)

    其中,N表示STU-DOM数中包含的结点总数。

    动态分块网页主题信息与该结点的上下文相关度计算公式为

    ContextualCorrelativity(SΤUi)=linkcount(SΤUi)contentlength(SΤUΡi)   (4)

    式中,STUPi代表STUi树中的父结点。

    假设LCm和CCm分别表示动态分块网页主题信息与STU-DOM树结点的局部相关度阈值和上下文相关度阈值,利用剪枝器判断态分块网页主题信息与STU-DOM树结点之间的关联度,如果满足以下式(5),则说明态分块网页主题信息与STU-DOM树结点局部相关,可以进行动态分块网页主题信息提取,如果满足以下式(6),则说明态分块网页主题信息与STU-DOM树结点上下文相关,也可以进行动态分块网页主题信息提取

    Correlativity(SΤUi)≥LCm   (5)ConteltualCorrelativity(SΤUi)≥CCm   (6)

    3 动态分块网页主题信息自动提取研究

    3.1 动态分块网页主题信息预处理

    利用解析器将网页解析成DOM树,分别计算DOM树叶子节点包含的网页主题信息量和非叶子节点包含的网页主题信息量,并通过设置阈值变量判断DOM树中节点包含的网页主题信息量及其相对于子节点包含的网页主题信息量变化情况来判断该节点是否为互联网网页动态分块节点,实现动态分块网页主题信息预处理[8]。

    利用解析器将网页解析成DOM树,分别计算DOM树叶子节点包含的网页主题信息量和非叶子节点包含的网页主题信息量。由于动态分块网页文档信息通常是完整的句子序列,可以通过利用DOM树叶子节点包含的字数以及标点符号贡献值来表征叶子节点包含的网页主题信息量,计算公式如下

    weight=Τext_length*punct_weight   (7)

    式中,Text_length代表网页文档长度,即DOM树节点中文档包含的总字数;punct_weight代表DOM树节点所包含的标点符号贡献值,其计算公式如下

    punct_weight={2,punt_num≥51,2≤punt_num<50.5,punt_num≤1   (8)

    通常情况下,标点符号的引入是为了人们更好地理解网页文档内容,是书面语言表达中不可或缺的一部分,但通常情况下网页主题信息中基本不会有标点符号的出现或者只有极少的标点符号出现,设置其权值为1。

    由于DOM树中非叶子节点本身不包含任何网页文档信息,网页主题信息量主要取决于其包含的子节点信息量,研究计算节点信息量的主要目的是为了识别数互联网网页动态分块节点。

    ①假设M和Mi,i=1,2,…,n分别表示用于描述互联网网页布局的非叶子节点及其子节点,计算互联网网页布局非叶子节M包含的网页主题信息量为:

    weight′=λ1∑Μiweight   (9)

    在DOM树中,越外层的非叶子节点包含的子节点数目越多,为了找到网页主题信息所在的信息块,在计算非叶子节点主题信息量时,为其设置的权值为λ1<1,假设非叶子节点所在层次为m,则λ1=1/m。

    ②对于非叶子节点中包含的重要网页主题信息,其信息量计算公式如下:

    weight″=∑Μiweightλ2   (10)

    这类非叶子节点的作用是对其包含的网页主题信息的有效规范,λ2为该类非叶子节点主题信息量的权值。

    ③对于可能与网页主题信息相关的超链接,其信息量计算公式如下

    weight‴=∑Μiweightλ3   (11)

    其中,λ3表示对应权值。

    根据上述计算,通过设置阈值变量R判断DOM树中节点包含的网页主题信息量及其相对于子节点包含的网页主题信息量变化情况来判断该节点是否为互联网网页动态分块节点,实现动态分块网页主题信息预处理,计算公式如下:

    R=maxiweightweight′λ1⋅weight″λ2⋅weight‴λ3   (12)

    如果DOM树节点包含的网页主题信息量增加到T时,如果阈值变量R小于阀值Q,则说明DOM树节点包含的网页主题信息量不具有明显增加的趋势,此时网页主题信息基本都在DOM树子节点中,则该节点可能为动态分块节点。

    3.2 网页主题信息自动提取

    在上述预处理基础上,构建动态分块网页主题信息的分层树模型,确定网页主题信息的内在联系,采用一个二元集合序列来描述要提取的动态分块网页主题信息,计算不同的网页主题信息文本对全网页主题信息的贡献程度[9];采用空间向量模型描述动态分块网页主题信息特征,并利用混合加强的方法从空间向量模型中提取动态分块网页主题信息[10]。

    假设Score(Hi)表示分层树模型中动态分块网页主题信息Hi的权值,通常情况下是由动态分块网页主题信息Hi的显示权值ES(Hi)和隐式权值IS(Hi)的和构成,计算公式如下

    Score(Ηi)=ES(Ηi)+ΙS(Ηi)   (13)

    式中,动态分块网页主题信息Hi的显示权值ES(Hi)和隐式权值IS(Hi)分别是指显示和隐式动态分块网页主题信息在文档中出现的概率,用TFETE(Hi)和TFITE(Hi)代表,由于显示网页主题信息和隐式网页主题信息可能表示相同的内容,并且具有较大的随机性,为了提高计算权值的准确程度,设置d为显示网页主题信息权值的加权因子,则可得

    ES(Ηi)=ΤFEΤE(Ηi)⋅R+d⋅ΤFΙΤE(Ηi)   (14)

    已知隐式权值IS(Hi)与分层树模型中的子节点相关,则可得

    ΙS(Ηi)=α⋅R⋅∑i=1nScore(Ci)   (15)

    式中,C1,C2,…,Cn代表分层树模型中动态分块网页主题信息Hi的子节点序列;α代表模型子节点的信息量贡献度。

    根据上述计算,采用一个二元集合序列{sk,ck}来描述要提取的动态分块网页主题信息,其中,集合{sk}表示要提取的网页主题信息文本序列;集合{ck}表示对应要提取的网页主题信息文本序列的权值序列,用于描述不同的网页主题信息文本对全网页主题信息的贡献程度,其计算公式如下

    ck=L(sk)×V(sk)×Score(Ηi)   (16)

    式中,L(sk)和V(sk)分别表示对应的文本权重等级和对应的网页主题信息贡献度大小。

    根据上述计算,采用空间向量模型描述动态分块网页主题信息特征,假设{ti,ωi}表示网页主题信息中的关键词/权重序列;tfik表示关键词ti在网页主题信息文本中出现的频率;tfi表示对tfik进行混合加权处理的结果,根据经验可知,以中文词语为例,长度较长的中文词语在语义上的表达能力相对于长度较短的中文词语要强,对网页主题信息本文中每个关键词ti的权重ωi赋予一定的修正系数η,以区分不同的关键词长度,通过采用混合加权方法对网页主题信息本文中每个关键词ti的权重ωi进行归一化处理,实现动态分块网页主题信息自动提取,提取公式如下:

    ωi=η(∑ktfik×ck)ln(Udfi)∑j=1U(η(∑ktfik×ck)ln(Udfi))2   (17)

    式中,U代表分层树模型训练集中包含的动态分块网页主题信息文本总数;dfi代表分层树模型训练集中关键词ti的动态分块网页主题信息文本数目。

    5 结束语

    随着互联网的快速发展和大规模普及,网页主题信息量与日俱增,动态分块网页成为当今人们想要足不出户实现最新咨询获取的一个重要途径,通常情况下人们在浏览网页时,更多关注的是网页主题,为了实现动态分块网页主题信息的准确自动提取,研究提出了一种基于混合加权的动态分块网页主题信息自动提取方法,并通过与其它方法的对比测试,证明了所提方法的有效性。得出一下结论:

    1)从几大门户网站中选取动态分块网页作为实验测试数据集,利用本文方法与文献中方法对信息进行提取发现,所提方法对不同网页主题信息数据提取的准确性始终在90%以上,基本接近100%,优于不同文献中方法15%～20%。

    2)所提方法对动态分块网页主题信息提取的错误率始终在2%以内,较文献中方法降低4%～8%。

    3)相同条件下所提方法提取信息的耗时始终在0.1s内,而文献中方法的提取耗时是所提方法的4～5倍。

    下一步的研究工作将针对动态分块网页主题信息的批量提取,这样不仅能够增加实验的测试数据量,而且也能节省更多的提取耗时。
