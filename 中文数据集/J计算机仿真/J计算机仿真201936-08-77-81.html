    飞行员角运动感知模型研究

    周晓光 赵仁厚 王鹏 张沛帆

    海军航空兵学院飞行模拟训练系 空军工程大学装备管理与安全工程学院

    摘    要：

    为解决持续载荷飞行模拟器角运动感知与视觉感知错配问题, 对飞行员角运动感知模型进行深入研究。首先, 对基于飞行员角运动感知模型的持续载荷控制原理进行了介绍, 分析了角运动感知模型在持续载荷飞行模拟器控制系统中的地位和作用。其次搭建了飞行员角运动感知试验环境, 研究了飞行员角运动感知与旋转线性加速度向量和角加速度之间的关系, 给出了飞行员角运动感知传递函数, 构建了飞行员角运动感知模型。最后, 对所构建的飞行员角运动感知模型进行了仿真验证分析, 仿真结果显示所构建的飞行员角运动感知模型可有效解决持续载荷飞行模拟器角运动感知与视觉感知错配问题。研究结果可为持续载荷飞行模拟器控制算法设计提供理论参考。

    关键词：

    飞行员角运动感知模型; 飞行模拟器; 加速度; 角运动;

    Research on Pilot Perceptual Angular Response Model

    ZHOU Xiao-guang ZHAO Ren-hou WANG Peng ZHANG Pei-fan

    Flight Simulation Tranining Department, Naval Aviation Institute Equipment Management and Safety Engineering College, Air Force Engineering University

    Abstract：

    In order to solve the problem of mismatch exist between angular motions by Sustained-G Flight Simulator pilot through his proprioceptive receptors and visual receptors, the pilot perceptual angular model is researched. Firstly, the control principle of Sustained-G Flight Simulator based on the pilot perceptual angular response model is introduced, the position and function of pilot perceptual angular response model is analyzed. Secondly, the experimental environment for researching pilot perceptual angular response model is constructed. The pilot perceptual angular response to rotating linear acceleration vector and angular acceleration is researched. The pilot response transfer functions is researched based on centrifuge experiments. Finally, the verification simulation is made, the results of the simulation is shown that the pilot perceptual angular response model is effectiveness to solve the problem of mismatch exist between angular motions through proprioceptive receptors and visual receptors. The results of this paper can provide reference for design Sustained-G Flight Simulator control algorithm.

    Keyword：

    Pilot perceptual angular response Model; Flight simulator; Acceleration; Angular motion;

    1 引言

    先进的战斗机所产生的过载最大可达9g[1][2], 加载率最大可达6g/s, 过载持续时间长并可能反复出现。这种持续性高过载可使飞行员出现暂时黑视、空间定向错觉甚至意识丧失, 这将严重地影响海军飞行员的空间感和对战势的判断能力。持续载荷飞行模拟器以载人离心机为运动平台, 可以提供六自由度运动平台飞行模拟器没有的持续高过载模拟能力, 有效拓展了飞行模拟训练内容, 对于提升飞行员复杂海空条件下生理心理适应能力, 提高部队战斗力将起到关键作用[3]。持续载荷飞行模拟器处于动态模拟模式下[4], 飞行员在模拟座舱中控制操纵系统, 通过飞行解算后获得仿真飞机状态信息, 输入到持续载荷飞行模拟器驱动系统产生相应运动, 并驱动视景仪表系统, 为飞行员提供较为真实的过载、运动、视觉、听觉感受[5]。但是由于持续载荷飞行模拟器的结构限制, 进行精确载荷模拟过程中, 会产生与仿真飞机不相匹配的干扰角, 由于干扰角的存在, 使得飞行员前庭功能感知的角运动与视觉感知的角运动不相匹配, 从而造成飞行员眩晕, 飞行员输入控制与输出反馈与仿真飞机不一致等问题, 严重的影响了持续载荷飞行模拟器的应用。

    为解决持续载荷飞行模拟器角运动感知与视觉感知错配问题, 国外诸多学者对持续载荷飞行模拟器控制算法进行了研究[6]。飞行员角运动感知模型是持续载荷飞行模拟器控制算法中的核心模型, 是解决持续载荷飞行模拟器角运动感知与视觉感知错配的关键技术。国外在飞行员角运动感知方面进行了一定的研究, 推进了持续载荷飞行模拟器的发展[7][8]。国内在飞行员角运动感知方面还没有相关研究, 持续载荷飞行模拟器控制算法不能满足飞行模拟训练的要求, 严重制约了持续载荷飞行模拟器的应用。

    本文对飞行员角运动感知进行了深入研究, 分析了飞行员角运动感知与旋转线性加速度向量和角加速度之间的关系, 给出了飞行员角运动感知传递函数, 构建了飞行员角运动感知模型, 提出了弱化持续载荷飞行模拟器精确载荷模拟过程中产生的干扰角和应用旋转线性角速度向量产生所需要的角运动模拟提供了方法, 设计的基于飞行员角运动感知模型的控制算法有效地解决了持续载荷飞行模拟器角运动感知与视觉感知错配问题, 提高了持续载荷飞行模拟器仿真逼真度。论文研究结果可为持续载荷飞行模拟器控制算法设计提供理论参考。

    3 飞行员角运动感知度量

    飞行员产生角运动感知主要有两种方式[9]:一是旋转线性加速度向量;二是角加速度。以滚转角为例, 两种方式不同的组合, 飞行员感知角旋转的方式主要有3种, 如图3所示。图3 (1) 描述的是因为旋转线性加速度向量而产生的角运动感知, 飞行员在座舱内保持不动, 线性加速度向量进行旋转, 那么飞行员将感知自身向反方向左进行了滚转运动。图3 (2) 描述的是因角加速度而产生的角运动感知, 飞行员在座舱内, 线性加速度始终作用于飞行员头-脚方向, 飞行员因受角加速度作用, 与飞机座舱同时向右进行滚转, 飞行员感知自身向右发生滚转。图3 (3) 是旋转线性加速度向量和角加速度共同作用产生角运动感知的示意图, 也是通常状况下飞行员感知角运动环境, 在一个常量过载环境下, 飞行员因受角加速度作用, 与飞机座舱同时向右进行滚转, 飞行员感知自身向右发生滚转。

    图3 飞行员感知角旋转的3种方法   下载原图

    美国海军作战试验中心, 应用持续载荷飞行模拟器对飞行员角运动感知进行了度量实验。实验对象为4名飞行员, 两男两女。飞行员位于持续载荷飞行模拟器暗舱内进行实验。持续载荷飞行模拟器运行基础G值为1.25G, 峰值为2或者3G, 峰值持续时间为20s, 基础G值过渡到峰值和峰值过渡到基础G值的时间为2s或者4s。飞行员位于暗舱内应用感知杆舵进行俯仰角和滚转角控制, 在20s的峰值模拟时间段内, 持续跟踪飞行员关于俯仰轴和滚转轴的感知角度, 飞行汇报所感知的角运动方向。

    3.1 角加速度感知

    第一阶段, 主要是测量角运动感知与纯角加速度之间的关系。在实验过程中, 线性加速度始终作用于飞行员纵轴线上。在测量俯仰角运动感知的情况下, 忽略滚转角运动提示。同样, 在测量滚转角运动感知的情况下, 忽略俯仰角运动提示。那么飞行员分别对俯仰角和滚转角感知与角加速度之间的关系如下式所示[10]

    φ′ap=β12.48SS2+15.0S+50.0    (1) ϕ′ap=α12.48SS2+12.5S+25.0    (2) φap=φ′apG′    (3) ϕap=ϕ′apG′    (4)

    其中, φ′ap和ϕ′ap为在载荷为1G的情况下, 飞行员感知的俯仰角和滚转角, β和α为持续载荷飞行模拟器座舱实际的俯仰角和滚转角。

    ϕap和ϕap为相应的载荷G值下, 飞行感知的俯仰角和滚转角度, 其中G′为延迟1s的G值。

    3.2 旋转线性加速度向量感知

    第二阶段主要测量飞行员角运动感知与旋转线形加速度向量之间的关系。旋转的线形加速度向量通过控制万向节内部环产生。通过对试验数据的整理分析, 构建飞行员俯仰角和滚转角运动感知与俯仰轴和滚转轴旋转线性加速度向量的相应函数如下[11]

    φvp=φv2.66S+2.0S2+10.2S+2.0    (5) ϕvp=ϕv6.66S+5.0S2+10.55S+5.0    (6) φv=sin-1GxG    (7) ϕv=Τan-1GyGz    (8)

    其中, φvp和ϕvp为当持续载荷飞行模拟器产生绕俯仰轴和滚转轴旋转的线性加速度向量角度为φv和ϕv时, 飞行员感知的角运动。

    3.3 总角运动感知

    式 (1) (2) 描述了人体半规管对运动的感知函数, 其中φap和ϕap分别是万向节俯仰角β和滚转角α的比例函数。式 (5) (6) 不仅包含了前庭功能器官或耳石对运动的感知, 还包括触觉和肌肉运动感知, 其中φvp和ϕvp是旋转线性加速度的比例函数。那么飞行员总角运动感知方程为:

    φp=φ′apG′+φvp=φap+φvp    (9) ϕp=ϕ′apG′+ϕvp=ϕap+ϕvp    (10)

    其中, φp和ϕp分别为飞行员对俯仰角和滚转角的总角运动感知。

    5 结论

    针对持续载荷飞行模拟器控制算法设计需求, 论文对飞行员角运动感知模型进行了研究, 基于实验构建了飞行员角运动感知传递函数, 并对飞行员角运动感知模型进行了仿真验证分析, 验证了所构建的飞行员角运动感知模型的正确性和有效性。论文所构建的飞行员角运动感知模型可为设计持续载荷飞行模拟器奠定技术基础。下一步的研究工作, 是在所构建的飞行员角运动感知模型的基础上, 设计持续载荷飞行模拟器控制算法。
