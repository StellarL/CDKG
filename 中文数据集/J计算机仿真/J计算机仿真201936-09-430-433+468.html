    摘    要：

    传统方法在对大数据网络用户进行浏览隐式反馈信息检索时,存在查全率低、准确率不高等问题。针对上述问题,提出一种基于大数据网络的用户浏览隐式反馈信息检索方法。方法通过观测用户在浏览网络页面时所选取的动作来获取隐式反馈信息,并根据这些信息建立用户兴趣更新模型。采用向量来描述用户浏览的网页文档,为各个浏览行为赋予相应的权值,通过该权值从用户的浏览行为推算出用户对某一文档的感兴趣程度,并建立基于用户浏览隐式反馈信息的用户兴趣模型,并利用用户兴趣更新模型对兴趣进行更新,提高检索精度,以此实现大数据网络用户浏览隐式反馈信息检索。实验仿真证明,与传统方法相比,所提方法较能够有效提高检索有效性、查全率、准确率。

    Big Data Network User Browsing Implicit Feedback Information Retrieval Simulation

    QU Juan-juan

    Xinhua College of Sun Yat-Sen University

    Abstract：

    In traditional methods, recall rate and accuracy rate are low. Therefore, this paper puts forward a method to retrieve user browsing implicit feedback information based on big data network. This method obtained the implicit feedback information by observing the action selected by user in browsing web pages. On this basis, the user interest update model was established. Moreover, the method used vectors to describe the web documents browsed by user and gave corresponding weights to each browsing behavior. Based on this weight value, the user's interest in a document can be calculated from the user's browsing behavior. On this basis, our method deduced the user's interest degree on a document by the user's browsing behavior. Meanwhile, we built the user's browsing implicit feedback information and used this model to update the interest, so as to improve the retrieval accuracy. Thus, the implicit feedback information retrieval of use browse based on big data network could be achieved. Simulation results verify that, compared with the traditional methods, the proposed method can effectively improve the retrieval efficiency, recall rate and accuracy rate.

    Information retrieval; Big data network; User browsing implicit;

    1 引言

    随着互联网和信息检索技术的不断提升,网络已近渗透到了日常生活的各个领域。然而当前阶段的信息检索主要侧重于用户检索质量的提高,并未顾及在实际应用中用户信息检索结果的有效性。在此背景下,如何提高信息检索的有效性成为了网络信息查询领域亟待解决的主要问题。

    目前已有很多学者针对此问题进行了深入研究,并提出一些较为合理的信息检索方法。现存的主要信息检索方法包括基于关键词的信息检索方法、基于Solr检索模型的信息检索方法和基于概念格的信息检索方法。其中最常用的是基于关键词的浏览隐式反馈信息检索方法[1]。该方法引入反馈信息构建反馈信息检索图,设计反馈信息检索相关性评分机制,在反馈信息检索图搜索过程中引入反馈信息检索语义约束完成用户浏览隐式反馈信息检索,这种检索方式较为简单,但存在信息检索有效性较差的问题。基于Solr检索模型的浏览隐式反馈信息检索方法[2]是将Solr检索模型中的信息集群作为浏览隐式反馈信息检索引库,将HBase集群作为浏览隐式反馈信息检索存储库,引入信息与索引分离、redis缓存、动态参数调整、动态cache释放等设计,构建了大数据检索模型。该方法检索效率较高,但存在检索查全率和准确率均较低的问题。基于概念格的浏览隐式反馈信息检索方法[3]是采用概念格结构存储浏览隐式反馈信息,然后在信息展示时将概念格结构变更为树形结构,用户可以通过多条路径检索到隐式反馈信息。该方法改善了由高维反馈信息要素之间的紧耦合性造成检索效率低的问题,但检索精度较低。

    针对上述方法存在的缺陷,提出一种基于大数据网络的用户浏览隐式反馈信息检索方法。该方法通过观测用户对网页文档的浏览行为来获取隐式反馈信息,并根据这些信息创建用户兴趣模型,为大数据网络信息检索鉴定了基础。

    2 大数据网络用户浏览隐式反馈信息检索方法

    2.1 获取用户浏览隐式的反馈信息

    通过观测用户在浏览网络页面时所选取的动作来获取隐式反馈信息,并根据这些信息建立与更新用户兴趣模型,具体过程如下:

    通常情况下,对用于统计的文档建立索引的方式是将各个浏览文本描述为特定词的集合。为了有效表示各个词在表征某一类文档时所起所用的大小,需要为用户浏览文本空间内的各个词捆绑一个数值权,捆绑与任一文档di中词的权值可视为di在文档空间中的坐标,也就是说,将用户浏览的某一文档di视为文档空间中的一点,这样di即可描述为从Web文档空间的初始点到任意点的一个向量。文档表示方法的一个最关键问题就是如何为描述文档的词设定权值。当前阶段应用最广泛的生成权值的方案是tf*idf加权方案。tf表示某个词汇在Web文档中出现的次数,tf因文档差异而不同,其作用是衡量该词在Web文档中的重要性。idf表示全局统计数据,该值用于衡量该词在整个Web文档集中分布的广泛性。idf设定为In(N/n),N表示Web文档集中的文档数,n表示某个词汇的文档数。即含有该词的Web文档数量越少,idf值越大,假设Web文档集合中的各个文档均含有该词,则idf的值为0。

    采用基于向量的方法来描述用户浏览的网页文档。任意文档d的描述向量V对应的第i个元素可利用下式计算

    w(d,i)=tf(i,d)*idf(i) (1)

    其中,tf(i,d)表示词频统计数据,即词wi在网页文档d中出现的次数为

    idf(i)=In(N/n) (2)

    其中,N表示网络用户浏览的网页文档集中的文档数量,n表示含有词wi的文档数量。

    在大数据网络信息系统中,假设用户对检索结果的评价是直接通过用户获取的,可将这种反馈称之为显示反馈。显示反馈比较容易实现,但在很多应用中要求网络用户显性地评价浏览的网络页面文档的关联性所产生的副作用会约束用户兴趣模型的构建,降低了整个网络信息系统的可用性。与显示反馈相比,由于隐式反馈仅是和用户对浏览的文档可用性的评价存在相关性,收集过程较为简单,不会影响用户浏览网页行为,所以在支持用户兴趣模型处理方面较比显示反馈更加适用。

    一般情况下,人们均是采用网络浏览器来阅览网络资源,网络浏览器为用户提供浏览和处理文档的命令接口,在浏览过程中用户的浏览行为体现了用户对目标文档的感兴趣程度,可充分利用这些信息建立用户兴趣模型。将用户浏览行为划分为以下几种,即:(审查行为、滑动滚动条(sl)、页面浏览时间(rt);存留类型,存留网页文档(sd)、添加标签、打印网页文档;参考类型,追随超链接)。上述浏览行为能够反映网络用户对某种文档di的感兴趣程度,为了有效区分这些浏览行为所体现的不同兴趣程度,为各个浏览行为v赋予相应的权值Cv,通过该权值从用户的浏览行为推算出用户对某一文档di的感兴趣程度,即

    Ri=∑v∈FCvfv(di) (3)

    其中,∑v∈FCv=1,F={sl,rt,sd,bm,pd,fl},Cv表示划分给浏览行为v的权值,fv(di)表示一个二值函数,当观测到用户对文档di有浏览行为v时其相应的函数值为1,相反则为0。将所得二值函数值Ri应用到创建和更新用户兴趣模型中。

    2.2 基于用户浏览隐式反馈信息的用户兴趣模型

    通过用户显示反馈得到的数据较为全面、具体,由此更新的用户兴趣模型通常较为可靠,但这种方式耗费用户较多时间,导致很多用户不愿对检索结果进行评价,导致用户兴趣模型无法及时更新。对此,还需要追踪更多的用户网页浏览行为来更新用户兴趣模型,为动态追踪、预测用户的浏览需求提供保障。

    因为用户兴趣是通过观测用户对某类网页的浏览行为获得的,所以对用户兴趣的描述应当与浏览的网页文档的描述一致,可采用向量来表示用户对于某类网页的感兴趣程度。一个用户可能对网页中的多类文档感兴趣,则用户的感兴趣模型应当是由多个向量组成的向量集。

    假设用户最多有N个感兴趣区域,各个兴趣领域均能够以m个关键词表示,则可用一个m维的向量V来描述用户的单个兴趣领域。用户的兴趣模型可采用m维的向量集合S来描述。

    将用户兴趣和网页文档均通过向量来描述,采用统一的计算方法进行处理。用户的兴趣是通过观测用户的浏览行为逐渐学习获得的,为了有效掌握用户网页浏览行为信息,需要对Web文档进行预处理,预处理过程就是将Web文档调整成可直接使用的数据库记录。具体步骤如下:

    1)剔除Web文档中无用信息。

    2)创建的用户兴趣模型且仅针对注册用户服务,并将Web文档中的访问记录根据用户账号进行划分整合,将浏览记录分别存入不同的数据库表中。

    数据库中的各条浏览记录都真实的描述了用户每一次浏览行为,也是更新用户兴趣模型的依据。以下给出用户浏览隐式反馈信息更新用户兴趣模型的主要算法。

    1)通常情况下,用户对某类Web文档的浏览存留时间越长,说明用户对某类Web文档D所描述的主题内容越感兴趣。从基于概念的Web文档库中获取文档D所含有的索引词(dc1,…,dci)及其相应的权重,假设索引词dci为用户兴趣模型中的某个兴趣词条,则模型中dci相应的权重w更新式为

    Neww=r*t/α+Ri*Oldw (4)

    其中,Neww表示模型中dci的新权值,r表示dci在Web文档D中的权重,t表示用户对Web文档D的浏览存留时间相应的权值。Oldw表示模型中dci最初的权值,α表示一常数。

    假设,dci不属于模型中的某个兴趣词条,则可将dci视为一个兴趣词条,带入模型中,相应的权重计算式如下

    Neww′=r*t/α (5)

    由上式求解出的权值更新用户兴趣模型将导致兴趣词条迅速增加,需要设定一个阈值b,当式(5)计算获得的Neww′值比阈值b大时,才可将dci视为一个兴趣词条,带入用户兴趣模型。

    2)统计单位时间内用户输入查询窗口的各检索词的次数高于设定的某个值时,假设该检索词不属于模型中的感兴趣词条,可将该检索词视为一个兴趣词条,引入模型中,此时权值可设定为

    W=num/α (6)

    其中,num用于描述某个检索词被输入的次数,α表示一常数。

    3)统计模型中的各个兴趣词条被用户视为检索词最新时间,假设单位时间内用户没有检索到某兴趣词条所表示的信息资源,则该词条在模型中的权值将减小,利用下式更新权值

    Neww″=Oldw*(1-Mdate/α) (7)

    其中,Neww″表示旧的权值,Mdate表示上一次访问至现在的天数,α表示调节参数。

    4)用户对反馈信息检索结果和推荐文档的操作方式反映了用户对不同网页信息的兴趣程度。

    无论采用哪种方式更新用户兴趣模型,均需要进行后续这一步处理。Wi,j表示用户兴趣模型,将Wi,j进行规整化处理,则有Wi,j=Wi,j/∑t=1nWi,t。最终获得更新后的用户兴趣模型Ui,即

    Ui={(Ci,1,Wi,1),…,(Ci,n,Wi,n)} (8)

    通过对比用户兴趣模型的近似度来判断用户兴趣是否相近,利用式(9)确定与某个用户A兴趣相近的用户B,反馈与用户需求相关的检索结果。

    sim(A,B)=cos(A,B)=∑k=1nWak*Wbk[(∑k=1nWak2)*(∑k=1nWbk2)]1/2 (9)

    其中,n表示检索过程中所涉及的关键词数量,Wak表示某个关键词k对模型A的权重,Wbk表示某个关键词k对模型B的权重。假设模型近似度计算结果大于事先设定的阈值,则可认定用户A与用户B的浏览兴趣较为相近,提高了信息检索精度以及在一定程度上简化了信息检索流程。以上最终实现了大数据网络用户浏览隐式反馈信息检索。

    3 仿真证明

    为了验证基于大数据网络的用户浏览隐式反馈信息检索方法的优越性,需要进行仿真,主要是为了检验信息检索结果的有效性、查全率以及准确率。利用实验平台为Windows10系统,3.5GHz主频,8GB内存,利用ImageMatch软件进行仿真。

    浏览隐式反馈信息检索实验数据来源于中文文本信息资料集SPAN2012,从中抽取120个检索信息构成候选检索信息集。

    采用所提方法和基于VIS模型的信息检索方法对中文文本信息资料集进行信息检索有效性对比实验,比较2种不同方法信息检索的查准率和稳定性,对比结果见图1和图2。

    仿真证明,所提方法与传统检索方法在进行信息检索时,所提方法的检索有效性、查全率、准确率更高。

    4 结束语

    传统信息检索方法在对大数据网络用户进行信息检索时,存在信息检索有效性差、查全率低、准确率不高等问题。提出一种基于大数据网络的用户浏览隐式反馈信息检索方法。采用浏览隐式反馈信息检索方法进行信息检索时,信息检索有效性、查全率、准确率均优于传统信息检索方法。

    但是所提方法还存在一些需要改进的问题,未来的研究工作需要致力于改善信息检索的性能和发展信息检索技术的多样性,需要在信息检索领域中进行深入探究。
