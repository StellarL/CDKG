    摘    要：

    针对当前网络输入节点筛选方法筛选准确性差、耗时长的问题, 提出基于人工免疫的复杂网络输入节点在线筛选方法。利用网络节点特征子集的评估函数将网络输入节点数据中的冗余特征去除, 采用INTERACT算法分析网络输入节点属性对称不确定性和属性一致性, 设定判定阈值, 将属性一致性小于阈值的节点数据移除, 实现网络输入节点贡献值的判定。将节点的去冗和贡献值计算结果引入离差归一化中, 完成网络输入节点数据样本集合归一化。将网络取证信息数据流中样本当作抗原, 并将抗原动态作用域的B细胞 (DIB) 直接添加至人工免疫网络。对抗原和人工免疫网络的子网隶属度比较大时的DIB刺激度进行计算, 根据计算结果对DIB的作用域半径进行修正, 克隆和变异DIB, 同时抑制免疫网络, 以此得到新免疫网络。利用新免疫网络实现抗原分类, 设置节点过滤阈值判据, 将抗原数目比判据大的聚类筛选掉。实验结果表明, 上述方法筛选准确率平均为97%, 且筛选耗时短, 具有一定的实际意义。

    关键词：

    大数据; 复杂网络; 输入节点; 筛选;

    Online Screening of Complex Network Input Nodes Driven by Big Data

    HE Kang-le

    Open University of Zhuhai

    Abstract：

    At present, the network input node screening method has poor screening accuracy and high time consumption. Therefore, a method for screening complex network input node online based on artificial immune was proposed. At first, the evaluation function of network node feature subset was used to remove the redundant feature in the data of network input node. The INTERACT algorithm was used to analyze the symmetry uncertainty and attribute consistency of network input node attribute. Then, the decision threshold was set and the node data whose attribute consistency was less than the threshold was removed to determine the contribution value of network input node. Moreover, calculation results of redundancy reduction and contribution value of node were introduced into the dispersion normalization, and thus to complete the data sample set normalization of network input node. In addition, the sample in the network forensic information data stream was treated as the antigen and then cell B (DIB) of antigen dynamic scope was directly added to the artificial immune network. When the subnet membership degree was relatively large, DIB stimulation degree of antigen and artificial immune network was calculated. According to calculation result, the radiance of action scope of DIB was corrected. After that, DIB was cloned and mutated. Meanwhile, the immune network was suppressed to obtain a new immune network. The new immune network was used to achieve antigen classification. Finally, the node filtering threshold criterion was set, and then the cluster that number of antigen was more than that of criterion was screened out. Simulation results show that the average screening accuracy of this method is 97%. Meanwhile, the screening time is short.

    1 引言

    大数据驱动下复杂网络输入节点筛选技术是网络监测领域的热点课题[1]。但复杂网络中的输入节点自身存在隐藏形式多样、攻击形态多样等特点, 要对其进行快速筛选比较困难[2,3]。然而网络安全一直是大众所关心的问题, 为了保证大数据驱动下复杂网络输入节点安全性, 设计并构建在线筛选方法十分必要。

    当前基于D-S理论的网络输入节点筛选方法, 在筛选过程中融合自适应机制, 实现实时网络监测, 将计算代价比较小的数据特征和融合入规则结合, 完成输入节点筛选。该方法存在耗时长、准确性差等问题。该领域研究的不断发展, 相关研究成果逐渐涌现。

    谷川[4]等人提出基于阈值筛选模糊分簇的网络数据汇聚法。过程中, 利用模糊思想构建分簇阈值, 通过阈值水平的高低实现网络结构初始化, 根据剩余能量完成簇头节点周期轮换。确定簇头节点之后, 对剩余能量和Sink节点之间的物理距离进行综合考量, 同时对数据汇聚相关准则进行定义, 数据簇成员经该准则完成数据传输到中心节点的任务, 实现整轮数据筛选过程。实验结果表明, 该法可在一定程度上减少网络开销, 但筛选过程中存在将异常节点当作正常节点的现象, 筛选准确性较差。吴大鹏[5]等人提出基于行为特征的异常行为检测方法。该方法利用网络节点之间静态相似程度以及动态相似程度对网络中各个节点的影响能力进行评价, 同时筛选出可疑的网络节点, 以此对可疑节点异常行为进行观察, 通过马尔科夫模型对女巫节点隐藏的实际身份进行推测, 进而较为准确地对女巫节点进行检测。实验结果表明, 该方法具有一定的异常识别率, 但检测耗时较长。王林[6]等人提出基于多决策属性的网络节点检测方法。将多目标优化问题当作目标函数, 将社会网络与分析方法相结合实现核心节点集的识别, 同时根据各种决策属性对识别结果进行二次评价, 完成网络节点筛选。利用APRA网络与Zachary俱乐部网络完成实验, 结果表明, 该法稳定性较好, 但耗时较长。

    大数据驱动下网络数据海量增长, 复杂网络节点不断增加, 这些输入节点的筛选对于网络安全和运行效率而言十分重要。以上相关算法和方法性能有待完善, 提出基于人工免疫的复杂网络输入节点在线筛选方法。

    2 复杂网络输入节点在线筛选原理

    在复杂网络输入节点在线筛选原理中, 采用KCCA法完成网络节点检测。过程中, 根据典型相关分析法对复杂网络输入节点原始特征进行二次提取, 获取简约且重要的数据二次特征。基于二次特征提取结果, 通过二叉树多分类向量机对网络输入节点隶属类别进行判断, 以此实现复杂网络输入节点在线筛选。详细实现过程如下所示。

    将大数据驱动下复杂网络的状态定义成输出空间Y, X代表采集到的输入特征空间。在两个空间中各选取若干对节点数据样本组建训练样本集合Strain与测试样本集合Stest。则由Strain和Stest构成的KCCA最优化问题可表示为

    Ζ=maxα, βΚxΚyβs.t.ΚxΚxα+καΚx=1, βΚyΚy+καΚyβ=1    (1)

    式 (1) 中, κ代表泛化系数, Kx和Ky代表输入和输出的空间核矩阵, α和β代表输入和输出空间中的相关向量值。

    将Strain的容量设置为γ, 利用式 (2) 和式 (3) 获取式 (1) 的最优解 (λi, αi, βi) , i=1, 2, …, γ。

    β= (Κy+κΙ) -1Κxαλ    (2) λ2α= (Κx+κΙ) -1Κy (Κy+κΙ) Κx    (3)

    其中, I代表γ×γ单位矩阵, λ代表数据特征值。

    因在KCCA法中, 特征值与典型相关度相等, 所以仅选取特征值是正实数的状况, 可得到 (λj, αj, βj) , j=1, 2, …, C, 其中, λ1>λ2>…λC。

    将训练集合Strain原始N维输入数据特征xi, i=1, 2, …, γ, 通过式 (4) 降成仅为C维的二次特征向量, 其中C比N小, 再与特征yi相加, 组建二次特征空间的训练样本集合Ftrain。以此类推, 能够得到二次空间的测试样本集合Ftest。

    Ρx (xi) =xiΚxiα    (4)

    式 (4) 为一个投影过程, 将xi降成仅为C维的二次特征向量。

    将Ftrain当作训练样本, 通过二叉树多分类SVM分类器对Ftest中各待测样本隶属类别进行筛选, 筛选结果如式 (5) 所示

    Y′=SVΜ (λj, αj, βj) ⋅Ρx (xi)     (5)

    式 (5) 中, Y′代表基于KCCA的复杂网络输入节点筛选结果, SVM代表分类器。

    在复杂网络输入节点在线筛选原理中, 虽然对节点数据进行了降维, 但其中的冗余节点并没有进行处理, 导致该法运行耗时控制效果较差。为此, 提出基于人工免疫的复杂网络输入节点在线筛选方法。

    3 复杂网络输入节点在线筛选方法

    3.1 复杂网络输入节点数据预处理

    为解决当前方法中存在的筛选准确性差、耗时长的问题, 对网络输入节点数据进行预处理[7]。过程中, 利用网络节点特征子集的评估函数对复杂网络输入节点数据中的冗余特征去除, 采用INTERACT算法分析网络输入节点属性对称不确定性和属性一致性, 设定判定阈值, 将属性一致性小于阈值的节点数据移除, 实现网络输入节点贡献值的判定。将节点的去冗和贡献值计算结果引入离差归一化中, 完成网络输入节点数据样本集合归一化处理。详细过程如下:

    因网络节点数据的单个特征会被添加至特征子集, 这说明与该特征有关的特征会被忽略, 以此就能够达到去冗目的。式 (6) 为复杂网络节点特征子集的评估函数, 即去冗函数

    ΜS=krcfk+k (k-1) rff    (6)

    式 (6) 中, MS代表包含k个属性的子集S启发价值, rcf代表特征和类别之间联系强度均值, rff代表节点特征和节点特征间联系强度均值。式 (6) 中的分子可视为S的特征和类别之间联系强度反映值, 式中的分母可视为特征中存在多少冗余节点数据。

    在大数据驱动下复杂网络输入节点筛选实际运行中, 每个节点为筛选过程做出的贡献是不同的。INTERACT是依据对称不确定性SU的一种算法, 其可对两个特征x′和y′间信息收益IG与熵H′之间的比值度量进行描述, 以此得到INTERACT中的属性一致性, 进而获取各节点对网络输入节点在线筛选做出的贡献。其中, 对称不确定性中的IG与熵H′间关系可利用式 (7) 和式 (8) 表示

    SU (x′, y′) =2ΙG (x′/y′) Η′ (x′) +Η′ (y′)     (7) ΙG (x′/y′) =Η′ (x′) +Η′ (y′) -Η′ (x′, y′)     (8)

    根据式 (7) 和式 (8) 的计算可知, 除SU, INTERACT中还包含属性的一致性, 则网络输入节点特征一致性表达式为:

    SU′ (x′, y′) =ΙG (x′y′) [Η′ (x′) +Η′ (y′) ]-Η′ (x′, y′)     (9)

    由式 (9) 可知, 复杂网络一个节点特征属性的一致性可反映出节点特征元素对于一致性而生成的影响指标。此算法在寻求节点特征间相互影响程度时, 共包含以下两部分:大数据驱动下复杂网络输入节点特征均经其自身对称不确定性进行降序排列;由这些特征序列中排列在最后的值开始, 逐个进行评估[8,9]。假设式 (9) 计算结果小于设定阈值l, 则表示该值贡献小, 将被移除。通过网络输入节点贡献值的判定, 可直接对具有代表性的节点进行在线筛选, 以此提高筛选效率和准确率。

    数据归一化为网络输入节点在线筛选中的基础工作。为了降低各种因素对筛选结果产生的影响, 将网络输入节点的去冗和贡献值计算结果引入节点数据归一化中, 消除节点数据间不可比性, 提高节点筛选精度。在此, 利用离差归一化法对大数据驱动下复杂网络输入节点数据样本集合进行归一化。其表达式为

    y″=x″-minmax-minSU′ΜS    (10)

    其中, y″代表复杂网络输入节点数据样本归一化结果, x″代表网络输入节点数据样本集合, min代表样本数据最小值, max代表样本数据最大值。

    3.2 基于人工免疫的复杂网络输入节点在线筛选

    将3.1节网络输入节点处理结果代入基于人工免疫的复杂网络输入节点在线筛选中, 完成大数据驱动下网络输入节点过滤。过程中, 将网络取证信息数据流中样本当作抗原, 并将抗原动态作用域的B细胞 (DIB) 直接添加至人工免疫网络。对抗原和人工免疫网络的子网隶属度比较大时的DIB刺激度进行计算, 根据计算结果对DIB的作用域半径进行修正, 克隆和变异DIB, 同时抑制免疫网络, 以此得到新免疫网络。利用新免疫网络实现抗原分类, 设置节点过滤阈值判据, 将抗原数目比判据大的聚类筛选掉。

    假设网络取证信息数据流为X‴={x‴1, x‴2, ⋯x‴j″, ⋯}, 将数据流中样本设置为抗原。

    假设抗原和人工免疫网络的子网隶属度比较小, 则将抗原当作动态作用域的B细胞, 即DIB, 直接添加至人工免疫网络, 构成新一代人工免疫网;假设抗原和人工免疫网络的子网隶属度比较大, 对DIB刺激度进行计算。刺激度作为DIB回应抗原与被其它DIB刺激和抑制的程度, 其表达式如式 (11) 所示

    S‴i‴=ξ∑t′=1wi‴l′ψi‴J′-ζ∑t′=1wi‴l′ψi‴J′    (11)

    式 (11) 中, Si代表第i个DIB于第J′个抗原及其它DIB共同作用下产生的刺激度, ξ和ζ代表刺激与抑制程度, ψiJ′代表t′J′时刻第i个DIB作用域半径, wil′代表t′J′时刻下第l′个DIB隶属第i个DIB的隶属度。

    利用式 (11) 的计算, 对DIB作用域半径进行修正

    ψi‴J′=12wi‴l′di‴l′ξ∑t′=1wi‴l′-ζ∑t′=1wi‴l′S‴i‴    (12)

    式 (12) 中, dil′代表第l′个DIB到第i个DIB之间的距离。

    DIB克隆数量取决于刺激度, 为保持DIB种群具有多样性, 对未成熟及老化的DIB不进行克隆[10]。那么第i个DIB克隆数量可表示为

    Μci‴=ΜcS‴i‴∑S‴i‴⋅ψi‴J′    (13)

    其中, Mci代表第i个DIB克隆数量, Mc代表克隆的规模。

    对克隆出的DIB依据一定概率进行变异, 将变异之后的DIB添加至人工免疫网络。将最大种群规模设置为Mmax, 实现人工网络规模的抑制, 假设变异之后种群规模大于最大种群规模, 那么将DIB依据刺激度值大小进行降序排列, 形成新的人工免疫网络, 假设形成的新网络为Y″ (X) 。

    将全部抗原经Y″ (X) 实现分类, 设置过滤阈值判据Th, 将抗原数目比Th大的聚类滤除, 滤除掉的数据即为大数据驱动下复杂网络输入节点在线筛选掉的节点。则最终筛选结果可表示为

    复杂网络内容丰富, 节点数据多变, 对输入节点进行在线筛选, 可有效增强网络安全性, 提高网络运行效率。通过节点数据去冗、节点贡献值计算和归一化实现网络输入节点数据处理, 利用免疫网络实现输入节点在线筛选。实验结果表明, 该方法运行效率高且准确性好。
