    摘    要：

    针对传统的网络入侵特征检测中,一直存在由于构建入侵特征判断函数不准确,导致检测耗时过长,检测正确率较低、误报率和漏报率较高等问题,提出基于径向基核函数的网络入侵特征检测方法。通过将目标网络进行四元组合,确定入侵特征选择表达式;引入径向基核函数当作支持向量机的核函数,获取参数最优值;将网络入侵特征参数作为约束条件,采用粒子群算法确定支持向量机参数,获取粒子位置,并对粒子群速度进行更新,实现对入侵特征的获取;利用支持向量机,对入侵特征进行分类,当分类平面中出现线性不可分问题时,将其转化为线性优化问题,寻找出最优分类面,并对其进行判断,实现检测。实验结果表明,所提方法检测完成时间较短、检测正确率较高、误报率和漏报率均较低。

    关键词：

    级连层次图; 网络入侵; 特征检测; 支持向量机;

    FU Yao LI Ying

    College of Software, Nanchang University Information Engineering College, Nanchang University

    Abstract：

    In traditional network intrusion detection, the intrusion feature judgment function is not accurate, which results in long detection time and low detection accuracy. Meanwhile, the false alarm rate and false alarm rate are high. Therefore, a method to detect network intrusion based on radial basis function was proposed. The expression of intrusion feature selection was determined by quaternion combination of target network. Moreover, the radial basis function was introduced as the kernel function of support vector machine, so as to obtain the optimal parameters. The network intrusion feature parameters were taken as constraints. The particle swarm optimization algorithm was used to determine the parameters of support vector machine and obtain the particle position. By updating the particle swarm velocity, the intrusion features were obtained. In addition, the support vector machine was used to classify intrusion features. The linear inseparable problems occurred in the classification plane could be transformed into linear optimization problems. Finally, the optimal classification surface was found and judged. Thus, the detection was achieved. Simulation results show that the proposed method has shorter detection time, higher detection accuracy, lower false alarm rate and lower false alarm rate.

    Cascading hierarchical graph; Network intrusion; Feature detection; Support vector machine(SVM);

    1 引言

    网络信息时代的飞速发展,其规模也迅速扩大,网络非法入侵现象也层出不穷,同时种类增多、入侵模式多样,导致对网络安全使用造成不同程度的影响[1],为此对其进行自动检测是网络安全防御的首要任务,其可检测出网络各种来源的入侵类型,是研究网络安全的重中之重,已经得到各界人士的广泛关注,专家学者们利用大量时间进行深入研究,也取得了很多较好的研究结果。

    网络入侵是一种能够破坏计算机和网络系统数据完整性的行为方式,对其进行检测,就是为了检查出网络中是否存在入侵的情况[2],同时也可以将它看作成是一个分类问题,即对网络输入源进行分类,将入侵数据与正常数据进行区分。对此采用支持向量机对网络数据中小样本进行分类,对网络入侵特征进行检测,其检测结果能够和支持向量机参数进行紧密的结合,利用粒子群算法对参数进行优化,以此来提高检测准确性[3]。

    为了提高网络性能,提出基于径向基核函数的网络入侵特征检测方法。实验结果表明,所提方法检测完成时间较短、检测正确率较高、误报率较低、漏报率较低。

    2 级连层次图的网络入侵特征检测仿真

    对级连层次图的网络入侵特性进行检测过程中,很容易出现线性不可分问题,需将其转化为线性优化问题,寻找出最优分类面,对网络入侵特性进行分类,同时还需防止出现局部最优,导致分类不准确,无法实现特征检测。

    2.1 网络入侵特征参数选取

    通过对级连层次图网络进行分析,利用径向基核函数将网络入侵特征转换到高维空间中进行分析[4],获取网络入侵特征参数,具体过程如下所述:

    将目标网络视为一个四元组合,即N=(G,s,t,C),对与目标网络数据流图G关联的层次图L顶点,按照级次序从小到大进行排序。不同级层由左向右排列,相同级别自上而下重新编号并排序,可获得与L同构的同构图L′;将L′作为目标网络数据流图G的级连层次图,并根据目标网络级连层次图进行网络安全的定义。

    假设,在整个级连层次图的目标网络中存在N种网络入侵的方式,并且每种网络入侵方式样本数量为L,目标网络入侵特征集用F={f1,f2,…,fk}来表示。当目标网络入侵特征集中相邻的两个特征之间存在距离较大且方差较小的问题时,为了便于目标网络入侵建模,需构建目标网络入侵特征选择器,可用下式进行表示

    J(i)=|m1i-m2i|σ1i+σ2i (1)

    式中,m1i和m2i分别代表目标网络在第i个入侵特征条件下入侵特征获取的均值;σ1i和σ2i分别fi代表目标网络在第i个入侵特征条件1和2下的si方差值。

    有效提高级连层次图的网络入侵特征检测性能在当前互联网领域中具有重要的意义,为了改善当前网络入侵特征检测中存在的各种不足,提出基于粒子群算法和支持向量机结合的网络入侵特征检测方法,利用径向基核函数将网络入侵特征映射到高维空间中进行计算,获取网络入侵特征参数,构建网络入侵特征数学模型,利用粒子群算法对其进行求解,支持向量机参数寻优的过程中,容易误入局部最优的难题中,为此,需要加入动量项来加快粒子群收敛速度,从而降低网络入侵特征检测的误报率和漏报率,提高检测的准确率,具有较为广泛的应用前景。
