    相似性维数在点云中的应用

    刘妮 张志毅

    西北农林科技大学信息工程学院

    摘    要：

    3D点云配准、分割、识别等任务中都需要获取形状属性,传统的形状属性对尺度变化敏感且计算复杂,表达的几何意义简单。结合分形几何中相似性维数的概念,给出了一种可作为点云模型形状属性的维数定义。首先,求取模型中每个点的k邻域得到一个点集,计算其外接球半径;其次,计算由该点集组成的体积与面积信息,并通过缩放处理解决尺度敏感的问题;最后,利用相似性维数表达式计算点云模型中每个点的维数值,使用该值来表示点云模型的形状属性。实验结果表明,相似性维数具有表达形状的能力,并且能够清晰地表达模型的全局特征。

    关键词：

    相似性维数; 形状属性; 点云模型; 分形几何;

    Application of similarity dimension in point cloud

    LIU Ni ZHANG Zhi-yi

    College of Information Engineering,Northwest A & F University

    Abstract：

    Shape attribute acquisition is necessary in 3 D point cloud registration, segmentation, recognition and other tasks. Traditional shape attributes are sensitive to scale changes, complicated to calculate,and simple to express the geometric meaning. Thus, combined with the concept of similarity dimension in fractal geometry, we propose a dimension definition that can be used as the shape attribute of the point cloud model. Firstly, we obtain a set of points that composed of the k neighborhood of each point in the model, and calculate the radius of the circumscribed sphere of the set of points. Secondly, we calculate the volume and area of the set of points, and solve the scale-sensitive problem by scaling. Finally, the dimension value of each point in the point cloud model is calculated using the similarity dimension expression, and the value is used to represent the shape attribute of the point cloud model. Experimental results show that the similarity dimension has the ability to express the shape and can clearly express the global characteristics of the model.

    Keyword：

    similarity dimension; shape attribute; point cloud model; fractal geometry;

    1 引言

    在逆向工程中,获取点云模型的形状属性非常重要。点云形状一般有如下表达方式:协方差矩阵[1,2,1]、法向量的变化率[3]和曲率[4,5]等。这类方法通常使用邻域信息得到协方差矩阵,能较完整地保留模型的形状信息,需要依据经验与各自需求确定使用方式,没有统一的使用规则。曲率是最重要的形状属性之一,具有旋转不变性,使用便捷,仅能表示物体某一位置的弯曲程度。基于以上问题,我们试图去寻找一种几何意义更加丰富,使用更加便捷的形状属性。分形理论是由Mandelbrot[6,6]提出并建立的理论体系,主要研究欧几里得难以描述的形状。维数是分形几何的中心概念,利用维数可以间接描述形状的复杂程度,来定义点云的形状属性。分形维数常应用于图像处理等方面,主要用于表达图像纹理的粗糙程度[7,8,9,7,8,9]或连续曲面的弯曲程度[10,11,10,11]等,这些分形维数大多只定义在二维平面上,较难扩展到三维空间中。目前基于点云数据的分形维数定义的研究相对较少,大多数定义方法与计盒维数相关。Yang等[12]使用计盒维数对点云模型进行分割,他们将点空间分为若干个网格,计算格子中点个数的增长率作为该片点云的计盒维数。Thielen等[13]对文献[12]的方法进行改进并应用于表面重建,但这种方法更适用于空间信息差别较大的模型,不具有普适性。莫程威等[14,14]将分形维数应用于尺度差异的表征与衡量上,通过计算计盒维数来表示数据的尺度差异性,以实现跨尺度数据拼接。郭彩玲等[15,15]将计盒维数应用于苹果树冠层的分形表示上,预测树冠层的分形维数与光照分布的特点。这种计算方式较常用于整体分形结构明显的物体,如树[15,16,15,16]、矿岩结构[17,17]等。Guo等[18]提出了一种计算点云模型的相似性维数的方法,该维数更适合用于聚类应用。Gionis等[19]提出局部相关维数计算公式。Levina等[20,21]对计算相关维数的方法进行了改进,通常使用某属性的增长率表示维数,计算方法较为繁琐,且主要应用于聚类等应用。本文对相关维数进行改进,计算效率较高,更适合表达局部形状属性。

    2 相似性维数的重定义

    2.1 定义

    本文的维数计算基于相关维数的定义。相关维数使用点密度信息计算得到[19],通过使用直线拟合的方法,将某一数据集密度的对数值与尺度的对数值拟合,获取其斜率作为分形维数值,所需的数据量较大,因而更适合表达数据集的全局形状属性。本文采用面积与体积信息计算得到更适合表达点云的局部形状属性。

    定义1 对于2个相似的三维几何体G1和G2,外接球半径分别为R和r,表面积分别为SR和Sr,体积分别为VR和Vr,则其相关维数可表示为:

    DfS=logR/r(SR/Sr)         (1)DfV=logR/r(VR/Vr)         (2)

    定义1适合判断2个几何体是否相似。若作为形状属性,根据文献[19],需要计算每个点邻域内的面积或体积的变化率,计算复杂度高。因而,结合相似性维数定义[6,6]给出本文如下定义:

    定义2 对于三维几何体G,外接球半径为R,表面积为S,体积为V,则其相似性维数可表示为以外接球半径为底,取三维几何体表面积(体积)的对数,如式(3)和式(4)所示:

    DS=logRS         (3)DV=logRV         (4)

    与相关维数相比,使用式(3)和式(4)表示点云的形状属性,形式更为简洁,使用时仅需要2个参量作为输入,且这2个参量易获取,更适合表达点云的局部形状属性。

    一般情况下,几何体的表面积可表示为:S=k1R2,体积可表示为:V=k2R3,则以式(3)和式(4)计算得到三维几何体G的相似性维数分别为:

    DS=logRk1R2=2+logRk1         (5)DV=logRk2R3=3+logRk2         (6)

    其中,式(5)和式(6)右边的项“logRk1”“logRk2”分别体现了具体尺度变换下的形状参量;面积参量属于三维空间中的二维度量信息,体积属于三维度量信息,因而式(5)和式(6)中的项“2”“3”分别表示度量信息所对应的拓扑维数。

    2.2 尺度问题

    将G等比例放大n倍后变为G′,则G′半径为nR,表面积为S′=k1n2R2,体积为V′=k2n3R3,以式(3)和式(4)来计算相似性维数。

    G′的相似性维数为:

    D′S=lognRk1n2R2=2+lognRk1         (7)D′V=lognRk2n3R3=3+lognRk2         (8)

    由于G、G′仅在尺度上不同,得到的相似性维数中的形状参量不具有相同的值,我们需要解决这类问题。

    从根本上来说,引起相似性维数值不同的原因为尺度的变化,因而可以通过尺度缩放来解决这类问题。将G′缩放到G的尺度上,则半径变为原来的1/n倍,表面积变为原来的1/n2倍,体积变为原来的1/n3倍,则式(7)和式(8)将与式(5)和式(6)相同。

    3 相似性维数计算方法概述

    第2节分别基于面积和体积定义了三维空间中相似性维数的表达式。相似性维数表达式中含有区分形状的参量,因而可将其作为形状描述符应用在点云模型上。

    首先将点云模型中每个点表示为p,找出离p最近的k个点组成集合s;然后计算s的半径r,将s进行三角剖分得到四面体集合P,计算P中四面体体积和V,并且计算s所组成的表面积S;最后经过缩放处理后分别利用式(3)和式(4)计算得到每个点的描述符。为方便描述,在之后的表述中以式(3)计算得到的相似性维数,简称为面维数,以式(4)计算得到的相似性维数,简称为体维数。下面就相似性维数各参量的计算方法具体说明。

    3.1 k近邻的计算

    首先,使用k最近邻KNN(K-Nearest Neighbor)算法得到p的k个最近点,该算法需要确定2个参数,分别为k和距离参数。本文选用的距离参数为点与点之间的欧氏距离;参数k一般选取20～48。现阶段对于k的研究不是很成熟,但影响k的基本要素有:点云的密度、点云数量的多少和采集点云设备的类型[22]。文献[22,23]提到了k选取的具体方法,但都需要点云数据具有额外的几何信息。因而,结合前人的研究结果,本文设计出多组实验来确定k的取值范围。图1和图2分别为使用式(3)、式(4)对应描述符的值所作的灰度图,颜色越浅对应描述符的值越大。对比图1a～图1c、图2a～图2c,当k值较小时,模型的细节信息体现得更明显;随着k值逐步增大,细节部分逐渐消失,整体特征更加明显。因而,在确定k值的过程中,可以根据实际应用的具体需求进行调节。当然,使用时仍然会有一定的限制。经过分析发现,在网格模型中,每个模型的邻接点数量在4～12,k值过低会导致部分相似性维数计算不准确,影响实验效果。

    3.2 r的计算

    尺度不变性是形状描述符需要具备的重要特点,可通过第2节提到的缩放处理来解决这一问题。在选定k值得到点集s后,可计算得到s对应的外接球半径r。为了统一尺度,将点云模型中所有的r都缩放为r的平均值,对应面积与体积的变化与第2节中提到的变化相同。

    3.3 面积与体积的计算

    得到点集s后,计算s所组成的面积,实验模型采用的是网格模型,找出点集所组成的面信息,通过面积计算公式即可得到面积参量。体积的计算稍显复杂,通过三维三角剖分点集得到1组四面体集合,这组集合即为该点集的最小凸几何体,计算这组四面体的体积和作为体积参量。Delaunay三角剖分算法具有空圆特性和最大化最小角特性,这2种几何性质保证了剖分后的点集为最优,本文参考文献[24]中提到的算法。另外,当点集s中所有点处在同一平面时,无法进行三维三角剖分,因而这部分点集所对应的点需要单独处理——将这些点对应的体积值赋为整个模型体积值的最小值(缩放处理后的体积)即可。

    以上计算过程得到了相似性维数所需的各参量值,代入式(3)和式(4),即可得到点云模型中每个点的相似性维数,将其作为该点的形状描述符。

    3.4 点云中的维数特点分析

    在确定了相似性维数在点云模型中的具体定义后,有必要分析面维数与体维数所呈现的异同。面维数与体维数同时受面的弯曲程度的影响:模型中某点附近的面越平坦,面积与体积越小,当r>1时,面维数与体维数越小;当0<r<1时,情况正好相反。

    2种维数在呈现模型细节与全局特征方面各有特点。在确定空间中,r确定,平面具有最小面积,且面越曲折,面积越大,并且无上限,因而面维数取值范围广,无上界;体积是由点集的最小凸几何体求得,因而面越平坦,点集所组成的凸几何体体积也越小,且体积具有上限值,因而体维数取值范围窄,有上界。2种维数取值范围的特点决定了面维数可以兼顾更多的形状特征(如面的弯曲程度),而体维数更注重体现模型的整体特征。此外,体维数忽略模型的细节特征,直接使用凸几何体来计算体积,损失了模型细节特征,也正是因为这种“忽略”使体维数更能体现模型的全局特征,如图2所示。

    此外,相似性维数的表达更适用于点分布相对均匀的模型。对于点分布不均匀的模型,可以通过细分网格化后的点云模型,使得模型中的点分布尽可能均匀,在下一步的研究中会更深入地分析以寻找更合适的解决办法。

    4.2 与传统属性相比

    为了更清晰地验证体维数的形状表达能力,将体维数、曲率与泰森多边形协方差测量VCM(Voronoi Covariance Measure)算法[25]应用在特征提取上进行比较。曲率选用较为经典的Gauss-Bonnet方法[26]计算得到的高斯曲率。VCM算法具有很强的理论支撑,算法经典且实验结果较好,它使用邻域点的协方差矩阵计算得到对应的3个特征值,使用这些特征值作为点云的形状描述符(以下将这种描述符称为VCM描述符)。计算模型中每个点的体维数、曲率、VCM描述符,得到体维数组、曲率组与VCM描述符组(以下将由体维数值组成的一组值称为体维数组,由曲率值组成的一组值称为曲率组,由VCM描述符组成的一组值称为VCM描述符组)。体维数值所表达的面平坦程度与r有关,当r>1时,点附近的面越平坦,体维数越小;当0<r<1时,情况正好相反。因而,当r>1时,抽取大于阈值的点为特征点;当0<r<1时,则抽取小于阈值的点。曲率的绝对值与曲面的平坦程度成正比,因而选择曲率绝对值大于阈值的点,阈值选择标准为特征点数尽量与体维数组的特征点数相近。VCM算法选取特征点的标准与特征值的比值有关(每个点的VCM描述符具有3个特征值,为方便描述,将这3个特征值从小到大分别称为特征值1、特征值2、特征值3),在文献[25]中有具体说明:使特征值2与特征值1的比值大于10～40的某个数,即可获取特征点。在阈值属于10～40的前提下,阈值选择标准为特征点数尽量与体维数组的特征点数相近。

    如表1所示为每种模型的总点数以及3种方法提取对应模型特征点的个数以及运行时间的对比,图5为3种方法抽取对应模型特征点后的效果对比图。

    其中,表1中的模型a、模型i、模型m分别对应图5a、图5i、图5m。图5第1列为测试数据的实物模型图,其中图5a、图5e为同一模型的正面与背面;图5b～图5d、图5f～图5h、图5j～图5t、图5n～图5p为分别以体维数、曲率、VCM描述符为形状参量抽取的特征点后生成的效果图,其中黑色的点即为抽取得到的特征点。以下将以体维数为形状参量抽取的特征称为体维数特征,以曲率为形状参量抽取的特征称为曲率特征,以VCM为形状参量抽取的特征称为VCM特征。从图5中可以看到,在取到相近特征点个数的情况下,体维数特征与VCM特征相较于曲率特征,线特征更加清晰、明朗,即使在具有复杂表面结构的图5i中仍能提取到清晰的线特征。同时,体维数特征、VCM特征将模型的整体特征体现得较为完整,而曲率特征则有较多干扰点。如在图5m～图5p中,图5m除去本身的一些线条特征外,模型较为光滑,图5k(体维数)与图5p(VCM)较为完整地提取出了特征线上的点,而图5o(曲率)由于局部细节特征点的存在,使得整体特征变得模糊。从表1的运行时间对比来看,体维数方法相对于VCM算法运行效率更高。综上所述,体维数方法相较于曲率方法,所需时间稍长,但能够提取到更加清晰、完整的线特征;体维数方法与VCM算法均能提取到较完整的全局特征,但体维数方法的时间效率较高。

    5 结束语

    本文对相似性维数进行重新定义,给出了体维数和面维数的表达式,将这2种维数应用在点云模型的形状表达上,并使用缩放处理解决了尺度敏感的问题。理论分析发现,面维数无上限值可以兼顾更多的局部特征,而体维数有上限,更注重体现模型的全局特征。实验表明:2种相似性维数都具有表达形状的能力,体维数相对于面维数更适合表达模型的整体特征,且线条特征明显;在体维数与曲率以及VCM的对比实验中,体维数方法提取到的特征更清晰、完整,且运行效率较高。在下一步更深入的研究计划中,可将面维数与体维数相结合,来提取模型的局部细节特征。
