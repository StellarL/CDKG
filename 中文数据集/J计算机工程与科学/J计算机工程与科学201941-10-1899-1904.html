    摘    要：

    孪生支持向量机TWSVMs分类过程的计算量和样本的数量成正比,当样本个数较多时,其分类过程将会比较耗时。为了提高样本集的稀疏性,从而提高TWSVMs的分类速度,提出了一种基于AP聚类的约简孪生支持向量机快速分类算法FCTSVMs-AP。首先对原始数据集进行AP聚类操作。聚类的中心为约简后新的样本集,按照分类误差最小的原则构建优化模型,用二次规划方法求解新的决策函数的系数,并证明了当样本集压缩时,收紧新的快速决策函数和原始决策函数之间的误差等价于在样本空间对原始数据集进行AP聚类操作。在人工数据集和UCI数据集上的实验表明,保持分类精度的损失在统计意义上不明显的前提下,FCTSVMs-AP可以通过有效压缩样本数量的方式提高分类速度。

    A fast classification algorithm of reduced twin support vector machines based on AP clustering

    WEI Xiu-xi HUANG Hua-juan ZHOU Yong-quan

    College of Information Science and Engineering,Guangxi University for Nationalities Guangxi Higher School Key Laboratory of Complex Systems and Intelligent Computing (Guangxi University for Nationalities)

    Abstract：

    The computation of the classification process of Twin Support Vector Machines(TSVMs) is proportional to the number of samples. When the number of samples is large, the classification process will be time-consuming. In order to improve the sparsity of sample sets, a Fast Classification algorithm of Twin Support Vector Machines based on Affinity Propagation clustering(FCTSVMs-AP) is proposed. FCTSVMs-AP first performs adaptive AP clustering on the original data set. The center of the cluster is be used as the new sample set after reduction. According to the principle of minimum classification error, an optimization model is constructed. Moreover, the coefficients of the new decision tree is solved by quadratic programming. Furthermore, we prove that, when the sample set is compressed, the error between the new fast decision function and the original decision function is equivalent to the adaptive AP clustering of the original data set in the sample space. Experiments on artificial datasets and UCI datasets show that FCTSVMs-AP can improve the classification speed by effectively compressing the number of samples, while the loss of classification accuracy is not statistically significant.

    1 引言

    支持向量机SVM(Support Vector Machine)自1995年由Cortes等人提出以来就受到理论研究和工程应用两方面的重视,目前已成为机器学习的研究热点,并在很多领域如时间序列预测、手写数字识别、人脸图像识别等得到了成功的应用[1,2,3,4,5,3,4]。2007年,Jayadeva等人[6]在深入研究标准SVM形式的基础上,提出了孪生支持向量机TWSVMs(TWin Support Vector Machines)。TWSVMs是要得到2个不平行的分类超平面,使得每1个超平面靠近其中的一类样本,而远离另一类样本。TWSVMs在形式上完全类似于标准SVM,只是标准SVM解决的是1个大的二次规划问题,所有样本点都出现在这个二次规划问题的约束条件里,而TWSVMs需要解决的是2个小的二次规划问题,只把与目标函数的样本点相对的另一类样本点放在了约束条件里,这样使得TWSVMs的训练速度大大提高了。鉴于其明显的分类优势,TWSVMs已被应用于说话人识别[7,7]、医学检测[8,9,10,8]等领域。目前,许多学者对TWSVMs进行了改进研究,但这些改进算法大部分都是通过使用Lagrange定理来求解带有不等式约束的二次规划QPP(Quadratic Programming Problems)的对偶问题,然而QPP问题的求解在样本数目较大的时候受到了时间和内存的制约。

    在实际应用中,分类速度对提升TWSVMs的性能是非常重要的,特别是在当前云计算和物联网兴起的背景下,一些应用比如网络入侵检测、时间序列异常检测[11,12,11,12]等在线应用业务,其训练过程可以通过具有强大计算能力的后台云端完成,后台云端将训练得到的业务规则(分类模型)分发给泛联的各种客户端。对于手持移动设备这类计算能力较弱的客户端,如何利用从云端得到的分类模型快速实现分类将直接影响用户的使用体验。因此,研究TWSVMs的分类速度是现实应用中1个迫切需要解决的问题。

    仿射传播AP(Affinity Propagation)算法[13],也称为近邻传播算法,是在2007年的Science杂志上出现的一种新的聚类算法。AP算法的基本思想是将全部数据点都当作潜在的聚类中心(称之为exemplar),然后数据点两两之间连线构成1个网络(相似度矩阵),再通过网络中各条边的消息(responsibility和availability)传递计算出各样本的聚类中心。与k-均值算法或k中心点算法不同,AP算法不需要在运行算法之前确定聚类的个数,因此计算速度比传统聚类算法要快得多。

    为了提高TWSVMs求解大数据样本集的分类速度,结合AP聚类算法的优势,本文将通过自适应AP聚类方法对TWSVMs训练得到的原始样本集进行压缩,并求解稀疏化后的样本集对应的权重系数以期得到新的决策函数。同时进一步证明,收紧新的决策函数和原始决策函数之间的分类误差等价于在样本空间对原始支持向量进行自适应AP聚类操作。由此,本文提出一种基于AP聚类的TWSVMs快速分类算法FCTSVMs-AP(Fast Classification algorithm of Twin Support Vector Machines based on Affinity Propagation clustering)。该算法使用AP聚类方法将原始的 s1个样本集聚集为s2个,在分类精度损失误差最小的原则下构造1个二次规划优化问题,求解出新的支持向量的权重系数,使得FCTSVMs-AP的计算代价减小为原分类函数的s2/s1,并且求解权重系数的时间复杂度仅为O(s13)。

    2 AP聚类算法原理

    AP算法不需要事先确定聚类中心,而是将所有样本点都看作潜在的聚类中心,通过循环迭代,在迭代过程中不断搜索合适的聚类中心,自动从数据点间识别类中心的位置及个数。设样本个数为N的样本空间,任意2个样本点xi和xj之间的相似度S(i,j)用负的欧氏距离进行度量,其值储存于N×N的相似度矩阵 S中。矩阵对角线上的元素为偏向参数p,它表示各样本点被选作聚类中心的倾向性。调节p值可以改变聚类的结果,当p值小于一定的阈值时会引起分类数目的改变。

    定义1R(i,k)为xk适合作为xi的聚类中心的程度(responsibilty,吸引度);A(i,k)为xi选择xk作为其聚类中心的适合程度(availability,归属度)。

    为了找到合适的聚类中心xk,AP算法不断地从数据样本中搜集证据R(i,k)和A(i,k)。R(i,k)和A(i,k)的迭代公式为:

    R(i,k)=S(i,j)-maxj≠k{A(i,j)+S(i,j)}         (1)A(i,k)=min{0,R(k,k)+∑j∉{i,k}max{0,R(j,k)}}         (2)

    AP聚类过程就是根据式(1)和式(2)不断循环迭代从而更新证据的过程。迭代更新的快慢可以通过调节阻尼系数λ实现。对于数据点xi,若数据点xk使得R(i,k)+A(i,k)为R(i,j)+A(i,j),j=1,2,…,N中的最大值,那么数据点xk就是xi的聚类中心。通过迭代竞争的方式,AP聚类可以得到最优的聚类中心和各个样本点的类属情况。

    3 基于AP聚类的约简TWSVMs快速分类算法FCTSVMs-AP

    3.1 主要思想

    给定训练样本集(xi,yi),i=1,2,…,n,xi∈Rn,yi∈{+1,-1},核化后的TWSVMs的分类函数可以写成:

    f(x)=∑i=1s1αiyiΚ(xi,x)+b         (3)

    其中,xi∈Rn是支持向量,αi∈R是Lagrange系数,yi∈{+1,-1}是对应的xi的标签,b∈R是偏移量,K(·)是核函数。

    从式(3)中可以直观地看出,TWSVMs分类过程的计算量取决于s1的大小。传统的TWSVMs分类函数中包含了全部的支持向量,而较多支持向量又导致分类计算量较大,其分类速度必然较慢。因此,提高分类速度的关键是减少支持向量的数量s1。基于以上分析,我们定义新的快速分类函数:

    g(x)=∑j=1s2βjyjΚ(x′j,x)+b         (4)

    其中,s2<s1,{x′j}j=1s2为新的支持向量集,x′j∈Rn,βj是Lagrange系数。

    定义如下损失函数:

    L(x)=∥g(x)-f(x)∥2         (5)

    传统TSVMs将训练样本集{xk}k=1n稀疏化为支持向量集{xi}i=1s1,FCTSVMs-AP则将支持向量集{xi}i=1s1再稀疏化为新的支持向量集{x′j}j=1s2。s2/s1称为压缩率,压缩率越小,支持向量越少,稀疏程度越高,分类速度越快。在不影响分类精度的前提下,如何构造{x′j}j=1s2和新决策函数的系数βj是本文的关键,下面分别介绍{x′j}j=1s2和βj的构造方法。

    3.1.1 新支持向量集的构造方法

    本文采用自适应AP聚类方法把原始支持向量聚为s2类,聚类中心x′j为新的支持向量。下面证明采用自适应AP聚类方法构造新的支持向量集是有效的,即采用此方法可以使得式(5)是有上界的。

    定理1 将原始支持向量集{xi}i=1s1聚集为以{Dj}j=1s2为划分,以{x′j}j=1s2为中心的s2类,Dj表示第j类支持向量的集合,损失函数式(5)的上界可以表示为:

    L(x)≤C∑j=1s2∑xi∈Dj∥xi-x′j∥2         (6)

    其中,C是常数。

    证明 式(5)可以改写为:

    L(x)=∑j=1s2Lj(x)=∑j=1s2((∑xi∈DjαiyiΚ(xi,x))-βjyjΚ(x′j,x))2=∑j=1s2(∑xi∈Dj(αiyiΚ(xi,x)-βj|Dj|yjΚ(x′j,x)))2         (7)

    其中,|Dj|表示第j类中元素的个数。对于L(x)的每个子项Lj(x),由Cauchy-Schwarz不等式(∑i=1nai)2≤n∑i=1n(ai)2得到:

    Lj(x)=(∑xi∈Dj(αiyiΚ(xi,x)-βj|Dj|yjΚ(x′j,x)))2≤|Dj|∑xi∈Dj(αiyiΚ(xi,x)-βj|Dj|yjΚ(x′j,x))2         (8)

    并且存在常数cij使得:

    (αiyiΚ(xi,x)-βj|Dj|yjΚ(x′j,x))2=cij(Κ(xi,x)-Κ(x′j,x))2         (9)

    令z=σ∥a-b∥,其中σ是高斯核函数的参数,取高斯核时有K(a,b)=k(z)=exp(-z2)=exp(-σ‖a-b‖2),并且k′(z)=-2z·k(z),k′(z)≤k′(1/2)=2/e,由Lagrange中值定理,∃ξ使得:

    (Κ(xi,x)-Κ(x′j,x))2=(σk′(ξ)(∥xi-x∥-∥x′j-x∥))2≤(2σ/e)(∥xi-x∥-∥x′j-x∥)2         (10)

    由三角不等式定理(三角形的两边之差小于第三边)得:

    ∥xi-x∥-∥x′j-x∥≤∥(xi-x)-(x′j-x)∥=∥xi-x′j∥         (11)

    把式(8)、式(10)和式(11)一起代入式(7)有:

    L(x)=((2σe)|Dj|maxcijmax)∑j=1s2∑xi∈Dj∥xi-x′j∥2         (12)

    其中,|Dj|max=max(|Dj|),cijmax=max(cij),则令C=(2σ/e)max(|Dj|)max(cij)。证毕。

    3.1.2 新决策函数系数βj的求解方法

    为了使分类精度损失达到最小,就必须最小化损失函数。由此可得到无约束优化问题:

    minβj∑k=1n{|∑j=1s2βjyjΚ(xj,xk)-∑i=1s1αiyiΚ(xi,xk)|2}         (13)

    求解式(13)即可得到在分类精度损失最小情况下的新决策函数对应的权重系数{βj}j=1s2。

    为了求解方便,可将各个变量表示为向量形式。

    令α=[α1,α2,…,αs1]T,β=[β1,β2,…,βs2]T,有:

    Κα(x)=(Κ(x1,x)Κ(x2,x)  ⋮Κ(xs1,x))s1×1         (14)Κβ(x)=(Κ(x′1,x)Κ(x′2,x)  ⋮Κ(x′s2,x))s2×1         (15)

    下面我们证明式(13)可以用二次规划的形式表示,从而可以用Matlab优化工具箱中的quadprog函数求解二次规划问题,进而求出βj的值。

    定理2 式(13)表示的最小优化问题可以表示为如下的二次规划问题形式:

    minβ12βΤΗβ+hΤβ         (16)

    证明 先分析式(13)中的每个xk的通项式:

    ∥g(x)-f(x)∥2=g(x)2-2f(x)g(x)+f(x)2         (17)

    g(x)2=(∑j=1s2βjyjΚ(x′j,x))2=∑i=1s2∑j=1s2βiβjΚ(xi,x)Κ(x′j,x)yiyj=βΤ(Κβ(x)ΚβΤ(x))β         (18)f(x)g(x)=(∑i=1s1αiyiΚ(xi,xk))(∑j=1s2βjyjΚ(x′j,x))=∑i=1s1∑j=1s2αiβjΚ(xi,x)Κ(x′j,x)yiyj=αΤ(Κα(x)ΚβΤ(x))β         (19)

    Η=∑k=1n(Κβ(xk)ΚβΤ(xk))         (21)

    证毕。

    3.2 FCTSVMs-AP的算法步骤

    FCTSVMs-AP算法首先对原始的支持向量进行AP聚类操作。聚类的中心为约简后新的支持向量,得到稀疏性更好的支持向量集,随后对新的支持向量集进行二次训练。具体的算法步骤如下:

    Step 1 提取原始支持向量集。采用传统的TWSVMs对样本集进行训练,提取出支持向量集。

    Step 2 提取新的支持向量集。采用AP算法对原始支持向量集进行聚类,类中心作为新的支持向量集。

    Step 3 对TWSVMs进行二次训练。采用Step 2的支持向量集重新训练TWSVMs,得到新的TWSVMs分类模型。

    Step 4 测试TWSVMs的分类能力。采用Step 3的分类模型对测试样本进行测试。

    3.3 FCTSVMs-AP的算法分析

    二次规划问题式(16)的变量规模为s2,所以其求解的时间复杂度为O(s23)。AP聚类的时间复杂度为O(s12),所以二次训练的时间总和是O(s23)+O(s12)。原始TWSVMs的训练时间为O(N3),而s1≼N,s2≼N,所以FCTSVMs-AP的二次训练时间是在可以接受的范围内。FCTSVMs-AP的支持向量规模为s2,TWSVMs的支持向量规模为n,而且有s2<n,所以FCTSVMs-AP的分类时间要少于TWSVMs的分类时间。
