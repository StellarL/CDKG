    摘    要：

    为提高遥感影像草地分类的精度, 分析了卷积神经网络中提取图像特征的特点, 提出了一种基于特征整合深度神经网络的遥感影像特征提取算法。首先, 将遥感影像数据进行PCA白化处理, 降低数据之间的相关性, 加快神经网络学习的速率;其次, 将从卷积神经网络中提取到的浅层特征和深层特征进行双线性整合, 使得整合后的新特征更加完善和优化;最后, 对遥感数据进行训练, 由于新特征中有效信息的增加, 使得特征表达能力得到提高, 达到提高草地分类准确率的目的。实验结果表明:该算法能够有效地提高草地分类的准确率, 分类精度达到94.65%, 相较于卷积神经网络、BP神经网络和基于SVM的分类算法分别提高了4.3%、10.39%和15.33%。

    A grassland classification algorithm using convolutional neural network based on feature integration

    ZHANG Meng QIAN Yu-rong DU Jiao FAN Ying-ying

    Software College, Xinjiang University

    Abstract：

    In order to improve the precision of grassland classification from remote sensing images, we analyze the characteristics of image features extracted from convolutional neural networks (CNNs) , and propose a remote sensing image feature extraction method based on feature-integrated depth neural networks. Firstly, PCA whitening is performed on the remote sensing image to reduce the correlation between data and accelerate the learning rate of neural networks. Secondly, both low-level features and high-level features are bilinearly integrated to enhance and optimize the integrated features. Finally, the remote sensing data is trained. As the introduction of effective information in new features, both feature expression ability and the grassland classification accuracy are improved. Experimental results show that the proposed algorithm can effectively improve the accuracy of grassland classification. The classification accuracy reaches up to 94.65%. Compared with the traditional convolutional neural network, BP neural network and SVM algorithm, our accuracy is increased by 4.3%, 10.39% and 15.33% respectively.

    1 引言

    我国新疆地域广阔, 具有土地利用类型多样、对环境变化敏感等特点。天然的草地作为一个完整的生态系统, 在保护生态安全、发展牧业和传承草原文化等方面做出了突出贡献[1,1]。目前草地遥感科学发展为集现代遥感技术、地理信息系统和草原科学于一身的新兴边缘学科[2,2]。快速、准确地从各种植被中分离出草地信息, 对检测草地面积、观察草地长势、制定科学合理的区域生态保护计划以及经济开发决策等有重要的意义[3,3]。传统的基于遥感图像植被分类的方法有BP (Back Propagation) 神经网络和支持向量机SVM (Support Vector Machine) 等有监督分类方法[4,4]、目视解译法[5,5]、决策树法[6,6]等。这些方法虽然可以取得较好的分类效果, 但分类精度不高, 因此利用遥感数据实现对草地的分类是一个值得研究的问题。

    近年来, 随着深度学习 (Deep Learning) 的发展, 卷积神经网络CNNs (Convolutional Neural Networks) 因其强大的函数表达能力和在复杂分类上高效的识别率而受到广大学者的重视。从开始的ALexNet[7]到2015年的ResNet[8], 图像分类任务的错误率就从16.4%降低到了3.75%。在此之前, 传统的图像分类算法的错误率达到了25.8%。卷积神经网络为图像分类领域带来如此巨大的进步与提高, 究其原因是不同的感受野以及不同层级的信息使卷积神经网络提取的图像特征具有更强的可区分性。一些传统的图像特征, 例如方向梯度直方图HOG (Histogram of Oriented Gradient) 特征[9]、局部二值模式LBP (Local Binary Patten) 特征[10]以及尺度不变特征转换SIFT (Scale-Invariant Feature Transform) [11]特征, 均是研究者为使图像特征包含某类特定的信息而精心设计的, 目的是使其具有很强的语义。然而卷积神经网络提取的特征则不同, 它以损失函数为基准来划分图像分类的准确率, 进而提取到更适合分类任务的图像特征。单独运用卷积神经网络所提取的浅层特征或者深层特征都存在着一定的局限性, 因为一些重要的特征经过卷积层和池化层后可能会丢失。近期研究工作发现, 特征整合能够提高卷积神经网络在图像分类任务上的准确率, 这表明卷积神经网络所提取的特征的确丢失了一些有用的信息。

    鉴于以上研究, 本文提出一种基于特征整合的卷积神经网络分类算法, 用于提高对遥感影像中草地分类的准确率, 通过对卷积神经网络中浅层特征和深层特征进行双线性整合, 达到减少特征提取过程中信息损耗, 提高特征表达能力的目地。在大规模遥感影像组成的数据集基础上进行实验, 并对比分析了卷积神经网络 (CNNs) 、BP神经网络、基于SVM的分类算法和本文算法在对草地分类时的分类精度以及分类时耗。

    2 研究区域和数据来源

    2.1 研究区域介绍

    实验选取新疆维吾尔自治区乌鲁木齐县南山为研究区域。研究区域位于东经86°37′56″～88°58′22″、北纬43°01′08″～44°06′11″, 处于乌鲁木齐南部, 是北天山喀拉乌成北山麓的一部分, 距离市区75 km, 海拔2 252 m, 面积119 km2, 地处中山与低山过渡带, 由东西走向的10条大小沟谷组成, 是典型的山岳、森林草原结构型, 其风景秀丽, 是优秀的天然大牧场, 适用于草地识别研究。

    2.2 数据源

    研究选取的数据为来自新疆乌鲁木齐县, 空间分辨率为30 m的多光谱Landset 8号数据, 成像时间为2013年7月3日 (云量<5%) , 7月是草地生长的旺盛时期, 比较适于草地识别研究。

    由于遥感图像中目标间的相似度较高, 所以在对卷积神经网络训练之前, 需要对遥感影像数据集进行PCA (Principal Component Analysis) 白化处理, 用于降低像素间的相关性, 加快神经网络的学习速率, 并在降低特征之间相关性的同时, 使所有的特征都具有相同的方差。

    PCA是一种典型的降维方法, 它能在搜索主要成分的同时, 降低数据中的噪声, 广泛用于模式识别领域。由于遥感图像中相邻像素高度相关, 造成数据冗余, 在误差很小的情况下, 利用PCA可以将输入的向量转化为一个低维度的近似向量。对数据进行PCA白化处理的方法如下所示:

    (1) 计算协方差矩阵。

    Σ=1n∑i=1n (x (i) ) (x (i) ) Τ          (1)

    其中, n为样本个数, x (i) 为均值为0的样本。

    (2) 计算式 (1) 中Σ的特征向量, 并以列的方式排放, 组成矩阵V:

    V=[v11v12⋯v1pv21v22⋯v2p⋮⋮⋮vp1vp2⋯vpp]          (2)

    其中, p为特征向量的总数, v11为矩阵中向量对应的最大特征值, v22是次特征向量对应的最大特征值, 依次类推。

    (3) 主成分个数的选择。

    在选取不同的主成分个数w时, w的取值可以由保留的方差百分比来决定。令y1, y2, …, yp表示Σ的从大到小的特征值, 则yj为对应于特征向量vj的特征值。由此可得在取前w个成分时, 保留的方差百分比为:

    z=∑j=1wyj∑j=1pyj          (3)

    根据以往的经验, w应该保留99%的方差, 即w的取值应为z≥0.99的最小值。

    (4) 白化。

    经过白化后的遥感数据为:

    pΡCAwhite, i=prot, iyi+α          (4)

    其中, prot, j=vTixi, 为了避免某些特征值过小在相除的过程中造成结果溢出或数值不稳定, α取值为0.000 1。

    (5) 降维。

    把原始的数据降到w维, 可以得到:

    pΤilde=V (:, 1:w) Τ↔pΡCAwhite          (5)

    即为降维后的数据。

    3.3 特征整合卷积神经网络

    传统的卷积神经网络的工作流程是卷积层提取特征后, 经采样层进行组合, 形成更加抽象的特征, 再经过卷积层提取特征, 经采样层进行组合最后得到对图片对象的描述特征。

    相对于传统的卷积神经网络, 特征整合卷积神经网络主要有3方面的变化。首先, 特征选择, 选取哪一层的特征作为浅层特征进行整合;其次, 特征整合的方法, 本文采用了特征双线性整合, 分类器使用单线性的SVM分类器;最后, 利用双线性整合得到的特征对分类器进行训练。网络中的某2层特征函数可表示为fA、fB。特征整合卷积神经网络的结构如图3所示。

    图3中, 遥感影像经过PCA白化处理后输入到深度卷积神经网络中, 然后网络中第24层和第35层输出数据将会被整合到一起作为最终的遥感影像特征并进行分类。

    3.4 特征选择

    特征整合首先需选择要整合的特征, 本文以VGGNet-16为例[12], 该网络有16层, 每一层的网络均会产生不同的输出。选择不同层网络的输出进行特征整合就会得到不同的特征。

    如果选择较浅层的网络, 会因此层特征没有得到有效的筛选, 导致特征中包含较多的冗余信息为之后的特征降维带来困难, 并且选择浅层网络的特征会引入较多的噪声, 当引入的噪声多于引入的信息时, 将会产生适得其反的效果。

    如果选择较深层的网络, 则特征经过卷积层和池化层后可能会丢失较多有价值的信息, 就不能为新特征带来更多有价值的信息, 甚至会影响整个网络的性能。因此, 选择合适的要整合的特征至关重要, 需要权衡浅层特征和深层特征之间的利弊。另外, 在选择的过程中尽量选择池化后的输出, 因为池化仅能使特征具有良好的位移不变性, 而且达到了降低特征维度、减少噪声的目的。

    一个比较好的的特征整合选择就是双线性整合。

    其中, p是一个位置矩阵, c为遥感图像, 这个函数输出的维度为l×J。

    由以上方式得到2个特征通过矩阵外积进行整合, 在每个具体的位置γ, 双线整合θ可被写成:

    θ (γ, c, fA, fB) =fA (γ, c) ΤfB (γ, c)           (8)

    这里需要说明的是, 特征函数fA和fB具有相同的某一个输出维度, 例如, 如果fA提取到的特征维度为L×M, fB提取到的维度为L×N, 那么线性整合θ的输出即为M×N的矩阵。

    当双线性模型用于卷积神经网络时, 卷积神经网络的特征函数会随之发生一些相应的变化。原因在于一个位置矩阵λ, 采用训练过的网络模型作为特征函数能够得到好的特征。在特征整合的过程中还是需要进行池化, 本文选用最大池化, 这样有利于忽略位置信息, 保持位移不变性, 更适合整个特征的稳定。

    最后, 将得到的结果向量化, 就得到了图像的特征。

    5 结束语

    为了提高草地分类精度, 本文提出了一种将卷积神经网络中浅层和深层特征经过双线性整合产生新特征的特征提取算法, 为了降低输入遥感影像的冗余性, 先对遥感影像进行了PCA白化处理, 然后对提取的不同层中的特征进行整合。实验表明, 本文算法的总体分类精度明显高于支持向量机 (SVM) 分类算法和BP神经网络的分类精度, 证明了该算法在遥感影像分类中的可行性, 可为遥感影像处理领域以及乌鲁木齐县当地畜牧业的相关工作提供一定的参考价值。在接下来的工作中, 将研究大数据多GPU时该算法的可行性以及通过调整网络的学习率、迭代次数等方式来进一步优化。
