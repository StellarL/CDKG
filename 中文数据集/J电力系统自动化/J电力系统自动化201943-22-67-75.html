    摘    要：

    针对大规模交直流混联系统的快速动态安全风险防控需求,研究了系统动态安全风险智能评估系统的架构与关键技术。结合电力系统安全风险评估的一般流程,提出了基于机器学习的安全风险智能评估系统的总体框架和动态安全风险统一评估模型结构;构建了包含发电负荷运行方式、网络拓扑结构和故障位置特征的训练样本集合,采用样本平衡技术提高评估模型精度;基于深度学习提取动态安全风险高级特征,采用主流机器学习框架构建和更新动态安全风险统一评估模型。以省级电网为例验证了所提动态安全风险智能评估系统的可行性。

    Intelligent Assessment System for Dynamic Security Risk of Large-scale Power Grid

    LI Changgang LI Huarui LIU Yutian WU Haiwei ZHANG Qibing FAN Haifeng

    Key Laboratory of Power System Intelligent Dispatch and Control of Ministry of Education (Shandong University) State Grid Jiangsu Electric Power Co., Ltd.

    Abstract：

    The architecture and key technologies of the intelligent assessment system for dynamic security risk of power systems are studied for preventing and controlling fast dynamic security risk of large-scale AC-DC hybrid systems. The overall framework of intelligent assessment based on machine learning and the structure of unified dynamic security risk assessment model are proposed considering the general assessment process. A training sample set containing fault locations, generation and load modes, and network topology structures is constructed. The sample balance technique is used to improve the accuracy of the assessment model. Advanced features for assessing dynamic security risk are extracted based on deep learning, and a unified assessment model of dynamic security risk is constructed and updated with mainstream machine learning framework. A provincial power grid is taken as an example to verify the feasibility of the proposed intelligent assessment system for dynamic security risk.

    0 引言

    随着大容量直流输电工程的集中馈入馈出,中国电力系统已经成为典型的大规模交直流混联电网[1]。直流输电线路等暴露于自然环境中,可能发生各类故障导致直流闭锁,系统遭受严重功率扰动的风险增加[2]。因此,快速感知系统动态安全风险,及时预警各种高风险运行方式,是现代电力系统安全稳定运行的基础,也是电网智能调度控制系统的基本功能[3]。

    传统调度系统的动态安全风险评估方法多基于时域仿真技术[4,5,6,7,8,9]。时域仿真技术计及电力系统各元件物理模型,通过逐步积分的方法对各元件数学模型构成的微分—代数方程组(differential algebraic equations,DAE)进行求解,计算结果准确。但是,对于大规模电力系统,系统详细DAE规模庞大,时域仿真方法存在计算量大、耗时严重问题[10]。随着新能源发电的大规模接入,系统运行方式的不确定性增加,动态安全评估需构建大量典型运行方式[11]。在此背景下,基于时域仿真技术评估系统动态安全风险耗时长,难以满足动态安全风险评估快速性的需求[12]。

    随着人工智能技术的快速发展,基于机器学习的动态安全风险智能评估方法也大量涌现。针对暂态稳定[13]、电压安全[14]和频率安全[15]等安全风险属性,不同方法采用稳态潮流特征[15]或综合使用潮流特征与故障后特征[16]构建动态安全风险评估模型。根据其对故障位置的处理方式,现有方法可分为针对特定故障建立专用模型[17]、基于故障编码建立通用模型[18]和基于动态特征隐式表达故障位置建立通用模型[19,20]等3种。

    与基于物理模型的时域仿真方法不同,基于机器学习的动态安全风险评估方法是数据驱动的。此类方法首先通过离线训练建立动态安全风险评估模型,然后针对需要进行评估的未来场景,使用提前训练好的模型进行在线评估。与时域仿真方法相比,基于机器学习的动态安全评估方法无须进行逐步积分,具有评估速度快的优点,是未来电网智能调度系统实现动态安全风险快速感知的重要途径[21]。

    现有基于机器学习的方法多针对单一安全风险属性或指标,且主要针对发电负荷运行方式等特征开展研究,缺乏故障位置和网络拓扑结构特征的准确表达方法,无法提取故障位置和网络拓扑结构的关键特征,动态安全评估输入特征集不完备。这导致系统动态安全风险评估的模型集合规模庞大,模型训练、更新和维护困难。

    针对上述问题,本文在现有研究基础上构建了基于机器学习的大规模交直流混联系统动态安全评估系统框架,并提出了能够适应故障位置和网络拓扑结构变化的动态安全统一评估模型;充分考虑未来待评估样本与训练样本之间的联系,生成符合历史运行规律且能充分反映未来待评估样本可能分布的训练样本;采用复杂深度学习与简单顶层模型相结合的方法构建动态安全风险评估模型,并以某省级电网为算例,对所提动态安全风险智能评估系统的有效性进行验证。

    电力系统动态安全风险评估需要适应不同的发电负荷运行方式、网络拓扑结构和预想事故。对于数小时后的未来态运行方式,受新能源出力不确定性和负荷预测误差的影响,系统的运行方式存在较大的不确定性,发电负荷运行方式复杂多变。为提高动态安全风险评估模型对发电负荷运行方式的适应性,理论上针对具体的网络拓扑结构和预想事故分别建立专用的评估模型具有最高的精度。但是,由于系统网络拓扑结构和预想故障数量众多,采用这种方式构建的智能评估模型库规模巨大,模型训练和更新维护困难。

    为降低动态安全风险智能评估模型库的规模,可通过离线研究辨识系统关键支路,进而利用One-Hot方法或网络电气距离等指标对系统网络拓扑结构进行统一编码,形成网络拓扑结构特征。对于预想故障,不同类型的故障及其对应的主导动态安全属性存在显著差异,难以构造统一的故障类型特征。对于类型相同但位置不同的故障,其差异主要表现于故障在电网结构中的相对位置,可以使用故障编号或与网络拓扑结构特征相似的构建方法形成统一的故障位置特征。

    2 动态安全风险训练样本生成

    动态安全风险智能评估模型基于样本进行训练,模型的适应性与样本的分布密切相关。在构造用于实际电力系统动态安全评估的训练样本时,可能会出现不稳定样本比例远低于稳定样本比例的情况,使得不稳定样本成为少数类。而人工智能方法进行分类时存在着对于多数类的倾向性,因此会导致模型的泛化能力下降。同时,如果生成的训练样本不具有代表性,没有覆盖待评估样本可能出现的场景,也会导致模型的泛化能力下降。因此,在生成训练样本时,需要考虑到训练样本不均衡和训练样本缺乏代表性等问题,对训练样本的分布进行调整。

    2.1 训练样本的结构

    系统动态安全风险智能评估模型训练的样本集可表示为{z1,z2,…,zi,…,zN},其中N为训练样本个数,zi为第i个样本,并可表示为zi=(xi,yi),xi为样本特征向量,yi为样本对应的安全风险属性指标。

    xi的主要数据为系统发电负荷运行方式、网络拓扑结构和故障位置特征。发电负荷运行方式与网络拓扑结构信息可由潮流数据得到。潮流数据的生成一般有2种途径:一是调度平台运行产生的大量历史运行方式;二是调度人员根据系统可能发生的极端场景,通过合理的运行方式生成程序构建的极端运行方式。

    对于yi,可针对各潮流样本,结合预想事故集和系统动态参数,使用PSASP,BPA或PSS/E等全时域仿真软件分析系统的动态安全性,进而分析各动态安全风险指标。

    2.2 动态安全风险指标

    对于功角稳定,通常将是否存在任意 2 台机的转子角度差值大于180°作为暂态稳定的定性判断依据[23]。为定量衡量系统的功角稳定安全风险,可以考虑进一步引入极限切除时间(critical clearing time,CCT)[24]等定量评估指标。

    对于电压安全,除电压偏移量外,还可根据故障后电压在指定时间内能否恢复到指定阈值以上[25]来定量衡量系统的电压安全风险,也可进一步引入CCT[26]和基于多二元表判据的实用暂态电压稳定裕度指标等定量评估电压安全性[27]。

    对于频率安全,除频率偏移量指标外,还可根据频率二元表判断系统频率安全性[28]。为定量衡量系统的频率安全风险,可进一步引入基于二元表的暂态频率偏移可接受性量化裕度指标[29]和基于累积效应的暂态频率稳定指标等[30]。

    2.3 训练样本的生成

    由于系统中用户负荷水平的变化具有一定的周期性,同时系统网络拓扑结构、开停机方式和出力计划等短时间内不会发生较大改变,因此系统运行方式往往具有一定的变化规律。当历史数据足够多时,可以认为历史数据中各种运行方式的分布情况可反映系统中实际各种运行方式的概率分布规律。

    基于系统历史运行数据,使用传统方法对系统运行方式的概率分布情况进行拟合时,往往需要对概率分布函数进行假设[31]。当实际概率分布函数与假设相差较大时,会导致较大拟合误差。此时,可以采用文献[32]所提的改进生成式对抗网络 (Wasserstein generative adversarial network,WGAN)模型,在不对概率分布函数类型进行假设的情况下,基于系统历史数据得到系统运行方式的概率分布。

    基于历史数据中各运行方式的概率分布情况,拟合得到系统实际运行方式的概率分布,并认为待评估样本遵循相同的概率分布。基于相应的概率分布规律,通过蒙特卡洛抽样生成大量覆盖评估样本可能运行场景的训练样本,对模型进行训练,可以充分挖掘待评估样本与训练样本之间的内在联系,从而提高模型的预测能力。

    2.4 安全样本与不安全样本的数量平衡

    由于电力系统中绝大多数故障对系统的安全风险影响较小,会导致样本中低安全风险的样本过多,而高安全风险以及失稳的样本过少,存在样本不均衡问题。此时,以总体分类准确率为学习目标的传统分类算法会过多地关注多数类,从而使得少数类样本的分类性能下降,模型的泛化能力减弱。

    对于样本不平衡问题,常用的解决方法如下。

    1)基于欠采样和过采样方法[33],通过改变原始样本分布,增加少数类样本的数量或者减少多数类样本的数量,来消除或减小样本的不平衡问题。

    2)采用合成少数类过采样技术(synthetic minority oversampling technique,SMOTE),结合已有的少数类样本,通过线性插值等方法按照一定规则合成少数类样本,最终达到平衡数据集的目的[34]。

    3)在Bagging等集成学习算法的基础上,优化样本抽样方法,降低各基学习器的样本不平衡度[35]。

    4)引入代价敏感度学习方法,对少数类样本赋予较高的错分代价,提高模型对少数类样本的评估精度[36]。

    上述各方法在现有研究中均有应用,在构建具体的安全风险智能评估模型时,可根据实际模型和样本特点选择合适的样本平衡方法。

    3.2 顶层模型学习

    安全风险指标根据其可能的取值类型分为离散型和连续型2类,可利用决策树(decision tree,DT)、支持向量机(support machine vector,SVM)和人工神经网络(artificial neural network,ANN)等机器学习方法中的分类和回归方法分别建立相应模型。在构建顶层模型时,需要综合比较不同人工智能方法的有效性,选择合适方法。

    针对复杂电力系统的动态安全风险智能评估问题,使用单一方法建立机器学习模型往往存在精度不足、泛化能力差等问题。此时,可以利用以下集成学习方法,通过集成多个元学习器来提高模型的预测精度。

    1)AdaBoost集成学习方法[44]

    其基本思想是逐步构造出一组元学习器,每构造一个新的元学习器都着重弥补前一个元学习器的缺陷,最后将它们各自的预测输出集成,获得更理想的预测效果。具体来讲,数值初始化时对每一个训练例赋予相等的权重1/n,然后用该学习算法对训练集训练一轮,每次训练后对训练失败的训练例赋予较大的权重,也就是让学习算法在后续的学习中集中对比较难的训练例进行学习,从而得到一个预测函数序列,预测效果好的预测函数权重较大,反之较小。最终的预测函数对于分类问题采取加权投票的方法,对于回归问题采取加权平均的方法,得到模型最后的输出结果。

    2)Bagging集成学习方法[45]

    Bagging算法是一种用来提高弱学习器准确率的集成方法。这种方法的基本思想是对训练集实施有放回的随机样本重新抽样,形成多个与训练集规模相近但各不相同的训练子集,在此基础上形成多个元学习器,在多个元学习器的基础上,对分类问题采取多数投票原则,对回归问题采取简单取平均的方法,得到模型最后的输出结果。它利用训练集中随机抽取出的子集来训练每个模型,有助于降低方差并避免过拟合现象的发生。

    构造顶层学习模型时,如果存在单个学习器精度不足的情况,可以根据Adaboost集成和Bagging集成的具体效果,选择合适的集成学习方法,提高模型的预测精度。

    3.3 模型更新

    对于通过在线评估得到的低置信度场景,需进一步使用时域仿真方法进行详细校验。受限于计算资源,给定时间内时域仿真能够详细校验的场景数是有限的。如果存在大量低置信度场景,则时域仿真校验所需计算量可能会超出当前可用计算能力。因此,评估模型将无法在给定时间内完成对所有未来场景的准确评估,模型适应性不足。此时,需将低置信度场景加入训练样本集,对模型进行训练更新,以提高评估的置信度和模型适应性。

    模型训练更新可采用增量学习(incremental learning)[46]或迁移学习(transfer learning)[47]等方法,以充分利用已有训练样本,挖掘其与新加入样本间的关系,简化训练过程,降低训练耗时。

    4.1 样本平衡处理

    在基于本文方法对系统动态安全进行评估时,功角稳定、电压安全和频率安全评估都可能存在训练样本集中安全样本数量较多、不安全样本数量较少的情况,从而导致训练样本中样本分布不平衡的问题。而相较于频率安全评估和电压安全评估的回归问题,功角稳定的分类问题受样本分布不平衡的影响更大,会使得功角稳定评估的评估结果倾向于稳定,从而使得评估结果不可信。因此,本文以功角稳定评估问题中的样本平衡为例,验证了本文样本平衡处理方法的有效性。

    为构建模型样本集,在生成训练样本时,本文根据峰谷差率约为40%的全网负荷变化曲线,分别构建服从正态分布的各负荷曲线和直流功率曲线,并按照全网网损最小计算最优潮流,优化常规机组出力,共得到1 398个不同负荷水平、不同直流运行状态的运行方式,其中功角稳定的样本943个,功角失稳样本455个,稳定样本明显多于失稳样本。

    在生成测试样本时,采用与生成训练样本时相似的运行方式生成方法,生成测试样本970个,其中功角稳定样本649个,功角不稳定样本321个。为验证本文样本平衡方法的有效性,剔除测试样本中的328个稳定样本,保留321个稳定样本,使得测试样本中失稳与稳定样本比例基本一致。

    在未进行样本平衡前,采用本文暂态稳定评估模型,测试样本中失稳样本正确判断率为90.4%,稳定样本正确判断率为99.7%,全部样本正确判断率为97.8%。由于训练样本中稳定样本明显多于失稳样本,因此模型对于测试样本的评估结果趋向于功角稳定,导致模型对于失稳样本的正确判断率偏低。为解决此问题,采用本文前述几种样本平衡处理方法对训练样本进行调整,样本调整方法和相应仿真结果如下。

    1)基于过采样技术,增加稳定样本,调整样本中稳定样本和不稳定样本比例,使得平衡样本和不平衡样本比例一致,即稳定样本和失稳样本均为943个,然后再使用本文暂态稳定评估模型对测试样本进行评估,得到调整后失稳样本正确判断率为98.4%,稳定样本正确判断率为97.9%,全部样本正确判断率为98.2%。

    基于欠采样技术,增加训练样本中失稳样本比例,使得稳定样本和不稳定样本比例基本一致,失稳样本与不失稳样本均为455个,得到经过调整后的失稳样本正确判断率为97.8%,稳定样本正确判断率为98.5%,全部样本正确判断率为98.4%。

    2)基于SMOTE插值的方法合成新的失稳训练样本,增加训练样本中不稳定样本比例,使得训练样本中失稳和不失稳样本均为943个,得到经过调整后的失稳样本正确判断率为97.5%,稳定样本正确判断率为97.1%,全部样本正确判断率为97.3%。

    3)基于自助采样的方法,随机从943个稳定样本中随机抽取455个样本,与455个失稳样本共同构成元学习器训练样本集。共进行20次随机抽样,形成20个稳定样本子集,与不稳定样本共同构成元学习器训练样本集用于元学习器的训练,共得到20个元学习器,共同构成最终的集成学习器。对20个元学习器进行集成,得到最终的评估模型。经过调整后失稳样本正确判断率为97.3%,稳定样本正确判断率为98.2%,全部样本正确判断率为98.1%。

    4)使用代价—敏感度学习方法处理训练样本中失稳与稳定样本比例不平衡问题,对失稳样本取较高错分类代价,经过调整后失稳样本正确判断率为97.2%,稳定样本正确判断率为97.9%,全部样本正确判断率为97.6%。

    综合考虑以上几种解决样本不平衡问题的方法在针对稳定与不稳定样本不平衡上的效果,各种方法的效果相差不大,而过采样方法相较于其他方法精度略高,因此本文选择使用过采样方法处理暂态稳定评估问题中稳定与不稳定样本不平衡的问题。

    5 结语

    中国交直流混联电网遭受严重功率扰动的风险增加,需要快速评估系统动态安全风险,预警高风险场景。本文提出了基于机器学习的大规模交直流电网动态安全风险智能评估系统框架,充分利用平台历史数据,通过离线训练的方式得到功角稳定、电压安全与频率安全的统一评估模型,针对未来不确定场景和预想事故进行动态安全在线评估。通过对低置信度样本的全时域仿真,校验模型适应性,并进行模型训练更新。

    在本文基础上进一步完善系统结构与功能,并针对具体系统优化评估模型选择,将有助于快速评估未来高不确定性场景的动态安全风险,并为高风险运行场景的快速前瞻预警、趋势分析和预防控制奠定基础。
