    摘    要：

    现有就地型馈线自动化在应对网络拓扑变动、多联络点自动恢复等方面存在不足。文中提出了一种基于区域序号的自适应就地型馈线自动化故障处理新方法,所有开关均配置统一定值,并按区域序号定义原则为每个开关分配逻辑序号,各开关的运行参数将根据逻辑序号自动调整,基于配电一二次成套设备并配合变电站出口保护,即可自适应网络拓扑变动,完成故障的定位隔离及多联络恢复供电。通过典型应用案例进一步说明方法在配电网发生新增分支线、多联络运行方式调整、高阻单相接地故障等情况下具有自适性。采用PSCAD/EMTDC进行仿真测试,验证了所提方法的正确性和有效性。

    区域序号; 参数自适应; 就地型馈线自动化; 配电网;

    Adaptive Fault Processing Method for Local Type Feeder Automation Based on Region Sequence Numbers

    LI Zhaotuo JIN Songmao ZHANG Hua

    Guodian Nanjing Automation Co., Ltd.

    Abstract：

    Existing local type feeder automation is insufficient in dealing with network topology variations and automatic recovery of multiple contact points. A new adaptive local type feeder automation fault processing method based on regional sequence number is proposed. All switches in the distribution region are configured with uniform setting and each switch is assigned with a logical number according to the principle of region sequence number definition. The operation parameters of each switch are automatically adjusted according to the logical number. Cooperating with the substation protection and primary/secondary combined equipment, network topology changes can be adaptive, and the fault location and isolation and multi-contact power restoration can be completed. The typical application schemes are used to further illustrate the self-adaptability of the method in the case of adding branch lines, multi-contact operation mode adjustment and high-impedance single-phase grounding faults in the distribution network. The validity of the proposed method is verified by simulation on PSCAD/EMTDC.

    Keyword：

    region sequence number; parameter self-adaptive adjustment; local type feeder automation; distribution network;

    馈线自动化作为配电自动化系统的重要组成部分,对快速处理配电网故障、提升供电可靠性、缩短抢修时间具有重要作用[1,2,3,4]。馈线自动化分为集中型和就地型。就地型馈线自动化无需通信系统,能够快速部署,可在配电自动化建设进程中率先应用实施[5,6,7,8,9]。国内外已有多种就地型馈线自动化技术,包括电压时间型、电压电流型、自适应综合型等[10],但上述方法在应对网络拓扑变动、多联络点自动恢复等方面存在着不足。

    对于电压时间型和电压电流型馈线自动化,整定参数与网络拓扑相关[11],如果配电网新增分段开关、配电分支、或联络点位置变化时,原有整定参数可能无法确保故障处理流程的正确性,需要根据变更后的拓扑结构进行定值修改[12]。例如,当主干线延长增加分段开关时,就需要根据主干线开关数量重新计算调整各支线开关的得电延时合闸时限(X时间),否则原有整定值将造成干线开关与分支线开关同时合闸,有可能引起逻辑误判。

    自适应型是近几年提出的一种就地型馈线自动化实现方式,通过一次初始配置,即可自动适应后续配电网拓扑结构变化。现有的自适应综合型馈线自动化系统,所有开关采用统一定值,故障发生后,故障路径上的开关X时间不变,非故障路径上的开关X时间自动延长,以此来保证优先完成故障路径的隔离操作。当故障下游存在多个联络开关,且故障下游开关到联络开关通路上的开关数量相同时,如果联络开关采用统一定值,故障下游恢复阶段各联络开关同时合闸,会造成系统非计划合环,影响系统稳定性;否则需要针对每种可能存在的联络点分布情况分别进行参数整定。

    另外,在高阻接地故障发生时,如果接地过渡电阻过大,导致接地故障检测算法无法产生接地故障信号[13,14],自适应综合型馈线自动化中所有开关的X时间均会延长,在网络拓扑中存在分支的情况下,会有2个以上开关同时合闸的情况,如果其中一个开关合于接地故障检测到零序电压,同一时刻合闸的其他健全线路开关均检测到零序电压,现有处理机制将认定刚刚合闸开关的下游为故障区域,即误将健全线路隔离,进而执行后继错误的操作。

    本文提出一种基于区域序号的自适应就地型馈线自动化处理方法,对供电区域内的自动化开关进行逻辑编号,通过逻辑编号这个新增差异化条件,进一步丰富判断依据,达到降低现场实施难度、自适应网络拓扑变化的需求。

    1 基于区域序号的馈线自动化故障处理方法

    1.1 区域序号定义原则

    基于区域序号的自适应就地型馈线自动化,需要对区域内的开关分配逻辑序号,逻辑序号定义原则见图1。图中QF为断路器,QS为负荷开关。

    图1 区域序号定义原则   下载原图

    Fig.1 Definition principle of regional sequence number

    1)区域范围划定

    在正常运行状态下,以联络开关作为分界点,每个供电区域即为一个逻辑区域。图1中,QS15和QS16为联络点,将配电网划分为3个逻辑区域。

    2)逻辑序号分配

    每个逻辑区域内分段开关独立分配序号,起始序号为1,序号按主干线路、大分支线路、分支线路顺序分配。以逻辑区域Ⅱ为例,主干线路开关QS21,QS22和QS23序号依次为1,2和3,大分支线路开关QS24和QS25序号依次为4和5。

    如果遇到逻辑区域内因新增分支、线路延长等有新分段开关加入的情况,新加入开关在区域内已有序号基础上继续编号即可,无需考虑拓扑位置。以逻辑区域Ⅱ为例,新增分支开关QS26,该开关序号在区域Ⅱ内已有序号基础上继续增加,序号为6。

    联络开关序号统一为0。

    3)逻辑运行时间调整

    所有开关设置统一的基础定值,由于开关可在分段开关状态和联络开关状态间切换,开关基础定值包含分段开关基础定值和联络开关基础定值。其中,分段开关基础定值包括X时间、合闸确认时限(Y时间),X时间设置值大于Y时间设置值。联络开关基础定值包括单侧失压延时合闸时间(XL时间),实际运行时,分段开关状态的得电延时合闸时限(X运行时间)和联络开关状态的单侧失压延时合闸时间(XL运行时间),将根据逻辑序号自适应调整。

    X运行时间的数值在X时间统一基础定值上增加开关逻辑序号数值:

    ΤXr=ΤXb+L         (1)

    式中:TXr为X运行时间;TXb为X时间基础定值;L为开关逻辑序号。

    例如,当X时间基础定值为7 s时,2号开关的X运行时间为9 s。

    XL运行时间的数值在XL时间统一基础定值上增加开关逻辑序号一半的数值:

    ΤXLr=ΤXLb+0.5L         (2)

    式中:TXLr为XL运行时间;TXLb为XL时间基础定值。

    例如,当XL时间基础定值为60 s时,3号开关的XL运行时间为61.5 s。

    本文中X时间基础定值统一设置为7 s,Y时间统一设置为5 s;XL时间基础定值按不同运行方式下最大故障隔离时间来统一整定:

    ΤXLb>Τp+Τr+Τmax+Τd         (3)

    式中:Tp为断路器保护动作时间;Tr为一次重合闸时间;Tmax为最长分段情况X时间基础定值之和;Td为安全余量,可取0.5Tmax。

    配电网馈线主干线分段数量多为3分段或4分段,考虑联络点调整最多形成6分段情况(线路上有5个分段开关),最长隔离时间发生在末端分段开关与联络开关之间,取Tp=0.5 s,Tr=2 s,此时的隔离时间为55 s。如果对恢复时间没有特别严格的需求,可以根据配电网规模将XL时间基础定值统一设置为比最大故障隔离时间更大的数值(例如在上述示例中设置为60 s),从而覆盖所有可能的恢复情况。

    1.2 故障处理方法

    发生故障后,变电站出口断路器保护动作跳闸,供电区域内分段开关失电分闸,随后变电站出口断路器第1次重合闸,分段开关依次得电延时合闸,得电合闸延时时间为X运行时间,分段开关在执行合闸操作前判断是否存在零序电压,无零序电压后方可合闸。

    故障上游分段开关合闸后,线路故障再次发生,如果是短路故障或大电流接地故障,变电站出口断路器再次跳闸,供电区域内所有分段开关失电分闸,故障上游分段开关通过正向合闸闭锁逻辑闭锁合闸;如果是小电流接地故障,故障上游分段开关分闸,并通过正向合闸闭锁逻辑闭锁合闸。故障下游分段开关通过反向合闸闭锁逻辑闭锁合闸。

    分段开关正向合闸闭锁的判据有以下2个:①在合闸后Y时间内,检测到故障信号,开关失电分闸后进入正向合闸闭锁状态,其中,故障信号包括过流信号和接地故障信号,不包含电压相关判据;②合闸后未检测到①中所述故障信号,同时立即检测到零序电压,开关执行延时分闸,如果分闸后零序电压立即消失,则开关进入正向合闸闭锁状态,其中,延时时间为逻辑序号对应时间(例如:3号开关延时3 s分闸)。

    如果合闸后没有立即检测到零序电压,或者在延时计时过程中零序电压消失,又或者分闸后零序电压没有立即消失,则逻辑恢复正常运行状态。如果合闸后Y时间内没有检测到故障信号,也没有立即检测到零序电压,则逻辑也恢复正常运行状态。闭锁逻辑如图2所示。

    图2 正向合闸闭锁流程   下载原图

    Fig.2 Blocking process of positive direction closing

    反向合闸闭锁的判据有以下2个:①分段开关处于分位且检测到未保持X运行时间的瞬时电压;②分段开关处于分位,正向得电已保持X运行时间,同时检测到零序电压,在零序电压消失时,正向电压同时消失。

    分段开关具备反向合闸闭锁自动解除逻辑,当处于反向合闸闭锁的开关检测到原得电侧再次得电,电压持续时间大于X运行时间且未检测到零序电压,则开关得电合闸,解除反向合闸闭锁。逻辑流程如图3所示。

    非故障区域恢复供电包括故障上游区域恢复供电和故障下游区域恢复供电。对于短路故障或大电流接地故障,故障隔离后,故障上游恢复供电通过变电站出口断路器第2次重合闸实现;如果是小电流接地故障,故障隔离后故障上游未失电,无需恢复操作。故障下游区域恢复供电通过联络开关合闸转供,下游非故障区域分段开关依次得电合闸完成。

    2.3 高阻单相接地故障处理流程

    发生单相接地故障时,配电终端或集中选线装置可根据配电网中性点不同接地方式选择相应算法,故障处理流程在配电终端或集中选线装置选线跳闸并重合闸后开始执行。

    如果接地过渡电阻不是特别大,故障零序电流超过接地算法的最小动作电流阈值,接地故障由线路首个分段开关选线跳开并延时重合闸,后续处理方法与上述相间短路故障处理方法一致。

    如果由于接地过渡电阻较大导致算法不能正常工作,此时可通过变电站集中接地选线装置、手动拉线选线等方法跳开单相接地线路并重合闸,再通过高阻单相接地故障处理流程完成后续处理[16,17]。

    2)QF1第1次重合,QS11(8 s)和QS12(9 s)依次得电合闸,QS12合闸后QS13,QS14和QS15同时进入得电合闸逻辑。QS13经过10 s后合闸,QS13合闸后未检测到故障信号和零序电压,恢复正常运行状态;QS14经过11 s后合闸,合闸后未检测到故障信号,但立即检测到零序电压,开始11 s延时分闸计时;QS15经过12 s后检测到零序电压,将暂停得电合闸逻辑,等待零序电压消失,具体流程见附录A图A12。

    3)QS14合闸后经过11 s分闸,分闸后零序电压立即消失,进入正向合闸闭锁状态。QS72处于分位,正向得电已保持X运行时间,同时检测到零序电压,当检测到零序电压消失时,正向电压同时消失,进入反向合闸闭锁状态。QS15在QS14分闸后检测到零序电压消失,正向电压未消失,执行合闸。同理,QS71和QS23依次得电合闸,具体流程见附录A图A13。

    3 仿真验证

    结合某地区实际配电网拓扑结构,选取一个3电源点架空供电回路作为仿真对象,如图5所示。在PSCAD/EMTDC中搭建线路模型,同时编写基于区域序号的自适应就地型馈线自动化逻辑模块,将逻辑序号以及仿真产生的电压电流、开关位置等信号输入至各开关对应的逻辑模块,再由逻辑模块控制开关完成相应分合闸操作。

    3.1 相间故障处理流程验证

    在F1处设置相间短路故障(故障起始时刻为0 s),QF21保护动作时间为0.5 s,一次重合闸时间2 s,二次重合闸时间为10 s。首先选择QS14和QS25为联络点进行仿真(序号1),再将联络点QS14切换为QS13再次仿真(序号2)。仿真结果见附录A表A1。表中的双压闭锁,指分段开关处于分位情况下检测到开关两侧均有电压而产生的合闸闭锁状态。

    3.2 接地故障处理流程验证

    在F2处设置接地短路故障(故障起始时刻为0 s),集中选线动作(QF21动作)时间和首开关动作时间均为20 s,一次重合闸时间为2 s,二次重合闸时间为10 s,QS14和QS25为联络点。首先设置接地电阻为1 000 Ω进行仿真,此时配电终端保护可动作(序号1),再将接地电阻设置为5 000 Ω,此时由集中选线装置动作(序号2)。仿真结果见附录A表A2。

    通过表A2仿真结果可以看出,当多联络运行方式调整后,本文所述方法可以自适应网络拓扑变动,完成故障隔离与恢复。仿真结果进一步说明了在小电流接地故障发生后,无论是配电终端判断故障或配合集中选线装置判断故障,本文所述方法均可完成故障处理流程。

    4 结语

    本文提出了一种基于区域序号的自适应就地型馈线自动化故障处理方法,分析了运行参数自动调整原则以及不同类型故障的处理流程,利用PSCAD/EMTDC仿真验证了方法有效性。应用该方法后,当配电网发生新增分支线、多联络运行方式调整等影响网络拓扑的变动时,无需考虑更改原有整定参数,自动化系统可自适应运行。

    本文方法中,开关逻辑动作时间与区域序号密切相关,目前区域序号是靠人工分配,后续将进一步研究区域序号的自动分配功能,以实现对更为复杂的网络拓扑结构或接线方式发生较大变化时的自动管理。
