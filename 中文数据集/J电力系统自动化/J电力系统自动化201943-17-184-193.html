    摘    要：

    计算并分析了调相机采用静止变频器启动的谐波特性, 探讨了变频启动系统的变频特性及谐波特性对调相机启机保护的影响。为滤除变频状态下的主要谐波, 提高调相机启机保护的计算精度, 提出一种改进型的定频采样变数据窗离散傅里叶变换算法。该算法对fs/f0 (采样频率/系统频率) 为非整数时的固有误差进行补偿, 并根据系统频率自适应优化每周期傅里叶变换计算的点数, 同时采用拟合函数实时计算正余弦系数表。在此基础上, 对调相机启机保护进行了改进, 并给出了动作判据。在保护装置上进行了实验验证, 结果表明改进后的启机过流保护与启机零序电压保护在不同频率下精度较高, 在变频工况下能够准确计算故障中的基波分量。此外, 启机差动保护使用峰峰值算法进行差动启动计算, 并结合改进型定频采样变数据窗离散傅里叶变换算法进行差流动作判别, 能够确保该保护的可靠性与速动性。

    变频; 同步调相机; 启机保护; 定频采样; 变数据窗; 傅里叶变换算法;

    Analysis and Improvement of Starting Protection for Synchronous Condenser in High-voltage Variable Frequency Condition

    LI Yuping SANG Jianbin BAO Minglei YANG Qingsong ZHU Yucong

    Nanjing Guodian Nanzi Grid Automation Co., Ltd.

    Abstract：

    The harmonic characteristics of the synchronous condenser using static frequency converter (SFC) starting system are calculated and analyzed, and the influence of frequency conversion characteristics and harmonic characteristics on the starting protection is discussed. In order to filter out the main harmonics in the variable frequency state and increase the calculation accuracy of the synchronous condenser starting protection, an improved fixed frequency sampling variable data window discrete Fourier transform (FFSVDW-DFT) algorithm is proposed. The algorithm compensates for the inherent error when fs/f0 (sampling frequency/system frequency) is a non-integer, and optimizes the points number of the Fourier transform algorithm adaptively in each cycle according to the current system frequency. At the same time, the fitted function is adopted to calculate the sine and cosine coefficient table in real time. On this basis, the synchronous condenser starting protection is improved and the action equation is given. The improved starting protection algorithm is experimentally verified on the protection device. The results show that the starting protection using the improved FFSVDW-DFT algorithm has high precision. The fundamental component during the fault could be accurately calculated under the variable frequency condition. In addition, the peak-to-peak algorithm is used for the differential start calculation in the proposed starting protection method by combining with the improved FFSVDW-DFT algorithm for the differential action calculation. Thus, the reliability and rapidity of the protection could be both ensured.

    Keyword：

    variable frequency; synchronous condenser; starting protection; fixed frequency sampling; variable data window; Fourier transform;

    近年来随着特高压直流输电快速发展, 电网的“强直弱交”特性较为突出, 尤其对于直流多馈入受端电网, 存在多回直流换相失败、电压支撑不足等问题[1,2]。其中受端电网的无功补偿问题可通过配置大型同步调相机解决[3,4], 在交流电站, 就无功补偿问题, 大型同步调相机同样适用[5]。大型同步调相机通常使用静止变频器 (static frequency converter, SFC) 启动, 该启动方式控制精准、并网冲击小, 配合同期装置能够实现软启动[6]。

    调相机变频启动时投入了励磁, 且启动时间较长, 因此调相机启机保护不容忽视。现有大型同步调相机的启机保护设计较少考虑变频状态下的谐波影响, 为了提高启机保护性能, 启机保护算法有改进的必要[7,8]。启机保护采用的频率无关算法有多种实现方式, 主要包括峰值原理、变采样率原理以及定频采样原理等。

    峰值原理即检测采样数据中同向过零点之间的最大值与最小值, 进而计算基波幅值, 该方法较为简单, 无须测频, 但算法受谐波影响较大, 无法滤除整次谐波[8], 同时算法本身不具备抗电磁干扰能力, 因此在SFC启动系统中直接使用性能欠佳。

    变采样率原理即预先检测系统频率, 实时调整采样率, 实现一周期内数据点数恒定, 在程序中实现离散傅里叶变换 (discrete Fourier transform, DFT) [9,10]。该方式在变频工况仍具备工频下常规傅氏算法的同等性能, 但需要保护装置支持实时更改采样率。由于目前继电保护装置多为定频采样, 且数字化电站的合并单元输出的采样报文多为定频方式, 因此, 变采样原理在保护装置上进一步改造较为困难。

    定频采样算法中, 文献[11]在频率偏移上进行改进, 利用带有窗口函数的DFT, 能够减少频率偏移带来的测量误差, 文献[12]在基于准同步采样下DFT相角计算的精确误差表达式, 得到精确的频率、幅值、相位计算公式, 但文献[11,12]对于大范围偏离额定频率的变频启动工况时并不适用。实际上, 定频采样算法更多的是根据系统频率实时修正数据窗长度, 在新的数据窗中即可实现精确的幅值、相位计算。文献[13]在定频采样下进行频率跟踪, 采用均方根算法, 考虑了采样频率不是信号频率整数倍的影响, 可以获得较精确的有效值, 但无法计算基波及各次谐波。文献[14,15]均采用了定频采样变数据窗的傅氏算法, 根据频率调整傅氏算法的数据窗长度, 但当频率较低时, 一周期点数N较多, 算法的计算量巨大, 当频率较高时, 点数N又较少, N为非整数时误差较大。文献[16]将自适应窗长与对称平移采样窗相结合, 提出基于对称平移1/12采样窗的自适应DFT相量算法, 该原理同样基于测频得到自适应窗长, 动态误差小, 但是在继电保护装置环境下, 其计算过程仍较为复杂。

    针对将现有文献中的算法直接用于调相机启机保护存在精度、效率以及可靠性等方面的问题, 本文对调相机SFC启动系统的谐波特性及变频特性进行分析, 提出一种改进型的定频采样变数据窗离散傅里叶变换 (fixed frequency sampling variable data window-discrete Fourier transform, FFSVDW-DFT) 算法, 对调相机启机保护的计算精度及效率进行优化, 并进行实验验证。

    1 调相机SFC启动系统的谐波及其对启机保护的影响

    本文分析基于动模试验, 实验中的SFC由实际工程供货厂家提供, 变频器拓扑结构为12-6脉波, 支持输出最大电压为20 kV、输出最大功率为60 MW;动模发电机的额定电压为400 V, 额定容量为15 kvar。启动主接线图如图1所示。

    调相机带励磁变频启动过程中, 转子的转速亦同步上升, 调相机电气量中的基波与谐波分量均处于变频过程, 启机保护需充分考虑变频启动过程中的变频特性与谐波特性。

    2 调相机启机保护算法改进

    变频启动过程中无论定子接地还是定子相间故障, 基波分量才是故障特征量, 因此启机保护宜将变频状态的谐波滤除。本文在不改变采样频率的前提下, 提出一种改进型FFSVDW-DFT算法, 仍通过预先测频, 在变数据窗的基础上实现傅氏算法, 在计算数据窗时采用精确长度, 并动态优化数据窗内的计算点数, 同时采用拟合函数实时计算正余弦系数表。

    常用测频方法有傅氏测频与过零点测频。目前, 关于测频的算法研究较多, 主要是在傅氏测频的基础上进行改进[17,18], 虽然傅氏测频不易受整次谐波的影响, 但其对于系统频率未严重偏离工频时更为适用, 对于大范围变频的系统, 傅氏测频需要进行多次迭代计算。

    过零点测频算法的测频速度较快, 但当存在谐波时过零点检测可能受影响, 文献[19]为抽水蓄能SFC拖动系统下的测频功能设计了数字低通滤波器, 改善了机桥侧电压波形。考虑保护装置对测频算法的响应速度、计算效率要求较高, 本文仍采用低通滤波与过零点插值测频相结合的方式进行频率跟踪, 优先采用电压测频, 如果三相电压均消失, 则转为电流测频 (电流测频时需要再叠加差分滤波) 。具体的测频方法本文不再赘述, FFSVDW-DFT算法需要假设系统频率已被准确测量。

    2.1 常规FFSVDW-DFT算法

    文献[15,20]均采用常规FFSVDW-DFT算法, 根据当前系统频率, 计算出一周期采样点数的整数近似值, 并通过查表或调用正余弦高级函数获取傅氏算法的正余弦系数, 最后计算相量的实部与虚部。为将改进后的算法与常规FFSVDW-DFT算法进行对比, 先对常规算法进行说明。

    2.3 调相机启机保护的改进

    调相机启机保护在采用改进型FFSVDW-DFT算法时, 需确保测频的准确性, 尤其要注意三相短路后电压突降为0的故障情况, 因此, 如果检测到三相电压采样值过低, 需要立即切换到电流测频。

    调相机启机过流保护、调相机启机零压保护作为启动期间的后备保护, 由于动作延时设置较长, 可以直接使用改进型FFSVDW-DFT算法设置动作判据, 启机过流保护动作条件为:

    Iφ′>Iset (16)

    式中:Iφ′为调相机机尾相电流, 下标φ为相别;Iset为启机过流保护定值。

    启机零压保护动作条件为:

    U0>U0, set (17)

    式中:U0, set为零序电压定值。

    调相机启机差动保护作为变频启动时的主保护, 动作时间较短, 考虑短路故障后第1个周期内调相机频率可能存在跃变, 如果直接使用改进型FFSVDW-DFT算法进行差动保护的启动计算并不准确, 因此采用峰峰值算法判据作为启动条件, 表达式为:

    Id, φ′>0.4Id, set (18)

    式中:Id, φ′为峰峰值算法计算得到的差流, 即先求得差流波形中两个同向过零点之间的最大值与最小值之差, 再归算得到差流有效值;Id, set为启动差流定值, 启动系数固定设置为0.4。

    调相机启机差动保护的动作条件为:

    Id, φ>Id, set (19)

    式中:Id, φ为改进型FFSVDW-DFT算法计算得到的差流。当启动条件 (18) 满足后, 动作条件 (19) 持续满足一个电气周期, 则启机差动保护动作。

    3 改进后启机保护的验证

    为了验证本文所提算法的有效性, 对改进后的启机保护进行了实验验证。保护装置采用MPC8247的硬件核心以及VxWorks的嵌入式操作系统, 保护算法采样频率fs=1 200 Hz。算法中的正弦函数、余弦函数采用简化函数进行替换。继电保护测试仪型号为PW466, 实验主要采用测试仪合成波形以及物理动模故障波形回放的方式进行验证。

    分别为采用常规FFSVDW-DFT算法的启机过流保护与改进型FFSVDW-DFT算法的启机过流保护输入阶跃电流, 调整电流幅值由0 A突变为2 A, 当频率分别为34 Hz, 32 Hz, 15 Hz时, 电流幅值计算结果如图3所示。图3显示改进前的启机过流保护计算得到的幅值均存在明显误差, 且误差与Nf的小数部分直接相关。当小数部分大于0.5时, 计算值偏大;当小数部分小于0.5时, 计算值偏小。改进后的启机过流保护启机过流保护对Nf的小数部分进行了补偿, 能够计算得到稳定、准确的幅值。

    4 结语

    本文计算并分析了调相机在SFC拖动时的谐波特性, 阐述了变频启动过程中的变频特性及谐波特性对调相机启机保护的影响。在此基础上, 为提高启机保护性能, 提出一种改进型的FFSVDW-DFT算法, 在此基础上, 给出了启机保护的动作判据, 并进行了实验验证。

    本文通过计算以及实验得到以下结论:改进型FFSVDW-DFT算法考虑了采样频率不能被调相机频率整除的情况, 精度较高, 并根据变频特征优化了效率, 适用于对效率敏感的调相机保护装置;启机零压保护采用改进型FFSVDW-DFT算法后能将变频状态的3次谐波电压滤除, 有利于提高启机零压保护的计算精度与灵敏度;变频启动时启机过流保护采用FFSVDW-DFT算法能够更准确地算出故障电流的基波幅值;启机差动保护采用峰峰值算法进行差流启动计算, 并使用改进型FFSVDW-DFT算法进行差流动作判别, 能够确保启机差动保护的可靠性与速动性;为保证变频环境下启机保护相量计算的准确性, 适用于不同工况、不同故障状态的测频算法仍有待进一步研究。
