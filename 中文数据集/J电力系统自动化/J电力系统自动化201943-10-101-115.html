    基于随机矩阵谱分析的输电线路故障检测算法

    童晓阳 余森林

    西南交通大学电气工程学院

    摘    要：

    针对基于基波相量体系和时域波形相似度的故障检测算法抗干扰能力不强, 提出了一种基于随机矩阵谱分析的输电线路故障检测算法。首先, 利用故障线路两侧电流故障分量波形几乎反向的特性, 通过滑动的定长时间窗, 将线路的一侧电流故障分量与另一侧电流故障分量相反量形成原始矩阵, 进行复制、平移、加噪声等处理构造随机矩阵, 计算其平均谱半径, 并进行谱分析, 给出了线路的两个故障检测判据。然后, 结合随机矩阵理论的单环定理, 给出了阈值的确定办法, 分析了各种因素对所提算法的影响。最后, 利用IEEE 39节点系统的大量仿真算例验证了算法的有效性, 实验表明该算法不受故障点位置、线路重载、非全相运行等因素的影响, 在系统发生振荡时也不会误切除正常线路, 耐受过渡电阻能力强, 对信息的同步性要求不高。

    关键词：

    随机矩阵; 故障检测; 过渡电阻; 系统振荡; 异常数据;

    Fault Detection Algorithm for Transmission Lines Based on Random Matrix Spectrum Analysis

    TONG Xiaoyang YU Senlin

    School of Electrical Engineering, Southwest Jiaotong University

    Abstract：

    Considering the lack of anti-interference ability of fault detection algorithm based on the fundamental wave phasor system and similarity of time-domain waveform, a novel transmission line fault detection algorithm based on random matrix spectrum analysis is proposed. Firstly, using the almost reverse characteristics of waveforms for the current fault components at both ends of the line, the original matrix is formed by a fixed time window that consists of current fault component on one side and opposite current fault component on the other side. A random matrix is constructed by copying, translating, and adding noise, its mean spectral radius is calculated and spectrum analysis is taken, which brings about two criteria of fault detections for lines. Then, the single ring theorem of the theory for random matrix is used to set the threshold and the influence of various factors on the proposed algorithm is analyzed. Finally, a large number of simulation examples in the IEEE-39 bus system prove the effectiveness of the algorithm. The simulation results show that the algorithm is not affected by the fault location, non-full-phase operation and heavy load and it will not cut off the normal line when the system oscillates. It also has a strong ability of resisting the transition resistance and does not require high information of synchronization.

    Keyword：

    random matrix; fault detection; transition resistance; system oscillation; abnormal data;

    现代电网规模大、结构复杂、交换容量大, 故障发生机率增加[1], 快速检测与切除故障线路是保证电网安全运行的重要措施。已有保护算法多数是在基波相量体系下, 受制于滤波算法数据窗, 抗干扰能力不足, 在异常数据较多的情况下, 已有算法无法对错误数据进行修正[2], 造成保护的拒动或误动。

    近年来学者在时域中以电流波形研究故障线路检测算法[2,3,4,5,6]。文献[2]计算线路两侧电流的豪斯多夫距离评价两侧波形的相似度, 通过增设条件提升算法抗异常数据能力。文献[3]利用线路两端的电流故障分量计算灰色关联度及其相对大小识别故障。文献[4]分析区内故障与区外故障的差动电压和时间的一阶导数与差动电流的关系, 利用相关系数构建基于时域相关性的保护判据。文献[5]对线路两侧电流采样值进行绝对值运算后获得其相关度, 与给定阈值相比较来判断故障。但上述算法均是利用单一时刻数据构造保护判据, 没有充分利用历史数据。

    随机矩阵理论 (random matrix theory, RMT) 在1928年被提出, 它说明了当系统中只存在无序的噪声和量测误差时, 可由系统的状态变量构成的矩阵得到随机的统计特性;当状态变量含有非噪声信息时, 由状态变量构成的矩阵将出现变化, 其对应的统计特性也会改变[7]。有限维随机矩阵理论将这门分析技术从理论分析推向工程应用[8]。最近几年, 随机矩阵理论在金融去噪、信号处理、智能电网等领域引起了广泛研究[9,10,11,12,13,14,15,16]。

    文献[11]利用随机矩阵理论对电网大数据进行分析, 提出了一种新的数据分析架构。文献[12]将节点电压幅值和支路有功功率作为状态变量, 利用增广矩阵法构造数据矩阵, 通过随机矩阵理论分析了各节点有功负荷对电网运行状态的影响。文献[13]只将节点电压作为状态变量应用于随机矩阵中, 减少了数据种类, 缩短了扰动定位的时间。文献[14]运用随机矩阵理论和相量测量单元 (PMU) 数据结合, 研究了一种由数据驱动的电网暂态稳定分析方法。目前, 将随机矩阵理论应用于输电线路故障检测的研究尚不多见。

    在已有研究成果的基础上, 本文尝试引入随机矩阵理论到输电线路故障检测。选取和分析了谱分析的输入量, 研究应用随机矩阵理论于故障线路检测的方法和故障判据。

    1 随机矩阵理论

    1.1 单环定理

    任意数据矩阵X^={x^i, j, 1≤i≤m, 1≤j≤n}可按归一化公式变换为非Hermitian矩阵X∼[12]:

    x∼i, j= (x^i, j-μ (x^i) ) σ (x∼i) σ (x^i) +μ (x∼i) (1)

    式中:x∼i, j为非Hermitian矩阵的元素;i=1, 2, …, m;j=1, 2, …, n;μ (·) 和σ (·) 分别为求期望和求方差函数, μ (x∼i) =0, σ (x∼i) =1。

    X∼的奇异值等价矩阵Xu为:

    Xu=UX∼X∼Η (2)

    式中:U∈Cm×m, 是由魔方阵经过奇异值分解得到的Haar酉矩阵, m阶魔方阵可由MATLAB语句生成[16];上标H表示矩阵的共轭转置。

    X∼X∼Η=XuXuΗ (3)

    1.2 数字特征的选取

    随机变量的数字特征能够反映随机变量的统计特性。随机矩阵的特征值λi (i=1, 2, …, n) 为随机变量, 有相应的线性特征值统计量 (linear eigenvalue statistics, LES) 刻画其分布, LES定义为[19]:

    Νn (φ) =∑i=1nφ (λi) (7)

    式中:φ (·) 为测试函数, 不同的φ (·) 可构成不同的LES。

    平均谱半径 (mean spectral radius, MSR) 作为随机变量λi (i=1, 2, …, n) 的一种数字特征被广泛应用在单环定理中。MSR值r是复平面中所有特征值到原点的距离, 即

    r=1n∑i=1n|λi| (8)

    式中:|λi|为随机矩阵各特征值的模;n为特征值的个数。

    MSR的定义中包含了所有的特征值, 能有效地反映随机矩阵的统计特性, 很多文献将它作为数据相关性分析的指标[10,11,12,13,14,15,16]。本文将利用MSR的变化来检测线路是否故障。

    -ΔΙ⋅iΔΙ⋅j=- (1-a) Ζ+ΖjaΖ+Ζi (11)

    由式 (11) 易知, 当线路内部故障时, ΔΙ⋅i与-ΔΙ⋅j的相对大小和相角差只受线路两端背侧等效阻抗、线路阻抗Z及故障位置的影响。高压电网的线路阻抗和系统背侧阻抗几乎为纯感性, 式 (11) 的相位arg (ΔΙ⋅i/ (-ΔΙ⋅j) ) ≈180°[20]。

    当线路外部发生故障时, 由基尔霍夫电流定律规定的电流故障分量正方向易知|ΔΙ⋅i|和|-ΔΙ⋅j|基本相等, 而arg (ΔΙ⋅i/ (-ΔΙ⋅j) ) 的值约等于零。

    根据以上分析可知, 正常线路与故障线路两端电流故障分量ΔΙ⋅i与-ΔΙ⋅j的波形有着明显差异。区外故障时ΔΙ⋅i与-ΔΙ⋅j的波形基本重合;而区内故障时ΔΙ⋅i与-ΔΙ⋅j对应时刻的采样值大小不等、方向相反。区内故障和区外故障时, 线路两端的电流故障分量波形如附录A图A1 (a) 和 (b) 所示。

    2.2 故障检测矩阵的生成

    随机矩阵理论需要高维的数据矩阵来进行相关性分析, 下面将给出高维数据矩阵的形成方法。

    将线路两端电流故障分量序列组成一个2×n阶矩阵Iorig:

    Ιorig= [Δi1 (k1) Δi1 (k2) ⋯Δi1 (kn) -Δi2 (k1) -Δi2 (k2) ⋯-Δi2 (kn) ]2×n          (12)

    式中:Δi1 (k) 和-Δi2 (k) 分别为线路一侧电流故障分量和对侧电流故障分量相反量;k=k1, k2, …, kn为采样时刻。

    对Iorig的两行数据复制m-1次, 向下平移, 扩展为2m×n阶矩阵Icopy:

    Ιcopy= [Δi1 (k1) Δi1 (k2) ⋯Δi1 (kn) -Δi2 (k1) -Δi2 (k2) ⋯-Δi2 (kn) ⋮⋮⋮Δi1 (k1) Δi1 (k2) ⋯Δi1 (kn) -Δi2 (k1) -Δi2 (k2) ⋯-Δi2 (kn) ]2m×n          (13)

    将一个2m×n阶的高斯白噪声矩阵叠加到Icopy上, 使矩阵中各元素变为随机数。记叠加高斯白噪声后得到的随机矩阵为Ior, 即

    Ior=Icopy+ANgas (14)

    式中:Ngas为2m×n阶噪声矩阵且Ngas的各个元素均为服从标准正态分布的随机数;A为噪声矩阵的幅值。每将定长时间窗向前移动一个采样点, 会得到一个Ior。

    噪声幅值A设置的过大会淹没线路故障时采样值序列的特征;设置的太小则不能有效抵消由线路分布电容对正常线路两侧电流故障分量的影响。半补偿方式可消除或部分消除分布电容对正常线路两端电流故障分量的影响, 因此, 本算法将噪声幅值A设置为半补偿电流幅值的一半[16], 即

    A=12UΝY2 (15)

    式中:UN为额定相电压;Y为线路导纳, 由具有均匀分布参数高压线路的π形等效网络计算得到。

    将Ior按照式 (1) 至式 (5) 进行矩阵变换, 得到标准矩阵积Ifinal, Ifinal∈R2m×2m, 作为本文需要的故障检测数据矩阵。

    式中:det (·) 为求方阵的行列式;λE为对角矩阵, 对角线上的值为λ。λi为满足式 (16) 的一个解。

    2.4 基于随机矩阵谱分析的保护判据

    根据2.1节分析可知, 除有限个等值点, 故障线路两端电流故障分量对应时刻互为正负, 相差较大。因此, 故障线路的Ifinal中各行元素的统计特性将不是相同的高斯分布, Ifinal的特征值主要分布在内环内部, 即MSR将小于理论计算内环半径值。而正常线路两端的电流故障分量ΔΙ⋅i和-ΔΙ⋅j基本处处相等, Ifinal中各行元素会具有相同的高斯分布, MSR的值将介于两个圆环半径之间。

    2) 保护判据抗过渡电阻能力

    由式 (11) 可知, 两侧电流故障分量对应时刻采样值的相对大小与过渡电阻无关, 即故障检测矩阵中各行元素的关联性不受过渡电阻影响, 因此本文提出的保护判据抗过渡电阻能力较强。

    3) 动作速度的分析

    本算法通过定长时间窗对一段时间内的采样数据进行分析并计算出该线路该时刻的MSR。某时刻MSR的计算矩阵Ifinal包含该线路此时刻之前的历史采样值。故障发生后Ifinal将含有不是同分布的故障数据, 但Ifinal中故障数据需要积累到一定的量后才能使矩阵各行元素之间的关联性发生变化, 因此本算法需要经过一定的时间才能判断故障线路。算法的动作速度与Ifinal由几个周期的数据组成有关, 在满足数据维数的要求下, 提高采样率, 使一个周期内包含更多的采样数据, 算法的动作速度将加快。本文选取的采样周期数为2。经过大量仿真实验可得到, 本文算法的动作时间约为18 ms。

    4) 抗异常数据能力分析

    当数据在传输中受到噪声干扰或连续出现几个离散脉冲时, 已有的基于单一时刻采样数据构造的波形相似度保护算法将不能保证得到正确的识别结果, 异常数据会使保护动作的可靠性降低。本保护算法是利用一个定长时间窗内所有的采样值数据进行关联性分析, 几个异常脉冲和高斯白噪声不会改变时间窗内数据的统计特性, 即线路的MSR不会发生变化。因此, 本保护算法有良好的耐受异常数据能力。

    5) 信息同步对算法的影响

    已有算法大多需要精确的时间同步来保证算法的有效性。对于正常线路, 本算法在形成故障检测矩阵时已对原数据矩阵叠加了高斯白噪声, 各元素叠加的噪声将减少由时间不同步引起的采样数据差异;而故障线路两侧电流故障分量的采样值数据除零点外互为正负, 一定程度内的时间不同步不会影响保护算法结果的正确性。本文所应用的π形等值输电线路模型一般应用于中等长度线路 (100～300 km) 。在分布参数模型中, 两侧电流存在相位差, 本文通过叠加相应的噪声可减小由电容电流造成的两端电流不同步的影响, 能够正确区分正常线路和故障线路。

    附录C图C6为正常线路L28-29的多个非同步时差仿真结果, 由图可知, 本算法能够承受2 ms的不同步时间差, 对同步性的要求并不高。

    4 结语

    本文提出了基于随机矩阵谱分析的线路故障检测算法, 利用实时和历史数据序列扩展成矩阵, 求得MSR进行线路故障检测, 具有以下优点。

    1) 本算法具有良好的抗过渡电阻能力。

    2) 本算法不受故障位置、非全相运行、潮流转移影响, 在系统振荡时也不会误切除正常线路。

    3) 本算法针对一定时间窗的电流故障分量信号进行分析, 有良好的抗脉冲噪声能力和一定的抗白噪声能力。

    4) 线路两端电流故障分量的相位同步误差不超过22° (约1 ms) 时, 不会造成正常线路的误判, 本算法对同步性的要求并不高。

    本文需要大量数据构成故障检测矩阵, 很难达到暂态量超高速保护的动作速度要求。用更少的采样数据更准确地估计总体分布, 将有助于本算法动作速度的提升, 这是本文下一步要开展的工作。
