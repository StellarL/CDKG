    摘    要：

    针对描述逻辑无法表示语义网中模糊和非单调知识的问题, 在模糊描述逻辑f-SHOIQ基础上增加弱否定构造算子, 提出模糊非单调的描述逻辑f-SHOIQN。使用弱否定标记模糊非单调原子概念, 进而表示模糊非单调规则。将模糊非单调规则引入模糊描述逻辑f-SHOIQ, 用来表示模糊和非单调知识。构建f-SHOIQN的知识库, 给出该知识库中模糊非单调知识和模糊单调知识处理方式。为处理模糊非单调知识库中规则的竞争并满足描述逻辑中概念包含、相等的问题, 提出f-SHOIQN中竞争规则的优先级判定算法。分析f-SHOIQN具有的性质, 并给出相关证明。

    f-SHOIQN:A Fuzzy and Non-monotonic Description Logic

    WANG Xing ZHAO Qiaoxia CHEN Ji LI Jia

    School of Electronics and Information Engineering, Liaoning Technical University Graduate School, Liaoning Technical University

    Abstract：

    There is a lot of fuzzy and non-monotonic knowledge in the semantic Web, which cannot be expressed by the current description logic.Aiming at the problem, this paper adds weak negative constructor and proposes a fuzzy and non-monotonic description logic f-SHOIQN based on the fuzzy description logic f-SHOIQ.The weak negative is used to mark fuzzy and non-monotonic atomic concepts, and further represent fuzzy and non-monotonic rules, which are then introduced into fuzzy description logic to represent fuzzy and non-monotonic knowledge.The knowledge base of f-SHOIQN is constructed, and the processing methods on fuzzy non-monotonic knowledge and fuzzy monotonic knowledge are given.In order to deal with the problem of rule competition while satisfying the rule inclusion and rule equivalence in the description logic, this paper proposes a priority decision algorithm about competition rules in f-SHOIQN.analyzes the properties of f-SHOIQN and gives relevant proofs.

    Keyword：

    semantic Web; description logic; weak negative; fuzzy and non-monotonic knowledge; constructor; competition rules;

    0 概述

    描述逻辑[1]是一种基于对象的知识表示, 它建立在概念和关系之上, 具有形式化语义和可判定性, 作为本体语言的逻辑基础, 使语义Web具备可理解性。基于较强的表达能力和有效的推理能力, 描述逻辑引起了人们的广泛关注, 并且应用于知识表示[2,3,4]、信息检索[5,6]、应用系统[7]等领域。

    现有的描述逻辑无法对模糊和非单调的知识进行表示。事实上, 现实世界存在大量模糊、受限的信息, 人们很难对其进行准确归类和全面掌握, 并且当新的知识加入时, 原结论可能被推翻。大量的模糊非单调信息被广泛地应用于各个领域, 例如, 在中国刑法中, 刑法规定是具体有限的, 而现实生活中的案例千变万化, 某个刑法案例对应具体的法条是模糊的, 在原有的刑法结论中加入新的条件, 原先的结论可能被推翻。因此, 如何对模糊非单调信息进行表示和推理变得十分必要。

    本文对模糊描述逻辑f-SHOIQ[8]进行扩展, 提出能够处理模糊非单调知识的描述逻辑f-SHOIQN。使用弱否定∸标记弱否定概念或角色, 进而表示模糊非单调规则, 引入模糊非单调规则表示描述逻辑中模糊非单调知识, 提出模糊规则优先级判断算法, 解决因非单调带来的规则库不一致的问题, 并研究f-SHOIQN的性质。

    1 描述逻辑基础

    描述逻辑通过增加构造算子来增强表达能力和算法复杂性。为了对模糊信息进行表示, 文献[9]在模糊描述逻辑f-ALC中加入具体域 (D) 构造算子, 提出模糊描述逻辑f-ALC (D) 。文献[10]在f-ALC的基础上增加了传递角色 (S) 、角色层次 (H) 、逆角色 (I) 、数量约束 (N) 构造算子, 提出模糊描述逻辑f-SHIN。随后, 文献[11]在f-SHIN的基础上增添枚举概念 (O) 构造算子, 提出模糊描述逻辑f-SHOIN。文献[8]在f-SHOIN的基础上引入了基数约束 (Q) , 提出模糊描述逻辑f-SHOIQ。文献[12]在D-S证据理论的基础上加入元本体, 提出了D-S模糊描述逻辑。尽管上述的模糊描述逻辑通过添加构造算子来增强其表达模糊信息的能力, 但无法对非单调知识进行表示。

    国内外学者做了大量的描述逻辑对非单调知识的研究工作。文献[13]在ALC+T的基础上引入最小模型的语义, 提出非单调描述逻辑ALC+Tmin, 提高了例外实例继承的推理能力。文献[14]在ALC+Tmin基础上增添了关系优先权, 提出非单调描述逻辑ALC+Tmin, 解决了受限实例属性阻塞继承的问题。文献[15]通过对古典模型的扩展, 提出一种可废止的描述逻辑, 并给出缺省知识的论证。文献[16]将本体加入描述逻辑DL-LiteA, 提出修复数据不一致的非单调逻辑描述, 解决了规则和本体在相互作用时不一致的问题。文献[17]引入冲突检测机制, 提出非单调描述逻辑DLN, 消除了传统非单调描述逻辑中的继承阻塞、封闭世界假设效应、超出正常范围等缺陷。以上研究虽然弥补了非单调知识表示和推理的缺陷, 但对模糊信息的处理却不适用。

    描述逻辑表达能力越强, 其复杂性越高, 推理语义Web中的知识越困难。目前, 较先进的描述逻辑为f-SROIQ, 它在f-SHOIN的基础上增加了XML的语法和语义, 用来表示和推理语义Web模糊领域中的多类型文件。f-SHOIQ通过增加传递性和基数约束构造算子, 提高模糊描述逻辑表达能力。f-SHOIQN主要目的是处理语义Web中的文本文件, 并且需要f-SHOIQ中的角色或概念的传递性和基数约束的条件, 所以, 本文在f-SHOIQ的基础上进行扩展, 提出模糊非单调描述逻辑f-SHOIQN。

    2 模糊非单调描述逻辑f-SHOIQN

    在介绍模糊非单调描述逻辑之前, 需要引入弱否定[18]∸对模糊非单调的知识进行表示和处理。强否定 ¬和弱否定∸分别为2种不同否定的表达形式。其中, ¬用于准确知识的对立面, 即显示的完全否定, ∸表示非真, 用于受限或不完全的知识, 即暂时不清晰、不能证明的信息。将∸引入模糊描述逻辑, 标记弱否定的概念或角色, 进而表示模糊非单调的信息。

    2.1 语法和语义

    与经典的描述逻辑类似, 在f-SHOIQN中, 本文使用C和D表示概念, R和S表示角色, I为个体, a, b∈I, n∈N。为了与强否定的概念名 ( ¬C) 和关系名 ( ¬R) 加以区分, 弱否定概念名和关系名分别用∸C和∸R表示, 并且对于任意的∸C和∸R, 其同样为f-SHOIQN的概念名和关系名。f-SHOIQN语法定义如下:

    模糊非单调描述逻辑f-SHOIQN的语义由模糊解释FI= (ΔFI, ·FI) 给出, 其中, ΔFI是解释论域, ·FI是解释函数, n∈[0, 1], 则:

    1) 对于∀ a, 模糊解释函数·FI将a映射到ΔFI的一个元素, 即aFI∈ΔFI。

    2) 对于∀C, 模糊解释函数·FI将C映射到隶属函数, 即CFI:ΔFI→[0, 1]。

    3) 对于∀R, 模糊解释函数·FI将R映射到隶属函数, 即RFI:ΔFI*ΔFI→[0, 1]。

    4) 对于∀∸a, 模糊解释函数·FI将∸a映射到ΔFI的一个元素, 即∸aFI∈ΔFI。

    5) 对于∀∸C, 模糊解释函数·FI将∸C映射到隶属函数, 即∸CFI:ΔFI→[0, 1]。

    6) 对于∀∸R, 模糊解释函数·FI将∸R映射到隶属函数, 即∸RFI:ΔFI*ΔFI→[0, 1]。

    2.2 f-SHOIQN模糊非单调规则

    一个模糊非单调规则形如:

    ∧i=1nΡi (xi) *ui∶∸C (xk) *uk→ω    (R (yj) *uj) (R° (yj) *uj)

    其中, Pi为任意的普通概念, ∸C为弱否定的概念, R (yi) *uj、R° (yi) *uj 分别为模糊单调规则结果和模糊非单调规则结果, ui、uj、uk为模糊权重, ω为该规则中弱否定原子概念对应的模糊单调规则集。

    模糊非单调规则的含义为:条件P1 (x1) *u1∧P2 (x2) *u2∧…∧Pn (xn) *un在执行模糊单调规则情况下得到模糊结论R (yi) *uj。然后, 将弱否定原子概念的条件加入上述规则, 该规则变成了模糊非单调规则, 此时R (yi) *uj的结论反而被推翻, 即原有的结论不成立。如果想得到模糊非单调规则的结论, 需要对弱否定原子在该模糊非单调规则的情况下进行分析, 得到不确定或不完整信息可能对应的规则ω, 根据条件从ω确定与弱否定原子概念相应的模糊非单调规则信息, 执行新的条件才能得到模糊非单调规则的结果R° (yi) *uj, 并对模糊非单调规则解释如下:

    1) ∧i=1nPi (xi) *ui∶∸C (xk) *uk为模糊非单调规则体, (R (yi) *uj) (R° (yi) *uj) 为模糊非单调规则头, 模糊非单调规则是由模糊非单调规则体为条件, 推理得到模糊非单调规则头组成。

    2) xi、yj为常量或者变量, Pi为普通的概念或角色原子, ui、uk、uj分别Pi、∸C、R或R°规则中的权重。

    3) 模糊非单调规则表明, 对于任意的权重ui不小于模糊非单调规则的权重, 且对于任意0≤ui≤1, 才能推理得到模糊头。

    4) ω为含有弱否定概念的模糊非单调规则转换的一个或多个模糊单调规则。

    2.3 f-SHOIQN的知识库

    f-SHOIQN的知识库KB由基本知识库B-KB和模糊非单调规则知识库N-KB组成。B-KB包含基本的概念和角色、弱否定的概念和角色。N-KB在B-KB基础上, 通过模糊非单调概念之间的关系组成模糊非单调规则, 同时使规则具有概念包含、概念等价、角色传递等优点。

    B-KB是个二元组<TN, AN>, 其中, TN、AN分别为模糊非单调知识库中的TBox和ABox。TBox是一个关于公理的有限集, 由一系列概念包含和概念等价组成, 本文将概念分为一般的概念和弱否定概念, 和其他模糊描述逻辑一样, 一般的概念用C或D表示, 为了和一般概念加以区分, 弱否定概念使用∸C或∸D表示, 并且含有∸C或∸D概念的规则为模糊非单调规则。对于任意概念可以是原子概念, 也可以是复杂的概念, 一般形式为CD或C≡D和∸C∸D或∸C≡∸D。ABox是实例断言和关系断言的有限集合, 包含一般的实例断言和关系断言与弱否定的实例断言和关系断言。一般的实例断言和弱否定实例断言的一般形式分别表示为< a∶Cn>和<a∶∸Cn>, 一般的关系断言和弱否定关系断言的一般形式分别表示为< (a, b) ∶Rn>和< (a, b) ∶∸Rn>, 其中, 表示>、≥、<、≤、=、, n∈[0, 1]。

    当且仅当存在一个概念解释FI对于∀a∈ΔFI满足∸CFI (a) =n, f-SHOIQN概念∸C是可满足的。在TBox的约束下, 对于∀ a∈ΔFI且∸CFI (a) ≤∸DFI (a) , 概念∸C包含于概念∸D (即∸C∸D) 成立。当且仅当对于∀ a∈ΔFI, ∸CFI (a) ≡∸DFI (a) , 概念∸C等价于∸D (即∸C≡∸D) 成立。在ABox约束下, 对于∀ a∈I, 当且仅当∸CFI (a) n, 个体断言< a∶∸Cn>成立。对于任意的角色断言, 当且仅当∸RFI (a, b) n, < (a, b) ∶∸Rn>成立。当且仅当存在一个解释FI同时满足于TBox和ABox, 知识库B-KB是可满足的。

    N-KB是模糊非单调规则集ω—, 在推理过程中只要遇到弱否定的概念就必须调用N-KB中的规则。对于任意的ω∈ω—, 其由B-KB中的概念和角色组成, 这些概念和角色包含普通的概念和角色与弱否定的概念和角色, 并且TBox中的概念包含和概念等价、ABox中的概念断言和关系断言等功能同样在模糊非单调规则中适用。

    对于知识库中的普通概念和角色以及弱否定的概念和角色, 本文除了用不同的标记进行标记外, 对于不同的概念, 采用不同的处理方式。

    对于普通概念θ, ϑ, oC, ∸θ′, ∸ϑ′, ∸o′∸C′, ϵ′⊆ω, SR, ∸S′∸R′, 有:

    1) 若θϑ, 且ϑo, 则当且仅当∀Oθ, KB⊧Oo。

    2) 若∸θ′∸ϑ′, 且∸ϑ′∸o′, 则当且仅当∀∸θ′ ⊆ ϵ′, KB⊧∸θ′∸o′。

    3) 若θ≡ϑ, 且ϑ ≡ o, 则当且仅当∀O≡θ, KB⊧O≡ o。

    4) 若∸θ′≡∸ϑ′, 且∸ϑ′ ≡∸o′, 则当且仅当∸θ′ ⊆ ϵ′, KB⊧∸θ′ ≡∸o′。

    5) 若SR, 则当且仅当∀<a, b>∈ΔFI*ΔFI, KB⊧SFI (a, b) ≤RFI (a, b) 。

    6) 若∸S′∸R′, 则且仅当∀<a, b>∈ΔFI*ΔFI且∸R′ ⊆ ϵ′, KB⊧∸S′FI (a, b) ∸RFI (a, b) 。

    7) 若S ≡ R, 则当且仅当∀<a, b>∈ΔFI*ΔFI, KB⊧SFI (a, b) ≡ RFI (a, b) 。

    8) 若∸S′ ≡∸R′则当且仅当∀<a, b>∈ΔFI*ΔFI且∸R′ ⊆ ϵ′, KB⊧∸S′FI (a, b) ≡∸R′FI (a, b) 。

    对于任意的模糊非单调规则, 本文使用符号δ表示规则头, 则:

    1) 如果δ含有普通的概念包含公理、概念断言, 则δ∶={∧i=1nPi (xi) }。

    2) 如果δ是含有弱否定的原子概念, 则δ∶={∧i=1nPi (xi) }∪{∸C (xk) }。

    3) 如果δ是含有弱否定的原子概念并且∸C∶=∸C1∧∸C2, 则δ ∶={∧i=1nPi (xi) }∪{∸C1 (xk1) }∪{∸C2 (xk2) }。

    2.4 f-SHOIQN中模糊非单调规则优先级判定算法

    在模糊非单调规则知识库中, 规则间的竞争是不可避免的, 本文对规则优先级算法[19]进行改进, 在原有的规则优先级算法的基础上增加了概念包含和概念等价相关的规则。

    模糊非单调规则包含以下基本符号和关系:

    1) 概念集sub (c) , sub (c′) C, ∸c, ∸c′, ∸d, ∸d′∸C。

    2) 规则s, s′∈ω。

    3) 权重符号ui, uj, uk∈s, u′i, u′j, u′k∈s′。

    4) 优先级符Δ, 规则优先级、、从高到低依次降低, Δs表示冲突规则s的优先级, Δs′表示冲突规则s′的优先级。

    为了简化描述, 本文只讨论2个冲突规则的优先级判定方法, 多个规则的优先级判定方法以此类推。2个冲突规则的优先级判断方法如下:

    1) 对于要加入知识库的规则, 如果该规则与知识库中的规则没有冲突, 则为该规则赋予最高优先级。

    2) 对于任意的2个冲突规则s和s′, 如果规则s和s′中普通的原子概念和弱否定的概念完全匹配且原子个数相等, 则通过比较s和s′的权重来确定优先级, 即分别计算每个规则体原子的乘积, 然后通过比较乘积大小确定优先级。乘积大的规则优先级高, 为;乘积小的规则优先级低, 为。

    3) 如果2个冲突规则的原子概念 (可以为普通的原子概念, 也可以为弱否定的原子概念) 存在单个的包含关系, 则认为包含范围小的原子概念规则优先级高。正如当看到一只企鹅时, 首先想到的是它是企鹅, 而不是动物。

    4) 如果冲突规则原子概念之间存在互相包含的关系, 即s中的原子概念包含s′中的原子概念, 反之亦然, 同样通过权重的乘积比较冲突规则的优先级。

    2.5 f-SHOIQN的应用实例

    本节通过具体的刑法案例对模糊非单调规则进行解释。

    案例 张某为先天性中度精神病人 (用ModeratePsychosis表示) , 经过长期治疗后病状得到缓解, 基本上恢复了自制能力, 但仍然有残留的精神症状, 有时候会喃喃自语。一天在做午饭时, 他与妻子发生争吵 (具体矛盾原因不详) 。之后, 张某对妻子实施暴力, 并用刀把妻子后腰处、右前臂及左手腕砍伤 (用WristChop表示) , 致使妻子左手腕离断 (用WristSeparation表示) , 腕关节及左手功能丧失 (用Afunction表示) 。经专家鉴定, 该残疾为五级伤残。根据案例中的描述并结合刑法[20]第234条:“故意伤害他人身体的, 处三年以下有期徒刑、拘役或者管制。致人死亡或者以特别残忍 (用SeriousCruel表示) 手段致人重伤 (用SeriousWound表示) 造成严重残疾 (用SeriousDisability表示) 的, 处十年以上有期徒刑、无期徒刑或者死刑。”

    遵照刑法量刑过程, 首先, 刑法人员结合刑法知识给出正常人犯罪 (不考精神病人的情况下) 模糊刑法规则1。

    规则1 SeriousWound (WristChop) *0.8∧SeriousCruel (WristSeparation) *0.7∧SeriousDisability (Afunction) *0.9→12YearsImprisonment (MrZhang) *0.7

    其次, 根据2.4节中竞争规则的优先级判定算法, 将规则1的一般形式规则2加入刑法知识库:

    1) 调用规则优先级判定算法, 检测该规则是否与知识库中的规则发生冲突, 程序返回结果为没有冲突。

    2) 将该规则加入刑法知识库。

    规则2 SeriousWound*0.8∧SeriousCruel*0.7∧SeriousDisability*0.9→12YearsImprisonment*0.7

    然后, 考虑其他特殊条件。在本案例中该犯罪人为特殊人员 (中度精神病人) , 在当时的情况下, 没有办法确定张某犯罪时刻的精神状态, 不可以得到张某在犯罪的时候精神状态是正常或完全不正常。刑法人员不得不根据张某的病情和案例描述, 经法定程序鉴定确认, 得到张某犯罪时尚未完全丧失辨认能力的结果。根据刑法[20]第18条规定:“尚未完全丧失辨认或者控制自己行为能力的精神病人犯罪, 应当负刑事责任, 但是可以从轻或者减轻处罚。”在本案例中, 当得知张某为间接性精神病人时, 执法机关根据上述刑法规定和案例给出了减轻处罚的刑法结果, 结合刑法规则和案例, 把中度精神病人的条件加入上述规则, 同时, 根据减轻处罚的条件和模糊非单调规则的定义, 得到规则3为模糊非单调规则。因为, 当把弱否定的概念ModeratePsychosis加入规则中, 没有使规则头中12YearsImprision的结果增加, 同时, 也没有完全否定规则头, 而是部分否定了已推出的结论, 使得推理不得不重新进行。

    规则3 SeriousWound (WristChop) *0.8∧SeriousCruel (WristSeparation) *0.7∧SeriousDisability (Afunction) *0.9∧∸ΜoderateΡsychosis (ΜrΖhang) *0.6→Reduce*0.86YearsΙmprisonment (ΜrΖhang) *0.6

    同样, 得到规则3的一般形式为规则4, 并将其加入刑法知识库:

    1) 调用2.4节中竞争规则的优先级判定算法后, 返回结果为无冲突。

    3 f-SHOIQN的性质

    f-SHOIQN具有以下性质:

    定理1 如果C∸C且Cω, 则N-KB⊨∧i=1nΡi (xi) *ui∶C (xk) *uk→ω (R° (yi) *uj) 结果是唯一的。

    定理2 如果C∸C, 则N-KB⊨∧i=1nΡi (xi) *ui∶C (xk) *uk→ω (R (yi) *uj) (R° (yi) *uj) 为N-KB⊨∧i=1nΡi (xi) *ui∶∸C (xk) *uk→ω (R (yi) *uj) (R′ (yi) *uj) 的真子集。

    定理3 如果C1∸C, C2∸C, C1C2且C2C1, 则N-KB⊨∧i=1nΡi (xi) *ui∶C1 (xk) *uk→ω (R (yi) *uj) (R° (yi) *uj) ≡Ν-ΚB⊨∧i=1nΡi (xi) *ui∶C2 (xk) *uk→ω (R (yi) *uj) (R° (yi) *uj) 。

    定理4 如果 C1∸C, C2∸C, C = C1 ∩C2, 则N-KB⊨∧i=1nΡi (xi) *ui∶C (xk) *uk→ω (R (yi) *uj) (R° (yi) *uj) 与N-KB⊨∧i=1nΡi (xi) *ui∶C1 (xk1) *uk1∧C2 (xk2) *uk2→ω (R (yi) *uj) (R′ (yj) *ue) 是一致的。

    下文对f-SHOIQN满足的定理逐一证明。

    1) 定理1证明

    ∀∧i=1nΡi (xi) *ui∶C (xk) *uk→ω (R° (yi) *uj) , 设R′ (yi) *uj, R* (yi) *uj为∧i=1nPi (xi) *ui∶C (xk) *uk规则得到的2个不同结果。因为∧ni=1Pi (xi) *ui ≡ P1 (x1) *u1 ∧P2 (x2) *u2∧…∧Pn (xn) *un, 所以R′ (yi) *ujP1 (x1) *u1 ∧P2 (x2) *u2∧…∧Pn (xn) *un∧C (xk) *uk, R* (yi) *ujP1 (x1) *u1 ∧P2 (x2) *u2 ∧…∧Pn (xn) *un∧C (xk) *uk, 则∧i=1nPi (xi) *ui ⊆ R′ (yi) *uj和∧i=1nPi (xi) *ui ⊆ R* (yi) *uj 。又因为R′ (yi) *uj中∧i=1nPi (xi) *ui和R* (yi) *uj中∧i=1nPi (xi) *ui是相同, 且C∸C, Cω, 所以R′ (yi) 与 R* (yi) 是相同的。根据模糊非单调规则, R′ (yi) *uj与 R* (yi) *uj是同一个值, 假设失败, 即N-KB⊨∧i=1nΡi (xi) *ui∶C (xk) *uk→ω (R° (yi) *uj) 结果是唯一的。

    2) 定理2证明

    4 结束语

    本文研究描述逻辑中模糊和非单调知识表示, 通过增加弱否定构造算子, 增强f-SHOIQ的表达能力, 提出模糊非单调描述逻辑f-SHOIQN。使用∸标记模糊非单调的概念, 引入模糊非单调规则弥补描述逻辑无法表示模糊非单调知识的不足。同时, 为了解决模糊非单调规则知识库中的模糊非单调规则之间的冲突问题, 本文提出模糊非单调规则的优先级判定算法, 保证规则知识库的一致性, 并给出f-SHOIQN的相关性质以及证明。下一步将研究f-SHOIQN的推理算法, 设计相应的推理机。
