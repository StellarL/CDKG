    摘    要：

    使用线性回归模型预测公共建筑能耗数据时, 存在不确定性影响因素和精度偏差问题。为此, 建立一种遗传优化决策树模型。采用遗传算法优化梯度提升决策树的子树生成过程, 以模型预测的R-Square值作为迭代的评估标准, 从而达到能耗预测的目的。实验结果表明, 与传统的回归预测模型相比, 该模型预测精度较高。

    Short-term Prediction Model of Building Energy Consumption Based on Genetically Optimized Decision Tree

    DING Feihong LIU Peng LU Tun GU Ning DING Xianghua YANG Baoming DAI Wenqi ZOU Chaojun

    School of Computer Science, Fudan University Shanghai Key Laboratory of Data Science, Fudan University Shanghai Institute of Intelligent Electronics and System Research Shanghai Luban Softw are Co., Ltd.

    Abstract：

    When using linear regression models to predict public building energy consumption data, there are uncertainties and precision deviations.Therefore, a Genetically Optimized Decision Tree (GODT) model is established.The Genetic Algorithm (GA) is used to optimize the subtree generation process of the Gradient Boosting Decision Tree (GBDT) .The R-Square value predicted by the model is used as the evaluation criterion of the iteration to achieve the purpose of energy consumption prediction.Experimental results show that the prediction accuracy of this model is higher than that of the traditional regression prediction model.

    building energy consumption; regression prediction; Gradient Boosting Decision Tree (GBDT) ; Genetic Algorithm (GA) ; Genetically Optimized Decision Tree (GODT) ;

    0 概述

    随着智能电网不断普及和能耗数据的指数型膨胀, 公共建筑能耗数据的统计与管理成为能源系统及电力行业的研究重点。为收集与管理能耗数据, 许多城市建立了能源监管系统, 通过分析建筑能耗特征, 挖掘节能潜力, 并合理分配能源, 达到节约建筑设施能源的目的。

    建筑能耗相关影响因素较多, 如建筑布局因素、设备分布与关联、气候和环境、人类行为因素和能耗数据的周期性等, 导致人们在建立相关预测模型时难以选取恰当的影响因素。同时, 复杂的城市建筑结构使能耗预测存在无法消除的偏差。因此, 研究建筑能耗的变化规律是预测模型面临的难题。

    针对公共建筑能耗相关影响因素较多、传统预测方法误差较大的问题, 本文提出一种基于遗传优化决策树 (Genetically Optimized Decision Tree, GODT) 的建筑能耗短期预测模型。以某酒店能源监测平台提供的实际电耗数据为例, 分析与建筑能耗相关的影响因素, 包括历史能耗数据、相关电表能耗数据以及天气等, 给出遗传优化决策树模型。采用遗传算法 (Genetic Algorithm, GA) [1,2]对梯度提升决策树 (Gradient Boosting Decision Tree, GBDT) [3]子树生成过程的划分决策进行特征子集优化, 从而准确预测建筑设备的能耗值。

    1 相关工作

    根据预测时间不同, 能耗问题主要分为短期能耗预测、中期能耗预测和长期能耗预测[4]。其中, 精准的短期能耗预测有助于能耗管理部门合理安排发电方案, 实现电网供求平衡, 保障电力送配过程的安全。

    短期能耗预测的方法主要有传统基于线性的回归模型、基于核函数的模型、神经网络模型和集成学习模型等。传统基于线性的回归模型原理简单, 便于操作, 但是依赖于经验, 且建筑能耗影响因素较多、相关性复杂, 会存在精度偏差[5]。时间序列分析[6,7]也是一种传统的回归模型, 它根据历史数据的自相关性, 分析数据随时间随动态变化的规律, 这与能耗数据的变化特征相吻合。但是可考虑的实际影响因素比较少, 限制了模型的精准性。本文模型中采用部分具有时间序列特性的数据作为模型的特征。文献[8,9,10]提出基于核函数的支持向量回归 (Support Vector Regression, SVR) 的能耗数据预测模型, 但更大程度上依赖于特征选择、核函数选择以及参数调优。人工神经网络模型 (Artificial Neural Network, ANN) [11]具有较强的自适应能力和自主学习能力, 且具有较好的泛化能力, 但网络结构依赖于经验, 很难达到理想的拟合效果。另外, 人工神经网络也存在着学习速度慢、数据依赖性强等缺陷。

    集成学习模型[12,13]是将多个学习器集成在一起共同完成训练任务的模型, 如Boosting算法[14]、Bagging算法[15], 可以使多个学习器共同决策、各取所长, 具有较高的预测准确性。因此, 基于集成学习方法, 本文提出一种改进梯度提升决策树模型[16]。

    2 建筑能耗特征分析

    本文采用上海市浦东新区某酒店能源监测平台提供的能耗数据, 该数据集记录了2009年—2018年的建筑能耗数据, 包含49个用能设备的对应电表读数, 根据《公共建筑节能设计标准》分为照明插座用电、空调用电、动力用电和特殊用电4类。数据按时间顺序排序, 记录间隔为10 min, 经过预处理后获得每小时能耗数据。此外, 从天气网站获取了这段时间内浦东新区天气数据, 数据时间间隔为小时, 主要包括温度、湿度、风速、风向、雨水等。本文采用的实验数据日期范围为2018年3月15日—5月10日, 共65 856条能耗数据和1 344条天气数据。

    针对建筑能耗数据的短期预测问题, 将能耗数据预测的相关影响因素分为3类:建筑其他相关电表读数, 时间序列历史数据, 天气及其他因素。本节分别对3类相关特征进行分析, 主要理解特征和预测值之间相关关系, 并根据相关性快速过滤无关特征。

    2.3 天气及其他因素分析

    天气因素对于高精度的能耗数据预测具有十分重要的影响。文献[18]分析得出季节和能耗具有很高的相关性。文献[19]以干球温度、露点温度、一天中的小时、星期作为能预测模型的特征, 但是并没有对特征进行相关性分析和特征选择。

    本文获取了浦东新区每小时的天气状况数据作为候选特征, 包括温度 (T) 、湿度 (H) 、雨水 (R) 、风向 (WD) 、风力 (WP) 。另外, 由于建筑能耗跟人的行为活动有所关联, 建筑能耗在不同日期具有一定差异性, 因此也将预测日期是本周第几天 (周日期偏移DoW) 和本月第几天 (月日期偏移DoM) 作为候选特征。图3给出天气及其他因素分析结果。从图3可以看出, 温度和建筑能耗数据的相关性存在中等程度的相关, 其次是风力, 而风向、星期等因素的相关性很低。

    3 遗传优化决策树模型

    在建筑能耗预测模型中, 预筛选后的能耗特征集合仍具较高维度, 难以区分特征优劣。文献[21]证明从特征集合中探索最优子集是无法求出严密解的NP完全问题, 需要一种在可接受的时间与空间范围内求出近似解的方法。遗传算法对于建筑能耗预测问题场景是一个很好的选择。

    针对建筑能耗预测中包含高维度特征集合的问题, 梯度提升决策树的模型可以发现多种有区分性的特征以及特征组合, 利用遗传算法对cart树进行切分决策[22], 以解决建筑能耗预测中特征集合维度过高的问题。同时其具有较高预测精度和较低的算法时间损耗, 适合作为遗传算法迭代过程中适应度评估回归模型。

    另外, 遗传算法以及梯度提升决策树都可以独立于问题场景。例如, 针对建筑空调能耗和电灯能耗的预测, 本文模型可以根据两者能耗曲线动态趋势以及相关电表、环境因素进行迭代, 使模型对不同设备的能耗预测具有泛化性。

    综上, 遗传算法是一种高扩展性的评价函数启发[23]的特征优化算法, 梯度提升决策树是一种对依赖于特征划分子树的集成学习模型。针对建筑能耗预测问题, 遗传算法和梯度提升决策树具有很高的契合性。因此本文基于遗传优化决策树模型对建筑能耗进行短期预测。模型具体流程如图4所示。

    3.1 能耗特征子集映射规则

    经皮尔逊特征选择筛选后的能耗特征维度仍高达60维, 本文模型采用二进制方式对基因进行编码, 恰好可以表示建筑能耗遗传算法基因选中与未选中2种状态。对预筛选后的能耗特征集合的特征子集F={f1, f2, …, fm}进行二进制编码G={g1, g2, …, gm}, 规则如下:

    gi=encode (fi) ={1, fiisselected0, fiisnotselected          (1)

    在能耗预测模型中, 每一个特征被映射为基因串上的一个基因。当该特征被选中时, 对应基因标识1;否则为0, 其反向过程即为解码过程。

    种群初始化过程生成n条初始的随机二进制基因串, 可以理解为随机初始化能耗特征子集群, 即Population={G1, G2, …, Gn}, Gi (1≤i≤n) 代表一种遗传进化过程中考虑到的建筑能耗特征子集。

    3.2 能耗特征子集评估过程

    在建筑能耗短期预测问题中, 需要计算能耗特征子集对模型的适合度, 作为下一步能耗择优过程的评估标准。由于遗传迭代过程是一个嵌入式优化过程, 因次本文模型选择基因串映射的特征子集对梯度提升决策树模型的拟合程度作为评估标准。图5所示为能耗特征子集评估过程。

    其中, yi表示数据样本S中第j个数据的真实值, y^j表示最终模型对第j个变量的预测值, y—表示数据样本S中响应变量y的均值。

    3.3 能耗特征子集择优过程

    计算种群基因串适应度后, 需要按一定策略, 从当前种群择优挑选Q条基因串形成新的种群, 即对能耗特征的择优过程。其中, Q表示种群大小。本文选择轮盘赌方法[24]进行基因串选择。假设当前种群Population={G1, G2, …, GQ}代表Q个建筑能耗特征子集, 从种群中有放回的取出Q条基因串组成新的种群, 每一条基因串被选中的概率为:

    ΡGi=fitness (Gi) ∑j=1nfitness (Gj)           (11)

    其中, fitness (Gi) 是指第i个基因的适应度, i=1, 2, …, Q。

    由于R-Square评估值的数据分布较集中, 不同能耗特征子集适应度的差异较小, 优秀基因串与劣势基因串被选中的概率相近, 导致收敛速度过慢。假设该基因串所映射的能耗特征子集中有n个特征, 其对应的适应度按递增排序为:

    {f+d0, f+d1, …, f+dn}

    d0=0≤d1≤d2≤…≤dn

    其中, f表示基因串 (特征子集) 适应度基值 (最小值) , di表示第i个基因串的适应度和基值之差, i∈[1, n]。对每一个基因串i, 在轮盘赌中被选中概率为:

    ΡGi=f+din×f+∑j=1ndj          (12)

    根据特征子集映射的基因串适应度特征, 不同基因串的适应度差别非常小, 即f>>di, i∈[1, n], 因此ΡGi≈1n, 种群中每一个基因被选中的概率约等于1/n, 且优秀基因的优先性不明显。

    本文采用精英策略[25]来优化基因选择过程, 使种群基因串拟合度的差距更加明显, 通过每一个基因串i的适应度减去种群适应度基值, 则该基因串在轮盘赌中被选中的概率为:

    ΡGi=di∑j=1ndj=fitness (Gi) -min  (fitness (Gi) ) i∑j=1n[fitness (Gj) -min  (fitness (Gi) ) i]          (13)

    对于优势基因串, 其被选中概率会高于1/n, 对于劣势基因串, 其被选中的概率会低于1/n。因此, 本文策略保证了能耗模型的特征值具有较高的优先性。

    3.4 新特征子集生成过程

    由于建筑能耗特征的维度较高, 种群无法一次性涵盖所有可能的特征子集, 因此在遗传算法对能耗特征的选择过程中需要根据一定的随机性产生新的特征子集, 产生新特征子集的方式即基因串的变异和交叉操作。基因的变异操作指基因串有一定概率Pm在某一位基因发生位反转, 使对应特征的选中状态改变。其中, Pm为变异率。基因突变可以摆脱局部最优状态。基因交叉是指2条基因串有一定概率Pc互换部分基因, 从而产生2个新的基因串, 即新的能耗特征子集。其中, Pc为交叉概率, 它影响算法的收敛性。

    由于能耗特征维度较大, 在固定迭代次数内, 优势特征有一定几率从未选中, 使模型收敛于局部最优, 造成早熟现象, 因此需要一种策略增加基因串进化变异的可能性, 以避免早熟现象发生。

    鉴于方差可以衡量集合元素的差异性, 因此采用方差作为衡量当前种群是否发生早熟现象的标准。对于种群Population={G1, G2, …, GQ}, 其种群 (特征) 多样性可表示为:

    varΡopulation=∑j=1n[fitness (Gj) -avg (fitness (G) ) ]2n          (14)

    其中, fitness (Gj) 即第j个基因串的适应度, j=1, 2, …, Q, avg (fitness (G) ) 表示种群平均适应度。

    在初始化时可以适当调高初始交叉率与变异率, 然后对其进行自适应地调节。

    {Ρc=Ρc-α×varΡopulationΡm=Ρm-α×varΡopulation          (15)

    基于自适应的调节策略, 当种群中某种建筑能耗特征占据大多数时, 种群基因串方差会降低, 变异率和交叉率会提升, 使新个体的产生概率增大, 能涵盖更多的建筑能耗特征, 避免陷入局部最优;当种群建筑能耗特征多样性升高时, 种群基因串方差会升高, 变异率和交叉率会适当降低, 使新个体的生成趋于稳定。此方法可以保证建筑能耗特征的选择过程保持动态稳定的状态。

    根据上文分析, 精英策略和自适应性概率调节似乎存在着矛盾。前者会增加建筑能耗特征集合中的优势特征被选中的可能性, 而后者会致使最优特征被进化迭代。但两者并不矛盾, 当变异率和交叉率提高时, 若产生了包含更优建筑特征的新特征子集, 则种群演化会趋向于更优的建筑特征子集迭代;若新特征子集适应度低于种群原个体, 在选择操作中, 适应性高的建筑能耗特征子集仍然会保持优先性, 有更大的几率被选中, 且精英策略在每一次选择操作中都会执行。而概率的自适应性调节会利用权重值将概率变化控制在合理范围内, 不会产生过大的影响。

    3.5 模型应用及效率分析

    经过大量的特征子集评估、择优以及生成的迭代后, 模型的迭代过程会终止, 一般终止条件为迭代次数限制或者收敛于最优值。由于是否收敛于最优值较难评估, 因此采用迭代次数限制作为终止条件。

    当遗传优化决策树模型迭代结束, 由于遗传算法的随机性, 模型收敛后会处于小范围的震荡, 最终的结果不一定是最优解, 因此根据评估函数, 选择收敛后基于最优特征子集的最优值模型作为最终的输出模型。模型用于短期能耗预测时, 会将预测目标设备的前24 h的能耗特征作为输入, 根据最优特征子集对输入数据进行预处理, 然后输入到训练好的模型中, 最终输出预测能耗。

    模型迭代过程中涉及回归模型适应度计算过程, 因此时间损耗一定程度取决于回归模型的选择。假设执行一次回归预测的时间复杂度用函数f表示, 种群规模为Q, 迭代G代, 其时间复杂度为O (G×Q×f) 。

    对于GDBT回归模型生成cart树的过程中, 需要遍历每一个特征的每一个切点。对于深度为D的梯度提升决策树, 当样本大小为S, 特征数量为F时, 每层子树计算的时间复杂度是O (S×F) , D层树的时间复杂度是O (S×F×D) 。对于一个M棵子树的梯度提升决策树, 算法时间复杂度即为O (M×S×F×D) 。

    5 结束语

    本文建立一种基于遗传优化策略树的建筑能耗短期预测模型。对建筑能耗预测问题的时间序列历史数据、建筑其他相关电表读数、天气及其他因素进行分析和预筛选, 将其作为回归模型的输入数据集。对于筛选后的特征, 使用遗传提升决策树模型对建筑能耗进行短期预测。利用遗传算法对梯度提升决策树生成的cart子树的特征进行择优, 并以此特征集对梯度提升决策树的拟合程度作为特征迭代的适应度评估标准。实验结果表明, 本文模型准确度显著高于其他回归模型, 且对不同建筑设备能耗数据具有一定的泛化能力。由于本文模型在寻找子树切分点过程中全局遍历模型耗时较长, 因此下一步将考虑用随机搜索策略来优化最优切分的的寻找过程。此外, 将根据数据特征和分布, 分析波动情况, 以解决不同设备能耗预测收敛时的局部波动问题。
