    摘    要：

    为快速准确地找到复杂网络的社团划分结果, 引入相似度指标, 提出一种改进的社团划分算法。将网络节点拓扑结构信息作为节点的多维属性, 在不改变网络连边的情况下, 使输入网络转换为节点多属性网络, 并定义节点之间的混合相似度与社团相似度, 运用层次聚类思想得到最终的社团划分结果。在真实网络、计算机生成网络上的实验结果表明, 该算法能够发现明显的社团结构, 并且具有较高的社团划分准确率。

    关键词：

    复杂网络; 多属性; 社团划分; 模块度; 相似度;

    Community Division Algorithm Based on Similarity Index

    DING Mingzhu MA Yinghong LI Yun

    School of Management Science and Engineering, Shandong Normal University

    Abstract：

    In order to find the result of complex network community division more accurately and quickly, an improved community division algorithm introduced similarity index is proposed. This paper extracts the network node topology information as a multi-dimensional attribute for each node, converts network into a multi attribute network by extracting topology information of nodes without changing the network side, defines the concepts of mixed node similarity and community similarity, and uses the hierarchical clustering idea to get the final community division result. This paper verifies that the algorithm can find the obvious community structure, improve the accuracy rate of community division by experiment on true network and computer generated network.

    Keyword：

    complex network; multiple attribute; community division; modularity; similarity;

    0概述

    社团结构是复杂网络的一个重要特性, 社团内部节点之间的联系非常紧密, 而社团之间的联系相对比较疏松[1]。研究复杂网络的社团结构对于分析复杂网络内部的拓扑结构等都具有十分重要的理论意义。基于节点相似度的社团划分算法是一类重要的算法, 近年来基于该方法的社团划分算法也逐渐走向成熟。文献[2]提出网络中节点相似度的概念。文献[3]提出一种节点分裂式算法SGN, 提高了算法精度。文献[4]给出度与接近度的评估标准, 克服了相异性指数在度量社团内节点相似度时的不足。文献[5]将局部模块度和相似度相结合, 提出基于相似度的社团划分算法, 可避免模块度缺陷, 提高算法准确度, 降低算法计算复杂度。文献[6]针对不能快速发现最优节点加入社团的问题, 提出根据节点亲密度由大到小选择节点加入社团, 以局部模块度为指标终止局部社团扩展, 具有较好的时间效率和准确性, 适用于大规模网络的社团划分。文献[7]针对网络存在重叠现象, 提出Strsim社团检测方法, 该方法在不用事先知道网络最终划分的社团数量的情况下, 通过量化节点隶属度, 发现重叠社团结构。文献[8]结合余弦指标和RA指标, 提出顶点相似性指标, 并将其运用到推荐系统, 得到了很好的应用效果。这些算法中相似度的构建大部分是间接地利用两节点之间的共同邻居数, 衡量两节点之间的相似性, 而很少从节点自身的角度进行考虑, 并且它们所采用的社团划分算法大部分是通过单一指标来划分, 比如只是根据网络节点的相似度、边介数、聚集系数等进行社团划分。

    本文着眼于节点的拓扑结构特性, 将其作为节点属性, 在不改变网络连边的情况下, 使输入的网络重新构造为一种节点拥有多属性的网络, 而节点的拓扑结构信息不仅衡量了自身的重要性, 也包含其邻居的重要性, 能够做到综合利用网络的多种拓扑结构指标来进行社团划分。结合欧氏距离相似度E (X, Y) [9]与余弦相似度C (X, Y) [10]提出混合相似度。该相似度不仅考虑2个量之间在方向上的相似性, 而且考虑2个量之间在距离上的相似性。类比聚集系数, 构造社团之间的相似度。基于相似度指标, 本文提出一种能够对具有多属性的网络进行社团划分的算法。

    1.2 算法设计

    本文算法采用层次聚类的思想, 先将最相似的节点进行合并, 达到网络快速聚合的目的。最相似节点合并后, 节点集合变成了社团块集合, 得到初步的社团划分结果。再根据社团之间的相似程度, 逐步将相似的社团块进行合并, 反复迭代, 最终达到社团划分的目的。以空手道俱乐部为例, 初步的社团划分后, 34个节点聚合成了7个社团块, 网络中不再存在离散的点, 那么根据社团之间的相似程度将社团进行聚合, 具体的情况如图1所示, 其中, 圆代表初划分后得到的7个社团块, 水平虚线代表社团之间的相似度阈值r。

    图1 空手道俱乐部的社团划分结果   下载原图

    若社团之间的相似度阈值r>0.6, 则只有块1与块2可以合并;若阈值r>0.4, 则在块1与块2可以合并的基础上, 块3和块4之间的相似程度也达到条件, 可以合并在一起;若不对社团相似度阈值进行设定, 则任意2个社团之间都存在连边, 会发现网络将会整个的聚合到一起。模块度是衡量网络划分后社团结构是否明显的一个重要指标, 能够衡量一个网络划分的好坏, 取值一般在0.3～0.7之间, 模块度越大, 社团结构越明显, 划分效果越好[11]。为了找到网络的最优划分, 本文以模块度作为算法终止的条件。将模块度最大的划分作为网络结果的最优划分。

    在不改变原始网络连接的情况下, 通过提取网络节点的拓扑结构性质作为节点的属性, 使输入的网络变成节点拥有多个属性的网络。利用多种拓扑结构属性进行社团划分, 本身就会造成复杂度的提高, 如果能通过牺牲复杂度换取精度, 不失为一种可取方法。比如, 图2 (a) 是一个只有3个节点A、B、C以及3条边的原始网络, 对该图中的每个节点提取n种拓扑结构性质, 使得每个节点都成为具有多属性的节点, 如图2 (b) 所示。

    输入无权无向网络G (V, E)

    输出社团划分结果及对应的模块度Q

    步骤1若, 则算法结束;否则抽取节点的n种拓扑结构信息, 对网络进行重构。

    步骤2对原始数据X进行标准化:Z= (X-min) / (max-min) , Z∈[-1, 1]。

    步骤3计算节点相似度矩阵H={hij}, 其中, i=1, 2, …, n, j=1, 2, …, n。

    步骤4合并网络中所有相似程度最大的节点, 进行网络的初划分。

    步骤5由步骤4节点合并之后, 得到社团块的集合C={ci}, i=1, 2, …, m, 计算此时对应的模块度Q0。

    步骤6计算社团相似度矩阵S={sij}, i=1, 2, …, m, j=1, 2, …, m。

    步骤7设定社团相似度阈值r, 若sij>r, 则认为社团i与j足够相似, 合并为同一个社团;否则不进行合并。

    步骤8重复迭代步骤7, 直至社团模块度Q达到最大值, 算法终止。

    为了更好地验证算法的有效性, 本文运用模块度、准确率[13]以及NMI指标[14]作为检测社团划分结果质量好坏的标准。

    2 算法模拟及实例分析

    本文将算法分别应用到计算机生成网络以及目前多个公认常用的现实网络中。从划分社团的模块度、时间复杂度以及划分的社团数量上与现有算法划分结果进行对比, 分析该算法的有效性。

    2.1 可行性分析

    为了验证本文算法的可行性, 将算法应用到人工图与计算机生成的图上, 称为基准图[15]。该图共有128个节点, 1 024条边, 被均匀地分成4个顶点数都是32的社团, 顶点对之间的边是随机的, 但是内部连边概率大于外部连边概率, 并且使得网络的平均度为16。这样就生成了已知社团结构的网络, 可以用来验证算法的可行性。下文将运用该算法对网络进行社团划分。

    首先对该网络提取节点的拓扑结构信息进行重构。然后计算节点之间的相似度, 将最相似的2个节点进行两两连接, 得到初步的社团划分。初步的社团划分得到了31个社团块。其次对相似社团块进行合并, 在该实验中可以发现, 随着社团块的聚合, 模块度呈现出一种先增加后下降的趋势, 结果如图3所示。

    2.2 实例分析

    2.2.1 Zachary网

    Zachary网[16]是由34个空手道俱乐部成员组成的网络, 每个人抽象为网络中的一个节点, 人与人之间的联系抽象为网络的连边, 34人共有78条连边。对于是否增加俱乐部会费问题, 内部成员产生了分歧, 最终社团分裂成了以俱乐部主管 (节点1) 和校长 (节点33) 为中心的2个派别, 该网络被视为一个标准网络, 经常被用来检验社团划分算法的正确性。为验证本文算法的正确性以及优越性, 对Zachary网应用该算法进行社团划分。

    首先对Zachary网络提取节点的拓扑结构信息进行网络重构。对于Zachary网络, 本文算法提取了每个节点的拓扑结构信息, 比如接近中心性、聚集系数、离心率等。然后根据定义的混合相似度, 计算节点之间的相似程度, 得到节点相似度矩阵, 因为空手道俱乐部总共有34个节点, 所以得到一个34×34的相似度矩阵。

    根据节点相似度矩阵, 每个节点都可以找到与之相似程度最大的一个节点, 把它们进行两两合并, 比如节点6与节点17的相似度最大, h617=0.86, 那么就把节点6与节点17进行合并, 而节点7又与节点6的相似度最大, h76=0.94, 那么就把节点7与节点6进行合并, 以此类推。具体结果如图5所示。

    从该矩阵中可以看出, 若社团之间的相似程度阈值r≥0.6, 则只有块1与块2可以合并。若阈值r≥0.4, 则块1与块2可以合并的基础上, 块3和块4之间的相似程度也达到条件, 可以合并在一起。若不对社团相似度阈值进行设定, 则任意2个社团之间都存在连边, 会发现网络将会整个聚合到一起, 模块度为0, 不能给出最优的社团划分结果, 所以需要为社团相似度找到一个合适的阈值。给社团相似度设定一个阈值, 对应一种网络划分结果, 而每一种网络划分结果, 都会使得网络对应一个模块度值, 所以本文采用网络模块度最大化作为算法的终止条件。经过实验发现, 随着社团块的聚合, 模块度先增加后下降, 具体结果如图6所示。

    3 结束语

    本文利用节点之间的相似度构造基于相似度指标的社团划分算法, 并通过实验证明该算法的可行性及准确性。此外, 该算法无需事先知道最终划分的社团数目, 并且能够充分考虑节点的多种拓扑结构属性, 具有较高的划分准确性及较好的实用性。然而, 本文算法在进行社团划分时存在节点被错误划分的情况, 因此下一步将对相似度定义加以改进, 并对数据量更大的网络进行实验验证, 加强算法普适性。
