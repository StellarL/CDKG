    摘    要：

    miRNA是一类重要的非编码小RNA分子, 与癌症等疾病有密切的关系。目前研究者已经识别大量miRNA, 但是多数miRNA的功能仍然未知。为此, 提出一种网络表示学习的miRNA功能相似性计算方法。通过miRNA的相关数据集如目标基因和关联疾病可以有效地计算miRNA的功能相似性, 从而预测疾病相关的候选miRNA。利用不同类型生物数据集构建miRNA相关多源网络, 采用网络表示学习的方式为网络中的每一个miRNA节点学习一个特征向量, 进而使用特征向量来衡量miRNA的相似性。实验结果表明, 与DeepWalk方法相比, 该方法在同一家族的miRNA中能够取得较高的得分, 并且可以在已有的数据库中找到疾病候选miRNA验证记录。

    功能相似性; 网络表示学习; 网络嵌入; 多源网络; 特征向量;

    Research on Functional Similarity of miRNA Based on Network Representation Learning

    RUAN Lu XIONG Yun

    School of Computer Science, Fudan University

    Abstract：

    miRNA is an important class of non-coding small RNA molecules that are closely related to diseases such as cancer. Researchers have identified a large number of miRNA, but the function of most miRNA remains unknown.Therefore, a method for calculating the similarity of miRNA functions based on network representation learning is proposed.The functional similarity of miRNA can be efficiently calculated by relevant data sets of miRNA such as target genes and associated diseases, thereby predicting disease-related candidate miRNA. miRNA-related multi-source network is constructed by utilizing different types of biological data sets. The network representation learning method is used to learn a feature vector for each miRNA node in the network, and the similarity of the miRNA is measured by the learned feature vector.Experimental results show that compared with the DeepWalk method, the method can obtain higher scores in the same family of miRNA, and the disease candidate miRNA verification records can be found in the existing database.

    Keyword：

    functional similarity; network representation learning; network embedding; multi-source network; feature vector;

    0概述

    miRNA是一种内源性非编码单链RNA分子, 对基因的表达具有重要的调控作用[1]。miRNA参与动植物的细胞分化、增殖、凋亡、组织和器官形成等生命过程, miRNA的功能失调会导致各种疾病, 包括免疫系统的失调以及多种癌症[2]。

    国内外学者已经识别出数以万计的miRNA, 并通过生物医学实验方法确定一部分miRNA的功能, 但大部分miRNA的功能仍然未知[3]。研究者需要有效的方法推测可靠的候选miRNA集, 以提高实验效率。随着近年来生物数据的大量积累和开放共享, 通过生物信息学的方法预测miRNA功能的方法成为一种高效的方法[4,5]。

    研究miRNA主要的任务是确定miRNA之间的功能相似性, 即通过计算功能明确的miRNA和功能未知的miRNA之间的功能相似性推测功能未知miRNA的功能。尽管可以通过miRNA序列和结构来度量miRNA的功能相似性, 但研究表明有些具有相似序列和结构的miRNA却有不同的功能[6]。因此, 寻找一个有效的方法度量miRNA之间的功能相似性是研究的一个重点。通过生物信息学来计算miRNA功能相似性的方法主要分为2类, 一类方法基于miRNA的目标基因, miRNA的功能由其调控的下游目标基因决定[7], 2个miRNA调控的目标基因群越相似, 它们有相似功能的可能性就越高。另一类方法基于与miRNA关联的疾病。研究表明功能相似的miRNA经常涉及表型相似的疾病, 而与相似表型疾病相关的miRNA具有相似的功能[8]。相关基因和相关疾病对miRNA相似性研究有较大作用, 然而多数研究主要考虑其中一类信息, 并且这些方法针对特定的数据集, 无法简单地扩展到其他类型的数据集或者融合不同类型的数据来源[9]。

    本文基于miRNA-基因关系、基因-基因相似性、miRNA-疾病关系和疾病-疾病相似性等多个数据集构建多源miRNA相关网络, 通过网络表示学习 (或称为网络嵌入) 方法学习miRNA的特征向量, 根据特征向量的相似性衡量miRNA的功能相似性, 从而提供可靠的miRNA功能相似性度量方式。

    1 miREmb方法描述

    为充分利用现有的生物实验数据, 本文提出从多个公开数据集构建miRNA相关网络, 并将网络表示学习方法应用到构建的多源网络, 为网络中的每一个miRNA节点学习一个特征向量。学习到的网络表示向量不仅能直接根据余弦相似性计算miRNA功能相似分, 而且可以作为特征向量运用到其他任务中。

    1.1 多源miRNA相关网络

    在生物信息学中, 网络结构描述生物网络的特性, 节点表示实际的生物物质, 边表示它们之间的相互作用关系。比如在基因相互作用网络中, 节点代表单个基因, 节点之间的边表示基因之间存在着相互作用关系, 边的权重表示节点间的相关性或相似性程度。随着生物实验技术的进步, 各种生物数据的种类越来越多, 规模也越来越大, 将多个数据集进行整合, 构建miRNA相关网络, 进一步挖掘数据的深层信息。

    现有研究主要通过miRNA的目标基因和关联疾病计算miRNA的功能相似性, 这些方法主要用到miRNA-基因关系网络和miRNA-疾病关系网络。此外, 生物学家对一些基因和疾病之间的关系做了研究, 提供基因-疾病关系网络。除了不同生物物质之间的相互作用关系网络, 同一种物质也存在相互作用和相似关系, 其中可以利用的数据包括基因相互作用网络和疾病相似关系网络, 根据上述列举的相关数据构建多源miRNA相关网络, 网络结构如图1所示。

    2 实验结果与分析

    2.1 实验数据集

    本文构建多源miRNA相关网络用到的数据包括以下5个方面:

    1) miRNA-基因关系网络

    miRTarBase[14]是一个专门收集有实验证据支持miRNA及其靶基因间关系的数据库, 并按来源分为文献、免疫印迹实验、新一代测序技术等, 共有超过360 000条miRNA-基因关系记录。本文选择其中和人类miRNA相关的18 519条miRNA-目标基因关系记录, 其中5 504条由报告基因分析和蛋白质印迹法得到的关系可信度较高, 因此在网络中边的权重设为1, 其余13 015条关系的边权重设为0.3。

    2) miRNA-表型疾病关系网络

    文献[15]提供99个miRNA和51种疾病间共计242条miRNA-表型疾病关系, 其源自于文献中实验验证的记录, 且在网络中的边权重均为1。

    3) 基因-疾病关系网络

    本文使用的DisGeNET[16]是目前最大的公开数据集, 本文提取19 714条基因-疾病关联, 每一条关系都根据来源的可靠性对应一个0～1的可信度分数, 将此分数作为构建网络中边的权重。

    4) 基因相互作用网络

    基因相互作用网络数据来源于数据库HPRD (Human Protein Reference Database) [17]。该数据库提供目前最为广泛使用的蛋白质相互作用数据, 由于蛋白质可链接其编码基因, 因此可以看作基因-基因关系网络。HRPD包含39 122条相互作用数据, 边权重均设为1。

    5) 疾病表型相似网络

    疾病表型是指一组描述该疾病的临床症状的词汇, 目前针对人类疾病之间的研究还不是很完善, 疾病相似性可以通过对2个疾病之间症状的重叠进行量化。本文疾病相似性网络来源于MimMiner数据库[18], 根据文献[19]描述的疾病进行文本挖掘处理后得到5 081个疾病两两之间相似度的量化关系。本文选择其中相似度大于0.5的9 716条记录, 并将相似值作为网络中边的权重。

    利用上述数据集构建如图1所示的多源miRNA相关网络, 其中基因和miRNA由各自的符号表示, 疾病由OMIM数据库中的编号表示。在多源网络上运行miREmb算法, 得到该网络的网络表示向量, 并根据式 (4) 计算miRNA两两之间的余弦相似度, 以此作为miRNA的功能相似性得分。

    2.2 对比算法

    本文实验对比算法如下:

    1) miRFunSim方法[20]是用miRNA目标基因在整合的蛋白质相互作用网络中基因连通性来定量分析miRNA之间的功能相似性。该方法从数据库中获取miRNA目标基因数据, 对每对miRNA基于整合的蛋白质相互作用网络构建目标基因子图, 在子图上计算这2个miRNA目标基因对之间最短路径长度的平均值, 取倒数作为miRNA的功能相似性得分。

    2) DeepWalk是一种网络表示学习方法, 在网络上采样生成大量的随机游走序列, 对随机游走序列中每个局部窗口内的节点对进行概率建模, 通过最大化随机游走序列的似然概率学习节点的向量表示。该方法同等地对待网络中的所有类型节点, 并且不考虑关联路径的信息。

    3) miREmb＿single方法是本文提出的第1种方法, 在学习网络表示时只关注miRNA节点, 即只将关联路径中具有相关性的miRNA节点进行训练, 而忽略路径中的其他类型节点。

    4) miREmb方法是本文提出的第2种方法, 在采样得到关联路径之后, 将路径中所有类型的节点两两组对进行训练。

    2.3 miRNA家族的功能相似性

    属于同一个家族的miRNA通常具有相似的miRNA序列和完全相同的miRNA目标种子区域, 因此, 同一个家族的miRNA具有相似的功能[20]。为了评估方法计算得到的miRNA之间功能相似性得分的可靠性, 从miRBase数据库[21]中下载miRNA的家族信息。网络中的miRNA分为3组:家族间miRNA组, 跨家族miRNA组和随机miRNA组, 分别计算3个组内的miRNA相似度, 结果如图3所示。从图3可以看出, 属于同一个家族的miRNA之间的功能相似性得分比属于随机组和跨家族组的功能相似性得分都要高, 而跨家族组的得分比随机组低。miREmb方法计算出的家族内得分要比其他方法高, 而跨家族得分则要更低。DeepWalk方法由于无法区分多源网络中不同类型节点的差异, 学习的miRNA特征向量对家族内和家族外的差异较小, 效果较差。miREmb＿single方法尽管考虑关联路径的信息, 但在训练时只利用相关的miRNA节点对而忽略其他类型节点之间的关系, miRFunSim方法只利用基因相互作用网络的信息。这2种方法和miREmb方法相比都只利用部分信息, 效果都略差于miREmb方法。miREmb方法充分利用miRNA和相关基因以及相关疾病之间的关系, 计算出的miRNA家族内得分要比其他方法高, 而跨家族得分则要更低, 效果最好。

    3 结束语

    本文提出基于多源网络表示学习的miRNA功能相似性计算方法。该方法将多个来源的数据集进行融合, 包括miRNA相关的基因和表型疾病, 以及基因和疾病的相似关系, 构造miRNA相关多源网络。通过多源网络表示学习的方法将构造的网络映射到向量空间, 为网络中的每一个miRNA节点学习对应的特征向量。学习到的特征向量可以通过向量余弦相似度度量miRNA功能相似性, 也可以作为特征向量应用到miRNA相关疾病预测等任务中。实验结果表明, 该方法在区分家族内和家族外miRNA时优于DeepWalk方法。此外, 通过计算miRNA相似值, 可以准确地预测疾病的候选miRNA。
