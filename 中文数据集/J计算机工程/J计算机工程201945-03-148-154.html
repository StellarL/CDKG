    基于部分支撑集检测的WBAN数据处理机制

    摘    要：

    在利用无线体域网监测生理信号的过程中, 直接运用已恢复信号的支撑集会降低网络能量的有效性。为此, 提出一种基于部分支撑集检测的数据处理机制。根据信号的多尺度小波系数衰减特征, 以迭代支撑集检测的方式求解初始支撑集, 结合已恢复信号的支撑集并采取交集运算分离出正确支撑集信息。在此基础上, 通过正交多匹配追踪 (OMMP) 算法恢复数据。仿真结果表明, 与OMP、OMMP算法相比, 该机制在保证信号恢复质量的同时可以提高数据压缩率。

    关键词：

    无线体域网; 数据处理; 能效; 压缩感知; 部分支撑集;

    WBAN Data Processing Mechanism Based on Partial Support Set Detection

    WU Dapeng XIAO Bowen YAN Junjie

    College of Communication and Information Engineering, Chongqing University of Posts and Telecommunications Municipal Key Laboratory of Optical Communication and Network

    Abstract：

    In the process of monitoring physiological signals by Wireless Body Area Network (WBAN) , the support set of recovered signals is directly used to reduce the effectiveness of network energy.To solve this problem, a data processing mechanism based on partial support set detection is proposed.According to the attenuation characteristics of multi-scale wavelet coefficients of signals, the initial support set is solved by iteration support set detection, and then the correct support set information is separated by intersection operation combined with the restored support set of signals.On this basis, the data is recovered by Orthogonal Multi-Matching Pursuit (OMMP) algorithm.Simulation results show that compared with OMP and OMMP algorithm, this mechanism can improve the data compression rate while guaranteeing the quality of signal recovery.

    Keyword：

    Wireless Body Area Network (WBAN) ; data processing; energy efficiency; Compressed Sensing (CS) ; partial support set;

    0 概述

    无线体域网 (Wireless Body Area Network, WBAN) 由多个传感器节点组成, 用于持续监测人体的生理特征信息[1]。在通常情况下, 由电池供电的节点可用能量有限, 长期不间断监测会加速能量消耗。因为数据传输耗能较大[2], 所以亟需采取适宜方式对WBAN中的生理数据, 特别是在数据采样率和量化精度等方面数值较高的生理数据, 如心电图 (Electrocardiogram, ECG) 信号等, 进行压缩处理。ECG时频特征丰富, 不会在短时间内发生突变, 小波变换能灵活捕捉信号时频特性, 运用压缩感知 (Compressed Sensing, CS) 时可根据少量观测值恢复原始信号。因此, 结合CS设计能量有效的数据处理机制, 符合资源有限WBAN的实际应用需求。

    目前, 在WBAN中, 已有一些研究将CS应用于心电信号处理。为提高信号稀疏水平并进一步缩减数据量, 文献[3]将在线字典学习 (Online Dictionary Learning, ODL) 算法引入到数据压缩过程中, 但是其并未充分探究ECG的信号特征, 导致信号重构效果较差。考虑到ECG中QRS波谱能量占比较大的特点以及特定ECG的周期特性, 文献[4]利用Sinc函数进行拟合并压缩信号残差, 但是其拟合不够精准, 导致重构信号质量较差。根据连续ECG信号段对应的小波系数中存在的相关关系, 文献[5,6]提出在信号恢复性能方面优于传统方法的算法。文献[5]将ECG信号恢复问题转化为支撑集检测问题, 利用连续序列的支撑集一致性进行数据恢复, 但是其忽略了ECG的信号动态特性。文献[6]利用各数据段的多尺度小波系数构成二叉树结构, 但其仅探究各子带内小波系数幅值关系。综合考虑系数衰减特征以及连续信号段支撑集的一致性, 文献[7]在数据恢复算法中引入先验支撑集, 能够提升信号恢复效率。但是, 贪婪算法CoSaMP的原子筛选过程繁琐, IHT算法易导致算法收敛缓慢。

    上述ECG信号压缩处理机制均依据预先设定的随机矩阵进行信号观测, 不能有针对性地获取信号内部特征。此外, 他们均假设信号稀疏度与支撑集恒定不变, 直接将已恢复ECG信号段的支撑信息作为当前信号段的先验支撑集, 该操作会引入错误支撑集信息, 导致信号质量下降。

    基于以上分析, 本文提出一种结合正确支撑集的检测算法, 以剔除已知信号的支撑集中所包含的不准确信息。根据多尺度小波系数呈现的衰减特性, 利用快速支撑集检测算法检测出信号支撑集信息, 结合已恢复信号的支撑集以及检测算法所得的支撑集, 通过集合相交操作提高先验支撑集利用率。在此基础上, 基于正交多匹配追踪 (Orthogonal Multi-Matching Pursuit, OMMP) [8]算法在单次迭代过程中扩展多个支撑集元素。

    1 结合部分支撑集的压缩感知

    CS是一种新兴的信号获取方式, 适用于在特定变换域中呈现稀疏特征信号, 已广泛应用于图像处理、模式识别以及无线通信等领域。根据CS理论, 用于稀疏信号恢复的观测数据值在体量上远少于传感器节点的实际感测数据值。

    1.1 压缩感知

    定义一段长度为N的离散时间信号x, 正交基矩阵Ψ满足:

    x=Ψθ=∑i=0ΚθiΨi    (1)

    其中, Ψi为基矩阵Ψ中的列向量, θi为向量θ中的元素, 存在K<<N。若信号θ为K稀疏信号, 则满足:

    Κ=card{i|θi≠0}    (2)

    其中, card (·) 表示有限元集合中包含的元素数目。

    利用CS技术获得观测数据的数学模型为:

    y=Φx (3)

    其中, y为观测数据值构成的向量, 数据长度为M。

    小波域系数向量θ的恢复过程对应的优化模型为:

    min‖θ‖0 s.t.y=ΦΨθ (4)

    其中, ‖·‖0为L0范数, 表示向量内非零元个数。由于L0范数优化问题的时间复杂度随信号长度N呈现指数级增长, 因此用L1范数优化算式对式 (4) 进行转换:

    min‖θ‖1 s.t.‖y-ΦΨθ‖2≤τ (5)

    其中, ‖·‖1为L1范数, 表示向量内的元素绝对值之和, ‖·‖2为L2范数, 表示向量内所有元素的平方之和开平方后的数值, τ表示观测过程中混入的噪声数据的L2范数上界。

    文献[9]引入约束等距性质 (Restricted Isometry Property, RIP) 评估原始信号成功恢复的可能性。依据RIP理论, 为准确恢复原始信号, 约束等距常数δ2K应满足:

    (1-δ2K) ‖θ‖22≤‖ΦΨθ‖≤ (1+δ2K) ‖θ‖22 (6)

    由文献[10]可知, 根据式 (6) 得到的原始信号应满足:

    ∥x-x^∥2≤cτ    (7)

    其中, c是一个与δ2K正相关的常数, x^为根据式 (5) 求得的估计值。由式 (7) 可知, 常数c取值越小, 即δ2K越小, 重构效果越好。

    1.2 部分支撑集

    假设一段ECG信号x的长度为N, 则信号在多尺度小波域可以表示为:

    x=∑i=0Νi-1a1, iφ1, i+∑j=1L∑i=0Νj-1dj, iψj, i    (8)

    其中, j表示当前的尺度索引值, L表示信号解构尺度值, φ、ψ分别表示尺度函数和小波函数, Νj=Ν2L-j+1表示位于尺度j的系数总量, i表示位置索引值, 满足0≤i≤Nj-1。在小波变换后, 系数矩阵中包含尺度系数a1, i和小波系数dj, i。

    根据CS理论, 原始信号x在小波域中的多尺度表示为x=Ψz。其中, 尺度系数和小波系数构成的向量用z=[a1, 0a1, 1 … a1, N1-1d1, 0 … d1, N1-1 … dL, 0 …dL, NL-1]T表示。正交矩阵Ψ各列依次为小波函数和尺度函数。向量z又可划分为L+1个子向量, 第1个子向量为尺度系数, 可记为a1, 其余子向量记为dj, 且整数j的取值范围为[1, L]。此外, 向量z中第j+1个子向量表示尺度j内的全体小波系数。

    为分析真实ECG信号经过多尺度小波变换后系数值的分布情况, 在麻省理工学院提供的MIT-BIH Arrhythmia数据库[11]中选取编号为100的ECG信号, 从中截取一段长度N=128的信号段, 运用式 (8) 所示的模型对信号段进行多尺度分解。将系数幅度值按递减顺序排列, 保持其在多尺度小波系数中的索引值不变, 依次选取若干系数组成系数子集。因为非典型的非零元素对信号恢复的影响极小, 所以本文中ECG信号稀疏度值的选取过程并未考虑该类元素, 即在求解信号稀疏度的过程中对式 (2) 进行了松弛处理。当系数子集中元素的总能量达到全体系数总能量的99%时, 定义当前系数子集中元素个数为ECG信号的稀疏度, 用符号K表示。另外, 定义该系数子集中前若干元素的索引值构成的集合为信号支撑集, 用符号supp (·) 表示。当小波分解尺度L=4时, ECG信号段在多尺度小波域的系数幅度值变化情况如图1所示。从图1可以看出, ECG信号多尺度系数的幅度值总体上呈现出衰减特性。在基于CS架构设计算法的过程中充分运用这一先验性质, 可以在保证重构信号准确度的同时适当提高时效性。

    图1 ECG数据段在多尺度小波域中的系数幅度值   下载原图

    为分析连续ECG信号段对应的支撑集间的变化情况, 按照长度N=128对ECG信号段依序截断, 取连续的128个ECG信号段, 假定其对应的小波系数序列用符号zi表示, 其中, 参数i的取值满足i=1, 2, …, 100。定义小波域稀疏信号段zi的支撑集为Ωi。为探索连续ECG序列支撑集变化, 明确支撑集中元素的新添和删除情况, 本文对支撑集变化比例进行分析, 该参数的数学表达式为:

    ri, i+1=card[Ωi+1-Ωi∩Ωi+1]Κ    (9)

    其中, K表示信号段zi对应的完整支撑集内元素个数或信号段稀疏度, i的取值范围为[1, 99]。

    相邻信号段支撑集中元素的变化情况如图2所示。从图2可以看出, 该支撑集有所改变, 但不会发生非常剧烈的变化。此外, 支撑集内新添加的元素数量与支撑集中元素总数的比值一直处于较高水平, 但并不为1。由此可知, 连续ECG信号段的支撑集中元素在一定程度上存在一致性, 但是, 直接利用前一段ECG信号序列对应的支撑集信息, 并将其视为先验信息来估计当前序列的支撑集, 这一方式在WBAN生理信号监测过程中并不会一直产生较大的性能提升, 甚至会因为错误支撑集信息的引入而导致性能下降。

    图2 相邻ECG数据段的支撑集变化情况   下载原图

    考虑到连续ECG序列对应的支撑集信息呈现出来的部分一致性特征, 可采取支撑集检测的方式以及已有支撑集信息用于当前ECG信号段的正确支撑集检测。因为ECG信号段在多尺度小波域中的系数呈现出衰减特征, 可采取适用于幅度衰减信号支撑集检测的迭代支撑检测 (Iterative Support Detection, ISD) 算法[12], 该迭代机制在信号支撑集信息未知的情况下可快速高效地探索出信号支撑集。然而, 经过ISD算法处理后得到的信号支撑集中难免会混入错误支撑信息, 为在恢复算法执行初始阶段引入正确支撑集信息以增强信号恢复质量, 根据由ISD算法得到的支撑集信息和前一段恢复出来的ECG信号段的支撑集信息, 取两者的交集, 可在充分利用已有先验支撑信息的同时保证信号恢复质量。经过支撑集检测和正确支撑集提取2个阶段, 信号支撑集中得到的信号较为准确, 将其应用于ECG信号恢复过程, 能够保证信号恢复效果并增强CS系统中的生理信号还原质量。

    2 结合部分支撑集的ECG信号处理

    本文提出的适用于WBAN的ECG信号数据处理机制系统架构如图3所示。

    图3 ECG信号数据处理机制系统架构   下载原图

    图3中的系统架构主要包含3个处理单元:信号观测, 观测矩阵训练, 信号恢复。其中, 资源受限的传感器节点执行信号观测任务, 剩余处理单元均位于资源充裕的远程终端。在远程终端训练观测矩阵的过程中, 预处理操作可以减轻基线漂移以及高频干扰给训练样本带来的影响。为更好地保证信号恢复效果, 对训练样本采取滤波等预处理操作。通过一定的限制条件对观测矩阵加以约束, 可以对其进行优化并提升数据观测的有效性, 从而为数据的良好恢复提供保障。观测矩阵在远程终端上训练完毕后, 将其发送至对应的普通传感器节点进行数据观测。特定对象的生理信号会依据监测场景的改变而发生变化, 导致ECG信号的支撑集也会发生变化。为从包含错误信息的支撑集中尽可能地提取出真实支撑集, 实现原始信号的最佳逼近, 本文根据观测矩阵和观测数据进行支撑集估计, 在估计出来的支撑集中分离出真实支撑集信息, 然后利用真实支撑集信息、当前信号段稀疏度以及相应的数据恢复算法, 求解原始信号的近似信号。

    2.1 观测矩阵优化

    本文使用的数据相关型观测矩阵满足行满秩条件, 并且其相应的等距常数值较小。前者可保证观测后所得数据值的有效性, 后者对观测矩阵加以约束, 有利于增强稀疏系数恢复效果。同时, 传感器节点采用训练得到的观测矩阵进行数据的高效观测, 有利于提升WBAN在生理信号监测过程中的能量有效性。

    由于数据在观测过程中会受到噪声影响, 并且数据量化误差也会影响观测数据的准确性, 因此在观测矩阵设计的过程中, 本文引入某些约束条件以保证算法的数据恢复能力。在观测矩阵Φ满足RIP的前提下, 利用L1范数最小化操作可以从含噪声的观测数据中恢复稀疏数据。按照RIP对观测矩阵进行优化处理, 可以抑制WBAN系统在实际生理数据观测过程中受到的噪声影响。

    为简化式 (6) 的计算过程, 对θ进行归一化处理, 其等价算式为:

    |∥ΦΨθ∥22-∥θ∥22|≤δ2Κ    (10)

    其中, Ψ取同维度单位阵, 向量θ为ECG信号在多尺度小波域中的系数值, 满足θ=z。

    假定训练集中包含l组训练数据向量, 它们对应于l个稀疏系数向量, 记为zi, i=1, 2, …, l。各稀疏向量均需满足式 (6) , 用公式表示为:

    |ziΤ (ΨΤΦΤΦΨ-Ι) zi|≤δ2Κ, i=1, 2, ⋯, l    (11)

    定义矩阵A=ΦΨ, Y=ATA, 式 (11) 可转化为:

    |ziΤ (Y-Ι) zi|≤δ2Κ, i=1, 2, ⋯, l    (12)

    其中, 矩阵Y为实矩阵, 其特征值均大于0。

    观测矩阵Φ的秩用符号rank (Φ) 表示, 其反映随机观测后数据集中元素的有效性水平。当观测矩阵为行满秩矩阵时, 远程终端获得的数据更有价值。参数δ2K的取值对观测矩阵的准确性影响很大, 最小化该参数有助于获得最优的观测矩阵。Y的优化过程为:

    min{|ziΤ (Y-Ι) zi|}, i=1, 2, ⋯, ls.t.diag (Y) =[1, 1, ⋯, 1]    (13)

    求解得到矩阵Y后, 采用乔里斯基分解方式获得矩阵A, 该过程用公式表示为:

    Y=AΤA=USUΤ, A= (US) Τ    (14)

    利用奇异值分解 (Singular Value Decomposition, SVD) 方式得到合适的观测矩阵Φ, 该过程用公式表示为:

    Ψ=USVT, Ψ*=VS-1UT, Φ=AΨ* (15)

    通过上述过程可以获得实数域观测矩阵。然而, 仅含有0和1两种元素的观测矩阵优势明显, 更适用于能量受限的WBAN。根据文献[13]方法将实值矩阵转化为二进制伯努利矩阵, 然后计算观测矩阵是否满足行满秩条件。为缩短观测矩阵优化时间, 设定满足rank (Φ) ≈M时的观测矩阵即为所求目标, 否则循环执行上述优化过程, 直至满足条件后终止。

    远程终端采取上述过程优化观测矩阵后, 能够充分利用ECG信号的小波域系数中包含的丰富结构特征, 有助于提升观测数据的质量, 最终实现原始信号的最优逼近。观测矩阵在优化过程结束之后被传递给资源受限的传感器节点, 节点按照随机投影的方式对数据进行降维处理。采集生理信号的传感器节点根据ECG信号的小波域结构特征可求解当前信号段稀疏度水平。将各小波系数取绝对值后对数值按照降序进行排列, 当一组典型的小波域系数的能量总和在信号总能量中的占比达到99.99%时, 取当前的小波系数数量为当前ECG信号段的稀疏度值K, 然后将该数值传递给远程终端并应用于信号恢复过程, 以提升恢复信号质量。

    由式 (16) 、式 (17) 可知, 利用CS机制进行信号恢复的过程可以等价于相关信号的支撑集选取过程。在传统的迭代机制, 如正交匹配追踪 (Orthogonal Matching Pursuit, OMP) [14]中, 支撑集的元素数量随轮数逐渐增多, 但增长速率缓慢。为平衡信号恢复时间与信号恢复质量, 本文依据ISD机制进行支撑集选取。与传统的迭代算法相比, ISD算法具有2个典型特征:

    1) 全体稀疏系数值在每轮循环中均得到更新。

    2) 在搜索稀疏系数中的非零元素索引时, ISD不依赖信号先验信息。

    此外, ECG信号在多尺度小波域中的系数幅度值呈现阶梯状下降的趋势符合ISD算法的应用需求。因此, 根据ISD设计支撑集检测机制可满足WBAN的实际应用要求。但ISD算法求得的支撑集中会混入错误信息, 因此, 在部分支撑集已知的情况下, 本文采用矩阵取交集的方式精确识别先验支撑集中的正确支撑集信息。在原始信号恢复过程中, 采取该方式可以有效减少初始支撑集以及ISD算法所得支撑集中含有的错误信息, 从而更好地保证信号恢复效果, 增强CS系统中的信号还原质量。

    定义ΦQ为矩阵Φ的子矩阵, 其由矩阵Φ中若干列向量构成, 列索引值构成集合Q, 第i轮获得的扩展支撑集记为Di。文献[8]指出一旦扩展集Di包含真实支撑集D, 即D⊆Di, 则能通过最小二乘估计方法重构原始信号。 (Di) C为集合Di的补集, 远程终端可根据传感器节点发送过来的数据Φ和y, 按照如下步骤求解原始信号中可能的支撑集信息:

    步骤1 初始化迭代次数i以及支撑集D。

    步骤2 根据当前支撑集, 按照式 (17) 更新z。

    步骤3 通过更新后的向量z, 更新支撑集D得到Di+1, 该过程中系数值应满足:

    |z[j+1]|-|z[j]|>∥z∥∞/Μ, j=1, 2, ⋯, Ν    (18)

    D={j:|xj|≥ε}, j=0, 1, ⋯, Ν    (19)

    其中, z[j+1]表示系数幅度值绝对值按降序排列后位于位置j+1处的元素, ε为满足式 (18) 的第1个系数的绝对值, ‖z‖∞为集合中数值最大的元素, ‖z‖∞/M为阈值条件。

    步骤4 评估迭代算法是否满足终止条件, 若不满足, 则更新迭代次数, 将i+1的值赋给i。

    支撑集检测算法执行完毕后得到支撑集Df, 该支撑集中会混入错误信息, 当传感器节点向远程终端发送数据信息后, 远程终端便存在有关支撑集Df的信息, 但不能确定其中的正确支撑集信息。因为相邻ECG数据段经过多尺度小波分解后其系数支撑集中含有部分相同元素, 所以本文利用集合相交的方式从估计出来的支撑集信息以及先验支撑集信息中尽可能地剔除错误信息, 最终提取出正确信息。为更精确地恢复原始信号, 本文以正确的支撑集子集作为先验支撑信息, 根据经典的迭代算法求解原始信号逼近值。在迭代算法中, OMP算法执行效率较高, OMMP算法作为OMP算法的改进, 在各轮迭代过程中均挑选多个列向量, 可提升信号恢复效率, 在OMMP算法中引入先验支撑信息有助于提高信号恢复质量。

    综合考虑OMMP算法和集合Df, 本文提出一种应用于WBAN中带有部分支撑集的数据检测算法DSSD。利用ISD算法获得初始信号支撑集, 然后利用集合的交集运算尽可能提取出正确支撑集信息, 以增强先验支撑集中所含元素的有效性。最后, 根据所得支撑集信息, 引入OMMP算法恢复出原始信号。定义函数k (v, m) , 其返回值是一个与向量v同维度的向量, 通过保留原始向量v中最大的m个元素及其位置而得到函数k (v, m) 。参数n表示OMMP算法在各轮循环中加入支撑集的索引数量。DSSD算法的迭代次数用Γ表示, D-1代表已恢复ECG信号段对应的支撑集, (·) †表示伪逆操作。DSSD算法伪代码如下:

    3 仿真结果与分析

    本文采用Matlab 7.11.0工具, 在台式计算机 (3.20 GHz Intel Core i3 CPU, 2.0 GB RAM) 上进行仿真, 以评估DSSD算法的性能。本次实验所使用数据库为MIT-BIH Arrhythmia数据库, 其中各项数据文件均包含双通道ECG信号, 信号采样率为360 Hz, 量化精度为11 bit。本文选择在WBAN信号处理领域被研究人员广泛使用的编号为100的ECG信号进行仿真分析。实验中的对比算法包括结合随机高斯观测矩阵和多尺度训练字典的检测机制[3]等。在仿真过程中, 保持观测矩阵F维度为M×N, 从通道V5中选择连续的N个数据值构成长度为N的信号段x, 各项参数均是重复多次实验后得到的平均值。在结果分析过程中, 本文用参量压缩率 (Compression Ratio, CR) 衡量信号压缩程度, 该参量数值越大, 则远程终端获得的数据观测信息越少。参量CR的数学表达式为:

    CR=a-ba×100%    (20)

    其中, 参数b为节点采集到的原始数据长度, a为节点发送给远程终端的数据段长度。此外, 本文采取重构信噪比 (Reconstructed Signal-Noise Ratio, RSNR) 来衡量恢复算法的性能。RSNR数值越大, 表示信号恢复质量越好。RSNR的数学表达式为:

    RSΝR=10 lg∥x∥22∥x-x^∥22    (21)

    其中, x表示原始信号段, x^表示恢复出来的信号段。

    3.3 压缩率对系统性能的影响

    为验证各算法在压缩率变化时的系统性能差异, 本文设置压缩率变化范围为[40%, 85%], 对不同算法下的重构信噪比变化情况进行仿真分析。其中, 对比算法包括基于在线字典学习的ODL算法[3]、基于固定基字典的DCT-DWT算法[17]、OMMP算法和OMP算法。图5所示为稀疏度值K取40时重构信噪比随信号压缩率CR的变化情况。从图5可以看出, 当信号压缩率在区间[40%, 85%]内变化时, 与OMP、OMMP等算法相比, DSSD在RSNR上的性能更优, 这是因为DSSD算法在识别准确支撑集信息方面具有一定优势, 使得其能够精准识别出信号中的有用支撑集信息, 并且DSSD算法会利用更多的先验支撑集信息, 该过程有利于ECG数据恢复算法识别稀疏系数中的有效支撑集, 从而提升重构信号质量。另外, 随着压缩率值的增大, 各算法的RSNR指标呈现下降趋势, 这是因为随着压缩率数值的提高, 远程终端上可得到的观测数据量会减少, 最终导致信号恢复质量下降。将ODL算法与DCT-DWT算法进行对比, 可以看出, 字典矩阵的使用可以有效提升ECG信号的恢复效果, 这是因为在信号稀疏表示方面, 学习型字典比普通的固定基矩阵具有更强的针对性, 其能够灵活表示原始信号并获得更优的信号稀疏系数, 最终提升信号质量。由于多尺度学习型字典的使用过程中并未深入探索ECG信号在多尺度小波域中的结构特征, 而DSSD算法考虑到额外的小波系数特征, 支撑集探索与分离有助于获得准确的支撑集信息, 这不仅加快了信号恢复过程, 而且在一定程度上保证了信号恢复质量。因此, 与利用多尺度小波字典的算法相比, 采用DSSD算法所获得的信号质量更高。

    4 结束语

    本文提出一种应用于生理信号远程监测的数据处理机制DSSD。利用ISD算法和先验支撑信息提升支撑集信息准确性, 在已有部分支撑集的基础上采取OMMP算法深入筛选支撑集信息。同时, 对数据相关型观测矩阵进行优化与设计, 以增强观测信号的有效性并提高数据恢复质量。分析结果表明, 该机制能够充分运用ECG信号的多尺度特征, 在信号恢复阶段引入部分支撑集信息可以改善信号恢复效果, 使ECG信号压缩率维持在较低水平, 有利于传感器节点节省存储空间与能量开销, 最终达到提升WBAN能量有效性的目的。下一步将选择传感器节点能量作为研究对象, 以验证本文机制的能量有效性。
