    摘    要：

    为在不完美频谱检测环境下对资源进行优化分配, 提出一种分布式认知无线电网络资源分配算法。根据贝叶斯理论给出子载波状态信任指数与统计平均干扰功率的概念, 利用拉格朗日对偶分解理论, 将原分配问题分解为独立的子问题并进行求解, 以实现多用户资源的有效分配。仿真结果表明, 相对基于回避准则的优化算法, 该算法能有效提高系统容量, 且能更快地达到循环结束条件。

    关键词：

    认知无线电网络; 贝叶斯理论; 资源分配; 对偶分解理论; 不完美频谱检测;

    Resource Allocation in Cognitive Radio Network Based on Imperfect Spectrum Detection

    TANG Liping JIANG Yilun

    Abstract：

    To optimize resource allocation in imperfect spectrum detection environment, a distributed cognitive radio network resource allocation algorithm is proposed.Based on Bayesian theory, the concepts of subcarrier state trust index and statistical average interference power are given.By using Lagrange dual decomposition theory, the original allocation problem is decomposed into independent sub-problems and solved to achieve effective resource allocation for multiple users.Simulation results show that compared with the optimization algorithm based on avoidance criterion, the proposed algorithm can effectively improve the system capacity and achieve the end condition of the cycle faster.

    0 概述

    随着无线通信技术的快速发展, 用户对频谱的需求大幅提升。为解决频谱资源不足的问题, 文献[1]提出认知无线电的概念, 并指出通信设备和通信网络具备学习能力是认知无线电技术的标志性特征[2]。认知无线电系统中的可用资源根据授权用户的活动情况而动态变化, 此外, 认知通信系统资源分配的前提是其不能影响授权用户的通信。因此, 认知无线电系统中的资源分配与传统通信系统存在较大差异, 在不过分干扰授权用户的情况下, 如何合理利用资源的不确定性来为认知用户提供确定且可靠的服务, 是认知无线电系统资源分配的重难点问题, 目前有较多学者对该问题进行了研究[3,4,5]。文献[6]基于不完美感知结果, 提出一种联合资源分配与频谱检测的算法, 该算法大幅提升了系统性能。文献[7]在不完美频谱检测环境下, 设计基于部分可观察马尔科夫决策过程 (Partially Observable Markov Decision Process, POMDP) 的频谱分配算法, 该算法采用跨层的资源分配方法, 能有效降低授权用户和认知用户间的冲突。文献[8]建立一种基于抢占式排队论的频谱切换模型。文献[9]同时考虑频谱分配与功率控制2个因素, 利用多目标优化算法对资源分配问题进行求解。文献[10]在蝙蝠算法的基础上, 提出一种基于统计特性的认知无线电频谱分配方案, 其系统性能得到了明显提升。

    在实际应用中, 认知无线电系统中的频谱检测单元极其重要, 其提供的频谱检测信息是认知无线电系统进行决策的依据, 因此, 频谱检测单元是系统实现频谱二次利用、提高频谱利用率的基础。此外, 目前的频谱检测技术水平较低, 这导致其较难得到与现实频谱状况相符的频谱检测结果, 尤其是针对微弱信号进行检测时。已有关于认知无线电网络资源分配的研究多数关注完美频谱检测, 较少在不完美频谱检测下对资源进行分配。针对该问题, 本文结合概率统计理论与凸优化理论, 提出一种基于不完美频谱检测的资源分配算法。

    1 系统模型

    1.1 不完美频谱检测下的干扰模型

    根据授权频谱是否被占用, 不完美频谱检测的结果可以分为2种:

    1) 某一频谱资源实际上正被授权用户占用, 但被频谱检测单元检测为空闲。

    2) 某一频谱资源实际上为空闲, 但被频谱检测单元检测为被占用。

    第1种不完美检测结果会导致认知用户在该频谱已被占用的情况下依然占用该频谱, 从而造成对授权用户的进一步干扰;第2种不完美检测结果会导致认知用户错过接入空闲频谱的机会, 致使系统吞吐量下降[11]。

    考虑一个不完美频谱检测的认知无线电系统, 其中的某个子载波没有被授权用户占用而被频谱检测单元判断为被占用, 该情况被称为误报警, 简称误报, 其概率用Qalan表示。某子载波被授权用户占用但被频谱检测单元判断为空闲, 该情况被称为误判断, 简称误判, 其概率用Qmisn表示。用Qn表示子载波正被授权用户占用的概率, setocc表示被授权用户占用的子载波集合, setvac表示空闲的子载波集合。

    对于一个被判断为被授权用户占用的子载波j∈setocc, 其确实被授权用户占用的概率用Bayesian公式表示为:

    pjo-o= (1-Qjmis) Qj (1-Qjmis) Qj+ (1-Qj) Qjala          (1)

    piv-o=QimisQiQimisQi+ (1-Qiala) (1-Qi)           (2)

    根据全概率公式可知, 一个被判断为空闲的子载波i∈setvac, 其确实空闲的概率为:

    ri=piv-v=1-piv-o (3)

    用ri表示资源分配的信任指数。在上述不完美频谱检测的认知无线电系统中, 某认知用户对授权用户造成的统计平均干扰功率为:

    其中, Ι∼ki表示认知用户k在子载波i上传输对授权用户造成干扰的统计平均干扰因子。在不完美频谱检测的认知无线电系统中进行资源分配, 目标是实现系统容量最大化。本文基于传统分配模型, 考虑2个约束条件:

    1) 各认知用户的发射功率受限制。

    2) 认知用户对授权用户造成的干扰功率低于一个门限值。

    1.2 资源分配模型

    2 基于凸优化理论的分配问题求解

    2.1 原分配问题的拉格朗日对偶问题

    本文引入如下2个拉格朗日因子:

    1) 发射功率因子向量μ。

    2) 干扰功率因子λ。

    将目标优化函数联合约束条件转换成拉格朗日函数:

    L (a, p, μ, λ) =∑k=1Κ∑i∈setvacak, irilb (1+hk, ipk, iσ2) +   ∑k=1Κμk (Ρk-∑i∈setvacak, ipk, i) +λ (Ι-∑k=1Κ∑i∈setvacak, ipk, iΙ∼ki) =   ∑k=1Κ∑i∈setvacak, irilb (1+hk, ipk, iσ2) -∑k=1Κμk∑i∈setvacak, ipk, i-   λ∑k=1Κ∑i∈setvacak, ipk, iΙ∼ki=∑k=1Κ∑i∈setvacak, irilb (1+hk, ipk, iσ2) -   ∑k=1Κμk∑i∈setvacak, ipk, i-λ∑k=1Κ∑i∈setvacak, ipk, iΙ∼ki+∑k=1ΚμkΡk+λΙ          (10)

    其中, a为分配指数向量, p为功率分配向量。进一步得到拉格朗日对偶函数:

    d (μ, λ) =maxa, pL (a, p, μ, λ)           (11)

    对偶函数是式 (5) 的上界, 对偶函数的最小值即为原优化函数的最大值。对偶函数的最小值表示为[14]:

    minμ, λ>0d (μ, λ)           (12)

    约束条件为∑k=1Κak, i≤1, ∀i和ak, i∈{0, 1}。

    2.2 原分配问题的对偶分解

    由式 (12) 可知, 拉格朗日因子数目较多, 且同时涉及2个变量向量a、p, 直接求解时难度较大。因此, 本文采用分解理论将对偶优化问题分解成多个子问题并进行求解。对于一个空闲的子载波i, 有:

    di (μk, λ) =∑k=1Κak, irilb (1+hk, ipk, iσ2) -   ∑k=1Κμkak, ipk, i-λ∑k=1Κak, ipk, iΙ∼ki          (13)

    由于每个子载波只能分配给一个认知用户, 如果将空闲子载波i分配给认知用户k, 则只有ak, i的值为1, 其他分配指数都为0。因此, 对于认知用户k, 有:

    dk, i (μk, λ) =rilb (1+hk, ipk, iσ2) -μkpk, i-λpk, iΙ∼ki          (14)

    对式 (14) 求导, 可得认知用户k在空闲子载波i上的最优功率分配为:

    pk, i*=[rilb (μk+λΙ∼ki) -σ2hk, i]+          (15)

    其中, [ ]+表示计算结果是一个非负数, 当式内结果为负数时, 其值将取0。为使系统容量达到最大值, 需要将各子载波i分配给能使dk, i (μk, λ) 最大的认知用户:

    ak*, i={1, dk*, i (μk*, λ) 最大0, 其他          (16)

    认知用户k所占用子载波i上的功率分配由式 (15) 确定, 这就是子载波与功率的分配准则。

    分配准则式 (15) 、式 (16) 是分别关于发射功率因子μ→、干扰功率因子λ的函数, 对他们求关于μ→和λ的偏导数较难。本文采用次梯度法对μ→和λ进行迭代更新:

    μk (t+1) =[μk (t) -α (Ρk-∑i∈setvacak, ipk, i) ]+          (17) λ (t+1) =[λ (t) -β (Ι-∑k=1Κ∑i∈setvacak, ipk, iΙ∼ki) ]+          (18)

    其中, t为迭代次数, α、β为非负的更新步长, 如果α、β取值合适, 次梯度算法的迭代更新一定能够收敛到最优值。当|Δμk|<ε、|Δλ|<ε时, 迭代更新结束。

    由上述过程可以看出, 考虑频谱检测误差的资源优化分配算法, 其本质是通过在资源分配时参考子载波信任指数信息和统计平均干扰因子信息, 将信任指数较高的子载波分配给能使统计平均干扰因子Ι∼ki较低的认知用户, 并给该用户在子载波上分配较多传输功率。认知用户分配子载波的统计平均干扰因子Ι∼ki较高, 则该认知用户在子载波上分配的功率较小, 即给统计平均干扰因子Ι∼ki较低且子载波信任指数ri较高的认知用户下的子载波分配较多功率, 而给统计平均干扰因子Ι∼ki较高且子载波信任指数ri较低的认知用户下的子载波分配较少功率, 从而在不完美频谱检测环境下实现认知用户对授权用户干扰功率的有效管理与控制, 并在授权用户干扰门限限制下进行系统容量优化。

    3 分布式网络中的资源分配算法

    与中央控制式网络相比, 分布式网络没有中心控制单元, 因此, 其不会因为控制中心发生故障而造成全网瘫痪。考虑一个同步时隙的分布式网络, 每个时隙分为3个子时隙:感知子时隙, 分配子时隙, 传输子时隙, 如图2所示。

    图2 时隙结构示意图   下载原图

    感知子时隙检测每个子载波的信道信息并计算每个信道的统计信息[15]。在感知时隙检测结束后, 将进入分配子时隙阶段, 此时调用资源分配算法确定这一轮的资源分配方案。在传输子时隙阶段, 按照已确定的资源分配方案进行传输。本文给每个认知用户配备一个GPS设备, 以确保时隙同步。

    分配子时隙涉及具体的分配算法, 本文对其进行重点研究。在分配子时隙阶段, 为使每个子载波i都能分配给能使dk, i (μk, λ) 最大的认知用户k, 本文采用倒计时的方式, 根据每个认知用户的dk, i (μk, λ) 值设置不同的倒计时初值, 最先完成倒计时的用户将占用该子载波。倒计时初值设定为:

    Οk, i=cdk, i (μk, λ)           (19)

    其中, c是一个足够小的常数。在分配子时隙中, 对于子载波i, 当某认知用户率先完成倒计时后, 向周围广播即将占用子载波i的信息, 其他用户收到该广播信息后将停止在子载波i上的倒计时。采用这种倒计时方式, 可在分布式网络中使用户自主地完成资源分配, 而无需中心控制单元。

    在分布式网络资源分配算法中, 各用户的发射功率因子μk根据式 (17) 确定, 由每个认知用户自行更新。干扰因子λ由授权用户根据式 (18) 更新并广播给认知用户。

    本文分布式资源分配算法具有自主调节认知用户功率的能力。如式 (17) 所示, 当认知用户k的功率大于其设定的发射功率时, μk将增大, 导致下一轮能够分配到的功率pk, i减小, 直到满足设定的功率发射限制条件。而当系统统计平均干扰功率大于干扰功率门限时, 参数λ将会增大, 这也会导致下一轮能够分配到的功率pk, i减小, 直到满足设定的干扰功率门限限制条件。另一方面, 当认知用户k的功率小于其设定的发射功率时, 系统容量存在提升空间, μk将减小, 导致下一轮能够分配到的功率pk, i增大, 从而提升系统容量, 直到满足设定的功率发射限制条件。而当系统统计平均干扰功率小于干扰功率门限时, 参数λ将会减小, 这也会导致下一轮能够分配到的功率pk, i增大, 从而提升系统容量, 直到满足设定的干扰功率门限限制条件。

    本文认知无线电网络资源分配算法具体步骤如下:

    步骤1 设定初始值。

    步骤2 根据上一轮发射功率与干扰功率的情况, 认知用户更新发射功率因子μ和干扰功率因子λ。

    步骤3 计算k个认知用户分别在各空闲子载波上的dk, i (μk, λ) 和pk, i。

    步骤4 根据式 (19) , 采用倒计时方式确定子载波分配方案。

    步骤5 根据步骤4的方案进行网络传输。

    步骤6 判断|Δμk|<ε和|Δλ|<ε是否成立。如果成立, 结束迭代, 然后按步骤4的方案进行网络传输, 直到本次传输任务结束;否则, 传输完成后回到步骤3。

    4 实验结果与分析

    本文对资源分配算法的性能进行仿真评估。认知用户采用OFDM调制, 授权用户采用QAM调制。假设认知网络中存在1个授权用户、3个认知用户, 3个认知用户的发射功率限制分别为6.0 W、6.5 W、7.0 W。每个子载波带宽为8 Hz, 只考虑信道平坦衰落。迭代的收敛条件为ε<0.005, 更新步长为α=0.002 5, β=0.002 5。设空闲子载波集合setvac中共有16个子载波, 每个空闲子载波的误判概率Qmisn满足[0.2, 0.5]的均匀分布, 每个空闲子载波正被授权用户占用的概率Qn满足[0.4, 0.6]的均匀分布。设已被占用子载波集合etocc中共有5个子载波, 每个子载波的误报警概率Qalan满足[0.2, 0.5]的均匀分布, 每个子载波正被授权用户占用的概率Qn满足[0.5, 0.7]的均匀分布。

    在一个不完美频谱检测环境下将本文算法与同样考虑频谱检测误差的基于回避准则的优化算法[7]进行对比。基于回避准则的优化算法根据每个信道状态信息的可靠性来设置信道的优先级, 对可靠性较高的信道设置高优先级。在该算法中, 授权用户干扰门限设为4.5 W。在未考虑频谱检测误差的对比算法中, 不存在信任指数与统计平均干扰因子的概念, 因此, 式 (15) 变为:

    pk, i*=[1lb (μk+λΙki) -σ2hk, i]+          (20)

    式 (18) 中的∑k=1Κ∑i∈setvacak, ipk, iΙ∼ki干扰功率值是由授权用户检测所得的结果, 直接由授权用户广播给各认知用户。在未考虑频谱检测误差的算法中, 不存在非空闲子载波集, 因此, 式 (18) 变为:

    λ (t+1) =[λ (t) -β (Ι-∑k=1Κ∑iΙak, ipk, iΙki) ]+          (21)

    在同一个不完美频谱检测环境下, 本文优化算法与基于回避准则的优化算法的系统总容量与迭代次数的关系如图4所示。由图4可以看出, 本文算法数值变化较平稳, 基于回避准则的优化算法数值震荡较大。原因是本文算法在进行子载波与功率分配时, 考虑每个子载波检测结果的可信性 (即信任指数) , 在信任指数较低的空闲子载波上分配较少的功率, 有效避免了给授权用户带来干扰。2种算法在迭代40次后均趋于平稳, 而本文算法能更快地达到循环结束条件。

    5 结束语

    本文设计一种在不完美频谱检测环境下对无线资源进行分配的算法。结合概率统计知识, 提出子载波状态信任指数与统计平均干扰功率的概念, 利用拉格朗日对偶理论, 将资源分配问题分解为和每个认知用户相对应的独立子问题, 对各子问题进行求解, 得到一种分布式多用户资源分配方案。仿真结果表明, 该算法能有效控制由频谱检测误差导致的统计平均干扰, 提高不完美频谱检测环境下的系统容量。下一步将在资源分配过程中对能效问题进行研究。
