    摘    要：

    KiK-net和K-NET是由日本防灾科学技术研究所 (简称:NIED) 负责维护管理的强震动台网。KiK-net和K-NET的观测设施广泛分布在日本全境, 其中KiK-net在观测点同时安装地表及井下强震仪, 进行对比观测, 观测到的强震动记录直接发送到NIED的强震观测中心, 通过网站 (http://www.kyoshin.bosai.go.jp) 对外公开共享。此外, 还提供了观测井的地质钻孔柱状图和钻孔剪切波测试数据。该强震动数据库为世界各地研究人员推进地震各方面研究提供了宝贵资源。根据NIED规则, 用户需要在NIED网站注册获得用户名和密码, 并且确保数据用于本人研究。本文基于C#语言开发了软件系统。设置好用户名和密码, 就可利用本文开发的软件系统自动下载KiK-net和K-NET数据, 并存入或更新研究者的本地数据库, 进而管理和使用本地的强震动数据, 此外系统还集成了强震动数据常用的分析处理算法, 提高研究人员的使用数据的效率。

    LI Jinggang YAO Yunsheng ZHANG Lifen

    Abstract：

    KiK-net and K-net are the strong ground motion observation networks maintained by the Disaster Prevention Institute of Science and Technology (hereinafter referred to as NIED) . KiK-net observation facilities are distributed all over the country, and particularly accelerometers are installed both at surface and underground in each observation point for the comparative observation. Observation data are directly send to NIED strong earthquake center which shares the data widely through the website (http://www.kyoshin.bosai.go.jp) . Moreover, the website also provides the geological profile and shear-wave test data of borehole. The strong ground motion database provides valuable resources for researchers around the world to promote all aspects of earthquake research. The user should register in the NIED website and obtain the user name and password, and the data should be used without distributing to others by NIED convention. In this paper, we developed a software system with C# language. After setting user name and password into the software, it can automatically download NIED earthquake data from NIED website, and update/insert the data into local database. Moreover, it provides management tools and many strong motion analysis algorithms, which may improve the use efficiency of the researchers.

    引言

    强震动台网是布设于地震活动水平较高地区的固定台站或流动台站, 以及服务于特定研究目的的专用台阵组成的强震动观测网。台网记录的强震动数据是研究地震动衰减规律, 进行地震危险性分析、震害预测, 验证和改进结构抗震分析及设计方法, 确定新结构抗震设计标准和抗震措施的基础资料。地震多发国家都建设了不同规模的强震动观测台网, 如中国数字强震动观测台网, 日本K-NET, KiK-net台网, 美国Tri-Net台网等。中国地震局在21个地震重点监视区防御区建设了数字强震动台网, 该台网与2007年完成。在21个地震重点监视防御区布设1154个自由场固定地震动遥测台。中国数字强震动台网除全部采用先进的数字强震动观测仪器外, 对每个固定台站都做了详细的场地勘察, 包括钻孔、波速测量和地脉动测量等。

    日本K-NET台网是由日本防灾科学技术研究所 (NIED) 负责建设管理的全国性强震动台网。日本汲取1995年阪神地震的经验教训, 提出了一系列推进地震灾害预防的措施, 其中之一是大力加强强震动观测, 促进地震工程研究和应用。1996年, NIED完成了K-NET网络建设, 共布设1031个观测台站, 平均间距20km, 使日本成为世界上强震仪布设密度最高的国家。台站设施配置K-NET 95型三分量强震动仪。KiK-net (Kiban-Kyoshin Net:基岩强震动台网) 的观测设施在全国范围内共有近700处, 其特点是在每个观测设施处设置了地表和井下两套强震仪。每个台站均提供了地质钻孔柱状图和剪切波速测试等资料。观测到的强震动数据直接发送到强震观测中心, 由防灾科研的网站 (http://www.kyoshin.bosai.go.jp) 向全球广泛地公开。观测点的地质钻孔柱状图和剪切波测试数据也能在该网站直接下载。为世界各地研究人员提供了宝贵的数据资源[1,2,3,4,5]。

    场地条件对地震动的影响显著, 研究其影响的最直接有效的方法是基于强震动观测的三维台站记录, 分析局部场地条件对地震动的影响。强震动观测的三维台阵中国有唐山响堂台阵和通海台阵用于场地效应的研究, 自贡西山公园山体地形影响台阵用于研究地形起伏对地震动的影响。日本KiK-net台站是世界上较密集的强震动台网、地震频发, 记录了大量强震动数据。利用日本强震动台网KiK-net中不同类型台站中大量强震动记录, 对比不同场地的地震动特性, 可以直接分析傅里叶谱卓越频率与震级和距离的关系, 以及上覆岩土层对地震动的影响。地震预警是目前地震服务社会的热点, 地震预警中广泛使用强震动记录数据、利用强震动数据快速确定震源位置以及地震震级是关键。因此要充分发掘KiK-net强震动数据在地震预警中的应用, 改进地震定位和震级确定的精度[6,7,8,9]。NIED提供的KiK-net和K-NET数据在地震学及地震预警、场地效应等地震工程学研究中得到了广泛的应用并取得了众多的成果。为了更便于数据的维护使用, 本文基于.Net平台, 采用C#语言, 使用SQLite作为数据库, 开发了数据分析管理系统, 为研究人员提高数据使用效率, 自动下载、管理NIED强震动。

    SQLite是一款稳定、开源的轻型数据库, 遵守 ACID 的关联式数据库管理系统, 能够与包括 C 语言在内的其他多种程序语言相结合, 能够执行大部分的 SQL 命令, 具有 ODBC 接口。SQLite 程序本身很小, 只在 250 K 左右, 却可以支持最大达到 2TB 的数据库大小, 这足可以满足一般科学与工程需要。相比于其他数据库软件, SQLite 安装简单方便, 不需要任何其他附加软件的安装。鉴于SQLite 的诸多优点, 在数据量不大时使用该数据库来管理数据是一个很好的选择, 这将大大的降低开发和维护成本。SQLite是目前比较流行的轻型嵌入式数据库, 占用资源较少, 效率高可靠性强, 支持主流操作系统, 并能跟多种程序语言相结合, 是存储数据库的首选. 与其他数据管理系统如 SQL Server、Orcalce等不同, SQLite数据库包含在磁盘上的一个文件中, 备份、复制非常方便 (表1) 。用户只需当作一般的文件进行处理即可, 无需了解专业数据库方面的知识。本系统采用SQLite作为数据库进行系统开发。

    由于NIED强震动数据库保存了1996年至目前 (2016) 所有台站的数据, 以NIED提供的数据格式, 数据量大约360 GB, 采用本文提到的数据压缩方法, 大约可以压缩到原来的1/4, 即数据量大约100 GB。

    2 系统功能

    2.1 自动下载NIED的K-NET和KiK-net数据

    用户需要登录http://www.kik.bosai.go.jp/进行注册, 待网站管理员验证通过后, 即可下载数据。可以通过网页界面逐条数据、逐个地震进行下载。这就要求经常下载更新, 比较繁琐。NIED的KiK-net和K-NET数据是保存在一个FTP服务器上的。IP地址为158.203.31.151, 端口21. 由于数据实际上是存放在FTP服务器上, 用户也可以使用通用的FTP软件进行下载。NIED的FTP服务器上数据的根目录下包含all、kik、knet、utility四个文件夹 (表9) 。knet存储该台网的强震动波形、多种格式台站信息。utility为提供的工具软件。

    2.2 数据压缩原理及效果

    NIED文本格式*.asc的数据为有符号整数, 每行8个数据, 每个数据为9字节文本格式, 若采用二进制整型存储, 则可以减少到4个字节。考虑到四字节整型范围为从-2147483648～2147483647, 大多数数据用二进制表示时高位基本上是0, 波形数据采用压缩方式存储。首先对一个地震波形的所有数据进行判断, 统一偏移, 保证所有数据转为正数, 记下偏移量, 存储起来;每个数据都是由四字节组成, 确定中每一位 (bit) 均为的0的字节数, 从而确定每个数据的有效字节数neff, 也就是为使所有数据的信息不丢失, 每个数据的最小字节数, 无效字节的每一位都是0, 解码时, 自动补上, 无需存储。压缩后的数据是一串字节数组, 依次是该地震波形的数据量ndata (int, 4字节) , 单个数据有效字节数neff (byte, 1字节) , 为使所有数据为正补上的偏移量数据 (int, 4字节) , 字节数组 (ndata*neff) 。表11中给出了不同大小的数据的压缩效果, 可见数据越小, 压缩的效果越好。

    4 结论

    为方便研究人员对日本强震动数据的研究和使用, 本文开发了NIED强震动数据管理分析系统。

    (1) 该系统自动从日本防灾研究所FTP服务器下载地震、台站钻孔剪切波速度结构、波形数据, 更新本地数据库系统, 省时省力, 采用仅存储有效字节的二进制存储方式, 可将数据存储空间缩减到原格式的1/4左右, 节省磁盘空间, 便于维护、管理、更新和应用。

    (2) 提供常用的计算反应谱、相关、互相关、谱比、P波到时拾取、预警震级计算等强震动数据分析处理方法。

    (3) 可视化管理数据, 快速了解数据时间、空间、震级、加速度、震源机制的分布情况, 便于检索地震学及地震工程需研究中所需的地震数据。

    (4) 提供研究人员集成自行开发算法扩展开发的接口。

    (5) 本系统稍作修改即可应用到其他台网的数据管理分析中去。

    (6) 根据NIED的规则, 个人不能分发台网的数据, 本软件主要是供个人研究者使用, 提高个人研究中数据使用的效率。
