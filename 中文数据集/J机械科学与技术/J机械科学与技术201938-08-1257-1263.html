    刀具磨损的机器视觉监测研究

    摘    要：

    为提高铣削加工时的刀具利用率、降低刀具成本, 提出采用机器视觉技术在机监测铣刀磨损状态, 及时更换刀具。建立刀具磨损监测系统, 由电荷耦合器件 (Charge coupled device, CCD) 相机获取刀具磨损图像, 通过图像预处理、阈值分割、基于Canny算子和亚像素的边缘检测方法建立刀具磨损边界, 提取刀具磨损量。开展GH4169镍基高温合金铣削实验, 将监测系统检测的磨损量与超景深显微镜的测量结果进行比对, 结果表明:该系统具有较高的检测精度, 可实现铣削加工时刀具磨损状态的在机监测。

    Study on Tool Wear Monitoring using Machine Vision

    Peng Ruitao Jiang Haojian Xu Ying Tang Xinzi Zhang Shan

    Engineering Research Center of Ministry of Education for Complex Track Processing Technology and Equipment, School of Mechanical Engineering, Xiangtan University AECC Hunan South Astronautics Industry Co., Ltd.

    Abstract：

    In order to improve tool utilization and reduce tool costs in milling process, a new approach to monitor tool wear state and replace tool in time by using machine vision technology was presented. A tool wear monitoring system was established. The wear images of the tool were obtained by using a charge coupled device (CCD) camera, and the wear boundaries were established by using image preprocessing, threshold segmentation and edge detection based on Canny operator and sub-pixel, then the wear value of the tool was extracted. Milling experiments of Superalloy GH4169 were carried out. The wear values detected by using the monitoring system were compared with that obtained by using ultra-deepth microscope. The results showed that the wear monitoring system had a high detection accuracy and enabled on-machine monitoring of tool wear in milling process.

    Keyword：

    tool wear; machine vision; image processing; edge detection;

    铣削加工中, 刀具磨损状态对工件的加工精度和表面完整性有显著影响, 刀具过度磨损将导致加工表面质量的恶化[1]。为降低刀具磨损对加工造成的不利影响, 通常根据人工经验对刀具进行定期更换。据统计, 在加工过程中, 由于更换刀具引起的机床停机时间可占总停机时间的20%[2], 刀具成本可占总加工成本的3%至12%[3]。因此, 探索有效的刀具磨损监测技术, 对降低加工成本、提高加工质量与效率具有重要意义。

    近年来, 国内外学者对刀具磨损监测技术进行了大量研究, 依照检测原理可将其分为间接法与直接法两种类型。由于刀具磨损量直接影响切削力、机床功率、刀具/工件振动、声发射信号、工件表面纹理等参量的特性, 间接法通过监控这些参量的变化判断刀具的磨损状态。谢庆陆, 王国锋[4]分析了刀具磨损情况对切削力时域特性的影响, 基于支持向量机 (SVM) 技术对铣刀磨损量进行了监测。Rao等[5]采用人工神经网络分析加工中工件振动的时域、频域特性, 建立了刀具磨损状态的预测模型。Ren等[6]设计了一种二型模糊逻辑系统, 通过分析铣削过程中声发射信号的变化情况分析刀具的磨损状态。然而, 间接法通常需要采用专门的测量设备, 成本高昂, 且监测的变量同时受加工参数、冷却方式等的影响, 检测精度一般低于直接法[7]。

    直接法指在加工过程中直接监测刀具的磨损状态, 主要采用基于图像处理方法的机器视觉技术提取刀具磨损区域、计算刀具磨损量。Su等[8]利用机器视觉采集刀具磨损图像, 通过边缘检测方法提取刀具磨损边界, 检测刀具后刀面磨损状态, 具有较高的效率。秦国华等[9]提出一种局部方差阈值分割算法。实验结果表明, 该算法可以有效提取刀具磨损区域, 获得清晰完整的磨损边界。Jurkovic等[10]采用机器视觉技术建立了刀具磨损在线监测系统, 包括电荷耦合器件 (CCD) 相机、光源、线性投影仪、计算机等, 可以准确获取刀具磨损区域的三维图像。Kim等[11]利用图像处理软件建立了基于机器视觉的宏程序, 可对刀具磨损的几何参数进行自动测量。

    通过分析和总结前人的研究思路, 本文提出一种基于机器视觉的铣刀磨损监测系统, 通过图像预处理、阈值分割和边缘检测方法在机监测刀具的磨损量。为检验监测系统的性能, 开展GH4169镍基高温合金平面铣削实验, 结果表明:此监测系统具有较高的检测精度。

    1 刀具磨损评估指标

    切削过程中, 刀具、切屑和工件间的力-热耦合作用使刀具发生磨损和破损。刀具磨损状态直接影响加工质量, 磨损到一定程度就需换刀。实际生产中, 通常依据刀具后刀面磨损情况建立刀具磨钝标准, 原因在于后刀面与刚度较大的工件相接触, 磨损一般较前刀面剧烈[12]。ISO 3685标准规定, 车削时可将刀具后刀面磨损带宽度VB作为衡量刀具磨损状态的评估指标。当后刀面呈现规则磨损时, 磨损带平均宽度VBave不得超过0.3 mm;当后刀面呈现不规则磨损时, 磨损带最大宽度VBmax不得超过0.6 mm。铣削为多切削刃断续切削过程, 刀具磨损情况更为复杂, 仅考虑后刀面磨损宽度难以真实、完整的评估铣刀的磨损状态。有学者指出, 铣刀后刀面磨损面积AVB可对刀具磨损情况作二维度量, 能更准确地反映铣刀磨损的真实状况[8,13]。

    基于以上讨论, 本文将铣刀后刀面平均磨损宽度VBave、最大磨损宽度VBmax、后刀面磨损面积AVB作为磨损评估指标进行监测, 如图1所示。

    2 刀具磨损监测系统

    基于机器视觉的刀具磨损监测系统主要由硬件和软件两部分组成。硬件部分包括CCD相机、光学镜头、相机支架、光源和计算机。其中, CCD相机选用Cognex公司生产的In-Sight 5000工业相机, 工作时可直接采集刀具的灰度图像;镜头配备Computar公司生产的M0814-MP2FA工业镜头, 最近物象距离为0.1 m;光源为LED环形光源, 加装在镜头前方, 提供必要的照明条件。软件部分包括Cognex In-Sight explorer软件和设计的图像处理软件, 前者控制相机进行对焦和图像采集, 后者根据采集的刀具磨损图像检测刀具磨损量。监测系统总体架构如图2所示。

    图2 刀具磨损监测平台   下载原图

    系统工作原理如下所述。CCD相机通过相机支架安装在机床工作台上, 相机与刀具之间的距离约为0.3 m, 以便获取刀具的完整图像。铣削过程中, 铣刀在一次走刀完成后停止转动, 调整被测刀具位置, 采用CCD相机拍摄刀具后刀面的灰度图像并通过图像采集卡将图像传输至计算机。由图像处理软件对刀具图像进行分析, 提取刀具磨损边界, 结合像素当量大小计算刀具的实际磨损量。

    3.1 图像预处理

    CCD相机拍摄并传输到计算机的原始图像包含刀具整体部分和不必要的背景环境, 如机床导轨、主轴、工件等。为减少计算量, 在磨损量检测之前需要对原始图像进行裁剪和旋转, 以消除环境因素的干扰并保留刀具的磨损区域, 如图4所示。

    图4 对原始图像裁剪和旋转   下载原图

    采集刀具图像时, 环境的亮度及光照条件等通常对图像产生噪声干扰。对基于像素点灰度值大小进行计算的图像处理方法而言, 噪声对后续图像处理结果影响较大, 需要对刀具图像进行降噪处理。中值滤波是一种高效的降噪算法, 可以在保持图像细节信息的同时最小化噪声的影响[14]。采用7×7中值滤波算法, 对图像中任意像素点 (i, j) , 在以该点为中心的7×7邻域内对所有像素点灰度值进行统计排序, 将排序中值定为该点处理后的灰度大小。图像预处理见图5。

    3.2 阈值分割

    图像预处理完成后, 采用区域生长法对图像进行阈值分割, 得到刀具后刀面的二值化图像。区域生长法基于定义的生长准则将图像中同类像素点聚合至相同的区域, 其处理流程如下所述。首先, 在刀具图像中随机选取像素点作为生长点, 在其邻域内将灰度值相差范围在[0, 1] 内的像素点或区域与其聚合, 形成新的区域。此后不断重复以上过程, 直到计算达到收敛为止。最后, 依据生长完成的区域对图像进行二值化处理。铣刀后刀面阈值分割完成后的图像如图7所示。

    3) 对梯度幅值进行非极大值抑制, 将边缘强度大于梯度方向上相邻两点的像素点标记为候选边缘, 将非局部极大值点赋予零值。

    3.4 磨损量的计算

    3.4.1 像素当量的标定

    基于图像处理方法获取的铣刀磨损量以像素为单位, 需要转换成实际尺寸才能得到真实的刀具磨损值。以铣刀直径作为参考标准进行像素当量的标定, 若铣刀直径为D, 在图像中的像素长度为N, 则该图像的像素当量表示为

    k=DΝ          (7)

    将由图像处理方法检测得到的磨损量与像素当量相乘, 即可得到铣刀后刀面的实际磨损量大小。

    3.4.2 刀具磨损量的计算

    计算铣刀后刀面磨损量时, 需要对刀具磨损区域的上边界进行重建, 以消除切削刃的不规则破损对检测结果造成的不利影响[15]。铣刀切削刃本为螺旋线, 由于采集的刀具图像经过局部放大处理, 边界重建时可将其近似为直线, 以提高计算效率。对刀具切削刃的上边缘进行列扫描, 提取各边缘像素点纵坐标的平均值建立刀具磨损区域的上边界, 如图9所示。

    图9 铣刀后刀面磨损区域提取   下载原图

    依据建立的刀具磨损边界, 进行铣刀后刀面磨损量的计算。对刀具磨损区域进行列扫描, 上下边界纵坐标的差值即为后刀面磨损宽度VB。将各像素点处的VB值进行统计排序, 其平均值和最大值分别为后刀面平均磨损宽度VBave和后刀面最大磨损宽度VBmax;将各点VB值沿水平方向进行积分, 即可得到后刀面磨损面积AVB。

    3.5 自动检测软件

    依据上述图像处理流程, 基于Matlab 2014平台将相应图像处理方法集成到一个GUI界面, 设计了基于机器视觉的铣刀磨损量自动检测软件, 如图10所示。将设计的软件与硬件部分相结合, 构成刀具磨损监测系统, 以实现铣刀磨损量的在机监测。

    5 结论

    1) 提出采用机器视觉技术对铣刀磨损状态进行在机监测, 建立了刀具磨损监测系统。系统由硬件和软件两部分组成, 通过硬件部分采集刀具后刀面磨损图像, 由软件部分进行图像处理, 自动检测铣刀后刀面磨损量。

    2) 进行镍基高温合金平面铣削试验, 检验刀具磨损监测系统的性能。结果表明:监测系统与显微镜测量的刀具磨损量相对误差在7.53%以内, 设计的刀具磨损监测系统具有较高的精度, 可用于铣削加工时刀具磨损状态的在机监测。
