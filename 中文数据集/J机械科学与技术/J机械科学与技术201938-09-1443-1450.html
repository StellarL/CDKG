    摘    要：

    掌握智能滚齿机的智能化状态, 改进智能滚齿机的薄弱环节, 是提升滚齿机智能化程度的重要途径。通过借鉴能力成熟度模型集成技术, 构建智能滚齿机成熟度模型并给出其主要特征以描述滚齿机智能化发展进程, 根据滚齿机的智能特性建立智能滚齿机成熟度综合评价指标体系。在此基础上, 提出一种区间层次分析法和模糊综合评价相结合的智能滚齿机成熟度评价方法, 其量化结果能合理地评估智能滚齿机成熟度, 直观得到滚齿机智能化的改进方向。通过对企业某台智能滚齿机的评价, 验证所提方法的有效性。

    关键词：

    智能滚齿机; 成熟度; 区间层次分析法; 模糊综合评价法;

    A Method to Evaluate Maturity of Intelligent Hobbing Machine

    Wang Xingrong Yan Chunping Wan Lu Cao Weidong

    The State Key Laboratory of Mechanical Transmission, Chongqing University

    Abstract：

    It is an important way to improve the intelligent degree of hobbing machine by grasping the intelligent state and improving the weak link of intelligent hobbing machine. To describe the smart development process of hobbing machine, a maturity model for smart hobbing machine was built by referring capability maturity model integration technology. On the other hand, a smart hobbing machine maturity comprehensive evaluation index system was established to describe the characteristics of intelligent hobbing machine. In order to evaluate the maturity of smart hobbing machines, a method combined with interval analytic hierarchy process and fuzzy comprehensive evaluation was proposed. The quantitative evaluation results can reasonably assess the maturity of smart hobbing machines and point out the improvement direction for smarter. An example with a smart hobbing machine in a business verified the present effectiveness.

    Keyword：

    smart hobbing machine; maturity; interval analytic hierarchy process; fuzzy comprehensive evaluation;

    目前智能制造已成为制造业转型升级的趋势和方向, 智能化是提升齿轮制造装备的重要手段, 机床智能化是机床装备发展的趋势[1]。滚齿机作为典型的齿轮制造装备, 在整个制齿加工机床中的比例约占45%[2], 在我国制造加工业中占据重要地位。智能滚齿机技术研究中, 国内外主要集中在以下几个方面:1) 滚齿机加工准备智能化方面;文献[3]提出了工艺参数智能优化决策方法, 文献[4]提出基于成组技术和标准模板的滚齿CNC自动编程方法, 文献[5]讨论了现有虚拟制造系统的焦点和方法, 文献[6]研究了最优化节能感知调度方法, 文献[7]提出一种基于任务的可靠性分配方法;2) 滚齿机加工过程智能化方面, 文献[8]研究了具有在线进给率优化功能的先进数控系统, 文献[9]研究了自动调整过程监控系统, 文献[10]提出一种用于干切滚齿机热偏差计算和补偿的方法;文献[11]对齿轮精加工滚齿加工系统进行研究以提高齿轮加工精度。文献[12]提出一种用于数控机床运行过程能量效率分析, 能量消耗预测和节能优化的数学模型。文献[13]提出了一种刀具破损在线监测方法;3) 滚齿机的维修维护智能化方面, 胡林桥针对滚齿机故障诊断、维护知识智能决策以及滚齿机维护服务的网络化、智能化设计与应用进行了研究, 实现了网络智能滚齿机故障诊断及维护系统的设计与应用[14]。

    综上所述, 国内外对智能滚齿机的单项技术研究较多, 也取得较好的应用成果, 但对于滚齿机整体的智能化提升还没有一致认识, 对滚齿机智能化的发展路径尚未明确, 如何确定滚齿机智能化提升方向还缺乏较为有效的指引。“成熟度”是产品设计中间状态可参考性的衡量标准, 即产品成熟度越高, 后续改进的必要性就越小[15]。随着智能化研究的不断深入及滚齿机智能化发展要求的不断提高, 进行智能滚齿机成熟度评价, 使其各方面性能朝着最优化发展, 符合制造智能化发展的要求。通过确定滚齿机智能化发展路径以及其不同等级成熟度的关键特征和要素, 建立合理的智能滚齿机成熟度评价指标体系, 采用恰当的评价方法, 得到智能滚齿机成熟度等级, 可了解并掌握滚齿机智能化开发的进展和完成情况;指导智能滚齿机开发者实现对智能滚齿机成熟度关键因素的全程控制, 为滚齿机智能化提升指明方向。

    1 智能滚齿机成熟度模型

    1.1 智能滚齿机特征描述

    智能滚齿机是通过自动感知、智能决策以及智能执行方式, 组合智能辅助决策、智能监测、智能调节和智能维护等智能功能, 支持所在制造系统高效、精密、优质和低碳等多目标优化运行[1]的制齿机床。智能滚齿机具有加工准备智能化、加工执行智能化、维修维护智能化的功能特征, 具有智能规划与决策、持续制造评估优化、智能检测与诊断、自适应优化控制、集成与互联互通、故障智能维护及知识智能维护的技术特征, 能够支持齿轮加工过程的工艺参数智能优化决策、数控程序自动编制等智能功能。

    1.2 成熟度模型建立

    “成熟度模型”是一个概念性框架, 一般包含对相关领域成熟度的定义和如何对成熟度进行评估、改进的方法及过程等内容[16]。借鉴能力成熟度模型集成 (Capability maturity model integration, CMMI) 技术中的目标分级建立智能滚齿机成熟度模型, 综合考虑智能滚齿机的整体与局部的协调性、智能的恰当性与无止性、自学习及其能力持续提高性、结构的开放性和可扩展性、制造和加工的绿色性及智能的贯穿性[1]等多个方面进行成熟度等级划分, 根据智能滚齿机从初级、提高、成熟及持续改进的发展历程, 将智能滚齿机成熟度等级划分为依次递增的4个等级, 实现智能滚齿机的一个可持续改进的层次划分, 确定滚齿机智能化评定和发展的标尺。其成熟度模型如图1所示。

    1) 级别1:规范级智能化

    这个级别的智能滚齿机处于滚齿机智能化提升的初级阶段;具有完备的电气系统、数控系统、液压、冷却及润滑系统, 各个系统各自能实现自动化控制;彼此独立工作, 不能实现整体协同, 不能进行计算机控制操作, 滚齿机工作安全可靠。

    2) 级别2:集成级智能化

    这个级别的智能滚齿机处于滚齿机智能化提升的中级阶段;各个系统间实现协同工作, 计算机、滚齿机协同工作但没有重视人在智能滚齿机中的作用;有智能控制软件, 支持远程监控, 但滚齿机结构不开放, 不具备自学习及其能力提升功能, 加工质量较高但加工绿色性不能保障, 滚齿机工作安全可靠。

    3) 级别3:优化级智能化

    这个级别的智能滚齿机处于其智能化的高级阶段;人、计算机、滚齿机以及相应智能控制软件相互关联、协同工作, 人、机床装备在信息库支撑下相互配合, 能基本实现整体与局部协调, 机构部分开放;可配置指定功能部件和软件, 滚齿机具有加工过程绿色性, 可实现低碳加工;支持自学习相关控制和决策算法, 但是能力提升幅度小;支持远程监控, 部分可实现优化加工和维修维护, 有较高的加工效率、加工质量和较为合理的加工成本。

    4) 级别4:引领级智能化

    这个级别的智能滚齿机处于其智能化的顶级阶段;人、计算机、滚齿机以及相应智能控制软件相互关联、协同工作;各智能系统可实现局部协调, 人、滚齿机及控制软件在信息库支撑下相互配合, 可实现整体协调, 智能滚齿机结构开放可扩展;随时可根据新的需要, 配置各种功能部件和软件, 可保证滚齿机设计、生产, 加工过程及加工产品的绿色性;支持自主学习、自主检测、自我优化及共享方式等提升智能化水平, 支持远程的监控和自主维护维修, 能实现高效、精密、优质和低碳等多目标优化运行。

    四梯级的分层递阶智能滚齿机成熟度模型是评估智能滚齿机智能化成熟度的框架, 它描述了滚齿机智能化过程功能水平持续增长的路径, 并且能够为提高滚齿机智能化水平提供指导依据。

    1.3 智能滚齿机成熟度评价指标体系

    为细化评估指标, 简化评估过程, 将评估指标进行适当的层次划分, 构建智能滚齿机成熟度评价指标体系。根据智能滚齿机加工准备、加工执行、维修维护的功能特征构建智能滚齿机成熟度评价指标体系, 目标层为智能滚齿机成熟度评价;准则层包含智能规划与决策能力、持续制造评估优化能力、智能检测与诊断能力、自适应优化控制能力、集成与互联互通能力、故障智能维护能力及知识智能维护能力;指标层则包含对准则层各元素的评价指标;智能滚齿机成熟度评价指标体系如图2所示。

    智能滚齿机成熟度评价指标体系的基本描述如下:

    2 采用区间层次分析法的模糊综合评价方法

    2.1 确定区间判断矩阵和权重向量

    指标权重的确定, 即将影响滚齿机智能化水平的各个因素, 按其对智能化成熟度的影响程度大小分别赋予不同的权重, 以充分体现各个因素在智能滚齿机成熟度综合评价过程中的地位和作用。考虑到智能滚齿机成熟度评价定性指标在评价过程中的模糊性, 将层次分析法 (Analytic hierarchy process, AHP) 在模糊环境下进行拓展, 采用区间分析法对传统AHP中的 (1, 9) 标度扩展为连续型0～9标度[21], 即采用区间层次分析法 (Interval analytic hierarchy process, IAHP) 确定智能滚齿机成熟度评价指标权重。

    准则层指标权重的确定, 由专家对准则层各因素对目标层的重要程度进行两两比较, 并根据实际需要选取0～9范围内的任意区间数, 同时适当限制区间数的上下限差值以防止结果失真。专家根据连续性0～9标度打分, 得出区间判断矩阵为

    A-= (a∼ij) 7×7=[A-, A+]          (1)

    式中:aij表示准则层上第i个因素与第j个因素相对于目标的重要性比值, aij=[a-ij, a+ij], 1≤i≤7, 1≤j≤7;A-= (a-ij) 7×7;A+= (a+ij) 7×7。因为a∼ij=1a∼ij, 故只需要给出矩阵上三角的元素的值。用权重向量法 (Interval eigenvector method, IEM) [22]进行运算求解区间矩阵得权重向量ω-为

    ω-= (ω∼i) 1×7=[kx-, mx+]          (2)

    式中:

    k=[∑j=171∑i=17aij+]12, m=[∑j=171∑i=17aij-]12          (3)

    x+、x-分别为A-、A+的归一化特征向量。当0≤k≤m时, 该区间判断矩阵的一致性较好, 即可采用。若A-=A+, 则IEM与传统AHP中的特征根法所求出的权重向量相同。

    同理, 采用准则层权重确定方法可确定指标层指标的权重, 即

    ω-= (ω∼in) 1×l=[kixi-, mixi+]          (4)

    式中:l为第i个准则层因素下所含的指标个数, 1≤i≤7, 1≤n≤l;ω∼in为第i个准则层因素下第n个元素的权重。如ω∼1表示准则层第一个指标, 即加工任务管理与规划能力中各个子指标相对重要程度的权重向量, ω∼12表示准则层智能规划与决策能力的第二个子指标层属性智能规划与决策能力的相对区间权重值。

    以上所求得的权重向量是一个区间数, 需要对其进行模糊化处理。这里采用ω∼i的上下限均值wi= (ωi++ωi-) 2作为指标的权重值构成指标权重向量, 则

    WX= (w1, w2, ⋯, wn)           (5)

    2.3 模糊综合评价

    模糊综合评价法是以模糊数学为基础, 应用模糊关系合成的原理, 将一些边界不清、不易定量的因素定量化, 根据因素对被评价事物隶属等级状况进行综合评价的一种方法[23]:

    1) 对单因素进行模糊综合评价。确定评价因素集U={u1, u2, ⋯, un}和评价等级集V={v1, v2, …, vn};建立单因素评判矩阵, 即对ui进行评判, 得到V上的模糊集 (ri1, ri2, …, rin) , 其中rij表示ui对vj的隶属度, 则评判矩阵为

    R= (R1, R2, ⋯, Rn) = (rij) n×m          (6)

    用权重矢量W与矩阵R综合, 即可得到从总体上看被评价对象对个评价等级的隶属程度, 即

    B=W⋅R= (b1, b2, ⋯, bm)           (7)

    式中bj=∑i=1nwirij, j=1, 2, ⋯, m。

    2) 模糊综合评判。二级模糊综合评判模型为

    Μ=Wi⋅Bi= (m1, m2, ⋯, mn)           (8)

    式中:Wi为第i个因素的等级权重, Wi= (ai1, ai2, …, aik) ;Bi为第i个因素的一级模糊评判结果;M为各因素之间的模糊综合评判结果。

    3) 计算综合评价分数。若用总分数表示综合评价结果, 取评价标准的隶属度集为μ={很高, 高, 一般, 低, 很低}T, 各级指标的智能化成熟度综合评价得分为

    G=Μμ×100          (9)

    1) 根据滚齿机的智能化本质特性和功能特性等, 从智能规划与决策、可持续制造评估与优化、智能检测及诊断等7个方面构建智能滚齿机成熟度评价指标体系, 为滚齿机床成熟度评价与智能化改进提供了理论依据, 为齿轮智能制造装备的研究提供支持。

    2) 采用区间分析、层次分析法与模糊综合评价法相结合的方法评价智能滚齿机成熟度, 运用区间数代替具体数值构造判断矩阵, 克服了传统方法中定性指标经验判断的模糊性问题;经实例验证, 该方法可获得滚齿机成熟度等级与智能化发展的不足, 为滚齿机智能化改进指明方向。
