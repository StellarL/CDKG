    基于旋量理论3-RRR平面并联机构的误差分析

    屈淑维 李瑞琴 郭志宏

    中北大学机械工程学院

    摘    要：

    机构的输出误差决定着应用性能。以旋量理论为基础, 运用旋量参数与齐次变换矩阵的转换关系, 将影响机构输出误差的铰链间隙及连杆参数等效为虚拟旋量。以3-RRR平面并联机构为对象, 根据其拓扑结构的对称性, 拆分为三个四杆机构。运用虚拟旋量, 建立了四杆机构由输入到输出的误差模型;运用Matlab软件分别对三个输入支链引起的误差进行了理论计算分析。结果表明, 该方法简洁有效, 为各类平面并联机构的误差分析提供了有效的理论参考依据, 对机构的应用分析具有重要的参考价值。

    关键词：

    旋量理论; 铰链间隙; 连杆参数; 误差分析, 3-RRR平面并联机构;

    作者简介： 屈淑维 (1978-) , 女, 博士, 讲师;主要研究方向:机构理论与机器人装备, 已发表论文20余篇。, E-mail:shuweiqu1222@nuc.edu.cn;

    Error Analysis of 3-RRR Plane Parallel Mechanism Based on Screw Theory

    QU Shuwei LI Ruiqin GUO Zhihong

    College of Mechanical Engineering, North University of China

    Abstract：

    The output error of the mechanism determines the application performance. Based on the conversion relation between screw parameters and the transformation matrix, the joint clearance and link parameters which mainly affect the output error of the mechanism are equivalent to virtual screws. The 3-RRR planar parallel mechanism is divided into three four-bar mechanisms according to the symmetry topological structure. The error model of four-bar mechanism from input to output is established by using virtual screw. Matlab software is used to calculate and analyze the errors caused by three transmission drivers. The results show that the method is simple and effective, which provides effective theoretical basis for error analysis of various mechanisms and has important reference value for the application analysis of mechanisms.

    Keyword：

    screw theory; joint clearance; link parameters; error analysis; 3-RRR plane parallel mechanism;

    Received： 2018-10-10

    机构输出误差大小决定着应用性能, 受系统各种因素的影响, 如制造公差[1], 杆件变形[2], 运动过程的冲击磨损[3]等因素。这些因素可归结为两类:第一类为结构参数[4], 另一类为铰链间隙[5]。文献[6]以制造误差及运动副间隙为因素, 运用矢量法建立了有效杆长的计算模型及精度算法。文献[7]运用杆件增量法, 以相邻构件之间的作用力方位以及输出构件参考点的运动轨迹, 建立了机构误差分析传递矩阵。Chen等应用几何方法预测了铰链间隙对输出特征的影响[8,9]。铰链间隙及结构参数作为影响输出特性的主要因素, 具有不可重复的随机特征, 通常采用统计方法进行预测。设计一种准确有效的分析方法对机构输出误差的预测具有重要的理论及应用价值。Qu 等考虑铰链间隙及连杆参数提出了开链机构误差分析的统一模型[10]。如何建立并联机构的误差分析模型, 是机构学研究的重要课题。3-RRR并联机构具有柔性机构和少自由度并联机构两者的优点, 广泛用于平面定位及高速运转场合, 引起了学者们的广泛关注[11,12]。

    本文以旋量理论依据, 借助虚拟旋量对平面3-RRR并联机构的输出误差进行了理论及模拟分析。该方法对其它机构输出误差的分析提供了重要的理论参考价值。

    1 旋量理论

    6 结 论

    本文基于旋量理论, 运用旋量参数与齐次矩阵的变换关系, 将铰链间隙及连杆的参数误差等效为虚拟旋量;得到了平面机构输出误差的统一表达式。

    依据3-RRR平面并联机构的拓扑结构特征, 得到其中一条支链的输出误差, 借助并联机构末端执行器的输出特征, 运通Matlab软件分别对三个驱动输入引起的末端累积误差进行了分析。

    该方法简洁有效, 对各类机构的误差分析提供了理论参考依据。
