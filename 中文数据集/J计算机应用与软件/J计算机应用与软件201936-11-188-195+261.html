    摘    要：

    针对工艺操作说明文本中的命名实体,提出一种基于BiLSTM-CRF模型与词典、规则相结合的识别方法,旨在识别图纸编号、参考标准、零件和零件号等11种实体。基于BiLSTM-CRF模型,使用BERT模型预训练的向量,对相关命名实体进行初始识别;针对工艺操作说明文本中零件和零件号表达方式复杂多样的问题,使用基于词典和规则的方法对此类实体的标注结果进行校正。实验结果表明,该方法在工艺操作说明文本中能较好地完成命名实体识别任务,在测试语料上F1值达到94.03%,比基线提升了4.14%。

    NAMED ENTITY DEEP LEARNING RECOGNITION METHOD FOR PROCESS OPERATION DESCRIPTION TEXT

    Zhang Nana Wang Peiyan Zhang Guiping

    Abstract：

    This paper proposed a recognition method based on BiLSTM-CRF model combined with dictionary and rules for the named entities in the process operation description text. The purpose was to identify 11 entities such as drawing numbers, reference standards, parts and part numbers. The method was based on the BiLSTM-CRF model and used the pre-trained vector of the BERT model to initially identify the relevant named entities. Then, the method based on dictionary and rules was used to correct the labeling results of such entities, which aiming at the complex and diverse expressions of parts and part numbers in the process operation description text. The experimental results show that the method can successfully complete the named entity recognition task in the process operation description text. The F1 value reaches 94.03% in the test corpus, which is 4.14% higher than the baseline.

    Keyword：

    Process operation description text; Named entity recognition; Two-way long-term memory network; Conditional random fields;

    命名实体识别[1](Named Entity Recognition,NER)也称为专名识别,是自然语言处理(Nature Language Processing,NLP)中的一项基础任务,是正确理解文本的基础[2]。其在通用领域中的研究已经相对成熟,如在新闻语料中,人名、地名和机构名的识别已经取得了较好的效果[3]。相比通用领域,专业领域中命名实体的识别往往以该领域的知识为依据,兼顾其语言规律特性,使得专业领域的命名实体识别具有一定的难度。在飞机装配工艺中,工艺操作说明文本是用于指导飞机装配过程中的工艺描述文件。

    面向工艺操作说明文本的命名实体包括工程图纸、参考标准、结构特征、零件和零件号、属性和属性值等11种,其中零件和零件号属于复杂实体,是本文识别工作的难点。由于飞机装配精度要求很高,该文本所涉及的零件总类众多且构成结构复杂,此类命名实体的长度没有一定的限制。例如由单个词的“螺栓”、“高锁螺栓”到词和相关符号组成的“5/32高锁螺栓”。由于零件的种类众多,其对应的零件号也是多样,零件号存在简写问题,如:“-002”。零件和零件号在工艺操作说明文本中所占的比重较大且表达方式复杂多样,并且零件本身没有严格的构词规律可循。

    针对以上问题,本文提出了一种面向工艺操作说明文本的命名实体识别方法。首先基于BiLSTM-CRF-BERT模型对该文本中的相关实体进行初始识别;然后利用基于词典的方法先对零件进行校正;最后,在上述基础上,充分利用零件和零件号的外部信息,用基于规则的方法对零件和零件号进行校正。该方法降低了仅使用BiLSTM-CRF-BERT模型对大规模语料的依赖性,有效地提高了零件和零件号的识别效果。

    1 相关研究

    命名实体识别是指识别出文本中具有特定意义或者指代性强的短语,并对其进行分类。早期的命名实体包括三大类,即实体类、时间类和数字类,这三大类又可以进一步细分为人名、地名、组织机构名、时间、日期、货币和百分比七小类[4]。但随着科学技术和社会的快速发展,命名实体识别的应用范围在不断扩大,从通用领域逐渐扩展到专业领域,针对不同领域内的不同问题,命名实体的种类也越来越多,不再局限于上述几类范围。

    传统的命名实体识别的主要技术方法分为:基于词典和规则的方法、基于统计的机器学习方法以及上述两者相合的方法等。

    基于词典的方法最早应用到命名实体识别任务中,是最简单的一种方法。在识别过程中,使用完全匹配或者模糊匹配这两种词典匹配方式在所构建的词典中查找字符串。Yang等[5]使用了基于词典的方法,采用改进编辑距离来进行词典匹配,在JNLPBA2004语料集上达到了53.68%的综合分类率。该方法虽然简单,但是由于词典的容量有限,不能有效识别未登录词,总体识别的效果在很大程度上受限于词典的规模和质量。

    基于规则的方法多采用领域专家手工构造的规则模板,选用统计信息、标点符号、关键字、指示词和方向词、中心词、词性、词形等特征,以模式匹配和字符串相匹配为主要手段来识别文本中的实体。Olsson等[6]构建的基于规则的系统获得了67.1%的综合分类率。基于规则的方法往往依赖于具体语言、领域和文本风格,编制过程耗时且难以涵盖所有的语言现象。同时这些实体并非按照一定规则来命名,很难建立一套一致的规则,系统可移植性不好,对于不同的系统需要语言学专家重新书写不同的规则。

    在基于统计的机器学习方法中,命名实体识别被当作序列标注问题,即利用大规模的语料学习出标注模型,从而对句子的各个位置进行标注。常用的统计机器学习方法有:支持向量机模型[7](Support Vector Machines,SVM)、隐马尔可夫模型[8](Hidden Markov Model,HMM)、最大熵隐马尔可夫模型[9](Maximum Entropy Markov Model,MEMM)、条件随机场模型[10](Conditional Random Fields,CRFs)等,其中条件随机场模型的表现优于其他统计机器学习模型。同以往的方法相比,基于统计机器学习的方法在系统健壮性、迁移性、识别效果等方面表现更好。但单独使用基于统计机器学习的方法会使得状态搜索空间非常庞大,其主要存在以下两个缺点:(1) 对特征选取的要求较高,需要从文本中选择出对该项任务有影响的各种特征,并将这些特征加入到特征向量中,该过程耗时耗力;(2) 对语料库的依赖比较大,而可以用来建立和评估命名实体识别系统的大规模通用语料库又比较少。

    两者相结合的方法是指在基于统计的机器学习方法中引入规则或者加入词典信息,是近年来解决命名实体识别任务的主流方法之一。文献[11]提出了一种CRF与规则相结合的方法来实现维吾尔文地名的识别。首先采用CRF的适应性优势对维吾尔文地名进行初始识别,然后利用规则对初始识别错误的实体进行后处理。虽然两者相结合的方法比上述方法单独使用时,具有较好的准确性及鲁棒性,但是该方法的算法复杂度较高,并且在面对大规模多领域复杂的文本数据时,召回率不稳定,识别效果较差。

    近年来,深度学习模型被广泛用在NER任务中,通过深度神经网络自动学习高层的特征,不依赖于特征工程,减少了人工定义特征和对领域知识的过度依赖,实现了端到端的学习模式。常用的模型有长短时记忆网络[12,13](Long Short-term Memory,LSTM)、卷积神经网络[14](Convolutional Neural Network,CNN)以及门限循环单元[15](Gated Recurrent Unit,GRU)等。上述网络通常使用语言模型预训练的词向量来处理中文NLP任务,如命名实体识别、文本分类、机器翻译等任务存在一定的相关性,底层网络参数抽取的特征可通用,使用预训练语言模型如BERT(Bidirectional Encoder Representations from Transformers)[16],可将已学到的模型参数共享给新模型从而加快并优化新模型的学习效率。深度学习模型具备较强的序列建模以及神经网络拟合非线性函数能力,能够很好地捕捉上下文信息,这些也是优于其他统计机器学习的地方。

    通用领域的命名实体识别已经投入了大量的研究,F值可以达到90%以上[17]。但在飞机装配工艺中,工艺操作说明文本不同于通用领域的命名实体,有其独特命名法则,识别存在一定的难度,该领域中的命名实体识别的相关研究较少。本文使用深度学习模型结合基于词典和规则的方法,取词典和规则之长,补深度学习之短,不仅能减少机器学习模型中人工设计特征的工作量,而且能够自动从字或者词获取更为有效的特征。

    2 基于BiLSTM-CRF模型的初始识别

    本文采用的BiLSTM-CRF模型框架如图1所示,该模型共有三部分组成:向量层、BiLSTM 网络层、CRF层。首先将输入的语句转化为向量形式,然后输入到BiLSTM网络层,最后结合CRF层来获取全局最优的输出序列。

    2.1 向量层

    向量的特征表示与传统的基于统计来记录上下文特征的方法相比,可以更好地捕捉到数据之间的固有联系,包含了更丰富的信息,使得深度神经网络模型具有比传统模型更强的特征抽取能力。因为神经网络的输入是向量形式, 所以应先将工艺操作说明文本转化为向量。文本向量化有两种表示方法: One-hot 表示和分布式表示[18]。在One-hot向量表示方法中,任意两个词之间是孤立的,无法表达不同词之间的语义关系,容易出现矩阵稀疏和维度灾难的问题。词语的分布式表示通过学习文本,用词向量的方式表征词的语义信息,能把词映射到低维的向量空间,用向量空间的相似度表示文本语义上的相似度,可以较好地表达不同词之间的相似和类比关系,有利于解决工艺操作说明文本语义含糊的问题。

    本文所使用的神经网络模型依赖于底层的向量的特征表示,因此本文分别使用了两种向量生成方法:BERT模型和Word2Vec模型。前者采取Transformer[19]构成的深度神经网络作为特征提取器,第一阶段利用双向语言模型,即根据当前词的上下文来预测当前词,进行预训练,把相关的语言学特征抽取出来并编码到特定任务的网络结构中;第二阶段利用第一阶段预训练好的参数来初始化特定任务的网络结构,并通过微调(Fine-tuning)的模式解决特征集成等下游任务。对于后者,其包含跳字模型(Skip-gram)和连续词袋模型(Continuous Bag of Words,CBOW),由于本文所使用的语料规模相对较小,所以选用CBOW模型。

    2.2 BiLSTM网络层

    工艺操作说明文本中命名实体的正确识别取决于其所在的上下文,即当前状态之前的状态和之后的状态对预测标签是同等重要的,句子中各个字或者词应该是平权的,如果能够获取过去和将来的信息,对命名实体识别任务很有帮助。如工艺操作说明语句:“按某标准检查______去毛刺和倒角质量。”和“按某标准检查______安装质量。”这两个工艺操作说明前面的内容是一样的,仅从当前词的上文可能预测出“凸头高锁螺栓孔”、“凸头高锁螺栓”等,但若融入文本序列的下文信息,那么工艺操作说明一的当前词是结构特征“凸头高锁螺栓孔”,而工艺操作说明二的当前词是零件“凸头高锁螺栓”的概率就增大了。在BiLSTM中,每一个训练序列向前和向后是两个LSTM,并且都连接着一个输出层,此结构提供给输出层输入序列中每一个点过去和未来的信息,其模型结构如图2所示。

    2.3 CRF层

    CRF模型是Lafferty 等[20]基于最大熵模型和隐马尔科夫模型,提出的一种判别式概率无向图学习模型。由于CRF能考虑到相邻标签之间的关系,所以将CRF融合到BiLSTM模块的最后一层中,即相当于对BiLSTM层的信息进行再利用,进而获得全局最优的标记序列。对于每一个给定的工艺操作说明语句,其对应的输入序列为X={x1,x2,…,xn},将特征向量映射到k维,k是标签的种类,定义P=(p1,p2,…,pn)∈Rn×k是BiLSTM层输出的分数矩阵,n是训练集中词的个数。Pij表示的是句子中第i个词xi分类到第j个标签的概率值。对于句子长度为n的标签预测序列y=(y1,y2,…,yn),它的打分为:

    S(X,y)=∑i=0nAyi,yi+1+∑i=1nΡi,yi (2)

    为了描述标签间的依赖关系,引入CRF层进行句子级别的序列标注。CRF 层的参数是一个为(k+2)2 的转移分数矩阵A,其中Aij表示从第i个标签转移到第j个标签的概率值。这样在为一个字或者词进行标注的时候可以利用之前已经标注过的标签信息。在输入序列X的条件下产生标记序列y的概率为:

    p(y|X)=eS(X,y)∑y˜∈YXS(X,y∼) (3)

    在训练的过程中,我们要最大化正确标签序列的似然概率:

    log(p(y|X))=S(X,y)-log(∑y∼∈YXS(X,y∼)) (4)

    式中:YX是指一个输入序列X对应的所有可能的标签序列。预测时,通过以下公式输出整体概率最大的一组标记序列:

    y*=argmaxy∼∈YXS(X,y∼) (5)

    3 基于词典与规则的校正识别

    经过人工分析发现,工艺操作说明文本中的零件和零件号的外部构成存在一定的规律。为进一步提高零件和零件号的识别效果,本文通过分析零件和零件号前后的用词特点,充分发挥零件和零件号所在上下文对其的约束作用,先用基于词典的方法对初始识别后的零件进行校正,再用规则对零件和零件号的标注结果进行校正。

    3.1 基于词典的方法

    本文人工收集建立零件命名实体库即词典,词条数总计1 259个,如“托板螺母”、“剪切角片”等,词典中的一个词条表示一个零件类型的命名实体。本文依据该词典对BiLSTM-CRF-BERT模型初始识别的结果进行逆向最大匹配,此处校正的是零件。基于词典的文本匹配算法流程如图3所示。

    3.2 基于规则的方法

    工艺操作说明文本中频繁使用大量相似的句型句式,如“按照图纸…制…与…的连接孔(导孔或者终孔)”、“按照…安装…”、“按…及参考…定位…并钻定位孔”等,此类句型句式归纳总结为21条。零件和零件号经常嵌套在上述结构中,并且零件后面一般紧跟其对应的零件号。

    上述句型句式中,包含一类固定的词,这些词的出现预示着零件或零件号的出现,将这些词称为触发词。工艺操作说明文本中趋向动词的使用有助于零件或零件号的识别,如“定位”、“安装”、“制出”、“钻制”、“拆除”等,指示词“零件”后面常跟零件号,此类触发词归纳总结为15个。这些词有着鲜明的特征,在一定程度上标识着零件和零件号的类别和边界。 因此,可通过对相似的句型句式结合触发词的识别来校正以上两种实体。如当前句型句式是“根据…完成对…的安装。”时,首先判断“对”和“的”之间的字段是包含英文字母及数字,若没有,则该字段是零件;若有,继续判断是否包含中文字符。若没有中文字符,则该字段是零件号,若有中文字符,则中文字符和英文字母及数字分别是零件和零件号。

    4.2 标注策略

    命名实体识别任务常用的标注方式有以下三种:BIO、BIEO、BISEO,其中:B表示实体首字,I表示实体中间,S表示单个实体,E表示实体尾字,O表示非实体。理论而言,标注得越精细,识别后的效果也越好。结合本文所使用的语料中单个字的实体出现次数非常少的情况,同时为了更好地找出起始和结束边界, 本文采用BIEO形式的标注方式,即定义4种实体范围标签。以下面的工艺操作说明语句为例:

    按照D02344200要求,制出D02313770-005与框之间的16个螺栓孔。

    评判一个命名实体是否被正确识别通常包括以下两个方面:首先,判断实体的边界是否正确,即确定哪些字或者词属于命名实体;其次,判断实体的类型是否标注正确。本文主要通过准确率P、召回率R及F1值这三个指标来对工艺操作说明文本的实体识别结果进行评测,具体公式为:

    本文使用统计机器学习模型CRF测试的结果作为baseline,在十折交叉验证下,比较其与神经网络模型以及本文所提方法在工程图纸、图区、参考标准、零件等11类实体识别上的表现,如表4-表6所示。表4中实验11即本文提出的方法在大部分类别的实体识别效果上,取得比baseline和其他神经网络模型更好的表现,F1值比baseline提高了4.14%。通过分析实验结果,可推得以下研究结论:

    (1) 工艺操作说明文本中的方法图、结构图、图区、属性、零件数量都是属于简单实体,此类实体的特征相对明显,如结构图“US7058240-42”,从词构成而言,结构图的构成具有一定的规律,主要表现形式是“[US]+数字和横线”,虽然构成结构图的各部分数字具有任意性,但总体还是表现出相对集中的特点。并且结构图的前缀“US”是此类实体识别的有用知识。所以,baseline和神经网络模型对以上实体的识别效果表现相当。而baseline 在参考标准这类实体的F1值优于神经网络模型,因为参考标准在工艺操作说明文本中分布较少,并且特征不明显,神经网络模型难以更好地抽象出此类实体所包含的特征。

    CRF是通过特征来区分实体及其类型和非实体,难以捕捉到那些特征不明显的实体,如零件。工艺操作说明文本中的零件存在一定的嵌套及交叉现象,边界错误是影响此类实体识别正确与否的原因之一。如:零件“高锁螺栓”覆盖了零件“高锁”,在有些工艺操作说明语句中,CRF模型只能识别出“高锁螺栓”中的“高锁”;零件“龙骨梁下纵梁”和“翼下纵梁”具有交叉“下纵梁”的现象,CRF只识别出“下纵梁”,即在判断实体的边界时发生错误,而BiLSTM-CRF-BERT+校正方法能在一定程度上克服此问题。在零件识别效果上,不加校正的BiLSTM-CRF-BERT模型较baseline的F1值提升了4.73%。究其原因,主要在于向量的分布中隐含了相对丰富的上下文语法和语义信息,经过BiLSTM-CRF-BERT模型的抽象,可以较好地学习出来。

    本文提出的方法在BiLSTM-CRF-BERT模型的基础上,利用零件前后的用词特点,依次采用基于词典和规则的方法对零件的识别结果进行校正,零件的准确率、召回率和F1值最终可达到83.93%、90.14%和86.92%。该方法在BiLSTM-CRF-BERT模型的基础上继承了基于词典和规则方法的优点,如工艺操作说明语句“按861B3205-000-001的26D区完成SD541框与补偿片、角材、角片间连接的4个标准件的安装。”中“SD541框”在BiLSTM-CRF-BERT模型中被识别为非实体,通过构造的零件词典和规则可以确定“SD541框”是零件。校正后,零件的准确率、召回率和F1值分别提升了2.23%、4.08%和3.27%。

    (2) 表4中的实验结果表明,BiLSTM-CRF-BERT模型对工艺操作说明文本中零件号的识别效果有所提升,较baseline的F1值提升了3.04%,但是其准确率和F1值较其他类型实体如图纸仍然较差。

    通过文本分析发现,零件号的识别需要更加充分的上下文语境信息来区分,并且零件号的上下文,即外部信息存在一定的规律。零件后面一般紧跟其对应的单个或多个零件号,且零件号是由大写的英文字母、数字和横线构成。若零件识别错误,在一定程度上会导致零件号识别出错,即发生错误传播现象。对于此类情况,本文先使用基于词典和句型句式结合触发词规则的方法校正标注错误的零件,然后再使用句法句式结合触发词的规则校正零件号,能在一定程度上提高零件号的识别效果。由表6可知,校正后的零件号的准确率、召回率和F1值与没有校正的BiLSTM-CRF-BERT模型相比,分别提升了3.02%、3.05%和3.21%。

    此外,有些零件号和某些工程图纸组成结构相似,造成有些零件号的类型标注发生错误。由于工艺操作说明文本的某些工程图纸和零件号等是由英文字母、数字和横线组成,并且长度一样,此类实体之间的界限不清晰,特征区分不明显,没有校正的模型将有些零件号识别为工程图纸或者非实体。例如,工艺操作说明语句:“按照〈DPAS151-010〉〈DPAS157-028〉根据772C0000-000-001和684B0000-100-001湿安装紧固件。”,没有校正的模型把“684B0000-100-001”识别为零件号,而校正后的模型就能正确识别为工程图纸。

    (3) 对比表4和表5的实验结果,本文方法的表现比IDCNN-CRF要好,原因在于IDCNN在卷积之后,末层神经元可能只是得到了原始输入数据中一小块的信息,即仅仅考虑到整句话的部分字或词,提供的上下文信息有限。而对NER来讲,整个句子的每个字或者词都有可能都会对当前需要识别的字或者词做出影响,而使用BiLSTM能控制和保存内部记忆,该网络的记忆力能记住更长的信息来对某个字或者词进行识别。比如,下面的工艺操作说明语句:“与顶部蒙皮相连的对接区域连接带板需填角密封。”,IDCNN-CRF把“连接带板”识别为非实体,而本文方法能正确识别为零件。综上所述,本文提出的方法在整体识别效果上表现更佳。

    5 结 语

    针对工艺操作说明文本的命名实体识别的问题,本文提出了BiLSTM-CRF-BERT模型与词典、规则相结合的方法。该方法在BiLSTM-CRF-BERT模型初始识别的基础上,根据零件和零件号的外部信息的规律特征,用词典和规则对零件和零件号进行校正。实验结果表明,该方法取得了较好的效果,F1值达到了94.03%,比baseline提高了4.14% ,能有效解决工艺操作说明文本的命名实体识别问题。

    本文在初始识别时所使用的BiLSTM-CRF-BERT模型,其输入是基于字级的向量特征信息,而字级向量可能丢失一部分工艺操作说明句子本身潜在的特征信息。后期将考虑把句子信息作为一种补充输入,期许在隐层抽象出更加丰富且有针对性的文本信息。
