    摘    要：

    基于可见光红外成像辐射仪多波段、宽覆盖、长重访周期的特点, 以及云层在可见光到热红外通道的分布及变化特性, 提出了一种适用于可见光红外成像辐射仪数据的改进的动态阈值云检测算法;通过遥感目视解译的方法对云检测结果进行精度验证, 并与通用动态阈值云检测算法、可见光红外成像辐射仪云掩膜产品的结果进行对比。结果表明:所提算法能以较高的精度识别不同地表上空的云层, 平均总体精度为93.23%, 平均Kappa系数为0.821, 对薄、碎云的整体识别精度得到了明显提高, 错分和漏分误差明显减小;所提算法的云检测结果整体优于通用动态阈值云检测算法和可见光红外成像辐射仪云掩膜产品的云检测结果。

    Improved Dynamic Threshold Cloud Detection Algorithm for Suomi-NPP Visible Infrared Imaging Radiometer

    Chi Yulei Sun Lin Wei Jing

    College of Geomatics, Shandong University of Science and Technology College of Global Change and Earth System Science, Beijing Normal University

    Abstract：

    Herein, we propose an improved dynamic threshold cloud detection algorithm (I-DTCDA) for visible infrared imaging radiometers (VIIRS) based on the multi-channel, wide coverage, and short revisit period features of a VIIRS.In addition, the algorithm is also based on the characteristics of the cloud distributions and variations in the visible and thermal infrared channels.We validated the accuracy of the cloud detection results using the remote sensing visual interpretation method.We compared our results with those using the universal dynamic threshold cloud detection algorithm (UDTCDA) and the VIIRS cloud mask (VCM) products.The results show that the proposed algorithm has average overall accuracy of 93% (Kappa=0.821) over different surface features.In particular, for the thin and broken clouds, the overall accuracy is significantly improved and the commission and omission errors are obviously reduced.The cloud detection results using the proposed algorithm are superior to those using UDTCDA and VCM.

    remote sensing; cloud detection; improved dynamic threshold cloud detection algorithm (I-DTCDA) ; dynamic threshold cloud detection algorithm (UDTCDA) ; visible infrared imaging radiometers cloud mask (VCM) ; brightness temperature;

    1 引言

    云在遥感影像中普遍存在, 它因遮挡地物而削弱了卫星传感器对地表真实辐射特性的记录, 从而改变了地物的光谱信息[1,2]。因此, 遥感影像云识别是一种非常关键的数据预处理技术, 云识别精度对土地利用分类、多时相变化检测和遥感参数定量化提取等具有重要影响[3,4], 因此, 国内外学者提出了固定阈值法[5]、面向对象法[6]、神经网络法[7]等多种云检测算法。其中的固定阈值法具有简单、快速和稳定的特点, 成为应用最广泛的云检测方法, 在多源卫星云检测中能够达到较高的精度[8,9,10]。然而, 由于薄云或碎云的整体云量较少, 且形状不规则, 与下垫面形成的混合像元保留了与地物相似的光谱特征[11,12], 因此, 很难通过光谱差异确定合适的阈值, 特别是高亮地表, 导致云检测非常困难, 漏判现象比较严重[13]。

    SuomiNPP (National Polar-orbiting Partnership) 卫星搭载的可见光红外成像辐射仪 (VIIRS) 于2011年10月成功发射, 它是高分辨率辐射仪 (AVHRR) 和地球观测系列中分辨率成像光谱仪 (MODIS) 的改进与延续, 是一种新型的对地观测卫星, 被广泛应用于陆地、海洋和大气中地球反照率、海洋水色、云和气溶胶等特性的研究。VIIRS传感器提供了覆盖可见光到热红外波谱范围 (0.3～14μm) 共计22个光谱通道的数据记录, 星下点空间分辨率约为350m, 扫描带边缘空间分辨率约为750m, 刈幅宽度为3000km, 每隔4h经过赤道1次。VIIRS数据使用采样合成技术提高了刈幅边缘区域数据的观测精度, 同时增加了可昼夜连续观测的白天/夜间波段 (DNB) 。VIIRS可收集陆地、大气、冰层和海洋在可见光和红外波段的辐射信息, 对环境监测、灾害预警等具有重要意义。

    国内外学者针对VIIRS数据云检测进行了大量研究。其中, VIIRS云掩膜 (VCM) 算法由联合极地卫星系统 (JPSS) 根据生产环境数据记录过程中对云位置精确定位的需求, 利用多通道固定阈值法生产得到[14]。Hutchison等[15]和Kopp等[14]分别对VCM产品进行了性能预期分析和结果验证, 结果表明, VCM具有生成陆地和海洋表面产品、云物理产品等性能, 并且在确定晴空像元方面具有较大优势。Piper等[16]在VCM算法和Landsat＿7自动云量评估模型的基础上, 提出了一种VIIRS I波段云掩膜 (VIBCM) 快速算法, 该方法利用I波段反射率和亮度温度的测试阈值进行云检测, 简单实用, 具有相对较高的精度。然而, 该算法仅使用单波段的亮度温度, 对冰雪表面的云识别存在一定缺陷。Parmes等[17]提出了一种VIIRS数据自动云检测算法, 该算法的云识别正确率可达94%, 但该算法没有考虑混合像元, 对冰雪及薄云的识别精度相对较低。夏浪等[18,19]结合VIIRS与MODIS数据的光谱特性, 根据不同热红外波段的温度差异, 提出了一种适合中国的云检测算法, 并将云检测结果与遥感目视解译结果进行对比, 整体精度可达到85%, 但该方法的适用性有限。

    尽管国内外学者提出了多种云检测算法, 但这些算法都存在一定缺陷, 大部分算法没有考虑影像中云与冰雪、水体、沙漠和裸地等多种地表形成的混合像元问题。针对这一问题, 本文基于文献[13,20]中的通用动态阈值云检测算法 (UDTCDA) , 提出一种适用于VIIRS数据的改进的动态阈值云检测算法———I-DTCDA算法。该算法基于大气辐射传输方程理论, 利用第二代太阳光谱波段卫星信号模拟程序6S模型, 模拟不同观测几何和大气条件下, 可见光到近红外通道表观反射率与地表反射率之间的关系, 构建不同光谱通道的动态阈值云检测模型;然后分析典型地表在不同热红外通道的亮度温度差异, 引入红外固定阈值法, 以辅助提高遥感影像中薄、碎云及冰雪的识别精度。随机选择大量VIIRS数据进行云检测实验, 采用遥感目视解译方法对云识别结果进行精度验证, 并与原算法的云掩膜结果、VIIRS官方提供的VCM产品进行对比。

    2.2 MODIS数据

    选择经过严格大气校正的MODIS地表反射率产品, 构建先验地表反射率数据库。MOD09A1是由MODIS经8d合成的地表反射率产品, 该产品基于高观测覆盖、低视角、无云或云阴影, 以及气溶胶浓度进行精确定位和几何校正, 并选择每个产品像素在8d内的最佳L2G (Level-2Grid) 进行观测, 具有较高的整体精度[23], 空间分辨率为500m。本研究使用MOD09A1产品的蓝光波段 (0.459～0.479μm) 、绿光波段 (0.545～0.565μm) 、红光波段 (0.62～0.67μm) 和近红外波段 (0.841～0.876μm) , 采用次最小反射率值合成技术构建得到月合成先验地表反射率数据库, 并重采样到750m。该方法能够有效降低云/云阴影和大气气溶胶的影响, 从而较准确地表达真实的地表信息。

    3 VIIRS云检测算法

    3.1 动态阈值云检测模型

    将所提算法利用最小合成法构建的先验地表反射率数据集作为真实的下垫面地表反射率。根据大气辐射传输理论可知[24], 在特定的观测条件和大气条件下, 表观反射率与地表反射率存在强烈的函数关系[25], 因此, 本研究利用6S模型[26]模拟不同几何观测、大气气溶胶光学厚度、气溶胶类型等因子与表观反射率、地表反射率之间的关系。如果待测像元的表观反射率大于模拟得到的表观反射率的最大值, 则该像元被判定为云像元, 否则为晴空像元。利用最小二乘法拟合二者极大值间的函数关系, 构建得到可见光和近红外4个通道的动态阈值云检测模型, 并取4个通道云检测结果的并集作为最终的云检测结果。M3、M4、M5和M7这4个光谱通道的动态阈值云检测模型为

    3.3 薄、碎云的识别

    UDTCDA算法能够较准确地检测出厚云、大部分薄云及碎云, 但对于沙漠、裸地/岩石、冰雪等高亮地表和水体上空的薄、碎云, 仍可能会出现薄云的误提现象。针对该问题, 在原有算法的基础上, 通过引入和分析VIIRS热红外波段的亮度温度及其差异进行辅助识别。随机提取薄、碎云以及厚云在高亮地表 (包括沙漠、裸地、岩石和冰雪等) 和暗地表水体上VIIRS数据热红外通道的亮度温度特性及反射波段的反射率特性, 结果如图3 (a) 所示。由图3 (a) 可知, 沙漠与裸地具有相似的反射能力, 二者的反射率与薄云反射率间的差异较小, 而薄云的反射率波动较大, 导致沙漠与裸地上空的薄云容易漏判。裸地和沙漠的平均亮度温度 (>310K) 明显高于其他地表的, 而冰雪在两个热红外波段的亮度温度分别为261.4 K和264.8K, 三者与薄云的亮度温度之间存在明显差异。在热红外通道, 地表特征的辐射能量是地面发射、地面反射与表面特征反射的总和, 薄云对地面的遮挡使得地面发射和反射的能量降低, 总的辐射能量最低, 亮度温度整体低于其他地表的。统计得到了各类地表在M15和M16波段的平均亮度温度差及标准偏差, 如图3 (b) 所示, 可知, 裸地和沙漠的亮度温度差明显低于其他地表的。

    4.2 精度验证

    为了定量评价不同云识别算法的精度, 采用遥感目视解译的方法提取影像中的云层, 并进行二值化处理, 将处理后的值作为真值。分别计算3种算法在沙漠、裸地、水体和冰雪上的平均总体精度、漏分误差、错分误差和Kappa系数, 结果如表2所示。由表2可知:对于沙漠地区, UDTCDA和VCM产品的云识别总体精度分别约为84.56%和86.34%, 漏判现象比较明显, 漏分误差均大于15%;而所提算法能够较准确地识别沙漠上的云层, 总体精度最高, 为91.98%, 漏分和错分误差最小, Kappa系数达到0.805。与沙漠地区的评价结果类似, 裸地和冰雪表面的地表反射率也较高, 与薄、碎云形成的混合像元导致云识别比较困难, 因此, 尽管UDTCDA和VCM产品具有整体较高的检测精度, 但云识别的不确定性也比较明显, 特别是在冰雪表面, 仍存在明显的错分现象, VCM产品将大量冰雪错识别为云;而所提算法能够改进这些高亮地表上空的整体云识别精度, 漏分误差和错分误差最小, 对裸地、冰雪的Kappa系数分别为0.809和0.801。相对于高亮地表, UDTCDA和VCM这两种产品对反射率相对较低的水体的云识别效果相对较好, 错分误差较小, Kappa系数较大, 但漏分现象增加;而所提算法能够更准确地识别水体上的云, 总体精度为95.58%, 漏分和错分误差均小于5%, Kappa系数为0.868。可见, 改进的VIIRS云检测算法能够整体改善不同地表的云识别结果。总体上, 与其他两种算法相比, 所提算法的云识别效果最好, 平均总体精度为93.23%, 平均漏分误差和错分误差最小, 均小于8%, 平均Kappa系数为0.821。

    5 结论

    针对当前VIIRS云检测算法对遥感影像中薄、碎云检测困难的问题, 提出了一种适用于VIIRS数据的I-DTCDA算法。该算法基于适用多源传感器的UDTCDA算法, 利用6S模型模拟不同观测条件和大气条件下VIIRS数据在可见光到近红外波段的动态阈值云检测模型。同时, 为了提高不同地表上空薄云检测的精度, 结合固定阈值法, 分析了薄云与典型地物在热红外通道的亮度温度差异, 并改进了冰雪识别方法。最后, 采用遥感目视解译结果, 对所提算法的云识别结果进行精度验证, 并与UDTCDA和VCM产品进行对比。结果表明:相比于UDTCDA和VCM的云检测结果, I-DTCDA能够更准确地识别影像中的大部分云层, 云识别结果与遥感目视解译结果具有更高的一致性 (R2=0.9895) , 估计偏差更小 (HRMSE=0.0088, HMAE=0.0049) 。特别地, 所提算法明显改善了沙漠、裸地、水体和冰雪等地表上空的薄云识别效果, 特别是对于云边缘的识别更加细致, 空间连续性较好。所提算法的云识别平均总体精度为93.23%, 平均漏分误差和错分误差均小于8%, 平均Kappa系数为0.821。尽管所提算法的整体精度较高, 但仍存在一定的局限性。例如, 由于沙尘的反射和温度特性与薄云极为相似, 沙尘识别本身较为困难, 因此, 所提算法未考虑沙尘的影响。在地表反射率变化明显的区域, 地表反射率数据的准确性会在一定程度上影响云的识别精度。
