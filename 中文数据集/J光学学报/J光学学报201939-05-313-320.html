    马识途 郭大波 薛哲 贺超

    摘    要：

    在连续变量量子密钥分发 (CVQKD) 多维数据协调过程中, 低密度奇偶校验码 (LDPC) 的纠错性能直接影响协调效率和传输距离。构造了一种双边类型的低密度奇偶校验码 (TET-LDPC) , 引入了类似于重复累积码中的累积结构以提高其纠错性能, 在多维数据协调算法中得到了更小的收敛信噪比、更高的协调效率以及更远的传输距离。仿真结果表明:当TET-LDPC的码率为0.5, 分组码长为2×105时, 收敛信噪比降至1.02dB, 协调效率达到了98.58%, 安全密钥率达到17.61kb/s, CVQKD系统的传输距离提高为44.9km。

    Multidimensional Reconciliation for Continuous-Variable Quantum Key Distribution Based on Two-Edge Type Low-Density Parity-Check Codes

    Ma Shitu Guo Dabo Xue Zhe He Chao

    College of Physics and Electronic Engineering, Shanxi University

    Abstract：

    In the multidimensional reconciliation process of continuous-variable quantum key distribution (CVQKD) , the error correction performance of the low-density parity-check code (LDPC) directly affects the reconciliation efficiency and transmission distance.Herein, a two-edge type low-density parity-check code (TET-LDPC) is constructed.We introduce a cumulative structure similar to that of a repeat-accumulate code into the TET-LDPC to improve its error correction performance.These codes obtain a smaller convergence signal-to-noise ratio, whereas the reconciliation system achieves higher coordination efficiency and longer transmission distance.The simulation results indicate that at a TET-LDPC code rate of 0.5 and a block length of 2×105, the convergence signal-to-noise ratio of the system is reduced to 1.02 dB, the data reconciliation efficiency is 98.58%, the security key rate reaches17.61 kb/s, and the CVQKD transmission distance increases to 44.9 km.

    Keyword：

    quantum optics; quantum key distribution; two-edge type low density parity check code; progressive edge-growth algorithm; multidimensional reconciliation; multi-edge type low density parity check code;

    1 引言

    20世纪80年代, Bennett等[1]在海森堡测不准原理、量子不可克隆定理和单光子不可再分的理论基础上提出了量子密钥分发 (QKD) 和量子密钥分配协议 (BB84协议) , 该协议的完全保密特性得到了证明[2]。在QKD方案中信息以单光子为载体, 但是, 这种单光子源的产生和探测装置复杂, 易受时间抖动和色散效应等的影响, 这限制了QKD的进一步发展。在1999年, Timothy[3]提出了一种基于连续变量的QKD方案, 称为连续变量量子密钥分发 (CVQKD) 协议, 该协议在摆脱单光子低效率限制的同时, 增大了信道容量, 提高了密钥传输速率。

    自1999年开始, 连续变量量子态, 如相干态、压缩态、纠缠态等, 就已经作为信号载体用于CVQKD方案, 并且CVQKD方案的安全性也得到了相应的证明[3,4,5,6]。经过对比, 相干态光源以易获取的优点成为首选方案。这种稳定的光源可以通过使用改进的零拍探测器[7,8]进行捕获, 不易受到干扰, 具有较快的传输速率, 得到了国内外学者的广泛关注。

    在过去几十年, CVQKD技术得到了很大的发展。但是, 传输距离短仍然是CVQKD面临的首要问题。对此, 国内外的学者们进行了大量的研究。2007年, Lodewyck等[9]在25km单模光纤上演示了反向协调的高斯调制相干态的CVQKD协议。2015年, 上海交通大学Wang等[10]在50km光纤中实现了高速的连续变量量子密钥分发。山西大学基于EPR纠缠态实现了无信号调制连续变量量子密钥分发实验, 并开展了全光纤器件连续变量量子密钥分发的研究, 相继实现了30km四态分离调制CVQKD[11]和50km高斯调制CVQKD[12]。2018年, Zhao等[13]证明了多维数据协调在CVQKD中的重要性。

    在CVQKD方案中, 数据协调指从相关连续变量中提取相同信息的过程。2004年, Van Assche等[14]首次提出了基于分层纠错协议 (SEC) 的逆向数据协调算法, 即对连续变量进行多电平的分层量化和编码, 可在一个光脉冲上携带大于1bit的信息, 但是收敛信噪比约为4dB～5dB, 信号传输距离只有30km左右。2008年, Leverrier等[15]提出了一种多维数据协调方法, 其核心思想是通过状态点的球面化过程解决高斯随机变量数值期望为0且易受噪声干扰的问题, 理论上安全距离超过了50km。2014年, 王云艳等[16]在多维数据协调方法中, 使用码率为0.3、码长为2×105的低密度奇偶校验码 (LDPC) 作为纠错码, 传输距离从30km增加到47km, 密钥传输率可以到达8.61kb/s。2016年, 窦磊等[17]对CVQKD多维数据协调算法进行了优化, 在分组码长为106时, 协调效率达到90.27%。

    本文在渐进边增长 (PEG) 算法的基础上, 设计一种双边类型的低密度奇偶检验码 (TET-LDPC) , 将其作为CVQKD多维数据协调中的纠错码, 在分组码长为2×105时, 收敛信噪比降至1.02dB, 协调效率达到98.58%, 安全密钥率达到17.61kb/s, 与文献[16]的结果相比得到了显著提升。

    2 CVQKD多维数据协调方案

    2.1 多维数据协调

    多维数据协调的核心思想是多个高斯随机变量的平方和服从χ2分布, 从而解决因高斯随机变量数学期望为0导致的大量随机信息不易辨别的问题, 并采用多维空间旋转操作, 将高斯信号传入高斯噪声信道中, 将其转化为一个虚拟的二进制输入的高斯噪声信道[15,16,17]。

    在高斯调制的CVQKD协议中, Alice端呈高斯分布的随机数是以0为中心, 在低信噪比时, 随机数X的方差较小, 都集中在0附近, 难以分辨。多维数据协调是将呈高斯分布的d维矢量进行单位球面化, 然后通过旋转操作最大化球面上的信号点距离, 以提高信号辨识度, 如图1所示。

    3 TET-LDPC

    多维数据协调必须依靠性能优越的纠错码, 比如LDPC。LDPC是由Gallager[20]于1962年首次提出的一种信道纠错码。在1995年, David等[21]证明了LDPC优越的纠错性能。并且将LDPC应用到多维数据协调中可提高协调过程的纠错率和降低收敛信噪比, 也得到了论证[16,17]。LDPC是一种线性分组码, 可以通过校验矩阵来定义, 所以构造LDPC就是构造一个对应的稀疏校验矩阵H。

    3.1 PEG算法

    本研究提出的TET-LDPC构造法是一种基于PEG算法的构造方法。PEG算法是Hu等[22]提出的一种校验矩阵构造法。研究表明:使用PEG算法构造得到的不规则LDPC具有非常好的译码性能[23]。PEG算法是根据节点的度分布, 逐步增加边的方法, 在Tanner图中的校验节点和变量节点之间添加新边, 并且在增加新边的同时寻找此变量节点路径最长的校验节点, 尽可能保证对Tanner图中短环的长度 (girth) 影响最小, 从而使本地girth达到最大 (如图3所示) 。

    5 结论

    本研究利用密度进化和差分进化方法找到最佳度分布, 利用PEG算法构造出了TET-LDPC, 在CVQKD多维数据协调算法中进行了仿真, 在码率为0.5, 分组码长为2×105时, 收敛信噪比降至1.02dB, 协调效率可达98.58%, 传输距离超过44km[16], 并且在码长为2×105时, 安全密钥率为17.61kb/s, 达到了国际水平。

    本研究的TET-LDPC的边类型有待增加, 而且可以构造更低码率的MET-LDPC, 以得到更低的收敛信噪比和更高的协调效率。除此之外, 还可以考虑有控制地加入度为1的边, 以及加入删余变量节点, 对门限值的提高和错误地板的降低有一定益处, 从而可用于得到更高的协调效率。
