    摘    要：

    cellular代数是一种近年来备受关注的代数结构.cellular代数有一个指标集Λ, 用以表示这个代数的复杂度.文中研究了单参数cellular代数, 即当Λ只包含一个元素时的情形.一般的cellular代数可以看成由这些最简单的cellular代数合成而得.观察到单参数cellular代数作为集合与某个矩阵代数完全相同, 但是具有一个由某个对称矩阵决定的乘法.此外还详细研究了对应于各种不同对称矩阵时, 相应单参数cellular代数的结构和表示.此外还尝试了在这些代数上构造类似于行列式和迹的函数.

    关键词：

    cellular代数; 结合代数; 迹; 行列式; 转置;

    Structure of Cellular Algebras with a Single Parameter

    YU Hai-yan

    School of Mathematics, Hefei University of Technology

    Abstract：

    Cellular algebra is a algebraic structure received considerable attention in recent years. There is index set Λ which shows complexity of the corresponding cellular algebra in some sense. We study in detail the so called one parameter cellular algebras whose index set Λ contain only one element. General cellular algebras can be viewed as been composed from one parameter cellular algebras. We observe a one parameter cellular algebra is the same thing with certain matrix algebra as a set, but equipped with a different multiplication rule determined by a symmetric matrix. We study in detail the one parameter cellular algebras corresponding to different symmetric matrices, especially their structure and representations. We also try to construct certain functions on those algebras which could be called determinant and trace.

    Keyword：

    cellular algebra; associate algebra; trace; determinant; transposition;

    1 引 言

    近年来, cellular代数受到很多学者的关注, Graham和 Lehrer [1]提出了cellular代数的定义和结构, 并研究了它们的表示.Green[2]介绍了由Hecke 代数构造的cellular代数的结构.Changchang Xi[3,4]证明了partition 代数是cellular代数, 又对cellular代数进行了进一步的讨论.Axler[5]对群和代数的表示给出了详细的介绍.在cellular代数的定义中, 有一个指标集合Λ量度cellular代数的复杂性.本文主要考虑当指标集Λ只含一个λ时的cellular代数.这些情形是最基本的cellular代数, 结构也相对较简单.发现一个参数的cellular代数是由一个对称矩阵决定的.考虑到当这个对称矩阵为单位矩阵时, 所对应的 cellular 代数正好为矩阵代数.文中主要将矩阵代数中的性质在 cellular代数中进行推广.在此感谢导师陈智副教授在写作过程中的细心指导, 考虑代数的行列式函数和迹函数的想法来源于他的建议.

    (C1) Λ是一个偏序集, 且对任意λ∈Λ, M (λ) 是一个有限集.其中C是一个从∏λ∈ΛM (λ) ×M (λ) 到H的单射, 其像是H的一组K基.

    3 结 论

    当对应的对称矩阵为非退化时, 单参数cellular代数同构于一个矩阵代数.一般情况下单参数cellular代数可以看作由一个矩阵代数的子代数和一个平凡代数合成而得.一般情况下单参数cellular代数上也具有类似于行列式和迹的函数.
