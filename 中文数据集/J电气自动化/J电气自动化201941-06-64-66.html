    基于可编程序控制器和组态软件的升降横移式立体车库监控系统设计与仿真

    王晓瑜

    西安航空学院电子工程学院

    摘    要：

    为了提高立体车库控制系统的自动化程度,使用西门子(SIMATIC)S7-1200可编程序控制器(programmable logic controller, PLC)和昆仑通泰(MCGS)组态软件实现对六层升降横移式立体车库监控系统的模拟设计及仿真。介绍了控制系统的方案设计思路及主程序设计流程图, SIMATIC S7-1200系列PLC为下位机主控器,使用MCGS嵌入版组态软件完成上位机监控环境组态。仿真结果表明,可有效实现存车、取车的优化调度和协调控制,结合MCGS组态画面达到立体车库运行状态可视化和控制智能化的目的,系统易于操作,运行可靠,稳定性高。

    关键词：

    升降横移式立体车库; SIMATIC S7-1200可编程序控制器; 博图V13; MCGS; 监控系统;

    作者简介： 王晓瑜(1974-),女,陕西西安人,博士,副教授、高级工程师,现从事电气自动化、机械电子工程教学与科研工作。;

    Design and Simulation of a Monitoring System for Traverse Lift Stereo Garages Based on Programmable Logic Controller and Configuration Software

    Wang Xiaoyu

    College of Electronic Engineering, Xi'an Aeronautical University

    Abstract：

    To improve the automation level of stereo garage control systems, simulation design and emulation of a six-floor traverse lift stereo garage was realized by using Siemens SIMATIC S7-1200 programmable logic controller(PLC) and MCGS modular software. Schematic design ideas and main program design flow chart of the control system were introduced. SIMATIC S7-1200 series PLC, as main controller of the lower computer, adopted MCGS embedded configuration software to complete configuration of the monitoring environment of the upper computer. Simulation results indicated that the proposed system could effectively realize optimal dispatch and coordinated control for parking and picking up. Its combination with MCGS configuration pictures could realize visualized operational status and intelligent control of the stereo garage. The proposed system was easy to operate and had reliable operation and high stability.

    Keyword：

    cross slide type lift stereo garage; SIMATIC S7-1200 programmble logic controller(PLC); Portal V13; MCGS; monitoring system;

    近年来,随着城市规模扩大,导致城市生活用地逐渐减小,人们出行停车难已成为棘手问题。因此,可实现最大限度利用空间的立体车库近年来发展迅速。基于此,下位机可编程序控制器(programmable logic controller, PLC)选用西门子SIMATIC S7-1200 作为该系统的主控制器,使用上位机PC在博图V13环境下,编写PLC控制程序,以MCGS嵌入版组态软件进行上位机监控,完成六层升降横移式立体车库监控环境组态。仿真试验证明,该系统可有效实现存车、取车的优化调度和协调控制,结合MCGS组态画面,达到立体车库运行状态可视化和人机交互的目的,系统运行可靠,易于操作,稳定性好,同时符合节约国土资源的要求[1]。

    1 系统总体方案设计

    六层升降横移式立体车库监控系统硬件主要包括上位机PC、下位SIMATIC S7-1200 PLC、信号板SB和信号模块SM及电池板; 软件主要包括下位机PLC控制梯形图、MCGS7.7嵌入版组态监控程序、上位机PC与下位机PLC通信程序等。

    图1 升降横移式立体车库监控系统总体设计方案框图   下载原图

    六层升降横移式立体车库总体设计方案框图如图1所示。SIMATIC S7-1200 PLC作为主控制器,实时实现立体车库存车、取车的自动运行,工控对象则由MCGS嵌入版组态软件在 PC机上建立,实现对SIMATIC S7-1200 PLC的监控;并利用MCGS嵌入版组态软件中的脚本程序,来构建动画,直观、准确地模拟存车、取车的实际运行状况[2]。

    六层升降横移式立体车库的主要功能包括:平台车辆检测、平台人员检测、1#车位停车到位至12#车位到位检测及指示、横移电机的左移和右移控制、升降电机的上行和下行控制,以及紧急情况的电机抱闸控制等。上位机PC通过SIMATIC 7-1200 PLC,控制车库位置的载车板升、降、横移等动作到达地面完成存车/取车的动作。MCGS嵌入版组态界面则完成监控环境组态,实时监控车库系统当前运行状态,包括车辆在1#车位到12#车位运行方向及所在楼层及显示;当前车位占用情况汇总、运行(异常)状态监测、停车、取车信号及选层信号和急停按钮等。监控系统总体设计方案见图1[3]。

    3 软件系统设计

    3.1 控制系统流程图设计

    本设计研究对象为2×6层升降横移式立体车库,采用车库对应位置的载车板进行存取车辆,当选定车库位置,该车库位置的载车板就会通过横移电机及升降电机驱动,实现载车完成横移、升降等运动,到达地面完成存/取车的动作。天井式为本立体车库的结构特点,车库地面一层可进行路径最短,用时最少的停/取车,要求每一层车位共用中间空位进行停/取车的交换车位,在车位被需求时,被选中车位的下方车位需向每层车库的空车位移出一个车位的空间,以供选中的车位进行存取车的动作,根据控制要求,编写立体车库控制系统主程序设计流程图,如图3所示。

    3.3 车库组态界面设计与仿真运行

    MCGS 7.7嵌入版组态软件,为上位机监控软件,利用变量编辑器创建下位机PLC 外部连接变量,将组态画面中操作对象地址对应下位机 PLC的 地址。上位机PC与下位机SIMATIC S7-1214C主控制器采用以太网通信方式,其通信协议为TCP/IP协议,通信接口选择工业以太网的通用网口,PC操作站组态站地址为102.168.0.241,PLC中的IP地址为192.168.0.1,子网掩码设置为默认,查看以太网通信口。设置上位机PC的IP地址前三段与下位机PLC相同,上位机静态IP地址为(192.168.0.4),子网掩码为(255.255.255.0),建立通信连接,可得到PLC与MCGS通信诊断图,若通道号0的数值为0,则代表通信成功。

    在立体车库监控系统的仿真运行中,将博图V13软件环境下调试通过的程序下载至PLC并置位RUN 状态,打开MCGS嵌入版软件下的立体车库监控人机界面,确保下位机PLC与上位机正常通信。

    MCGS人机监控界面如图6所示[7]。此界面显示车位全满,停车输入和取车输入按钮及其他相关按钮为组态状态。

    图6 MCGS人机监控界面   下载原图

    图7 升降横移式立体车库运行组态画面   下载原图

    图7为进入模拟运行画面,显示当车主发现2#、3#、6#、7#、10#和11#车位已占,而4#车位为空,选择该车位操作停车时,车运行至4层天井空位的组态画面。

    仿真结果表明:系统运行正常,符合升降横移式立体车库停车和取车的实际运行规律,达到立体车库控制运行状态可视化和人机交互的目的。

    4 结束语

    本文采用SIMATIC S7-1200 PLC 作为主控器,在V13 TIA博途软件环境下,编写控制系统梯形图,使用上位机PC及MCGS7.7嵌入版组态软件组建立体车库的组态画面,模拟实现六层升降横移式立体车库的控制系统设计。仿真试验证明,该系统可监视立体车库实际运行工况,还可在上位机上通过MCGS组态画面,有效实现存车、取车的优化调度和协调控制,系统运行可靠,易于操作,稳定性好,同时可以减少占地面积。
