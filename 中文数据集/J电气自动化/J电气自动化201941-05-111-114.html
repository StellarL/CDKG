    基于PC端的智能变电站IRIG-B(DC)测试方法研究

    王治国 孙浩 于哲 笃峻 高晟辅

    南京南瑞继保电气有限公司

    摘    要：

    IRIG-B(DC)对时是智能变电站的一种常用对时方式,如何提高检测效率,具有重要意义。通过对组成波形的三种码元分析,提出了一种基于PC侧实现B码对时编码与解码的新方法。方法充分利用PC机优势,通过串口向装置提供B码对时服务或解析授时装置的B码输出,并完成数据存储和波形实时展示。同时利用被测装置提供的时间查询变量服务,实现了闭环自动测试。试验结果表明,方法使用简单、性能可靠,具有较好的使用价值。

    关键词：

    IRIG-B(DC); 码元; 时钟同步装置; 智能变电站; 继电保护装置;

    作者简介： 王治国(1978-),男,河南人,高级工程师,主要研究方向:电力系统继电保护及其智能化测试技术。;

    Research on Smart Substation IRIG-B(DC) Testing Methods Based on PC Terminal

    Wang Zhiguo Sun Hao Yu Zhe Du Jun Gao Shengfu

    NARI Relays Co., Ltd.

    Abstract：

    IRIG-B(DC) is a common time check method for smart substations. How to improve its detection efficiency is of great significance. In this paper, a novel method based on the PC side was proposed to realize encoding and decoding for code B time check through analysis of three code elements of its constituent waveform. According to this approach, PC advantages were fully used to provide code B time check service to the device or analyze code B output to the timing device through the serial port, and complete data storage and real-time waveform display. Furthermore, by using the time query variable service offered by the tested device, the system could realize automatic closed-loop tests. Test results showed that the presented method was simple to use and had reliable performance and good use value.

    Keyword：

    IRIG-B(DC); code element; clock synchronization device; smart substation; relay protection device;

    电力系统智能变电站建设中所有二次设备及监控系统必须与授时装置时间同步[1],如果现场环境复杂,授时装置接收GPS信号受天气、时钟晶振和线路传输等因素干扰,会导致时间特性发生变化。这些异常时间信息会给继保装置的正常运行带来一定影响,甚至引起继电保护装置的误动[2,3]。

    因此电力二次设备无论在变电站现场还是厂内,对其时间特性的检测,都是一个不可或缺的硬性指标。文献[4,5,6]提出了一种基于现场可编程门阵列的IRIG-B(DC)时间码解码的设计方法,具有体积小、成本低和工作稳定等特点。文献[7]提出了一种利用高性能FPGA实现解调IRIG-B(AC)码的解码器,它通过调用FPGA的IP核生成乘法器与FIR低通滤波器,将B码中的交流分量滤除掉,并根据其幅值进行解调的新方法。以上设计虽具有精度高等优点,但无法自动检测继保装置的时间特性,更不能长期跟踪授时装置实现其信号异常输出的自动捕获。

    本文通过对组成IRIG-B(DC)波形三种码元的分析,在得出三种码元遵循异步串行通信规律的基础上,提出了一种基于PC侧的IRIG-B(DC)编码和解码的新方法。最后开发了B码时间测试系统软件。该软件充分利用PC机数据存储、图形化操作与展示等优势,提供了多种B码码元异常测试检测功能。该方案解决了以往智能变电站和厂内授时装置与继保装置时间异常不能快速有效检测的问题,降低了电力系统运行风险。

    1 IRIG-B(DC)对时技术

    IRIG-B采用串行时间码发布对时,时间帧速率为1帧/s,传递100字节信息量。作为IRIG-A、B、D、E、G、H中应用最为广泛的时间码,IRIG-B具有携带信息量大,经译码后可获得1、10、100、1 000 c/s 的脉冲信号,时间信息采用BCD编码,控制功能信息及高分辨率等特点。该方式是智能变电站普遍采用的对时方式,对时精度可达亚毫秒级[8]。授时装置IRIG-B码是每秒一帧的时间串行码,波形见图1。图中每个码元的宽度是10 ms,每10个码元有一个位置识别标志P。一个帧周期前60个码元能够表示包含秒、分、时、日、年等时间信息,如图1所示。

    2 基于PC端的IRIG-B编解码原理分析

    通过PC端串口发送和解析B码波形,需深入了解串行通信协议。根据起止式异步通信协议的特性,结合B码三种码元特点,来完成对时波形的发送和解析。按照串行通信协议标准,数据起始位输出为逻辑0(低电平,表示数据同步),总线空闲及停止位输出逻辑1(高电平,表示数据帧结束)。UART串口通信对应的波特率设置为1 000 bps,数据帧格式为1位起始位,8位数据位(要传送的数据信息),1位停止位,无校验等特性,报文格式如图2所示。图3中信号A是组成IRIG-B波形的三种码元:位置码元“P”,逻辑“0”,逻辑“1”。三种码元波形不同,但持续时间相同,都是10 ms。从报文传输的角度来看,三种码元均遵循异步串行通信的规律。而一帧B码波形有100个码元组成,每个码元10 ms,一帧波形持续1 000 ms,而以上串行通信设置刚好能满足这一刚性需求,也就是说通过计算机串口可以输出IRIG-B(DC)时间帧波形。

    3.1 对时编码功能设计

    系统对指定时间数据进行编码,按照设置输出正常状态或异常状态下的时间数据波形,用以测试保护和测控装置接收时间信息后的逻辑动作,包括接受或放弃对时。

    Binary-Coded Decimal(BCD)码是一种二进制数字编码,是用二进制编码的十进制代码。该编码利用四个位元来储存一个十进制数码,使二进制和十进制之间得以快捷转换。B码数据帧采用BCD码对秒、分、时、日、世纪年等进行编码。B码每1 s输出一帧波形,一帧数据包含100字节。为保证B码波形连续性,系统通过Windows底层API函数WriteFile将100字节的编码数据,再通过异步操作写入串口接收缓冲区,并组织下1 s数据包。而写入缓冲区的数据有计算机根据BPS设置进行数据流的输出。由于装置接收该时间帧时,实际时间已经为下1 s,因此在传输时间时,需将这1 s加上去。

    B码异常对时测试是继电保护研发测试中的重要组成部分。基于PC端的图形化B码对时测试系统主要提供以下异常测试功能,各种异常码元设计如图5所示。异常对时测试功能如表1所示。

    3.2 IRIG-B(DC)录波功能设计

    从图3可以看出,如果IRIG-B(DC)波形正确,那么通过PC侧串口通信解析出来的数据只能是0x80、0xFE和0xF0,而实际上各厂家对时装置可能会出现异常输出,造成保护测控装置对时失败。而通过PC端图形化的IRIG-B录值功能可以实现异常问题的精准定位及快速排查。

    B码录值有两大功能:一是全部录值,利用PC机硬盘优势,实现解码数据的全部存储,再按照B码帧格式进行统一解码与分析,这样可以清楚了解每一帧数据的详细情况;二是部分录值功能,仅存储组成B码的三种码元,实时解析年、日、时、分、秒等信息。由于B码每秒输出一帧波形,各厂家对时装置时间输出又是连续的,通过判断相邻两帧解码信息,可以断定某一时刻所出现的问题,比如时间跳变等。

    系统通过Windows底层API函数ReadFile处理来自串口接收缓冲区的数据。每读取一个字节即保存在内存中,并根据测试参数进行相应处理:直接保存或进行判断处理。根据B码帧格式,找出帧头,解析出年、日、时、分、秒等信息。最后将100字节帧数据发往显示工作线程。系统每接收完整一帧数据,就更新一次数据展示和图形展示区。

    4 应用与分析

    基于以上技术开发的变电站时间测试系统软件主界面如图7所示。该软件通过PC机串口向装置发送对时报文及对授时装置进行解码。两种功能都充分利用PC机图形化操作、长期跟踪监视存储不受限制等优势。手动或自动改变B码异常输出状态,任意组合波形异常码,用来测试对时接收逻辑;对授时装置进行B码实时解码并实时显示数字波形及数据。装置产生的异常波动数据,均可记录在案。

    测试系统作为对时测试仪时,串口的GND接到保护测控装置的SYN-上,TX接到SYN+上。该方案已在南瑞继保保护测控装置对时逻辑的自动测试上进行试验验证。而作为时间解码测试仪时,串口的GND接到授时装置RS485输出正上,串口的RX接到授时装置的RS485输出负上,该方案已在南瑞继保PCS-9785授时装置B码录波测试中验证。

    5 结束语

    通过对串行通信编码规范和IRIG-B(DC)三种码元的特点进行深入分析,基于PC侧提出了一种测试IRIG-B(DC)的新方法。该方法充分利用PC侧图形操作和数据存储优势,实现了多种异常码元组合自动测试及录波自动检测。该方法为授时装置对时输出逻辑与继保装置时间接收逻辑的厂内测试和变电站现场测试,提供了简易有效的测试手段。实际应用证明,该方法具有一定的应用价值。
