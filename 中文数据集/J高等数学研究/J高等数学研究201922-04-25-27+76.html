    非负半群上保持对称阵的项秩的可加映射

    霍东华

    牡丹江师范学院数学科学学院

    摘    要：

    本文讨论了非负半群上的保持对称阵的项秩的可加映射, 即设A是非负半群, 2≤l<n, An (0) 为非负半群A上的主对角线上元素都是0的n×n对称阵的全体, T是An (0) →An (0) 的加法映射.若T保持项秩l, 只要2≤t<l, 都存在X0∈An (0) , 使τ (Τ (X0) ) ≥τ (X0) .

    关键词：

    非负半群; 项秩; 可加映射;

    作者简介： 霍东华 (1978-) , 女, 黑龙江人, 博士, 副教授, 代数方向, Email:i94donghua@163.com;

    收稿日期：2018-07-16

    基金： 黑龙江省省属高等学校基本科研业务重点项目 (1354ZD007); 牡丹江师范学院青年一般项目 (QN2018006);

    Additive Maps of Preserving Term Rank of Symmetric Matrices over Nonnegative Semigroup

    HUO Donghua

    School of Mathematical Science, Mudanjiang Normal University

    Abstract：

    In this paper, we discuss the additive maps of preserving term rank of symmetric matrices over nonnegative semigroup. Let A be a nonnegative semigroup, An (0) be the set of all symmetric matrices oforder n over A whose elements in the main diagonal are 0, and T: An (0) →An (0) be an additive map. If T preserves term rank l (2≤l<n) , then, as long as 2≤t<l, there exists X0∈An (0) such that τ (T (X0) ) ≥τ (X0) .

    Keyword：

    nonnegative semigroup; term rank; additive maps;

    Received： 2018-07-16

    1 引言

    矩阵的一行或一列称为矩阵的一条线.两个元素不在同一条线上指既不在同一行上也不在同一列上.

    1987年, Beasley等人提出项秩与零项秩的概念.

    定义1.1[1] 一个矩阵A的项秩τ (A) 是指覆盖A的所有非0元的线数的最小数.

    定义1.2[2] 设A是一个m行n列的矩阵, A的元素只取0或1这两个数, 则称A为一个m×n型的 (0, 1) -矩阵.

    例1.1 矩阵

    A= (0110010100110000011000000)

    覆盖所有非零元的行与列的最小数是4, 故A的项秩是4.

    例1.2 矩阵

    A= (100000011111001100000100000000000000)

    覆盖所有非零元的行与列的最小数是4, 故A的项秩是4.

    1991年, Omladic和Semrl展开了对加法保持谱问题的研究, 用加法映射代替线性映射[3], 并在1993年得到了保持矩阵秩1的结果[4].

    2012年, Kang和Song等人研究了非负半环上线性映射保持 (0, 1) -对称阵的项秩的结果[5].2015年庞丽娜在文献[6]中提出了保持对称阵项秩的问题, 将其推广到非负半群上, 设An (0) 为非负半群A上的主对角线上元素都是0的n×n对称阵的全体, 得到的定理如下:

    定理1.1[6] 设A是一个非负半群, 2≤l<k≤n, T是An (0) →An (0) 的加法映射:

    (1) 若T保持项秩l, 则存在X0∈An (0) , 使

    τ (T (X0) ) ≥τ (X0) , 且τ (X0) =2.

    (2) 若T不保持项秩l, 但T保持项秩k, 且k与l奇偶性相同, 则存在X0∈An (0) , 使

    τ (T (X0) ) >τ (X0) .

    (3) 若T不保持项秩2, 保持项秩k, 则存在X0∈An (0) 使τ (T (X0) ) >τ (X0) 且τ (X0) =2.

    本文在以上文献的基础上, 推广了文献 [6] 中结论 (1) 的内容, 在同样的前提下, 只要2≤t<l, 都存在X0∈An (0) , 使

    τ (T (X0) ) ≥τ (X0) .

    为了证明这个结论, 我们先回顾一些相关的概念和引理.

    2 基本概念

    定义2.1[6] 设 (A, +) 是一个加法半群, 0是 (A, +) 中的零元.若A有非零元且非零元都没有负元, 则称 (A, +) 是非负半群.

    定义2.2[6] 取定a∈A, a≠0, 记Ei, j (a) 为第 (i, j) 位置元素是a, 其它位置元素是0的矩阵, 也称其为单元.

    定义2.3[6] 取定a∈A, a≠0, 记Di, j (a) 第 (i, j) 位置与第 (j, i) 位置元素是a, 其它位置元素是0的对称阵, 也称其为二角原子.

    定义2.4[6] 设 (A, +) 是非负半群, Mm×n (A) 是定义在A上的m×n矩阵的全体, T为Mm×n (A) 到自身的映射.T (A+B) =T (A) +T (B) , ∀A, B∈Mm×n (A) , 则称T加法映射.

    定义2.5[6] 假设T:Mm×n (A) →Mm×n (A) 为映射.若对∀A∈Mm×n (A) , 当τ (A) =k时, τ (T (A) ) =k成立, 其中k≤n, 则称T保持项秩k.

    定义2.6[6] 设A, B∈Mm×n (A) , 若bi, j≠0⇒ai, j≠0, 则称A支配B.记为A≥B或者B≤A.

    定义2.7[6] 设A是一个非负半群, X= (xi, j) , Y= (yi, j) ∈Mm×n (A) . 若存在Z= (zi, j) ∈Mm×n (A) , 使得X=Y+Z, 则称X强支配Y, 记为XY或YX.

    有非负半群上矩阵之间支配的定义, 同样适用于对称阵之间.即设A, B∈An (0) , 若bi, j≠0⇒ai, j≠0, 则A≥B或B≤A.

    引理2.1[6] 设A, B∈Mm×n (A) , 则τ (A) ≤τ (A+B) 及τ (A+B) ≤τ (A) +τ (B) .

    引理2.2[6]A, B∈Mm×n (A) , 若A≤B, 则τ (A) ≤τ (B) .

    引理2.3[6] 设T是Mm×n (A) 到Mm×n (A) 的加法映射, X, Y∈Mm×n (A) , 若XY, 则τ (T (X) ) ≥τ (T (Y) ) .

    3 主要结果

    本文在以上内容的基础上, 做了进一步加强, 结果如下:

    定理3.1 设A是非负半群, 2≤l<n, T是An (0) →An (0) 的加法映射:若T保持项秩l, 只要2≤t<l, 都存在X0∈An (0) , 使

    τ (T (X0) ) ≥τ (X0) .

    证明 若存在项秩为t的矩阵X0∈An (0) , 使得τ (T (X0) ) ≥τ (X0) =t, 则命题成立. 不妨设对任意项秩为t的矩阵X∈An (0) , 都有τ (T (X) ) <t.

    情形1:t和l同为偶数

    不妨设XD12 (a) +D34 (a) +…+Dt-1, t (a) , 其中a∈A, a≠0, 由t<l可知

    τ (D12 (a) +D34 (a) +…+Dt-1, t (a) +Dt+1, t+2 (a)

    +…+Dl-1, l (a) ) =l.

    由T保持项秩l, 知

    τ (T (D12 (a) +D34 (a) +…+Dt-1, t (a) +Dt+1, t+2 (a)

    +…+Dl-1, l (a) ) ) =l.

    于是由引理2.1

    τ (T (D12 (a) +D34 (a) +…+Dt-1, t (a) )

    +τ (T (Dt+1, t+2 (a) +…+Dl-1, l (a) ) ) ≥l

    而由引理2.3

    τ (T (D12 (a) +D34 (a) +…

    +Dt-1, t (a) ≤τ (T (X) ) <t

    故

    τ (T (Dt+1, t+2 (a) +…+Dl-1, l (a) ) ) >l-t.

    取

    X0=Dt+1, t+2 (a) +…+Dl-1, l (a) ,

    则

    τ (T (X0) ) >τ (X0) .

    情形2:t和l同为奇数

    不妨设

    XD12 (a) +D23 (a) +D31 (a) +D45 (a) +

    D67 (a) +…+Dt-1, t (a)

    其中a∈A, a≠0, 因t<l, 故

    τ (D12 (a) +D23 (a) +D31 (a) +D45 (a) +D67 (a) +…

    +Dt-1, t (a) +Dt+1, t+2 (a) +…+Dl-1, l (a) ) =l

    由T保持项秩l, 知

    τ (T (D12 (a) +D23 (a) +D31 (a) +D45 (a) +

    D67 (a) +…+Dt-1, t (a) +Dt+1, t+2 (a) +…

    +Dl-1, l (a) ) ) =l

    于是由引理2.1

    τ (T (D12 (a) +D23 (a) +D31 (a) +D45 (a) +

    D67 (a) +…+Dt-1, t (a) ) +τ (T (Dt+1, t+2 (a) +…

    +Dl-1, l (a) ) ) ≥l

    但是由引理2.3

    τ (T (D12 (a) +D23 (a) +D31 (a) +D45 (a) +

    D67 (a) +…+Dt-1, t (a) ) ) ≤τ (T (X) ) <t

    故

    τ (T (Dt+1, t+2 (a) +Dt+3, t+4 (a) +…

    +Dl-1, l (a) ) ) >l-t.

    取

    X0=Dt+1, t+2 (a) +Dt+3, t+4 (a) +…+Dl-1, l (a) ,

    则

    τ (T (X0) ) >τ (X0) .

    情形3:t为奇数, l为偶数

    不妨设

    XD12 (a) +D23 (a) +D31 (a) +D45 (a) +

    D67 (a) +…+Dt-1, t (a)

    其中a∈A, a≠0, 因t<l, 故

    τ (D12 (a) +D23 (a) +D31 (a) +D45 (a) +D67 (a) +…

    +Dt-1, t (a) +Dt+1, t+2 (a) +Dt+2, t+3 (a) +Dt+3, t+1 (a)

    +Dt+4, t+5 (a) +Dt+6, t+7 (a) …+Dl-1, l (a) ) =l

    由T保持项秩l, 知

    τ (T (D12 (a) +D23 (a) +D31 (a) +D45 (a) +D67 (a) +…

    +Dt-1, t (a) +Dt+1, t+2 (a) +Dt+2, t+3 (a) +Dt+3, t+1 (a)

    +Dt+4, t+5 (a) +Dt+6, t+7 (a) …+Dl-1, l (a) ) ) =l

    于是由引理2.1

    τ (T (D12 (a) +D23 (a) +D31 (a) +D45 (a) +D67 (a) +…+

    Dt-1, t (a) ) +τ (T (Dt+1, t+2 (a) +Dt+2, t+3 (a) +Dt+3, t+1 (a)

    +Dt+4, t+5 (a) +Dt+6, t+7 (a) …+Dl-1, l (a) ) ) ≥l

    但是由引理2.3

    τ (T (D12 (a) +D23 (a) +D31 (a) +D45 (a) +

    D67 (a) +…+Dt-1, t (a) ) ≤τ (T (X) ) <t

    故

    τ (T (Dt+1, t+2 (a) +Dt+2, t+3 (a) +Dt+3, t+1 (a) +

    Dt+4, t+5 (a) +Dt+6, t+7 (a) +…+Dl-1, l (a) ) ) >l-t.

    取

    X0=Dt+1, t+2 (a) +Dt+2, t+3 (a) +Dt+3, t+1 (a) +

    Dt+4, t+5 (a) +Dt+6, t+7 (a) …+Dl-1, l (a)

    则

    τ (T (X0) ) >τ (X0) .

    情形4:t为偶数, l为奇数

    不妨设

    XD12 (a) +D34 (a) +D56 (a) +…+Dt-1, t (a) ,

    其中a∈A, a≠0, 因t<l, 故

    τ (D12 (a) +D34 (a) +D56 (a) +…+Dt-1, t (a) +

    Dt+1, t+2 (a) +Dt+2, t+3 (a) +Dt+3, t+1 (a) +

    Dt+4, t+5 (a) +Dt+6, t+7 (a) +…+Dl-1, l (a) ) =l

    由T保持项秩l, 知

    τ (T (D12 (a) +D34 (a) +D56 (a) +…+Dt-1, t (a) +

    Dt+1, t+2 (a) +Dt+2, t+3 (a) +Dt+3, t+1 (a) +

    Dt+4, t+5 (a) +Dt+6, t+7 (a) …+Dl-1, l (a) ) ) =l

    于是由引理2.1

    τ (T (D12 (a) +D34 (a) +D56 (a) +…+Dt-1, t (a) ) +

    τ (T (Dt+1, t+2 (a) +Dt+2, t+3 (a) +Dt+3, t+1 (a) +

    Dt+4, t+5 (a) +Dt+6, t+7 (a) …+Dl-1, l (a) ) ) ≥l

    但是由引理2.3

    τ (T (D12 (a) +D34 (a) +D56 (a) +…+Dt-1, t (a) ) )

    ≤τ (T (X) ) <t,

    故

    τ (T (Dt+1, t+2 (a) +Dt+2, t+3 (a) +Dt+3, t+1 (a) +

    Dt+4, t+5 (a) +Dt+6, t+7 (a) …+Dl-1, l (a) ) ) >l-t.

    取

    X0=Dt+1, t+2 (a) +Dt+2, t+3 (a) +Dt+3, t+1 (a)

    +Dt+4, t+5 (a) +Dt+6, t+7 (a) …+Dl-1, l (a) ,

    则

    τ (T (X0) ) >τ (X0) .

    定理得证.
