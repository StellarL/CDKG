    空间闭曲线上第二类曲线积分的计算

    张莉莉 郝新生 张小英

    山西农业大学文理学院

    摘    要：

    介绍了具有普适性的计算空间闭曲线上第二类曲线积分的三种方法, 通过求解同一问题体现不同解法之间的区别与联系, 以及各种方法的使用技巧.

    关键词：

    曲线积分; 定积分; 格林公式; 斯托克斯公式;

    作者简介： 张莉莉 (1981-) , 女, 山西广灵人, 硕士, 讲师.研究方向:高等数学的教学及应用.;

    收稿日期：2018-06-02

    基金： 山西农业大学科技创新基金项目 (2017zz06);

    Calculation of Second Type Curve Integral on Spatial Closed Curve

    ZHANG Lili HAO Xinsheng ZHANG Xiaoying

    College of Arts and Sciences, Shanxi Agricultural University

    Abstract：

    This paper introduces three methods for calculating the second type curve integral on spatial closed curve. By solving the same problem with these methods, the differences and connections among them are reflected, as well as the techniques of using these methods.

    Keyword：

    curve integral; definite integral; Green formula; Stokes formula;

    Received： 2018-06-02

    第二类曲线积分是高等数学的主要内容之一, 通过定义可将此类积分转化为定积分.对于平面曲线积分, 还可通过格林公式将其转化为二重积分;对于空间曲线积分, 还可通过斯托克斯公式将其转化为曲面积分.因此, 通过第二类曲线积分的计算, 可将定积分、二重积分、曲线积分以及曲面积分建立起密切的联系.在高等数学的教学过程中, 随着教学内容中积分类型的增多, 学习难度进一步加大, 有效的总结不同类型积分间的区别与联系, 寻找它们之间互相转化的方法, 对有效掌握各种积分的内涵及计算有重要的意义.

    本文通过利用定义转化为定积分、利用投影法转化为平面曲线积分继而使用格林公式转化为二重积分、利用斯托克斯公式转化为曲面积分三种方法对空间闭曲线上的第二类曲线积分的计算进行讨论, 其中第二种方法大多教科书上没有, 方法如下:

    设空间分段光滑的空间曲线Γ在光滑曲面Σ:z=f (x, y) 上, 且Σ在xoy面上的投影区域为Dxy, Lxy为Dxy的取正向的边界曲线, P (x, y, z) , Q (x, y, z) , R (x, y, z) 在Γ上具有一阶连续偏导数, 则

    ∮ΓΡ (x, y, z) dx+Q (x, y, z) dy+R (x, y, z) dz=∮LxyΡ (x, y, f (x, y) ) dx+Q (x, y, f (x, y) ) dy+R (x, y, f (x, y) ) ⋅ (fx´dx+fy´dy) =∮Lxy[Ρ (x, y, f (x, y) ) +R (x, y, f (x, y) ) ⋅fx´]dx+[Q (x, y, f (x, y) ) +R (x, y, f (x, y) ) ⋅fy´]dy=∬Dxy[∂ (Q+R⋅fy´) ∂x-∂ (Ρ+R⋅fx´) ∂y]dxdy

    也可将曲线投影到另外两个坐标面上, 方法类似, 在此不予赘述.此三种方法都具有普适性, 其中前两种方法对不封闭的曲线积分也适用, 下面通过实例说明其有效性.

    本文讨论的三种方法将空间曲线积分与定积分、平面曲线积分、二重积分以及曲面积分紧密的联系在一起, 展示出了不同类型积分之间相互转化的具体方法.当积分曲线的参数方程容易写出, 而转化成的定积分也容易计算时, 可考虑用方法一, 即直接转化为定积分求解, 此种方法是较简单也较容易理解的方法;如若积分曲线的参数方程不易求, 而其在某一坐标面上的投影曲线容易求出, 且利用格林公式转化成的二重积分也容易计算时, 可考虑用方法二, 即转化为平面曲线积分继而转化为二重积分求解, 此种方法可将三维的空间曲线积分降到二维的平面曲线积分进行计算, 且格林公式简单易理解;当空间曲线积分利用斯托克斯公式后转化成的曲面积分容易计算时, 可考虑方法三, 即转化为曲面积分求解, 此种方法要求熟练掌握斯托克斯公式以及曲面积分的计算方法, 难度系数较高.

    此外, 除上述三种方法外, 还可以利用对称性[2,3]、换元法[4]等方法求解第二类曲线积分, 本文介绍的三种方法具有普适性, 对一般问题均可使用, 教师在教学以及学生在学习过程中, 可根据具体情况选择一种较简单的方法进行计算.
