    Abstract：

    Modeled by the method for establishing Hermite-Hadamard type inequalities in existing literatures, Hermite-Hadamard type inequalities for h-F-convex functions satisfying conditions P1 and P2 are given.

    1 引言

    Hermite-Hadamard不等式是Jensen不等式的加细, 即

    fa+b2≤1b-a∫bafxdx≤fa+fb2,

    其中f是区间I上的凸函数, a, b∈I, a<b.

    Hermite-Hadamard不等式在积分估值、建立不等式等方面有着重要作用[1,2,3].关于Hermite-

    Hadamard不等式的证明及几何意义可分别参见文[4]和[5].连续凸函数的Hermite-Hadamard不等式等号成立当且仅当该函数是线性函数[6].

    随着新的类型的凸函数概念不断被众多学者提出, 对相应的Hermite-Hadamard型不等式的研究也取得了许多结果[4,7,8,9].1995年, Mohan S R与Neogy S K在研究预不变凸函数时提出了条件 C[10].

    定义1[10] 设K⊂R是关于η:K×K→R的不变凸集, 若对任意x, y∈K, t∈[0, 1], 有

    ηy, y+tηx, y=-tηx, y,

    ηx, y+tηx, y=1-tηx, y,

    则称η满足条件C.

    文[11,12]定义了广义凸集和 F-G广义凸函数等概念, 并将条件C推广为条件P1、P2.

    定义2[11,12] 设KRn, 若存在向量值映射F:K×K×[0, 1]→Rn, 使得λ∈[0, 1], x, y∈K, 都有F (x, y, λ) ∈K.则称K是关于F的广义凸集.

    定义3[11,12] 设KR是关于F的广义凸集, 若α, β∈[0, 1], 且α<β, x, y∈K, 有

    P1:Fy, Fx, y, β, 1-αβ=Fx, y, α;

    P2:Fx, Fx, y, α, β-α1-α=Fx, y, β,

    则称F在K上满足条件P1、P2.

    引理1[11,12,13] 设KR是关于F的广义凸集, 若F在K上满足条件P1、P2, 则λ∈[0, 1], u1, u2∈[0, 1], u1≠u2, x, y∈K, 有

    Fx, y, λu1+1-λu2=

    FFx, y, u1, Fx, y, u2, λ.

    设η:K×K→R满足条件C, 在引理1中取

    Fx, y, t=y+tηx, y, 则得到

    ηy+u1ηx, y, y+u2ηx, y=u1-u2ηx, y,

    其中x, y∈K, u1, u2∈[0, 1].

    文[14]引入s-F凸函数的概念.文[15]引入h-F凸函数的概念, 推广了s-F凸函数.

    定义4[15] 设集合KRn是关于F的广义凸集, f:K→[0, +∞) , [0, 1]JR.h:J→[0, +∞) 且h不恒为零, 若λ∈[0, 1], x, y∈K, 有

    fFx, y, λ≤hλfx+h1-λfy,

    则称f是K上的h-F凸函数.

    定理1[15] 设KR是关于F的广义凸集, F在K上满足条件P1、P2, f:K→[0, +∞) 是K上的h-F凸函数, h∈L[0, 1], x, y∈K, fFx, y, t∈L[0, 1], 则有

    12h12fFx, y, 12≤∫10fFx, y, tdt≤

    fx+fy∫10htdt. (1)

    在定理1中取ht=ts, s∈0, 1, 则有

    2s-1fFx, y, 12≤∫10fFx, y, tdt. (2)

    文[15]还通过建立积分恒等式得到h-F凸函数的其它Hermite-Hadamard型不等式, 这给了我们很大启发.我们发现条件P1、P2在建立积分恒等式时起了重要的作用.利用条件P1、P2还可以尝试将推导经典凸函数的Hermite-Hadamard不等式的方法移植到h-F凸函数.比如, 设f是区间[a, b]上的凸函数, 利用积分恒等式及中点凸性得

    1b-a∫bafxdx=

    1b-a∫a+b2afxdx+∫ba+b2fxdx=

    12b-a∫bafa+x2+fx+b2dx≤

    12b-a∫bafa+fx2+fx+fb2dx=

    fa+fb4+12b-a∫bafxdx,

    由此证得Hermite-Hadamaerd不等式的右边部分.将这种方法移植到h-F凸函数就有

    ∫10fFa, b, tdt=

    ∫120fFa, b, tdt+∫112fFa, b, tdt=

    12∫10fFa, Fa, b, t, 12dt+

    12∫10fFb, Fa, b, t, 12dt≤

    12∫10h12fa+h12fFa, b, tdt+

    12∫10h12fb+h12fFa, b, tdt,

    由此得到h-F凸函数的Hermite-Hadamard型不等式 (在定理4中将给出这个不等式的另证)

    21-h12∫10fFa, b, tdt≤

    h12fa+fb.

    关于h-F凸函数的其它结果可参见文献[15,16,17].本文的目的是建立h-F凸函数的一些新的Hermite-Hadamard型不等式.

    定义5[18] 设h是区间J上的非负函数, 如果对于任意x, y∈J, 有

    hxy≥hxhy, (3)

    则称h是J上的超可乘函数.若式 (3) 反向, 则称h是J上的次可乘函数.若式 (3) 等号成立, 则称h是J上的可乘函数.
