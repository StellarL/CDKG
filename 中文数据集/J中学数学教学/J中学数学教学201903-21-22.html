    割补法在三视图还原空间几何体中的应用

    何业亮

    安徽省芜湖市沈巷中学

    收稿日期：2019-04-08

    Received： 2019-04-08

    三视图是高中新教材新增内容, 也是高考的常考知识点.不难发现, 由三视图还原几何体的试题经“考”不衰, 且其载体的复杂程度呈增加之势, 特别是以简单组合体为载体的试题备受命题专家亲睐.虽然很多学生能将“长对正, 高平齐, 宽相等”烂记于心, 也对简单几何体的三视图了如指掌, 但处理这类问题仍力不从心, 无法可依, 甚至不少教师的讲解和很多资料的答案往往是直接告知几何体, 不作详细解释.笔者经过深入研究, 发现割补法 (对三视图进行分割或修补) 能有效处理复杂的三视图还原几何体问题, 且简单易行, 直观明了.

    1 合理分割, 各个击破

    例1 (2015·重庆·文5) 某几何体的三视图如图1所示, 则该几何体的体积为 ( )

    A.13+2πB.13π6C.7π3D.5π2

    图1   下载原图

    解析 若整体认识, 很难得出几何体的结构特征.我们将三视图分割成两部分, 如图1.1-1.2所示.易知:图1.1是底面半径为1, 高为1的半圆锥的三视图, 图1.2对应底面半径为1, 高为2的放倒的圆柱, 所以该几何体是半圆锥与圆柱拼接而成的简单组合体, 则该几何体体积为V=12×13×π×12×1+π×12×2=13π6, 故选B.

    例2 (2014·安徽·理7) 一个多面体的三视图如图2所示, 则该多面体的表面积为 ( )

    A.21+3B.18+3C.21 D.18

    解析 此题易被正方形中的虚线和实线所迷惑, 从而无法得出该多面体的直观图.我们可以将三视图分割成三部分后重新组合, 如图2.1-2.3所示.

    易知图2.1是正方体的三视图, 图2.2和图2.3均是三棱锥 (有三条共端点、两两垂直且长度都是1的棱) 的三视图.所以本题中的几何体是棱长为2的正方体从后面右上角和前面左下角分别截去以一个小三棱锥后剩余的部分 (如图2.4) , 所以该多面体的表面积为S=6×22选A.

    图2   下载原图

    评述 在“长对正, 高平齐, 宽相等”的前提下, 可将一些复杂的三视图分割成若干个熟悉几何体或简单几何体的三视图, 后将所得的那些几何体进行符合题意的组合, 即可得所求几何体.

    2 恰当修补, 无中生有

    例3 (2014·重庆·理7) 某几何体的三视图如图3所示, 则该几何体的表面积为 ( )

    A.54 B.60 C.66 D.72

    图3   下载原图

    解析 本题很多学生不能准确地还原几何体.我们可以将三视图修补成图3.1, 易知图3.1对应的是底面为直角三角形的直三棱柱, 结合图3将此三棱柱截去一个同底且高为3的三棱锥, 如图3.2所示, 即为符合题意的几何体.所以该几何体的表面积为S=12×3×4+12×3×42+ (5-2) 2+12× (5+2) ×4+12× (5+2) ×32+42+3×5=60, 故选B.

    例4 (2012·湖北·理4) 已知某几何体的三视图如图4所示, 则该几何体的体积为 ( )

    A.8π3B.3π C.10π3D.6π

    图4   下载原图

    解析 很多学生看到图中两个圆, 会联想到与球有关, 最后发现怎么都得不到正视图, 进而束手无策.我们可将三视图修补成图4.1, 这样便得到一个底面半径为1, 高为4的圆柱的三视图.最后联系图4, 可将圆柱截去一部分 (半径为1, 高为2的圆柱的一半) , 便可得到图4对应的几何体, 如图4.2.所以该几何体的体积是半径为1, 高为4的圆柱的体积的34, 即V=34×π×12×4=3π, 故选B.

    评述 本着“长对正, 高平齐, 宽相等”的原则, 可将有些三视图修补成一个简单几何体或熟悉几何体的三视图, 然后在所得几何体上进行恰当的添补或剪裁, 直到得到符合题设的几何体.

    3 割补合用, 双管齐下

    例5 (2014·辽宁·理7) 某几何体的三视图如图5所示, 则该几何体的体积为 ( )

    A.8-2π B.8-π C.8-π2D.8-π4

    图5   下载原图

    解析 本题可先将图5的俯视图补成正方形, 再将所得三视图分成三部分, 如图5.1-5.3所示.由于图5.1是正方体的三视图, 图5.2和5.3图对应的几何体都是14圆柱, 则图5对应的几何体是棱长为2的正方体切去两个体积相等的14圆柱.所以所求几何体的体积为V=23-2×14×π×12×2=8-π, 故选B.

    评述 有些三视图还原几何体的问题, 可对三视图进行分割和修补, 这样能有效地转化为熟悉几何体的三视图, 但是一定要遵循“长对正, 高平齐, 宽相等”的原则, 否则对后续几何体的确定不利.

    由此可见, 处理复杂的三视图还原空间几何体问题, 割补法有着奇效, 可以化陌生为熟悉, 化复杂为简单.
